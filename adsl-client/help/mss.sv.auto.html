Begränsa paketstorleken?
Det här alternativet kan användas för att begränsa den maximala storleken på paket som skickas ut via din ADSL-anslutning. Om du har ett internt LAN som är ansluten till Internet via ADSL, bör det ställas in på 1412 för att förhindra problem med överdrivna Ethernet-paketstorlekar.
