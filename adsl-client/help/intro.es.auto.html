El módulo de cliente ADSL
Este módulo le permite configurar el programa cliente ADSL RP-PPPoE, que se puede utilizar para establecer una conexión a Internet a través de cualquier módem ADSL conectado a Ethernet que admita el protocolo PPPoE (PPP sobre Ethernet). No funcionará con módems ADSL conectados por USB, ni con aquellos en modo puente.

Antes de poder usarlo, debe tener una cuenta con un proveedor de servicios ADSL, y su módem debe estar conectado a una interfaz Ethernet no utilizada en su sistema Linux. Esta interfaz debe ser visible para el núcleo, pero no debe tener asignada ninguna dirección IP.
