¿Conecte bajo demanda?
Cuando se selecciona No, la conexión ADSL solo se iniciará cuando se solicite explícitamente. Pero si se elige Sí, se abrirá cuando sea necesario y se mantendrá activo mientras haya tráfico que lo atraviese. El tiempo de espera determina cuánto tiempo puede permanecer inactiva la conexión antes de que se cierre.
