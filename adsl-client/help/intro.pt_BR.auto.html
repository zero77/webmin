O Módulo Cliente ADSL
Este módulo permite configurar o programa cliente RP-PPPoE ADSL, que pode ser usado para estabelecer uma conexão com a Internet através de qualquer modem ADSL conectado à Ethernet que suporte o protocolo PPPoE (PPP over Ethernet). Ele não funciona com modems ADSL conectados por USB, nem com aqueles no modo de ponte.

Antes de poder usá-lo, você deve ter uma conta com um provedor de serviços ADSL e seu modem deve estar conectado a uma interface Ethernet não usada no sistema Linux. Essa interface deve estar visível para o kernel, mas não deve ter nenhum endereço IP atribuído.
