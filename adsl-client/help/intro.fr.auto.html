Le module client ADSL
Ce module vous permet de configurer le programme client RP-PPPoE ADSL, qui peut être utilisé pour établir une connexion Internet via tout modem ADSL connecté à Ethernet qui prend en charge le protocole PPPoE (PPP sur Ethernet). Il ne fonctionnera pas avec les modems ADSL connectés par USB, ni avec ceux en mode pontage.

Avant de pouvoir l'utiliser, vous devez avoir un compte auprès d'un fournisseur de services ADSL et votre modem doit être connecté à une interface Ethernet non utilisée sur votre système Linux. Cette interface doit être visible pour le noyau, mais aucune adresse IP ne doit lui être attribuée.
