Server Webmin del cluster
Questo modulo consente di gestire moduli, temi, utenti e gruppi su più server Webmin da un'unica interfaccia. Combina le funzioni dei moduli Configurazione Webmin e Utenti Webmin con la possibilità di eseguire azioni (come l'installazione di un tema o la creazione di un utente) su più server contemporaneamente.

La parte superiore della pagina principale sotto l'intestazione Server gestiti elenca altri server Webmin i cui moduli e utenti sono gestiti da questo modulo. Per aggiungere un server a questo elenco, è necessario innanzitutto aggiungerlo al modulo Server Webmin, con un nome utente e una password specificati per accedere a Webmin su quel server. È quindi possibile selezionare il server dall'elenco accanto al pulsante Aggiungi server.

Quando viene aggiunto un server, verrà verificato per assicurarsi che stia eseguendo una versione supportata di Webmin (0.985 o successive) e che abbia i moduli necessari installati. Gli elenchi di tutti i moduli, temi, utenti e gruppi dal server verranno quindi scaricati e memorizzati nella cache in locale.

Una volta che c'è almeno un'icona in Server gestiti, è possibile utilizzare i pulsanti in Utenti e gruppi di Webmin per modificare, creare o impostare ACL su utenti e gruppi su qualsiasi server. Gli utenti e i gruppi creati verranno creati su tutti i server gestiti, ma quelli già esistenti che vengono modificati o eliminati verranno modificati solo sui server su cui già esistono.

Nella parte inferiore della pagina in Moduli e temi sono presenti pulsanti per la modifica di moduli e temi installati su qualsiasi server gestito e un modulo per l'installazione di un nuovo modulo o tema su tutti i server.
