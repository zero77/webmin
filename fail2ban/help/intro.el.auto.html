Ανιχνευτής εισβολής Fail2Ban
Το Fail2Ban είναι ένας διακομιστής που σαρώνει αρχεία καταγραφής για καταχωρήσεις που υποδεικνύουν αποτυχημένες συνδέσεις ή άλλες επιθέσεις και στη συνέχεια εκτελεί ενέργειες όπως τείχος προστασίας ή άλλον αποκλεισμό των πηγών αυτών των επιθέσεων. Αυτό μπορεί να χρησιμοποιηθεί για να αποφευχθούν απόπειρες μαντείας κωδικού πρόσφυσης με μπλοκάρισμα του εισβολέα πριν μπορέσει να δοκιμάσει ένα ευρύ φάσμα κωδικών πρόσβασης.

Οι τρεις κύριοι τύποι αντικειμένων διαμόρφωσης στο Fail2Ban είναι:

Φίλτρα καταγραφής
Ένα φίλτρο είναι βασικά μια κανονική έκφραση που μπορεί να εφαρμοστεί σε ένα αρχείο καταγραφής για αντιστοίχιση αποτυχημένων συνδέσεων ή άλλων επιθέσεων.

Ενέργειες αγώνα
Μια ενέργεια είναι ένα σύνολο εντολών που εκτελούνται για τον αποκλεισμό μιας επίθεσης. Μια ενέργεια μπορεί επίσης να ορίσει εντολές για κατάργηση αποκλεισμού μιας IP και εντολές που εκτελούνται όταν το Fail2Ban ξεκινά ή τερματίζεται.

Φίλτρο φυλακών δράσης
Το jail είναι ένας συνδυασμός φίλτρου, μιας ή περισσότερων ενεργειών και ενός ή περισσότερων αρχείων καταγραφής. Το αρχείο καταγράφεται συνεχώς για γραμμές που ταιριάζουν με το φίλτρο και όταν βρεθεί, εκτελούνται οι επιλεγμένες ενέργειες.
