Encaminhar nomes de usuário não qualificados para o host
A menos que a opção Entrega local seja escolhida, qualquer email enviado por usuários locais para um nome de usuário não qualificado (como jcameron ) será encaminhado para o host especificado. Útil se o seu sistema é uma estação de trabalho compartilhada e você deseja que todo o correio vá para um servidor de correio comum.
