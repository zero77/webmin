Псевдоним Для
Каждый псевдоним может перенаправлять почту нескольким получателям, и каждый получатель может быть одним из следующих:
Адрес электронной почты
Либо полный адрес электронной почты, например, foo@bar.com , либо имя другого пользователя в вашей системе, например, jcameron .
Адреса в файле
Для этого типа псевдонима вы должны указать полный путь к файлу, содержащему список адресов электронной почты, по одному на строку. Каждый адрес может быть либо полностью квалифицированным, либо именем другого пользователя в вашей системе.
Написать в файл
В этом случае письмо будет записано в указанный файл. Чтобы выбросить всю электронную почту на какой-либо адрес, укажите /dev/null качестве имени файла.
Подача в программу
Каждый раз, когда приходит электронное сообщение для этого псевдонима, указанная программа будет выполняться, и все сообщения электронной почты (заголовки и тело) будут переданы ему на стандартный ввод.
