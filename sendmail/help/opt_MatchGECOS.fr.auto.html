Accepter le courrier pour les vrais noms des utilisateurs?
Si cette option est activée, sendmail acceptera les e-mails dont la partie nom d'utilisateur est le vrai nom d'un utilisateur plutôt que simplement son nom de connexion. Par exemple, si l'utilisateur Joe Bloggs avec login jbloggs existait sur votre système, vous pouvez lui envoyer des messages à Joe.Bloggs@votredomaine.com ainsi que jbloggs@votredomaine.com.
