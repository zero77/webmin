Leiten Sie E-Mails weiter, damit lokale Benutzer sie hosten können
Sofern nicht Lokal zustellen ausgewählt ist, werden alle E-Mails, die normalerweise an lokale Benutzer zugestellt werden, an den angegebenen Host weitergeleitet. Nützlich, wenn Ihr System eine gemeinsam genutzte Workstation ist und alle E-Mails an einen gemeinsamen Mailserver gesendet werden sollen.
