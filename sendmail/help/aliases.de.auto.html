Mail Aliase
Mail-Aliase sind die Methode von sendmail, um E-Mails von einer Adresse auf Ihrem System an ein anderes Ziel umzuleiten. Dieses Ziel kann ein Benutzer auf Ihrem System, eine E-Mail-Adresse auf einem anderen System, eine Datei oder die Eingabe in ein Programm sein. Aliase gelten für alle Domänen, für die Ihr System E-Mails akzeptiert. Wenn Ihr System also E-Mails für die Domains abc.com und xyz.com akzeptiert , leitet ein Alias für die Adresse foo E-Mails weiter, die sowohl an foo@abc.com als auch an foo@xyz.com gesendet werden .

Oben auf dieser Seite befindet sich ein Formular zur Eingabe der Details eines neuen Mail-Alias. Unter dem Formular befindet sich eine Liste der vorhandenen Aliase und der Ziele für umgeleitete E-Mails. Um einen Alias zu bearbeiten, klicken Sie auf seinen Namen.

Wenn Sie das Majordomo-Modul verwenden, werden Sie möglicherweise mehrere Aliase wie Majordomo , Eigentümerlistenname oder Listennamenanforderung bemerken . Ändern Sie diese Aliase nur, wenn Sie wissen, was Sie tun.
