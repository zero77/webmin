Mode de livraison
Contrôle quand les messages envoyés à ce serveur de messagerie sont remis. Les options possibles sont:
  • Contexte
    Les messages sont livrés dès que possible en arrière-plan.

  • File d'attente uniquement
    Les messages soumis sont simplement placés dans la file d'attente et ne sont pas remis tant que la file d'attente n'a pas été vidée manuellement.

  • Interactif
    Les messages sont livrés immédiatement.

  • Différé
    Comme Queue uniquement , mais aucune recherche DNS ou NIS n'est effectuée pour les messages soumis.
