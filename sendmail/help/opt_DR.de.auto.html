Leiten Sie nicht qualifizierte Benutzernamen an den Host weiter
Sofern nicht Lokal zustellen ausgewählt ist, werden alle E-Mails, die von lokalen Benutzern an einen nicht qualifizierten Benutzernamen (wie jcameron ) gesendet werden, an den angegebenen Host weitergeleitet. Nützlich, wenn Ihr System eine gemeinsam genutzte Workstation ist und alle E-Mails an einen gemeinsamen Mailserver gesendet werden sollen.
