Trier la file d'attente de messagerie par
Contrôle l'ordre dans lequel la file d'attente de messagerie est traitée. Les options possibles sont:
  • Priorité
    Les messages sont traités par ordre de coût estimé des ressources.

  • Nom d'hôte
    Les messages sont classés par nom d'hôte du serveur de destination.

  • Date de réception
    Les messages sont traités dans l'ordre où ils ont été reçus.
