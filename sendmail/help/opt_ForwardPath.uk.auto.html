Файли переадресації користувача
Значення, вказане для цієї опції, повинно бути розділеним двокрапкою списком імен шляхів, що визначає, де sendmail шукає файл переадресації для користувача під час доставки електронної пошти. Змінні $z для домашнього каталогу користувача та $w для імені системи можуть бути використані у шляху.
