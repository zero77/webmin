Senden Sie ausgehende E-Mails über den Host
Sofern die Zustellung nicht direkt ausgewählt ist, leitet sendmail alle ausgehenden E-Mails, die nicht für Benutzer Ihres Systems bestimmt sind, an den angegebenen Host weiter. Dies kann nützlich sein, wenn sich Ihr System hinter einer Firewall befindet und daher keine Internet-E-Mails direkt übermitteln kann.
