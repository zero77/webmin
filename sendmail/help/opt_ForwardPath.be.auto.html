Перанакіраваць файлы карыстальніка
Значэнне, вызначанае для гэтай опцыі, павінна быць падзеленым двукроп'е спісам імёнаў шляхоў, якія вызначаюць, дзе sendmail шукае файл перанакіравання для карыстальніка пры дастаўцы электроннай пошты. У шляху могуць выкарыстоўвацца зменныя $z для хатняга каталога карыстальніка і $w для імя сістэмы.
