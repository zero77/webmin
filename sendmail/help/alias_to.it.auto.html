Alias To
Ogni alias può reindirizzare la posta a più destinatari e ciascun destinatario può essere uno dei seguenti:
Indirizzo email
O un indirizzo email completo come foo@bar.com o il nome di un altro utente sul tuo sistema come jcameron .
Indirizzi nel file
Per questo tipo di alias è necessario specificare il percorso completo di un file contenente un elenco di indirizzi e-mail, uno per riga. Ogni indirizzo può essere completo o il nome di un altro utente sul sistema.
Scrivi su file
In questo caso, l'e-mail verrà scritta nel file specificato. Per avere tutta la posta elettronica inviata a qualche indirizzo, dare /dev/null come nome file.
Feed al programma
Ogni volta che arriva un messaggio di posta elettronica per questo alias, il programma specificato verrà eseguito e l'intero messaggio di posta elettronica (intestazioni e corpo) gli verrà passato sull'input standard.
