Διευθυντής Samba

Αυτή η σελίδα βοήθειας είναι ελλιπής

Εισαγωγή

Το Samba είναι ένα πρόγραμμα δωρεάν λογισμικού που επιτρέπει στους υπολογιστές που εκτελούν Windows 95 και NT να έχουν πρόσβαση σε αρχεία και εκτυπωτές στο σύστημα Unix, σαν να είναι διακομιστής αρχείων / εκτύπωσης των Windows. Ορισμένα συστήματα Unix όπως το Redhat Linux περιλαμβάνουν το Samba ως μέρος της τυπικής εγκατάστασης, ενώ τα περισσότερα άλλα απαιτούν από εσάς να το κατεβάσετε και να το εγκαταστήσετε ξεχωριστά.

Εάν δεν έχετε εγκαταστήσει το Samba στο σύστημά σας, η κύρια σελίδα της μονάδας θα παραπονεθεί ότι δεν μπορεί να βρει το αρχείο διαμόρφωσης Samba και δεν θα μπορείτε να μοιραστείτε αρχεία ή καταλόγους. Εάν έχετε εγκαταστήσει το Samba αλλά λαμβάνετε αυτό το μήνυμα σφάλματος, ελέγξτε τη διαμόρφωση της λειτουργικής μονάδας για να βεβαιωθείτε ότι το Webmin ψάχνει στο σωστό μέρος για το αρχείο διαμόρφωσης και τα προγράμματα Samba.

Υποθέτοντας ότι το Samba έχει εγκατασταθεί σωστά, η κύρια σελίδα της μονάδας θα εμφανίσει μια λίστα με όλα τα αρχεία και τα κοινόχρηστα στοιχεία του εκτυπωτή που ορίζονται στο σύστημά σας. Εάν το Webmin διαπιστώσει ότι ο διακομιστής Samba δεν εκτελείται, θα σας δοθεί η επιλογή απόπειρας εκκίνησης του διακομιστή. Αυτό μπορεί να είναι απαραίτητο εάν το Samba δεν ξεκινήσει κατά την εκκίνηση.

Δημιουργία κοινής χρήσης αρχείων

Ένα κοινόχρηστο αρχείο Samba είναι ένας κατάλογος που μπορεί να τοποθετηθεί από υπολογιστές-πελάτες. Κάθε κοινή χρήση έχει ένα σύντομο όνομα, όπως docs ή www , το οποίο αντιστοιχεί σε κάποιον κατάλογο στο σύστημά σας Unix. Ένα κοινόχρηστο στοιχείο αναφέρεται συνήθως από έναν πελάτη χρησιμοποιώντας τη σημείωση \\server\share .

Για να δημιουργήσετε μια κοινή χρήση, κάντε κλικ στο σύνδεσμο Create File Share κάτω από τη λίστα των υπαρχόντων κοινόχρηστων στοιχείων στην κύρια σελίδα. Αυτό θα εμφανίσει μια φόρμα που σας επιτρέπει να εισαγάγετε τις λεπτομέρειες της νέας κοινοποίησης. Αν και το Samba υποστηρίζει μεγάλο αριθμό παραμέτρων για κάθε κοινή χρήση, οι πιο σημαντικές είναι:

Κοινή χρήση ονόματος Κάθε μετοχή πρέπει να έχει ένα όνομα, το οποίο χρησιμοποιείται από τους πελάτες για να το αναφέρει. Μια ειδική περίπτωση είναι η κοινή χρήση All home directories , που εξηγείται αργότερα.
Μονοπάτι Ο κατάλογος που θα κοινοποιηθεί. Οι πελάτες που χρησιμοποιούν αυτό το κοινόχρηστο στοιχείο μπορούν να έχουν πρόσβαση μόνο σε αρχεία εντός ή κάτω από αυτόν τον κατάλογο.
Εγγράψιμο Καθορίζει εάν οι πελάτες μπορούν να γράψουν σε αρχεία και καταλόγους σε αυτό το κοινόχρηστο στοιχείο.
Πρόσβαση επισκέπτη Εάν ένα κοινόχρηστο στοιχείο επιτρέπει στους επισκέπτες επισκέπτες, οι πελάτες μπορούν να έχουν πρόσβαση σε αυτό χωρίς να χρειάζεται να παρέχουν έγκυρο όνομα και κωδικό πρόσβασης. Ένα κοινόχρηστο στοιχείο μπορεί είτε να απαγορεύει τους επισκέπτες, να επιτρέπει σε επισκέπτες και σε κανονικούς χρήστες, είτε να επιτρέπει σε επισκέπτες μόνο.
Για πληροφορίες σχετικά με άλλες επιλογές κοινής χρήσης, ανατρέξτε στην ενότητα Επιλογές για προχωρημένους παρακάτω.

Μόλις εισαγάγετε τις λεπτομέρειες μιας νέας κοινής χρήσης και κάνετε κλικ στο κουμπί Create , θα είναι άμεσα διαθέσιμη στους υπολογιστές-πελάτες. Υποθέτοντας ότι όλα έχουν ρυθμιστεί σωστά στο δίκτυό σας, οι χρήστες θα μπορούν να χρησιμοποιούν τη λειτουργία Network Neighbourhood στα Windows για να βλέπουν τον διακομιστή σας και όλα τα κοινόχρηστα στοιχεία του. Το νέο κοινόχρηστο στοιχείο μπορεί στη συνέχεια να αντιστοιχιστεί σε ένα γράμμα μονάδας δίσκου στον υπολογιστή.

Δημιουργία κοινής χρήσης εκτυπωτή

Ένα κοινόχρηστο εκτυπωτή Samba είναι ένας τοπικός ή απομακρυσμένος εκτυπωτής Unix που θέλετε να διαθέσετε σε υπολογιστές-πελάτες. Όπως τα κοινόχρηστα αρχεία, κάθε κοινή χρήση εκτυπωτή έχει ένα σύντομο όνομα (όπως hp5si ) και αναφέρεται με τη χρήση της συμβολής \\server\share χρήσης \\server\share .

Για να δημιουργήσετε ένα νέο κοινόχρηστο εκτυπωτή, κάντε κλικ στο Create Printer Share όπως από την κύρια σελίδα. Αυτό θα εμφανίσει μια φόρμα παρόμοια με αυτήν που χρησιμοποιείται για τη δημιουργία κοινόχρηστων αρχείων, στην οποία μπορείτε να εισαγάγετε τις λεπτομέρειες της νέας κοινής χρήσης. Οι πιο σημαντικές επιλογές για ένα νέο κοινόχρηστο εκτυπωτή είναι:

Κοινή χρήση ονόματος Κάθε μετοχή πρέπει να έχει ένα όνομα, το οποίο χρησιμοποιείται από τους πελάτες για να το αναφέρει. Μια ειδική περίπτωση είναι η κοινή χρήση All printers , που εξηγείται αργότερα.
Εκτυπωτής Ο εκτυπωτής Unix για κοινή χρήση. Αυτός μπορεί να είναι οποιοσδήποτε εκτυπωτής στον οποίο θα εκτυπώσατε χρησιμοποιώντας την εντολή lpr .
Μονοπάτι Ο κατάλογος στον οποίο τα προσωρινά αρχεία εκτύπωσης
Πρόσβαση επισκέπτη Εάν ένα κοινόχρηστο στοιχείο επιτρέπει στους επισκέπτες επισκέπτες, οι πελάτες μπορούν να έχουν πρόσβαση σε αυτό χωρίς να χρειάζεται να παρέχουν έγκυρο όνομα και κωδικό πρόσβασης. Ένα κοινόχρηστο στοιχείο μπορεί είτε να απαγορεύει τους επισκέπτες, να επιτρέπει σε επισκέπτες και σε κανονικούς χρήστες, είτε να επιτρέπει σε επισκέπτες μόνο.
Τύπος εκτυπωτή Το όνομα του προγράμματος οδήγησης εκτυπωτή που πρέπει να χρησιμοποιηθεί για αυτόν τον εκτυπωτή. Εάν εισαγάγετε μια τιμή για αυτήν την επιλογή και ταιριάζει με μια μονάδα δίσκου εκτυπωτή σε υπολογιστές-πελάτες, τότε οι χρήστες δεν θα χρειαστεί να επιλέξουν ένα πρόγραμμα οδήγησης εκτυπωτή κατά την προσθήκη αυτού του εκτυπωτή.

Όπως και με τα κοινόχρηστα αρχεία, είναι διαθέσιμες πολλές περισσότερες επιλογές για κοινές εκτυπώσεις. Μόλις προστεθεί ένα κοινόχρηστο εκτυπωτή, θα είναι διαθέσιμο στους πελάτες. Στα Windows 95 και NT, ένας εκτυπωτής μπορεί να προστεθεί χρησιμοποιώντας την επιλογή Add Printer στον πίνακα ελέγχου των εκτυπωτών. Μόλις προστεθεί, ο χρήστης μπορεί να εκτυπώσει σε αυτόν όπως οποιονδήποτε τοπικό ή εκτυπωτή δικτύου.

Δημιουργία αντιγράφου

Επεξεργασία κοινής χρήσης

Για να επεξεργαστείτε μια υπάρχουσα κοινή χρήση, κάντε κλικ στο όνομά της από τη λίστα μεριδίων στην κύρια σελίδα. Αυτό θα εμφανίσει την ίδια φόρμα που χρησιμοποιείται για τη δημιουργία μιας κοινής χρήσης, επιτρέποντάς σας να τροποποιήσετε όλες τις επιλογές και τις παραμέτρους. Για να αποθηκεύσετε τις αλλαγές σας, κάντε κλικ στο κουμπί Apply στο κάτω αριστερό μέρος της σελίδας.

Για να διαγράψετε μια κοινή χρήση, κάντε κλικ στο κουμπί Delete στο κάτω δεξί μέρος της σελίδας. Οι υπάρχοντες χρήστες δεν θα αποσυνδεθούν αμέσως - ωστόσο, κανένας νέος χρήστης δεν θα μπορεί να συνδεθεί στο κοινόχρηστο στοιχείο.

Χρήστες και ασφάλεια

Για κοινές χρήσεις που δεν επιτρέπουν την πρόσβαση επισκέπτη, η Samba αναμένει από τον πελάτη να παρέχει όνομα χρήστη και κωδικό πρόσβασης. Συνήθως, ελέγχονται από τη λίστα χρηστών Unix, η οποία μπορεί να ενημερωθεί από τη λειτουργική μονάδα User Manager . Το Samba μπορεί επίσης να διατηρήσει τη δική του λίστα χρηστών και κωδικών πρόσβασης - δείτε την ενότητα Κρυπτογραφημένοι κωδικοί πρόσβασης για περισσότερες λεπτομέρειες.

Ορισμένες εκδόσεις των Windows δεν επιτρέπουν στο χρήστη να παρέχει ένα όνομα χρήστη κατά την πρόσβαση σε ένα κοινόχρηστο στοιχείο - αντ 'αυτού, το όνομα χρήστη που παρέχεται κατά τη «σύνδεση» στα Windows χρησιμοποιείται. Εάν όλοι οι χρήστες πελατών συνδεθούν στα Windows χρησιμοποιώντας το ίδιο όνομα χρήστη που χρησιμοποιούν για το Unix, τότε αυτό δεν είναι πρόβλημα. Εάν όχι, έχετε 2 επιλογές:

  1. Τα Windows 95 και NT μπορούν να προσαρμόσουν ένα κοινόχρηστο στοιχείο με μη αυτόματο τρόπο, είτε πληκτρολογώντας το net use drive: share στη γραμμή εντολών ή κάνοντας κλικ στο κουμπί της μονάδας δικτύου δικτύου χάρτη σε ένα παράθυρο αρχείου και εισάγοντας τη διαδρομή κοινής χρήσης. Και στις δύο περιπτώσεις, το κοινόχρηστο στοιχείο μπορεί να έχει τη μορφή \\server\share%user , όπου ο χρήστης είναι το όνομα χρήστη για σύνδεση στο διακομιστή.

  2. Το Samba μπορεί να ρυθμιστεί ώστε να προσπαθεί να ταιριάζει με τον κωδικό πρόσβασης που παρέχεται με μια λίστα χρηστών, προκειμένου να βρει το σωστό όνομα χρήστη. Η επιλογή κοινής χρήσης Unix users σάς επιτρέπει να καθορίσετε τους χρήστες ή τις ομάδες χρηστών για έλεγχο. Λάβετε υπόψη ότι η χρήση αυτής της επιλογής με μεγάλο αριθμό χρηστών δεν είναι πολύ ασφαλής, επειδή 2 άτομα μπορεί να έχουν τον ίδιο κωδικό πρόσβασης.

    Κρυπτογραφημένοι κωδικοί πρόσβασης

    Κανονικά, οι κωδικοί πρόσβασης που αποστέλλονται μέσω του δικτύου μεταξύ διακομιστή Samba και υπολογιστών-πελατών δεν είναι κρυπτογραφημένοι. Αυτό είναι κακό για 2 λόγους:
    1. Οποιοσδήποτε μπορεί να ακούσει στο δίκτυό σας και να καταγράψει τους κωδικούς πρόσβασης άλλων ατόμων.
    2. Οι τελευταίες εκδόσεις στα Windows 9x και NT αρνούνται την αποστολή κωδικών πρόσβασης χωρίς κρυπτογράφηση. Οι χρήστες που προσπαθούν να αποκτήσουν πρόσβαση στον διακομιστή Samba από αυτά τα συστήματα θα λάβουν ένα μήνυμα σφάλματος όπως ??? . Αυτή η αλλαγή εμφανίστηκε για πρώτη φορά στο NT service pack 3.

    Ευτυχώς, το Samba μπορεί να χρησιμοποιήσει κρυπτογραφημένους κωδικούς πρόσβασης. Ωστόσο, δεν είναι δυνατή η αντιστοίχιση κωδικών πρόσβασης που αποστέλλονται από πελάτες με το κανονικό αρχείο κωδικού πρόσβασης Unix λόγω των διαφορετικών μορφών κρυπτογράφησης που χρησιμοποιούνται. Για το λόγο αυτό, η Samba πρέπει να διατηρεί τη δική της λίστα χρηστών και κωδικών πρόσβασης.

    Για να αλλάξετε το Samba σε κρυπτογραφημένη λειτουργία κωδικού πρόσβασης, πρέπει:

    1. Βεβαιωθείτε ότι η έκδοση του Samba υποστηρίζει κρυπτογράφηση κωδικού πρόσβασης. Στη σελίδα Password Options , ορίστε την επιλογή Encrypted passwords σε ναι. Εάν αυτό αποτύχει, θα πρέπει να κατεβάσετε ή να συντάξετε μια έκδοση με κρυπτογραφημένη υποστήριξη κωδικού πρόσβασης.

    2. Κάντε κλικ στον σύνδεσμο Convert Unix users to Samba users στο κάτω μέρος της κύριας σελίδας. Αυτό θα σας οδηγήσει σε μια σελίδα στην οποία μπορείτε να μετατρέψετε μερικούς ή όλους τους υπάρχοντες χρήστες Unix σε χρήστες Samba. Τα ονόματα χρήστη, τα πραγματικά ονόματα, τα UID και τα κελύφη θα μετατραπούν - αλλά οι κωδικοί πρόσβασης δεν θα.

    3. Για να εκχωρήσετε κωδικούς πρόσβασης στους χρήστες, χρησιμοποιήστε την επιλογή Edit Samba user list στην κύρια σελίδα. Αυτό θα εμφανίσει μια λίστα με όλους τους χρήστες του Samba, από τους οποίους μπορείτε να επιλέξετε έναν χρήστη για επεξεργασία. Ο κωδικός πρόσβασης για κάθε χρήστη μπορεί να είναι Locked (δεν επιτρέπεται η είσοδος), No Password (δεν απαιτείται κωδικός πρόσβασης) ή κάποιος κωδικός πρόσβασης που εισάγετε.
