iSCSI-Client
Mit diesem Modul können Sie auf Festplattengeräte zugreifen, die von anderen Systemen in Ihrem Netzwerk mithilfe des iSCSI-Protokolls gemeinsam genutzt werden. Sobald ein Gerät verbunden wurde, kann es wie eine lokal angeschlossene Festplatte partitioniert, gemountet oder für RAID oder LVM verwendet werden.

Wenn für den iSCSI-Server, auf den Sie zugreifen möchten, eine Authentifizierung erforderlich ist, rufen Sie zunächst die Seite Authentifizierungsoptionen auf, um einen Benutzernamen und ein Kennwort festzulegen, die für nachfolgende Verbindungen verwendet werden sollen.

Um ein neues Gerät anzuschließen, rufen Sie die Seite iSCSI-Verbindungen auf und geben Sie den Hostnamen oder die IP-Adresse des iSCSI-Servers in das Formular unten auf der Seite ein. Webmin ruft verfügbare Zielgeräte vom angegebenen Server ab und ermöglicht Ihnen, eines oder alle für die Verbindung auszuwählen. Unter der Annahme, dass die Verbindung erfolgreich hergestellt wurde, wird ein lokales SCSI-Gerät wie /dev/sdb erstellt und in den Modulen Partitionen auf lokalen Datenträgern und Datenträgern und Netzwerkdateisystemen verfügbar gemacht.

Bestehende Verbindungen können entfernt werden, indem Sie das Kontrollkästchen neben ihren Einträgen auf der Seite iSCSI-Verbindungen aktivieren und auf die Schaltfläche Trennen klicken. Das Entfernen einer Verbindung, die derzeit in einem RAID- oder LVM-Volume bereitgestellt oder verwendet wird, wird nicht empfohlen.
