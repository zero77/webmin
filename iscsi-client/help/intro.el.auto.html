Πελάτης iSCSI
Αυτή η ενότητα σάς επιτρέπει να έχετε πρόσβαση σε συσκευές δίσκου που μοιράζονται χρησιμοποιώντας το πρωτόκολλο iSCSI από άλλα συστήματα στο δίκτυό σας. Μόλις συνδεθεί μια συσκευή, μπορεί να χωριστεί, να τοποθετηθεί ή να χρησιμοποιηθεί για RAID ή LVM όπως ένας τοπικά συνδεδεμένος δίσκος.

Εάν ο διακομιστής iSCSI στον οποίο σκοπεύετε να αποκτήσετε πρόσβαση απαιτεί έλεγχο ταυτότητας, επισκεφθείτε πρώτα τη σελίδα Επιλογές ελέγχου ταυτότητας για να ορίσετε ένα όνομα χρήστη και κωδικό πρόσβασης που θα χρησιμοποιηθούν για τις επόμενες συνδέσεις.

Για να επισυνάψετε μια νέα συσκευή, μεταβείτε στη σελίδα Συνδέσεις iSCSI και εισαγάγετε το όνομα κεντρικού υπολογιστή ή την IP του διακομιστή iSCSI στη φόρμα στο κάτω μέρος της σελίδας. Το Webmin θα πάρει τις διαθέσιμες συσκευές-στόχους από τον καθορισμένο διακομιστή και θα σας επιτρέψει να επιλέξετε μία ή όλες για σύνδεση. Υποθέτοντας ότι η σύνδεση ολοκληρώνεται με επιτυχία, μια τοπική συσκευή SCSI, όπως /dev/sdb, θα δημιουργηθεί και θα διατεθεί στις ενότητες Partitions on Local Disk και Disk και Network Filesystems .

Οι υπάρχουσες συνδέσεις μπορούν να αφαιρεθούν επιλέγοντας το πλαίσιο δίπλα στις καταχωρίσεις τους στη σελίδα Συνδέσεις iSCSI και κάνοντας κλικ στο κουμπί Αποσύνδεση . Δεν συνιστάται η κατάργηση μιας σύνδεσης που είναι εγκατεστημένη ή χρησιμοποιείται σε τόμο RAID ή LVM.
