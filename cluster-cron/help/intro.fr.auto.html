Emplois Cluster Cron
Ce module est presque identique au module Scheduled Cron Jobs , mais permet aux commandes planifiées d'être exécutées sur plusieurs serveurs à la fois. Cela est utile si votre réseau contient de nombreux systèmes et que vous souhaitez exécuter la même commande sur certains d'entre eux ou tous sur une planification régulière.

Son interface utilisateur est presque identique au module Cron standard, mais avec l'ajout d'un champ Exécuter sur les serveurs sur le formulaire d'édition de travail. Cela vous permet de sélectionner un ou plusieurs serveurs ou groupes qui ont été créés dans le module Webmin Servers Index avec un identifiant et un mot de passe.
