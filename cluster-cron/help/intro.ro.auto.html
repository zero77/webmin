Cluster Cron Jobs
Acest modul este aproape identic cu modulul Schedched Cron Jobs , dar permite rularea comenzilor programate pe mai multe servere simultan. Acest lucru este util dacă rețeaua dvs. conține multe sisteme și doriți să executați aceeași comandă pe unele sau pe toate acestea într-un program regulat.

Interfața sa de utilizator este aproape identică cu modulul Cron standard, dar cu adăugarea unui câmp Run pe servere pe formularul de editare a lucrărilor. Acest lucru vă permite să selectați unul sau mai multe servere sau grupuri care au fost create în modulul Index Web Servers cu o autentificare și o parolă.
