Proprietà LDAP per tutti i nuovi utenti
Questa opzione può essere utilizzata per specificare le proprietà che verranno assegnate agli utenti LDAP quando vengono create, oltre a quelle create da Webmin. Devono essere inseriti nel seguente formato:

nome1: valore1
nome2: valore2

È possibile utilizzare le seguenti sostituzioni, che vengono sostituite quando viene creato l'utente:

$ {USER} Il nome di accesso dell'utente
$ {PLAINPASS} La password non crittografata dell'utente, se disponibile
$ {REAL} Il vero nome dell'utente
$ {UID} Il numero UID Unix dell'utente
$ {GID} Il numero ID del gruppo Unix dell'utente
$ {HOME} La home directory dell'utente
$ {SHELL} La shell di accesso dell'utente
