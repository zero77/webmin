모든 신규 사용자를위한 LDAP 속성
이 옵션을 사용하면 Webmin에서 만든 속성 외에 LDAP 사용자가 생성 될 때 LDAP 사용자에게 할당 될 속성을 지정할 수 있습니다. 다음 형식으로 입력해야합니다.

이름 1 : 값 1
name2 : 값 2

다음 대체를 사용할 수 있으며 이는 사용자가 작성 될 때 대체됩니다.

$ {USER} 사용자의 로그인 이름
$ {PLAINPASS} 사용자의 암호화되지 않은 비밀번호 (제공된 경우)
$ {REAL} 사용자의 실명
$ {UID} 사용자의 Unix UID 번호
$ {GID} 사용자의 유닉스 그룹 ID 번호
$ {HOME} 사용자의 홈 디렉토리
$ {SHELL} 사용자의 로그인 쉘
