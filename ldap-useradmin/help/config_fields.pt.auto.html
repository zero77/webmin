Propriedades extras do usuário LDAP para permitir a edição de
Se o seu banco de dados LDAP usar atributos não suportados diretamente pelo Webmin, essa opção de configuração poderá ser usada para adicionar campos extras à página de edição do usuário para gerenciá-los. Pode ser útil para editar propriedades Qmail + LDAP, como mailForwardingAddress e mailMessageStore.

Os atributos para editar devem ser inseridos no formato:

nome1 descrição1
name2 description2

O nome é o nome LDAP do atributo e a descrição é o texto a ser exibido no Webmin. O nome pode ser opcionalmente seguido por um +, indicando que este atributo pode permitir vários valores.
