خواص LDAP برای همه کاربران جدید
از این گزینه می توان علاوه بر موارد ایجاد شده توسط Webmin ، برای مشخص کردن خصوصیاتی که هنگام ایجاد آنها به کاربران LDAP اختصاص می یابد ، استفاده کرد. آنها باید در قالب زیر وارد شوند:

name1: مقدار 1
name2: value2

از تعویض زیر می توان استفاده کرد که در هنگام ایجاد کاربر جایگزین می شوند:

$ {USER name نام ورود کاربر
$ {PLAINPASS password در صورت گذراندن رمزعبور رمزگذاری نشده رمزگذاری شده کاربر
$ {REAL name نام واقعی کاربر
$ {UID number شماره UID کاربر یونیکس
$ {GID number شماره شناسه گروه یونیکس کاربر
$ {HOME directory فهرست خانه کاربر
$ {SHELL shell پوسته ورود کاربر
