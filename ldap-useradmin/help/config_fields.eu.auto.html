LDAP erabiltzailearen propietate gehigarriak editatzeko baimena emateko
LDAP datu-baseak Webmin-ek zuzenean onartzen ez dituen atributuak erabiltzen badituzu, konfigurazio aukera hau erabiltzailea editatzeko orrialdean eremu osagarriak gehitzeko erabil daiteke. Qmail + LDAP propietateak editatzeko erabilgarria izan daiteke, hala nola mailForwardingAddress eta mailMessageStore.

Editatu beharreko atributuak formatuan sartu behar dira:

izena1 deskribapena1
izena2 deskribapena2

Izena atributuaren LDAP izena da eta deskribapena Webmin-en bistaratzeko testua da. Izena + aukeraz jarrai daiteke izena, eta atributu honek balio anitz baimendu ditzake.
