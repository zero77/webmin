Proprietăți de utilizator LDAP suplimentare pentru a permite editarea
Dacă baza de date LDAP utilizează atribute care nu sunt acceptate direct de Webmin, această opțiune de configurare poate fi utilizată pentru a adăuga câmpuri suplimentare la pagina de editare a utilizatorului pentru gestionarea acestora. Poate fi util pentru editarea proprietăților Qmail + LDAP, cum ar fi mailForwardingAddress și mailMessageStore.

Atributele de editat trebuie introduse în format:

nume1 descriere1
nume2 descriere2

Numele este numele LDAP al atributului, iar descrierea este textul de afișat în Webmin. Numele poate fi urmat opțional de un +, care indică faptul că acest atribut poate permite mai multe valori.
