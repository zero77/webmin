Дополнительные пользовательские свойства LDAP, позволяющие редактировать
Если ваша база данных LDAP использует атрибуты, которые не поддерживаются Webmin напрямую, этот параметр конфигурации можно использовать для добавления дополнительных полей на страницу редактирования пользователя для управления ими. Это может быть полезно для редактирования свойств Qmail + LDAP, таких как mailForwardingAddress и mailMessageStore.

Атрибуты для редактирования должны быть введены в формате:

имя1 описание1
name2 description2

Имя - это имя атрибута LDAP, а описание - это текст, отображаемый в Webmin. За именем может следовать знак «+», указывающий, что этот атрибут может содержать несколько значений.
