Papildomos LDAP vartotojo savybės, leidžiančios redaguoti
Jei jūsų LDAP duomenų bazėje naudojami atributai, kurių tiesiogiai nepalaiko „Webmin“, ši konfigūracijos parinktis gali būti naudojama norint pridėti papildomų laukų vartotojo redagavimo puslapyje jiems valdyti. Tai gali būti naudinga redaguojant „Qmail + LDAP“ ypatybes, tokias kaip mailForwardingAddress ir mailMessageStore.

Redaguoti atributus reikia įvesti tokiu formatu:

name1 aprašymas1
name2 aprašymas2

Pavadinimas yra LDAP atributo pavadinimas, o aprašymas - tekstas, kurį reikia rodyti „Webmin“. Prie pavadinimo pasirinktinai gali būti pridedamas +, nurodantis, kad šis atributas gali leisti kelias reikšmes.
