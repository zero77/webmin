Propiedades LDAP para todos los usuarios modificados.
Esta opción se puede utilizar para especificar propiedades que se actualizarán en usuarios LDAP cuando se modifiquen. Deben ingresarse en el siguiente formato:

nombre1: valor1
nombre2: valor2

Se pueden utilizar las siguientes sustituciones, que se reemplazan cuando se guarda el usuario:

$ {USER} El nombre de inicio de sesión del usuario
$ {PLAINPASS} La contraseña no cifrada del usuario, si se le dio una
$ {REAL} El nombre real del usuario
$ {UID} El número de UID de Unix del usuario
$ {GID} El número de identificación del grupo Unix del usuario
$ {HOME} El directorio de inicio del usuario
$ {SHELL} El shell de inicio de sesión del usuario
