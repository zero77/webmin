Властивості LDAP для всіх нових користувачів
Ця опція може бути використана для визначення властивостей, які будуть призначені користувачам LDAP під час їх створення, на додаток до створених Webmin. Вони повинні бути введені у такому форматі:

name1: value1
name2: value2

Можуть використовуватися такі підстановки, які замінюються під час створення користувача:

$ {USER} Ім'я користувача для входу
$ {PLAINPASS} Незашифрований пароль користувача, якщо такий був наданий
$ {REAL} Справжнє ім'я користувача
$ {UID} Юнікс UID номер користувача
$ {GID} Ідентифікаційний номер групи Unix групи
$ {HOME} Домашній каталог користувача
$ {SHELL} Оболонка для входу користувача
