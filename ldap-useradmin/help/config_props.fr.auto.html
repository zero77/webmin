Propriétés LDAP pour tous les nouveaux utilisateurs
Cette option peut être utilisée pour spécifier les propriétés qui seront attribuées aux utilisateurs LDAP lors de leur création, en plus de celles créées par Webmin. Ils doivent être saisis dans le format suivant:

nom1: valeur1
nom2: valeur2

Les substitutions suivantes peuvent être utilisées, qui sont remplacées lors de la création de l'utilisateur:

$ {USER} Le nom de connexion de l'utilisateur
$ {PLAINPASS} Le mot de passe non chiffré de l'utilisateur, s'il a été donné
$ {REAL} Le vrai nom de l'utilisateur
$ {UID} Le numéro UID Unix de l'utilisateur
$ {GID} Le numéro d'identification du groupe Unix de l'utilisateur
$ {HOME} Le répertoire personnel de l'utilisateur
$ {SHELL} Le shell de connexion de l'utilisateur
