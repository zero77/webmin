Propriétés utilisateur LDAP supplémentaires pour permettre la modification de
Si votre base de données LDAP utilise des attributs qui ne sont pas directement pris en charge par Webmin, cette option de configuration peut être utilisée pour ajouter des champs supplémentaires à la page de modification utilisateur pour les gérer. Il peut être utile pour modifier les propriétés Qmail + LDAP telles que mailForwardingAddress et mailMessageStore.

Les attributs à modifier doivent être saisis au format:

nom1 description1
nom2 description2

Le nom est le nom LDAP de l'attribut et la description est le texte à afficher dans Webmin. Le nom peut éventuellement être suivi d'un +, indiquant que cet attribut peut autoriser plusieurs valeurs.
