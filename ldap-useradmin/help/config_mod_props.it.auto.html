Proprietà LDAP per tutti gli utenti modificati
Questa opzione può essere utilizzata per specificare le proprietà che verranno aggiornate negli utenti LDAP quando vengono modificate. Devono essere inseriti nel seguente formato:

nome1: valore1
nome2: valore2

È possibile utilizzare le seguenti sostituzioni, che vengono sostituite quando l'utente viene salvato:

$ {USER} Il nome di accesso dell'utente
$ {PLAINPASS} La password non crittografata dell'utente, se disponibile
$ {REAL} Il vero nome dell'utente
$ {UID} Il numero UID Unix dell'utente
$ {GID} Il numero ID del gruppo Unix dell'utente
$ {HOME} La home directory dell'utente
$ {SHELL} La shell di accesso dell'utente
