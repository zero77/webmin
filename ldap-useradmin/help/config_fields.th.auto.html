คุณสมบัติผู้ใช้ LDAP เพิ่มเติมเพื่ออนุญาตให้แก้ไข
หากฐานข้อมูล LDAP ของคุณใช้คุณลักษณะที่ไม่ได้รับการสนับสนุนโดยตรงจาก Webmin ตัวเลือกการกำหนดค่านี้สามารถใช้เพื่อเพิ่มเขตข้อมูลพิเศษในหน้าการแก้ไขผู้ใช้สำหรับการจัดการ มันจะมีประโยชน์สำหรับการแก้ไขคุณสมบัติ Qmail + LDAP เช่น mailForwardingAddress และ mailMessageStore
