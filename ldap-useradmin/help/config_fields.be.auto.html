Дадатковыя ўласцівасці карыстальніка LDAP, каб дазволіць рэдагаванне
Калі ваша база дадзеных LDAP выкарыстоўвае атрыбуты, якія не падтрымліваюцца непасрэдна Webmin, гэтая опцыя канфігурацыі можа быць выкарыстана для дадання дадатковых палёў на старонку карыстальніка для рэдагавання для кіравання імі. Гэта можа быць карысна для рэдагавання уласцівасцей Qmail + LDAP, такіх як mailForwardingAddress і mailMessageStore.

Атрыбуты для рэдагавання павінны быць уведзены ў фармаце:

name1 апісанне1
name2 description2

Імя атрыбута - гэта LDAP, а апісанне - тэкст для адлюстравання ў Webmin. Неабавязкова імя можа ісці праз +, што азначае, што гэты атрыбут можа дазволіць некалькі значэнняў.
