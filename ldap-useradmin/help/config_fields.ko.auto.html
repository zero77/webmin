편집 할 수있는 추가 LDAP 사용자 속성
LDAP 데이터베이스가 Webmin에서 직접 지원하지 않는 속성을 사용하는 경우이 구성 옵션을 사용하여 사용자 편집 페이지에 필드를 추가하여 관리 할 수 있습니다. mailForwardingAddress 및 mailMessageStore와 같은 Qmail + LDAP 속성을 편집하는 데 유용 할 수 있습니다.

편집 할 속성은 다음 형식으로 입력해야합니다.

이름 1 설명 1
name2 설명 2

이름은 속성의 LDAP 이름이고 description은 Webmin에 표시 할 텍스트입니다. 선택적으로 이름 뒤에 +가 올 수 있으며이 속성은 여러 값을 허용 할 수 있음을 나타냅니다.
