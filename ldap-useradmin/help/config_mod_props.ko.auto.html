수정 된 모든 사용자의 LDAP 속성
이 옵션은 LDAP 사용자가 수정 될 때 업데이트 될 등록 정보를 지정하는 데 사용할 수 있습니다. 다음 형식으로 입력해야합니다.

이름 1 : 값 1
name2 : 값 2

사용자가 저장 될 때 대체되는 다음 대체를 사용할 수 있습니다.

$ {USER} 사용자의 로그인 이름
$ {PLAINPASS} 사용자의 암호화되지 않은 비밀번호 (제공된 경우)
$ {REAL} 사용자의 실명
$ {UID} 사용자의 Unix UID 번호
$ {GID} 사용자의 유닉스 그룹 ID 번호
$ {HOME} 사용자의 홈 디렉토리
$ {SHELL} 사용자의 로그인 쉘
