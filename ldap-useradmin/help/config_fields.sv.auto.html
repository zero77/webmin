Extra LDAP-användaregenskaper för att tillåta redigering av
Om din LDAP-databas använder attribut som inte stöds direkt av Webmin kan detta konfigurationsalternativ användas för att lägga till extra fält på användarredigeringssidan för att hantera dem. Det kan vara användbart för att redigera Qmail + LDAP-egenskaper som mailForwardingAddress och mailMessageStore.

Attribut som ska redigeras måste anges i formatet:

namn1 beskrivning1
namn2 beskrivning2

Namnet är attributet LDAP-namn och beskrivningen är texten som ska visas i Webmin. Namnet kan eventuellt följas av en +, vilket indikerar att detta attribut kan tillåta flera värden.
