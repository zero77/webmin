Properti LDAP untuk semua pengguna yang diubah
Pilihan ini dapat digunakan untuk menentukan sifat yang akan diperbarui pada pengguna LDAP ketika mereka diubah. Mereka mesti dimasukkan dalam format berikut:

nama1: nilai1
nama2: nilai2

Penggantian berikut dapat digunakan, yang diganti ketika pengguna disimpan:

$ {PENGGUNA} Nama log masuk pengguna
$ {PLAINPASS} Kata laluan pengguna yang tidak disulitkan, jika diberikan
$ {SEBENAR} Nama sebenar pengguna
$ {UID} Nombor UID Unix pengguna
$ {GID} Nombor ID kumpulan Unix pengguna
$ {RUMAH} Direktori rumah pengguna
$ {SHELL} Shell log masuk pengguna
