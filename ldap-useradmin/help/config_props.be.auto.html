Уласцівасці LDAP для ўсіх новых карыстальнікаў
Гэтая опцыя можа быць выкарыстана для вызначэння уласцівасцей, якія будуць прызначаны карыстальнікам LDAP пры іх стварэнні, акрамя тых, якія створаны Webmin. Яны павінны быць уведзены ў наступным фармаце:

name1: value1
name2: value2

Можна выкарыстоўваць наступныя падмены, якія замяняюцца пры стварэнні карыстальніка:

$ {USER} Імя карыстальніка
$ {PLAINPASS} Незашыфраваны пароль карыстальніка, калі такі быў дадзены
$ {REAL} сапраўднае імя карыстальніка
$ {UID} Нумар UID карыстальніка
$ {GID} Ідэнтыфікацыйны нумар групы Unix карыстальніка
$ {HOME} Хатні каталог карыстальніка
$ {SHELL} Абалонка для ўваходу карыстальніка
