Properti LDAP untuk semua pengguna baru
Pilihan ini dapat digunakan untuk menentukan sifat yang akan diberikan kepada pengguna LDAP ketika mereka dibuat, selain yang dibuat oleh Webmin. Mereka mesti dimasukkan dalam format berikut:

nama1: nilai1
nama2: nilai2

Penggantian berikut dapat digunakan, yang diganti ketika pengguna dibuat:

$ {PENGGUNA} Nama log masuk pengguna
$ {PLAINPASS} Kata laluan pengguna yang tidak disulitkan, jika diberikan
$ {SEBENAR} Nama sebenar pengguna
$ {UID} Nombor UID Unix pengguna
$ {GID} Nombor ID kumpulan Unix pengguna
$ {RUMAH} Direktori rumah pengguna
$ {SHELL} Shell log masuk pengguna
