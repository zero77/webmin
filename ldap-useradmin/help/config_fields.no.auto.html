Ekstra LDAP-brukeregenskaper for å tillate redigering av
Hvis LDAP-databasen din bruker attributter som ikke støttes direkte av Webmin, kan dette konfigurasjonsalternativet brukes til å legge til ekstra felt på brukerredigeringssiden for å administrere dem. Det kan være nyttig for redigering av Qmail + LDAP-egenskaper som mailForwardingAddress og mailMessageStore.

Attributter som skal redigeres må legges inn i formatet:

navn1 beskrivelse1
navn2 beskrivelse2

Navnet er LDAP-navnet på attributtet, og beskrivelse er teksten som skal vises i Webmin. Navnet kan eventuelt følges av en +, som indikerer at dette attributtet kan tillate flere verdier.
