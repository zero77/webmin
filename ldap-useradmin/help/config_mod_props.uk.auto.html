Властивості LDAP для всіх модифікованих користувачів
Ця опція може бути використана для визначення властивостей, які будуть оновлені у користувачів LDAP, коли вони будуть змінені. Вони повинні бути введені у такому форматі:

name1: value1
name2: value2

Можуть бути використані такі підстановки, які замінюються, коли користувач зберігається:

$ {USER} Ім'я користувача для входу
$ {PLAINPASS} Незашифрований пароль користувача, якщо такий був наданий
$ {REAL} Справжнє ім'я користувача
$ {UID} Юнікс UID номер користувача
$ {GID} Ідентифікаційний номер групи Unix групи
$ {HOME} Домашній каталог користувача
$ {SHELL} Оболонка для входу користувача
