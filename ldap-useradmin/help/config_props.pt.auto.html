Propriedades LDAP para todos os novos usuários
Esta opção pode ser usada para especificar propriedades que serão atribuídas aos usuários LDAP quando eles forem criados, além daquelas criadas pelo Webmin. Eles devem ser inseridos no seguinte formato:

name1: value1
name2: value2

As seguintes substituições podem ser usadas, que são substituídas quando o usuário é criado:

$ {USER} O nome de login do usuário
$ {PLAINPASS} A senha não criptografada do usuário, se houver uma
$ {REAL} O nome real do usuário
$ {UID} O número UID do usuário Unix
$ {GID} O número de identificação do grupo Unix do usuário
$ {HOME} O diretório inicial do usuário
$ {SHELL} O shell de login do usuário
