Επιπλέον ιδιότητες χρήστη LDAP που επιτρέπουν την επεξεργασία του
Εάν η βάση δεδομένων LDAP σας χρησιμοποιεί χαρακτηριστικά που δεν υποστηρίζονται απευθείας από το Webmin, αυτή η επιλογή διαμόρφωσης μπορεί να χρησιμοποιηθεί για την προσθήκη επιπλέον πεδίων στη σελίδα επεξεργασίας του χρήστη για τη διαχείρισή τους. Μπορεί να είναι χρήσιμο για την επεξεργασία ιδιοτήτων Qmail + LDAP, όπως mailForwardingAddress και mailMessageStore.

Τα χαρακτηριστικά προς επεξεργασία πρέπει να εισαχθούν με τη μορφή:

όνομα1 περιγραφή1
όνομα2 περιγραφή2

Το όνομα είναι το όνομα LDAP του χαρακτηριστικού και η περιγραφή είναι το κείμενο που θα εμφανίζεται στο Webmin. Το όνομα μπορεί προαιρετικά να ακολουθείται από ένα +, υποδεικνύοντας ότι αυτό το χαρακτηριστικό μπορεί να επιτρέψει πολλές τιμές.
