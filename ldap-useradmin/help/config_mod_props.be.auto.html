Уласцівасці LDAP для ўсіх змененых карыстальнікаў
Гэтая опцыя можа быць выкарыстана для вызначэння уласцівасцей, якія будуць абнаўляцца ў карыстальнікаў LDAP пры іх змене. Яны павінны быць уведзены ў наступным фармаце:

name1: value1
name2: value2

Наступныя падмены могуць быць заменены, калі карыстальнік захоўваецца:

$ {USER} Імя карыстальніка
$ {PLAINPASS} Незашыфраваны пароль карыстальніка, калі такі быў дадзены
$ {REAL} сапраўднае імя карыстальніка
$ {UID} Нумар UID карыстальніка
$ {GID} Ідэнтыфікацыйны нумар групы Unix карыстальніка
$ {HOME} Хатні каталог карыстальніка
$ {SHELL} Абалонка для ўваходу карыстальніка
