ویژگی های کاربر اضافی LDAP برای ویرایش
اگر پایگاه داده LDAP شما از ویژگیهایی استفاده می کند که مستقیماً توسط Webmin پشتیبانی نمی شوند ، می توانید از این گزینه پیکربندی برای اضافه کردن فیلدهای اضافی به صفحه ویرایش کاربر برای مدیریت آنها استفاده کنید. این می تواند برای ویرایش خصوصیات Qmail + LDAP از جمله پست الکترونیکی پست الکترونیکی و آدرس ایمیلMessageStore مفید باشد.

ویژگی های ویرایش باید در قالب وارد شوند:

name1 description1
name2 شرح2

نام این اسم LDAP از ویژگی است و توضیحات متنی است که در Webmin نمایش داده می شود. نام ممکن است به صورت اختیاری توسط a + دنبال شود و نشان می دهد که این ویژگی می تواند مقادیر متعددی را مجاز سازد.
