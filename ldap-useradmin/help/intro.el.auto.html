Χρήστες και ομάδες LDAP
Εάν το σύστημά σας έχει ρυθμιστεί να χρησιμοποιεί διακομιστή LDAP για αποθήκευση χρηστών και ομάδων Unix, αυτή η λειτουργική μονάδα μπορεί να χρησιμοποιηθεί για τη διαχείριση χρηστών σε μια βάση δεδομένων LDAP. Το LDAP χρησιμοποιείται συχνά ως εναλλακτική λύση στα NIS για συγχρονισμό πληροφοριών χρήστη σε πολλά συστήματα σε ένα δίκτυο, καθώς μια βάση δεδομένων μπορεί να αποθηκεύσει πρόσθετα χαρακτηριστικά για κάθε χρήστη εκτός από το τυπικό όνομα χρήστη Unix, κέλυφος και ούτω καθεξής.

Φυσικά, ένας διακομιστής LDAP πρέπει να εκτελείται στο σύστημά σας ή σε κάποιο κεντρικό υπολογιστή στο δίκτυό σας για να χρησιμοποιηθεί αυτή η λειτουργική μονάδα. Επιπλέον, ένας ή περισσότεροι κεντρικοί υπολογιστές πρέπει να ρυθμιστούν ώστε να χρησιμοποιούν LDAP για έλεγχο ταυτότητας χρήστη και / ή ομάδας. Κανένα από αυτά τα καθήκοντα δεν μπορεί να γίνει μέσα από τη λειτουργική μονάδα - πρέπει να γίνει χειροκίνητα ή με άλλες λειτουργικές μονάδες Webmin για το σκοπό αυτό.

Εάν έχετε ρυθμίσει το Samba να χρησιμοποιεί και διακομιστή LDAP, αυτή η λειτουργική μονάδα μπορεί επίσης να δημιουργήσει και να διαχειριστεί τα απαραίτητα χαρακτηριστικά κωδικού πρόσβασης για κάθε χρήστη LDAP ώστε να μπορεί να συνδεθεί και στον διακομιστή Samba. Για να λειτουργήσει αυτή η δυνατότητα, ο διακομιστής LDAP πρέπει να ρυθμιστεί ώστε να υποστηρίζει τα πρόσθετα χαρακτηριστικά Samba και την κατηγορία αντικειμένων για κάθε χρήστη στο σχήμα του. Μόνο όταν η σύνδεση Samba; Η επιλογή έχει οριστεί σε Ναι, θα δημιουργηθούν τα χαρακτηριστικά Samba.

Η μονάδα μπορεί επίσης να δημιουργήσει χρήστες σε διακομιστή Cyrus IMAP, εάν έχει ρυθμιστεί για έλεγχο ταυτότητας με την ίδια βάση δεδομένων LDAP. Θα δημιουργήσει γραμματοκιβώτια για κάθε νέο χρήστη, θα εγγραφεί ο χρήστης στα γραμματοκιβώτια του και θα παραχωρήσει πρόσβαση διαχειριστή σε αυτούς. Εάν ένας χρήστης LDAP διαγραφεί, το γραμματοκιβώτιο IMAP θα είναι επίσης. Προς το παρόν, αυτή η δυνατότητα λειτουργεί μόνο με τον διακομιστή Cyrus IMAP - δεν έχει δοκιμαστεί και μάλλον δεν είναι απαραίτητο για άλλους διακομιστές, οι οποίοι χρησιμοποιούν τους οικιακούς καταλόγους των χρηστών για αποθήκευση αλληλογραφίας.
