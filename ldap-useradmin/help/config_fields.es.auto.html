Propiedades de usuario LDAP adicionales para permitir la edición de
Si su base de datos LDAP usa atributos que no son compatibles directamente con Webmin, esta opción de configuración se puede usar para agregar campos adicionales a la página de edición del usuario para administrarlos. Puede ser útil para editar propiedades de Qmail + LDAP como mailForwardingAddress y mailMessageStore.

Los atributos para editar deben ingresarse en el formato:

nombre1 descripción1
nombre2 descripción2

El nombre es el nombre LDAP del atributo y la descripción es el texto que se mostrará en Webmin. El nombre puede ir seguido opcionalmente por un +, que indica que este atributo puede permitir múltiples valores.
