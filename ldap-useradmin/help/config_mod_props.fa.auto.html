خواص LDAP برای همه کاربران اصلاح شده
از این گزینه می توان برای مشخص کردن خصوصیات استفاده شده در هنگام تغییر در کاربران LDAP استفاده کرد. آنها باید در قالب زیر وارد شوند:

name1: مقدار 1
name2: value2

از تعویض زیر می توان استفاده کرد که با ذخیره کاربر جایگزین می شوند:

$ {USER name نام ورود کاربر
$ {PLAINPASS password در صورت گذراندن رمزعبور رمزگذاری نشده رمزگذاری شده کاربر
$ {REAL name نام واقعی کاربر
$ {UID number شماره UID کاربر یونیکس
$ {GID number شماره شناسه گروه یونیکس کاربر
$ {HOME directory فهرست خانه کاربر
$ {SHELL shell پوسته ورود کاربر
