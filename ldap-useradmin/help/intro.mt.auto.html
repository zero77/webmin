Utenti u Gruppi LDAP
Jekk is-sistema tiegħek hija kkonfigurata biex tuża server LDAP għall-ħażna ta 'utenti u gruppi Unix, dan il-modulu jista' jintuża biex jimmaniġġja l-utenti f'database LDAP. LDAP ħafna drabi jintuża bħala alternattiva għall-NIS għas-sinkronizzazzjoni tal-informazzjoni tal-utent bejn sistemi multipli f'netwerk, peress li database tista 'taħżen attributi addizzjonali għal kull utent minbarra l-istandard tal-username Unix, il-qoxra u l-bqija.

Naturalment, server LDAP irid ikun qed jaħdem fuq is-sistema tiegħek jew fuq xi host fuq in-netwerk tiegħek biex jintuża dan il-modulu. Barra minn hekk, wieħed jew ħafna mill-ospiti għandhom ikunu kkonfigurati biex jużaw LDAP għall-utent u / jew l-awtentikazzjoni tal-grupp. L-ebda minn dawn il-kompiti ma jista 'jsir minn ġewwa l-modulu - għandhom isiru manwalment, jew b'xi moduli Webmin oħra għall-iskop.

Jekk Samba kkonfigurat biex juża server LDAP ukoll, dan il-modulu jista 'wkoll joħloq u jimmaniġġja l-attributi tal-password meħtieġa għal kull utent LDAP biex ikun jista' jidħol fis-server Samba ukoll. Sabiex din il-karatteristika taħdem, is-server LDAP tiegħek għandu jiġi kkonfigurat biex jappoġġja l-attributi tas-Samba addizzjonali u l-klassi tal-oġġett għal kull utent fl-iskema tiegħu. Meta biss iddaħħal is- Samba? l-għażla hija ssettjata għal Iva se jinħolqu l-attributi tas-Samba.

Il-modulu jista 'wkoll joħloq utenti fuq server Cyrus IMAP, jekk huwa mwaqqaf biex jawtentika fl-istess database LDAP. Se toħloq kaxxi tal-posta għal kull utent ġdid, tissottoskrivi l-utent għall-kaxxi tal-posta tiegħu u tagħti aċċess għall-amministratur għalihom. Jekk utent LDAP jitħassar, il-kaxxa postali tiegħu IMAP tkun ukoll. Bħalissa din il-karatteristika taħdem biss mas-server IMAP Cyrus - ma ġiex ittestjat u probabbilment lanqas huwa meħtieġ għal servers oħra, li jużaw id-direttorji tad-dar tal-utenti għall-ħażna tal-posta.
