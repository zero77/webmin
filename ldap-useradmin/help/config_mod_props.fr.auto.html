Propriétés LDAP pour tous les utilisateurs modifiés
Cette option peut être utilisée pour spécifier les propriétés qui seront mises à jour chez les utilisateurs LDAP lors de leur modification. Ils doivent être saisis dans le format suivant:

nom1: valeur1
nom2: valeur2

Les substitutions suivantes peuvent être utilisées, qui sont remplacées lorsque l'utilisateur est enregistré:

$ {USER} Le nom de connexion de l'utilisateur
$ {PLAINPASS} Le mot de passe non chiffré de l'utilisateur, s'il a été donné
$ {REAL} Le vrai nom de l'utilisateur
$ {UID} Le numéro UID Unix de l'utilisateur
$ {GID} Le numéro d'identification du groupe Unix de l'utilisateur
$ {HOME} Le répertoire personnel de l'utilisateur
$ {SHELL} Le shell de connexion de l'utilisateur
