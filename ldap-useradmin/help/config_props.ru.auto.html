Свойства LDAP для всех новых пользователей
Этот параметр можно использовать для указания свойств, которые будут назначены пользователям LDAP при их создании, в дополнение к свойствам, созданным Webmin. Они должны быть введены в следующем формате:

имя1: значение1
имя2: значение2

Можно использовать следующие замены, которые заменяются при создании пользователя:

$ {USER} Логин пользователя
$ {PLAINPASS} незашифрованный пароль пользователя, если он был задан
$ {REAL} Настоящее имя пользователя
$ {UID} номер UID пользователя Unix
$ {GID} идентификационный номер группы Unix пользователя
$ {HOME} Домашний каталог пользователя
$ {SHELL} Оболочка для входа пользователя
