Sifat pengguna LDAP tambahan untuk membolehkan penyuntingan
Sekiranya pangkalan data LDAP anda menggunakan atribut yang tidak didukung langsung oleh Webmin, pilihan konfigurasi ini dapat digunakan untuk menambahkan bidang tambahan ke halaman penyuntingan pengguna untuk mengurusnya. Ia berguna untuk mengedit sifat Qmail + LDAP seperti mailForwardingAddress dan mailMessageStore.

Atribut untuk diedit mesti dimasukkan dalam format:

nama1 keterangan1
nama2 keterangan2

Nama itu adalah nama LDAP atribut, dan keterangan adalah teks yang akan dipaparkan di Webmin. Nama itu mungkin diikuti dengan tanda +, yang menunjukkan bahawa atribut ini dapat membenarkan beberapa nilai.
