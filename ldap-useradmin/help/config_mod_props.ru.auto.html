Свойства LDAP для всех измененных пользователей
Эту опцию можно использовать для указания свойств, которые будут обновляться у пользователей LDAP при их изменении. Они должны быть введены в следующем формате:

имя1: значение1
имя2: значение2

Можно использовать следующие замены, которые заменяются при сохранении пользователя:

$ {USER} Логин пользователя
$ {PLAINPASS} незашифрованный пароль пользователя, если он был задан
$ {REAL} Настоящее имя пользователя
$ {UID} номер UID пользователя Unix
$ {GID} идентификационный номер группы Unix пользователя
$ {HOME} Домашний каталог пользователя
$ {SHELL} Оболочка для входа пользователя
