Propriedades LDAP para todos os usuários modificados
Esta opção pode ser usada para especificar propriedades que serão atualizadas nos usuários LDAP quando eles forem modificados. Eles devem ser inseridos no seguinte formato:

name1: value1
name2: value2

Podem ser usadas as seguintes substituições, que são substituídas quando o usuário é salvo:

$ {USER} O nome de login do usuário
$ {PLAINPASS} A senha não criptografada do usuário, se houver uma
$ {REAL} O nome real do usuário
$ {UID} O número UID do usuário Unix
$ {GID} O número de identificação do grupo Unix do usuário
$ {HOME} O diretório inicial do usuário
$ {SHELL} O shell de login do usuário
