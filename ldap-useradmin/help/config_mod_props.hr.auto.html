LDAP svojstva za sve modificirane korisnike
Ova se opcija može koristiti za određivanje svojstava koja će se ažurirati kod korisnika LDAP-a kada se izmijene. Moraju se unijeti u sljedećem formatu:

ime1: vrijednost1
ime2: vrijednost2

Mogu se koristiti sljedeće zamjene koje se zamjenjuju kada se korisnik spremi:

$ {USER} Korisničko ime za prijavu
$ {PLAINPASS} Korisnikova šifrirana lozinka, ako je data
$ {REAL} Pravo ime korisnika
$ {UID} Unix UID broj korisnika
$ {GID} ID broja korisnika Unix grupe
$ {HOME} Kućni imenik korisnika
$ {SHELL} Korisnička ljuska za prijavu
