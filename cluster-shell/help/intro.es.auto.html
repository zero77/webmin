Comandos de clúster
Este módulo es similar al Command Shell , pero permite que se ejecuten comandos simples en varios servidores simultáneamente en lugar de solo en este servidor. En la página principal, puede ingresar un comando para ejecutar (o seleccionar uno antiguo) y elegir uno o más servidores o grupos para ejecutarlo. Solo aparecerán en la lista los servidores que se hayan creado en el módulo Índice de servidores Webmin con un nombre de usuario y contraseña.
