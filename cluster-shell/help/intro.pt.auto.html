Comandos do shell de cluster
Este módulo é semelhante ao Command Shell , mas permite que comandos simples sejam executados em vários servidores simultaneamente, em vez de apenas neste servidor. Na página principal, você pode inserir um comando para executar (ou selecionar um antigo) e escolher um ou mais servidores ou grupos para executá-lo. Somente servidores que foram criados no módulo Índice de servidores Webmin com login e senha aparecerão na lista.
