Εντολές συμπλέγματος συμπλέγματος
Αυτή η ενότητα είναι παρόμοια με το Command Shell , αλλά επιτρέπει την εκτέλεση απλών εντολών σε πολλούς διακομιστές ταυτόχρονα αντί μόνο σε αυτόν τον διακομιστή. Στην κύρια σελίδα μπορείτε να εισαγάγετε μια εντολή για εκτέλεση (ή να επιλέξετε μια παλιά) και να επιλέξετε έναν ή περισσότερους διακομιστές ή ομάδες για να την εκτελέσετε. Στη λίστα θα εμφανίζονται μόνο διακομιστές που έχουν δημιουργηθεί στη μονάδα ευρετηρίου διακομιστών Webmin με σύνδεση και κωδικό πρόσβασης.
