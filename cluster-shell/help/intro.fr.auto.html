Commandes de shell de cluster
Ce module est similaire à Command Shell , mais permet d'exécuter des commandes simples simultanément sur plusieurs serveurs au lieu de simplement sur ce serveur. Sur la page principale, vous pouvez entrer une commande à exécuter (ou en sélectionner une ancienne) et choisir un ou plusieurs serveurs ou groupes sur lesquels l'exécuter. Seuls les serveurs créés dans le module Webmin Servers Index avec un identifiant et un mot de passe apparaîtront dans la liste.
