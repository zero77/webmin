Cluster Shell -komennot
Tämä moduuli on samanlainen kuin Command Shell , mutta sallii yksinkertaisten komentojen suorittamisen useilla palvelimilla samanaikaisesti vain tämän palvelimen sijasta. Pääsivulle voit kirjoittaa suoritettavan komennon (tai valita vanhan) ja valita yhden tai useamman palvelimen tai ryhmän suorittamaan sen. Vain palvelimet, jotka on luotu Webmin Servers Index -moduulissa sisäänkirjautumisella ja salasanalla, näkyvät luettelossa.
