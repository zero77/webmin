Comandi Shell Cluster
Questo modulo è simile a Command Shell , ma consente l'esecuzione semplice di comandi su più server contemporaneamente anziché solo su questo server. Nella pagina principale è possibile immettere un comando da eseguire (o selezionarne uno precedente) e scegliere uno o più server o gruppi su cui eseguirlo. Nell'elenco verranno visualizzati solo i server che sono stati creati nel modulo Indice server Webmin con un login e una password.
