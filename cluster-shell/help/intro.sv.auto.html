Cluster Shell-kommandon
Denna modul liknar Command Shell , men tillåter enkla kommandon att köras på flera servrar samtidigt istället för bara på den här servern. På startsidan kan du ange ett kommando som ska köras (eller välja ett gammalt) och välja en eller flera servrar eller grupper för att köra det på. Endast servrar som har skapats i Webmin Servers Index- modulen med inloggning och lösenord visas i listan.
