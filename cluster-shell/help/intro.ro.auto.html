Comenzi Cluster Shell
Acest modul este similar cu Command Shell , dar permite rularea comenzilor simple simultan pe mai multe servere în loc de doar pe acest server. Pe pagina principală puteți introduce o comandă pentru a rula (sau selecta una veche) și alege unul sau mai multe servere sau grupuri pentru a o executa. Doar serverele care au fost create în modulul Index Web Servers cu o autentificare și parolă vor apărea în listă.
