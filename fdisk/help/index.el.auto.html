Hdparm
Βοηθήστε να διαμορφώσετε διάφορα ioctls σκληρού δίσκου που υποστηρίζονται από το πρόγραμμα οδήγησης συσκευής Linux IDE / ST-506.

Απαιτείται έκδοση πυρήνα Linux 1.2.13 ή μεταγενέστερη. Ορισμένες επιλογές ενδέχεται να μην λειτουργούν σωστά με παλαιότερους πυρήνες. Επιπλέον, υποστηρίζονται πολλές επιλογές μόνο για πυρήνες που περιλαμβάνουν το νέο πρόγραμμα οδήγησης συσκευής (E) IDE, όπως η έκδοση 2.0.10 ή μεταγενέστερη.

Η στήλη Drive εμφανίζει όλη τη μονάδα IDE. Κάντε κλικ για να τις διαμορφώσετε και / ή επεξεργαστείτε την καταχώριση στο αρχείο σεναρίου.

Κατάσταση ισχύος εμφανίζει την κατάσταση της μονάδας δίσκου, εάν είναι σε κατάσταση αναμονής χαμηλής ισχύος, η μονάδα έχει διακοπεί, κατάσταση αναστολής λειτουργίας χαμηλότερης ισχύος, η μονάδα είναι εντελώς κλειστή ή η ενεργή / αδράνεια κανονική λειτουργία.
Εάν η κατάσταση της μονάδας είναι ένα είδος λειτουργίας εξοικονόμησης ενέργειας, κάντε κλικ στην κατάσταση για επανεκκίνηση της περιστροφής της μονάδας.

Έχει γίνει δοκιμή δύο ταχυτήτων, εκτελούν χρονισμούς ανάγνωσης κρυφής μνήμης και ανάγνωσης μονάδας δίσκου για σκοπούς αναφοράς και σύγκρισης. Εάν η μονάδα βρίσκεται σε λειτουργία ασφαλούς λειτουργίας, η δοκιμή αυτή αγνοείται.
