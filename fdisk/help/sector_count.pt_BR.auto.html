Contagem Setorial
Defina a contagem do setor para leitura antecipada do sistema de arquivos.

Isso é usado para melhorar o desempenho em leituras seqüenciais de arquivos grandes, buscando previamente blocos adicionais, antecipando que eles sejam necessários para a tarefa em execução. Na versão atual do kernel (2.0.10), isso tem uma configuração padrão de 8 setores (4 kB). Esse valor parece bom para a maioria dos propósitos, mas em um sistema em que a maioria dos acessos a arquivos é feita aleatoriamente, uma configuração menor pode fornecer um melhor desempenho. Além disso, muitas unidades IDE também possuem uma função de leitura antecipada interna separada, o que alivia a necessidade de uma leitura antecipada do sistema de arquivos em muitas situações.
