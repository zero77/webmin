Modo de transferência
Defina o modo de transferência IDE para unidades (E) IDE / ATA2 mais recentes.

Isso geralmente é usado em combinação com o uso do DMA ao ativar o DMA para / de uma unidade em um chipset de interface suportado.

Além disso, o uso desse sinalizador é .Eu raramente necessário, pois a maioria / todas as unidades IDE modernas têm como padrão o modo de transferência PIO mais rápido na inicialização. Brincar com isso pode ser desnecessário e arriscado.

Use isso com extrema cautela!
