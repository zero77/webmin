hdparm
Ajută la configurarea mai multor dispozitive de hard disk suportate de driverul de dispozitiv Linux IDE / ST-506.

Necesită kernel Linux versiunea 1.2.13 sau o versiune ulterioară. Este posibil ca unele opțiuni să nu funcționeze corect cu sâmburele anterioare. În plus, mai multe opțiuni sunt acceptate numai pentru kerneluri care includ noul driver de dispozitiv ID (E), cum ar fi versiunea 2.0.10 sau o versiune ulterioară.

Coloana Drive arată toate unitățile IDE. Faceți clic pentru a le configura și / sau a edita intrarea în fișierul script.

Starea de alimentare afișează starea unității, dacă este în modul standby de putere redusă, unitatea a răsturnat, modul de repaus cel mai mic modul de putere, unitatea este complet închis sau funcționarea normală activ / inactiv.
Dacă starea unității este un fel de mod de economisire a energiei, faceți clic pe starea pentru a reporni rotirea unității.

Se efectuează două teste de viteză, se efectuează cronologii de citire a cache-urilor și de lectură de disc pentru scopuri de referință și comparație. Dacă unitatea se află într-un mod sigur de energie, testul este ignorat.
