Τρόπος μεταφοράς
Ρυθμίστε τη λειτουργία μεταφοράς IDE για νεότερες (E) μονάδες IDE / ATA2.

Αυτό χρησιμοποιείται συνήθως σε συνδυασμό με τη χρήση DMA κατά την ενεργοποίηση του DMA από / προς μια μονάδα δίσκου σε ένα υποστηριζόμενο chipset διεπαφής.

Εκτός από αυτό, είναι απαραίτητη η χρήση αυτής της σημαίας. Σπάνια είναι απαραίτητο, καθώς οι περισσότερες / όλες οι σύγχρονες μονάδες IDE είναι προεπιλεγμένες στην ταχύτερη λειτουργία μεταφοράς PIO κατά την ενεργοποίηση. Το να παίζεις αυτό μπορεί να είναι περιττό και επικίνδυνο.

Χρησιμοποιήστε αυτό με εξαιρετική προσοχή!
