32-Bit-E / A-Unterstützung
Stellen Sie die (E) IDE 32-Bit-E / A-Unterstützung ein.

Dieser Wert funktioniert mit fast allen 32-Bit-IDE-Chipsätzen, verursacht jedoch etwas mehr Overhead. Beachten Sie, dass sich "32-Bit" nur auf Datenübertragungen über einen PCI- oder VLB-Bus zur Schnittstellenkarte bezieht. Alle (E) IDE-Laufwerke haben immer noch nur eine 16-Bit-Verbindung über das Flachbandkabel von der Schnittstellenkarte.
