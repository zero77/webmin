ช่วยในการกำหนดค่าฮาร์ดดิสก์ต่างๆ ioctls ที่ได้รับการสนับสนุนโดยไดรเวอร์อุปกรณ์ Linux IDE / ST-506 หุ้น

ต้องการ Linux kernel รุ่น 1.2.13 หรือใหม่กว่า ตัวเลือกบางอย่างอาจทำงานไม่ถูกต้องกับเมล็ดก่อนหน้า นอกจากนี้ยังมีตัวเลือกต่าง ๆ ที่รองรับเฉพาะในเคอร์เนลซึ่งรวมถึงไดรเวอร์อุปกรณ์ใหม่ (E) IDE เช่นรุ่น 2.0.10 หรือใหม่กว่า

คอลัมน์ ไดรฟ์ แสดงไดรฟ์ IDE ทั้งหมด คลิกเพื่อกำหนดค่าและ / หรือแก้ไขรายการในไฟล์สคริปต์

สถานะพลังงาน แสดงสถานะของไดรฟ์หากเป็น โหมด พลังงานต่ำ สแตนด์บาย ไดรฟ์ได้ปั่นลง โหมดสลีปโหมดพลังงานต่ำสุดไดรฟ์ถูกปิดลงอย่างสมบูรณ์หรือการ ใช้งาน ปกติ / ไม่ได้ใช้ งาน
หากสถานะของไดรฟ์เป็นโหมดประหยัดพลังงานหนึ่งประเภทให้คลิกที่สถานะเพื่อเริ่มการหมุนของไดรฟ์ใหม่

ทำการทดสอบความเร็วสองครั้งโดยจะทำการกำหนดเวลาการอ่านแคชและอ่านไดรฟ์เพื่อการวัดประสิทธิภาพและเปรียบเทียบ หากไดรฟ์อยู่ในโหมดประหยัดพลังงานการทดสอบเหล่านั้นจะถูกละเว้น
