Übertragungsmodus
Stellen Sie den IDE-Übertragungsmodus für neuere (E) IDE / ATA2-Laufwerke ein.

Dies wird normalerweise in Kombination mit der Verwendung von DMA verwendet, wenn DMA zu / von einem Laufwerk auf einem unterstützten Schnittstellen-Chipsatz aktiviert wird.

Abgesehen davon ist die Verwendung dieses Flags selten erforderlich, da die meisten / alle modernen IDE-Laufwerke beim Einschalten standardmäßig ihren schnellsten PIO-Übertragungsmodus verwenden. Damit herumzuspielen kann sowohl unnötig als auch riskant sein.

Verwenden Sie dies mit äußerster Vorsicht!
