Ζώνη ώρας PHP

Προεπιλεγμένη ζώνη ώρας που χρησιμοποιείται από όλες τις συναρτήσεις ημερομηνίας / ώρας. Η ρύθμιση σε κενή τιμή χρησιμοποιεί την προεπιλογή για την έκδοση PHP που διαθέτετε.

Ανατρέξτε στην τεκμηρίωση του εγχειριδίου PHP για την επιλογή διαμόρφωσης για περισσότερες λεπτομέρειες.

(Διαθέσιμο από PHP 5.1.0)
