Часовой пояс PHP

Часовой пояс по умолчанию, используемый всеми функциями даты и времени. Установка этого значения в пустое значение использует значение по умолчанию для вашей версии PHP.

Пожалуйста, обратитесь к документации по PHP для опции конфигурации для более подробной информации.

(Доступно с PHP 5.1.0)
