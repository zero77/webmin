Παρακολούθηση εύρους ζώνης
Αυτή η ενότητα μπορεί να χρησιμοποιηθεί για τη δημιουργία απλών αναφορών σχετικά με τη χρήση εύρους ζώνης ανά θύρα, κεντρικό υπολογιστή, πρωτόκολλο και χρόνο για την κυκλοφορία που αποστέλλεται από ή δρομολογείται μέσω του συστήματός σας. Είναι χρήσιμο τόσο για τους αυτόνομους κεντρικούς υπολογιστές όσο και για εκείνους που λειτουργούν ως πύλη (πιθανώς με NAT) για ένα δίκτυο.

Προτού μπορέσει να χρησιμοποιηθεί, η μονάδα πρέπει να ρυθμίσει πολλούς κανόνες τείχους προστασίας και μια καταχώριση syslog για την καταγραφή της κίνησης που αποστέλλεται και λαμβάνεται μέσω του συστήματός σας. Κατά τον χρόνο εγκατάστασης πρέπει να επιλέξετε τη διεπαφή δικτύου του εξωτερικού συστήματος, στην οποία θα παρακολουθείται η κίνηση. Αυτό συνήθως θα είναι η διεπαφή PPP που χρησιμοποιείται για τη σύνδεση μέσω τηλεφώνου ή ADSL, ή η διασύνδεση Ethernet που είναι συνδεδεμένη στο καλωδιακό μόντεμ ή στον ανάντη δρομολογητή.

Μόλις καταγραφεί τουλάχιστον μία ώρα κυκλοφορίας, η λειτουργική μονάδα μπορεί να χρησιμοποιηθεί για τη δημιουργία αναφορών που συνοψίζουν την κίνηση από μία από τις ακόλουθες κατηγορίες:

  • Ωρα
    Η ημερομηνία και ώρα λήψης των δεδομένων.
  • Ημέρα
    Η ημερομηνία παραλαβής των δεδομένων.
  • Πλήθος
    Ο κεντρικός υπολογιστής στο εσωτερικό σας δίκτυο ή ο κεντρικός υπολογιστής τείχους προστασίας, που έστειλε ή έλαβε τα δεδομένα.
  • Εσωτερική θύρα
    Η θύρα του εσωτερικού ή του τείχους προστασίας στον οποίο εστάλησαν τα δεδομένα. Αυτή η λειτουργία είναι χρήσιμη για τον καθορισμό της επισκεψιμότητας που δημιουργείται από συνδέσεις σε καθέναν από τους διακομιστές σας.
  • Εξωτερική θύρα
    Η θύρα σε κάποιο διακομιστή εκτός του δικτύου σας στον οποίο εστάλησαν τα δεδομένα. Αυτό μπορεί να χρησιμοποιηθεί για τον προσδιορισμό των υπηρεσιών στις οποίες έχουν πρόσβαση οι χρήστες του δικτύου σας.
  • Λιμάνι
    Τόσο εσωτερικές όσο και εξωτερικές θύρες. Χρήσιμο για τη σύγκριση όλης της χρήσης δικτύου ανά τύπο υπηρεσίας.
Μπορείτε επίσης να επιλέξετε να περιορίσετε την αναφορά σε επιλεγμένο κεντρικό υπολογιστή, θύρα ή πρωτόκολλο. Ο κεντρικός υπολογιστής μπορεί να εισαχθεί με διεύθυνση IP, όνομα κεντρικού υπολογιστή ή διεύθυνση δικτύου όπως 192.168.1.0. Η θύρα μπορεί να εισαχθεί με όνομα ή αριθμό και θα ταιριάζει με το TCP και το UDP.

Η αναφορά μπορεί επίσης να περιορίζεται στην επισκεψιμότητα που συλλέγεται μεταξύ επιλεγμένων ωρών, χρησιμοποιώντας τα πεδία Για κυκλοφορία μετά και Για κυκλοφορία πριν . Επειδή η επισκεψιμότητα συνοψίζεται ανά ώρα, δεν μπορείτε να περιορίσετε την αναφορά με μεγαλύτερη ακρίβεια από την ωριαία.

Η επιλογή Θύρες διακομιστή μόνο; είναι χρήσιμο κατά την αναφορά μέσω εισερχόμενων, εξερχόμενων ή όλων των θυρών. Περιορίζει την οθόνη σε θύρες που χρησιμοποιούνται συνήθως από διακομιστές (αυτές κάτω από 1024 ή με ονόματα), για να αποφευχθεί η ακαταστασία της οθόνης με μετρήσεις για θύρες πελάτη που δεν είναι συνήθως χρήσιμες.

Η επιλογή Επίλυση ονομάτων κεντρικών υπολογιστών; μπορεί να επιλεγεί κατά την αναφορά από τον κεντρικό υπολογιστή. Θα προκαλέσει την επαναφορά όλων των διευθύνσεων IP σε ονόματα κεντρικών υπολογιστών, όπου είναι δυνατόν.
