Denne hjelpesiden er ufullstendig

Dependency / Dependent Tree Viewer

Informasjonen som vises på denne siden tilsvarer informasjon om avhengighet / avhengig av tre, samlet av svcs -d / -D, dvs. hvilke tjenester er avhengige av denne tjenesten, og hvilke tjenester er den avhengig av? Tilstandsendringer kan brukes på avhengige / avhengigheter, og ved å klikke på en avhengighet / avhengighet fmri kommer du til statusredigeringssiden.

Avhengighets / avhengige trær kan utvides / kollapses til hvilket som helst ønsket nivå ved å klikke på henholdsvis "+" og "-". Tilstandsendringer kan brukes på tjenesten og valgte avhengigheter / avhengige ved å bruke avmerkingsbokser for å velge tjenestene.

Hvis du klikker på en FMRI i avhengighet / avhengig tre, vil du bringe deg til avhengighet / avhengig tre.
