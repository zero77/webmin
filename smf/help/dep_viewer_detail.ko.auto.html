종속성 / 의존 트리 뷰어

이 페이지에 표시되는 정보는 svcs -d / -D에 의해 수집 된 종속성 / 종속 트리 정보에 해당합니다. 즉,이 서비스에 어떤 서비스가 의존하고 어떤 서비스에 의존합니까? 상태 변경은 종속 / 종속성에 적용 할 수 있으며 종속 / 종속성 fmri를 클릭하면 상태 편집기 페이지로 이동합니다.

"+"및 "-"버튼을 각각 클릭하여 종속성 / 종속 트리를 원하는 수준으로 확장 / 축소 할 수 있습니다. 상태 변경은 서비스에 적용 할 수 있으며 확인란을 사용하여 서비스를 선택하여 선택한 종속성 / 의존성을 선택할 수 있습니다.

종속성 / 의존 트리에서 FMRI를 클릭하면 해당 인스턴스의 종속성 / 의존 트리로 이동합니다.

자세한 내용은

SMF를 이해하는 데 도움이되는 다음 매뉴얼 페이지를 참조하십시오.
  • smf (5)
  • service_bundle (4)
  • svccfg (1M)
  • svcadm (1M)
  • svc.startd (1M)
  • svc.configd (1M)
  • svcprop (1M)
  • svcs (1M)
