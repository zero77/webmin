SMF-modul

Den här hjälpsidan är ofullständig

Introduktion

Den här modulen låter dig visa / ändra / uppdatera / lägga till SMF-tjänster i Solaris 10. Huvudsidan består av en lista med tjänster, och du kan välja en specifik kategori, visa funktionshindrade serier, sortera poster etc. Härifrån kan du också starta ett verktyg av guiden som hanterar tjänstskapande, visa servicetillstånd, beroenden / beroende och visa egenskapsgrupper / egenskaper för varje tjänst.

Huvudfönster

Som standard visar huvudfönstret i SMF-modulen de aktiverade tjänsterna i systemet och visar deras tillstånd, FMRI som identifierar tjänsten och beskrivningen som medföljer tjänsten. Du kan tillämpa tillståndsändringar på flera tjänster genom att kontrollera de önskade seriverna och klicka på aktivera / inaktivera etc. Du kan också ta bort de angivna tjänsteinstanserna, men uppenbarligen undvika att göra detta för viktiga tjänster.

Instansvisningsfönster

Genom att klicka på FMRI för en viss instans i huvudfönstret visas instansvisaren, som visar instansnamn / beskrivning, tillstånd, tillhörande manuella sidor (som kan klickas och visas), förklaringar till ovanliga tillstånd (som kan klickas och visade), referenser till tillhörande loggfiler (som igen kan klickas och visas). Härifrån är det också möjligt att se beroendet / beroende träd för instansen genom att klicka på lämplig länk och visa / redigera egendomsgrupper / egenskaper genom att också klicka på lämplig länk.

Fastighetsgrupp / fastighetsredigerare Windows

Om du klickar på länken Visa / redigera egendomsgrupper / egenskaper i inställningsvisaren öppnas redigeraren för fastighetsgrupper och visar egenskapsgrupperna som är associerade med den valda FMRI. Du kan lägga till / ta bort / ändra egendomsgrupper här och göra samma sak för de tillhörande egenskaperna genom att klicka på den relevanta egendomsgruppen i egendomsgruppredigeraren.

Fönster för beroende / beroende

Om du klickar på länken Visa beroende / beroende träd i instansvisaren får du upp beroende / beroende träd för den instansen. Som standard visar vyn beroende / beroende på första nivån, men genom att klicka på + -knappen kan du utöka trädet för att se vad som beror på tjänsterna som är beroende av tjänsten etc. På samma sätt som i huvudfönstret kan tillståndsändringar tillämpas till flera tjänster med hjälp av kryssrutor för att identifiera de tjänster du vill ändra. Om du klickar på en instans FMRI kommer det upp beroende / beroende träd för den instansen.

Guiden för skapande av tjänster

Om du klickar på knappen "Skapa ny serivce" öppnas guiden för skapande av SMF-tjänster, som samlar in den information som krävs för att skapa en ny SMF-tjänst, formaterar den till ett manifest och importerar den till SMF-förvaret. Framgångsrik import kommer att resultera i att tjänsten visas i listan över huvudsakliga tjänster.

Serviceinstanssökningsformulär

En rudimentär FMRI-sökning tillhandahålls, varvid tillträde till en del av tjänsten FMRI är tillräckligt för att hitta alla matchande FMRI, t.ex. "nfs" hämtar alla tjänster med "nfs" i FMRI. En "bläddringsanläggning" tillhandahålls också.
