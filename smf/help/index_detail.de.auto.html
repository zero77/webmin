Serviceliste

Die auf dieser Seite angezeigte Liste ist das Ergebnis der Ausführung des Befehls svcs unter Verwendung eines Platzhalterabgleichs, um die erforderliche Dienstkategorie anzuzeigen, z. B. svcs network / *. Verschiedene Kategorien (einschließlich aller Dienste) können ausgewählt werden, indem Sie die Kategorie aus dem Dropdown-Menü auswählen und auf "Ansicht aktualisieren" klicken. Standardmäßig werden nur aktivierte Dienste angezeigt. Dies kann jedoch geändert werden, indem Sie das Kontrollkästchen "Deaktivierte Dienste einschließen" aktivieren und erneut auf "Ansicht aktualisieren" klicken. Der Ansicht können auch zusätzliche Felder hinzugefügt werden (NextState und StartTime).

Statusänderungen / -löschungen können auf die im Hauptfenster aufgelisteten Dienste angewendet werden. Die Sortierung kann nach jeder der Spalten durchgeführt werden. Standardmäßig werden Dienste so sortiert, dass die "ungewöhnlichen" Zustände oben in der Liste angezeigt werden.

Durch Klicken auf das FMRI-Feld für einen Dienst können wir alle zugehörigen Details untersuchen, von der Basis (auf der Seite der Hauptinstanzanzeige) wie Name / Beschreibung bis zu allen Eigenschaften.

Durch Klicken auf die Schaltfläche "Neue Dienstleistung erstellen" wird der Assistent zum Erstellen von SMF-Diensten aufgerufen, der die zum Erstellen eines neuen SMF-Dienstes erforderlichen Informationen sammelt, in ein Manifest formatiert und in das SMF-Repository importiert. Ein erfolgreicher Import führt dazu, dass der Dienst in der Liste der Hauptdienste angezeigt wird.

Durch Klicken auf die Schaltfläche "Nach Diensten suchen" wird ein Suchformular angezeigt. Das Eingeben eines Teils des Dienst-FMRI reicht aus, um alle übereinstimmenden FMRIs zu finden, z. B. "nfs" ruft alle nfs-Dienste ab. Eine "Durchsuchen" -Funktion wird ebenfalls bereitgestellt.
