ตัวแสดงการพึ่งพาโมดูล SMF / ขึ้นอยู่กับ

หน้าช่วยเหลือนี้ไม่สมบูรณ์

Tree Dependency / Dependent

ข้อมูลที่แสดงในหน้านี้สอดคล้องกับข้อมูลการพึ่งพา / พึ่งพาต้นไม้ที่รวบรวมโดย svcs -d / -D คือบริการใดที่ขึ้นอยู่กับบริการนี้และบริการใดขึ้นอยู่กับอะไร การเปลี่ยนแปลงสถานะสามารถนำไปใช้กับผู้ติดตาม / ผู้ติดตามและการคลิกผู้ติดตาม / ผู้ติดตาม FMR จะพาคุณไปยังหน้าเครื่องมือแก้ไขสถานะ

แผนผังการพึ่งพา / ขึ้นอยู่กับต้นไม้สามารถขยาย / ยุบในระดับที่ต้องการได้โดยคลิกที่ปุ่ม "+" และ "-" ตามลำดับ การเปลี่ยนแปลงสถานะสามารถนำไปใช้กับบริการและการพึ่งพา / ผู้ติดตามที่เลือกโดยใช้ช่องทำเครื่องหมายเพื่อเลือกบริการ

การคลิกที่ FMRI ในแผนผังการพึ่งพา / ขึ้นอยู่กับจะนำคุณไปสู่แผนภูมิการพึ่งพา / การพึ่งพาสำหรับตัวอย่างนั้น
