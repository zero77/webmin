SMF-moduuli

Tämä ohjesivu on puutteellinen

esittely

Tämän moduulin avulla voit tarkastella / muokata / päivittää / lisätä SMF-palveluita Solaris 10: ssä. Pääsivu koostuu palveluiden luettelosta. Voit valita tietyn luokan, tarkastella estettyjä serivejä, lajitella merkintöjä jne. Täältä voit myös käynnistää ohjatun tyyppinen apuohjelma, joka hallinnoi palvelun luomista, tarkastele palvelun tilaa, riippuvuuksia / riippuvaisia ja tarkastele ominaisuuksien ryhmiä / ominaisuuksia jokaiselle palvelulle.

Pääikkuna

Oletusarvoisesti SMF-moduulin pääikkuna luetteloi järjestelmän käytössä olevat palvelut, näyttäen niiden tilat, palvelun tunnistavan FMRI: n ja palvelun mukana toimitetun kuvauksen. Voit soveltaa tilamuutoksia useisiin palveluihin tarkistamalla halutut serivit ja napsauttamalla Ota käyttöön / poista käytöstä jne. Voit myös poistaa määritetyt palvelunäkymät, mutta luonnollisesti välttää tämän tekemistä tärkeille palveluille.

Esimerkki-näyttöikkuna

Napsauttamalla pääikkunassa tietyn ilmentymän FMRI-painiketta, esiin tulee ilmentymän katseluohjelma, joka näyttää ilmentymän nimen / kuvauksen, tilan, kaikki siihen liittyvät manuaaliset sivut (joita voidaan napsauttaa ja katsella), selitykset epätavallisille tiloille (joita voidaan napsauttaa ja katsottu), viittaukset liittyviin lokitiedostoihin (joita taas voi napsauttaa ja tarkastella). Täältä on myös mahdollista tarkastella esiintymän riippuvuus- / riippuvuuspuita napsauttamalla asianmukaista linkkiä ja katsella / muokata ominaisuusryhmiä / ominaisuuksia napsauttamalla myös asianmukaista linkkiä.

Ominaisuusryhmä / Ominaisuuseditori Windows

Napsauttamalla Näytä / muokkaa ominaisuusryhmiä / Ominaisuuksia -linkkiä ilmentymän katseluohjelmassa, näkyviin tulee ominaisuusryhmäeditori, joka näyttää valittuun FMRI: hen liittyvät ominaisuusryhmät. Voit lisätä / poistaa / muokata ominaisuusryhmiä täällä ja tehdä samoin liittyviin ominaisuuksiin napsauttamalla vastaavaa omaisuusryhmää omaisuusryhmäeditorissa.

Riippuvuus / riippuvainen katsoja -ikkuna

Napsauttamalla Näytä riippuvuus / Riippuva puu -linkkiä ilmentymän katseluohjelmassa, tuo esiin kyseisen esiintymän riippuvuus- / riippuvuuspuu. Oletusarvon mukaan näkymässä on ensimmäisen tason riippuvuudet / riippuvaiset, mutta napsauttamalla + -painiketta voit laajentaa puuta nähdäksesi mikä riippuu palvelusta riippuvista palveluista jne. Samoin kuin pääikkunassa, tilamuutoksia voidaan soveltaa useisiin palveluihin tunnistamalla muutettavat palvelut valintaruuduilla. Napsauttamalla esiintymää FMRI tuo esiin kyseisen esiintymän riippuvuus- / riippuvuuspuut.

Ohjattu palvelun luominen

Napsauttamalla "Luo uusi palvelu" -painiketta, esiin tulee ohjattu SMF-palvelun luontitoiminto, joka kerää tarvittavat tiedot uuden SMF-palvelun luomiseksi, muotoilee sen manifestiksi ja tuo se SMF-arkistoon. Onnistunut tuonti johtaa siihen, että palvelu näkyy pääpalveluiden luettelossa.

Palvelun instanssin hakulomake

Toimitetaan alkeellinen FMRI-haku, jolloin palvelun osaan syöttäminen FMRI riittää löytämään kaikki vastaavat FMRI: t, esim. "Nfs" hakee kaikki palvelut, joissa "nfs" on FMRI: ssä. Mukana on myös "selata" -toiminto.
