Ενότητα SMF

Αυτή η σελίδα βοήθειας είναι ελλιπής

Εισαγωγή

Αυτή η ενότητα σάς επιτρέπει να προβάλετε / τροποποιήσετε / ενημερώσετε / προσθέσετε υπηρεσίες SMF στο Solaris 10. Η κύρια σελίδα αποτελείται από μια λίστα υπηρεσιών και μπορείτε να επιλέξετε μια συγκεκριμένη κατηγορία, να δείτε απενεργοποιημένες υπηρεσίες, ταξινόμηση καταχωρήσεων κ.λπ. Από εδώ μπορείτε επίσης να ξεκινήσετε ένα βοηθητικό πρόγραμμα τύπου οδηγού που διαχειρίζεται τη δημιουργία υπηρεσιών, προβολή της κατάστασης υπηρεσίας, εξαρτήσεις / εξαρτώμενα και προβολή ομάδων ιδιοτήτων / ιδιοτήτων για κάθε υπηρεσία.

Κύριο παράθυρο

Από προεπιλογή, το κύριο παράθυρο της μονάδας SMF παραθέτει τις ενεργοποιημένες υπηρεσίες στο σύστημα, δείχνοντας τις καταστάσεις τους, το FMRI που προσδιορίζει την υπηρεσία και την περιγραφή που παρέχεται με την υπηρεσία. Μπορείτε να εφαρμόσετε αλλαγές κατάστασης σε πολλές υπηρεσίες ελέγχοντας τις επιθυμητές υπηρεσίες και κάνοντας κλικ στο Ενεργοποίηση / απενεργοποίηση κ.λπ. Μπορείτε επίσης να διαγράψετε τις παρουσίες υπηρεσίας που καθορίστηκαν, αλλά προφανώς αποφύγετε να το κάνετε αυτό για σημαντικές υπηρεσίες.

Παράθυρο προβολής Instance

Κάνοντας κλικ στο FMRI μιας συγκεκριμένης παρουσίας στο κύριο παράθυρο εμφανίζεται το πρόγραμμα προβολής παρουσίας, το οποίο εμφανίζει το όνομα / περιγραφή της παρουσίας, την κατάσταση, τυχόν σχετικές μη αυτόματες σελίδες (στις οποίες μπορεί να γίνει κλικ και προβολή), εξηγήσεις για ασυνήθιστες καταστάσεις (στις οποίες μπορεί να γίνει κλικ και προβολή), αναφορές σε συσχετισμένα αρχεία καταγραφής (στα οποία μπορείτε να κάνετε κλικ και να δείτε ξανά). Από εδώ μπορείτε επίσης να δείτε την εξάρτηση / εξαρτώμενα δέντρα για την παρουσία κάνοντας κλικ στον κατάλληλο σύνδεσμο και να δείτε / επεξεργαστείτε ομάδες ιδιοτήτων / ιδιοτήτων κάνοντας επίσης κλικ στον κατάλληλο σύνδεσμο.

Ομάδα ιδιοκτησίας / Windows Editor Editor

Κάνοντας κλικ στο σύνδεσμο Προβολή / Επεξεργασία ομάδων ιδιοτήτων / ιδιοτήτων στο πρόγραμμα προβολής εμφανίσεων εμφανίζεται το πρόγραμμα επεξεργασίας ομάδας ιδιοτήτων, εμφανίζοντας τις ομάδες ιδιοτήτων που σχετίζονται με το επιλεγμένο FMRI. Μπορείτε να προσθέσετε / διαγράψετε / τροποποιήσετε ομάδες ιδιοτήτων εδώ και να κάνετε το ίδιο με τις σχετικές ιδιότητες κάνοντας κλικ στη σχετική ομάδα ιδιοτήτων στον επεξεργαστή ομάδας ιδιοτήτων.

Εξάρτηση / Εξαρτώμενο παράθυρο προβολής

Κάνοντας κλικ στον σύνδεσμο Προβολή Εξάρτησης / Εξαρτώμενου Δέντρου στο πρόγραμμα προβολής παρουσιών εμφανίζεται η εξάρτηση / εξαρτώμενο δέντρο για αυτήν την παρουσία. Από προεπιλογή, η προβολή εμφανίζει εξαρτήσεις / εξαρτώμενα άτομα πρώτου επιπέδου, αλλά κάνοντας κλικ στο κουμπί +, μπορείτε να επεκτείνετε το δέντρο για να δείτε τι εξαρτάται από τις υπηρεσίες που εξαρτώνται από την υπηρεσία κ.λπ. Όπως και στο κύριο παράθυρο, μπορούν να εφαρμοστούν αλλαγές κατάστασης σε πολλές υπηρεσίες, χρησιμοποιώντας πλαίσια ελέγχου για τον προσδιορισμό των υπηρεσιών που θέλετε να αλλάξετε. Κάνοντας κλικ σε μια παρουσία FMRI εμφανίζεται η εξάρτηση / εξαρτώμενα δέντρα για αυτήν την παρουσία.

Οδηγός δημιουργίας υπηρεσιών

Κάνοντας κλικ στο κουμπί "Δημιουργία νέου Serivce" εμφανίζεται ο οδηγός δημιουργίας υπηρεσιών SMF, ο οποίος συλλέγει τις απαιτούμενες πληροφορίες για τη δημιουργία μιας νέας υπηρεσίας SMF, τη μορφοποιεί σε μια δήλωση και τις εισάγει στο αποθετήριο SMF. Η επιτυχής εισαγωγή θα έχει ως αποτέλεσμα η υπηρεσία να εμφανίζεται στην κύρια λίστα υπηρεσιών.

Φόρμα αναζήτησης παρουσίας υπηρεσίας

Παρέχεται μια στοιχειώδης αναζήτηση FMRI, με την οποία εισάγετε ένα μέρος της υπηρεσίας Το FMRI αρκεί για να βρείτε όλα τα αντίστοιχα FMRI, π.χ. το "nfs" ανακτά όλες τις υπηρεσίες με το "nfs" στο FMRI. Παρέχεται επίσης δυνατότητα «περιήγησης».

Για περισσότερες πληροφορίες

Συμβουλευτείτε τις ακόλουθες μη αυτόματες σελίδες που θα σας βοηθήσουν να γνωρίσετε το SMF:
  • smf (5)
  • service_bundle (4)
  • svccfg (1 εκατ.)
  • svcadm (1M)
  • svc.startd (1M)
  • svc.configd (1M)
  • svcprop (1M)
  • svcs (1M)
