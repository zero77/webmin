Module SMF

Cette page d'aide est incomplète

introduction

Ce module vous permet de visualiser / modifier / mettre à jour / ajouter des services SMF dans Solaris 10. La page principale se compose d'une liste de services, et vous pouvez sélectionner une catégorie spécifique, afficher les services désactivés, trier les entrées, etc. Vous pouvez également lancer ici un utilitaire de type assistant qui gère la création de service, affiche l'état du service, les dépendances / dépendantes et affiche les groupes de propriétés / propriétés pour chaque service.

Fenêtre principale

Par défaut, la fenêtre principale du module SMF répertorie les services activés sur le système, affichant leurs états, l'IRMF qui identifie le service et la description fournie avec le service. Vous pouvez appliquer des changements d'état à plusieurs services en vérifiant les services souhaités, en cliquant sur activer / désactiver, etc. Vous pouvez également supprimer les instances de service spécifiées, mais évitez évidemment de le faire pour les services importants.

Fenêtre de visualisation d'instance

En cliquant sur la FMRI d'une instance particulière dans la fenêtre principale, le visualiseur d'instance affiche le nom / la description de l'instance, l'état, toutes les pages de manuel associées (sur lesquelles vous pouvez cliquer et afficher), des explications sur les états inhabituels (sur lesquels cliquer et consulté), des références aux fichiers journaux associés (qui peuvent à nouveau être cliqués et consultés). À partir d'ici, il est également possible d'afficher la dépendance / les arbres dépendants pour l'instance en cliquant sur le lien approprié, et d'afficher / modifier les groupes de propriétés / propriétés en cliquant également sur le lien approprié.

Fenêtre Groupe de propriétés / Éditeur de propriétés

En cliquant sur le lien Afficher / Modifier les groupes de propriétés / Propriétés dans la visionneuse d'instances, l'éditeur de groupe de propriétés s'affiche, affichant les groupes de propriétés associés à l'IRMF sélectionnée. Vous pouvez ajouter / supprimer / modifier des groupes de propriétés ici, et faire de même pour les propriétés associées en cliquant sur le groupe de propriétés approprié dans l'éditeur de groupe de propriétés.

Fenêtre Dépendance / Visionneuse dépendante

En cliquant sur le lien Afficher la dépendance / l'arborescence dépendante dans la visionneuse d'instances, l'arborescence des dépendances / dépendances s'affiche pour cette instance. Par défaut, la vue affiche les dépendances / dépendances de premier niveau, mais en cliquant sur le bouton +, vous pouvez développer l'arborescence pour voir ce qui dépend des services qui dépendent du service, etc. De la même manière que pour la fenêtre principale, des changements d'état peuvent être appliqués à plusieurs services, en utilisant des cases à cocher pour identifier les services que vous souhaitez modifier. Cliquer sur une instance FMRI fait apparaître la dépendance / les arbres dépendants pour cette instance.

Assistant de création de service

Cliquez sur le bouton "Créer un nouveau service" pour ouvrir l'assistant de création de service SMF, qui collecte les informations requises pour créer un nouveau service SMF, le formate dans un manifeste et l'importe dans le référentiel SMF. Une importation réussie fera apparaître le service dans la liste des services principaux.

Formulaire de recherche d'instance de service

Une recherche rudimentaire d'IRMF est fournie, par laquelle la saisie d'une partie du service IRMF est suffisante pour trouver tous les IRMF correspondants, par exemple "nfs" récupère tous les services avec "nfs" dans l'IRMF. Une fonction "parcourir" est également fournie.
