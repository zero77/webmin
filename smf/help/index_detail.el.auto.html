Λίστα υπηρεσιών

Η λίστα που εμφανίζεται σε αυτήν τη σελίδα είναι το αποτέλεσμα της εκτέλεσης της εντολής svcs χρησιμοποιώντας αντιστοίχιση μπαλαντέρ για την εμφάνιση της απαιτούμενης κατηγορίας υπηρεσίας, π.χ. δίκτυο svcs / *. Μπορείτε να επιλέξετε διαφορετικές κατηγορίες (συμπεριλαμβανομένων όλων των υπηρεσιών) επιλέγοντας την κατηγορία από το αναπτυσσόμενο μενού και κάνοντας κλικ στην επιλογή "Ενημέρωση προβολής". Από προεπιλογή, εμφανίζονται μόνο οι ενεργοποιημένες υπηρεσίες, αλλά αυτό μπορεί να αλλάξει επιλέγοντας το πλαίσιο "Συμπερίληψη απενεργοποιημένων υπηρεσιών" και κάνοντας ξανά κλικ στην επιλογή "Ενημέρωση προβολής". Επιπλέον πεδία μπορούν να προστεθούν και στην προβολή (NextState και StartTime).

Οι αλλαγές / διαγραφές κατάστασης μπορούν να εφαρμοστούν στις υπηρεσίες που ελέγχονται στην κύρια λίστα παραθύρων. Η ταξινόμηση μπορεί να πραγματοποιηθεί από οποιαδήποτε από τις στήλες και, από προεπιλογή, οι υπηρεσίες ταξινομούνται έτσι ώστε οι "ασυνήθιστες" καταστάσεις να εμφανίζονται στην κορυφή της λίστας.

Κάνοντας κλικ στο πεδίο FMRI για μια υπηρεσία, μπορούμε να εξετάσουμε όλες τις σχετικές λεπτομέρειες, από τη βασική (στη σελίδα προβολής της κύριας εμφάνισης) όπως το όνομα / περιγραφή, έως όλες τις ιδιότητες.

Κάνοντας κλικ στο κουμπί "Δημιουργία νέου Serivce" εμφανίζεται ο οδηγός δημιουργίας υπηρεσιών SMF, ο οποίος συλλέγει τις απαιτούμενες πληροφορίες για τη δημιουργία μιας νέας υπηρεσίας SMF, τη μορφοποιεί σε μια δήλωση και τις εισάγει στο αποθετήριο SMF. Η επιτυχής εισαγωγή θα έχει ως αποτέλεσμα η υπηρεσία να εμφανίζεται στην κύρια λίστα υπηρεσιών.

Κάνοντας κλικ στο κουμπί "Αναζήτηση για υπηρεσίες" εμφανίζεται μια φόρμα αναζήτησης. Η εισαγωγή ενός μέρους της υπηρεσίας FMRI αρκεί για να βρείτε όλα τα αντίστοιχα FMRI, π.χ. το "nfs" ανακτά όλες τις υπηρεσίες nfs. Παρέχεται επίσης δυνατότητα «περιήγησης».

Για περισσότερες πληροφορίες

Συμβουλευτείτε τις ακόλουθες μη αυτόματες σελίδες που θα σας βοηθήσουν να γνωρίσετε το SMF:
  • smf (5)
  • service_bundle (4)
  • svccfg (1 εκατ.)
  • svcadm (1M)
  • svc.startd (1M)
  • svc.configd (1M)
  • svcprop (1M)
  • svcs (1M)
