โมดูล SMF

หน้าช่วยเหลือนี้ไม่สมบูรณ์

บทนำ

โมดูลนี้ช่วยให้คุณดู / แก้ไข / อัปเดต / เพิ่มบริการ SMF ใน Solaris 10 หน้าหลักประกอบด้วยรายชื่อของบริการและคุณสามารถเลือกหมวดหมู่ที่เฉพาะเจาะจงดู serivces ปิดการใช้งานรายการเรียงลำดับจากที่นี่คุณยังสามารถเปิด ยูทิลิตี้ประเภทตัวช่วยสร้างที่จัดการการสร้างบริการดูสถานะบริการการพึ่งพา / ขึ้นอยู่กับและดูกลุ่มคุณสมบัติ / คุณสมบัติสำหรับแต่ละบริการ

หน้าต่างหลัก

ตามค่าดีฟอลต์หน้าต่างโมดูล SMF หลักจะแสดงรายการบริการที่เปิดใช้งานบนระบบแสดงสถานะของพวกเขา FMRI ที่ระบุบริการและคำอธิบายที่ให้มาพร้อมกับบริการ คุณสามารถใช้การเปลี่ยนแปลงสถานะกับบริการหลายรายการได้โดยตรวจสอบ serivces ที่ต้องการและคลิกเปิด / ปิดการใช้งาน ฯลฯ นอกจากนี้คุณยังสามารถลบอินสแตนซ์ของบริการที่ระบุ แต่เห็นได้ชัดว่าหลีกเลี่ยงการทำเช่นนี้สำหรับบริการสำคัญ

หน้าต่างตัวแสดงอินสแตนซ์

การคลิกที่ FMRI ของอินสแตนซ์เฉพาะในหน้าต่างหลักจะแสดงอินสแตนซ์ของตัวแสดงซึ่งแสดงชื่อ / คำอธิบายอินสแตนซ์รัฐหน้าคู่มือใด ๆ ที่เกี่ยวข้อง (ซึ่งสามารถคลิกและดู) คำอธิบายสำหรับสถานะผิดปกติ (ซึ่งสามารถคลิกและ ดูแล้ว) อ้างอิงถึงไฟล์บันทึกที่เกี่ยวข้อง (ซึ่งสามารถคลิกและดูอีกครั้ง) จากที่นี่คุณสามารถดูแผนผังการพึ่งพา / ขึ้นอยู่กับอินสแตนซ์ได้โดยคลิกที่ลิงค์ที่เหมาะสมและดู / แก้ไขกลุ่ม / คุณสมบัติของคุณสมบัติโดยคลิกที่ลิงค์ที่เหมาะสม

กลุ่มคุณสมบัติ / ตัวแก้ไขคุณสมบัติ Windows

คลิกที่ลิงค์ดู / แก้ไขกลุ่มคุณสมบัติ / คุณสมบัติในมุมมองอินสแตนซ์จะปรากฏตัวแก้ไขกลุ่มคุณสมบัติขึ้นมาแสดงกลุ่มคุณสมบัติที่เกี่ยวข้องกับ FMRI ที่เลือก คุณสามารถเพิ่ม / ลบ / แก้ไขกลุ่มคุณสมบัติที่นี่และทำเช่นเดียวกันกับคุณสมบัติที่เกี่ยวข้องโดยคลิกกลุ่มคุณสมบัติที่เกี่ยวข้องในตัวแก้ไขกลุ่มคุณสมบัติ

หน้าต่างตัวแสดงการอ้างอิง / ขึ้นอยู่กับ

การคลิกที่ลิงก์ดูการพึ่งพา / การพึ่งพาต้นไม้ในมุมมองอินสแตนซ์จะแสดงแผนผังการพึ่งพา / ขึ้นอยู่กับอินสแตนซ์นั้น ตามค่าเริ่มต้นมุมมองจะแสดงการพึ่งพา / ผู้อยู่ในระดับแรก แต่ด้วยการคลิกปุ่ม + คุณสามารถขยายแผนภูมิเพื่อดูสิ่งที่ขึ้นอยู่กับบริการที่ขึ้นอยู่กับบริการอื่น ๆ เช่นเดียวกับหน้าต่างหลักสามารถเปลี่ยนสถานะได้ ไปยังบริการต่างๆโดยใช้ช่องทำเครื่องหมายเพื่อระบุบริการที่คุณต้องการเปลี่ยน การคลิกที่อินสแตนซ์ FMRI จะแสดงแผนผังการพึ่งพา / พึ่งพาสำหรับอินสแตนซ์นั้น

วิซาร์ดการสร้างบริการ

การคลิกปุ่ม "สร้างใหม่ Serivce" จะแสดงตัวช่วยสร้างการสร้างบริการ SMF ซึ่งรวบรวมข้อมูลที่จำเป็นในการสร้างบริการ SMF ใหม่จัดรูปแบบลงในรายการและนำเข้าสู่ที่เก็บ SMF การนำเข้าที่ประสบความสำเร็จจะส่งผลให้บริการปรากฏในรายการบริการหลัก

แบบฟอร์มการค้นหาอินสแตนซ์ของบริการ

การค้นหา FMRI พื้นฐานนั้นจะให้บริการโดยการป้อนส่วนหนึ่งของบริการ FMRI ก็เพียงพอที่จะค้นหา FMRIs ที่ตรงกันทั้งหมดเช่น "nfs" ดึงข้อมูลบริการทั้งหมดด้วย "nfs" ใน FMRI สิ่งอำนวยความสะดวก "เรียกดู" ถูกจัดเตรียมไว้ด้วย
