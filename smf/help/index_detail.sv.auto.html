SMF Module Service Listing

Den här hjälpsidan är ofullständig

Tjänsteförteckning

Listningen som visas på denna sida är resultatet av att köra kommandot svcs med jokerteckenmatchning för att visa önskad servicekategori, t.ex. svcs nätverk / *. Olika kategorier (inklusive alla tjänster) kan väljas genom att välja kategori från rullgardinsmenyn och klicka på "Uppdatera vy". Som standard visas endast aktiverade tjänster, men det kan ändras genom att kryssa i rutan "Inkludera inaktiverade tjänster" och igen klicka på "Uppdatera vy". Extra fält kan också läggas till i vyn (NextState och StartTime).

Tillståndsändringar / raderingar kan tillämpas på de tjänster som kontrolleras i listan med huvudfönster. Sortering kan utföras med vilken som helst av kolumnerna, och som standard sorteras tjänster så att de "ovanliga" tillstånden visas högst upp i listan.

Genom att klicka på FMRI-fältet för en tjänst kan vi granska alla tillhörande detaljer, från den grundläggande (på huvudvisningsvisarens sida), t.ex. namn / beskrivning, till alla egenskaper.

Om du klickar på knappen "Skapa ny serivce" öppnas guiden för skapande av SMF-tjänster, som samlar in den information som krävs för att skapa en ny SMF-tjänst, formaterar den till ett manifest och importerar den till SMF-förvaret. Framgångsrik import kommer att resultera i att tjänsten visas i listan över huvudsakliga tjänster.

Om du klickar på knappen "Sök efter tjänster" visas ett sökformulär. Att gå in i en del av tjänsten FMRI räcker för att hitta alla matchande FMRI: er, till exempel "nfs" hämtar alla nfs-tjänster. En "bläddringsanläggning" tillhandahålls också.
