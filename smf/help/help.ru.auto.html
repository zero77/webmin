Модуль SMF

Эта страница справки является неполной

Введение

Этот модуль позволяет просматривать / изменять / обновлять / добавлять SMF-сервисы в Solaris 10. Главная страница состоит из списка сервисов, и вы можете выбрать определенную категорию, просмотреть отключенные сервисы, отсортировать записи и т. Д. Отсюда вы также можете запустить утилита типа мастера, которая управляет созданием службы, просматривает состояние службы, зависимости / зависимые и просматривает группы свойств / свойства для каждой службы.

Главное окно

По умолчанию в главном окне SMF-модуля перечислены включенные сервисы в системе, показаны их состояния, FMRI, идентифицирующий сервис, и описание, которое поставляется с сервисом. Вы можете применить изменения состояния к нескольким сервисам, проверив нужные сервисы и нажав «включить / отключить» и т. Д. Вы также можете удалить указанные экземпляры сервисов, но, очевидно, избегайте делать это для важных сервисов.

Окно просмотра экземпляра

При нажатии на FMRI определенного экземпляра в главном окне открывается окно просмотра экземпляра, которое отображает имя / описание экземпляра, состояние, любые связанные страницы руководства (которые можно щелкать и просматривать), объяснения необычных состояний (которые можно нажимать и просматриваются), ссылки на связанные файлы журналов (которые снова можно нажать и просмотреть). Отсюда также можно просмотреть деревья зависимостей / зависимостей для экземпляра, щелкнув соответствующую ссылку, и просмотреть / отредактировать группы свойств / свойства, также щелкнув соответствующую ссылку.

Группа свойств / Редактор свойств Windows

При щелчке по ссылке «Просмотр / редактирование групп свойств / свойств» в средстве просмотра экземпляров открывается редактор групп свойств, в котором отображаются группы свойств, связанные с выбранным FMRI. Здесь вы можете добавить / удалить / изменить группы свойств и сделать то же самое со связанными свойствами, щелкнув соответствующую группу свойств в редакторе группы свойств.

Окно Зависимости / Зависимого просмотра

Щелкнув ссылку View Dependency / Dependent Tree в средстве просмотра экземпляра, вы увидите дерево зависимостей / зависимостей для этого экземпляра. По умолчанию в представлении отображаются зависимости / зависимости первого уровня, но, нажав кнопку +, можно развернуть дерево, чтобы увидеть, что зависит от служб, зависящих от службы и т. Д. Как и в главном окне, изменения состояния могут быть применены. для нескольких служб, используя флажки для определения служб, которые вы хотите изменить. Щелчок по экземпляру FMRI вызывает деревья зависимостей / зависимостей для этого экземпляра.

Мастер создания сервиса

При нажатии кнопки «Создать новую службу» открывается мастер создания службы SMF, который собирает необходимую информацию для создания новой службы SMF, форматирует ее в манифест и импортирует в хранилище SMF. Успешный импорт приведет к тому, что сервис появится в списке основных сервисов.

Форма поиска экземпляра сервиса

Предоставляется элементарный поиск FMRI, при котором достаточно ввести часть FMRI службы, чтобы найти все соответствующие FMRI, например, «nfs» извлекает все службы с «nfs» в FMRI. «Просмотр» объекта также поставляется.
