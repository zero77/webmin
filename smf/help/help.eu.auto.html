SMF modulua

Laguntza orri hau osatu gabe dago

Sarrera

Modulu honek Solaris-en 10. SMF zerbitzuak ikusi / aldatu / eguneratu / gehitu ditzakezu. Orrialde nagusia zerbitzuen zerrenda osatzen du. Kategoria jakin bat aukeratu dezakezu, desgaitutako zerbitzariak ikusi, sarrerak ordenatu ... Eta hemendik ere abiarazi ditzakezu. morroi-motako zerbitzua, sorkuntza kudeatzen duena, zerbitzuaren egoera, menpekotasunak / menpekoak eta zerbitzu bakoitzeko propietate-taldeak / propietateak ikusteko.

Leiho nagusia

Lehenespenez, SMF moduluko leiho nagusiak sisteman gaitutako zerbitzuak zerrendatzen ditu, eta haien egoera erakusten du, zerbitzua identifikatzen duen FMRI eta zerbitzuarekin ematen den deskribapena. Zerbitzu aldaketak egoera anitzetan eska ditzakezu nahi dituzun zerbitzariak egiaztatuz, eta gaitu / desgaitu klik eginez. Gainera, zehaztutako zerbitzu instantziak ezabatu ditzakezu, baina saihestu hori egitea zerbitzu garrantzitsuengatik.

Instantzia bistaratzailearen leihoa

Leiho nagusian instantzia jakin baten FMRI klik egitean, instantzia-ikuslea agertzen da, non instantzia izena / deskribapena, egoera, lotutako eskuzko orrialdeak (klikatu eta ikusi daitezke), ezohiko egoerei buruzko azalpenak (klik egin daitezkeenak eta ikusi), lotutako fitxategiei buruzko aipamenak (egin klik berriro ikusi eta ikusteko). Hemendik posible da mendekotasuna / menpeko zuhaitzak kasurako esteka egokia klikatuz eta jabetza / talde / propietateak ikusteko / editatzeko esteka egokia sakatuz.

Jabetza Taldea / Jabetza Editorea Windows

Instantziako ikuslean Talde / Propietate taldeak / Ikusi / Editatu estekan klik egitean jabetza taldeen editorea agertuko da hautatutako FMRIrekin lotutako propietate taldeak bistaratzeko. Hemen propietate taldeak gehitu / ezabatu / aldatu ditzakezu eta gauza bera egin erlazionatutako propietateekin jabetza propietate taldeko editorean klik eginez.

Mendekotasuna / Menpeko Bistaratzailearen leihoa

Instantziaren ikuspegian View Dependency / Dependent Tree Tree estekan klik egitean instantzia horren mendekotasuna / menpeko zuhaitza agertzen da. Berez, ikuspegiak lehen mailako mendekotasunak / mendekotasunak erakusten ditu, baina + botoia sakatzean zuhaitza zabaldu dezakezu zerbitzuaren menpeko zerbitzuen araberakoa den ikusteko. Erakusleiho nagusiaren antzera, egoera aldaketak aplika daitezke. hainbat zerbitzuetara, kontrol-laukiak erabiliz, aldatu nahi dituzun zerbitzuak identifikatzeko. Instantzia baten gainean FMRI klik egiteak mendekotasuna / menpeko zuhaitzak azaltzen dira kasu horretarako.

Zerbitzua sortzeko morroia

"Sortu seribazio berria" botoian klik egitean SMF zerbitzua sortzeko morroia agertzen da. SMF zerbitzu berri bat sortzeko behar den informazioa biltzen du, manifestu batean formatu eta SMF biltegira inportatzen du. Inportazio arrakastatsua zerbitzua zerbitzu nagusien zerrendan agertuko da.

Zerbitzuaren instantzien bilaketa orria

FMRI bilaketak egiten dira. Zerbitzuaren zatiren bat sartzea nahikoa da FMRI guztiak aurkitzeko. Adibidez, "nfs" -ek FMRIn "nfs" dituzten zerbitzu guztiak berreskuratzen ditu. "Arakatzeko" instalazio bat ere eskaintzen da.
