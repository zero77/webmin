Φόρμα αναζήτησης παρουσίας

Αυτή η σελίδα παρέχει μια δυνατότητα αναζήτησης, όπου μπορείτε είτε να χρησιμοποιήσετε μέρος του ονόματος της υπηρεσίας στο πλαίσιο κειμένου ως όρο αναζήτησης (π.χ. nfs) είτε να περιηγηθείτε στο δέντρο FMRI.

Για περισσότερες πληροφορίες

Συμβουλευτείτε τις ακόλουθες μη αυτόματες σελίδες που θα σας βοηθήσουν να γνωρίσετε το SMF:
  • smf (5)
  • service_bundle (4)
  • svccfg (1 εκατ.)
  • svcadm (1M)
  • svc.startd (1M)
  • svc.configd (1M)
  • svcprop (1M)
  • svcs (1M)
