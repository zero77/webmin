Instance Viewer

Informasjonen som vises på denne siden tilsvarer ordentlig tjenesteinformasjon, samlet av svcs -x. Manuell side (hvis tilstede) i serviceinformasjonsdetaljene kan klikkes på og vises, i tillegg til tilknyttede loggfiler (r) og forklaring på tjenestetilstand.

Fra denne siden kan avhengighets / avhengige trær og eiendomsgrupper for forekomsten vises ved å klikke på de aktuelle koblingene nederst på siden.
