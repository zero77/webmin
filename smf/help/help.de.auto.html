Einführung

Mit diesem Modul können Sie SMF-Dienste in Solaris 10 anzeigen / ändern / aktualisieren / hinzufügen. Die Hauptseite besteht aus einer Liste von Diensten. Sie können eine bestimmte Kategorie auswählen, deaktivierte Dienste anzeigen, Einträge sortieren usw. Von hier aus können Sie auch starten Ein Dienstprogramm vom Typ eines Assistenten, das die Erstellung von Diensten verwaltet, den Dienststatus, Abhängigkeiten / Abhängigkeiten anzeigt und Eigenschaftsgruppen / Eigenschaften für jeden Dienst anzeigt.

Hauptfenster

Standardmäßig listet das Hauptfenster des SMF-Moduls die aktivierten Dienste auf dem System auf und zeigt deren Status, den FMRI, der den Dienst identifiziert, und die Beschreibung an, die mit dem Dienst geliefert wird. Sie können Statusänderungen auf mehrere Dienste anwenden, indem Sie die gewünschten Dienste überprüfen und auf Aktivieren / Deaktivieren usw. klicken. Sie können auch die angegebenen Dienstinstanzen löschen, dies jedoch für wichtige Dienste natürlich vermeiden.

Instanz-Viewer-Fenster

Wenn Sie im Hauptfenster auf die FMRI einer bestimmten Instanz klicken, wird der Instanz-Viewer angezeigt, in dem der Name / die Beschreibung der Instanz, der Status, alle zugehörigen Handbuchseiten (die angeklickt und angezeigt werden können) sowie Erklärungen für ungewöhnliche Status (die angeklickt werden können und angezeigt werden) angezeigt werden angezeigt), Verweise auf zugehörige Protokolldateien (die wiederum angeklickt und angezeigt werden können). Von hier aus können Sie auch die Abhängigkeits- / abhängigen Bäume für die Instanz anzeigen, indem Sie auf den entsprechenden Link klicken, und Eigenschaftsgruppen / Eigenschaften anzeigen / bearbeiten, indem Sie auch auf den entsprechenden Link klicken.

Eigenschaftsgruppe / Eigenschafteneditor Windows

Wenn Sie im Instanz-Viewer auf den Link Eigenschaftsgruppen / Eigenschaften anzeigen / bearbeiten klicken, wird der Eigenschaftsgruppen-Editor geöffnet, in dem die dem ausgewählten FMRI zugeordneten Eigenschaftsgruppen angezeigt werden. Sie können hier Eigenschaftsgruppen hinzufügen / löschen / ändern und das Gleiche mit den zugehörigen Eigenschaften tun, indem Sie im Eigenschaftsgruppen-Editor auf die entsprechende Eigenschaftsgruppe klicken.

Fenster "Abhängigkeit / abhängiger Viewer"

Durch Klicken auf den Link Abhängigkeit / Abhängiger Baum anzeigen im Instanz-Viewer wird der Abhängigkeits- / abhängige Baum für diese Instanz angezeigt. Standardmäßig werden in der Ansicht Abhängigkeiten / Abhängigkeiten der ersten Ebene angezeigt. Durch Klicken auf die Schaltfläche + können Sie jedoch die Baumstruktur erweitern, um zu sehen, was von den Diensten abhängt, die vom Dienst usw. abhängen. Ähnlich wie im Hauptfenster können Statusänderungen angewendet werden Verwenden Sie Kontrollkästchen, um die Dienste zu identifizieren, die Sie ändern möchten. Durch Klicken auf eine Instanz FMRI werden die abhängigen / abhängigen Bäume für diese Instanz angezeigt.

Assistent zum Erstellen von Diensten

Durch Klicken auf die Schaltfläche "Neue Dienstleistung erstellen" wird der Assistent zum Erstellen von SMF-Diensten aufgerufen, der die zum Erstellen eines neuen SMF-Dienstes erforderlichen Informationen sammelt, in ein Manifest formatiert und in das SMF-Repository importiert. Ein erfolgreicher Import führt dazu, dass der Dienst in der Liste der Hauptdienste angezeigt wird.

Suchformular für Serviceinstanzen

Es wird eine rudimentäre FMRI-Suche bereitgestellt, bei der die Eingabe eines Teils des Dienstes FMRI ausreicht, um alle übereinstimmenden FMRIs zu finden, z. B. "nfs" ruft alle Dienste mit "nfs" in FMRI ab. Eine "Durchsuchen" -Funktion wird ebenfalls bereitgestellt.
