Introducción

Este módulo le permite ver / modificar / actualizar / agregar servicios SMF en Solaris 10. La página principal consiste en una lista de servicios, y puede seleccionar una categoría específica, ver servicios deshabilitados, ordenar entradas, etc. Desde aquí también puede iniciar Una utilidad de tipo asistente que gestiona la creación de servicios, ver el estado del servicio, dependencias / dependientes y ver grupos de propiedades / propiedades para cada servicio.

Ventana principal

De manera predeterminada, la ventana principal del módulo SMF enumera los servicios habilitados en el sistema, mostrando sus estados, el FMRI que identifica el servicio y la descripción que se proporciona con el servicio. Puede aplicar cambios de estado a múltiples servicios marcando los servicios deseados y haciendo clic en habilitar / deshabilitar, etc. También puede eliminar las instancias de servicio especificadas, pero obviamente evite hacer esto para servicios importantes.

Ventana del visor de instancias

Al hacer clic en el FMRI de una instancia en particular en la ventana principal, aparece el visor de instancias, que muestra el nombre / descripción de la instancia, el estado, cualquier página de manual asociada (que se puede hacer clic y ver), explicaciones de estados inusuales (que se puede hacer clic y visto), referencias a los archivos de registro asociados (que de nuevo se puede hacer clic y ver). Desde aquí también es posible ver la dependencia / árboles dependientes para la instancia haciendo clic en el enlace apropiado, y ver / editar grupos de propiedades / propiedades haciendo clic también en el enlace apropiado.

Grupo de propiedades / Editor de propiedades Windows

Al hacer clic en el enlace Ver / Editar grupos de propiedades / Propiedades en el visor de instancias, aparece el editor de grupos de propiedades, que muestra los grupos de propiedades asociados con el FMRI seleccionado. Puede agregar / eliminar / modificar grupos de propiedades aquí, y hacer lo mismo con las propiedades asociadas haciendo clic en el grupo de propiedades relevante en el editor de grupos de propiedades.

Ventana de dependencia / visor dependiente

Al hacer clic en el enlace Ver árbol de dependencia / dependiente en el visor de instancias, aparece el árbol de dependencia / dependencia para esa instancia. De forma predeterminada, la vista muestra dependencias / dependientes de primer nivel, pero al hacer clic en el botón +, puede expandir el árbol para ver qué depende de los servicios que dependen del servicio, etc. De manera similar a la ventana principal, se pueden aplicar cambios de estado a múltiples servicios, usando casillas de verificación para identificar los servicios que desea cambiar. Al hacer clic en una instancia, FMRI muestra la dependencia / árboles dependientes para esa instancia.

Asistente de creación de servicios

Al hacer clic en el botón "Crear nuevo servicio", aparece el asistente de creación de servicios SMF, que recopila la información requerida para crear un nuevo servicio SMF, lo formatea en un manifiesto y lo importa al repositorio SMF. La importación exitosa hará que el servicio aparezca en la lista de servicios principales.

Formulario de búsqueda de instancia de servicio

Se proporciona una búsqueda FMRI rudimentaria, mediante la cual ingresar una parte del servicio FMRI es suficiente para encontrar todas las FMRI coincidentes, por ejemplo, "nfs" recupera todos los servicios con "nfs" en FMRI. También se proporciona una función de "exploración".
