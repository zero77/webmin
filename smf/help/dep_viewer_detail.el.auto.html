Εξάρτηση / Εξαρτώμενη προβολή δέντρου

Οι πληροφορίες που εμφανίζονται σε αυτήν τη σελίδα αντιστοιχούν σε εξάρτηση / εξαρτώμενες πληροφορίες δέντρου, όπως συγκεντρώθηκαν από το svcs -d / -D, δηλαδή σε ποιες υπηρεσίες εξαρτώνται από αυτήν την υπηρεσία και σε ποιες υπηρεσίες εξαρτάται; Οι αλλαγές κατάστασης μπορούν να εφαρμοστούν σε εξαρτώμενα άτομα / εξαρτήσεις και κάνοντας κλικ σε μια εξαρτώμενη / εξάρτηση fmri σας μεταφέρει στη σελίδα επεξεργασίας κατάστασης.

Η εξάρτηση / εξαρτώμενα δέντρα μπορούν να επεκταθούν / καταρρεύσουν σε οποιοδήποτε επιθυμητό επίπεδο κάνοντας κλικ στα κουμπιά "+" και "-" αντίστοιχα. Οι αλλαγές κατάστασης μπορούν να εφαρμοστούν στην υπηρεσία και σε επιλεγμένες εξαρτήσεις / εξαρτώμενα άτομα χρησιμοποιώντας πλαίσια ελέγχου για να επιλέξετε τις υπηρεσίες.

Κάνοντας κλικ σε ένα FMRI στο δέντρο εξάρτησης / εξάρτησης θα μεταφερθείτε στο δέντρο εξάρτησης / εξάρτησης για αυτήν την περίπτωση.

Για περισσότερες πληροφορίες

Συμβουλευτείτε τις ακόλουθες μη αυτόματες σελίδες που θα σας βοηθήσουν να γνωρίσετε το SMF:
  • smf (5)
  • service_bundle (4)
  • svccfg (1 εκατ.)
  • svcadm (1M)
  • svc.startd (1M)
  • svc.configd (1M)
  • svcprop (1M)
  • svcs (1M)
