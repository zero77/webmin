SMF Module Instance Viewer

Denne hjælpeside er ufuldstændig

Instance Viewer

Oplysningerne, der vises på denne side, svarer til verbose serviceinformation, samlet af svcs -x. Den manuelle side (hvis den er til stede) i detaljerne om serviceoplysninger kan klikkes på og vises, det kan også være de tilknyttede logfiler og forklaringer på servicetilstanden.

Fra denne side kan afhængighed / afhængige træer og egenskabsgrupper for eksemplet ses ved at klikke på de relevante links nederst på siden.
