SMF 모듈

이 도움말 페이지가 불완전합니다

소개

이 모듈을 사용하면 Solaris 10에서 SMF 서비스를 보거나 수정 / 업데이트 / 추가 할 수 있습니다. 기본 페이지는 서비스 목록으로 구성되며 특정 범주를 선택하고 비활성화 된 서비스를보고 항목을 정렬 할 수 있습니다. 여기에서 시작할 수도 있습니다. 서비스 생성을 관리하고 서비스 상태, 종속성 / 종속성을보고 각 서비스의 속성 그룹 / 속성을 확인하는 마법사 유형 유틸리티.

메인 창

기본적으로 기본 SMF 모듈 창에는 시스템에서 사용 가능한 서비스가 표시되며 상태, 서비스를 식별하는 FMRI 및 서비스와 함께 제공되는 설명이 표시됩니다. 원하는 서비스를 확인하고 활성화 / 비활성화 등을 클릭하여 여러 서비스에 상태 변경을 적용 할 수 있습니다. 지정된 서비스 인스턴스를 삭제할 수도 있지만 중요한 서비스에 대해서는이를 피할 수 있습니다.

인스턴스 뷰어 창

기본 창에서 특정 인스턴스의 FMRI를 클릭하면 인스턴스 이름 / 설명, 상태, 관련 매뉴얼 페이지 (클릭하여 볼 수 있음), 비정상 상태에 대한 설명 (클릭 및 관련 로그 파일에 대한 참조 (다시 클릭하고 볼 수 있음). 여기에서 적절한 링크를 클릭하여 인스턴스의 종속성 / 종속 트리를보고 해당 링크를 클릭하여 속성 그룹 / 속성을 보거나 편집 할 수도 있습니다.

속성 그룹 / 속성 편집기 창

인스턴스 뷰어에서 속성 그룹 / 속성보기 / 편집 링크를 클릭하면 속성 그룹 편집기가 나타나고 선택한 FMRI와 관련된 속성 그룹이 표시됩니다. 여기에서 특성 그룹을 추가 / 삭제 / 수정할 수 있으며 특성 그룹 편집기에서 관련 특성 그룹을 클릭하여 연관된 특성과 동일하게 수행 할 수 있습니다.

종속성 / 의존 뷰어 창

인스턴스 뷰어에서 종속성 / 의존 트리보기 링크를 클릭하면 해당 인스턴스의 종속성 / 의존 트리가 나타납니다. 기본적으로보기에는 첫 번째 레벨 종속성 / 종속성이 표시되지만 + 단추를 클릭하면 트리를 확장하여 서비스 등에 의존하는 서비스에 의존하는 서비스를 확인할 수 있습니다. 기본 창과 유사하게 상태 변경 사항을 적용 할 수 있습니다. 확인란을 사용하여 변경하려는 서비스를 식별합니다. 인스턴스 FMRI를 클릭하면 해당 인스턴스의 종속성 / 종속 트리가 나타납니다.

서비스 생성 마법사

"새 서비스 작성"단추를 클릭하면 SMF 서비스 작성 마법사가 나타나 새 SMF 서비스를 작성하는 데 필요한 정보를 수집하고이를 매니페스트로 형식화 한 후 SMF 저장소로 가져옵니다. 가져 오면 서비스가 기본 서비스 목록에 나타납니다.

서비스 인스턴스 검색 양식

기본적인 FMRI 검색이 제공되어 서비스의 일부를 입력하면 일치하는 모든 FMRI를 찾기에 충분합니다. 예를 들어 "nfs"는 FMRI에서 "nfs"를 가진 모든 서비스를 검색합니다. "찾아보기"기능도 제공됩니다.
