Επεξεργαστής ιδιοκτησίας

Οι πληροφορίες που εμφανίζονται σε αυτήν τη σελίδα αντιστοιχούν στη λίστα ιδιοτήτων υπηρεσίας για την επιλεγμένη ομάδα ιδιοτήτων επιπέδου υπηρεσίας / επιπέδου εμφάνισης. Οι ιδιότητες μπορούν να προστεθούν, να διαγραφούν και να ενημερωθούν.

Για περισσότερες πληροφορίες

Συμβουλευτείτε τις ακόλουθες μη αυτόματες σελίδες που θα σας βοηθήσουν να γνωρίσετε το SMF:
  • smf (5)
  • service_bundle (4)
  • svccfg (1 εκατ.)
  • svcadm (1M)
  • svc.startd (1M)
  • svc.configd (1M)
  • svcprop (1M)
  • svcs (1M)
