Abhängigkeits- / Abhängiger Baum-Viewer

Die auf dieser Seite angezeigten Informationen entsprechen Informationen zu Abhängigkeiten / abhängigen Bäumen, die von svcs -d / -D erfasst wurden. Welche Dienste hängen von diesem Dienst ab und von welchen Diensten hängt er ab? Statusänderungen können auf Abhängige / Abhängigkeiten angewendet werden. Wenn Sie auf eine abhängige / Abhängigkeits-FMRI klicken, gelangen Sie zu deren Statuseditor-Seite.

Die abhängigen / abhängigen Bäume können durch Klicken auf die Schaltflächen "+" bzw. "-" auf eine beliebige Ebene erweitert / reduziert werden. Statusänderungen können auf den Dienst und ausgewählte Abhängigkeiten / Abhängige angewendet werden, indem Kontrollkästchen zum Auswählen der Dienste verwendet werden.

Durch Klicken auf einen FMRI im Abhängigkeits- / Abhängigkeitsbaum gelangen Sie zum Abhängigkeits- / Abhängigkeitsbaum für diese Instanz.
