Property Group Editor

Informationen som visas på denna sida överensstämmer med listan över servicegruppsgrupper för både tjänsten och vald instans. Egenskapsgrupper på servicenivå visas eftersom deras tillhörande egenskaper ärvs av instansen om de inte åsidosätts. Genom att klicka på egendomsgruppens namn kan visning / redigering av enskilda egenskaper inom den valda egendomsgruppen. Fastighetsgrupper kan också läggas till / raderas här.
