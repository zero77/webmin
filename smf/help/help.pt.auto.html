Introdução

Este módulo permite visualizar / modificar / atualizar / adicionar serviços SMF no Solaris 10. A página principal consiste em uma lista de serviços e você pode selecionar uma categoria específica, exibir serviços desativados, classificar entradas etc. A partir daqui, você também pode iniciar um utilitário do tipo assistente que gerencia a criação do serviço, exibe o estado do serviço, dependências / dependente e exibe grupos / propriedades de propriedades para cada serviço.

Janela principal

Por padrão, a janela principal do módulo SMF lista os serviços ativados no sistema, mostrando seus estados, o FMRI que identifica o serviço e a descrição fornecida com o serviço. Você pode aplicar alterações de estado a vários serviços, verificando os serviços desejados e clicando em ativar / desativar etc. Você também pode excluir as instâncias de serviço especificadas, mas obviamente evite fazer isso em serviços importantes.

Janela Visualizador de Instâncias

Clicar no FMRI de uma instância específica na janela principal exibe o visualizador da instância, que exibe o nome / descrição da instância, estado, quaisquer páginas de manual associadas (que podem ser clicadas e visualizadas), explicações para estados incomuns (que podem ser clicadas e visualizado), referências aos arquivos de log associados (que podem ser clicados e visualizados novamente). A partir daqui, também é possível visualizar as árvores de dependência / dependência da instância, clicando no link apropriado, e visualizar / editar grupos / propriedades de propriedades, clicando também no link apropriado.

Grupo de propriedades / Editor de propriedades do Windows

Clicar no link Visualizar / Editar Grupos / Propriedades de Propriedades no visualizador de instâncias exibe o editor de grupos de propriedades, exibindo os grupos de propriedades associados à FMRI selecionada. Você pode adicionar / excluir / modificar grupos de propriedades aqui e fazer o mesmo com as propriedades associadas clicando no grupo de propriedades relevante no editor de grupos de propriedades.

Janela Dependência / Visualizador Dependente

Clicar no link Exibir dependência / árvore dependente no visualizador de instâncias exibe a árvore de dependência / dependente para essa instância. Por padrão, a exibição mostra dependências / dependentes de primeiro nível, mas, clicando no botão +, você pode expandir a árvore para ver o que depende dos serviços que dependem do serviço, etc. Da mesma forma que na janela principal, as alterações de estado podem ser aplicadas para vários serviços, usando caixas de seleção para identificar os serviços que você deseja alterar. Clicar em uma instância FMRI exibe as árvores de dependência / dependentes dessa instância.

Assistente de Criação de Serviço

Clicar no botão "Create New Serivce" abre o assistente de criação de serviço SMF, que coleta as informações necessárias para criar um novo serviço SMF, formata-o em um manifesto e importa-o no repositório SMF. A importação bem-sucedida fará com que o serviço apareça na lista de serviços principal.

Formulário de Pesquisa de Instância de Serviço

É fornecida uma pesquisa FMRI rudimentar, em que entrar em uma parte do serviço FMRI é suficiente para encontrar todos os FMRIs correspondentes, por exemplo, "nfs" recupera todos os serviços com "nfs" no FMRI. Um recurso de "navegação" também é fornecido.
