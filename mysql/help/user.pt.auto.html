Para cada usuário, os seguintes parâmetros estão disponíveis:
Nome do usuário
Nesse campo, você pode inserir um nome de usuário específico ou escolher a opção Usuário anônimo. Nesse caso, as permissões serão aplicadas a qualquer usuário que não esteja na lista Permissões do usuário .

Senha
A senha exigida por este usuário ao efetuar login. Se Nenhum , nenhuma senha será necessária.

Anfitriões
O endereço IP ou o nome do host do qual este usuário tem permissão para efetuar login. Este campo também pode conter uma expressão regular SQL correspondente a vários hosts, como 192.168.1.% Ou %.foo.com .

Permissões
As operações do MySQL que este usuário pode executar em qualquer banco de dados.
