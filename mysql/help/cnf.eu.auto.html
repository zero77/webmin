MySQL zerbitzariaren konfigurazioa
Orrialde hau MySQL datu-baseko zerbitzariaren konfigurazio globalaren doikuntzak doitzeko erabil daiteke. Normalean ez duzu hauetakoren bat aldatu beharrik, nahiz eta bufferraren eta cachearen tamainak taulak eta datuak kopuru handia duten sistemetan handitu daitezkeen.

Aldaketarik izan dadin, MySQL zerbitzaria gelditu eta berriro hasi behar da moduluaren orri nagusiko botoiak erabiliz.
