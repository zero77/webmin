Редактировать поле
На этой странице вы можете редактировать существующее поле таблицы. Хотя базовый тип данных поля не может быть изменен, вы можете изменять другие параметры поля, такие как: По завершении нажмите кнопку « Сохранить» , чтобы обновить структуру таблицы, или кнопку « Удалить», чтобы удалить поле из таблицы.
