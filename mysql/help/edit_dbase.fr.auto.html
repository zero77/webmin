Modifier la base de données
En haut de cette page se trouvent des rangées d'icônes, chacune représentant une table dans la base de données. Pour éditer la structure d'un tableau, cliquez simplement sur l'une des icônes.

Vous trouverez ci-dessous des boutons pour créer une nouvelle table dans cette base de données, exécuter SQL sur les tables de la base de données et supprimer la base de données et toutes les tables qu'elle contient. Le bouton final doit être utilisé avec précaution, car il n'est pas réversible!
