Datenbank bearbeiten
Am oberen Rand dieser Seite befinden sich Reihen von Symbolen, die jeweils eine Tabelle in der Datenbank darstellen. Um die Struktur einer Tabelle zu bearbeiten, klicken Sie einfach auf eines der Symbole.

Im Folgenden finden Sie Schaltflächen zum Erstellen einer neuen Tabelle in dieser Datenbank, zum Ausführen von SQL für die Tabellen der Datenbank und zum Löschen der Datenbank und aller darin enthaltenen Tabellen. Der letzte Knopf sollte mit Vorsicht verwendet werden, da er nicht umkehrbar ist!
