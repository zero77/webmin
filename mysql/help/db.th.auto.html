มีพารามิเตอร์ต่อไปนี้:
ฐานข้อมูล
ฐานข้อมูลที่อนุญาตการเข้าถึง นี่อาจเป็น SQL regexp ถ้าต้องการ

ชื่อผู้ใช้
ผู้ใช้ที่ได้รับอนุญาตให้เข้าถึงฐานข้อมูลข้างต้น ผู้ใช้รายนี้ต้องมีอยู่แล้วในตาราง สิทธิ์ผู้ใช้

โฮสต์
โฮสต์ที่ผู้ใช้สามารถเข้าถึงฐานข้อมูลได้ หากเลือกตัวเลือก จากสิทธิ์ของโฮสต์ แล้ว MySQL ควรใช้ สิทธิ์ของโฮสต์ เพื่อควบคุมการเข้าถึง

สิทธิ์
การดำเนินงานของ MySQL ที่ผู้ใช้ได้รับอนุญาตให้ดำเนินการบนฐานข้อมูล
