Numele domeniului
Numele acestui câmp din tabel.

Valori enumerate (pentru tipurile de enumer și set )
O listă de valori posibile pentru acest câmp.

Lățime și zecimale (pentru tipuri float și zecimale )
Lățimea totală a acestui câmp și numărul de cifre din dreapta punctului zecimal.

Lățimea tipului (pentru alte tipuri)
Numărul maxim de caractere permis pentru datele din acest câmp.

Opțiuni de tip
Pentru câmpurile numerice, acest parametru vă permite să alegeți dacă câmpul este nesemnat sau căptușit la stânga cu zerouri. Pentru câmpurile de caractere, puteți alege dacă se ține cont de caz la sortare sau nu.

Permite zerourile?
Determină dacă valorile NULL sunt permise în acest câmp

Valoare implicită
Valoarea implicită a acestui câmp în înregistrările noi
