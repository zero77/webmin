Δημιουργία πίνακα
Αυτή η φόρμα προορίζεται για τη δημιουργία ενός νέου, κενού πίνακα. Πρέπει να εισαγάγετε ένα όνομα πίνακα και ένα ή περισσότερα πεδία. Για κάθε πεδίο, πρέπει να εισαγάγετε ένα όνομα, να επιλέξετε έναν τύπο δεδομένων και να εισαγάγετε το πλάτος του τύπου (εάν είναι κατάλληλο για τον τύπο δεδομένων). Όταν τελειώσετε, κάντε κλικ στο κουμπί Δημιουργία για να δημιουργήσετε αμέσως τον πίνακα.
