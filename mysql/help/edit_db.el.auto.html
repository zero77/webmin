Επεξεργασία δικαιωμάτων βάσης δεδομένων
Αυτή η σελίδα σάς επιτρέπει να επεξεργαστείτε ένα υπάρχον σύνολο αδειών χρήστη βάσης δεδομένων. Μετά την επεξεργασία, κάντε κλικ στο κουμπί Αποθήκευση για να εφαρμοστούν αμέσως οι αλλαγές. Μπορείτε επίσης να κάνετε κλικ στο Delete για να διαγράψετε τα δικαιώματα από τη MySQL.
