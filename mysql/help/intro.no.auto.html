MySQL Database Server
MySQL er en enkel databaseserver for Unix-systemer. Den støtter flere databaser, kraftige tilgangskontroller, mange datatyper og det meste av standard SQL-syntaks. Hver database administrert av serveren kan inneholde flere tabeller, og hver tabell inneholder flere rader med data.

Hovedsiden viser en liste over databaseikoner øverst, med en kobling for å legge til en ny database under dem. Hver MySQL-server har minst mysql- databasen, som lagrer konfigurasjons- og tilgangskontrolltabeller. Under databasene er ikoner for å konfigurere tilgangskontroll, og nederst er en knapp for å stoppe eller starte databaseserveren.
