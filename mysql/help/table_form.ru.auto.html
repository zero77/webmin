Создать таблицу
Эта форма предназначена для создания новой пустой таблицы. Вы должны ввести имя таблицы и одно или несколько полей. Для каждого поля необходимо ввести имя, выбрать тип данных и ввести ширину типа (если это соответствует типу данных). Когда закончите, нажмите кнопку Создать , чтобы немедленно создать таблицу.
