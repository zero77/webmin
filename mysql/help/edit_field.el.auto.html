Επεξεργασία πεδίου
Σε αυτήν τη σελίδα μπορείτε να επεξεργαστείτε ένα υπάρχον πεδίο πίνακα. Ενώ ο βασικός τύπος δεδομένων του πεδίου δεν μπορεί να αλλάξει, επιτρέπεται να τροποποιήσετε άλλες παραμέτρους πεδίου όπως: Όταν τελειώσετε, κάντε κλικ στο κουμπί Αποθήκευση για να ενημερώσετε τη δομή του πίνακα ή στο Διαγραφή για να αφαιρέσετε το πεδίο από τον πίνακα.
