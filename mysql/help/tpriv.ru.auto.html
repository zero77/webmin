Доступны следующие параметры:
Таблица
Таблица, к которой должны применяться эти разрешения.

имя пользователя
Пользователь, которому предоставлены разрешения. Этот пользователь уже должен существовать в таблице « Полномочия пользователя» .

Хосты
Хосты, с которых пользователь может получить доступ к таблице.

Таблица разрешений
Права, предоставленные пользователю для всех столбцов таблицы.

Полевые разрешения
Максимальные разрешения, которые могут быть предоставлены этому пользователю для любого столбца таблицы на странице Полевые разрешения .
