Konfigurace serveru MySQL
Tuto stránku lze použít k úpravě globálních nastavení konfigurace pro databázový server MySQL. Obvykle nebudete muset nic z toho měnit, ačkoli velikosti vyrovnávací paměti a mezipaměti lze v systémech s velkým počtem tabulek a dat zvětšit.

Aby se všechny změny projevily, musí být server MySQL zastaven a znovu spuštěn pomocí tlačítek na hlavní stránce modulu.
