Databázový server MySQL
MySQL je jednoduchý databázový server pre systémy Unix. Podporuje viac databáz, výkonné riadenie prístupu, veľa typov údajov a väčšinu štandardnej syntaxe SQL. Každá databáza riadená serverom môže obsahovať viac tabuliek a každá tabuľka obsahuje viac riadkov údajov.

Na hlavnej stránke sa v hornej časti zobrazuje zoznam ikon databázy s odkazom na pridanie novej databázy pod nimi. Každý server MySQL bude mať aspoň databázu mysql , ktorá ukladá tabuľky konfigurácie a riadenia prístupu. Pod databázami sú ikony na konfiguráciu riadenia prístupu a dole je tlačidlo na zastavenie alebo spustenie databázového servera.
