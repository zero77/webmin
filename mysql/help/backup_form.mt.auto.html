Database tad-Dejt
Din il-formola jippermettilek tagħmel backup ta 'database sħiħa (jew ringieli magħżula) għal fajl. Agħżel il-fajl li tikteb il-backup lil u l-għażliet l-oħra, imbagħad ikklikkja l-buttuna Backup Now biex tibda l-proċess ta 'backup. Bil-miktub fil-fajl se jkun hemm dikjarazzjonijiet SQL li meta jiġu eżegwiti se jerġgħu joħolqu t-tabelli tad-database u l-kontenut tagħhom. Dan il-fajl jista 'jiġi eżegwit billi tuża l-programm tal-linja ta' kmand mysql jew il-paġna Eżekita SQL, sabiex terġa 'tiġi stabbilita d-database.
