پیکربندی سرور MySQL
این صفحه می تواند برای تنظیم تنظیمات پیکربندی جهانی برای سرور پایگاه داده MySQL مورد استفاده قرار گیرد. به طور معمول نیازی به تغییر هیچ یک از این موارد نخواهید داشت ، اگرچه اندازه های بافر و حافظه نهان در سیستم هایی که تعداد زیادی جدول و داده دارند قابل افزایش است.

برای اعمال هرگونه تغییر ، سرور MySQL باید متوقف شود و مجدداً با استفاده از دکمه های صفحه اصلی ماژول مجدداً کار خود را شروع کنید.
