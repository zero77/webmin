เซิร์ฟเวอร์ฐานข้อมูล MySQL
MySQL เป็นเซิร์ฟเวอร์ฐานข้อมูลอย่างง่ายสำหรับระบบ Unix รองรับหลายฐานข้อมูลควบคุมการเข้าถึงที่มีประสิทธิภาพประเภทข้อมูลจำนวนมากและไวยากรณ์ SQL มาตรฐานส่วนใหญ่ แต่ละฐานข้อมูลที่จัดการโดยเซิร์ฟเวอร์สามารถมีหลายตารางและแต่ละตารางมีข้อมูลหลายแถว

หน้าหลักจะแสดงรายการไอคอนฐานข้อมูลที่ด้านบนพร้อมลิงค์สำหรับเพิ่มฐานข้อมูลใหม่ด้านล่าง เซิร์ฟเวอร์ MySQL ทุกตัวจะมีฐานข้อมูลอย่างน้อย mysql ซึ่งจัดเก็บการกำหนดค่าและตารางควบคุมการเข้าถึง ด้านล่างฐานข้อมูลคือไอคอนสำหรับการกำหนดค่าการควบคุมการเข้าถึงและที่ด้านล่างเป็นปุ่มสำหรับหยุดหรือเริ่มต้นเซิร์ฟเวอร์ฐานข้อมูล
