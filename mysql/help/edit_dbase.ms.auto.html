Edit Pangkalan Data
Di bahagian atas halaman ini terdapat baris ikon, masing-masing mewakili satu jadual dalam pangkalan data. Untuk mengedit struktur jadual, klik pada salah satu ikon.

Berikut adalah butang untuk membuat jadual baru dalam pangkalan data ini, menjalankan SQL pada jadual pangkalan data dan menjatuhkan pangkalan data dan semua jadual di dalamnya. Butang terakhir harus digunakan dengan berhati-hati, kerana ia tidak boleh diterbalikkan!
