Les paramètres suivants sont disponibles:
Champ
Le champ auquel ces autorisations doivent s'appliquer.

Nom d'utilisateur
L'utilisateur à qui les autorisations sont accordées. Cet utilisateur doit déjà exister dans la table Autorisations utilisateur .

Hôtes
Les hôtes à partir desquels l'utilisateur peut accéder à la table.

Autorisations
Les autorisations accordées à l'utilisateur sur la colonne choisie du tableau.
