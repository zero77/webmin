MySQL сервер баз даних
MySQL - це простий сервер баз даних для систем Unix. Він підтримує безліч баз даних, потужне управління доступом, багато типів даних і більшість стандартних синтаксисів SQL. Кожна база даних, керована сервером, може містити кілька таблиць, а кожна таблиця містить кілька рядків даних.

На головній сторінці розміщується список піктограм баз даних у верхній частині із посиланням для додавання нової бази даних під ними. Кожен сервер MySQL матиме принаймні базу даних mysql , в якій зберігаються таблиці конфігурації та управління доступом. Внизу баз даних значки для налаштування контролю доступу, а внизу - кнопка для зупинки або запуску сервера баз даних.
