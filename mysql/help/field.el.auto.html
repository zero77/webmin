Ονομα πεδίου
Το όνομα αυτού του πεδίου στον πίνακα.

Αριθμημένες τιμές (για τύπους enum και set )
Μια λίστα πιθανών τιμών για αυτό το πεδίο.

Πλάτος και δεκαδικά (για τύπους πλωτήρα και δεκαδικών )
Το συνολικό πλάτος αυτού του πεδίου και ο αριθμός των ψηφίων στα δεξιά της υποδιαστολής.

Πλάτος τύπου (για άλλους τύπους)
Ο μέγιστος αριθμός χαρακτήρων που επιτρέπονται για δεδομένα σε αυτό το πεδίο.

Πληκτρολογήστε επιλογές
Για αριθμητικά πεδία, αυτή η παράμετρος σας επιτρέπει να επιλέξετε εάν το πεδίο δεν είναι υπογεγραμμένο ή αριστερά με μηδενικά. Για τα πεδία χαρακτήρων, μπορείτε να επιλέξετε εάν η περίπτωση λαμβάνεται υπόψη κατά την ταξινόμηση ή όχι.

Επιτρέπω μηδενικά?
Καθορίζει εάν επιτρέπονται τιμές NULL σε αυτό το πεδίο

Προεπιλεγμένη τιμή
Η προεπιλεγμένη τιμή για αυτό το πεδίο σε νέες εγγραφές
