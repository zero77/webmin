Pangkalan Data Sandaran
Borang ini membolehkan anda membuat sandaran keseluruhan pangkalan data (atau baris yang dipilih) ke fail. Pilih fail untuk menulis cadangan dan pilihan lain, kemudian klik butang Sandaran Sekarang untuk memulakan proses sandaran. Tertulis ke fail tersebut akan menjadi pernyataan SQL yang apabila dijalankan akan membuat semula jadual pangkalan data dan isinya. File ini dapat dijalankan menggunakan program mysql baris perintah atau halaman Execute SQL, untuk memulihkan pangkalan data.
