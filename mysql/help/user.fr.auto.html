Pour chaque utilisateur, les paramètres suivants sont disponibles:
Nom d'utilisateur
Pour ce champ, vous pouvez soit saisir un nom d'utilisateur spécifique, soit choisir l'option Utilisateur anonyme, auquel cas les autorisations s'appliqueront à tout utilisateur qui ne figure pas dans la liste Autorisations utilisateur .

Mot de passe
Mot de passe requis par cet utilisateur lors de la connexion. Si aucun , aucun mot de passe n'est nécessaire.

Hôtes
L'adresse IP ou le nom d'hôte de l'hôte à partir duquel cet utilisateur est autorisé à se connecter. Ce champ peut également contenir une expression régulière SQL correspondant à plusieurs hôtes, comme 192.168.1.% Ou %.foo.com .

Autorisations
Les opérations MySQL que cet utilisateur est autorisé à effectuer sur n'importe quelle base de données.
