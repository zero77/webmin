Les paramètres suivants sont disponibles:
Bases de données
Les bases de données auxquelles l'accès est autorisé. Cela peut être une expression rationnelle SQL si vous le souhaitez.

Nom d'utilisateur
L'utilisateur est autorisé à accéder à la base de données ci-dessus. Cet utilisateur doit déjà exister dans la table Autorisations utilisateur .

Hôtes
Les hôtes à partir desquels l'utilisateur peut accéder à la base de données. Si l'option Depuis les autorisations d'hôte est choisie, MySQL doit utiliser les autorisations d'hôte pour contrôler l'accès.

Autorisations
Les opérations MySQL que l'utilisateur est autorisé à effectuer sur la base de données.
