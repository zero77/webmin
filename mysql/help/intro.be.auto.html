MySQL Server Database Server
MySQL - просты сервер баз дадзеных для сістэм Unix. Ён падтрымлівае некалькі баз дадзеных, магутны кантроль доступу, мноства тыпаў дадзеных і большасць стандартных сінтаксісаў SQL. Кожная база дадзеных, якая кіруецца серверам, можа ўтрымліваць некалькі табліц, і кожная табліца змяшчае некалькі радкоў дадзеных.

На галоўнай старонцы прадстаўлены спіс абразкоў баз дадзеных уверсе, а таксама спасылка для дадання новай базы дадзеных пад імі. Кожны сервер MySQL будзе мець як мінімум базу дадзеных mysql , у якой захоўваюцца табліцы налад і кантроль доступу. Унізе баз дадзеных знаходзяцца значкі для налады кантролю доступу, а ўнізе размешчана кнопка для прыпынку або запуску сервера баз дадзеных.
