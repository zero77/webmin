Резервная база данных
Эта форма позволяет сделать резервную копию всей базы данных (или выбранных строк) в файл. Выберите файл, укажите резервную копию и другие параметры, затем нажмите кнопку « Резервное копирование сейчас» , чтобы начать процесс резервного копирования. В файл будут записаны операторы SQL, которые при выполнении восстанавливают таблицы базы данных и их содержимое. Этот файл может быть выполнен, используя программу mysql командной строки или страницу Execute SQL, чтобы восстановить базу данных.
