แก้ไขตาราง
ตารางที่ใช้หน้านี้ส่วนใหญ่จะแสดงโครงสร้างของตารางที่เลือกในฐานข้อมูลของคุณ แต่ละแถวมีรายละเอียดหนึ่งฟิลด์ในตารางโดยมีชื่อประเภทข้อมูลและตัวเลือกอื่น ๆ ที่แสดง

ที่ด้านล่างซ้ายมีปุ่มสำหรับเพิ่มเขตข้อมูลใหม่ลงในตาราง ก่อนที่จะคลิกคุณต้องเลือกประเภทของฟิลด์ที่จะเพิ่ม - ดูเอกสาร MySQL สำหรับข้อมูลเพิ่มเติมเกี่ยวกับประเภทของฟิลด์ นอกจากนี้ที่ด้านล่างยังมีปุ่มสำหรับดูข้อมูลทั้งหมดในตารางและวางตารางทั้งหมดและข้อมูลทั้งหมดในตาราง
