Доступны следующие параметры:
Базы данных
Базы данных, к которым разрешен доступ. При желании это может быть регулярное выражение SQL.

имя пользователя
Пользователь разрешил доступ к вышеуказанной базе данных. Этот пользователь уже должен существовать в таблице « Полномочия пользователя» .

Хосты
Хосты, с которых пользователь может получить доступ к базе данных. Если выбран параметр « От разрешений узла» , то MySQL должен использовать разрешения узла для управления доступом.

права доступа
Операции MySQL, которые пользователь может выполнять в базе данных.
