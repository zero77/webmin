Permissões do usuário
Esta página permite criar e editar usuários que terão acesso aos seus bancos de dados MySQL. Para cada usuário, você pode especificar um nome de usuário, senha, hosts permitidos e um conjunto de permissões que determinam o que o usuário pode fazer.

Como qualquer usuário listado que tenha permissões terá essas permissões em todos os bancos de dados, geralmente é uma boa idéia não conceder permissões aos usuários nesta página. Em vez disso, use a página Permissões do banco de dados para conceder acesso aos usuários por banco de dados.
