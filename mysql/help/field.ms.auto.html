Nama bidang
Nama bidang ini dalam jadual.

Nilai yang dihitung (untuk jenis enum dan set )
Senarai kemungkinan nilai untuk bidang ini.

Lebar dan perpuluhan (untuk jenis apungan dan perpuluhan )
Lebar keseluruhan bidang ini, dan bilangan digit di sebelah kanan titik perpuluhan.

Lebar jenis (untuk jenis lain)
Jumlah aksara maksimum yang dibenarkan untuk data dalam bidang ini.

Jenis pilihan
Untuk medan angka, parameter ini membolehkan anda memilih sama ada medan tidak ditandatangani atau dibiarkan kiri dengan angka nol. Untuk bidang watak, anda boleh memilih sama ada kes diambil kira semasa menyusun atau tidak.

Benarkan nol?
Menentukan sama ada nilai NULL dibenarkan dalam bidang ini

Nilai asal
Nilai lalai untuk medan ini dalam rekod baru
