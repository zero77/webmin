ฐานข้อมูลสำรอง
ฟอร์มนี้ช่วยให้คุณสามารถสำรองฐานข้อมูลทั้งหมด (หรือแถวที่เลือก) ไปยังไฟล์ เลือกไฟล์เขียนข้อมูลสำรองและตัวเลือกอื่น ๆ จากนั้นคลิกปุ่ม สำรองข้อมูลตอนนี้ เพื่อเริ่มกระบวนการสำรองข้อมูล เขียนไปยังไฟล์จะเป็นคำสั่ง SQL ที่เมื่อดำเนินการจะสร้างตารางฐานข้อมูลและเนื้อหาของพวกเขา ไฟล์นี้สามารถเรียกใช้งานโดยใช้โปรแกรม mysql บรรทัดคำสั่งหรือหน้า Execute SQL เพื่อคืนค่าฐานข้อมูล
