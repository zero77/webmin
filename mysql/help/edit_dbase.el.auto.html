Επεξεργασία βάσης δεδομένων
Στην κορυφή αυτής της σελίδας υπάρχουν σειρές εικονιδίων, το καθένα αντιπροσωπεύει έναν πίνακα στη βάση δεδομένων. Για να επεξεργαστείτε τη δομή ενός πίνακα, απλώς κάντε κλικ σε ένα από τα εικονίδια.

Ακολουθούν κουμπιά για τη δημιουργία ενός νέου πίνακα σε αυτήν τη βάση δεδομένων, την εκτέλεση SQL στους πίνακες της βάσης δεδομένων και την απόρριψη της βάσης δεδομένων και όλων των πινάκων σε αυτήν. Το τελικό κουμπί πρέπει να χρησιμοποιείται με προσοχή, καθώς δεν είναι αναστρέψιμο!
