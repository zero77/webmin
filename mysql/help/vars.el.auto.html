Μεταβλητές συστήματος MySQL
Αυτή η σελίδα παραθέτει όλες τις εσωτερικές μεταβλητές που έχουν οριστεί αυτήν τη στιγμή στον διακομιστή MySQL. Εκείνες που ταξινομούνται ως μεταβλητές συστήματος μπορούν να επεξεργαστούν επιλέγοντας το πλαίσιο ελέγχου δίπλα τους και πατώντας το κουμπί Επεξεργασία στο κάτω μέρος της σελίδας.
