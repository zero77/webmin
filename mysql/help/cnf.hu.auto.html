MySQL Server konfiguráció
Ez az oldal használható a MySQL adatbázis-kiszolgáló globális konfigurációs beállításainak módosítására. Általában ezeknek egyikét nem kell megváltoztatnia, bár a puffer és a gyorsítótár mérete megnövelhető olyan rendszereknél, amelyekben nagyszámú tábla és adat található.

A módosítások hatályba lépéséhez a MySQL szervert le kell állítani és újra kell indítani a modul főoldalán található gombokkal.
