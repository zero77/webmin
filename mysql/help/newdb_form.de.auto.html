Datenbank erstellen
Mit diesem Formular können Sie eine neue MySQL-Datenbank mit einer optionalen Anfangstabelle erstellen. Sie müssen einen Datenbanknamen eingeben. Wenn Sie eine erste Tabelle erstellen, müssen Sie einen Tabellennamen und die Details mindestens eines Felds eingeben. Wenn Sie fertig sind, klicken Sie auf Erstellen , um die Datenbank sofort zu erstellen .
