Διακομιστής βάσεων δεδομένων MySQL
Η MySQL είναι ένας απλός διακομιστής βάσεων δεδομένων για συστήματα Unix. Υποστηρίζει πολλές βάσεις δεδομένων, ισχυρούς ελέγχους πρόσβασης, πολλούς τύπους δεδομένων και το μεγαλύτερο μέρος της τυπικής σύνταξης SQL. Κάθε βάση δεδομένων που διαχειρίζεται ο διακομιστής μπορεί να περιέχει πολλούς πίνακες και κάθε πίνακας περιέχει πολλές σειρές δεδομένων.

Η κύρια σελίδα εμφανίζει μια λίστα με εικονίδια βάσης δεδομένων στην κορυφή, με έναν σύνδεσμο για την προσθήκη μιας νέας βάσης δεδομένων κάτω από αυτά. Κάθε διακομιστής MySQL θα έχει τουλάχιστον τη βάση δεδομένων mysql , η οποία αποθηκεύει πίνακες διαμόρφωσης και ελέγχου πρόσβασης. Κάτω από τις βάσεις δεδομένων υπάρχουν εικονίδια για τη διαμόρφωση του ελέγχου πρόσβασης και στο κάτω μέρος υπάρχει ένα κουμπί για διακοπή ή εκκίνηση του διακομιστή βάσης δεδομένων.
