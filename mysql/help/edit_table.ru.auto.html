Редактировать таблицу
Таблица, занимающая большую часть этой страницы, показывает структуру выбранной таблицы в вашей базе данных. Каждая строка детализирует одно поле в таблице с указанием имени, типа данных и других параметров.

Внизу слева находится кнопка для добавления нового поля в таблицу. Прежде чем щелкнуть по нему, вы должны выбрать тип поля для добавления - см. Документацию MySQL для получения дополнительной информации о типах полей. Также внизу находятся кнопки для просмотра всех данных в таблице и удаления всей таблицы и всех данных в ней.
