Configuração do servidor MySQL
Esta página pode ser usada para ajustar as configurações globais do servidor de banco de dados MySQL. Normalmente, você não precisará alterar nada disso, embora os tamanhos de buffer e cache possam ser aumentados em sistemas com um grande número de tabelas e dados.

Para que quaisquer alterações entrem em vigor, o servidor MySQL deve ser parado e reiniciado usando os botões na página principal do módulo.
