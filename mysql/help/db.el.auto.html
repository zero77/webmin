Οι ακόλουθες παράμετροι είναι διαθέσιμες:
Βάσεις δεδομένων
Οι βάσεις δεδομένων στις οποίες επιτρέπεται η πρόσβαση. Αυτό μπορεί να είναι ένα SQL regexp, αν θέλετε.

Όνομα χρήστη
Ο χρήστης επέτρεψε την πρόσβαση στην παραπάνω βάση δεδομένων. Αυτός ο χρήστης πρέπει να υπάρχει ήδη στον πίνακα Δικαιώματα χρήστη .

Οικοδεσπότες
Οι κεντρικοί υπολογιστές από τους οποίους ο χρήστης μπορεί να έχει πρόσβαση στη βάση δεδομένων. Εάν έχει επιλεγεί η επιλογή Από δικαιώματα κεντρικού υπολογιστή , τότε η MySQL θα πρέπει να χρησιμοποιεί τα δικαιώματα κεντρικού υπολογιστή για τον έλεγχο της πρόσβασης.

Άδειες
Οι λειτουργίες MySQL που επιτρέπεται στον χρήστη να εκτελεί στη βάση δεδομένων.
