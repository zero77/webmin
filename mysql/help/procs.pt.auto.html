Conexões com o Banco de Dados
Esta página exibe todas as conexões atuais com o servidor de banco de dados MySQL, exceto a usada para realmente solicitar a lista de conexões. Você pode encerrar uma conexão (e a consulta que está sendo executada) selecionando-a e clicando no botão Eliminar conexões selecionadas.

Em alguns casos, uma conexão e sua consulta associada podem continuar sendo executadas mesmo quando o cliente sai. Ainda aparecerá nesta página.
