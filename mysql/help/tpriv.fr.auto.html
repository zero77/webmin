Les paramètres suivants sont disponibles:
Table
Le tableau auquel ces autorisations doivent s'appliquer.

Nom d'utilisateur
L'utilisateur à qui les autorisations sont accordées. Cet utilisateur doit déjà exister dans la table Autorisations utilisateur .

Hôtes
Les hôtes à partir desquels l'utilisateur peut accéder à la table.

Autorisations de table
Les autorisations accordées à l'utilisateur sur toutes les colonnes de la table.

Autorisations de champ
Autorisations maximales pouvant être accordées à cet utilisateur pour n'importe quelle colonne du tableau de la page Autorisations de champ .
