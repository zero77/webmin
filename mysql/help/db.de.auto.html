Folgende Parameter stehen zur Verfügung:
Datenbanken
Die Datenbanken, auf die zugegriffen werden darf. Dies kann bei Bedarf ein SQL-Regexp sein.

Nutzername
Der Benutzer darf auf die oben genannte Datenbank zugreifen. Dieser Benutzer muss bereits in der Benutzerberechtigungen Tabelle.

Gastgeber
Die Hosts, von denen aus der Benutzer auf die Datenbank zugreifen kann. Wenn die Option Von Host-Berechtigungen ausgewählt ist, sollte MySQL die Host-Berechtigungen verwenden , um den Zugriff zu steuern.

Berechtigungen
Die MySQL-Operationen, die der Benutzer an der Datenbank ausführen darf.
