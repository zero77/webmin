Datenbankberechtigungen
Auf dieser Seite können Sie festlegen, welche MySQL-Benutzer von welchen Hosts aus auf welche Datenbanken zugreifen dürfen. Jeder Benutzer muss bereits in den Benutzerberechtigungen Tabelle, obwohl sie hier zusätzliche Berechtigungen erteilt werden können.

Benutzer dürfen sich nicht von anderen Hosts als den in Benutzerberechtigungen angegebenen anmelden. Sie können jedoch unterschiedliche Berechtigungen für unterschiedliche Hosts erteilen, indem Sie mehrere Zeilen für denselben Benutzer erstellen.
