Sauvegarder la base de données
Ce formulaire vous permet de sauvegarder une base de données entière (ou des lignes sélectionnées) dans un fichier. Sélectionnez le fichier dans lequel écrire la sauvegarde et les autres options, puis cliquez sur le bouton Sauvegarder maintenant pour démarrer le processus de sauvegarde. Des instructions SQL seront écrites dans le fichier qui, une fois exécutées, recréeront les tables de la base de données et leur contenu. Ce fichier peut être exécuté à l'aide du programme mysql en ligne de commande ou de la page Execute SQL, afin de restaurer la base de données.
