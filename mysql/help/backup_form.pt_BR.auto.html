Banco de dados de backup
Este formulário permite fazer backup de um banco de dados inteiro (ou linhas selecionadas) em um arquivo. Selecione o arquivo em que o backup foi gravado e as outras opções e clique no botão Fazer backup agora para iniciar o processo de backup. Gravadas no arquivo serão instruções SQL que, quando executadas, recriarão as tabelas do banco de dados e seu conteúdo. Este arquivo pode ser executado usando o programa mysql da linha de comando ou a página Executar SQL, para restaurar o banco de dados.
