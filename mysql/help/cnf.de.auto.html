MySQL Server Konfiguration
Auf dieser Seite können Sie die globalen Konfigurationseinstellungen für den MySQL-Datenbankserver anpassen. In der Regel müssen Sie keine dieser Änderungen vornehmen, obwohl die Puffer- und Cache-Größe auf Systemen mit einer großen Anzahl von Tabellen und Daten erhöht werden kann.

Damit Änderungen wirksam werden, muss der MySQL-Server über die Schaltflächen auf der Hauptseite des Moduls gestoppt und neu gestartet werden.
