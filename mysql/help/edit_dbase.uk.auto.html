Редагувати базу даних
У верхній частині цієї сторінки розташовані рядки піктограм, кожна з яких представляє одну таблицю в базі даних. Щоб відредагувати структуру таблиці, просто натисніть на одну з піктограм.

Нижче розташовані кнопки для створення нової таблиці в цій базі даних, виконання SQL на таблицях бази даних та видалення бази даних та всіх таблиць у ній. Заключну кнопку слід використовувати обережно, оскільки вона не оборотна!
