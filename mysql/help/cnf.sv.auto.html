MySQL Server Configuration
Denna sida kan användas för att justera globala konfigurationsinställningar för MySQL-databasservern. Vanligtvis behöver du inte ändra något av dessa, även om buffert- och cache-storlekarna kan ökas på system med stort antal tabeller och data.

För att alla ändringar ska träda i kraft måste MySQL-servern stoppas och startas om med knapparna på modulens startsida.
