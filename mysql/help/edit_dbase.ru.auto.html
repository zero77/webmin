Редактировать базу данных
В верхней части этой страницы находятся строки значков, каждый из которых представляет одну таблицу в базе данных. Чтобы изменить структуру таблицы, просто нажмите на один из значков.

Ниже приведены кнопки для создания новой таблицы в этой базе данных, выполнения SQL для таблиц базы данных и удаления базы данных и всех таблиц в ней. Последняя кнопка должна использоваться с осторожностью, так как она необратима!
