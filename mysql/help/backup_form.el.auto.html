Εφεδρική βάση δεδομένων
Αυτή η φόρμα σας επιτρέπει να δημιουργήσετε αντίγραφα ασφαλείας ολόκληρης της βάσης δεδομένων (ή επιλεγμένων σειρών) σε ένα αρχείο. Επιλέξτε το αρχείο για να γράψετε το αντίγραφο ασφαλείας και τις άλλες επιλογές και, στη συνέχεια, κάντε κλικ στο κουμπί Δημιουργία αντιγράφων ασφαλείας τώρα για να ξεκινήσετε τη διαδικασία δημιουργίας αντιγράφων ασφαλείας. Θα γράφονται στο αρχείο οι δηλώσεις SQL που όταν εκτελούνται θα δημιουργήσουν εκ νέου τους πίνακες βάσης δεδομένων και τα περιεχόμενά τους. Αυτό το αρχείο μπορεί να εκτελεστεί χρησιμοποιώντας το πρόγραμμα mysql γραμμής εντολών ή τη σελίδα Εκτέλεση SQL, προκειμένου να επαναφέρετε τη βάση δεδομένων.
