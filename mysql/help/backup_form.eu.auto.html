Backup datu-basea
Inprimaki honek datu base oso bat (edo hautatutako errenkadak) fitxategi batean babeskopia egiteko aukera ematen du. Hautatu fitxategia idatzi babeskopiak eta beste aukerak, eta egin klik Backup Now botoian backup egiteko prozesua hasteko. Fitxategian idatzia SQL adierazpenak izango dira, exekutatutakoan datu baseko taulak eta haien edukiak berriro sortuko dituztenak. Fitxategi hau komando lerroko mysql programa edo Execute SQL orria erabiliz egin daiteke, datu-basea berreskuratu ahal izateko.
