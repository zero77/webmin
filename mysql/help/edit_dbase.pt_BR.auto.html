Editar banco de dados
No topo desta página, há linhas de ícones, cada um representando uma tabela no banco de dados. Para editar a estrutura de uma tabela, basta clicar em um dos ícones.

Abaixo estão os botões para criar uma nova tabela nesse banco de dados, executando o SQL nas tabelas do banco de dados e eliminando o banco de dados e todas as tabelas nele. O botão final deve ser usado com cuidado, pois não é reversível!
