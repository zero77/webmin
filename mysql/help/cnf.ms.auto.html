Konfigurasi Pelayan MySQL
Halaman ini dapat digunakan untuk menyesuaikan pengaturan konfigurasi global untuk pelayan pangkalan data MySQL. Biasanya anda tidak perlu mengubah semua ini, walaupun ukuran buffer dan cache dapat ditingkatkan pada sistem dengan sejumlah besar jadual dan data.

Agar perubahan berlaku, pelayan MySQL mesti dihentikan dan dimulakan semula menggunakan butang di halaman utama modul.
