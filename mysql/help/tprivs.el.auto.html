Δικαιώματα πίνακα
Η σελίδα σάς επιτρέπει να παραχωρήσετε επιπλέον πρόσβαση σε πίνακες από χρήστες, πέρα από αυτό που είναι διαθέσιμο στα δικαιώματα χρήστη ή δικαιώματα βάσης δεδομένων . Τα δικαιώματα που παρέχονται εδώ ισχύουν μόνο για πρόσβαση σε έναν πίνακα σε μία βάση δεδομένων από έναν χρήστη, επιτρέποντας έτσι να παραχωρηθεί αρκετά λεπτομερής πρόσβαση.

Για να προσθέσετε νέα δικαιώματα πίνακα, πρέπει πρώτα να επιλέξετε τη βάση δεδομένων στην οποία χρησιμοποιείται ο πίνακας χρησιμοποιώντας τη φόρμα στο κάτω μέρος της σελίδας. Στη συνέχεια, κάντε κλικ στο κουμπί Προσθήκη νέων δικαιωμάτων για να εμφανιστεί μια φόρμα για την επιλογή του πίνακα και άλλες λεπτομέρειες.
