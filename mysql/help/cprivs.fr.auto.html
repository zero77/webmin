Autorisations de champ
Ce champ vous permet d'accorder un accès supplémentaire à des champs spécifiques dans les tables par les utilisateurs. Les autorisations accordées ne peuvent pas être supérieures à celles spécifiées dans les autorisations de table sous autorisations de champ pour la table dans laquelle se trouve le champ.

Pour ajouter de nouvelles autorisations de champ, vous devez d'abord sélectionner la base de données et la table dans lesquelles se trouve le champ à l'aide du formulaire en bas de la page. Cliquez ensuite sur le bouton Ajouter de nouvelles autorisations pour afficher un formulaire de sélection du champ et d'autres détails.
