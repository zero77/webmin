การกำหนดค่าเซิร์ฟเวอร์ MySQL
หน้านี้สามารถใช้เพื่อปรับการตั้งค่าส่วนกลางสำหรับเซิร์ฟเวอร์ฐานข้อมูล MySQL โดยทั่วไปแล้วคุณไม่จำเป็นต้องเปลี่ยนแปลงสิ่งเหล่านี้แม้ว่าขนาดบัฟเฟอร์และแคชสามารถเพิ่มได้ในระบบที่มีตารางและข้อมูลจำนวนมาก

เพื่อให้การเปลี่ยนแปลงมีผลบังคับใช้เซิร์ฟเวอร์ MySQL จะต้องหยุดและเริ่มต้นใหม่โดยใช้ปุ่มบนหน้าหลักของโมดูล
