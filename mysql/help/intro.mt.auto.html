Database tad-Dejta MySQL
MySQL hija server tad-database sempliċi għas-sistemi Unix. Huwa jappoġġja bażijiet tad-dejta multipli, kontrolli ta 'aċċess qawwija, ħafna tipi ta' dejta u ħafna mis-sintassi SQL standard. Kull database ġestita mis-server jista 'jkun fiha tabelli multipli, u kull tabella fiha ringieli multipli ta' dejta.

Il-paġna ewlenija turi lista ta 'ikoni tad-database fil-parti ta' fuq, b'rabta biex iżżid database ġdida taħthom. Kull server MySQL għandu jkollu mill-inqas id-database mysql , li taħżen it-tabelli tal-kontroll tal-konfigurazzjoni u tal-aċċess. Hawn taħt il-bażijiet tad-dejta hemm ikoni għall-konfigurazzjoni tal-kontroll tal-aċċess, u fil-qiegħ hemm buttuna biex twaqqaf jew tibda s-server tad-database.
