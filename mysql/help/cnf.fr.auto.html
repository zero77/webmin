Configuration du serveur MySQL
Cette page peut être utilisée pour ajuster les paramètres de configuration globale du serveur de base de données MySQL. En règle générale, vous n'aurez pas besoin de modifier ces éléments, bien que la taille du tampon et du cache puisse être augmentée sur les systèmes avec un grand nombre de tables et de données.

Pour que toute modification prenne effet, le serveur MySQL doit être arrêté et redémarré à l'aide des boutons de la page principale du module.
