Назва поля
Назва цього поля в таблиці.

Перелічені значення (для enum та типів наборів )
Список можливих значень для цього поля.

Ширина та десяткові знаки (для типів з плаваючою комою та десяткових знаків )
Загальна ширина цього поля та кількість цифр праворуч від десяткової коми.

Ширина типу (для інших типів)
Максимальна кількість символів, дозволених для даних у цьому полі.

Введіть параметри
Для числових полів цей параметр дозволяє вибрати, чи поле не підписане або залишене нулями зліва. Для полів символів ви можете вибрати, чи буде враховуватися випадок при сортуванні чи ні.

Дозволити нулі?
Визначає, чи дозволено значення NULL у цьому полі

Значення за замовчуванням
Значення за замовчуванням для цього поля в нових записах
