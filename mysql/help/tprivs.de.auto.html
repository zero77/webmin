Tabellenberechtigungen
Auf dieser Seite können Sie Benutzern zusätzlichen Zugriff auf Tabellen gewähren, der über die in Benutzerberechtigungen oder Datenbankberechtigungen verfügbaren Optionen hinausgeht. Die hier erteilten Berechtigungen gelten nur für den Zugriff eines Benutzers auf eine Tabelle in einer Datenbank, sodass ein recht detaillierter Zugriff gewährt werden kann.

Um neue Tabellenberechtigungen hinzuzufügen, müssen Sie zuerst die Datenbank auswählen, in der sich die Tabelle befindet. Verwenden Sie dazu das Formular unten auf der Seite. Klicken Sie anschließend auf die Schaltfläche Neue Berechtigungen hinzufügen, um ein Formular zur Auswahl der Tabelle und anderer Details aufzurufen.
