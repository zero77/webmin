Конфігурація сервера MySQL
Ця сторінка може використовуватися для налаштування параметрів глобальної конфігурації для сервера баз даних MySQL. Зазвичай вам не потрібно буде змінювати жодне з них, хоча розміри буфера та кешу можуть бути збільшені в системах з великою кількістю таблиць і даних.

Щоб зміни набули чинності, сервер MySQL потрібно зупинити та запустити заново, використовуючи кнопки на головній сторінці модуля.
