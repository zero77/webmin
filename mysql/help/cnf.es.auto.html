Configuración del servidor MySQL
Esta página se puede utilizar para ajustar la configuración global del servidor de base de datos MySQL. Por lo general, no necesitará cambiar ninguno de estos, aunque los tamaños de búfer y caché se pueden aumentar en sistemas con grandes cantidades de tablas y datos.

Para que los cambios surtan efecto, el servidor MySQL debe detenerse y reiniciarse utilizando los botones de la página principal del módulo.
