Διαμόρφωση διακομιστή MySQL
Αυτή η σελίδα μπορεί να χρησιμοποιηθεί για την προσαρμογή καθολικών ρυθμίσεων διαμόρφωσης για το διακομιστή βάσης δεδομένων MySQL. Συνήθως δεν θα χρειαστεί να αλλάξετε κανένα από αυτά, αν και τα μεγέθη buffer και cache μπορούν να αυξηθούν σε συστήματα με μεγάλο αριθμό πινάκων και δεδομένων.

Για να εφαρμοστούν τυχόν αλλαγές, ο διακομιστής MySQL πρέπει να διακοπεί και να ξεκινήσει εκ νέου χρησιμοποιώντας τα κουμπιά στην κύρια σελίδα της μονάδας.
