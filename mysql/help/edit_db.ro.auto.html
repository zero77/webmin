Editați Permisiunile bazei de date
Această pagină vă permite să editați un set existent de permisiuni de bază de date-utilizator. După editare, faceți clic pe butonul Salvare pentru ca modificările să aibă efect imediat. Puteți de asemenea să faceți clic pe Ștergere pentru a șterge permisiunile din MySQL.
