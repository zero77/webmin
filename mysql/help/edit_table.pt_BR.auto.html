Editar tabela
A tabela que ocupa a maior parte desta página mostra a estrutura da tabela selecionada no seu banco de dados. Cada linha detalha um campo na tabela, com o nome, o tipo de dados e outras opções mostradas.

No canto inferior esquerdo, há um botão para adicionar um novo campo à tabela. Antes de clicar nele, você deve selecionar o tipo de campo a ser adicionado - consulte a documentação do MySQL para obter mais informações sobre os tipos de campo. Também na parte inferior estão os botões para visualizar todos os dados na tabela e soltar a tabela inteira e todos os dados nela.
