Permissões de banco de dados
Esta página permite definir quais usuários do MySQL terão acesso a quais bancos de dados a partir de quais hosts. Todos os usuários já devem existir na tabela Permissões do Usuário , embora possam receber permissões adicionais aqui.

Os usuários não podem efetuar login de nenhum host que não seja o especificado em Permissões do Usuário . No entanto, você pode conceder permissões diferentes para hosts diferentes, criando várias linhas para o mesmo usuário.
