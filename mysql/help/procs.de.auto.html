Datenbankverbindungen
Auf dieser Seite werden alle aktuellen Verbindungen zum MySQL-Datenbankserver angezeigt, mit Ausnahme derjenigen, mit der die Verbindungsliste tatsächlich angefordert wurde. Sie können eine Verbindung (und die Abfrage, die sie ausführt) beenden, indem Sie sie auswählen und auf die Schaltfläche Ausgewählte Verbindungen beenden klicken.

In einigen Fällen werden eine Verbindung und die zugehörige Abfrage möglicherweise auch dann weiter ausgeführt, wenn der Client beendet wurde. Es wird jedoch weiterhin auf dieser Seite angezeigt.
