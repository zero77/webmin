MySQL datu-base zerbitzaria
MySQL datu baseen zerbitzari sinplea da Unix sistemetarako. Datu-base ugari onartzen ditu, sarbide kontrol indartsuak, datu mota asko eta SQL sintaxi estandar gehienak. Zerbitzariak kudeatutako datu-base bakoitzak taula anitz eduki ditzake eta taula bakoitzak datu-ilara bat baino gehiago ditu.

Orrialde nagusiak datu baseko ikonoen zerrenda erakusten du goialdean, datu base berri bat horien azpian esteka bat azaltzeko. MySQL zerbitzari bakoitzak gutxienez mysql datu-basea izango du, konfigurazio eta sarbide kontrol taulak gordetzen dituena. Datu baseen azpian sarbide kontrola konfiguratzeko ikonoak daude, eta behealdean datu-basearen zerbitzaria gelditzeko edo hasteko botoia dago.
