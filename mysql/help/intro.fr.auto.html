Serveur de base de données MySQL
MySQL est un serveur de base de données simple pour les systèmes Unix. Il prend en charge plusieurs bases de données, de puissants contrôles d'accès, de nombreux types de données et la plupart de la syntaxe SQL standard. Chaque base de données gérée par le serveur peut contenir plusieurs tables et chaque table contient plusieurs lignes de données.

La page principale affiche une liste d'icônes de base de données en haut, avec un lien pour ajouter une nouvelle base de données en dessous. Chaque serveur MySQL aura au moins la base de données mysql , qui stocke les tables de configuration et de contrôle d'accès. Sous les bases de données se trouvent des icônes pour configurer le contrôle d'accès, et en bas se trouve un bouton pour arrêter ou démarrer le serveur de base de données.
