Créer une table
Ce formulaire sert à créer une nouvelle table vide. Vous devez saisir un nom de table et un ou plusieurs champs. Pour chaque champ, vous devez saisir un nom, sélectionner un type de données et saisir la largeur du type (le cas échéant pour le type de données). Une fois terminé, cliquez sur le bouton Créer pour que la table soit immédiatement créée.
