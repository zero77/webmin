Конфигурация MySQL Server
Эта страница может использоваться для настройки глобальных параметров конфигурации для сервера базы данных MySQL. Как правило, вам не нужно менять какие-либо из них, хотя размеры буфера и кэша могут быть увеличены в системах с большим количеством таблиц и данных.

Чтобы любые изменения вступили в силу, сервер MySQL должен быть остановлен и перезапущен с помощью кнопок на главной странице модуля.
