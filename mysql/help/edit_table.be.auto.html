Рэдагаванне табліцы
У табліцы, якая займае вялікую частку гэтай старонкі, паказана структура абранай табліцы ў вашай базе дадзеных. Кожны радок падрабязна апісвае адно поле ў табліцы з указаннем імя, тыпу дадзеных і іншых опцый.

Унізе злева знаходзіцца кнопка для дадання новага поля ў табліцу. Перад націскам на яго неабходна выбраць тып поля для дадання - для атрымання дадатковай інфармацыі пра тыпы палёў глядзіце дакументацыю MySQL. Таксама ўнізе размешчаны кнопкі для прагляду ўсіх дадзеных у табліцы і апускання ўсёй табліцы і ўсіх дадзеных у ёй.
