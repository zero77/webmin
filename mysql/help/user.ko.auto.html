각 사용자에 대해 다음 매개 변수를 사용할 수 있습니다.
사용자 이름
이 필드에 대해 특정 사용자 이름을 입력하거나 익명 사용자 옵션을 선택하면 사용자 권한 목록에없는 사용자에게 권한이 적용됩니다.

암호
로그인 할 때이 사용자에게 필요한 비밀번호입니다. None 이면 비밀번호가 필요하지 않습니다.

호스트
이 사용자가 로그인 할 수있는 호스트의 IP 주소 또는 호스트 이름입니다. 이 필드에는 192.168.1. % 또는 %.foo.com 과 같은 여러 호스트와 일치하는 SQL 정규식도 포함될 수 있습니다.

권한
이 사용자는 모든 데이터베이스에서 수행 할 수있는 MySQL 작업입니다.
