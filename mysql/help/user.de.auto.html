Für jeden Benutzer stehen folgende Parameter zur Verfügung:
Nutzername
Für dieses Feld können Sie entweder einen bestimmten Benutzernamen eingeben oder die Option Anonymer Benutzer auswählen. In diesem Fall gelten die Berechtigungen für alle Benutzer, die nicht in der Liste der Benutzerberechtigungen aufgeführt sind.

Passwort
Das Kennwort, das dieser Benutzer beim Anmelden benötigt. Wenn Keine , wird kein Kennwort benötigt.

Gastgeber
Die IP-Adresse oder der Hostname des Hosts, von dem aus sich dieser Benutzer anmelden darf. Dieses Feld kann auch einen regulären SQL-Ausdruck enthalten, der mehreren Hosts entspricht, z. B. 192.168.1.% Oder %.foo.com .

Berechtigungen
Die MySQL-Operationen, die dieser Benutzer für jede Datenbank ausführen darf.
