Rezervna baza podataka
Ovaj obrazac omogućuje sigurnosnu kopiju cijele baze podataka (ili odabranih redaka) u datoteci. Odaberite datoteku u koju upišite sigurnosnu kopiju i ostale mogućnosti, a zatim kliknite gumb Sigurnosna kopija sada da biste pokrenuli postupak izrade sigurnosne kopije. U datoteku će biti zapisani SQL izrazi koji će se nakon izvršenja ponovno stvoriti tablice baze podataka i njihov sadržaj. Ova se datoteka može izvršiti pomoću programa mysql naredbenog retka ili stranice Execute SQL kako bi se vratila baza podataka.
