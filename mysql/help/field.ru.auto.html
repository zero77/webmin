Имя поля
Название этого поля в таблице.

Перечисляемые значения (для перечислимых и множественных типов)
Список возможных значений для этого поля.

Ширина и десятичные дроби (для чисел с плавающей запятой и десятичных чисел)
Общая ширина этого поля и количество цифр справа от десятичной точки.

Ширина типа (для других типов)
Максимальное количество символов, допустимое для данных в этом поле.

Варианты типа
Для числовых полей этот параметр позволяет выбрать, будет ли поле без знака или дополнено слева нулями. Для символьных полей вы можете выбрать, будет ли учитываться регистр при сортировке или нет.

Разрешить нули?
Определяет, допустимы ли значения NULL в этом поле

Значение по умолчанию
Значение по умолчанию для этого поля в новых записях
