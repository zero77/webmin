MySQL Database Server
MySQL je preprost strežnik baz podatkov za sisteme Unix. Podpira več baz podatkov, močan nadzor dostopa, številne vrste podatkov in večino standardne sintakse SQL. Vsaka baza podatkov, ki jo upravlja strežnik, lahko vsebuje več tabel, vsaka tabela pa vsebuje več vrstic podatkov.

Na glavni strani se prikaže seznam ikon baz podatkov na vrhu, pod njimi pa povezava za dodajanje nove baze podatkov. Vsak strežnik MySQL bo imel vsaj mysql bazo podatkov, v kateri so shranjene tabele konfiguracije in nadzora dostopa. Spodaj v zbirkah podatkov so ikone za konfiguracijo nadzora dostopa, spodaj pa gumb za zaustavitev ali zagon strežnika baz podatkov.
