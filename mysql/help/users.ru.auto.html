Пользовательские разрешения
Эта страница позволяет вам создавать и редактировать пользователей, которые будут иметь доступ к вашим базам данных MySQL. Для каждого пользователя вы можете указать имя пользователя, пароль, допустимые хосты и набор разрешений, которые определяют, что пользователь может делать.

Поскольку любой пользователь в списке, у которого есть разрешения, будет иметь эти разрешения для всех баз данных, обычно рекомендуется не предоставлять пользователям на этой странице никаких разрешений. Вместо этого используйте страницу « Разрешения базы данных», чтобы предоставить пользователям доступ к базе данных.
