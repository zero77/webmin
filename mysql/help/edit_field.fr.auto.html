Modifier le champ
Sur cette page, vous pouvez modifier un champ de table existant. Bien que le type de données de base du champ ne puisse pas être modifié, vous êtes autorisé à modifier d'autres paramètres de champ tels que: Une fois terminé, cliquez sur le bouton Enregistrer pour mettre à jour la structure du tableau ou sur Supprimer pour supprimer le champ du tableau.
