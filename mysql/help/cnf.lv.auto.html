MySQL servera konfigurācija
Šo lapu var izmantot, lai pielāgotu MySQL datu bāzes servera globālos konfigurācijas iestatījumus. Parasti jums nekas no tiem nav jāmaina, lai gan buferšķīduma un kešatmiņas lielumus var palielināt sistēmās ar lielu skaitu tabulu un datu.

Lai izmaiņas stātos spēkā, MySQL serveris jāpārtrauc un jāatsāk, izmantojot pogas moduļa galvenajā lapā.
