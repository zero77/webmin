Os seguintes parâmetros estão disponíveis:
Campo
O campo ao qual essas permissões se aplicam.

Nome do usuário
O usuário a quem as permissões são concedidas. Este usuário já deve existir na tabela Permissões do Usuário .

Anfitriões
Os hosts dos quais o usuário pode acessar a tabela.

Permissões
As permissões concedidas ao usuário na coluna escolhida da tabela.
