Για κάθε χρήστη, είναι διαθέσιμες οι ακόλουθες παράμετροι:
Όνομα χρήστη
Για αυτό το πεδίο, μπορείτε είτε να εισαγάγετε ένα συγκεκριμένο όνομα χρήστη ή να επιλέξετε την επιλογή Ανώνυμος χρήστης , οπότε τα δικαιώματα θα ισχύουν για οποιονδήποτε χρήστη δεν είναι στη λίστα δικαιωμάτων χρήστη .

Κωδικός πρόσβασης
Ο κωδικός πρόσβασης που απαιτείται από αυτόν τον χρήστη κατά τη σύνδεση. Εάν δεν υπάρχει , τότε δεν απαιτείται κωδικός πρόσβασης.

Οικοδεσπότες
Η διεύθυνση IP ή το όνομα κεντρικού υπολογιστή του κεντρικού υπολογιστή από τον οποίο επιτρέπεται η σύνδεση. Αυτό το πεδίο μπορεί επίσης να περιέχει μια κανονική έκφραση SQL που ταιριάζει με πολλούς κεντρικούς υπολογιστές, όπως 192.168.1.% Ή %.foo.com .

Άδειες
Οι λειτουργίες MySQL που επιτρέπεται να εκτελεί αυτός ο χρήστης σε οποιαδήποτε βάση δεδομένων.
