Резервна база даних
Ця форма дозволяє створити резервну копію всієї бази даних (або вибраних рядків) у файл. Виберіть у файл записування резервної копії та інші параметри, а потім натисніть кнопку " Резервне копіювання зараз", щоб розпочати процес резервного копіювання. У файл буде записано SQL-висловлювання, які при виконанні повторно створять таблиці баз даних та їх вміст. Цей файл можна виконати за допомогою програми mysql командного рядка або сторінки Execute SQL для відновлення бази даних.
