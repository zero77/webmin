Назва поля
Назва гэтага поля ў табліцы.

Пералічаныя значэнні (для тыпаў enum і set )
Спіс магчымых значэнняў для гэтага поля.

Шырыня і дзесятковыя знакі (для паплаўкі і дзесятковых тыпаў)
Агульная шырыня гэтага поля і колькасць лічбаў справа ад дзесятковай коскі.

Шырыня тыпу (для іншых тыпаў)
Максімальная колькасць сімвалаў, дазволеных для дадзеных у гэтым полі.

Увядзіце параметры
Для лічбавых палёў гэты параметр дазваляе выбіраць, ці будзе поле без подпісу або налева з нулямі. Для знакавых палёў вы можаце выбраць, ці будзе ўлічвацца выпадак пры сартаванні ці не.

Дазволіць нулі?
Вызначае, ці дазволены значэнні NULL у гэтым полі

Значэнне па змаўчанні
Значэнне па змаўчанні для гэтага поля ў новых запісах
