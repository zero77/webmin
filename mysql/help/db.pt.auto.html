Os seguintes parâmetros estão disponíveis:
Bases de dados
Os bancos de dados aos quais o acesso é permitido. Pode ser um regexp SQL, se desejado.

Nome do usuário
O usuário permitiu acessar o banco de dados acima. Este usuário já deve existir na tabela Permissões do Usuário .

Anfitriões
Os hosts dos quais o usuário pode acessar o banco de dados. Se a opção De permissões de host for escolhida, o MySQL deverá usar as Permissões de Host para controlar o acesso.

Permissões
As operações do MySQL que o usuário tem permissão para executar no banco de dados.
