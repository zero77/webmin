สำหรับผู้ใช้แต่ละคนมีพารามิเตอร์ต่อไปนี้:
ชื่อผู้ใช้
สำหรับฟิลด์นี้คุณสามารถป้อนชื่อผู้ใช้เฉพาะหรือเลือกตัวเลือก ผู้ใช้แบบไม่ระบุชื่อ ซึ่งในกรณีนี้การอนุญาตจะมีผลกับผู้ใช้ที่ไม่ได้อยู่ในรายการการ อนุญาตผู้ใช้

รหัสผ่าน
รหัสผ่านที่ผู้ใช้รายนี้ต้องการเมื่อเข้าสู่ระบบหาก ไม่มี ดังนั้นไม่จำเป็นต้องใช้รหัสผ่าน

โฮสต์
ที่อยู่ IP หรือชื่อโฮสต์ของโฮสต์ที่ผู้ใช้รายนี้ได้รับอนุญาตให้เข้าสู่ระบบ ฟิลด์นี้ยังสามารถมีนิพจน์ปกติของ SQL ที่ตรงกับโฮสต์หลายตัวเช่น 192.168.1.% หรือ %.foo.com

สิทธิ์
การดำเนินงานของ MySQL ผู้ใช้นี้ได้รับอนุญาตให้ดำเนินการกับฐานข้อมูล ใด ๆ
