Erabiltzaileentzako baimenak
Orrialde honek zure MySQL datu-baseetara sarbidea izango duten erabiltzaileak sortu eta editatu ahal izango dituzu. Erabiltzaile bakoitzeko, erabiltzaile izena, pasahitza, ostalari baimenduak eta erabiltzaileak zer egin dezakeen zehazten duen baimen multzo bat zehaztu dezakezu.

Baimenak dituzten zerrendatutako erabiltzaileek baimen horiek datu base guztietan izango dituztenez, ideia ona da orrialde honetako erabiltzaileei baimenik ez ematea. Horren ordez, erabili datu-basearen baimenak orria datu base bakoitzeko erabiltzaileei sarbidea emateko.
