Créer une base de données
Ce formulaire vous permet de créer une nouvelle base de données MySQL, avec une table initiale optionnelle. Vous devez entrer un nom de base de données et si vous créez une table initiale, vous devez entrer un nom de table et les détails d'au moins un champ. Une fois terminé, cliquez sur Créer pour créer immédiatement la base de données.
