Δημιουργία βάσης δεδομένων
Αυτή η φόρμα σας επιτρέπει να δημιουργήσετε μια νέα βάση δεδομένων MySQL, με έναν προαιρετικό αρχικό πίνακα. Πρέπει να εισαγάγετε ένα όνομα βάσης δεδομένων και εάν δημιουργείτε έναν αρχικό πίνακα πρέπει να εισαγάγετε ένα όνομα πίνακα και τις λεπτομέρειες τουλάχιστον ενός πεδίου. Όταν τελειώσετε, κάντε κλικ στο Δημιουργία για να δημιουργήσετε αμέσως τη βάση δεδομένων.
