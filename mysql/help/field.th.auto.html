ชื่อฟิลด์
ชื่อของฟิลด์นี้ในตาราง

ค่าที่แจกแจง (สำหรับชนิด enum และ ชุด )
รายการค่าที่เป็นไปได้สำหรับฟิลด์นี้

ความกว้างและความทศนิยม (สำหรับลอยและทศนิยมชนิด)
ความกว้างทั้งหมดของฟิลด์นี้และจำนวนหลักทางด้านขวาของจุดทศนิยม

ความกว้างประเภท (สำหรับประเภทอื่น ๆ )
จำนวนอักขระสูงสุดที่อนุญาตสำหรับข้อมูลในฟิลด์นี้

ตัวเลือกประเภท
สำหรับฟิลด์ตัวเลขพารามิเตอร์นี้อนุญาตให้คุณเลือกว่าจะไม่ระบุเขตข้อมูลหรือมีเบาะซ้ายด้วยค่าศูนย์ สำหรับฟิลด์ตัวละครคุณสามารถเลือกได้ว่าจะคำนึงถึงกรณีและปัญหาเมื่อเรียงลำดับหรือไม่

อนุญาต null หรือไม่
พิจารณาว่าอนุญาตค่า NULL ในฟิลด์นี้หรือไม่

ค่าเริ่มต้น
ค่าเริ่มต้นสำหรับฟิลด์นี้ในบันทึกใหม่
