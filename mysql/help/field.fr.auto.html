Nom de domaine
Le nom de ce champ dans la table.

Valeurs énumérées (pour les types enum et set )
Une liste de valeurs possibles pour ce champ.

Largeur et décimales (pour les types flottants et décimaux )
La largeur totale de ce champ et le nombre de chiffres à droite du séparateur décimal.

Largeur du texte (pour les autres types)
Nombre maximal de caractères autorisé pour les données dans ce champ.

Options de type
Pour les champs numériques, ce paramètre vous permet de choisir si le champ est non signé ou rempli à gauche avec des zéros. Pour les champs de caractères, vous pouvez choisir si la casse est prise en compte lors du tri ou non.

Autoriser les valeurs nulles?
Détermine si les valeurs NULL sont autorisées dans ce champ

Valeur par défaut
La valeur par défaut de ce champ dans les nouveaux enregistrements
