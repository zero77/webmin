Serverul de baze de date MySQL
MySQL este un simplu server de baze de date pentru sistemele Unix. Suporta mai multe baze de date, controale de acces puternice, multe tipuri de date și majoritatea sintaxei SQL standard. Fiecare bază de date administrată de server poate conține mai multe tabele și fiecare tabel conține mai multe rânduri de date.

Pagina principală afișează o listă de pictograme ale bazei de date în partea de sus, cu un link pentru adăugarea unei noi baze de date sub ele. Fiecare server MySQL va avea cel puțin baza de date mysql , care stochează tabelele de configurare și control de acces. Mai jos de baze de date sunt pictograme pentru configurarea controlului de acces, iar în partea de jos este un buton pentru oprirea sau pornirea serverului de baze de date.
