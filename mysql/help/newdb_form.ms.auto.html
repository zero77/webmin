Buat Pangkalan Data
Borang ini membolehkan anda membuat pangkalan data MySQL baru, dengan jadual awal pilihan. Anda mesti memasukkan nama pangkalan data, dan jika membuat jadual awal, anda mesti memasukkan nama jadual dan perincian sekurang-kurangnya satu bidang. Setelah selesai, klik Buat untuk membuat pangkalan data segera dibuat.
