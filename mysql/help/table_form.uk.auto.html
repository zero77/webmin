Створити таблицю
Ця форма призначена для створення нової порожньої таблиці. Потрібно ввести ім'я таблиці та одне або кілька полів. Для кожного поля потрібно ввести ім’я, вибрати тип даних та ввести ширину типу (якщо це відповідає типу даних). Після закінчення натисніть кнопку Створити, щоб таблиця була негайно створена.
