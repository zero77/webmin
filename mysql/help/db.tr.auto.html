Aşağıdaki parametreler kullanılabilir:
Veritabanları
Erişime izin verilen veritabanları. İstenirse bu bir SQL normal ifadesi olabilir.

Kullanıcı adı
Kullanıcının yukarıdaki veritabanına erişmesine izin verildi. Bu kullanıcının zaten Kullanıcı İzinleri tablosunda olması gerekir.

Sunucular
Kullanıcının veritabanına erişebileceği ana bilgisayarlar. Ana bilgisayar izinlerinden Kimden seçeneği seçilirse, MySQL erişimi kontrol etmek için Ana Bilgisayar İzinlerini kullanmalıdır.

İzinler
Kullanıcının veritabanında gerçekleştirmesine izin verilen MySQL işlemleri.
