Datenbankberechtigungen bearbeiten
Auf dieser Seite können Sie einen vorhandenen Satz von Datenbankbenutzerberechtigungen bearbeiten. Klicken Sie nach der Bearbeitung auf die Schaltfläche Speichern , damit die Änderungen sofort wirksam werden. Sie können auch auf Löschen klicken, um die Berechtigungen aus MySQL zu löschen.
