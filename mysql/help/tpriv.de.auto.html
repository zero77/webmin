Folgende Parameter stehen zur Verfügung:
Tabelle
Die Tabelle, für die diese Berechtigungen gelten sollten.

Nutzername
Der Benutzer, dem die Berechtigungen erteilt wurden. Dieser Benutzer muss bereits in der Benutzerberechtigungen Tabelle.

Gastgeber
Die Hosts, von denen aus der Benutzer auf die Tabelle zugreifen kann.

Tabellenberechtigungen
Die Berechtigungen, die dem Benutzer für alle Spalten der Tabelle erteilt wurden.

Feldberechtigungen
Die maximalen Berechtigungen, die diesem Benutzer für eine beliebige Spalte der Tabelle auf der Seite Feldberechtigungen erteilt werden können.
