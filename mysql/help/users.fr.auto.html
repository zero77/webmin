Autorisations utilisateur
Cette page vous permet de créer et de modifier des utilisateurs qui auront accès à vos bases de données MySQL. Pour chaque utilisateur, vous pouvez spécifier un nom d'utilisateur, un mot de passe, des hôtes autorisés et un ensemble d'autorisations qui déterminent ce que l'utilisateur peut faire.

Étant donné que tout utilisateur répertorié disposant d'autorisations aura ces autorisations sur toutes les bases de données, il est généralement judicieux de ne pas accorder aux utilisateurs de cette page des autorisations. Utilisez plutôt la page Permissions de la base de données pour accorder l'accès aux utilisateurs par base de données.
