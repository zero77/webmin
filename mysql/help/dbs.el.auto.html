Δικαιώματα βάσης δεδομένων
Αυτή η σελίδα σάς επιτρέπει να ορίσετε ποιοι χρήστες MySQL θα έχουν πρόσβαση σε ποιες βάσεις δεδομένων από ποιον φιλοξενεί. Κάθε χρήστης πρέπει να υπάρχει ήδη στον πίνακα Δικαιώματα χρήστη , αν και μπορεί να του χορηγηθούν πρόσθετα δικαιώματα εδώ.

Οι χρήστες δεν επιτρέπεται να συνδέονται από άλλους κεντρικούς υπολογιστές εκτός από αυτούς που καθορίζονται στα δικαιώματα χρήστη . Ωστόσο, μπορείτε να εκχωρήσετε διαφορετικά δικαιώματα για διαφορετικούς κεντρικούς υπολογιστές, δημιουργώντας πολλές σειρές για τον ίδιο χρήστη.
