Δικαιώματα χρήστη
Αυτή η σελίδα σάς επιτρέπει να δημιουργείτε και να επεξεργάζεστε χρήστες που θα έχουν πρόσβαση στις βάσεις δεδομένων MySQL. Για κάθε χρήστη, μπορείτε να καθορίσετε ένα όνομα χρήστη, έναν κωδικό πρόσβασης, επιτρεπόμενους κεντρικούς υπολογιστές και ένα σύνολο δικαιωμάτων που καθορίζουν τι μπορεί να κάνει ο χρήστης.

Επειδή οποιοσδήποτε χρήστης που έχει δικαιώματα και έχει δικαιώματα θα έχει αυτά τα δικαιώματα σε όλες τις βάσεις δεδομένων, είναι συνήθως καλή ιδέα να μην παραχωρούνται δικαιώματα σε χρήστες αυτής της σελίδας. Αντ 'αυτού, χρησιμοποιήστε τη σελίδα Δικαιώματα βάσης δεδομένων για να παραχωρήσετε πρόσβαση στους χρήστες ανά βάση δεδομένων.
