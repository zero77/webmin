Baza de date de rezervă
Acest formular vă permite să faceți backup pentru o întreagă bază de date (sau rânduri selectate) într-un fișier. Selectați fișierul în care scrieți backup și celelalte opțiuni, apoi faceți clic pe butonul Backup Now pentru a începe procesul de backup. Scrise în fișier vor fi instrucțiuni SQL care, atunci când sunt executate, vor re-crea tabelele bazei de date și conținutul acestora. Acest fișier poate fi executat folosind programul mysql din linia de comandă sau pagina Execute SQL, pentru a restabili baza de date.
