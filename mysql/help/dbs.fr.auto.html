Autorisations de base de données
Cette page vous permet de définir quels utilisateurs MySQL auront accès à quelles bases de données à partir de quels hôtes. Chaque utilisateur doit déjà exister dans le tableau des autorisations utilisateur , bien qu'il puisse leur être accordé des autorisations supplémentaires ici.

Les utilisateurs ne peuvent pas se connecter à partir d'autres hôtes que ceux spécifiés dans les autorisations utilisateur . Cependant, vous pouvez accorder différentes autorisations pour différents hôtes en créant plusieurs lignes pour le même utilisateur.
