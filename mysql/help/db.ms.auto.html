Parameter berikut tersedia:
Pangkalan Data
Pangkalan data ke mana akses dibenarkan. Ini boleh menjadi regexp SQL jika dikehendaki.

Nama pengguna
Pengguna dibenarkan mengakses pangkalan data di atas. Pengguna ini mesti ada dalam jadual Kebenaran Pengguna .

Tuan rumah
Host yang pengguna dapat mengakses pangkalan data dari. Sekiranya pilihan Dari kebenaran host dipilih, maka MySQL harus menggunakan Host Permissions untuk mengawal akses.

Kebenaran
Operasi MySQL yang dibenarkan pengguna melakukan pangkalan data.
