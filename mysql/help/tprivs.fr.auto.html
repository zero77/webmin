Autorisations de table
La page vous permet d'accorder un accès supplémentaire aux tables par les utilisateurs, au-delà de ce qui est disponible dans les autorisations utilisateur ou les autorisations de base de données . Les autorisations accordées ici ne s'appliquent qu'à l'accès à une table dans une base de données par un seul utilisateur, permettant ainsi d'accorder un accès assez précis.

Pour ajouter de nouvelles autorisations de table, vous devez d'abord sélectionner la base de données dans laquelle se trouve la table à l'aide du formulaire en bas de la page. Cliquez ensuite sur le bouton Ajouter de nouvelles autorisations pour afficher un formulaire de sélection du tableau et d'autres détails.
