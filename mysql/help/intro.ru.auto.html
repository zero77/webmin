MySQL Database Server
MySQL - это простой сервер баз данных для систем Unix. Он поддерживает несколько баз данных, мощные средства управления доступом, множество типов данных и большую часть стандартного синтаксиса SQL. Каждая база данных, управляемая сервером, может содержать несколько таблиц, и каждая таблица содержит несколько строк данных.

Главная страница отображает список значков базы данных вверху, со ссылкой для добавления новой базы данных под ними. Каждый сервер MySQL будет иметь как минимум базу данных mysql , в которой хранятся таблицы конфигурации и контроля доступа. Ниже баз данных находятся значки для настройки контроля доступа, а внизу - кнопка для остановки или запуска сервера баз данных.
