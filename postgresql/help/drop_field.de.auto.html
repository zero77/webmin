Feld löschen
Auf dieser Seite können Sie ein vorhandenes Tabellenfeld löschen. Klicken Sie auf das Optionsfeld nach dem Feld, um anzugeben, welches Feld gelöscht werden soll. Bestätigen Sie außerdem im Kontrollkästchen, dass Sie das Löschen durchführen möchten und dass Sie verstehen, dass alle Daten in dieser Spalte verloren gehen. Beachten Sie, dass alle Definitionen (z. B. eine Definition eines Standardwerts) in der Tabelle außer den Spaltennamen, -typen und -größen verloren gehen. Klicken Sie auf die Schaltfläche, um das Feld zu löschen.
