Données de table
Cette page vous permet de visualiser les données dans un tableau, de modifier ces données et d'ajouter de nouveaux enregistrements.
Modification des enregistrements
Cochez les cases à gauche des lignes que vous souhaitez modifier et cliquez sur le bouton Modifier les lignes sélectionnées . Saisissez ensuite les nouvelles valeurs dans les champs et cliquez sur Enregistrer .

Suppression d'enregistrements
Cochez les cases à gauche des lignes que vous souhaitez supprimer et cliquez sur le bouton Supprimer les lignes sélectionnées .

Ajout d'un enregistrement
Cliquez sur le bouton Ajouter une ligne et entrez les valeurs du nouvel enregistrement dans les champs en bas du tableau, puis cliquez sur Enregistrer .
