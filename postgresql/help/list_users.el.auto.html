Χρήστες PostgreSQL
Αυτή η σελίδα σας επιτρέπει να δημιουργήσετε και να επεξεργαστείτε χρήστες που θα έχουν πρόσβαση στις βάσεις δεδομένων PostgreSQL. Για κάθε χρήστη μπορείτε να καθορίσετε ένα όνομα χρήστη, έναν προαιρετικό κωδικό πρόσβασης, μια προαιρετική ημερομηνία λήξης και να επιλέξετε εάν θα επιτρέπεται στον χρήστη να δημιουργεί άλλους χρήστες ή βάσεις δεδομένων.
