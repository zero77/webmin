Сервер баз данных PostgreSQL
PostgreSQL - это простой сервер баз данных для систем Unix. Он поддерживает несколько баз данных, мощные средства управления доступом, множество типов данных и большую часть стандартного синтаксиса SQL. Каждая база данных, управляемая сервером, может содержать несколько таблиц, и каждая таблица содержит несколько строк данных.

Главная страница отображает список значков базы данных вверху, со ссылкой для добавления новой базы данных под ними. Каждый сервер PostgreSQL будет иметь хотя бы одну базу данных шаблонов, обычно называемую template1 . Ниже баз данных находятся значки для настройки контроля доступа и разрешений, а внизу - кнопка для остановки или запуска сервера баз данных.
