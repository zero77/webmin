Створити таблицю
Ця форма призначена для створення нової порожньої таблиці. Потрібно ввести ім'я таблиці та одне або кілька полів. Для кожного поля потрібно ввести ім’я, вибрати тип даних та ввести ширину типу (якщо це відповідає типу даних). Ви також можете вибрати, чи поле є масивом з багатозначними значеннями, якщо воно дозволяє нульове значення, якщо це первинний ключ для індексації, і якщо значення в ньому повинні бути унікальними. Після закінчення натисніть кнопку Створити, щоб таблиця була негайно створена.
