Redigera tabell
Tabellen som tar upp det mesta av den här sidan visar strukturen för den valda tabellen i din databas. Varje rad anger ett fält i tabellen, med namn, datatyp och andra alternativ som visas.

Längst ner till vänster finns en knapp för att lägga till ett nytt fält i tabellen. Innan du klickar på det måste du välja vilken fälttyp du vill lägga till - se PostgreSQL-dokumentationen för mer information om fälttyper. Längst ner finns knappar för att visa alla data i tabellen och släppa hela tabellen och all data i den.
