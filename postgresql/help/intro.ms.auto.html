Pelayan Pangkalan Data PostgreSQL
PostgreSQL adalah pelayan pangkalan data ringkas untuk sistem Unix. Ia menyokong pelbagai pangkalan data, kawalan akses yang kuat, banyak jenis data dan sebahagian besar sintaks SQL standard. Setiap pangkalan data yang dikendalikan oleh pelayan dapat berisi beberapa tabel, dan setiap tabel berisi beberapa baris data.

Halaman utama memaparkan senarai ikon pangkalan data di bahagian atas, dengan pautan untuk menambahkan pangkalan data baru di bawahnya. Setiap pelayan PostgreSQL akan mempunyai sekurang-kurangnya satu pangkalan data templat, biasanya disebut templat1 . Di bawah pangkalan data adalah ikon untuk mengkonfigurasi kawalan akses dan izin, dan di bahagian bawah adalah butang untuk menghentikan atau memulakan pelayan pangkalan data.
