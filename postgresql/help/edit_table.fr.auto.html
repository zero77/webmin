Modifier le tableau
Le tableau qui occupe la majeure partie de cette page montre la structure du tableau sélectionné dans votre base de données. Chaque ligne détaille un champ du tableau, avec le nom, le type de données et d'autres options affichés.

En bas à gauche se trouve un bouton pour ajouter un nouveau champ à la table. Avant de cliquer dessus, vous devez sélectionner le type de champ à ajouter - voir la documentation PostgreSQL pour plus d'informations sur les types de champs. Vous trouverez également en bas des boutons pour afficher toutes les données du tableau et supprimer l'ensemble du tableau et toutes les données qu'il contient.
