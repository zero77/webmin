Ajuda do formulário de backup do Webmin PostgreSQL

O ID do usuário do sistema PostgreSQL deve ter permissão de gravação em qualquer diretório usado como destino para qualquer backup.

Independentemente do formato do arquivo de backup selecionado, é feito o backup de cada banco de dados em um único arquivo, com o seguinte formato: <name>. <extension>, em que <extension> será 'post', 'tar' ou 'sql'.

Os comandos especificados para execução antes e depois do backup são acionados antes e depois de cada banco de dados e NÃO apenas uma vez antes e uma vez após todo o conjunto.
