Privilèges accordés
Cette page vous permet de visualiser et de modifier les autorisations accordées sur les tables, vues, séquences et index aux utilisateurs et aux groupes. Le tableau montre tous les objets de toutes les bases de données sur lesquels des privilèges peuvent être accordés et à qui ces privilèges sont actuellement accordés (le cas échéant). Pour modifier les privilèges, cliquez sur le nom de l'objet et utilisez le formulaire pour sélectionner un utilisateur ou un groupe et les capacités qui lui seront accordées.
