Добавить поле
На этой странице вы можете добавить новое поле в таблицу. Редактируемые параметры для нового поля:
Имя поля
Название этого поля в таблице.

Ширина типа (для полей varchar и char)
Максимальное количество символов, допустимое для данных в этом поле.

Поле массива?
Если для этого параметра установлено значение «Да», поле сможет хранить несколько значений одного типа.

Когда закончите, нажмите кнопку Сохранить , чтобы обновить структуру таблицы новым полем.
