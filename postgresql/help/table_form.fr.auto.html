Créer une table
Ce formulaire sert à créer une nouvelle table vide. Vous devez saisir un nom de table et un ou plusieurs champs. Pour chaque champ, vous devez saisir un nom, sélectionner un type de données et saisir la largeur du type (le cas échéant pour le type de données). Vous pouvez également choisir si le champ est un tableau à plusieurs valeurs, s'il autorise null, s'il s'agit de la clé primaire d'indexation et si les valeurs qu'il contient doivent être uniques. Une fois terminé, cliquez sur le bouton Créer pour que la table soit immédiatement créée.
