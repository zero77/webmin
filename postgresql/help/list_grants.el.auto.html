Εκχωρημένα προνόμια
Αυτή η σελίδα σάς επιτρέπει να προβάλετε και να επεξεργαστείτε τα δικαιώματα που δίνονται σε πίνακες, προβολές, ακολουθίες και ευρετήρια σε χρήστες και ομάδες. Ο πίνακας δείχνει όλα τα αντικείμενα σε όλες τις βάσεις δεδομένων για τα οποία μπορούν να παραχωρηθούν δικαιώματα και σε ποια αυτά τα δικαιώματα χορηγούνται επί του παρόντος (εάν υπάρχουν). Για να αλλάξετε τα δικαιώματα, κάντε κλικ στο όνομα του αντικειμένου και χρησιμοποιήστε τη φόρμα για να επιλέξετε έναν χρήστη ή ομάδα και τις δυνατότητες που θα του δοθούν.
