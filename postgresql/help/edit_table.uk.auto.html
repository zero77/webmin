Редагувати таблицю
Таблиця, яка займає більшу частину цієї сторінки, показує структуру вибраної таблиці у вашій базі даних. Кожен рядок детально описує одне поле в таблиці із зазначенням імені, типу даних та інших параметрів.

Унизу зліва знаходиться кнопка для додавання нового поля до таблиці. Перш ніж натиснути його, потрібно вибрати тип поля для додавання - для отримання додаткової інформації про типи полів перегляньте документацію PostgreSQL. Також внизу розташовані кнопки для перегляду всіх даних у таблиці та опускання всієї таблиці та всіх даних у ній.
