Ajouter le champ
Sur cette page, vous pouvez ajouter un nouveau champ à la table. Les paramètres modifiables pour le nouveau champ sont:
Nom de domaine
Le nom de ce champ dans la table.

Largeur du texte (pour les champs varchar et char)
Nombre maximal de caractères autorisé pour les données dans ce champ.

Champ de tableau?
Si cette option est définie sur Oui, le champ pourra stocker plusieurs valeurs du même type.

Une fois terminé, cliquez sur le bouton Enregistrer pour mettre à jour la structure de la table avec le nouveau champ.
