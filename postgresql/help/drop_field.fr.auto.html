Drop Field
Sur cette page, vous pouvez supprimer un champ de table existant. Cliquez sur le bouton radio après le champ pour indiquer le champ à supprimer. Vérifiez également dans la case à cocher que vous souhaitez effectuer la suppression et que vous comprenez que toutes les données de cette colonne seront perdues. Notez que toutes les définitions (telles qu'une définition d'une valeur par défaut) dans le tableau sont perdues à l'exception des noms, types et tailles de colonne. Cliquez sur le bouton pour supprimer le champ.
