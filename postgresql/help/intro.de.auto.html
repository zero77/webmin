PostgreSQL-Datenbankserver
PostgreSQL ist ein einfacher Datenbankserver für Unix-Systeme. Es unterstützt mehrere Datenbanken, leistungsstarke Zugriffssteuerungen, viele Datentypen und den größten Teil der Standard-SQL-Syntax. Jede vom Server verwaltete Datenbank kann mehrere Tabellen enthalten, und jede Tabelle enthält mehrere Datenzeilen.

Auf der Hauptseite wird oben eine Liste der Datenbanksymbole mit einem Link zum Hinzufügen einer neuen Datenbank angezeigt. Jeder PostgreSQL-Server verfügt über mindestens eine Vorlagendatenbank, die normalerweise als template1 bezeichnet wird . Unterhalb der Datenbanken befinden sich Symbole zum Konfigurieren der Zugriffssteuerung und der Berechtigungen. Unten befindet sich eine Schaltfläche zum Stoppen oder Starten des Datenbankservers.
