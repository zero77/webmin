Guida del modulo di backup Webmin PostgreSQL

L'ID utente del sistema PostgreSQL deve disporre dell'autorizzazione di scrittura su qualsiasi directory utilizzata come destinazione per qualsiasi backup.

Indipendentemente dal formato del file di backup selezionato, verrà eseguito il backup di ciascun database in un singolo file con il formato: <nome>. <estensione> dove <estensione> sarà uno di 'post', 'tar' o 'sql'.

I comandi specificati per l'esecuzione prima e dopo il backup vengono attivati prima e dopo ciascun database e NON solo una volta prima e una volta dopo l'intero set.
