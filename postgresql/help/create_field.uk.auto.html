Додати поле
На цій сторінці ви можете додати нове поле до таблиці. Параметри, які можна редагувати для нового поля:
Назва поля
Назва цього поля в таблиці.

Ширина типу (для полів varchar та char)
Максимальна кількість символів, дозволених для даних у цьому полі.

Поле масиву?
Якщо для цього параметра встановлено значення Так, поле зможе зберігати кілька значень одного типу.

Після закінчення натисніть кнопку « Зберегти », щоб оновити структуру таблиці новим полем.
