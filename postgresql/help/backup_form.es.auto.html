Ayuda del formulario de respaldo Webgre PostgreSQL

El ID de usuario del sistema PostgreSQL debe tener permiso de escritura en cualquier directorio utilizado como destino para cualquier copia de seguridad.

Independientemente del formato de archivo de copia de seguridad seleccionado, cada base de datos se respaldará en un solo archivo que tiene la forma: <nombre>. <Extensión> donde <extensión> será uno de 'post', 'tar' o 'sql'.

Los comandos especificados para ejecutarse antes y después de la copia de seguridad se activan antes y después de cada base de datos y NO solo una vez antes y una vez después de todo el conjunto.
