PostgreSQL сервер баз даних
PostgreSQL - це простий сервер баз даних для систем Unix. Він підтримує безліч баз даних, потужне управління доступом, багато типів даних і більшість стандартних синтаксисів SQL. Кожна база даних, керована сервером, може містити кілька таблиць, а кожна таблиця містить кілька рядків даних.

На головній сторінці розміщується список піктограм баз даних у верхній частині із посиланням для додавання нової бази даних під ними. Кожен сервер PostgreSQL матиме принаймні одну базу даних шаблонів, яку зазвичай називають template1 . Внизу баз даних значки для налаштування контролю доступу та дозволів, а внизу - кнопка для зупинки або запуску сервера баз даних.
