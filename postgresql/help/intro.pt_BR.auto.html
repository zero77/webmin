Servidor de banco de dados PostgreSQL
O PostgreSQL é um servidor de banco de dados simples para sistemas Unix. Ele suporta vários bancos de dados, poderosos controles de acesso, muitos tipos de dados e a maioria da sintaxe padrão do SQL. Cada banco de dados gerenciado pelo servidor pode conter várias tabelas e cada tabela contém várias linhas de dados.

A página principal exibe uma lista de ícones do banco de dados na parte superior, com um link para adicionar um novo banco de dados abaixo deles. Todo servidor PostgreSQL terá pelo menos um banco de dados de modelos, geralmente chamado de template1 . Abaixo dos bancos de dados, há ícones para configurar permissões e controle de acesso e, na parte inferior, há um botão para parar ou iniciar o servidor de banco de dados.
