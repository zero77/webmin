Adicionar campo
Nesta página, você pode adicionar um novo campo à tabela. Os parâmetros editáveis para o novo campo são:
Nome do campo
O nome desse campo na tabela.

Digite largura (para campos varchar e char)
O número máximo de caracteres permitido para dados neste campo.

Campo de matriz?
Se esta opção estiver definida como Sim, o campo poderá armazenar vários valores do mesmo tipo.

Quando terminar, clique no botão Salvar para atualizar a estrutura da tabela com o novo campo.
