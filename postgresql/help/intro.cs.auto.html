Databázový server PostgreSQL
PostgreSQL je jednoduchý databázový server pro systémy Unix. Podporuje více databází, výkonné řízení přístupu, mnoho typů dat a většinu standardní syntaxe SQL. Každá databáze spravovaná serverem může obsahovat více tabulek a každá tabulka obsahuje více řádků dat.

Hlavní stránka zruší seznam ikon databáze v horní části s odkazem pro přidání nové databáze pod nimi. Každý server PostgreSQL bude mít alespoň jednu databázi šablon, obvykle nazývanou šablona1 . Pod databázemi jsou ikony pro konfiguraci řízení přístupu a oprávnění a dole je tlačítko pro zastavení nebo spuštění databázového serveru.
