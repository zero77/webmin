Feld hinzufügen
Auf dieser Seite können Sie der Tabelle ein neues Feld hinzufügen. Die bearbeitbaren Parameter für das neue Feld sind:
Feldname
Der Name dieses Feldes in der Tabelle.

Typ width (für varchar- und char-Felder)
Die maximal zulässige Anzahl von Zeichen für Daten in diesem Feld.

Array-Feld?
Wenn diese Option auf Ja gesetzt ist, kann das Feld mehrere Werte desselben Typs speichern.

Wenn Sie fertig sind, klicken Sie auf die Schaltfläche Speichern , um die Tabellenstruktur mit dem neuen Feld zu aktualisieren.
