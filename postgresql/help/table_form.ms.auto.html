Buat Jadual
Borang ini adalah untuk membuat jadual kosong yang baru. Anda mesti memasukkan nama jadual dan satu atau lebih medan. Untuk setiap bidang, anda mesti memasukkan nama, memilih jenis data dan memasukkan lebar jenis (jika sesuai untuk jenis data). Anda juga dapat memilih apakah bidang itu adalah array multi-nilai, jika itu memungkinkan nol, jika itu adalah kunci utama untuk pengindeksan, dan jika nilai di dalamnya mesti unik. Setelah selesai, klik pada butang Buat untuk membuat jadual segera dibuat.
