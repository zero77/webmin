Дадаць поле
На гэтай старонцы вы можаце дадаць новае поле ў табліцу. Параметры для новага поля:
Назва поля
Назва гэтага поля ў табліцы.

Шырыня тыпу (для палёў varchar і char)
Максімальная колькасць сімвалаў, дазволеных для дадзеных у гэтым полі.

Поле масіва?
Калі гэты параметр усталяваны ў Так, поле зможа захоўваць некалькі значэнняў аднаго тыпу.

Калі скончыце, націсніце кнопку Захаваць, каб абнавіць структуру табліцы новым полем.
