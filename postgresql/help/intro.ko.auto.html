PostgreSQL 데이터베이스 서버
PostgreSQL은 Unix 시스템을위한 간단한 데이터베이스 서버입니다. 여러 데이터베이스, 강력한 액세스 제어, 많은 데이터 유형 및 대부분의 표준 SQL 구문을 지원합니다. 서버가 관리하는 각 데이터베이스는 여러 테이블을 포함 할 수 있으며 각 테이블에는 여러 행의 데이터가 포함됩니다.

기본 페이지는 맨 위에 데이터베이스 아이콘 목록을 표시하고 그 아래에 새 데이터베이스를 추가하기위한 링크가 있습니다. 모든 PostgreSQL 서버에는 일반적으로 template1 이라는 템플릿 데이터베이스가 하나 이상 있습니다. 데이터베이스 아래에는 액세스 제어 및 권한을 구성하기위한 아이콘이 있으며 맨 아래에는 데이터베이스 서버를 중지하거나 시작하기위한 단추가 있습니다.
