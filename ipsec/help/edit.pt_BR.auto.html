Editar ou criar conexão
Esta página permite criar uma nova conexão IPsec ou editar os detalhes de uma conexão existente. Uma conexão com as mesmas configurações deve ser criada nos sistemas nas duas extremidades do túnel da VPN que você deseja criar.

Os campos na seção de detalhes da conexão VPN IPsec que devem ser fornecidos para cada conexão e suas configurações recomendadas estão listados abaixo:

Nome da conexão
Um nome abreviado para esta conexão sem espaços.
Na inicialização do IPsec
Se a opção Adicionar conexão for escolhida, essa conexão somente será estabelecida quando a explicitação for iniciada. Se a opção Iniciar conexão for escolhida, ela será iniciada assim que o servidor IPsec for iniciado. A primeira opção é melhor para túneis para sistemas remotos ativos apenas periodicamente, enquanto a segunda é adequada para links VPN permanentes.
Compactar dados?
Deve ser deixado definido como Padrão .
Tipo de conexão
Também deve ser deixado definido como Padrão .
Informações adicionais devem ser fornecidas para os sistemas em cada extremidade da conexão. Isso deve ser o mesmo nos dois sistemas, embora geralmente os detalhes Local e Remoto sejam trocados para que as configurações desse host estejam sempre na seção Configurações do sistema local ou esquerdo . Os campos em cada seção e suas configurações sugeridas são:
Endereço IP público
Este campo deve ser definido como o endereço IP fixo do sistema ou Automático para um host cujo IP é atribuído dinamicamente. As seções local e remota não podem ter esse campo definido como Automático ao mesmo tempo.
Identificador do sistema
A opção Nome do host deve ser selecionada e o nome do host do sistema inserido no campo de texto. O FreeSWAN usa essa configuração para determinar qual seção da conexão aplicada a qual host.
Sub-rede privada por trás do sistema
Se este sistema possui uma rede interna conectada à qual o outro host deve ter acesso, digite um endereço de rede e um comprimento de prefixo (como 192.168.1.0/24 ) neste campo.
Chave pública do sistema
Geralmente, você deve selecionar Digitado abaixo e inserir a chave pública RSA do sistema na caixa de texto. Isso pode ser visto na página Mostrar chave pública nesse sistema.
Próximo salto para outro sistema
A menos que você tenha uma configuração de rede incomum, esse campo deve ser definido como Rota padrão .
Após criar ou editar uma conexão, o botão Aplicar configuração na página principal do módulo deve ser usado para ativar as alterações.
