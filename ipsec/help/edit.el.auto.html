Επεξεργασία ή δημιουργία σύνδεσης
Αυτή η σελίδα σάς επιτρέπει να δημιουργήσετε μια νέα σύνδεση IPsec ή να επεξεργαστείτε τις λεπτομέρειες μιας υπάρχουσας. Πρέπει να δημιουργηθεί σύνδεση με τις ίδιες ρυθμίσεις στα συστήματα και στα δύο άκρα της σήραγγας VPN που θέλετε να δημιουργήσετε.

Τα πεδία στην ενότητα λεπτομερειών σύνδεσης IPsec VPN που πρέπει να παρέχονται για κάθε σύνδεση και οι συνιστώμενες ρυθμίσεις τους παρατίθενται παρακάτω:

Ονομα σύνδεσης
Ένα σύντομο όνομα για αυτήν τη σύνδεση χωρίς κενά.
Κατά την εκκίνηση του IPsec
Εάν επιλεγεί η Προσθήκη σύνδεσης , αυτή η σύνδεση θα δημιουργηθεί μόνο όταν ξεκίνησε η εξειδίκευση. Εάν επιλεγεί η σύνδεση Έναρξη , τότε θα ξεκινήσει μόλις ξεκινήσει ο διακομιστής IPsec. Η πρώτη επιλογή είναι καλύτερη για σήραγγες σε απομακρυσμένα συστήματα που είναι μόνο περιοδικά ενεργά, ενώ η δεύτερη είναι κατάλληλη για μόνιμους συνδέσμους VPN.
Συμπίεση δεδομένων;
Πρέπει να αφεθεί να οριστεί ως Προεπιλογή .
Τύπος σύνδεσης
Πρέπει επίσης να αφεθεί να οριστεί ως Προεπιλογή .
Πρέπει να παρέχονται πρόσθετες πληροφορίες για τα συστήματα και στα δύο άκρα της σύνδεσης. Αυτό πρέπει να είναι το ίδιο και στα δύο συστήματα, αν και γενικά οι τοπικές και απομακρυσμένες λεπτομέρειες ανταλλάσσονται έτσι ώστε οι ρυθμίσεις για αυτόν τον κεντρικό υπολογιστή να βρίσκονται πάντα στην ενότητα Ρυθμίσεις τοπικού ή αριστερού συστήματος . Τα πεδία σε κάθε ενότητα και οι προτεινόμενες ρυθμίσεις τους είναι:
Δημόσια διεύθυνση IP
Αυτό το πεδίο πρέπει να οριστεί στη σταθερή διεύθυνση IP του συστήματος ή σε Αυτόματο για έναν κεντρικό υπολογιστή του οποίου η IP έχει εκχωρηθεί δυναμικά. Και οι δύο τοπικές και απομακρυσμένες ενότητες δεν μπορούν να ρυθμίσουν αυτό το πεδίο σε Αυτόματη ταυτόχρονα.
Αναγνωριστικό συστήματος
Η επιλογή Hostname θα πρέπει να επιλεγεί και το όνομα κεντρικού υπολογιστή του συστήματος να εισαχθεί στο πεδίο κειμένου. Το FreeSWAN χρησιμοποιεί αυτήν τη ρύθμιση για να καθορίσει ποια ενότητα της σύνδεσης εφαρμόζεται σε ποιον κεντρικό υπολογιστή.
Ιδιωτικό υποδίκτυο πίσω από το σύστημα
Εάν αυτό το σύστημα έχει ένα εσωτερικό δίκτυο συνδεδεμένο σε αυτό, στον οποίο θα πρέπει να έχει πρόσβαση ο άλλος κεντρικός υπολογιστής, εισαγάγετε μια διεύθυνση δικτύου και ένα μήκος προθέματος (όπως 192.168.1.0/24 ) σε αυτό το πεδίο.
Το δημόσιο κλειδί του συστήματος
Γενικά θα πρέπει να επιλέξετε Entered παρακάτω και να εισαγάγετε το δημόσιο κλειδί RSA του συστήματος στο πλαίσιο κειμένου. Αυτό μπορεί να φανεί στη σελίδα Εμφάνιση δημόσιου κλειδιού σε αυτό το σύστημα.
Επόμενο άλμα σε άλλο σύστημα
Εάν δεν έχετε ασυνήθιστη ρύθμιση δικτύου, αυτό το πεδίο πρέπει να οριστεί σε Προεπιλεγμένη διαδρομή .
Μετά τη δημιουργία ή την επεξεργασία μιας σύνδεσης, πρέπει να χρησιμοποιηθεί το κουμπί Εφαρμογή διαμόρφωσης στην κύρια σελίδα της μονάδας για την ενεργοποίηση των αλλαγών.
