Contas PPP
Esta página lista as contas usadas pelo servidor PPTP para clientes de autenticação. Somente aqueles no arquivo de segredos PPP cujo nome do servidor corresponde ao nome do host do sistema (ou o nome do host definido na página Opções de PPP) são mostrados, pois são os únicos que serão usados para autenticação. Isso significa que quaisquer outros segredos PPP (como para uma conexão discada ou ADSL) não serão mostrados ou tocados.

Ao editar uma conta, você pode inserir uma lista de endereços IP que podem ser atribuídos. Isso permite que um IP estático diferente seja atribuído a usuários individuais, em vez de um escolhido aleatoriamente no intervalo definido na página Opções do servidor PPTP.
