PPP kontuak
Orrialde honetan zure PPTP zerbitzariak autentifikazio-bezeroentzako erabilitako kontuak zerrendatzen dira. PPPeko sekretuen fitxategian zure sistemaren izena zure sistemaren ostalariaren izenarekin bat datorren (edo PPP Aukerak orrian ezarritako ostalariaren izena) bakarrik agertzen dira, autentikaziorako erabiliko diren bakarrak baitira. Horrek esan nahi du PPPko beste edozein sekretu (esaterako markaketa bat edo ADSL konexio baterako) ez direla erakutsiko edo ukituko.

Kontua editatzerakoan, eslei dakizkiokeen IP helbideen zerrenda sartu dezakezu. Horri esker, IP estatiko desberdin bat erabiltzaile banaka esleitu behar zaio, PPTP Server Aukerak orrian ezarritako barruti batetik ausaz aukeratu beharrean.
