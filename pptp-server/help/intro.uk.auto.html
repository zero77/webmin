Сервер PPTP VPN
Цей модуль дозволяє налаштувати сервер PoPToP, щоб ваша система могла приймати PPTP-з'єднання від клієнтів. На головній сторінці значки для встановлення загальних параметрів сервера PPTP (наприклад, IP-адреси для призначення клієнтам), для налаштування параметрів PPP (наприклад, тип аутентифікації, що використовується), для редагування облікових записів PPP, що використовуються для аутентифікації клієнтів, та для перегляду будь-які активні сеанси PPTP.

Також на головній сторінці розміщені кнопки для зупинки або запуску сервера PPTP. Під час його запуску також показана кнопка Застосувати конфігурацію для активації поточних параметрів сервера PPTP для нових з'єднань. Будь-які зміни в параметрах PPP або акаунтах негайно застосовуватимуться до всіх нових з'єднань, встановлених з цього моменту, без необхідності натискати цю кнопку.
