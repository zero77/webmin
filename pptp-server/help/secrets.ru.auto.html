Счета PPP
На этой странице перечислены учетные записи, используемые вашим сервером PPTP для аутентификации клиентов. Отображаются только те файлы в файле секретов PPP, имя сервера которых совпадает с именем хоста вашей системы (или именем хоста, заданным на странице параметров PPP), поскольку они являются единственными, которые будут использоваться для аутентификации. Это означает, что любые другие секреты PPP (например, для соединения по телефонной линии или ADSL-соединения) не будут показаны или не затронуты.

При редактировании учетной записи вы можете ввести список IP-адресов, которым она может быть назначена. Это позволяет назначать отдельный статический IP-адрес отдельным пользователям, а не случайным образом выбирать один из диапазона, заданного на странице параметров сервера PPTP.
