Настройки сервера PPTP
Параметры на этой странице влияют на процесс главного сервера PPTP, который принимает сетевые подключения от клиентов, а затем запускает демон PPP для их обработки. Наиболее важными являются IP-адреса для клиентской и серверной сторон соединения. Адрес сервера может быть установлен на один IP (например, 192.168.1.1), но адреса клиентов должны быть диапазоном или диапазонами, достаточно большими, чтобы обрабатывать все ожидаемые клиенты (например, 192.168.1.100-200).
