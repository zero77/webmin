PPTP VPN server
Tento modul umožňuje nastavit server PoPToP tak, aby váš systém mohl přijímat připojení PPTP od klientů. Na hlavní stránce jsou ikony pro nastavení obecných možností serveru PPTP (například IP adresy, které mají být přiřazeny klientům), pro nastavení možností PPP (jako je typ použitého ověření), pro úpravu účtů PPP používaných k ověření klientů a pro prohlížení jakékoli aktivní relace PPTP.

Na hlavní stránce jsou také tlačítka pro zastavení nebo spuštění serveru PPTP. Když je spuštěno, zobrazí se také tlačítko Použít konfiguraci pro aktivaci aktuálních možností serveru PPTP pro nová připojení. Veškeré změny možností nebo účtů PPP se okamžitě použijí na všechna nová připojení navázaná od té doby, aniž by bylo nutné stisknout toto tlačítko.
