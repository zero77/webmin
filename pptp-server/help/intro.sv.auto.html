PPTP VPN-server
Denna modul låter dig ställa in PoPToP-servern så att ditt system kan acceptera PPTP-anslutningar från klienter. På startsidan finns ikoner för inställning av allmänna PPTP-serveralternativ (till exempel IP-adresser som ska tilldelas till klienter), för inställning av PPP-alternativ (t.ex. vilken typ av autentisering som används), för redigering av PPP-konton som används för att verifiera klienter och för visning alla aktiva PPTP-sessioner.

På huvudsidan finns också knappar för att stoppa eller starta PPTP-servern. När den körs visas också en Apply-konfigurationsknapp för att aktivera de aktuella PPTP-serveralternativen för nya anslutningar. Eventuella ändringar av PPP-alternativ eller konton kommer omedelbart att gälla för alla nya anslutningar som upprättats från och med sedan, utan att behöva trycka på den här knappen.
