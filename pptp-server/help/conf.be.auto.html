Параметры сервера PPTP
Параметры на гэтай старонцы аказваюць эфект вядучага працэсу сервера PPTP, які прымае сеткавыя злучэнні ад кліентаў, а затым запускае дэман PPP для іх апрацоўкі. Найбольш важнымі з'яўляюцца IP-адрасы для кліента і сервера на злучэнні. Адрас сервера можа быць усталяваны ў адзін IP (напрыклад, 192.168.1.1), але адрас кліента павінен быць дыяпазонам або дыяпазонам, дастаткова вялікім для апрацоўкі ўсіх чаканых кліентаў (напрыклад, 192.168.1.100-200).
