Cuentas PPP
Esta página enumera las cuentas utilizadas por su servidor PPTP para clientes de autenticación. Solo se muestran aquellos en el archivo de secretos PPP cuyo nombre de servidor coincide con el nombre de host de su sistema (o el nombre de host establecido en la página de Opciones de PPP), ya que son los únicos que se utilizarán para la autenticación. Esto significa que no se mostrarán ni tocarán otros secretos de PPP (como para una conexión de marcado o ADSL).

Al editar una cuenta, puede ingresar una lista de direcciones IP que se le pueden asignar. Esto permite asignar una IP estática diferente a usuarios individuales, en lugar de elegir una aleatoriamente del rango establecido en la página de Opciones del servidor PPTP.
