Servidor VPN PPTP
Este módulo permite que você configure o servidor PoPToP para que seu sistema possa aceitar conexões PPTP de clientes. Na página principal, há ícones para definir opções gerais do servidor PPTP (como os endereços IP a serem atribuídos aos clientes), para definir opções PPP (como o tipo de autenticação usado), para editar contas PPP usadas para autenticar clientes e para exibir quaisquer sessões PPTP ativas.

Também na página principal há botões para parar ou iniciar o servidor PPTP. Quando está em execução, um botão Aplicar configuração também é mostrado para ativar as opções atuais do servidor PPTP para novas conexões. Quaisquer alterações nas opções ou contas de PPP serão aplicadas imediatamente a todas as novas conexões estabelecidas a partir de então, sem a necessidade de pressionar este botão.
