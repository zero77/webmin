Serveur VPN PPTP
Ce module vous permet de configurer le serveur PoPToP afin que votre système puisse accepter les connexions PPTP des clients. Sur la page principale se trouvent des icônes pour définir les options générales du serveur PPTP (telles que les adresses IP à attribuer aux clients), pour définir les options PPP (telles que le type d'authentification utilisé), pour modifier les comptes PPP utilisés pour authentifier les clients et pour afficher toutes les sessions PPTP actives.

La page principale contient également des boutons pour arrêter ou démarrer le serveur PPTP. Lorsqu'il est en cours d'exécution, un bouton Appliquer la configuration s'affiche également pour activer les options actuelles du serveur PPTP pour les nouvelles connexions. Toute modification des options ou des comptes PPP s'appliquera immédiatement à toutes les nouvelles connexions établies à partir de ce moment, sans avoir besoin d'appuyer sur ce bouton.
