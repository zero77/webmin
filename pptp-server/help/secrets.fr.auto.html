Comptes PPP
Cette page répertorie les comptes utilisés par votre serveur PPTP pour les clients d'authentification. Seuls ceux du fichier secrets PPP dont le nom de serveur correspond au nom d'hôte de votre système (ou le nom d'hôte défini sur la page Options PPP) sont affichés, car ce sont les seuls qui seront utilisés pour l'authentification. Cela signifie que tout autre secret PPP (comme pour une connexion sortante ou ADSL) ne sera ni affiché ni touché.

Lors de la modification d'un compte, vous pouvez entrer une liste d'adresses IP qui peuvent lui être attribuées. Cela permet d'attribuer une IP statique différente à des utilisateurs individuels, au lieu d'être choisie au hasard dans la plage définie sur la page Options du serveur PPTP.
