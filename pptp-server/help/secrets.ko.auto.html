PPP 계정
이 페이지는 PPTP 서버가 인증 클라이언트를 위해 사용하는 계정을 나열합니다. 서버 이름이 시스템의 호스트 이름 (또는 PPP 옵션 페이지에 설정된 호스트 이름)과 일치하는 PPP 비밀 파일에있는 파일 만 인증에 사용될 것이므로 표시됩니다. 이는 다른 PPP 비밀 (예 : 전화 접속 또는 ADSL 연결)이 표시되거나 터치되지 않음을 의미합니다.

계정을 편집 할 때 할당 할 수있는 IP 주소 목록을 입력 할 수 있습니다. 이를 통해 PPTP 서버 옵션 페이지에 설정된 범위에서 임의로 선택된 IP 대신 다른 고정 IP를 개별 사용자에게 할당 할 수 있습니다.
