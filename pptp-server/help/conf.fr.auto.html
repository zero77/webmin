Options du serveur PPTP
Les options de cette page affectent le processus du serveur PPTP maître, qui accepte les connexions réseau des clients, puis démarre un démon PPP pour les gérer. Les plus importantes sont les adresses IP des extrémités client et serveur de la connexion. L'adresse du serveur peut être définie sur une seule IP (comme 192.168.1.1), mais les adresses client doivent être une ou plusieurs plages suffisamment grandes pour gérer tous les clients simultanés que vous attendez (tels que 192.168.1.100-200).
