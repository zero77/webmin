Opções do servidor PPTP
As opções nesta página afetam o processo do servidor PPTP principal, que aceita conexões de rede dos clientes e inicia um daemon PPP para lidar com eles. Os mais importantes são os endereços IP para as extremidades do cliente e servidor da conexão. O endereço do servidor pode ser definido como um único IP (como 192.168.1.1), mas os endereços do cliente devem ser um intervalo ou intervalos grandes o suficiente para lidar com todos os clientes simultâneos que você espera (como 192.168.1.100-200).
