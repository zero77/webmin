Επιλογές διακομιστή PPTP
Οι επιλογές σε αυτήν τη σελίδα επηρεάζουν την κύρια διαδικασία διακομιστή PPTP, η οποία δέχεται συνδέσεις δικτύου από πελάτες και στη συνέχεια ξεκινά έναν δαίμονα PPP για να τις χειριστεί. Οι πιο σημαντικές είναι οι διευθύνσεις IP για τα άκρα της σύνδεσης πελάτη και διακομιστή. Η διεύθυνση του διακομιστή μπορεί να οριστεί σε μια ενιαία IP (όπως 192.168.1.1), αλλά οι διευθύνσεις των πελατών πρέπει να είναι εύρος ή εύρος αρκετά μεγάλο ώστε να χειρίζονται όλους τους ταυτόχρονους πελάτες που περιμένετε (όπως 192.168.1.100-200).
