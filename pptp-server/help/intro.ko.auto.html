PPTP VPN 서버
이 모듈을 사용하면 시스템이 클라이언트의 PPTP 연결을 수락 할 수 있도록 PoPToP 서버를 설정할 수 있습니다. 기본 페이지에는 일반 PPTP 서버 옵션 (예 : 클라이언트에 할당 할 IP 주소) 설정, PPP 옵션 설정 (예 : 사용 된 인증 유형), 클라이언트 인증에 사용되는 PPP 계정 편집 및보기 아이콘이 있습니다. 활성 PPTP 세션

또한 메인 페이지에는 PPTP 서버를 중지하거나 시작하기위한 버튼이 있습니다. 실행 중일 때 새 구성에 대한 현재 PPTP 서버 옵션을 활성화하는 구성 적용 버튼도 표시됩니다. PPP 옵션 또는 계정에 대한 변경 사항은이 버튼을 누르지 않아도 그때부터 설정된 모든 새 연결에 즉시 적용됩니다.
