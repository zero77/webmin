ตัวเลือกเซิร์ฟเวอร์ PPTP
ตัวเลือกในหน้านี้มีผลต่อกระบวนการเซิร์ฟเวอร์ PPTP หลักซึ่งยอมรับการเชื่อมต่อเครือข่ายจากไคลเอนต์จากนั้นเริ่ม PPP daemon เพื่อจัดการกับพวกเขา ที่สำคัญที่สุดคือที่อยู่ IP สำหรับไคลเอ็นต์และเซิร์ฟเวอร์สิ้นสุดของการเชื่อมต่อ ที่อยู่เซิร์ฟเวอร์สามารถตั้งค่าเป็น IP เดียว (เช่น 192.168.1.1) แต่ที่อยู่ไคลเอนต์ควรเป็นช่วงหรือช่วงที่ใหญ่พอที่จะจัดการกับไคลเอนต์พร้อมกันทั้งหมดที่คุณคาดหวัง (เช่น 192.168.1.100-200)
