Servidor VPN PPTP
Este módulo le permite configurar el servidor PoPToP para que su sistema pueda aceptar conexiones PPTP de clientes. En la página principal hay iconos para configurar las opciones generales del servidor PPTP (como las direcciones IP para asignar a los clientes), para configurar las opciones PPP (como el tipo de autenticación utilizado), para editar las cuentas PPP utilizadas para autenticar clientes y para ver cualquier sesión PPTP activa.

También en la página principal hay botones para detener o iniciar el servidor PPTP. Cuando se está ejecutando, también se muestra un botón Aplicar configuración para activar las opciones actuales del servidor PPTP para nuevas conexiones. Cualquier cambio en las opciones o cuentas de PPP se aplicará inmediatamente a todas las conexiones nuevas establecidas a partir de ese momento, sin necesidad de presionar este botón.
