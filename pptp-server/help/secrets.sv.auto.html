PPP-konton
Den här sidan listar konton som används av din PPTP-server för autentiseringsklienter. Endast de i PPP-hemlighetsfilen vars servernamn stämmer med ditt systems värdnamn (eller värdnamnet som ställts in på sidan PPP-alternativ) visas eftersom de är de enda som kommer att användas för autentisering. Detta betyder att andra PPP-hemligheter (t.ex. för en uppringning eller ADSL-anslutning) inte kommer att visas eller vidröras.

När du redigerar ett konto kan du ange en lista med IP-adresser som det kan tilldelas. Detta gör att en annan statisk IP kan tilldelas enskilda användare, istället för att en slumpmässigt väljs från området som ställs in på PPTP-serverns sida.
