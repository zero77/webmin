Λογαριασμοί PPP
Σε αυτήν τη σελίδα παρατίθενται λογαριασμοί που χρησιμοποιούνται από το διακομιστή PPTP για πελάτες ελέγχου ταυτότητας. Εμφανίζονται μόνο εκείνα στο αρχείο μυστικών PPP των οποίων το όνομα διακομιστή ταιριάζει με το όνομα κεντρικού υπολογιστή του συστήματός σας (ή το όνομα κεντρικού υπολογιστή που ορίζεται στη σελίδα Επιλογές PPP), καθώς είναι τα μόνα που θα χρησιμοποιηθούν για έλεγχο ταυτότητας. Αυτό σημαίνει ότι τυχόν άλλα μυστικά PPP (όπως για σύνδεση μέσω τηλεφώνου ή σύνδεση ADSL) δεν θα εμφανίζονται ούτε θα αγγίζονται.

Κατά την επεξεργασία ενός λογαριασμού, μπορείτε να εισαγάγετε μια λίστα διευθύνσεων IP που μπορεί να εκχωρηθεί. Αυτό επιτρέπει μια διαφορετική στατική IP να εκχωρηθεί σε μεμονωμένους χρήστες, αντί να επιλέγεται τυχαία από το εύρος που έχει οριστεί στη σελίδα Επιλογές διακομιστή PPTP.
