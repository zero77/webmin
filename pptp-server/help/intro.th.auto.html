เซิร์ฟเวอร์ PPTP VPN
โมดูลนี้ช่วยให้คุณตั้งค่าเซิร์ฟเวอร์ PoPToP เพื่อให้ระบบของคุณสามารถยอมรับการเชื่อมต่อ PPTP จากไคลเอนต์ ในหน้าหลักมีไอคอนสำหรับการตั้งค่าตัวเลือกเซิร์ฟเวอร์ PPTP ทั่วไป (เช่นที่อยู่ IP ที่กำหนดให้กับลูกค้า) สำหรับการตั้งค่าตัวเลือก PPP (เช่นประเภทการตรวจสอบความถูกต้องที่ใช้) สำหรับการแก้ไขบัญชี PPP ที่ใช้รับรองความถูกต้องของไคลเอนต์ เซสชัน PPTP ใด ๆ ที่ใช้งานอยู่

นอกจากนี้ในหน้าหลักยังมีปุ่มสำหรับหยุดหรือเริ่มต้นเซิร์ฟเวอร์ PPTP เมื่อมันทำงานปุ่ม Apply Configuration จะปรากฏขึ้นเพื่อเปิดใช้งานตัวเลือกเซิร์ฟเวอร์ PPTP ปัจจุบันสำหรับการเชื่อมต่อใหม่ การเปลี่ยนแปลงใด ๆ ในตัวเลือก PPP หรือบัญชีจะมีผลทันทีกับการเชื่อมต่อใหม่ทั้งหมดที่จัดตั้งขึ้นตั้งแต่นั้นมาโดยไม่จำเป็นต้องกดปุ่มนี้
