Server VPN PPTP
Questo modulo consente di configurare il server PoPToP in modo che il sistema possa accettare connessioni PPTP dai client. Nella pagina principale sono presenti le icone per l'impostazione delle opzioni generali del server PPTP (come gli indirizzi IP da assegnare ai client), l'impostazione delle opzioni PPP (come il tipo di autenticazione utilizzato), la modifica degli account PPP utilizzati per autenticare i client e la visualizzazione eventuali sessioni PPTP attive.

Nella pagina principale sono presenti anche pulsanti per l'arresto o l'avvio del server PPTP. Quando è in esecuzione, viene mostrato anche un pulsante Applica configurazione per attivare le opzioni correnti del server PPTP per le nuove connessioni. Eventuali modifiche alle opzioni o agli account PPP verranno immediatamente applicate a tutte le nuove connessioni stabilite da quel momento in poi, senza la necessità di premere questo pulsante.
