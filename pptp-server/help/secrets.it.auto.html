Conti PPP
Questa pagina elenca gli account utilizzati dal server PPTP per i client di autenticazione. Vengono visualizzati solo quelli nel file dei segreti PPP il cui nome del server corrisponde al nome host del sistema (o il nome host impostato nella pagina Opzioni PPP), in quanto sono gli unici che verranno utilizzati per l'autenticazione. Ciò significa che qualsiasi altro segreto PPP (come per una connessione dial-out o ADSL) non verrà mostrato o toccato.

Quando si modifica un account, è possibile inserire un elenco di indirizzi IP che possono essere assegnati. Ciò consente di assegnare un IP statico diverso ai singoli utenti, anziché sceglierne uno a caso dall'intervallo impostato nella pagina Opzioni server PPTP.
