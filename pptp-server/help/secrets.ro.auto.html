Conturi PPP
Această pagină listează conturile utilizate de serverul dvs. PPTP pentru clienții de autentificare. Sunt afișate doar cele din fișierul secret PPP al cărui nume de server se potrivește cu numele de gazdă al sistemului (sau numele de gazdă setat în pagina Opțiuni PPP), deoarece sunt singurele care vor fi utilizate pentru autentificare. Acest lucru înseamnă că orice alte secrete PPP (cum ar fi pentru o conexiune dial-out sau ADSL) nu vor fi afișate sau atinse.

La editarea unui cont, puteți introduce o listă de adrese IP care pot fi atribuite. Aceasta permite alocarea unui IP static diferit utilizatorilor individuali, în loc să fie ales la întâmplare din intervalul setat în pagina Opțiuni server PPTP.
