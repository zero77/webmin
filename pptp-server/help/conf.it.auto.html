Opzioni del server PPTP
Le opzioni in questa pagina influiscono sul processo del server PPTP principale, che accetta connessioni di rete dai client e quindi avvia un demone PPP per gestirle. I più importanti sono gli indirizzi IP per le estremità client e server della connessione. L'indirizzo del server può essere impostato su un singolo IP (come 192.168.1.1), ma gli indirizzi client devono essere un intervallo o intervalli sufficientemente grandi da gestire tutti i client simultanei previsti (come 192.168.1.100-200).
