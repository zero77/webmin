PPP-Konten
Diese Seite listet Konten auf, die von Ihrem PPTP-Server für Authentifizierungsclients verwendet werden. Es werden nur diejenigen in der PPP-Geheimdatei angezeigt, deren Servername mit dem Hostnamen Ihres Systems (oder dem auf der Seite PPP-Optionen festgelegten Hostnamen) übereinstimmt, da sie als einzige für die Authentifizierung verwendet werden. Dies bedeutet, dass andere PPP-Geheimnisse (z. B. für eine Einwahl oder eine ADSL-Verbindung) nicht angezeigt oder berührt werden.

Beim Bearbeiten eines Kontos können Sie eine Liste von IP-Adressen eingeben, die ihm zugewiesen werden können. Auf diese Weise kann einzelnen Benutzern eine andere statische IP zugewiesen werden, anstatt dass eine zufällig aus dem auf der Seite PPTP-Serveroptionen festgelegten Bereich ausgewählt wird.
