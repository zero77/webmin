PPTP Server VPN
Гэты модуль дазваляе наладзіць сервер PoPToP, каб ваша сістэма магла прымаць PPTP-злучэнні ад кліентаў. На галоўнай старонцы размяшчаюцца абразкі для ўстаноўкі агульных параметраў сервера PPTP (напрыклад, IP-адрасы для прызначэння кліентам), для ўстаноўкі параметраў PPP (напрыклад, тып аўтэнтыфікацыі, які выкарыстоўваецца), для рэдагавання ўліковых запісаў PPP, якія выкарыстоўваюцца для праверкі сапраўднасці кліентаў, і для прагляду любыя актыўныя сеансы PPTP.

На галоўнай старонцы размешчаны кнопкі для прыпынку або запуску сервера PPTP. Калі ён працуе, таксама паказана кнопка Прымяніць канфігурацыю для актывацыі бягучых параметраў сервера PPTP для новых злучэнняў. Любыя змены ў параметры PPP або ўліковыя запісы будуць неадкладна распаўсюджвацца на ўсе новыя злучэнні, устаноўленыя з гэтага моманту, без неабходнасці націскаць на гэтую кнопку.
