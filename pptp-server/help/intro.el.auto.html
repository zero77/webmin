Διακομιστής VPN PPTP
Αυτή η ενότητα σάς επιτρέπει να ρυθμίσετε τον διακομιστή PoPToP έτσι ώστε το σύστημά σας να δέχεται συνδέσεις PPTP από πελάτες. Στην κύρια σελίδα υπάρχουν εικονίδια για τον ορισμό γενικών επιλογών διακομιστή PPTP (όπως οι διευθύνσεις IP που θα εκχωρηθούν σε πελάτες), για τη ρύθμιση επιλογών PPP (όπως ο τύπος ελέγχου ταυτότητας που χρησιμοποιείται), για την επεξεργασία λογαριασμών PPP που χρησιμοποιούνται για τον έλεγχο ταυτότητας πελατών και για προβολή οποιεσδήποτε ενεργές συνεδρίες PPTP.

Επίσης στην κύρια σελίδα υπάρχουν κουμπιά για διακοπή ή εκκίνηση του διακομιστή PPTP. Όταν εκτελείται, εμφανίζεται επίσης ένα κουμπί Εφαρμογή διαμόρφωσης που ενεργοποιεί τις τρέχουσες επιλογές διακομιστή PPTP για νέες συνδέσεις. Τυχόν αλλαγές στις επιλογές PPP ή στους λογαριασμούς θα ισχύουν αμέσως για όλες τις νέες συνδέσεις που έχουν δημιουργηθεί από τότε, χωρίς να χρειάζεται να πατήσετε αυτό το κουμπί.
