Opciones de servidor PPTP
Las opciones en esta página afectan el proceso del servidor PPTP maestro, que acepta conexiones de red de clientes y luego inicia un demonio PPP para manejarlas. Las más importantes son las direcciones IP para los extremos del cliente y el servidor de la conexión. La dirección del servidor se puede establecer en una sola IP (como 192.168.1.1), pero las direcciones del cliente deben ser un rango o rangos lo suficientemente grandes como para manejar todos los clientes simultáneos que espera (como 192.168.1.100-200).
