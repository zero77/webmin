Улік ППС
На гэтай старонцы прыведзены ўліковыя запісы, якія выкарыстоўваюцца вашым серверам PPTP для кліентаў аўтэнтыфікацыі. Адлюстроўваюцца толькі тыя з файлаў сакрэтаў PPP, імя сервера якіх адпавядае імя хаста вашай сістэмы (альбо імя хаста, устаноўленаму на старонцы Параметры PPP), бо яны з'яўляюцца адзінымі, якія будуць выкарыстоўвацца для праверкі сапраўднасці. Гэта азначае, што любыя іншыя сакрэты PPP (напрыклад, для набору або злучэння ADSL) не будуць адлюстроўвацца і не кранацца.

Падчас рэдагавання ўліковага запісу вы можаце ўвесці спіс IP-адрасоў, якім ён можа быць прызначаны. Гэта дазваляе індывідуальным карыстальнікам прысвоіць іншы статычны IP, а не выбраць выпадковым чынам з дыяпазону, устаноўленага на старонцы Параметры сервера PPTP.
