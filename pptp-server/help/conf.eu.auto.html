PPTP zerbitzariaren aukerak
Orrialde honetako aukerek PPTP zerbitzariaren prozesu nagusia eragiten dute; horrek bezeroen sareko konexioak onartzen ditu eta gero PPP daemon bat abiarazten du horiek kudeatzeko. Garrantzitsuenak bezeroaren eta zerbitzariaren konexioaren muturren IP helbideak dira. Zerbitzariaren helbidea IP bakarrean ezar daiteke (192.168.1.1 bezala), baina bezeroen helbideak nahikoa handiak izan beharko lirateke, espero dituzun bezero aldi berean kudeatzeko (adibidez, 192.168.1.100-200).
