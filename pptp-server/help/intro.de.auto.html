PPTP VPN Server
Mit diesem Modul können Sie den PoPToP-Server so einrichten, dass Ihr System PPTP-Verbindungen von Clients akzeptieren kann. Auf der Hauptseite befinden sich Symbole zum Festlegen allgemeiner PPTP-Serveroptionen (z. B. der IP-Adressen, die Clients zugewiesen werden sollen), zum Festlegen von PPP-Optionen (z. B. der Art der verwendeten Authentifizierung), zum Bearbeiten von PPP-Konten zur Authentifizierung von Clients und zum Anzeigen alle aktiven PPTP-Sitzungen.

Auf der Hauptseite befinden sich auch Schaltflächen zum Stoppen oder Starten des PPTP-Servers. Während der Ausführung wird auch die Schaltfläche Konfiguration anwenden angezeigt, um die aktuellen PPTP-Serveroptionen für neue Verbindungen zu aktivieren. Alle Änderungen an den PPP-Optionen oder -Konten gelten sofort für alle neuen Verbindungen, die von da an hergestellt werden, ohne dass Sie auf diese Schaltfläche klicken müssen.
