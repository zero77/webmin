con o sin autenticación
La primera columna en este campo le permite seleccionar el tipo de autenticación para esta exportación. Las opciones son:
sys
Seguridad IP basada en red o NFS de estilo antiguo, que se aplica completamente por la dirección IP del cliente.

krb5
Autenticación Kerberos (requiere que los sistemas cliente y servidor tengan configurado Kerberos).

libkey
El mecanismo de seguridad LIPKEY.

spkm-3
Mecanismo de autenticación SPKM3.
