avec ou sans authentification
La première colonne de ce champ vous permet de sélectionner le type d'authentification pour cette exportation. Les options sont les suivantes:
sys
Sécurité NFS IP ou réseau à l'ancienne, entièrement appliquée par l'adresse IP du client.

krb5
Authentification Kerberos (nécessite que les systèmes client et serveur aient Kerberos configuré).

libkey
Le mécanisme de sécurité LIPKEY.

spkm-3
Mécanisme d'authentification SPKM3.
