cu sau fără autentificare
Prima coloană din acest câmp vă permite să selectați tipul de autentificare pentru acest export. Opțiunile sunt:
sys
IP NFS vechi sau securitate bazată pe rețea, care este aplicată în totalitate de adresa IP a clientului.

krb5
Autentificare Kerberos (necesită ca sistemele client și server să fie configurate Kerberos).

libkey
Mecanismul de securitate LIPKEY.

spkm-3
Mecanism de autentificare SPKM3.
