Teilbaumprüfung deaktivieren?
Diese Option deaktiviert die Überprüfung von Teilbäumen, was geringfügige Auswirkungen auf die Sicherheit hat, unter bestimmten Umständen jedoch die Zuverlässigkeit verbessern kann.

Wenn ein Unterverzeichnis eines Dateisystems exportiert wird, das gesamte Dateisystem jedoch nicht, wenn eine NFS-Anforderung eingeht, muss der Server nicht nur überprüfen, ob sich die Datei, auf die zugegriffen wird, im entsprechenden Dateisystem befindet (was einfach ist), sondern auch, dass sie sich im exportierter Baum (was schwieriger ist).
