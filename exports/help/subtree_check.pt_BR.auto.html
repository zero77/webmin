Desativar verificação de subárvore?
Essa opção desabilita a verificação de subárvore, o que tem implicações de segurança leves, mas pode melhorar a confiabilidade em algumas circunstâncias.

Se um subdiretório de um sistema de arquivos for exportado, mas o sistema de arquivos inteiro não for, sempre que uma solicitação NFS chegar, o servidor deverá verificar não apenas se o arquivo acessado está no sistema de arquivos apropriado (o que é fácil), mas também se está no diretório árvore exportada (que é mais difícil).
