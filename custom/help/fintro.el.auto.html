Αυτή η ενότητα μπορεί επίσης να χρησιμοποιηθεί για τη ρύθμιση κουμπιών για την επεξεργασία αρχείων που επεξεργάζονται συχνά. Αυτό μπορεί επίσης να είναι χρήσιμο εάν θέλετε να παραχωρήσετε σε ορισμένους μη αξιόπιστους χρήστες τα δικαιώματα επεξεργασίας μόνο λίγων αρχείων.

Όπως τα κουμπιά εντολών, η κύρια σελίδα θα εμφανίζει επίσης ένα κουμπί για κάθε καθορισμένο πρόγραμμα επεξεργασίας αρχείων. Για να επεξεργαστείτε έναν υπάρχοντα επεξεργαστή, απλώς κάντε κλικ στον σύνδεσμο Επεξεργασία επεξεργασίας αρχείων κάτω από αυτόν. Για να δημιουργήσετε ένα νέο πρόγραμμα επεξεργασίας, κάντε κλικ στο σύνδεσμο Δημιουργία νέου προγράμματος επεξεργασίας αρχείων στο κάτω μέρος της σελίδας.
