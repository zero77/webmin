Este módulo também pode ser usado para configurar botões para editar arquivos editados com freqüência. Isso também pode ser útil se você deseja conceder a alguns usuários não confiáveis os direitos para editar apenas alguns arquivos.

Como os botões de comando, a página principal também exibirá um botão para cada editor de arquivo definido. Para editar um editor existente, basta clicar no link Editar editor de arquivos abaixo dele. Para criar um novo editor, clique no link Criar um novo editor de arquivos na parte inferior da página.
