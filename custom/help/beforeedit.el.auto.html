Εντολή για εκτέλεση πριν από την επεξεργασία
Όποια εντολή εισαχθεί εδώ θα εκτελεστεί ως root πριν εμφανιστούν τα περιεχόμενα του αρχείου για επεξεργασία. Αυτό θα μπορούσε να χρησιμοποιηθεί για παράδειγμα για την αντιγραφή του αρχείου από άλλο σύστημα και η εντολή για εκτέλεση μετά την αποθήκευση θα μπορούσε να χρησιμοποιηθεί για την αντιγραφή του.
