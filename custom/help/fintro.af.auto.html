Hierdie module kan ook gebruik word om knoppies op te stel vir die redigering van lêers wat gereeld geredigeer word. Dit kan ook nuttig wees as u 'n onbetroubare gebruiker die regte wil gee om slegs 'n paar lêers te wysig.

Soos die opdragknoppies, sal die hoofblad ook een knoppie vertoon vir elke gedefinieerde lêerredigeerder. Om 'n bestaande redakteur te redigeer, klik op die skakel Edit file editor hieronder. Om 'n nuwe redakteur te skep, klik op die skakel Maak 'n nuwe lêerredakteur onderaan die bladsy.
