Exécuter dans le répertoire
Par défaut, la commande spécifiée s'exécutera dans le répertoire contenant les programmes de ce module Webmin, qui sera quelque chose comme /usr/libexec/webmin/custom . Cette option peut être utilisée pour spécifier un répertoire différent, ce qui peut être utile pour éviter d'avoir à utiliser des chemins d'accès complets pour la commande et ses paramètres.
