Perintah untuk dijalankan sebelum mengedit
Apa sahaja arahan yang dimasukkan di sini akan dijalankan sebagai root sebelum kandungan fail dipaparkan untuk diedit. Ini dapat digunakan misalnya untuk menyalin file dari sistem lain, dan Perintah untuk dijalankan setelah menyimpan dapat digunakan untuk menyalinnya kembali.
