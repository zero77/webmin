Setiap perintah mempunyai keterangan (ditampilkan pada tombol di halaman utama), dan perintah sebenarnya untuk dilaksanakan. String arahan ini boleh mengandungi operator shell seperti |,> dan; untuk melaksanakan pelbagai arahan dan saluran paip. String juga boleh mengandungi parameter seperti $foo , yang digantikan oleh input pengguna ketika perintah dijalankan.

Parameter ini dapat dimasukkan ke dalam tabel di bagian bawah halaman. Untuk setiap parameter anda mesti memasukkan:

Nama
Kod unik untuk parameter ini. Sekiranya namanya foo , maka $foo akan digantikan oleh nilai parameter ketika perintah itu dijalankan.

Penerangan
Penerangan di sebelah parameter ini di halaman utama.

Jenis
Pilihan ini mengawal bagaimana parameter dimasukkan. Pilihan yang ada adalah:
  • Teks
    Input teks percuma.
  • Pengguna
    Nama pengguna dari sistem anda.
  • UID
    UID pengguna dari sistem anda.
  • Kumpulan
    Nama kumpulan dari sistem anda.
  • GID
    GID kumpulan dari sistem anda.
  • Fail
    Jalan penuh ke fail.
  • Direktori
    Jalan penuh ke direktori.
  • Pilihan
    Input Ya / Tidak yang akan menetapkan parameter ke apa sahaja yang ada di sebelah sebelah jenis input hanya jika Ya dipilih.
  • Kata Laluan
    Input teks sepenuhnya, tetapi dengan kata laluan diganti dengan *.
  • Menu
    Menu pilihan drop-down, diambil dari nama fail yang dimasukkan ke dalam medan teks kepadanya. Atau, bukannya nama fail anda boleh memasukkan perintah dengan | pada akhirnya, yang outputnya akan digunakan untuk menentukan pilihan yang ada.
  • Muat naik
    Kotak input untuk memilih file di sisi klien, yang akan dimuat ke server ketika perintah dijalankan. Ini akan ditempatkan dalam file sementara, dan jalan ke file tersebut akan menjadi nilai parameter ini ketika perintah dijalankan.
  • Kotak teks
    Medan teks percuma berbilang baris. Ketika perintah dijalankan, setiap watak baris baru dalam teks yang dimasukkan akan ditukar menjadi spasi.

Dalam kebanyakan kes, nilai lalai untuk parameter adalah apa pun yang anda masukkan di kotak teks di sebelah menu jenis parameter.

Parameter petikan?
Sekiranya Ya, parameter tersebut akan disebut dengan "sebelum penggantian, yang membolehkan pengguna memasukkan nilai yang mengandungi ruang kosong.
