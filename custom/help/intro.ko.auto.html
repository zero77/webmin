사용자 명령
이 모듈을 사용하면 선택적 매개 변수 대체를 통해 일반적으로 사용되는 명령을 설정하고 실행할 수 있습니다. 이것은 일반적으로 사용되는 스크립트 나 프로그램을 브라우저에서 쉽게 실행하거나 경험이없는 사용자가 쉽게 사용할 수 있도록하려는 경우 유용 할 수 있습니다.

기본 페이지에는 각 정의 된 명령마다 하나의 단추가 표시되며 각 단추 아래에 선택적 매개 변수가 있습니다. 적절한 옵션을 선택하고 버튼을 클릭하여 명령을 실행하면 명령의 출력이 표시됩니다.

기존 명령을 편집하려면 해당 명령 아래에있는 명령 편집 링크를 클릭하십시오. 새 명령을 작성하려면 페이지 맨 아래에서 새 사용자 정의 명령 작성 링크를 클릭하십시오. 일부 Webmin 사용자는이 옵션을 사용하지 못할 수 있습니다.
