Запустить в каталоге
По умолчанию указанная команда будет выполняться в каталоге, содержащем программы этого модуля Webmin, который будет выглядеть как /usr/libexec/webmin/custom . Эта опция может использоваться для указания другого каталога, который может быть полезен для избежания необходимости использовать полные пути для команды и ее параметров.
