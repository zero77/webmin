Команда для запуска перед редактированием
Какая бы команда здесь ни была введена, она будет запущена с правами root до того, как содержимое файла будет отображено для редактирования. Это может быть использовано, например, для копирования файла из другой системы, а команда для запуска после сохранения может быть использована для его копирования.
