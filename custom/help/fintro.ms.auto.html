Modul ini juga dapat digunakan untuk mengatur butang untuk mengedit file yang sering diedit. Ini juga berguna jika anda ingin memberikan hak kepada pengguna yang tidak dipercaya untuk mengedit hanya beberapa fail.

Seperti butang arahan, halaman utama juga akan memaparkan satu butang untuk setiap penyunting fail yang ditentukan. Untuk mengedit editor yang ada, cukup klik pada pautan Edit fail penyunting di bawahnya. Untuk membuat editor baru, klik pada pautan Buat editor fail baru di bahagian bawah halaman.
