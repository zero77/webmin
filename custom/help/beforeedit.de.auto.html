Befehl, der vor dem Bearbeiten ausgeführt werden soll
Der hier eingegebene Befehl wird als root ausgeführt, bevor der Inhalt der Datei zur Bearbeitung angezeigt wird. Dies kann beispielsweise zum Kopieren der Datei von einem anderen System verwendet werden, und der nach dem Speichern auszuführende Befehl kann zum erneuten Kopieren verwendet werden.
