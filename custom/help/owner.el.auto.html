Ιδιοκτησία αρχείου
Δύο επιλογές είναι διαθέσιμες εδώ - μπορείτε είτε να αποχωρήσετε από την ιδιοκτησία του αρχείου ως έχει, είτε να επιλέξετε έναν χρήστη και μια ομάδα που θα εκχωρηθεί ιδιοκτησία μετά την αποθήκευση. Αυτή η επιλογή είναι πιο χρήσιμη για αρχεία που δημιουργήθηκαν πρόσφατα. Εάν ένα αρχείο δεν υπάρχει και αυτή η επιλογή δεν έχει οριστεί, όταν δημιουργείται, θα ανήκει στον χρήστη και η ομάδα Webmin εκτελείται ως (πιθανότατα root και root ).
