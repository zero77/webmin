Ce module peut également être utilisé pour configurer des boutons permettant de modifier des fichiers fréquemment modifiés. Cela peut également être utile si vous souhaitez accorder à un utilisateur non fiable le droit de modifier uniquement quelques fichiers.

Comme les boutons de commande, la page principale affichera également un bouton pour chaque éditeur de fichier défini. Pour modifier un éditeur existant, cliquez simplement sur le lien Modifier l'éditeur de fichier en dessous. Pour créer un nouvel éditeur, cliquez sur le lien Créer un nouvel éditeur de fichier au bas de la page.
