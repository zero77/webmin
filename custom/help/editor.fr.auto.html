Chaque éditeur de fichier a une description (affichée sur le bouton de la page principale) et un nom de fichier à modifier. Il peut s'agir de n'importe quel fichier texte qui existe actuellement sur votre système ou du nom d'un fichier qui n'existe pas encore. Il existe également quelques autres options qui peuvent être définies pour déterminer ce qui se passe lorsque le fichier est enregistré.
