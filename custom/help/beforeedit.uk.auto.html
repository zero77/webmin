Команда для запуску перед редагуванням
Яка б команда тут не була введена, запускається як root перед тим, як вміст файлу буде показаний для редагування. Це може бути використано, наприклад, для копіювання файлу з іншої системи, а Команда, яку потрібно запустити після збереження, може бути використана для її копіювання.
