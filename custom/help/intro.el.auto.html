Προσαρμοσμένες εντολές
Αυτή η ενότητα σάς επιτρέπει να ρυθμίσετε και να εκτελέσετε εντολές που χρησιμοποιούνται συνήθως, με προαιρετική αντικατάσταση παραμέτρων. Αυτό μπορεί να είναι χρήσιμο εάν θέλετε να κάνετε σενάρια ή προγράμματα που χρησιμοποιούνται συνήθως εύκολα εκτελέσιμα από το πρόγραμμα περιήγησής σας ή εύκολα διαθέσιμα σε άπειρους χρήστες.

Η κύρια σελίδα εμφανίζει ένα κουμπί για κάθε καθορισμένη εντολή, με προαιρετικές παραμέτρους κάτω από κάθε κουμπί. Μπορείτε να εκτελέσετε την εντολή απλά επιλέγοντας τις κατάλληλες επιλογές και κάνοντας κλικ στο κουμπί, οπότε θα εμφανιστεί η έξοδος από την εντολή.

Για να επεξεργαστείτε μια υπάρχουσα εντολή, απλώς κάντε κλικ στο σύνδεσμο Επεξεργασία εντολής κάτω από αυτήν. Για να δημιουργήσετε μια νέα εντολή, κάντε κλικ στο σύνδεσμο Δημιουργία νέας προσαρμοσμένης εντολής στο κάτω μέρος της σελίδας. Σημειώστε ότι αυτές οι επιλογές ενδέχεται να μην είναι διαθέσιμες σε όλους τους χρήστες του Webmin.
