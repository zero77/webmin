Comandos personalizados
Este módulo permite configurar e executar comandos comumente usados, com substituição opcional de parâmetros. Isso pode ser útil se você deseja tornar scripts ou programas comumente usados facilmente executáveis no seu navegador ou facilmente disponíveis para usuários inexperientes.

A página principal exibe um botão para cada comando definido, com parâmetros opcionais abaixo de cada botão. Você pode executar o comando simplesmente selecionando as opções apropriadas e clicando no botão, e a saída do comando será exibida.

Para editar um comando existente, basta clicar no link Editar comando abaixo dele. Para criar um novo comando, clique no link Criar um novo comando personalizado na parte inferior da página. Observe que essas opções podem não estar disponíveis para todos os usuários do Webmin.
