Inclure les jeux de fichiers LANGUAGE correspondants?
Demande au système d'inclure tous les ensembles de fichiers de messages pour les ensembles de fichiers que vous installez actuellement ou avez déjà installés. La langue utilisée lors de l'installation du système d'exploitation de base est utilisée.

Il est recommandé de laisser cette option définie sur OUI si votre langue principale n'est pas l'anglais. Cette option est ignorée si la variable LANG sur votre système est C (LANG = C).
