Файли пакунків
На цьому екрані перераховані всі файли, що містяться в якомусь пакеті. Для кожного файлу відображається така інформація:
Шлях
Абсолютна назва шляху до цього пакунка. Якщо це символьне посилання, шлях буде включати ціль посилання. Клацніть по шляху, щоб переглянути додаткові відомості про цей файл.
Власник
Користувач Unix, який повинен володіти цим файлом.
Групувати
Ця група Unix, яка повинна володіти цим файлом.
Тип
Що це за файл чи каталог.
Розмір
Розмір цього файлу повинен бути.
Статус
Якщо файл не може бути редагований, у цьому стовпці відображатимуться розбіжності між правильним розміром, власністю та контрольною сумою файлу та їх поточними значеннями.
