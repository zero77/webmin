การอัพเดตซอฟต์แวร์ COMMIT
ระบุว่าคุณต้องการส่งมอบการอัพเดตซอฟต์แวร์ที่คุณกำลังติดตั้งหรือไม่ เมื่อคุณส่งมอบซอฟต์แวร์คุณต้องยอมรับว่าไม่จำเป็นต้องใช้ผลิตภัณฑ์รุ่นก่อนหน้านี้ รุ่นก่อนหน้าทั้งหมดที่คุณมุ่งมั่นถูกลบไปแล้ว ซอฟต์แวร์ที่ส่งมอบมีสองลักษณะพิเศษ ช่วยเพิ่มพื้นที่ว่างในดิสก์ที่ใช้ในการจัดเก็บซอฟต์แวร์รุ่นเก่ากว่า มันกำจัดความเป็นไปได้ที่จะสามารถปฏิเสธการอัปเดตและกลับไปเป็นเวอร์ชั่นก่อนหน้า

ใช่สั่งให้ระบบคอมมิตทำการอัพเดตซอฟต์แวร์ที่คุณกำลังติดตั้ง ทุกเวอร์ชันก่อนเวอร์ชันที่คุณยอมรับจะถูกลบ โดยทั่วไปจะไม่แนะนำให้คอมมิทอัพเดตซอฟต์แวร์ที่คุณติดตั้งเป็นครั้งแรก ตัวอย่างเช่นหลังจากใช้เวอร์ชันใหม่คุณอาจตัดสินใจว่าคุณต้องการใช้ซอฟต์แวร์เวอร์ชันก่อนหน้า เมื่อคุณยอมรับการอัปเดตซอฟต์แวร์ที่คุณกำลังติดตั้งคุณจะไม่มีทางดึงเวอร์ชันก่อนหน้า คุณต้องติดตั้งเวอร์ชันก่อนหน้าอีกครั้งเพื่อใช้งาน

ไม่สั่งให้ระบบไม่ยอมรับการอัพเดทซอฟต์แวร์ที่คุณกำลังติดตั้ง ซอฟต์แวร์ที่คุณติดตั้งจะถูกนำไปใช้ เมื่อซอฟต์แวร์ถูกนำไปใช้กับระบบมันจะกลายเป็นรุ่นที่ใช้งานของซอฟต์แวร์ หากมีการแทนที่ซอฟต์แวร์รุ่นก่อนหน้าเวอร์ชันก่อนหน้านี้จะถูกบันทึกไว้ในไดเรกทอรีพิเศษบนดิสก์ สามารถเรียกคืนเวอร์ชันก่อนหน้าได้หากจำเป็นโดยปฏิเสธเวอร์ชันปัจจุบัน เมื่อคุณพอใจกับการอัพเดตแล้วคุณควรคอมมิทเหล่านี้เพื่อเพิ่มพื้นที่ว่างในดิสก์ที่ใช้โดยไฟล์ที่บันทึก

หากคุณเลือกไม่สำหรับ "การอัปเดตซอฟต์แวร์ COMMIT" คุณจะต้องเลือก YES เป็น "SAVE แทนที่ไฟล์"

ไม่ว่าคุณจะต้องการอัพเดทซอฟต์แวร์หรือไม่นั้นขึ้นอยู่กับสถานการณ์ คุณควรพิจารณาสิ่งต่อไปนี้ในการตัดสินใจของคุณ

o หากคุณกำลังติดตั้งการอัปเดตแต่ละรายการเพื่อแก้ไขปัญหาที่รายงานขอแนะนำให้คุณตอบ NO จากนั้นคุณสามารถตรวจสอบว่าการแก้ไขแก้ไขปัญหาของคุณและไม่ทำให้เกิดปัญหาอื่น ๆ ก่อนที่จะกระทำ

o หากคุณมีพื้นที่ว่างบนฮาร์ดดิสก์ที่ จำกัด มากคุณอาจจำเป็นต้องตอบ YES เพื่อส่งมอบซอฟต์แวร์หรือใช้ไดเรกทอรีบันทึกสำรองด้วยระบบไฟล์ที่เมาท์ NFS

o การเลือกไม่เมื่ออัพเดทผลิตภัณฑ์ทั้งหมดของคุณให้อยู่ในระดับการบำรุงรักษาใหม่สามารถใช้พื้นที่ดิสก์จำนวนมากและจะทำให้การติดตั้งใช้เวลานานขึ้น คุณต้องชั่งน้ำหนักปัจจัยเหล่านี้ด้วยว่าคุณต้องการความสามารถในการสำรองการเปลี่ยนแปลงหรือไม่ถ้าคุณมีปัญหากับระดับใหม่

o หากคุณติดตั้งผลิตภัณฑ์ใหม่และอัปเดตผลิตภัณฑ์อาจไม่มีประโยชน์สำหรับคุณในการเลือกไม่

นี่ไม่ใช่โอกาสเดียวของคุณในการส่งมอบซอฟต์แวร์ หลังจากที่คุณติดตั้งซอฟต์แวร์และใช้งานได้ชั่วขณะหนึ่งคุณสามารถส่งมอบได้ ไปที่เมนูการติดตั้งและบำรุงรักษาซอฟต์แวร์บน SMIT เลือกบำรุงรักษาซอฟต์แวร์ที่ติดตั้งแล้วเลือก Commit Applied Software (ลบไฟล์ที่บันทึกไว้)
