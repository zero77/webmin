ΥΠΕΡΒΑΣΗ ίδιες ή νεότερες εκδόσεις;
Ορίστε σε ΝΑΙ εάν θέλετε να επανεγκαταστήσετε την ίδια ή παλαιότερη έκδοση λογισμικού με την έκδοση που υπάρχει στο σύστημά σας. Σημειώστε ότι η απάντηση ΝΑΙ δεν είναι έγκυρη για ενημερώσεις. μια αντικατάσταση μπορεί να γίνει μόνο σε βασικά αρχεία αρχείων. Η προεπιλεγμένη επιλογή είναι ΟΧΙ.

Σημείωση: Δεν χρειάζεται να ορίσετε αυτήν την επιλογή σε ΝΑΙ εάν εφαρμόζετε μια νεότερη έκδοση.

Εάν ορίσετε αυτήν την επιλογή σε ΝΑΙ, πρέπει να ορίσετε το "ΑΥΤΟΜΑΤΗ εγκατάσταση απαραίτητου λογισμικού;" επιλογή για ΟΧΙ.
