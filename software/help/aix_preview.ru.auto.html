Только предварительный просмотр? (операция установки НЕ произойдет)
Указывает, хотите ли вы предварительно просмотреть установку выбранных программных продуктов и обновлений. Предварительный просмотр определяет требования для успешной установки программного обеспечения.

ДА дает системе команду отображать информацию о требованиях к пространству, о том, прошла ли программа предварительную проверку, а также список программных продуктов и обновлений, которые будут установлены. Эта информация также может содержать список программных продуктов и обновлений, которые невозможно установить из-за проблем. Значение по умолчанию для этой опции - НЕТ.

Также, выбрав YES для «DETAILED output?» Опция, список программного обеспечения и обновлений будет отображаться в порядке установки.

НЕТ поручает системе выполнить установку.
