OVERWRITE versioni uguali o più recenti?
Impostare su SÌ se si desidera reinstallare la stessa o precedente versione del software della versione esistente sul sistema. Si noti che una risposta SÌ non è valida per gli aggiornamenti; una sovrascrittura può essere eseguita solo su set di file di base. L'opzione predefinita è NO.

Nota: non è necessario impostare questa opzione su SÌ se si applica una versione successiva.

Se si imposta questa opzione su SÌ, è necessario impostare "Installa AUTOMATICAMENTE il software richiesto?" opzione su NO.
