Installer AUTOMATIQUEMENT le logiciel requis?
Indique si vous souhaitez installer automatiquement le logiciel requis. Le logiciel requis est un logiciel qui est requis pour qu'un autre produit logiciel fonctionne.

OUI demande au système d'installer automatiquement tout logiciel requis pour les produits logiciels que vous sélectionnez pour l'installation. Pour les mises à jour sélectionnées, le choix de OUI indique également au système d'installer toutes les mises à jour logicielles de remplacement trouvées sur le support d'installation.

Si vous définissez l'option "OVERWRITE versions identiques ou plus récentes" sur OUI, vous devez définir l'option "Installer automatiquement le logiciel REQUISITE" sur NON.

Pour voir la liste des produits logiciels et des prérequis qui seront installés, définissez l'option "APERÇU de l'installation" sur OUI.

NON indique au système de NE PAS installer le logiciel requis pour les produits logiciels que vous sélectionnez pour l'installation. Si vous choisissez NON et que le système rencontre une condition requise manquante lors de l'installation d'un produit, l'installation de ce produit est annulée. Le système répertorie les éléments requis manquants, puis poursuit l'installation d'autres logiciels.
