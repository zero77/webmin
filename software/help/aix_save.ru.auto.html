СОХРАНИТЬ замененные файлы?
Указывает, хотите ли вы сохранить программные файлы, которые заменяются во время установки.

ДА дает команду системе сохранить предыдущие версии программного обеспечения, которое заменяется. В случае неудачной установки выбор YES позволяет процедуре очистки восстановить предыдущую версию программного обеспечения. Если вы выбрали НЕТ для «Программное обеспечение COMMIT?», То вы должны выбрать «ДА» для «СОХРАНИТЬ замененные файлы?». Выбор YES потребует дополнительного пространства для сохранения замененных файлов. Выбор ДА также требует больше времени для установки из-за дополнительной обработки для сохранения файлов.

НЕТ указывает системе НЕ сохранять предыдущие версии программного обеспечения во время его замены. В случае неудачной установки предыдущая версия программного обеспечения НЕ МОЖЕТ быть восстановлена и помечена как BROKEN. Вы должны переустановить обновление программного обеспечения, если это произойдет.
