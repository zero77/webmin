Mehrere Volumes verarbeiten?
Wählen Sie diese Option, wenn Sie die Verarbeitung von CDs mit mehreren Volumes aktivieren möchten. Der Standardwert ist "Ja".

Für die Verarbeitung von Software von einer CD-ROM-Quelle sind möglicherweise Dateigruppen erforderlich, die auf anderen CD-Volumes gespeichert sind. Wenn Sie diese Option auf "Ja" setzen, können Dateigruppen auf anderen CD-Volumes während derselben Sitzung verarbeitet werden. Wenn Sie "Nein" einstellen, werden auch CDs mit mehreren Volumes als einzelne Standalone-CD angesehen, und auf Dateigruppen auf anderen CD-Volumes kann nicht zugegriffen werden.
