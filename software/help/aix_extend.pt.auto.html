Estender os sistemas de arquivos, se houver espaço?
Indica se você deseja que o sistema estenda os sistemas de arquivos se for necessário mais espaço durante o processamento.

YES instrui o sistema a estender os sistemas de arquivos conforme necessário. Depois que o tamanho do sistema de arquivos é estendido, ele não pode ser reduzido. Você deve remover o sistema de arquivos inteiro. Se você tiver bastante espaço no disco rígido, escolha SIM. Para visualizar a quantidade estimada de espaço necessária, defina a opção "PREVIEW only?" para SIM.

NO instrui o sistema a NÃO estender os sistemas de arquivos. Se você tiver espaço limitado no disco rígido, escolha NÃO. Se você escolher NÃO e o sistema ficar sem espaço em disco durante o processamento, uma mensagem do sistema será exibida. A mensagem indica quanto espaço em disco é necessário e quais sistemas de arquivos exigem o espaço. O sistema não instala o produto que requer espaço adicional, mas continua o processo de instalação de outros produtos. Em alguns casos, uma atualização pode ser marcada como QUEBRADA se não houver espaço suficiente e o processo de limpeza não puder ser concluído.
