Файлы пакета
На этом экране перечислены все файлы, содержащиеся в каком-либо пакете. Для каждого файла отображается следующая информация:
Дорожка
Абсолютный путь к этому файлу пакета. Если это символическая ссылка, путь будет включать цель ссылки. Нажмите на путь, чтобы просмотреть дополнительную информацию об этом файле.
владелец
Пользователь Unix, который должен владеть этим файлом.
группа
Эта группа Unix, которая должна владеть этим файлом.
Тип
Какой это тип файла или каталога.
Размер
Размер этого файла должен быть.
Положение дел
Если файл не предназначен для редактирования, в этом столбце будут отображаться любые расхождения между правильным размером, владельцем и контрольной суммой файла и их текущими значениями.
