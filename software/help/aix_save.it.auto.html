SALVA i file sostituiti?
Indica se si desidera salvare i file software che vengono sostituiti durante l'installazione.

SÌ indica al sistema di salvare le versioni precedenti del software che viene sostituito. In caso di installazione non riuscita, la scelta di SÌ consente alla procedura di pulizia di ripristinare la versione precedente del software. Se hai scelto NO per "COMMIT software?", Devi scegliere SÌ su "SALVA i file sostituiti?". La scelta di SÌ richiederà dello spazio aggiuntivo per salvare i file sostituiti. La scelta di SÌ richiede anche più tempo di installazione a causa dell'elaborazione aggiuntiva per il salvataggio dei file.

NO indica al sistema di NON salvare le versioni precedenti del software mentre viene sostituito. In caso di installazione non riuscita, la versione precedente del software NON PU be essere ripristinata ed è contrassegnata BROKEN. In questo caso è necessario reinstallare l'aggiornamento del software.
