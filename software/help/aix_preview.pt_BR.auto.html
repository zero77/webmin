PREVIEW apenas? (a operação de instalação NÃO ocorrerá)
Indica se você deseja visualizar a instalação dos produtos e atualizações de software selecionados. Uma visualização identifica os requisitos para que a instalação do software seja bem-sucedida.

SIM instrui o sistema a exibir informações sobre os requisitos de espaço, se o software passou por uma verificação de pré-instalação e uma lista de produtos e atualizações de software que serão instalados. Essas informações também podem conter uma lista de produtos e atualizações de software que não podem ser instalados devido a problemas necessários. O padrão para esta opção é NÃO.

Ao selecionar também SIM na "Saída DETALHADA?" opção, a lista do software e as atualizações serão exibidas na ordem de instalação.

NO instrui o sistema a executar a instalação.
