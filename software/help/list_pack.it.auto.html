File del pacchetto
Questa schermata elenca tutti i file contenuti in alcuni pacchetti. Per ogni file, vengono visualizzate le seguenti informazioni:
Sentiero
Il percorso assoluto di questo file di pacchetto. Se si tratta di un collegamento simbolico, il percorso includerà la destinazione del collegamento. Fare clic sul percorso per visualizzare ulteriori informazioni su questo file.
Proprietario
L'utente Unix che dovrebbe possedere questo file.
Gruppo
Questo gruppo Unix che dovrebbe possedere questo file.
genere
Che tipo di file o directory è questo.
Dimensione
La dimensione di questo file dovrebbe essere.
Stato
A meno che non si supponga che il file sia modificabile, questa colonna mostrerà eventuali discrepanze tra la dimensione, la proprietà e il checksum corretti del file e i loro valori correnti.
