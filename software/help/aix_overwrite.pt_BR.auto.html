Substituir versões iguais ou mais recentes?
Defina como SIM se desejar reinstalar a mesma ou a versão mais antiga do software que a versão existente no seu sistema. Observe que uma resposta SIM não é válida para atualizações; uma substituição só pode ser feita em conjuntos de arquivos base. A opção padrão é NÃO.

Nota: Você não precisa definir esta opção como SIM se estiver aplicando uma versão posterior.

Se você definir esta opção como SIM, deverá definir "Instalar automaticamente o software necessário?" opção para NÃO.
