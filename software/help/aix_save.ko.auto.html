교체 된 파일을 저장 하시겠습니까?
설치 중 교체 할 소프트웨어 파일을 저장할지 여부를 나타냅니다.

YES는 시스템에 교체중인 이전 버전의 소프트웨어를 저장하도록 지시합니다. 설치에 실패한 경우 예를 선택하면 정리 절차에서 이전 버전의 소프트웨어를 복원 할 수 있습니다. "COMMIT software?"에 대해 NO를 선택한 경우, "Replace replace files?"로 YES를 선택해야합니다. YES를 선택하면 교체 된 파일을 저장하기위한 추가 공간이 필요합니다. YES를 선택하면 파일 저장을위한 추가 처리로 인해 더 많은 설치 시간이 필요합니다.

NO는 시스템이 소프트웨어를 교체하는 동안 이전 버전의 소프트웨어를 저장하지 않도록 지시합니다. 설치에 실패한 경우 이전 버전의 소프트웨어를 복원 할 수 없으며 BROKEN으로 표시됩니다. 이 경우 소프트웨어 업데이트를 다시 설치해야합니다.
