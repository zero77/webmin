Arquivos de pacote
Essa tela lista todos os arquivos contidos em algum pacote. Para cada arquivo, as seguintes informações são exibidas:
Caminho
O nome do caminho absoluto para este arquivo de pacote. Se este for um link simbólico, o caminho incluirá o destino do link. Clique no caminho para ver mais informações sobre este arquivo.
Proprietário
O usuário Unix que deve possuir este arquivo.
Grupo
Este grupo Unix que deve possuir este arquivo.
Tipo
Que tipo de arquivo ou diretório é esse.
Tamanho
O tamanho que esse arquivo deve ter.
Status
A menos que o arquivo seja editável, esta coluna exibirá quaisquer discrepâncias entre o tamanho correto, propriedade e soma de verificação do arquivo e seus valores atuais.
