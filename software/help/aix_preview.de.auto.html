Nur VORSCHAU? (Installationsvorgang wird NICHT ausgeführt)
Gibt an, ob Sie eine Vorschau der Installation der ausgewählten Softwareprodukte und Updates anzeigen möchten. In einer Vorschau werden die Voraussetzungen für eine erfolgreiche Softwareinstallation angegeben.

YES weist das System an, Informationen zum Platzbedarf anzuzeigen, ob die Software eine Vorinstallationsprüfung bestanden hat, sowie eine Liste der zu installierenden Softwareprodukte und Updates. Diese Informationen können auch eine Liste von Softwareprodukten und Updates enthalten, die aufgrund der erforderlichen Probleme nicht installiert werden können. Der Standardwert für diese Option ist NEIN.

Durch Auswahl von JA für "DETAILLIERTE Ausgabe?" Option, wird die Liste der Software und Updates in der Installationsreihenfolge angezeigt.

NEIN weist das System an, die Installation durchzuführen.
