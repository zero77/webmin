Ενημερώσεις λογισμικού COMMIT;
Υποδεικνύει εάν θέλετε να πραγματοποιήσετε τις ενημερώσεις λογισμικού που εγκαθιστάτε. Όταν δεσμεύετε λογισμικό, δεσμεύεστε ότι ΔΕΝ απαιτούνται προηγούμενες εκδόσεις αυτού του προϊόντος. ΟΛΕΣ ΟΙ ΕΚΔΟΣΕΙΣ ΠΡΙΝ ΑΠΟ ΕΝΑ ΔΕΣΜΕΥΣΕΙΣ ΔΙΑΓΡΑΦΟΝΤΑΙ. Η δέσμευση λογισμικού έχει δύο αποτελέσματα. Απελευθερώνει χώρο στο δίσκο που χρησιμοποιήθηκε για την αποθήκευση παλαιότερων εκδόσεων αυτού του λογισμικού. Εξαλείφει τη δυνατότητα απόρριψης της ενημέρωσης και επιστροφής στην προηγούμενη έκδοση.

ΝΑΙ δίνει εντολή στο σύστημα να κάνει τις ενημερώσεις λογισμικού που εγκαθιστάτε. Όλες οι εκδόσεις πριν από αυτήν που δεσμεύετε διαγράφονται. Γενικά, δεν συνιστάται η πραγματοποίηση ενημερώσεων λογισμικού που εγκαθιστάτε για πρώτη φορά. Για παράδειγμα, μετά τη χρήση μιας νέας έκδοσης, ενδέχεται να αποφασίσετε ότι προτιμάτε να χρησιμοποιήσετε μια παλαιότερη έκδοση του λογισμικού. Μόλις πραγματοποιήσετε τις ενημερώσεις λογισμικού που εγκαθιστάτε, δεν θα έχετε τρόπο να ανακτήσετε προηγούμενες εκδόσεις. Πρέπει να επανεγκαταστήσετε προηγούμενες εκδόσεις για να τις χρησιμοποιήσετε.

ΟΧΙ δίνει εντολή στο σύστημα να μην κάνει τις ενημερώσεις λογισμικού που εγκαθιστάτε. Θα εφαρμοστεί το λογισμικό που εγκαθιστάτε. Όταν το λογισμικό εφαρμόζεται στο σύστημα, γίνεται η ενεργή έκδοση του λογισμικού. Εάν αντικαθιστά μια προηγούμενη έκδοση του λογισμικού, η προηγούμενη έκδοση αποθηκεύεται σε έναν ειδικό κατάλογο του δίσκου. Η προηγούμενη έκδοση μπορεί να ανακτηθεί, εάν είναι απαραίτητο, απορρίπτοντας την τρέχουσα έκδοση. Όταν είστε ικανοποιημένοι με τις ενημερώσεις, θα πρέπει να τις δεσμεύσετε να ελευθερώσετε χώρο στο δίσκο που χρησιμοποιείται από τα αποθηκευμένα αρχεία.

Εάν επιλέξετε ΟΧΙ για "Ενημέρωση λογισμικού COMMIT", τότε πρέπει να επιλέξετε ΝΑΙ για να "ΑΠΟΘΗΚΕΥΣΗ αρχείων που αντικαταστάθηκαν".

Το εάν είναι επιθυμητό να κάνετε ενημερώσεις λογισμικού ή όχι εξαρτάται από τις περιστάσεις. Θα πρέπει να λάβετε υπόψη σας τα ακόλουθα για να λάβετε την απόφασή σας.

o Εάν εγκαθιστάτε μεμονωμένες ενημερώσεις για να διορθώσετε ένα αναφερόμενο πρόβλημα, συνιστάται να απαντήσετε ΟΧΙ. Στη συνέχεια, μπορείτε να επαληθεύσετε ότι η επιδιόρθωση διορθώνει το πρόβλημά σας και δεν προκαλεί άλλα προβλήματα προτού το επιλύσετε.

o Εάν έχετε πολύ περιορισμένο χώρο στο σκληρό δίσκο, ίσως χρειαστεί να απαντήσετε ΝΑΙ για να δεσμεύσετε λογισμικό ή να χρησιμοποιήσετε έναν εναλλακτικό κατάλογο αποθήκευσης με ένα σύστημα αρχείων που είναι τοποθετημένο στο NFS.

o Η επιλογή ΟΧΙ κατά την ενημέρωση όλων των προϊόντων σας σε ένα νέο επίπεδο συντήρησης μπορεί να χρησιμοποιήσει σημαντικό χώρο στο δίσκο και θα κάνει την εγκατάσταση να πάρει περισσότερο χρόνο. Πρέπει να σταθμίσετε αυτούς τους παράγοντες με το αν χρειάζεστε τη δυνατότητα να υποστηρίξετε τις αλλαγές εάν έχετε προβλήματα με το νέο επίπεδο.

o Εάν κάνετε μια νέα εγκατάσταση ενός προϊόντος και τις ενημερώσεις του, ενδέχεται να μην υπάρχει πλεονέκτημα για εσάς στην επιλογή ΟΧΙ.

Αυτή ΔΕΝ είναι η μόνη σας ευκαιρία να δεσμεύσετε λογισμικό. Αφού εγκαταστήσετε το λογισμικό και το χρησιμοποιήσετε για λίγο, μπορείτε να το δεσμεύσετε. Μεταβείτε στο μενού Εγκατάσταση και συντήρηση λογισμικού στο SMIT, επιλέξτε Συντήρηση εγκατεστημένου λογισμικού και, στη συνέχεια, επιλέξτε Commit Applied Software (Κατάργηση αποθηκευμένων αρχείων).
