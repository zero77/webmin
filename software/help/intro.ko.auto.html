소프트웨어 패키지

패키지 소개

소프트웨어 패키지는 프로그램을 구성하는 파일 모음입니다. 이러한 파일은 모두 배포 또는 다운로드를 위해 하나의 패키지 파일로 결합되어 패키지가 설치 될 때 추출되고 올바른 위치에 배치 될 수 있습니다.

패키지 관리 시스템은 프로그램 설치 및 삭제 작업을 대부분 자동화합니다. 패키지는 일반적으로 사전 컴파일되므로 설치시 Makefile 및 스크립트 구성 에 대해 걱정할 필요가 없습니다. 패키지 관리 시스템은 모든 패키지를 추적하므로 설치 제거시 프로그램의 모든 추적을 제거 할 수 있습니다.

설치된 패키지

패키지는 System 또는 X11/Utilities 와 같은 클래스와 서브 클래스로 구성됩니다. 메인 페이지의 맨 위에는 폴더 및 패키지 트리가 있으며 각 폴더는 클래스를 나타냅니다. 폴더를 열려면 클릭하십시오. 패키지에 대한 자세한 정보를 보려면 해당 이름을 클릭하십시오.

패키지 트리 아래에는 패키지 데이터베이스를 검색하기위한 양식이 있습니다. 이름이나 설명이 입력 한 텍스트와 일치하는 패키지를 검색합니다.

새 패키지 설치

패키지 검색 버튼 아래에는 새 패키지를 설치하기위한 양식이 있습니다. 시스템의 파일, 웹 브라우저를 실행중인 컴퓨터의 파일 또는 FTP 또는 HTTP URL에서 설치할 수 있습니다. 마지막으로 Webmin은 FTP 서버 또는 웹 사이트에 접속하여 패키지 파일을 다운로드합니다. 시스템이 방화벽 뒤에있는 경우 Webmin 구성 모듈에서 프록시 서버가 올바르게 구성되어 있는지 확인하십시오.

파일 식별

메인 페이지의 맨 아래에는 시스템의 일부 파일을 식별하기위한 양식이 있습니다. /etc/inetd.conf 와 같은 절대 경로 또는 elm 과 같은 명령을 입력 할 수 있습니다. Webmin은 패키지 관리 시스템에서 파일을 찾고 파일 및 파일을 사용하는 패키지에 대한 정보를 표시합니다.
