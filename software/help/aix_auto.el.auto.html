Εγκαθιστάτε αυτόματα το απαιτούμενο λογισμικό;
Υποδεικνύει εάν θέλετε να εγκαταστήσετε αυτόματα το απαιτούμενο λογισμικό. Απαιτούμενο λογισμικό είναι λογισμικό που απαιτείται για τη λειτουργία ενός άλλου προϊόντος λογισμικού.

ΝΑΙ δίνει εντολή στο σύστημα να εγκαταστήσει αυτόματα οποιοδήποτε λογισμικό είναι απαραίτητο για τα προϊόντα λογισμικού που επιλέγετε για εγκατάσταση. Για επιλεγμένες ενημερώσεις, επιλέγοντας ΝΑΙ καθοδηγεί επίσης το σύστημα να εγκαταστήσει τυχόν αντικαταστάτες ενημερώσεις λογισμικού που βρίσκονται στο μέσο εγκατάστασης.

Εάν ορίσετε την επιλογή "OVERWRITE ίδιες ή νεότερες εκδόσεις" σε ΝΑΙ, πρέπει να ορίσετε την επιλογή "Αυτόματη εγκατάσταση λογισμικού REQUISITE" σε ΟΧΙ.

Για να δείτε μια λίστα προϊόντων και απαιτήσεων λογισμικού που θα εγκατασταθούν, ορίστε την επιλογή "ΠΡΟΒΟΛΗ της εγκατάστασης" σε ΝΑΙ.

ΟΧΙ δίνει εντολή στο σύστημα ΔΕΝ να εγκαταστήσει λογισμικό που είναι απαραίτητο για τα προϊόντα λογισμικού που επιλέγετε για εγκατάσταση. Εάν επιλέξετε ΟΧΙ και το σύστημα αντιμετωπίζει μια απαίτηση που λείπει κατά την εγκατάσταση ενός προϊόντος, η εγκατάσταση αυτού του προϊόντος ακυρώνεται. Το σύστημα παραθέτει την απαίτηση που λείπει και στη συνέχεια συνεχίζει την εγκατάσταση άλλου λογισμικού.
