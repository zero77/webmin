Gleiche oder neuere Versionen überschreiben?
Stellen Sie YES ein, wenn Sie dieselbe oder eine ältere Softwareversion wie die auf Ihrem System vorhandene Version neu installieren möchten. Beachten Sie, dass eine JA-Antwort für Aktualisierungen nicht gültig ist. Ein Überschreiben kann nur für Basisdateigruppen durchgeführt werden. Die Standardoption ist NEIN.

Hinweis: Sie müssen diese Option nicht auf JA setzen, wenn Sie eine spätere Version anwenden.

Wenn Sie diese Option auf JA setzen, müssen Sie die Option "AUTOMATISCH erforderliche Software installieren?" Option zu NEIN.
