ΑΠΟΘΗΚΕΥΣΗ αντικαταστάθηκαν αρχεία;
Υποδεικνύει εάν θέλετε να αποθηκεύσετε αρχεία λογισμικού που αντικαθίστανται κατά την εγκατάσταση.

ΝΑΙ δίνει εντολή στο σύστημα να αποθηκεύσει προηγούμενες εκδόσεις λογισμικού που αντικαθίσταται. Σε περίπτωση αποτυχημένης εγκατάστασης, η επιλογή ΝΑΙ επιτρέπει τη διαδικασία εκκαθάρισης για την επαναφορά της προηγούμενης έκδοσης του λογισμικού. Εάν επιλέξατε ΟΧΙ για το "COMMIT software;", τότε πρέπει να επιλέξετε ΝΑΙ για να "ΑΠΟΘΗΚΕΥΣΗ αρχείων που αντικαταστάθηκαν;". Η επιλογή ΝΑΙ θα απαιτήσει επιπλέον χώρο για την αποθήκευση των αντικατασταθέντων αρχείων. Η επιλογή ΝΑΙ απαιτεί επίσης περισσότερο χρόνο εγκατάστασης λόγω της επιπλέον επεξεργασίας για την αποθήκευση αρχείων.

ΟΧΙ δίνει εντολή στο σύστημα ΔΕΝ να αποθηκεύει προηγούμενες εκδόσεις λογισμικού κατά την αντικατάστασή του. Σε περίπτωση αποτυχημένης εγκατάστασης, η προηγούμενη έκδοση του λογισμικού ΔΕΝ ΜΠΟΡΕΙ να αποκατασταθεί και φέρει την ένδειξη BROKEN. Εάν συμβεί αυτό, πρέπει να εγκαταστήσετε ξανά την ενημέρωση λογισμικού.
