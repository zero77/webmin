공간이 필요한 경우 파일 시스템을 확장 하시겠습니까?
처리 중에 추가 공간이 필요한 경우 시스템이 파일 시스템을 확장하도록할지 여부를 나타냅니다.

YES는 시스템이 필요에 따라 파일 시스템을 확장하도록 지시합니다. 파일 시스템 크기가 확장되면 줄일 수 없습니다. 전체 파일 시스템을 제거해야합니다. 하드 디스크 공간이 충분하면 예를 선택하십시오. 필요한 예상 공간을 미리 보려면 "PREVIEW only?"옵션을 설정하십시오. 그렇습니다.

NO는 시스템이 파일 시스템을 확장하지 않도록 지시합니다. 하드 디스크 공간이 제한되어 있으면 NO를 선택하십시오. NO를 선택하고 처리 중에 시스템에 디스크 공간이 부족하면 시스템 메시지가 표시됩니다. 메시지는 필요한 디스크 공간과 공간이 필요한 파일 시스템을 나타냅니다. 시스템은 추가 공간이 필요한 제품을 설치하지 않지만 다른 제품의 설치 프로세스를 계속합니다. 공간이 부족하고 정리 프로세스를 완료 할 수없는 경우 업데이트가 BROKEN으로 표시 될 수 있습니다.
