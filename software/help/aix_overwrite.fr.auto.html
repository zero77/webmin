ÉCRASER des versions identiques ou plus récentes?
Réglez sur OUI si vous souhaitez réinstaller la même version ou une version antérieure du logiciel que la version existante sur votre système. Notez qu'une réponse OUI n'est pas valide pour les mises à jour; un remplacement ne peut être effectué que sur des ensembles de fichiers de base. L'option par défaut est NON.

Remarque: vous n'avez pas besoin de définir cette option sur OUI si vous appliquez une version ultérieure.

Si vous définissez cette option sur OUI, vous devez définir l'option "Installer automatiquement le logiciel requis?" option à NON.
