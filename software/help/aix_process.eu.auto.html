Hainbat bolumen prozesatu?
Aukeratu aukera hau bolumen anitzeko CDen prozesamendua gaitu nahi baduzu. Lehenetsia "Bai" da.

CD-ROM iturritik prozesatzeko softwareak beste CD bolumenetan gordetako fitxategiak behar ditzake. Aukera hau "Bai" ezarrita, saio berean beste CD bolumenetako fitxategi multzoak prozesatu daitezke. "Ez" ezarrita baduzu, bolumen anitzeko CDak CD bakar gisa ikusten dira eta CDaren beste bolumen batzuetako fitxategiak ezin dira sartu.
