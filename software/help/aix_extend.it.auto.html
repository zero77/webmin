ESTENDERE i file system se è necessario spazio?
Indica se si desidera che il sistema estenda i file system se è necessario più spazio durante l'elaborazione.

SÌ indica al sistema di estendere i file system se necessario. Una volta estesa la dimensione del file system, non è possibile ridurla. È necessario rimuovere l'intero file system. Se si dispone di molto spazio sul disco rigido, scegliere SÌ. Per visualizzare in anteprima la quantità stimata di spazio necessario, impostare l'opzione "Solo ANTEPRIMA?" a SÌ.

NO indica al sistema di NON estendere i file system. Se lo spazio sul disco rigido è limitato, selezionare NO. Se si sceglie NO e il sistema esaurisce lo spazio su disco durante l'elaborazione, viene visualizzato un messaggio di sistema. Il messaggio indica la quantità di spazio su disco richiesta e quali file system richiedono lo spazio. Il sistema non installa il prodotto che richiede spazio aggiuntivo ma continua il processo di installazione per altri prodotti. In alcuni casi, un aggiornamento può essere contrassegnato come BROKEN se lo spazio non è sufficiente e il processo di pulizia non può essere completato.
