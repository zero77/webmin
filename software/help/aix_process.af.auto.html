Verwerk veelvuldige volumes?
Kies hierdie opsie as u die verwerking van multi-volume CD's wil aktiveer. Die standaard is "Ja".

Vir die verwerking van sagteware vanaf CD-ROM-bron kan lêersettings wat op ander CD-volumes gestoor is, nodig wees. As u hierdie opsie op "Ja" stel, kan lêestelle op ander CD-volumes gedurende dieselfde sessie verwerk word. As u dit op "Nee" stel, word selfs multi-volume CD's gesien as 'n enkele selfstandige CD, en daar is nie toegang tot lêersette op ander CD-volumes nie.
