SALVAR arquivos substituídos?
Indica se você deseja salvar os arquivos de software que estão sendo substituídos durante a instalação.

SIM instrui o sistema a salvar versões anteriores do software que está sendo substituído. Em caso de falha na instalação, a escolha de SIM permite que o procedimento de limpeza restaure a versão anterior do software. Se você escolheu NÃO para "COMMIT software?", Escolha SIM para "SALVAR arquivos substituídos?". Escolher SIM exigirá algum espaço adicional para salvar os arquivos substituídos. A escolha de SIM também requer mais tempo de instalação devido ao processamento extra para salvar arquivos.

NO instrui o sistema a NÃO salvar as versões anteriores do software enquanto ele está sendo substituído. Em caso de falha na instalação, a versão anterior do software NÃO PODE ser restaurada e está marcada como QUEBRADA. Você deve reinstalar a atualização de software, se isso ocorrer.
