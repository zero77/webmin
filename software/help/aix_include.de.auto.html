Entsprechende LANGUAGE-Dateigruppen einschließen?
Weist das System an, alle Nachrichtendateigruppen für Dateigruppen einzuschließen, die Sie gerade installieren oder zuvor installiert haben. Die bei der Installation des Basisbetriebssystems verwendete Sprache wird verwendet.

Es wird empfohlen, diese Option auf JA zu setzen, wenn Ihre Hauptsprache nicht Englisch ist. Diese Option wird ignoriert, wenn die LANG-Variable auf Ihrem System C ist (LANG = C).
