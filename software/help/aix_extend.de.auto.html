Dateisysteme ERWEITERN, wenn Speicherplatz benötigt wird?
Gibt an, ob das System Dateisysteme erweitern soll, wenn während der Verarbeitung mehr Speicherplatz benötigt wird.

YES weist das System an, Dateisysteme nach Bedarf zu erweitern. Sobald die Dateisystemgröße erweitert wurde, kann sie nicht mehr reduziert werden. Sie müssen das gesamte Dateisystem entfernen. Wenn Sie über genügend Festplattenspeicher verfügen, wählen Sie JA. Stellen Sie die Option "Nur VORSCHAU?" Ein, um eine Vorschau des geschätzten Speicherplatzbedarfs anzuzeigen. zu JA.

NEIN weist das System an, Dateisysteme NICHT zu erweitern. Wenn Sie nur über begrenzten Festplattenspeicher verfügen, wählen Sie NEIN. Wenn Sie NEIN wählen und dem System während der Verarbeitung der Speicherplatz ausgeht, wird eine Systemmeldung angezeigt. Die Nachricht gibt an, wie viel Speicherplatz benötigt wird und welche Dateisysteme den Speicherplatz benötigen. Das System installiert das Produkt nicht, das zusätzlichen Speicherplatz benötigt, sondern setzt den Installationsprozess für andere Produkte fort. In einigen Fällen kann ein Update als GEBROCHEN markiert werden, wenn nicht genügend Speicherplatz vorhanden ist und der Bereinigungsvorgang nicht abgeschlossen werden kann.
