แพคเกจซอฟต์แวร์

แพ็คเกจเบื้องต้น

แพคเกจซอฟต์แวร์คือชุดของไฟล์ที่ประกอบขึ้นเป็นโปรแกรม ไฟล์เหล่านี้อาจรวมกันเป็นไฟล์แพคเกจเดียวสำหรับการแจกจ่ายหรือดาวน์โหลดเพื่อทำการแตกไฟล์และวางในตำแหน่งที่ถูกต้องเมื่อติดตั้งแพ็คเกจ

ระบบการจัดการแพคเกจทำงานส่วนใหญ่โดยอัตโนมัติในการติดตั้งและลบโปรแกรม เนื่องจากแพคเกจมักจะรวบรวมล่วงหน้าคุณไม่ต้องกังวลกับ Makefiles และ กำหนดค่า สคริปต์เมื่อติดตั้ง เนื่องจากระบบการจัดการแพคเกจติดตามทุกแพ็คเกจมันสามารถลบร่องรอยทั้งหมดของโปรแกรมเมื่อถอนการติดตั้ง

แพคเกจที่ติดตั้ง

แพคเกจถูกจัดแบ่งเป็นคลาสและคลาสย่อยเช่น System หรือ X11/Utilities ที่ด้านบนของหน้าหลักเป็นแผนผังของโฟลเดอร์และแพ็คเกจซึ่งแต่ละโฟลเดอร์จะแทนคลาส เพื่อเปิดโฟลเดอร์เพียงคลิกที่มัน หากต้องการดูข้อมูลเพิ่มเติมเกี่ยวกับแพ็คเกจคลิกที่ชื่อแพ็คเกจ

ด้านล่างทรีแพ็กเกจเป็นแบบฟอร์มสำหรับค้นหาฐานข้อมูลแพ็คเกจ วิธีนี้จะค้นหาแพ็คเกจที่มีชื่อหรือคำอธิบายตรงกับข้อความที่คุณป้อน

ติดตั้งแพ็คเกจใหม่

ด้านล่างปุ่มค้นหาแพ็คเกจเป็นฟอร์มสำหรับติดตั้งแพ็คเกจใหม่ คุณสามารถติดตั้งได้ทั้งจากไฟล์ในระบบไฟล์ในคอมพิวเตอร์ที่คุณใช้งานเว็บเบราว์เซอร์หรือจาก FTP หรือ HTTP URL ในกรณีสุดท้าย Webmin จะติดต่อเซิร์ฟเวอร์ FTP หรือเว็บไซต์และดาวน์โหลดไฟล์แพ็คเกจ หากระบบของคุณอยู่หลังไฟร์วอลล์ตรวจสอบให้แน่ใจว่าคุณได้กำหนดค่าพร็อกซีเซิร์ฟเวอร์อย่างถูกต้องในโมดูลการ กำหนดค่า Webmin

ระบุไฟล์

ที่ด้านล่างของหน้าหลักเป็นแบบฟอร์มสำหรับการระบุไฟล์บางอย่างในระบบของคุณ คุณสามารถป้อนเส้นทางแบบสัมบูรณ์เช่น /etc/inetd.conf หรือคำสั่งเช่น elm Webmin จะพยายามค้นหาไฟล์ในระบบการจัดการแพ็คเกจและแสดงข้อมูลเกี่ยวกับไฟล์และแพ็คเกจที่ใช้งาน
