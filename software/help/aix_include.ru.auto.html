Включить соответствующие наборы файлов ЯЗЫКА?
Указывает системе включить все наборы файлов сообщений для наборов файлов, которые вы устанавливаете или ранее установили. Используется язык, используемый при установке базовой операционной системы.

Рекомендуется оставить для этого параметра значение YES, если основным языком не является английский. Эта опция игнорируется, если переменная LANG в вашей системе - C (LANG = C).
