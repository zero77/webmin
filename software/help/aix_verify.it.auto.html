VERIFICARE l'installazione e il controllo delle dimensioni dei file?
Indica se si desidera che il sistema esegua alcuni controlli sul software installato.

SÌ indica al sistema di eseguire un checksum per confermare che i file del software hanno la lunghezza corretta, contenere il numero corretto di cifre e caratteri e per verificare i requisiti per tutto il software sul sistema.

NO indica al sistema di NON verificare l'installazione. Il processo di verifica può aggiungere un notevole periodo di tempo al processo di installazione. Se si desidera una procedura di installazione più rapida, selezionare NO.
