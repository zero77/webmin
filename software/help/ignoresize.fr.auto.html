Vérifiez la taille du système de fichiers?
Par défaut, lorsque Oui est sélectionné, RPM vérifie l'espace libre sur tous les systèmes de fichiers montés lors de l'installation d'un package, pour s'assurer qu'il conviendra. Cependant, si vous avez des systèmes de fichiers montés à partir de serveurs NFS, vous souhaiterez peut-être sélectionner Non à la place, afin que le package puisse réellement être installé.
