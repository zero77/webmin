ΠΡΟΕΤΟΙΜΑΣΙΑ μόνο; (η λειτουργία εγκατάστασης ΔΕΝ θα συμβεί)
Υποδεικνύει εάν θέλετε να κάνετε προεπισκόπηση της εγκατάστασης των επιλεγμένων προϊόντων και ενημερώσεων λογισμικού. Μια προεπισκόπηση προσδιορίζει τις απαιτήσεις για την επιτυχή εγκατάσταση του λογισμικού.

ΝΑΙ δίνει εντολή στο σύστημα να εμφανίζει πληροφορίες σχετικά με τις απαιτήσεις χώρου, εάν το λογισμικό πέρασε έναν έλεγχο προεγκατάστασης και μια λίστα προϊόντων και ενημερώσεων λογισμικού που θα εγκατασταθούν. Αυτές οι πληροφορίες ενδέχεται επίσης να περιέχουν μια λίστα προϊόντων λογισμικού και ενημερώσεων που δεν μπορούν να εγκατασταθούν λόγω των απαιτούμενων προβλημάτων. Η προεπιλογή για αυτήν την επιλογή είναι ΟΧΙ.

Επιλέγοντας επίσης ΝΑΙ στο "ΛΕΠΤΟΜΕΡΗΣ έξοδος;" επιλογή, η λίστα του λογισμικού και οι ενημερώσεις θα εμφανίζονται με σειρά εγκατάστασης.

ΟΧΙ δίνει εντολή στο σύστημα να εκτελέσει την εγκατάσταση.
