Traiter plusieurs volumes?
Choisissez cette option si vous souhaitez activer le traitement des CD multi-volumes. La valeur par défaut est "Oui".

Le logiciel de traitement à partir d'une source de CD-ROM peut nécessiter des ensembles de fichiers stockés sur d'autres volumes de CD. La définition de cette option sur "Oui" permet de traiter les ensembles de fichiers sur d'autres volumes de CD au cours de la même session. Si vous le définissez sur "Non", même les CD multi-volumes sont considérés comme un seul CD autonome et les ensembles de fichiers sur d'autres volumes de CD ne sont pas accessibles.
