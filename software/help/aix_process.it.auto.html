Elaborare più volumi?
Scegliere questa opzione se si desidera abilitare l'elaborazione di CD multi-volume. L'impostazione predefinita è "Sì".

L'elaborazione del software dalla sorgente CD-ROM potrebbe richiedere set di file memorizzati su altri volumi di CD. Impostando questa opzione su "Sì" è possibile elaborare set di file su altri volumi di CD durante la stessa sessione. Se lo si imposta su "No", anche i CD multi-volume vengono visualizzati come un singolo CD autonomo e non è possibile accedere ai set di file su altri volumi di CD.
