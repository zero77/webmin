Pakkefiler
Denne skjermen viser alle filene som er i pakken. For hver fil vises følgende informasjon:
Sti
Det absolutte banenavnet til denne pakkefilen. Hvis dette er en symbolsk kobling, vil banen inkludere koblingsmålet. Klikk på banen for å se mer informasjon om denne filen.
Eieren
Unix-brukeren som bør eie denne filen.
Gruppe
Denne Unix-gruppen som bør eie denne filen.
Type
Hvilken type fil eller katalog dette er.
Størrelse
Størrelsen denne filen skal være.
Status
Med mindre filen er ment å kunne redigeres, vil denne kolonnen vise eventuelle avvik mellom riktig størrelse, eierforhold og kontrollsum for filen og deres nåværende verdier.
