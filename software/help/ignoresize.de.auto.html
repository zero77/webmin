Dateisystemgrößen prüfen?
Wenn Ja ausgewählt ist, überprüft RPM standardmäßig den freien Speicherplatz auf allen bereitgestellten Dateisystemen, wenn ein Paket installiert wird, um sicherzustellen, dass es passt. Wenn Sie jedoch einige Dateisysteme von heruntergefahrenen NFS-Servern bereitgestellt haben, möchten Sie möglicherweise stattdessen Nein auswählen, damit das Paket tatsächlich installiert werden kann.
