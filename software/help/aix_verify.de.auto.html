Überprüfen und überprüfen Sie die Dateigröße?
Gibt an, ob das System einige Überprüfungen der von Ihnen installierten Software durchführen soll.

YES weist das System an, eine Prüfsumme durchzuführen, um zu bestätigen, dass Ihre Softwaredateien die richtige Länge haben, die richtige Anzahl von Ziffern und Zeichen enthalten, und um die Anforderungen für die gesamte Software auf dem System zu überprüfen.

NEIN weist das System an, Ihre Installation NICHT zu überprüfen. Der Überprüfungsprozess kann den Installationsprozess erheblich verlängern. Wenn Sie eine schnellere Installation wünschen, wählen Sie NEIN.
