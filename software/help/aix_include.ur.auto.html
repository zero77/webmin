متعلقہ LANGUAGE فائلسیٹس شامل کریں؟
سسٹم کو ہدایت دیتا ہے کہ وہ فائل سیٹس کے لئے تمام میسج فائلسیٹس کو شامل کریں جو آپ فی الحال انسٹال کر رہے ہیں یا پہلے انسٹال کر چکے ہیں۔ بیس آپریٹنگ سسٹم کی تنصیب کے دوران استعمال ہونے والی زبان استعمال کی جاتی ہے۔

یہ تجویز کی جاتی ہے کہ اگر آپ کی بنیادی زبان انگریزی نہیں ہے تو آپ یہ اختیار YES پر چھوڑ دیں۔ اگر آپ کے سسٹم میں LANG متغیر C (LANG = C) ہے تو اس اختیار کو نظرانداز کردیا جاتا ہے۔
