Incluir conjuntos de arquivos LANGUAGE correspondentes?
Instrui o sistema a incluir todos os conjuntos de arquivos de mensagens dos conjuntos de arquivos que você está instalando atualmente ou que instalou anteriormente. O idioma usado durante a instalação do sistema operacional base é usado.

É recomendável deixar essa opção definida como YES se o seu idioma principal não for o inglês. Esta opção será ignorada se a variável LANG no seu sistema for C (LANG = C).
