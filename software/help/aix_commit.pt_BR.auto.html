COMMIT atualizações de software?
Indica se você deseja confirmar as atualizações de software que está instalando. Quando você confirma um software, compromete-se a não precisar de versões anteriores deste produto. TODAS AS VERSÕES ANTES DA ÚNICA COMPROMISSO SÃO EXCLUÍDAS. A confirmação do software tem dois efeitos. Ele libera espaço em disco usado para armazenar versões mais antigas desse software. Isso elimina a possibilidade de poder rejeitar a atualização e voltar à versão anterior.

SIM instrui o sistema a confirmar as atualizações de software que você está instalando. Todas as versões anteriores à que você está enviando são excluídas. Em geral, não é recomendável confirmar atualizações de software que você está instalando pela primeira vez. Por exemplo, depois de usar uma nova versão, você pode optar por usar uma versão anterior do software. Depois de confirmar as atualizações de software que você está instalando, você não terá como recuperar as versões anteriores. Você deve reinstalar as versões anteriores para usá-las.

NO instrui o sistema a não confirmar as atualizações de software que você está instalando. O software que você está instalando será aplicado. Quando o software é aplicado ao sistema, ele se torna a versão ativa do software. Se estiver substituindo uma versão anterior do software, a versão anterior será salva em um diretório especial no disco. A versão anterior pode ser recuperada, se necessário, rejeitando a versão atual. Quando estiver satisfeito com as atualizações, confirme-as para liberar espaço em disco usado pelos arquivos salvos.

Se você escolher NÃO para "Confirmar atualizações de software", deverá escolher SIM para "SALVAR arquivos substituídos".

Se é desejável que você confirme ou não atualizações de software, depende das circunstâncias. Você deve considerar o seguinte ao tomar sua decisão.

o Se você estiver instalando atualizações individuais para corrigir um problema relatado, é recomendável que você responda NÃO. Você pode verificar se a correção corrige seu problema e não causa outros problemas antes de se comprometer.

o Se você tiver um espaço em disco muito limitado, pode ser necessário responder ao SIM para confirmar o software ou usar um diretório de salvamento alternativo com um sistema de arquivos montado pelo NFS.

o Escolher NÃO ao atualizar todos os seus produtos para um novo nível de manutenção pode usar uma quantidade significativa de espaço em disco e fará com que a instalação demore mais. Você deve ponderar esses fatores com a necessidade de recuperar as alterações se tiver problemas com o novo nível.

o Se você estiver fazendo uma nova instalação de um produto e suas atualizações, pode não haver vantagem em escolher NÃO.

Esta não é sua única oportunidade de confirmar software. Depois de instalar o software e usá-lo por algum tempo, você pode confirmar. Vá para o menu Instalação e manutenção de software no SMIT, selecione Manter software instalado e, em seguida, selecione Confirmar software aplicado (remover arquivos salvos).
