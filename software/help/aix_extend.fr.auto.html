ÉTENDRE les systèmes de fichiers si l'espace est nécessaire?
Indique si vous souhaitez que le système étende les systèmes de fichiers si davantage d'espace est nécessaire pendant le traitement.

OUI demande au système d'étendre les systèmes de fichiers si nécessaire. Une fois la taille du système de fichiers étendue, elle ne peut pas être réduite. Vous devez supprimer l'intégralité du système de fichiers. Si vous disposez de suffisamment d'espace sur le disque dur, choisissez OUI. Pour prévisualiser la quantité estimée d'espace nécessaire, définissez l'option "APERÇU uniquement?" à OUI.

NON indique au système de NE PAS étendre les systèmes de fichiers. Si vous disposez d'un espace disque limité, choisissez NON. Si vous choisissez NON et que le système manque d'espace disque pendant le traitement, un message système s'affiche. Le message indique l'espace disque requis et les systèmes de fichiers qui nécessitent cet espace. Le système n'installe pas le produit nécessitant de l'espace supplémentaire mais poursuit le processus d'installation pour les autres produits. Dans certains cas, une mise à jour peut être marquée comme CASSÉE s'il n'y a pas assez d'espace et que le processus de nettoyage ne peut pas être terminé.
