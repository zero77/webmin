Software Paketeak

Paketeak aurkeztea

Software pakete bat programa osatzen duten fitxategien bilduma da. Fitxategi horiek guztiak pakete fitxategi bakarrean konbinatu edo deskargatu ahal izateko, paketea instalatuta dagoenean kokatu eta kokapen egokietan kokatu ahal izango dira.

Paketeak kudeatzeko sistema batek programak instalatu eta ezabatzeko lan gehienak automatizatzen ditu. Paketeak aurrez konpilatuak izan ohi direnez, ez duzu Makefiles eta kezkak konfiguratzen instalatzerakoan. Paketeak kudeatzeko sistemak pakete guztien jarraipena egiten duelako, desinstalatzen denean programa baten arrasto guztiak ken ditzake.

Instalatutako Paketeak

Paketeak klase eta azpiklasetan antolatuta daude, adibidez, Sistema edo X11/Utilities . Orrialde nagusiaren goiko aldean karpeta eta paketeen zuhaitza dago, karpeta bakoitzak klase bat adierazten du. Karpeta bat irekitzeko, egin klik gainean. Pakete bati buruzko informazio gehiago ikusteko, egin klik haren izenean.

Paketeen zuhaitzaren azpian paketeen datu-basea bilatzeko inprimakia dago. Hala, izena edo deskribapena paketeak bilatuko dira sartzen dituzun testu guztiekin.

Instalatu pakete berria

Paketen bilaketa botoiaren azpian pakete berri bat instalatzeko inprimakia dago. Zure sistemako fitxategi batetik, zure arakatzailea exekutatzen ari zaren ordenagailuan edo FTP edo HTTP URL batetik instalatu dezakezu. Azken kasuan, Webmin FTP zerbitzariarekin edo webgunearekin harremanetan jarriko da eta paketearen fitxategia deskargatuko du. Zure sistema firewall baten atzean badago, ziurtatu proxy zerbitzariak behar bezala konfiguratuta dituzula Webmin Konfigurazio moduluan.

Fitxategi bat identifikatu

Orrialde nagusiaren behealdean zure sisteman fitxategi batzuk identifikatzeko inprimakia dago. Edo /etc/inetd.conf bezalako bide absolutua edo elm bezalako komandoak sar ditzakezu. Webmin fitxategia paketeak kudeatzeko sisteman aurkitzen saiatuko da, eta fitxategiari eta erabiltzen dituzten paketeei buruzko informazioa bistaratuko da.
