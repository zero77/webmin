Обрабатывать несколько томов?
Выберите эту опцию, если хотите включить обработку многотомных компакт-дисков. По умолчанию «Да».

Для обработки программного обеспечения из источника CD-ROM могут потребоваться наборы файлов, хранящиеся на других томах CD. Установка этой опции на «Да» позволяет обрабатывать наборы файлов на других томах CD во время одного и того же сеанса. Если для этого параметра установлено значение «Нет», даже многотомные компакт-диски будут рассматриваться как отдельный автономный компакт-диск, и наборы файлов на других томах компакт-дисков будут недоступны.
