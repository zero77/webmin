Pacchetti software

Introduzione ai pacchetti

Un pacchetto software è una raccolta di file che compongono un programma. Tutti questi file possono essere combinati in un unico file del pacchetto per la distribuzione o il download, per essere estratti e collocati nelle posizioni corrette quando il pacchetto è installato.

Un sistema di gestione dei pacchetti automatizza la maggior parte del lavoro di installazione e cancellazione dei programmi. Poiché i pacchetti sono generalmente precompilati, non è necessario preoccuparsi dei Makefile e configurare gli script durante l'installazione. Poiché il sistema di gestione dei pacchetti tiene traccia di tutti i pacchetti, durante la disinstallazione può rimuovere tutte le tracce di un programma.

Pacchetti installati

I pacchetti sono organizzati in classi e sottoclassi, come Sistema o X11/Utilità . Nella parte superiore della pagina principale è presente un albero di cartelle e pacchetti, in cui ogni cartella rappresenta una classe. Per aprire una cartella, fai semplicemente clic su di essa. Per visualizzare ulteriori informazioni su un pacchetto, fare clic sul suo nome.

Sotto l'albero dei pacchetti è presente un modulo per la ricerca nel database dei pacchetti. Questo cercherà i pacchetti il cui nome o descrizione corrispondano a qualsiasi testo inserito.

Installa un nuovo pacchetto

Sotto il pulsante di ricerca del pacchetto è presente un modulo per l'installazione di un nuovo pacchetto. È possibile installare da un file sul proprio sistema, un file sul computer su cui si sta eseguendo il browser Web o da un URL FTP o HTTP. Nell'ultimo caso, Webmin contatterà il server FTP o il sito Web e scaricherà il file del pacchetto. Se il tuo sistema è protetto da un firewall, assicurati di aver configurato correttamente i server proxy nel modulo Configurazione Webmin .

Identifica un file

Nella parte inferiore della pagina principale c'è un modulo per identificare alcuni file sul tuo sistema. Puoi inserire un percorso assoluto come /etc/inetd.conf o un comando come elm . Webmin tenterà di trovare il file nel sistema di gestione dei pacchetti e visualizzerà le informazioni sul file e sui pacchetti che lo utilizzano.
