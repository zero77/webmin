Includeți seturi de fișiere LANGUAGE corespunzătoare?
Instruiește sistemul să includă toate seturile de fișiere de mesaje pentru seturile de fișiere pe care le instalați sau le-ați instalat anterior. Se folosește limba utilizată în timpul instalării sistemului de operare de bază.

Este recomandat să lăsați această opțiune setată la DA, dacă limba dvs. primară nu este engleza. Această opțiune este ignorată dacă variabila LANG din sistemul dvs. este C (LANG = C).
