SAVE ersetzte Dateien?
Gibt an, ob Sie Softwaredateien speichern möchten, die während der Installation ersetzt werden.

YES weist das System an, frühere Versionen der Software zu speichern, die ersetzt wird. Im Falle einer fehlgeschlagenen Installation können Sie durch Auswahl von JA die vorherige Version der Software wiederherstellen. Wenn Sie NEIN für "COMMIT-Software?" Gewählt haben, müssen Sie JA wählen, um "Ersetzte Dateien SPEICHERN?". Wenn Sie JA wählen, wird zusätzlicher Speicherplatz benötigt, um ersetzte Dateien zu speichern. Die Auswahl von YES erfordert aufgrund der zusätzlichen Verarbeitung zum Speichern von Dateien auch mehr Installationszeit.

NEIN weist das System an, frühere Softwareversionen NICHT zu speichern, während sie ersetzt werden. Im Falle einer fehlgeschlagenen Installation kann die vorherige Version der Software NICHT wiederhergestellt werden und ist mit BROKEN gekennzeichnet. In diesem Fall müssen Sie das Software-Update neu installieren.
