Instalați automat software-ul necesar?
Indică dacă doriți să instalați automat software-ul necesar. Software-ul necesar este software care este necesar pentru ca un alt produs software să funcționeze.

DA solicită sistemului să instaleze automat orice software care este necesar pentru produsele software selectate pentru instalare. Pentru actualizări selectate, alegând DA solicită sistemului să instaleze toate actualizările software de înlocuire găsite pe suportul de instalare.

Dacă setați opțiunea „OVERWRITE la fel sau versiuni mai noi” la DA, trebuie să setați opțiunea „Instalează automat software-ul REQUISITE” la NO.

Pentru a vedea o listă de produse și cerințe software care vor fi instalate, setați opțiunea „PREVEDEAZĂ instalarea” pe DA.

NO nu solicită sistemului să instaleze software-ul care este necesar pentru produsele software pe care le selectați pentru instalare. Dacă alegeți NU și sistemul întâmpină o cerință lipsă în timpul instalării unui produs, instalarea respectivului produs este anulată. Sistemul listează necesarul care lipsește și continuă instalarea altor programe software.
