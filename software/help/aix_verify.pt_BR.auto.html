VERIFICAR a instalação e verificar o tamanho dos arquivos?
Indica se você deseja que o sistema execute algumas verificações no software que você instalou.

SIM instrui o sistema a executar uma soma de verificação para confirmar se os arquivos do software têm o tamanho correto, conter o número correto de dígitos e caracteres e verificar os requisitos para todo o software no sistema.

NO instrui o sistema a NÃO verificar sua instalação. O processo de verificação pode adicionar uma quantidade significativa de tempo ao processo de instalação. Se você deseja um procedimento de instalação mais rápido, escolha NÃO.
