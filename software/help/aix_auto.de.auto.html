AUTOMATISCH die erforderliche Software installieren?
Gibt an, ob Sie die erforderliche Software automatisch installieren möchten. Erforderliche Software ist Software, die erforderlich ist, damit ein anderes Softwareprodukt funktioniert.

JA weist das System an, automatisch alle Software zu installieren, die für die für die Installation ausgewählten Softwareprodukte erforderlich ist. Wenn Sie für ausgewählte Updates die Option JA auswählen, wird das System auch angewiesen, alle auf dem Installationsmedium vorhandenen Software-Updates zu installieren.

Wenn Sie die Option "Gleiche oder neuere Versionen überschreiben" auf JA setzen, müssen Sie die Option "REQUISITE-Software automatisch installieren" auf NEIN setzen.

Um eine Liste der zu installierenden Softwareprodukte und -anforderungen anzuzeigen, setzen Sie die Option "VORSCHAU der Installation" auf JA.

NEIN weist das System an, KEINE Software zu installieren, die für die für die Installation ausgewählten Softwareprodukte erforderlich ist. Wenn Sie NEIN wählen und das System während der Installation eines Produkts auf eine fehlende Anforderung stößt, wird die Installation dieses Produkts abgebrochen. Das System listet die fehlenden erforderlichen Elemente auf und setzt die Installation anderer Software fort.
