Συμπερίληψη αντίστοιχων LANGUAGE σετ αρχείων;
Δίνει εντολή στο σύστημα να συμπεριλάβει όλα τα σύνολα αρχείων μηνυμάτων για σύνολα αρχείων που εγκαθιστάτε ή έχετε εγκαταστήσει προηγουμένως. Χρησιμοποιείται η γλώσσα που χρησιμοποιείται κατά την εγκατάσταση του βασικού λειτουργικού συστήματος.

Συνιστάται να αφήσετε αυτήν την επιλογή σε ΝΑΙ εάν η κύρια γλώσσα σας δεν είναι Αγγλικά. Αυτή η επιλογή αγνοείται εάν η μεταβλητή LANG στο σύστημά σας είναι C (LANG = C).
