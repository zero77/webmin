Includere i set di file LANGUAGE corrispondenti?
Indica al sistema di includere tutti i set di file dei messaggi per i set di file che si stanno attualmente installando o che sono stati precedentemente installati. Viene utilizzata la lingua utilizzata durante l'installazione del sistema operativo di base.

Si consiglia di lasciare questa opzione impostata su SÌ se la lingua principale non è l'inglese. Questa opzione viene ignorata se la variabile LANG sul sistema è C (LANG = C).
