Абнавіць квоты карыстальніка
Гэтая старонка можа выкарыстоўвацца для абнаўлення блока або абмежавання файлаў для некалькіх карыстальнікаў адначасова. Для кожнага тыпу квот вы можаце выбраць альбо пакінуць яго нязменным для карыстальніка, усталяваць яго неабмежавана, альбо ўсталяваць яго ў пэўны ліміт.
