Αντιγραφή ποσοστώσεων χρηστών
Η σελίδα σάς επιτρέπει να αντιγράψετε ποσοστώσεις σε όλα τα συστήματα αρχείων για ορισμένους χρήστες σε έναν ή περισσότερους άλλους χρήστες. Υπάρχουν τρεις τρόποι επιλογής στους χρήστες στους οποίους θα αντιγράψουν:
Ολοι οι χρήστες
Οι ποσοστώσεις θα αντιγραφούν σε κάθε άλλο χρήστη στο σύστημά σας.
Επιλεγμένοι χρήστες
Οι ποσοστώσεις θα αντιγραφούν στον χρήστη ή στους επιλεγμένους χρήστες.
Μέλη επιλεγμένων ομάδων
Οι ποσοστώσεις θα αντιγραφούν σε όλους τους χρήστες που έχουν μία από τις επιλεγμένες ομάδες ως κύρια ή δευτερεύουσα ομάδα.
