Kemas kini Kuota Pengguna
Halaman ini dapat digunakan untuk mengemas kini had blok atau fail untuk beberapa pengguna sekaligus. Untuk setiap jenis kuota, anda boleh memilih sama ada untuk membiarkannya tidak berubah untuk pengguna, menetapkannya menjadi tidak terhad, atau menetapkannya ke had tertentu.
