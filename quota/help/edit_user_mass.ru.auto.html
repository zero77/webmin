Обновление пользовательских квот
Эта страница может использоваться для обновления ограничений блоков или файлов для нескольких пользователей одновременно. Для каждого типа квоты вы можете выбрать либо оставить его неизменным для пользователя, либо установить его неограниченным, либо установить определенный предел.
