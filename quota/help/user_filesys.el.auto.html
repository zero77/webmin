Ποσοστά χρηστών
Αυτή η σελίδα απαριθμεί όλα τα συστήματα αρχείων στα οποία κάποιος χρήστης έχει όριο. Μπορείτε να κάνετε κλικ σε ένα σύστημα αρχείων από τη λίστα για να επεξεργαστείτε το όριό του ή να επιλέξετε ένα σύστημα αρχείων από το παρακάτω μενού και να κάνετε κλικ στο Επεξεργασία ορίου . Εάν ένα σύστημα αρχείων δεν περιλαμβάνεται στη λίστα, δεν υπάρχει όριο στον αριθμό αρχείων ή στο χώρο που μπορεί να καταλάβει ο χρήστης σε αυτό το σύστημα αρχείων.

Κάνοντας κλικ στο κουμπί Αντιγραφή ποσοστώσεων , μπορείτε να αντιγράψετε όλες τις ποσοστώσεις για αυτόν τον χρήστη σε έναν ή περισσότερους άλλους χρήστες. Αυτό μπορεί να είναι χρήσιμο εάν θέλετε να εκχωρήσετε σε έναν αριθμό χρηστών την ίδια ποσότητα χώρου στο δίσκο.
