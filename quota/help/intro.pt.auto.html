Cotas de disco

Introdução

As cotas são um método de controlar quanto usuários e grupos de espaço em disco podem usar. Para cada sistema de arquivos, você pode colocar limites em quanto espaço em disco e quantos arquivos cada usuário pode usar. Alguns sistemas Unix também podem permitir que você limite o espaço e os arquivos usados por todos os membros de algum grupo.

As cotas podem ser configuradas apenas para sistemas de arquivos Unix locais. Os sistemas de arquivos NFS montados a partir de outros sistemas usam as cotas do servidor, se houver. Os sistemas de arquivos não Unix, como o MS-DOS ou o ISO9660, não podem suportar cotas.

Para cada usuário, quatro limites são impostos:

Limite de bloqueio suave
O número de blocos de disco permitidos antes que uma mensagem de aviso seja exibida.
Limite de bloco rígido
O número máximo de blocos que o arquivo do usuário pode ocupar.
Limite de arquivo flexível
O número de arquivos permitidos antes que uma mensagem de aviso seja exibida.
Limite de arquivo rígido
O número máximo de arquivos que o usuário pode possuir.
Normalmente, os blocos de disco têm 512 ou 1024 bytes de tamanho, dependendo do sistema operacional. Ao calcular cotas, todos os tamanhos de arquivo são arredondados para o bloco mais próximo. Assim, mesmo um arquivo vazio usa pelo menos em bloco.

Para fins de cotas de arquivos, todo diretório, dispositivo, link simbólico ou arquivo normal conta para o número máximo de arquivos que um usuário pode possuir. Tecnicamente, a cota de arquivo é realmente uma cota em inodes .

O módulo de cotas de disco

A página principal deste módulo mostra uma lista de sistemas de arquivos locais que podem suportar cotas. Se atualmente, as cotas estão ativadas para um sistema de arquivos, você pode clicar no caminho para exibir as cotas para cada usuário. Caso contrário, clicar em Cotas ativadas ativará cotas para esse sistema de arquivos.

O sistema operacional Linux não permite que cotas sejam ativadas, a menos que o sistema de arquivos tenha a opção Usar cotas configurada no módulo Disks and Filesystems. Se nenhum dos seus sistemas de arquivos tiver essa opção configurada, nenhum será listado na página principal.

Abaixo da lista de sistemas de arquivos, há um botão para exibir todas as cotas para um usuário selecionado. Isso será exibido apenas se pelo menos um sistema de arquivos tiver cotas de usuário ativadas. Da mesma forma, se pelo menos um sistema de arquivos tiver cotas de grupo ativadas, haverá um botão para exibir todas as cotas de um grupo selecionado.
