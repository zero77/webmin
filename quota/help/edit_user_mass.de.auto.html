Benutzerkontingente aktualisieren
Diese Seite kann verwendet werden, um die Block- oder Dateilimits für mehrere Benutzer gleichzeitig zu aktualisieren. Für jeden Kontingenttyp können Sie auswählen, ob er für den Benutzer unverändert bleiben, auf unbegrenzt oder auf ein bestimmtes Limit festgelegt werden soll.
