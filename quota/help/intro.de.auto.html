Festplattenkontingente

Einführung

Kontingente sind eine Methode zum Steuern, wie viel Speicherplatz Benutzer und Gruppen verwenden können. Für jedes Dateisystem können Sie festlegen, wie viel Speicherplatz und wie viele Dateien jeder Benutzer verwenden darf. Bei einigen Unix-Systemen können Sie möglicherweise auch den Speicherplatz und die Dateien einschränken, die von allen Mitgliedern einer Gruppe verwendet werden.

Kontingente können nur für lokale Unix-Dateisysteme eingerichtet werden. Von anderen Systemen gemountete NFS-Dateisysteme verwenden gegebenenfalls die Kontingente vom Server. Nicht-Unix-Dateisysteme wie MS-DOS oder ISO9660 können Kontingente überhaupt nicht unterstützen.

Für jeden Benutzer sind vier Grenzwerte festgelegt:

Soft Block Limit
Die Anzahl der zulässigen Festplattenblöcke, bevor eine Warnmeldung angezeigt wird.
Harte Blockbegrenzung
Die maximale Anzahl von Blöcken, die die Benutzerdatei belegen darf.
Soft File Limit
Die Anzahl der Dateien, die zulässig sind, bevor eine Warnmeldung angezeigt wird.
Hard File Limit
Die maximale Anzahl von Dateien, die der Benutzer besitzen darf.
Festplattenblöcke haben je nach Betriebssystem normalerweise eine Größe von 512 oder 1024 Byte. Bei der Berechnung von Kontingenten werden alle Dateigrößen auf den nächsten Block aufgerundet. Somit verwendet auch eine leere Datei zumindest einen Block.

Für Dateikontingente zählt jedes Verzeichnis, Gerät, jeder symbolische Link oder jede normale Datei zur maximalen Anzahl von Dateien, die ein Benutzer besitzen darf. Technisch gesehen ist das Dateikontingent tatsächlich ein Kontingent für Inodes .

Das Disk Quotas-Modul

Die Hauptseite dieses Moduls zeigt eine Liste lokaler Dateisysteme, die Kontingente unterstützen können. Wenn derzeit Kontingente für ein Dateisystem aktiviert sind, können Sie auf dessen Pfad klicken, um die Kontingente für jeden Benutzer anzuzeigen. Wenn nicht, wird auf Aktiviert Quotas klicken Quoten für das Dateisystem aktivieren.

Das Linux - Betriebssystem erlaubt keine Quoten eingeschaltet werden , wenn das Dateisystem die Verwendung Quotas Option Satz in den Datenträger und Dateisysteme Modul hat. Wenn für keines Ihrer Dateisysteme diese Option festgelegt ist, wird keines auf der Hauptseite aufgelistet.

Unter der Liste der Dateisysteme befindet sich eine Schaltfläche zum Anzeigen aller Kontingente für einen ausgewählten Benutzer. Dies wird nur angezeigt, wenn in mindestens einem Dateisystem Benutzerkontingente aktiviert sind. Wenn in mindestens einem Dateisystem Gruppenkontingente aktiviert sind, wird eine Schaltfläche zum Anzeigen aller Kontingente für eine ausgewählte Gruppe angezeigt.
