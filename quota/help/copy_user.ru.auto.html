Копирование пользовательских квот
Страница позволяет вам копировать квоты во всех файловых системах для одного пользователя одному или нескольким другим пользователям. Существует три способа выбора пользователей для копирования:
Все пользователи
Квоты будут скопированы для всех остальных пользователей вашей системы.
Выбранные пользователи
Квоты будут скопированы для выбранного пользователя или пользователей.
Члены выбранных групп
Квоты будут скопированы для всех пользователей, у которых одна из выбранных групп является их основной или дополнительной группой.
