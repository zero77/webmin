Benutzerkontingente kopieren
Auf dieser Seite können Sie Kontingente auf allen Dateisystemen für einige Benutzer auf einen oder mehrere andere Benutzer kopieren. Es gibt drei Möglichkeiten, um auszuwählen, auf welche Benutzer kopiert werden soll:
Alle Nutzer
Kontingente werden auf jeden anderen Benutzer Ihres Systems kopiert.
Ausgewählte Benutzer
Kontingente werden auf den oder die ausgewählten Benutzer kopiert.
Mitglieder ausgewählter Gruppen
Kontingente werden an alle Benutzer kopiert, die eine der ausgewählten Gruppen als primäre oder sekundäre Gruppe haben.
