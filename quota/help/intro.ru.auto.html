Дисковые квоты

Вступление

Квоты - это метод контроля количества дискового пространства, которое могут использовать пользователи и группы. Для каждой файловой системы вы можете установить ограничения на количество дискового пространства и количество файлов, которые может использовать каждый пользователь. Некоторые системы Unix также могут позволять вам ограничивать пространство и файлы, используемые всеми членами какой-либо группы.

Квоты могут быть установлены только для локальных файловых систем Unix. Файловые системы NFS, смонтированные из других систем, используют квоты с сервера, если таковые имеются. Файловые системы не Unix, такие как MS-DOS или ISO9660, вообще не поддерживают квоты.

Для каждого пользователя накладываются четыре ограничения:

Предел мягкого блока
Количество дисковых блоков, разрешенных до появления предупреждающего сообщения.
Предел жесткого блока
Максимальное количество блоков, которое может занимать файл пользователя.
Мягкий лимит файлов
Количество файлов, разрешенных до появления предупреждающего сообщения.
Жесткий лимит файлов
Максимальное количество файлов, которыми может владеть пользователь.
Дисковые блоки обычно имеют размер 512 или 1024 байта, в зависимости от операционной системы. При расчете квот все размеры файлов округляются до ближайшего блока. Таким образом, даже пустой файл использует хотя бы один блок.

Для целей файловых квот каждый каталог, устройство, символическая ссылка или обычный файл учитывается по отношению к максимальному количеству файлов, которыми может владеть пользователь. Технически, квота файла на самом деле является квотой на inode .

Модуль дисковых квот

На главной странице этого модуля показан список локальных файловых систем, которые могут поддерживать квоты. Если квоты в настоящее время включены для файловой системы, вы можете щелкнуть по ее пути, чтобы отобразить квоты для каждого пользователя. Если нет, то, нажав на Enabled Квоты будет активировать квоты для этой файловой системы.

Операционная система Linux не позволяет включать квоты, если в файловой системе не установлен параметр « Использовать квоты» в модуле «Диски и файловые системы». Если ни в одной из ваших файловых систем этот параметр не установлен, то ни одна из них не будет указана на главной странице.

Ниже списка файловых систем находится кнопка для отображения всех квот для выбранного пользователя. Это будет отображаться, только если хотя бы в одной файловой системе включены пользовательские квоты. Аналогично, если хотя бы в одной файловой системе включены групповые квоты, будет кнопка для отображения всех квот для выбранной группы.
