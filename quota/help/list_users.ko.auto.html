파일 시스템 사용자
이 페이지에는 일부 파일 시스템에서 할당량이있는 모든 사용자가 나열됩니다. 당신은 자신의 할당량을 편집하거나 목록 아래의 메뉴에서 사용자를 선택하고 편집 할당량을 클릭하는 사용자 이름을 클릭 할 수 있습니다. 사용자가 목록에 없으면 파일 수나 사용자가 사용할 수있는 디스크 공간에 제한이 없습니다.

사용자 목록 아래에는 유예 시간 편집 버튼이 있습니다. 그러면 디스크 공간을 더 이상 사용할 수 없게되기 전에 사용자가 소프트 한계를 초과 할 수있는 시간을 편집 할 수있는 양식이 나타납니다.

할당량 확인 버튼을 클릭하면 전체 파일 시스템을 스캔하여 각 사용자의 블록 및 파일 사용량을 최신 상태로 만듭니다. 할당량이 한동안 비활성화되었거나 일부 사용자에 대해 할당 된 경우 필요할 수 있습니다.

페이지 하단에는 신규 사용자의 초기 차단 및 파일 할당량을 설정하는 양식이 있습니다. 이는 Webmin 사용자 및 그룹 모듈로 작성된 사용자에게만 적용됩니다.
