Скопіюйте квоти користувачів
Сторінка дозволяє копіювати квоти у всіх файлових системах для певного користувача до одного або декількох інших користувачів. Існує три способи вибрати, до яких користувачів скопіювати:
Всі користувачі
Квоти будуть скопійовані на всіх інших користувачів вашої системи.
Вибрані користувачі
Квоти будуть скопійовані користувачеві або вибраним користувачам.
Члени вибраних груп
Квоти будуть скопійовані для всіх користувачів, у яких одна із вибраних груп є їх основною або вторинною групою.
