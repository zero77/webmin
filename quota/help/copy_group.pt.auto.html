Copiar cotas de grupo
A página permite copiar cotas em todos os sistemas de arquivos de algum grupo para um ou mais outros grupos. Existem três maneiras de escolher para quais grupos copiar:
Todos os Grupos
As cotas serão copiadas para todos os outros grupos do seu sistema.
Grupos Selecionados
As cotas serão copiadas para o grupo ou grupos selecionados.
Grupos contendo usuários
As cotas serão copiadas para todos os grupos dos quais qualquer usuário selecionado é membro.
