Дискові квоти

Вступ

Квоти - це метод контролю, який обсяг дискового простору можуть використовувати користувачі та групи. Для кожної файлової системи ви можете розмістити обмеження на кількість дискового простору та скільки файлів може використовувати кожен користувач. Деякі системи Unix можуть також дозволяти обмежувати простір та файли, які використовуються всіма членами певної групи.

Квоти можна налаштувати лише для локальних файлових систем Unix. Файлові системи NFS, встановлені з інших систем, використовують квоти від сервера, якщо такі є. Файлові системи Unix, такі як MS-DOS або ISO9660, взагалі не можуть підтримувати квоти.

Для кожного користувача встановлено чотири ліміти:

Ліміт м'якого блоку
Кількість дозволених блоків дисків перед появою попереджувального повідомлення.
Межа жорсткого блоку
Максимальна кількість блоків, які може займати файл користувача.
Обмеження м'якого файлу
Кількість файлів, дозволених до появи попереджувального повідомлення.
Обмеження жорсткого файлу
Максимальна кількість файлів, якими користувач може володіти.
Зазвичай дискові блоки розміром 512 або 1024 байти, залежно від операційної системи. При обчисленні квот усі розміри файлів округляються до найближчого блоку. При цьому навіть порожній файл використовує принаймні на блок.

Для квот на файли кожен каталог, пристрій, символічне посилання або звичайний файл зараховується до максимальної кількості файлів, якими користувач може володіти. Технічно квота файлів - це фактично квота на inode .

Модуль квот диска

На головній сторінці цього модуля відображається перелік локальних файлових систем, які можуть підтримувати квоти. Якщо квоти на даний момент увімкнено для файлової системи, ви можете натиснути на її шлях, щоб відобразити квоти для кожного користувача. Якщо ні, натискання на Enabled Quotas активує квоти для цієї файлової системи.

Операційна система Linux не дозволяє включати квоти, якщо у файловій системі не встановлено параметр Використовувати квоти в модулі Disks and Filesystems. Якщо для жодної з ваших файлових систем цей параметр не встановлено, жодна з них не буде вказана на головній сторінці.

Під списком файлових систем розташована кнопка для відображення всіх квот для обраного користувача. Це відображатиметься лише в тому випадку, якщо для принаймні однієї файлової системи включені квоти користувачів. Аналогічно, якщо в принаймні одній файловій системі увімкнено групові квоти, з'явиться кнопка для відображення всіх квот для вибраної групи.
