Cotas de usuário
Esta página lista todos os sistemas de arquivos nos quais algum usuário possui uma cota. Você pode clicar em um sistema de arquivos da lista para editar sua cota ou selecionar um sistema de arquivos no menu abaixo e clicar em Editar cota . Se um sistema de arquivos não estiver na lista, não há limite para o número de arquivos ou a quantidade de espaço que o usuário pode ocupar nesse sistema de arquivos.

Clicar no botão Copiar cotas permite copiar todas as cotas desse usuário para um ou mais usuários. Isso pode ser útil se você deseja alocar um número de usuários com a mesma quantidade de espaço em disco.
