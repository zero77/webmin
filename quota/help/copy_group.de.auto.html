Gruppenkontingente kopieren
Auf dieser Seite können Sie Kontingente auf allen Dateisystemen für eine Gruppe in eine oder mehrere andere Gruppen kopieren. Es gibt drei Möglichkeiten, auszuwählen, in welche Gruppen kopiert werden soll:
Alle Gruppen
Kontingente werden in jede andere Gruppe auf Ihrem System kopiert.
Ausgewählte Gruppen
Kontingente werden in die ausgewählte Gruppe oder Gruppen kopiert.
Gruppen mit Benutzern
Kontingente werden in alle Gruppen kopiert, zu denen einer der ausgewählten Benutzer gehört.
