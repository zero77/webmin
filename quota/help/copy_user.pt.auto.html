Copiar cotas de usuário
A página permite copiar cotas em todos os sistemas de arquivos de algum usuário para um ou mais usuários. Existem três maneiras de escolher para quais usuários copiar:
Todos os usuários
As cotas serão copiadas para todos os outros usuários do seu sistema.
Usuários Selecionados
As cotas serão copiadas para o usuário ou usuários selecionados.
Membros de grupos selecionados
As cotas serão copiadas para todos os usuários que possuem um dos grupos selecionados como grupo primário ou secundário.
