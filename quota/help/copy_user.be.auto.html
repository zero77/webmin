Скапіруйце карыстацкія квоты
На гэтай старонцы вы можаце скапіяваць квоты ва ўсіх файлавых сістэмах для аднаго карыстальніка для аднаго або некалькіх іншых карыстальнікаў. Ёсць тры спосабы выбару, у якія карыстальнікі скапіяваць:
Усе карыстальнікі
Квоты будуць скапіяваны ў любы іншы карыстальнік вашай сістэмы.
Выбраныя карыстальнікі
Квоты будуць скапіяваны для карыстальніка альбо выбраных карыстальнікаў.
Члены выбраных груп
Квоты будуць скапіяваны ва ўсіх карыстальнікаў, якія маюць адну з выбраных груп у якасці асноўнай ці другаснай.
