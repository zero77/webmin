Benutzerkontingente
Diese Seite listet alle Dateisysteme auf, für die ein Benutzer ein Kontingent hat. Sie können aus der Liste auf einem Dateisystem klicken seine Quote zu bearbeiten, oder ein Dateisystem aus dem Menü aus und klicken Sie auf Bearbeiten Quote. Wenn ein Dateisystem nicht in der Liste enthalten ist, gibt es keine Begrenzung für die Anzahl der Dateien oder den Speicherplatz, den der Benutzer auf diesem Dateisystem belegen darf.

Die Copy Quotas Schaltfläche klicken , können Sie alle Quoten für diesen Benutzer ein oder mehr anderen Benutzer kopieren. Dies kann nützlich sein, wenn Sie mehreren Benutzern den gleichen Speicherplatz zuweisen möchten.
