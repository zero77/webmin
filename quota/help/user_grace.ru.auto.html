Пользователь Grace Times
Льготные периоды - это время, в течение которого пользователь может превысить свое мягкое ограничение, прежде чем ему будет запрещено использовать больше дискового пространства или создавать новые файлы. Как только пользователь превысит свой предел мягких блоков, у него будет определенное время, чтобы уменьшить использование своего диска ниже предела, прежде чем ему будет запрещено использовать больше блоков. То же самое относится к пределу мягкого файла.
