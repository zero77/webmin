Χρήστες συστημάτων αρχείων
Αυτή η σελίδα απαριθμεί όλους τους χρήστες που έχουν ποσοστώσεις σε ορισμένα συστήματα αρχείων. Μπορείτε να κάνετε κλικ σε ένα όνομα χρήστη για να επεξεργαστείτε το όριό τους ή να επιλέξετε έναν χρήστη από το μενού κάτω από τη λίστα και να κάνετε κλικ στο Επεξεργασία ορίου . Εάν ένας χρήστης δεν είναι στη λίστα, δεν υπάρχει όριο στον αριθμό των αρχείων ή στην ποσότητα χώρου στο δίσκο που μπορεί να χρησιμοποιήσει.

Κάτω από τη λίστα των χρηστών βρίσκεται το κουμπί Edit Grace Times . Αυτό σας μεταφέρει σε μια φόρμα για την επεξεργασία του χρόνου που επιτρέπεται σε έναν χρήστη να ξεπεράσει το μαλακό όριό του προτού αποτραπεί η χρήση περισσότερου χώρου στο δίσκο.

Κάνοντας κλικ στο κουμπί Έλεγχος ποσοστώσεων εκτελείται σάρωση ολόκληρου του συστήματος αρχείων για την ενημέρωση της χρήσης μπλοκ και αρχείων από κάθε χρήστη. Αυτό μπορεί να είναι απαραίτητο εάν οι ποσοστώσεις έχουν απενεργοποιηθεί για κάποιο χρονικό διάστημα ή εάν έχουν ενεργοποιηθεί μόνο για κάποιο χρήστη.

Στο κάτω μέρος της σελίδας υπάρχει μια φόρμα για τον καθορισμό των αρχικών ποσοστώσεων μπλοκ και αρχείων για νέους χρήστες. Αυτά θα ισχύουν μόνο για χρήστες που έχουν δημιουργηθεί με τη λειτουργική μονάδα Webmin Users and Groups.
