Ποσότητες δίσκου

Εισαγωγή

Οι ποσοστώσεις είναι μια μέθοδος ελέγχου του ποσού χώρου στο δίσκο που μπορούν να χρησιμοποιήσουν οι χρήστες και οι ομάδες. Για κάθε σύστημα αρχείων μπορείτε να ορίσετε όρια στο χώρο του δίσκου και πόσα αρχεία μπορεί να χρησιμοποιήσει κάθε χρήστης. Ορισμένα συστήματα Unix ενδέχεται επίσης να σας επιτρέψουν να περιορίσετε το χώρο και τα αρχεία που χρησιμοποιούνται από όλα τα μέλη μιας ομάδας.

Οι ποσοστώσεις μπορούν να ρυθμιστούν μόνο για τοπικά συστήματα αρχείων Unix. Τα συστήματα αρχείων NFS που είναι τοποθετημένα από άλλα συστήματα χρησιμοποιούν τις ποσοστώσεις από το διακομιστή, εάν υπάρχουν. Τα συστήματα αρχείων Non Unix όπως το MS-DOS ή το ISO9660 δεν μπορούν να υποστηρίξουν καθόλου ποσοστώσεις.

Για κάθε χρήστη επιβάλλονται τέσσερα όρια:

Όριο μαλακού μπλοκ
Ο αριθμός των μπλοκ δίσκου που επιτρέπονται πριν εμφανιστεί ένα προειδοποιητικό μήνυμα.
Όριο σκληρού μπλοκ
Ο μέγιστος αριθμός μπλοκ που μπορεί να καταλαμβάνει το αρχείο του χρήστη.
Όριο μαλακών αρχείων
Ο αριθμός των αρχείων που επιτρέπονται πριν εμφανιστεί ένα προειδοποιητικό μήνυμα.
Όριο σκληρού αρχείου
Ο μέγιστος αριθμός αρχείων που μπορεί να έχει ο χρήστης.
Τα μπλοκ δίσκου έχουν συνήθως μέγεθος 512 ή 1024 byte, ανάλογα με το λειτουργικό σύστημα. Κατά τον υπολογισμό των ποσοστώσεων, όλα τα μεγέθη αρχείων στρογγυλοποιούνται στο πλησιέστερο μπλοκ. Έτσι, ακόμη και ένα κενό αρχείο χρησιμοποιεί τουλάχιστον στο μπλοκ.

Για τους σκοπούς των ποσοστώσεων αρχείων, κάθε κατάλογος, συσκευή, συμβολικός σύνδεσμος ή κανονικός φάκελος μετράει στον μέγιστο αριθμό αρχείων που μπορεί να έχει ένας χρήστης. Τεχνικά, το όριο αρχείων είναι στην πραγματικότητα ένα όριο στα inodes .

Η ενότητα Quotas δίσκου

Η κύρια σελίδα αυτής της ενότητας εμφανίζει μια λίστα με τοπικά συστήματα αρχείων που μπορούν να υποστηρίξουν ποσοστώσεις. Εάν οι ποσοστώσεις είναι αυτήν τη στιγμή ενεργοποιημένες για ένα σύστημα αρχείων, μπορείτε να κάνετε κλικ στη διαδρομή του για να εμφανίσετε τις ποσοστώσεις για κάθε χρήστη. Εάν όχι, κάνοντας κλικ στο Enabled Quotas θα ενεργοποιηθούν οι ποσοστώσεις για αυτό το σύστημα αρχείων.

Το λειτουργικό σύστημα Linux δεν επιτρέπει την ενεργοποίηση των ποσοστώσεων, εκτός εάν το σύστημα αρχείων έχει ορίσει την επιλογή Χρήση ποσοστώσεων στη μονάδα δίσκων και αρχείων. Εάν κανένα από τα συστήματα αρχείων σας δεν έχει αυτήν την επιλογή, τότε κανένα δεν θα εμφανίζεται στην κύρια σελίδα.

Κάτω από τη λίστα των συστημάτων αρχείων υπάρχει ένα κουμπί για την εμφάνιση όλων των ποσοστώσεων για έναν επιλεγμένο χρήστη. Αυτό θα εμφανίζεται μόνο εάν τουλάχιστον ένα σύστημα αρχείων έχει ενεργοποιήσει τις ποσοστώσεις χρήστη. Ομοίως, εάν τουλάχιστον ένα σύστημα αρχείων έχει ενεργοποιήσει τις ομαδικές ποσοστώσεις, θα υπάρχει ένα κουμπί για την εμφάνιση όλων των ποσοστώσεων για μια επιλεγμένη ομάδα.
