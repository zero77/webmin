Ενημέρωση ποσοστώσεων χρηστών
Αυτή η σελίδα μπορεί να χρησιμοποιηθεί για την ενημέρωση των ορίων μπλοκ ή αρχείων για πολλούς χρήστες ταυτόχρονα. Για κάθε τύπο ποσόστωσης, μπορείτε να επιλέξετε είτε να το αφήσετε αμετάβλητο για τον χρήστη, να το ορίσετε σε απεριόριστο ή να το ορίσετε σε ένα συγκεκριμένο όριο.
