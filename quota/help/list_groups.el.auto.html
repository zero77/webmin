Ομάδες αρχείων συστήματος
Το σύστημα ποσοστώσεων Unix μετρά όλα τα αρχεία που ανήκουν σε μια ομάδα στο όριο αυτής της ομάδας. Οι πραγματικές ομάδες στις οποίες βρίσκεται ο χρήστης που κατέχει το αρχείο δεν έχει σημασία.

Αυτή η σελίδα απαριθμεί όλες τις ομάδες που έχουν ποσοστώσεις σε ορισμένα συστήματα αρχείων. Μπορείτε να κάνετε κλικ σε ένα όνομα ομάδας για να επεξεργαστείτε το όριό τους ή να επιλέξετε μια ομάδα από το μενού κάτω από τη λίστα και να κάνετε κλικ στο Επεξεργασία ορίου . Εάν μια ομάδα δεν περιλαμβάνεται στη λίστα, δεν υπάρχει όριο στον αριθμό των αρχείων ή στον χώρο του δίσκου που μπορεί να χρησιμοποιήσει.

Κάτω από τη λίστα των ομάδων βρίσκεται το κουμπί Edit Grace Times . Αυτό σας μεταφέρει σε μια φόρμα για την επεξεργασία του χρόνου που επιτρέπεται σε μια ομάδα να υπερβεί το όριο του μαλακού προτού αποτρέψει τη χρήση περισσότερου χώρου στο δίσκο.

Κάνοντας κλικ στο κουμπί Έλεγχος ποσοστώσεων εκτελείται σάρωση ολόκληρου του συστήματος αρχείων για την ενημέρωση της χρήσης μπλοκ και αρχείων από κάθε ομάδα. Αυτό μπορεί να είναι απαραίτητο εάν οι ποσοστώσεις έχουν απενεργοποιηθεί για κάποιο χρονικό διάστημα ή εάν έχουν ενεργοποιηθεί μόνο για κάποια ομάδα.

Στο κάτω μέρος της σελίδας υπάρχει μια φόρμα για τον καθορισμό των αρχικών ποσοστώσεων μπλοκ και αρχείων για νέες ομάδες. Αυτά θα ισχύουν μόνο για ομάδες που έχουν δημιουργηθεί με τη λειτουργική μονάδα χρηστών και ομάδων Webmin.
