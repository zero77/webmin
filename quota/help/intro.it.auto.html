Quote disco

introduzione

Le quote sono un metodo per controllare la quantità di spazio su disco che utenti e gruppi possono utilizzare. Per ogni filesystem puoi porre limiti su quanto spazio su disco e quanti file può usare ogni utente. Alcuni sistemi Unix possono anche consentire di limitare lo spazio e i file utilizzati da tutti i membri di alcuni gruppi.

Le quote possono essere configurate solo per filesystem Unix locali. I filesystem NFS montati da altri sistemi utilizzano le quote dal server, se presenti. I filesystem non Unix come MS-DOS o ISO9660 non supportano affatto le quote.

Per ogni utente vengono imposti quattro limiti:

Limite soft block
Il numero di blocchi disco consentiti prima che venga visualizzato un messaggio di avviso.
Limite del blocco rigido
Il numero massimo di blocchi che il file dell'utente può occupare.
Limite file soft
Il numero di file consentiti prima che venga visualizzato un messaggio di avviso.
Limite file rigido
Il numero massimo di file che l'utente può possedere.
I blocchi del disco hanno in genere una dimensione di 512 o 1024 byte, a seconda del sistema operativo. Quando si calcolano le quote, tutte le dimensioni dei file vengono arrotondate per eccesso al blocco più vicino. Quindi anche un file vuoto utilizza almeno sul blocco.

Ai fini delle quote di file, ogni directory, dispositivo, collegamento simbolico o file normale conta per il numero massimo di file che un utente può possedere. Tecnicamente, la quota del file è in realtà una quota sugli inode .

Il modulo quote disco

La pagina principale di questo modulo mostra un elenco di filesystem locali che possono supportare le quote. Se le quote sono attualmente abilitate per un filesystem, è possibile fare clic sul relativo percorso per visualizzare le quote per ciascun utente. In caso contrario, facendo clic su Quote abilitate verranno attivate le quote per quel file system.

Il sistema operativo Linux non consente l'attivazione delle quote a meno che nel file system non sia impostata l'opzione Usa quote nel modulo Dischi e filesystem. Se nessuno dei tuoi filesystem ha questa opzione impostata, nessuno verrà elencato nella pagina principale.

Sotto l'elenco dei filesystem c'è un pulsante per visualizzare tutte le quote per un utente selezionato. Questo verrà visualizzato solo se almeno un filesystem ha le quote utente abilitate. Allo stesso modo, se almeno un filesystem ha le quote di gruppo abilitate, ci sarà un pulsante per visualizzare tutte le quote per un gruppo selezionato.
