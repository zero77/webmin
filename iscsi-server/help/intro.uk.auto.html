iSCSI Server
Цей модуль дозволяє експортувати дискові пристрої в інші системи по мережі за допомогою протоколу iSCSI. Конфігурація здійснюється головним чином шляхом створення трьох типів об'єктів - пристроїв для спільного використання , комбінацій пристроїв та цілей обміну , описаних нижче. Ви також можете редагувати налаштування, що стосуються всього сервера iSCSI, використовуючи сторінку Параметри сервера iSCSI .

Пристрої для Частки (екстенти в конфігураційному файлі)
Кожен із них - це розділ, пристрій RAID, логічний том або файл, який можна повністю або частково експортувати через iSCSI. Спільні пристрої не повинні використовуватись для будь-яких інших цілей у цій системі.

Комбінації пристроїв ( пристрої у конфігураційному файлі)
Вони дозволяють комбінувати кілька пристроїв для спільного використання для збільшення експорту. Пристрої можуть бути об'єднані в стилі RAID0, щоб додати їх розміри разом, або дзеркальним стилем RAID1 для надмірності та покращення продуктивності читання.

Обмін цілями ( ціль у конфігураційному файлі)
Кожен запис у цьому списку експортує або комбінацію пристрою, або комбінацію пристроїв до всіх систем у вибраній мережі. Частка може бути як лише для читання, так і для читання / запису.

У нижній частині головної сторінки модуля знаходяться кнопки для зупинки, запуску та перезавантаження сервера iSCSI. Зміни конфігурації не застосовуватимуться, поки сервер не перезапущений.
