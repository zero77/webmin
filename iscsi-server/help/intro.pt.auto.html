Servidor iSCSI
Este módulo permite exportar dispositivos de disco para outros sistemas na rede usando o protocolo iSCSI. A configuração é feita principalmente pela criação de três tipos de objetos - Dispositivos para Compartilhar , Combinações de Dispositivos e Destinos de Compartilhamento , descritos abaixo. Você também pode editar as configurações que se aplicam a todo o servidor iSCSI usando a página Opções do servidor iSCSI .

Dispositivos para compartilhar ( extensões no arquivo de configuração)
Cada um deles é uma partição, dispositivo RAID, volume lógico ou arquivo que pode ser total ou parcialmente exportado via iSCSI. Os dispositivos compartilhados não devem ser usados para nenhuma outra finalidade neste sistema.

Combinações de dispositivos ( dispositivos no arquivo de configuração)
Isso permite combinar vários dispositivos para compartilhar em um dispositivo maior para exportação. Os dispositivos podem ser unidos no estilo RAID0 para adicionar seus tamanhos, ou no estilo RAID1 espelhado para redundância e desempenho de leitura aprimorado.

Compartilhando destinos ( destino no arquivo de configuração)
Cada entrada nesta lista exporta um dispositivo ou uma combinação de dispositivos para todos os sistemas em uma rede selecionada. Um compartilhamento pode ser somente leitura ou leitura / gravação.

Na parte inferior da página principal do módulo, existem botões para parar, iniciar e reiniciar o servidor iSCSI. As alterações na configuração não serão aplicadas até que o servidor seja reiniciado.
