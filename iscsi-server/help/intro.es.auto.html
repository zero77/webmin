Servidor iSCSI
Este módulo le permite exportar dispositivos de disco a otros sistemas a través de la red utilizando el protocolo iSCSI. La configuración se realiza principalmente mediante la creación de tres tipos de objetos: dispositivos para compartir , combinaciones de dispositivos y objetivos para compartir , que se describen a continuación. También puede editar la configuración que se aplica a todo el servidor iSCSI mediante la página de opciones del servidor iSCSI .

Dispositivos para compartir ( extensiones en el archivo de configuración)
Cada uno de estos es una partición, dispositivo RAID, volumen lógico o archivo que puede exportarse total o parcialmente a través de iSCSI. Los dispositivos compartidos no deben usarse para ningún otro propósito en este sistema.

Combinaciones de dispositivos ( dispositivos en el archivo de configuración)
Estos le permiten combinar múltiples dispositivos para compartir en un dispositivo más grande para la exportación. Los dispositivos se pueden unir en estilo RAID0 para agregar sus tamaños juntos, o en estilo RAID1 reflejado para redundancia y rendimiento de lectura mejorado.

Compartir objetivos ( objetivo en el archivo de configuración)
Cada entrada en esta lista exporta un dispositivo o una combinación de dispositivos a todos los sistemas en una red seleccionada. Un recurso compartido puede ser de solo lectura o de lectura / escritura.

En la parte inferior de la página principal del módulo hay botones para detener, iniciar y reiniciar el servidor iSCSI. Los cambios de configuración no se aplicarán hasta que se reinicie el servidor.
