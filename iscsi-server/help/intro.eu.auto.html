iSCSI zerbitzaria
Modulu honek disko gailuak sareko beste sistemetara esportatzeko aukera ematen du iSCSI protokoloa erabiliz. Konfigurazioa hiru objektu mota sortuz egiten da nagusiki: Partekatzeko gailuak , Gailuen konbinazioak eta Partekatzeko helburuak , jarraian deskribatzen direnak. ISCSI zerbitzari osora aplikatzen diren ezarpenak edita ditzakezu iSCSI Server Aukerak orria erabiliz.

Partekatzeko gailuak (konfigurazio fitxategian hedatuak )
Hauetako bakoitza partizio bat da, RAID gailua, iSCSI bidez osorik edo partzialki esportatu daitekeen bolumen logikoa edo fitxategia. Gailu partekatuak ez dira sistema honetako helburuetarako erabili behar.

Gailuen konbinazioak (konfigurazio fitxategian dauden gailuak )
Gailu anitz konbinatzeko aukera ematen dute esportaziorako gailu handiago batean partekatzeko . Gailuak RAID0 estiloan batu daitezke, tamaina batera gehitzeko, edo erredundantzia hobetzeko eta irakurtzeko errendimendua hobetzeko.

Helburuak partekatzea (konfigurazio fitxategiaren jomuga )
Zerrenda honetako sarrera bakoitzak gailu edo gailu konbinazio bat hautatutako sare bateko sistema guztietara esportatzen du. Akzio bat irakurtzeko edo irakurtzeko / idazteko aukera izan daiteke.

Moduluaren orrialde nagusiaren behealdean iSCSI zerbitzaria gelditzeko, berrabiarazteko botoiak daude. Konfigurazio aldaketak ez dira aplikatuko zerbitzaria berrabiarazi arte.
