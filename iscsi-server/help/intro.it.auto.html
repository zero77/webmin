Server iSCSI
Questo modulo consente di esportare i dispositivi disco su altri sistemi in rete utilizzando il protocollo iSCSI. La configurazione viene eseguita principalmente creando tre tipi di oggetti: Dispositivi da condividere , Combinazioni di dispositivi e Target di condivisione , descritti di seguito. È inoltre possibile modificare le impostazioni applicabili all'intero server iSCSI utilizzando la pagina Opzioni server iSCSI .

Dispositivi da condividere ( estensioni nel file di configurazione)
Ognuno di questi è una partizione, un dispositivo RAID, un volume logico o un file che può essere esportato in tutto o in parte tramite iSCSI. I dispositivi condivisi non devono essere utilizzati per altri scopi su questo sistema.

Combinazioni di dispositivi ( dispositivi nel file di configurazione)
Ciò consente di combinare più dispositivi da condividere in un dispositivo più grande per l'esportazione. I dispositivi possono essere uniti in stile RAID0 per sommare le loro dimensioni, oppure in stile RAID1 con mirroring per ridondanza e prestazioni di lettura migliorate.

Target di condivisione ( target nel file di configurazione)
Ogni voce in questo elenco esporta una combinazione di dispositivi o dispositivi a tutti i sistemi su una rete selezionata. Una condivisione può essere di sola lettura o di lettura / scrittura.

Nella parte inferiore della pagina principale del modulo sono presenti pulsanti per arrestare, avviare e riavviare il server iSCSI. Le modifiche alla configurazione non verranno applicate fino al riavvio del server.
