iSCSI-server
Denne modulen lar deg eksportere diskenheter til andre systemer over nettverket ved hjelp av iSCSI-protokollen. Konfigurasjon skjer primært ved å opprette tre typer objekter - enheter dele, enheter Kombinasjoner og delings Targets, beskrevet nedenfor. Du kan også redigere innstillinger som gjelder for hele iSCSI-serveren ved å bruke iSCSI Server Options- siden.

Enheter som skal deles ( utvides i konfigurasjonsfilen)
Hver av disse er en partisjon, RAID-enhet, logisk volum eller fil som kan eksporteres helt eller delvis via iSCSI. Delte enheter skal ikke brukes til noe annet formål på dette systemet.

Enhetskombinasjoner ( enheter i konfigurasjonsfilen)
Disse lar deg kombinere flere enheter som skal deles til en større enhet for eksport. Enheter kan enten kobles sammen i RAID0-stil for å legge til størrelser sammen, eller speiles RAID1-stil for overflødighet og forbedret leseytelse.

Delingsmål ( mål i konfigurasjonsfilen)
Hver oppføring i denne listen eksporterer enten en enhet eller enhetskombinasjon til alle systemer i et valgt nettverk. En andel kan være skrivebeskyttet eller lese / skrive.

Nederst på modulens hovedside er det knapper for å stoppe, starte og starte iSCSI-serveren på nytt. Konfigurasjonsendringer blir ikke brukt før serveren er startet på nytt.
