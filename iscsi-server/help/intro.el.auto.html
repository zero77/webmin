Διακομιστής iSCSI
Αυτή η ενότητα σάς επιτρέπει να εξάγετε συσκευές δίσκου σε άλλα συστήματα μέσω του δικτύου χρησιμοποιώντας το πρωτόκολλο iSCSI. Η διαμόρφωση γίνεται κυρίως με τη δημιουργία τριών τύπων αντικειμένων - Συσκευές για κοινή χρήση , συνδυασμοί συσκευών και στόχευση κοινής χρήσης , που περιγράφονται παρακάτω. Μπορείτε επίσης να επεξεργαστείτε ρυθμίσεις που ισχύουν για ολόκληρο το διακομιστή iSCSI χρησιμοποιώντας τη σελίδα Επιλογές διακομιστή iSCSI .

Συσκευές για την μετοχή (εκτάσεις στο αρχείο config)
Καθένα από αυτά είναι ένα διαμέρισμα, συσκευή RAID, λογικός τόμος ή αρχείο που μπορούν να εξαχθούν πλήρως ή μερικώς μέσω του iSCSI. Οι κοινόχρηστες συσκευές δεν πρέπει να χρησιμοποιούνται για οποιονδήποτε σκοπό σε αυτό το σύστημα.

Συνδυασμοί συσκευών ( συσκευές στο αρχείο διαμόρφωσης)
Αυτά σας επιτρέπουν να συνδυάσετε πολλές συσκευές για κοινή χρήση σε μια μεγαλύτερη συσκευή για εξαγωγή. Οι συσκευές μπορούν είτε να ενωθούν σε στυλ RAID0 για να προσθέσουν τα μεγέθη τους μαζί, είτε σε στυλ RAID1 για πλεονασμό και βελτιωμένη απόδοση ανάγνωσης.

Κοινή χρήση στόχων ( στόχος στο αρχείο config)
Κάθε καταχώριση σε αυτήν τη λίστα εξάγει είτε συνδυασμό συσκευών είτε συσκευών σε όλα τα συστήματα σε ένα επιλεγμένο δίκτυο. Ένα μερίδιο μπορεί να είναι μόνο για ανάγνωση ή για ανάγνωση / εγγραφή.

Στο κάτω μέρος της κύριας σελίδας της μονάδας υπάρχουν κουμπιά για διακοπή, έναρξη και επανεκκίνηση του διακομιστή iSCSI. Οι αλλαγές διαμόρφωσης δεν θα εφαρμοστούν έως ότου γίνει επανεκκίνηση του διακομιστή.
