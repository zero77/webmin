iSCSI 서버
이 모듈을 사용하면 iSCSI 프로토콜을 사용하여 네트워크를 통해 디스크 장치를 다른 시스템으로 내보낼 수 있습니다. 구성은 주로 공유 할 장치 , 장치 조합 및 공유 대상 의 세 가지 유형의 개체를 생성하여 수행됩니다. iSCSI 서버 옵션 페이지를 사용하여 전체 iSCSI 서버에 적용되는 설정을 편집 할 수도 있습니다.

공유 할 장치 (구성 파일의 범위 )
이들 각각은 iSCSI를 통해 완전히 또는 부분적으로 내보낼 수있는 파티션, RAID 장치, 논리 볼륨 또는 파일입니다. 이 시스템에서 공유 장치를 다른 목적으로 사용해서는 안됩니다.

장치 조합 (설정 파일의 장치 )
이를 통해 여러 장치 를 결합하여 더 큰 장치 로 공유 하여 내보낼 수 있습니다. 장치를 RAID0 스타일로 결합하여 크기를 추가하거나 중복 및 읽기 성능 향상을 위해 미러링 된 RAID1 스타일을 사용할 수 있습니다.

공유 대상 (구성 파일의 대상 )
이 목록의 각 항목은 장치 또는 장치 조합을 선택한 네트워크의 모든 시스템으로 내 보냅니다. 공유는 읽기 전용이거나 읽기 / 쓰기 일 수 있습니다.

모듈의 메인 페이지 하단에는 iSCSI 서버를 중지, 시작 및 재시작하는 버튼이 있습니다. 서버를 다시 시작할 때까지 구성 변경 사항이 적용되지 않습니다.
