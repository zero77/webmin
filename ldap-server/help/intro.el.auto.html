Διακομιστής LDAP
Αυτή η ενότητα σάς επιτρέπει να διαμορφώσετε τον διακομιστή καταλόγου OpenLDAP και να διαχειριστείτε αντικείμενα στη βάση δεδομένων του. Υποθέτοντας ότι έχετε εγκαταστήσει τον διακομιστή LDAP στο ίδιο σύστημα με το Webmin, η κύρια σελίδα θα εμφανίσει εικονίδια για την επεξεργασία της διαμόρφωσης του διακομιστή και τη διαχείριση της βάσης δεδομένων. Εάν η μονάδα έχει ρυθμιστεί ώστε να μιλά σε έναν απομακρυσμένο διακομιστή LDAP, θα εμφανίζει εικονίδια μόνο για πρόσβαση στη βάση δεδομένων.

Μπορείτε να ελέγξετε εάν χρησιμοποιεί έναν τοπικό ή απομακρυσμένο διακομιστή κάνοντας κλικ στο σύνδεσμο Module Config . Μόνο κατά τη διαχείριση ενός τοπικού διακομιστή θα είναι διαθέσιμες σελίδες για επεξεργασία αρχείων διαμόρφωσης και διακοπή, έναρξη και επανεκκίνηση της διαδικασίας διακομιστή OpenLDAP.

Εάν αναζητάτε έναν τρόπο διαχείρισης χρηστών και ομάδων Unix στη βάση δεδομένων LDAP, δοκιμάστε τη λειτουργική μονάδα LDAP Users and Groups στην κατηγορία Σύστημα Webmin. Για να διαμορφώσετε ένα σύστημα για τη λήψη χρηστών και ομάδων από έναν τοπικό ή απομακρυσμένο διακομιστή LDAP, χρησιμοποιήστε τη μονάδα πελάτη LDAP , επίσης στην κατηγορία Σύστημα .
