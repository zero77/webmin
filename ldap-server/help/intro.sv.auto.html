LDAP-server
Den här modulen låter dig konfigurera OpenLDAP-katalogservern och hantera objekt i sin databas. Förutsatt att du har LDAP-servern installerad på samma system som Webmin, kommer startsidan att visa ikoner för att redigera serverkonfigurationen och hantera databasen. Om modulen är konfigurerad att prata med en fjärr LDAP-server kommer den istället bara att visa ikoner för åtkomst till databasen.

Du kan kontrollera om den använder en lokal eller fjärrserver genom att klicka på länken Modulkonfig . Endast när man hanterar en lokal server kommer sidor att vara tillgängliga för att redigera konfigurationsfiler och stoppa, starta och starta om OpenLDAP-serverprocessen.

Om du letar efter ett sätt att hantera Unix-användare och grupper i din LDAP-databas kan du prova LDAP-användare och grupper- modulen i kategorin Webmin System istället. För att konfigurera ett system för att hämta användare och grupper från en lokal eller fjärr LDAP-server använder du LDAP-klientmodulen , även under systemkategorin .
