Gérer le schéma
Le schéma LDAP détermine quelles classes d'objets et quels attributs peuvent être stockés dans votre base de données LDAP. Cette page vous permet de sélectionner les types de schéma pris en charge par votre serveur, en utilisant la case à cocher en regard de chaque nom de fichier de schéma. Une fois les modifications apportées, cliquez sur Enregistrer pour mettre à jour le fichier de configuration OpenLDAP, puis sur Appliquer la configuration sur la page principale pour les activer.

Attention toutefois à désélectionner les fichiers de schéma existants, car cela peut endommager votre serveur LDAP si des objets existent déjà dans la base de données à l'aide des attributs définis dans ces fichiers. Le schéma principal qui contient les classes LDAP les plus élémentaires ne peut pas être désélectionné.

L'ordre des schémas est important, car les schémas ultérieurs ne peuvent se référer qu'à des attributs définis précédemment. Vous pouvez réorganiser la liste en utilisant les flèches haut et bas sur cette page, mais encore une fois, cela doit être fait avec soin pour les schémas existants afin d'éviter de casser la base de données LDAP.

Pour voir quelles classes et attributs un schéma définit, cliquez sur son lien Afficher . Si vous connaissez le format de schéma et souhaitez modifier un fichier, cliquez sur Modifier à la place.
