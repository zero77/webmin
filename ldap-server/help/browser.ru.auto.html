Просмотр базы данных
Эта страница позволяет вам перемещаться по иерархии объектов в вашей базе данных LDAP, создавать и удалять объекты и редактировать их атрибуты. DN для управляемого объекта всегда отображается в поле «Просмотр» - чтобы быстро перейти к другому объекту, введите новый DN и нажмите «Показать» . Чтобы перейти вверх по иерархии, нажмите Browse Parent .

Остальная часть страницы разделена на две вкладки:

Дочерние объекты
В этой таблице перечислены все объекты под текущим. Вы можете редактировать подобъект, нажав на его DN, или удалить объекты, отметив их и нажав кнопку «Удалить выбранных детей» . Чтобы изменить DN подобъекта, щелкните ссылку «Переименовать» и введите новое DN в появившемся поле.

Нажатие на кнопку Добавить новый подобъект вызовет форму для ввода сведений о новом пустом объекте под текущим.

Атрибуты объекта
В этой таблице перечислены все атрибуты текущего объекта. Чтобы изменить значения одного из них, нажмите на ссылку «Изменить» справа. Чтобы удалить атрибуты, установите флажки рядом с ними, а затем нажмите кнопку «Удалить выбранные атрибуты» . В обоих случаях сервер LDAP налагает ограничения на разрешенные и обязательные атрибуты для класса объекта.

Новый атрибут можно создать, щелкнув ссылку Добавить атрибут к объекту , которая отобразит поле для ввода имени и значения.
