OpenLDAP 서버 구성
이 페이지에서는 데이터베이스의 루트 DN, 관리 로그인, 캐시 크기 및 SSL 인증서와 같은 OpenLDAP 서버의 전역 설정을 구성 할 수 있습니다. 가장 일반적으로 변경되는 필드는 다음과 같습니다.
LDAP 데이터베이스의 루트 DN
이 필드는 LDAP 데이터베이스의 모든 객체를 저장해야하는 DN (고유 이름)을 입력하기위한 것입니다. 일반적으로 dc = yourcompany, dc = com 과 같이 회사 또는 조직 이름과 일치하는 것으로 변경해야합니다. 그러나 형식이 올바른 한 아무 것도 입력 할 수 없습니다.

관리 로그인 DN
이것은 사실상 LDAP 데이터베이스의 루트 사용자입니다. cn = Manager, dc = yourcompany, dc = com 과 같이 위의 필드에서 루트 세트 아래의 DN이어야합니다.

관리 비밀번호
이 필드에는 현재 비밀번호 (암호화 가능) 및 새 비밀번호 입력 옵션이 표시됩니다. 암호를 변경하면 데이터베이스에 연결할 때 모듈에서 암호를 자동으로 사용하여 변경합니다.

이 페이지에서 변경 한 사항은 모듈의 메인 페이지에서 Apply Configuration (구성 적용) 버튼을 클릭 할 때까지 적용 되지 않습니다.

LDAP 서버가 TLS 암호화 연결을 승인하도록하려면 먼저 SSL 인증서와 공개 키를 생성해야합니다. 페이지 하단의 SSL 인증서 생성 버튼을 클릭하면 대부분 자동화 할 수 있습니다.
