Server LDAP
Acest modul vă permite să configurați serverul de director OpenLDAP și să gestionați obiectele din baza de date a acestuia. Presupunând că aveți serverul LDAP instalat pe același sistem ca Webmin, pagina principală va afișa pictograme pentru editarea configurației serverului și gestionarea bazei de date. Dacă modulul este configurat pentru a vorbi cu un server LDAP de la distanță, în schimb, acesta va afișa doar pictograme pentru accesarea bazei de date.

Puteți controla dacă folosește un server local sau de la distanță făcând clic pe linkul Configurare modul . Doar atunci când gestionați un server local, paginile vor fi disponibile pentru a edita fișierele de configurare și a opri, porni și reporni procesul de server OpenLDAP.

Dacă sunteți în căutarea unei modalități de gestionare a utilizatorilor și grupurilor Unix din baza de date LDAP, încercați în schimb modulul Utilizatori și grupuri LDAP din categoria Sistem Webmin. Pentru a configura un sistem pentru a prelua utilizatorii și grupurile de la un server LDAP local sau la distanță, utilizați modulul Client LDAP , de asemenea, sub categoria Sistem .
