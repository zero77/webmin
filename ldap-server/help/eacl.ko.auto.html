액세스 제어 규칙
이 페이지에서 단일 액세스 제어 규칙을 편집하거나 작성할 수 있습니다. 부여되는 첫 번째 섹션은 규칙이 적용될 데이터베이스의 DN을 결정합니다. 전체 데이터베이스에 대해 모든 개체 를 선택하거나 인접한 필드에 입력 한 내용과 일치하는 개체에 대해 DN이 있는 개체 를 선택할 수 있습니다.

일치 유형 메뉴는 규칙이이 개체에만 적용되는지, 그 아래에있는 개체에 적용되는지 또는 DN이 정규식으로 처리되는지를 결정합니다. (objectClass = posixAccount) 와 같이 객체 필터 제한 필드에 LDAP 필터를 입력하여 부여 할 객체를 추가로 제어 할 수 있습니다.

이 페이지의 두 번째 부분은 객체에 액세스 할 수있는 LDAP 사용자를 선택하기위한 표입니다. 액세스 권한 부여 메뉴에서 여러 일반 사용자 클래스를 선택하거나 기타를 선택하고 특정 DN을 입력 할 수 있습니다.

액세스 수준 메뉴는 이러한 사용자가 개체로 수행 할 수있는 작업을 결정합니다. 더 강력한 수준이 낮을수록 위에있는 모든 수준을 의미하므로 쓰기 권한이있는 사용자는 읽기 및 검색 도 수행 할 수 있습니다.
