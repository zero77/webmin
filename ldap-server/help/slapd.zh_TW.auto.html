OpenLDAP服務器配置
該頁面允許您配置OpenLDAP服務器的全局設置，例如數據庫的根DN，管理登錄名，緩存大小和SSL證書。最常更改的字段是：
LDAP數據庫的根DN
此字段用於輸入DN（專有名稱），必須在其下存儲LDAP數據庫中的所有對象。通常，應將其更改為與您的公司或組織名稱匹配的名稱，例如dc = yourcompany，dc = com 。但是，只要格式正確，您就可以自由輸入。

管理登錄DN
這實際上是LDAP數據庫的root用戶。它應該是上述字段中您的根目錄下的DN，例如cn = Manager，dc = yourcompany，dc = com 。

管理密碼
此字段顯示當前密碼（可能已加密），以及輸入新密碼的選項。如果更改密碼，則模塊將在連接到數據庫進行更改時自動使用它。

在模塊主頁面上單擊“ 應用配置”按鈕之前，對該頁面所做的任何更改都不會生效。

如果希望LDAP服務器接受TLS加密的連接，則必須首先生成SSL證書和公共密鑰。通過單擊頁面底部的“ 生成SSL證書”按鈕，大部分操作可以自動進行。
