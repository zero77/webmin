การกำหนดค่าเซิร์ฟเวอร์ OpenLDAP
หน้านี้ช่วยให้คุณกำหนดการตั้งค่าส่วนกลางสำหรับเซิร์ฟเวอร์ OpenLDAP ของคุณเช่นรูท DN สำหรับฐานข้อมูลล็อกอินการบริหารขนาดแคชและใบรับรอง SSL ฟิลด์ที่มีการเปลี่ยนแปลงบ่อยที่สุดคือ:
รูท DN สำหรับฐานข้อมูล LDAP
ฟิลด์นี้ใช้สำหรับป้อน DN (ชื่อจำเพาะ) ภายใต้วัตถุทั้งหมดในฐานข้อมูล LDAP ของคุณที่ต้องจัดเก็บ โดยทั่วไปควรเปลี่ยนเป็นสิ่งที่ตรงกับชื่อ บริษัท หรือองค์กรของคุณเช่น dc = yourcompany, dc = com แต่คุณสามารถป้อนอะไรก็ได้ตราบใดที่มันมีรูปแบบที่ถูกต้อง

การบริหารล็อกอิน DN
นี่เป็นผู้ใช้ รูท สำหรับฐานข้อมูล LDAP ของคุณอย่างมีประสิทธิภาพ ควรเป็น DN ภายใต้รูทที่ตั้งไว้ในฟิลด์ด้านบนเช่น cn = Manager, dc = yourcompany, dc = com

รหัสผ่านการบริหาร
ฟิลด์นี้แสดงรหัสผ่านปัจจุบัน (อาจเข้ารหัส) และตัวเลือกในการป้อนใหม่ หากคุณเปลี่ยนรหัสผ่านโมดูลจะใช้โดยอัตโนมัติเมื่อเชื่อมต่อกับฐานข้อมูลเพื่อทำการเปลี่ยนแปลง

การเปลี่ยนแปลงใด ๆ ที่ทำในหน้านี้จะไม่มีผลจนกว่าจะมีการคลิกปุ่ม ใช้การกำหนดค่า บนหน้าหลักของโมดูล

หากคุณต้องการให้เซิร์ฟเวอร์ LDAP ยอมรับการเชื่อมต่อที่เข้ารหัส TLS คุณต้องสร้างใบรับรอง SSL และรหัสสาธารณะก่อน สิ่งนี้สามารถทำงานอัตโนมัติเป็นส่วนใหญ่โดยคลิกที่ปุ่ม สร้างใบรับรอง SSL ที่ด้านล่างของหน้า
