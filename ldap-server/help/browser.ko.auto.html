데이터베이스 찾아보기
이 페이지에서 LDAP 데이터베이스의 객체 계층 구조를 탐색하고 객체를 생성 및 제거하고 속성을 편집 할 수 있습니다. 관리중인 개체의 DN은 항상 찾아보기 필드에 표시 됩니다. 다른 개체로 빠르게 이동하려면 새 DN을 입력하고 표시를 클릭하십시오. 계층 구조를 위로 이동하려면 부모 찾아보기를 클릭하십시오.

나머지 페이지는 두 개의 탭으로 나뉩니다.

자식 개체
이 테이블에는 현재 오브젝트 아래의 모든 오브젝트가 나열됩니다. 하위 개체의 DN을 클릭하여 하위 개체를 편집하거나 개체를 확인하고 선택한 하위 항목 제거 버튼을 눌러 개체를 삭제할 수 있습니다. 하위 오브젝트의 DN을 변경하려면 이름 바꾸기 링크를 클릭하고 나타나는 필드에 새 DN을 입력하십시오.

새 하위 오브젝트 추가를 클릭하면 현재 비어있는 새 오브젝트의 세부 사항을 입력 할 수있는 양식이 나타납니다.

객체 속성
이 테이블은 현재 객체의 모든 속성을 나열합니다. 하나의 값을 편집하려면 오른쪽의 편집 링크를 클릭하십시오. 속성을 삭제하려면 옆에있는 확인란을 선택한 다음 선택한 속성 제거 버튼을 클릭하십시오. 두 경우 모두 LDAP 서버는 객체 클래스의 허용 및 필수 속성에 대한 제한을 시행합니다.

객체에 속성 추가 링크 를 클릭하여 새 속성을 작성할 수 있습니다. 이름 및 값을 입력하기위한 필드가 표시됩니다.
