Eskema kudeatu
LDAP eskemak zehazten du zein objektu klase eta atributu gorde daitezkeen zure LDAP datu-basean. Orrialde honek eskema-fitxategiaren izenaren ondoko kontrol-laukia erabilita zure zerbitzariak onartzen ditu aukeratzeko aukera ematen du. Aldaketak egin ondoren, egin klik Gorde Gorde OpenLDAP konfigurazio fitxategia eguneratzeko eta, ondoren, Aplikatu konfigurazioa orri nagusian, horiek aktibatzeko.

Kontuz ibili eskema fitxategiak hautatzen badituzu, izan ere, zure LDAP zerbitzaria hautsi dezake datu basean dagoeneko datu fitxategiak zehaztutako atributuak erabiliz. The core eskema horrek LDAP oinarrizko klaseak gehien dauka ezin izan de-aukeratu ditu.

Eskemak ordenatzea garrantzitsua da, gero eskemak aurretik zehaztutako atributuei baino ezin diezaieketen erreferentzia egin. Zerrenda eska dezakezu orrialde honetako gora eta behera geziak erabiliz, baina berriro egin beharko da lehendik dauden eskemak arretaz egin ahal izateko, LDAP datu-basea hautsi ez dadin.

Eskema batek definitzen dituen zer nolako klaseak eta atributuak ikusteko, egin klik Ikusteko estekan. Eskemaren formatua ezagutzen baduzu eta fitxategi bat editatu nahi baduzu, egin klik Editatu ordez.
