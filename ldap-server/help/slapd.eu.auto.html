OpenLDAP zerbitzariaren konfigurazioa
Orrialde honek zure OpenLDAP zerbitzariaren ezarpen orokorrak konfiguratu ahal izango dituzu, esaterako, datu baseko root helbidea, administrazio saioa, cache neurriak eta SSL ziurtagiriak. Gehien aldatutako eremuak hauek dira:
LDAP datu-basea sustraitzeko DN
Eremu hau zure LDAP datu-baseko objektu guztiak gorde behar diren DN (izen bereizgarria) sartzeko da. Normalean, zure enpresaren edo erakundearen izenarekin bat datorren zerbait aldatu beharko litzateke, dc = yourcompany, dc = com bezala . Baina askea zara beti formateatuta dagoen bitartean.

Administrazioaren saioa DN
Hau da LDAP datu-baseko root erabiltzailea. Goiko eremuan kokatutako erroaren azpian DN bat izan beharko luke, cn = Manager bezala, dc = yourcompany, dc = com .

Administrazioko pasahitza
Eremu honetan uneko pasahitza (ziurrenik enkriptatuta) eta berri bat sartzeko aukera erakusten da. Pasahitza aldatzen baduzu, moduluak automatikoki erabiliko du datu-basera konektatzean aldaketak egiteko.

Orrialde honetan eginiko aldaketek ez dute eragina izango Aplikatu konfigurazioa botoiaren moduluaren orrialde nagusian klik egin arte.

Zure LDAP zerbitzariak TLS enkriptatutako konexioak onartu nahi badituzu, lehendabizi SSL ziurtagiria eta gako publikoa sortu behar dituzu. Gehienean automatizatu daiteke orriaren behealdean Sortu SSL ziurtagiria botoian klik eginez.
