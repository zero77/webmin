Servidor LDAP
Este módulo permite configurar o servidor de diretório OpenLDAP e gerenciar objetos em seu banco de dados. Supondo que você tenha o servidor LDAP instalado no mesmo sistema que o Webmin, a página principal mostrará ícones para editar a configuração do servidor e gerenciar o banco de dados. Se o módulo estiver configurado para conversar com um servidor LDAP remoto, ele exibirá apenas ícones para acessar o banco de dados.

Você pode controlar se ele usa um servidor local ou remoto clicando no link Configuração do módulo . Somente ao gerenciar um servidor local, as páginas estarão disponíveis para editar arquivos de configuração e parar, iniciar e reiniciar o processo do servidor OpenLDAP.

Se você estiver procurando uma maneira de gerenciar usuários e grupos do Unix em seu banco de dados LDAP, tente o módulo Usuários e Grupos LDAP na categoria Sistema Webmin. Para configurar um sistema para buscar usuários e grupos de um servidor LDAP local ou remoto, use o módulo Cliente LDAP , também na categoria Sistema .
