浏览数据库
该页面允许您浏览LDAP数据库中对象的层次结构，创建和删除对象以及编辑它们的属性。被管理对象的DN始终显示在“ 浏览”字段中-要快速导航到另一个对象，请输入新的DN并单击Show 。要上移层次结构，请点击浏览父级 。

该页面的其余部分分为两个标签：

子对象
该表列出了当前对象下的所有对象。您可以通过单击子目录的DN来编辑子对象，也可以通过选中子对象并单击“ 删除所选子项”按钮来删除它们。要更改子对象的DN，请单击“ 重命名”链接，然后在出现的字段中输入新的DN。

单击添加一个新的子对象将弹出一个表单，用于在当前对象下输入一个新的空对象的详细信息。

对象属性
该表列出了当前对象的所有属性。要编辑1的值，请单击右侧的Edit链接。要删除属性，请选中它们旁边的框，然后单击“ 删除所选属性”按钮。在这两种情况下，LDAP服务器都会对对象类的允许和必需属性实施限制。

单击“ 将属性添加到对象”链接可以创建一个新属性，该链接将显示用于输入名称和值的字段。
