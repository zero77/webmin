LDAP-Server
Mit diesem Modul können Sie den OpenLDAP-Verzeichnisserver konfigurieren und Objekte in seiner Datenbank verwalten. Angenommen, Sie haben den LDAP-Server auf demselben System wie Webmin installiert, werden auf der Hauptseite Symbole zum Bearbeiten der Serverkonfiguration und zum Verwalten der Datenbank angezeigt. Wenn das Modul für die Kommunikation mit einem Remote-LDAP-Server konfiguriert ist, werden stattdessen nur Symbole für den Zugriff auf die Datenbank angezeigt.

Sie können steuern, ob ein lokaler oder ein Remote-Server verwendet wird, indem Sie auf den Link Module Config klicken. Nur bei der Verwaltung eines lokalen Servers stehen Seiten zum Bearbeiten von Konfigurationsdateien und zum Stoppen, Starten und Neustarten des OpenLDAP-Serverprozesses zur Verfügung.

Wenn Sie nach einer Möglichkeit suchen, Unix-Benutzer und -Gruppen in Ihrer LDAP-Datenbank zu verwalten, versuchen Sie stattdessen das Modul LDAP-Benutzer und -Gruppen unter der Kategorie Webmin- System . Um ein System zu konfigurieren , dass Benutzer und Gruppen aus einem lokalen oder Remote - LDAP - Server abzurufen, verwenden Sie den LDAP - Client - Modul, auch unter der Kategorie - System.
