Hantera schema
LDAP-schemat avgör vilka objektklasser och attribut som kan lagras i din LDAP-databas. På den här sidan kan du välja vilka schematyper som stöds av din server genom att använda kryssrutan bredvid varje schemafilnamn. När du har gjort ändringar klickar du på Spara för att uppdatera OpenLDAP-konfigurationsfilen och sedan Använd konfigurering på startsidan för att aktivera dem.

Var dock försiktig med att välja befintliga schemafiler, eftersom det kan bryta din LDAP-server om det redan finns objekt i databasen med attributen som definieras i dessa filer. Kärn schema som innehåller de mest grundläggande LDAP klasser kan inte vara de utvalda.

Det är viktigt att beställa scheman, eftersom senare scheman bara kan hänvisa till attribut som har definierats tidigare. Du kan ordna listan med hjälp av upp- och nedpilarna på denna sida, men återigen bör detta göras med omsorg för befintliga scheman för att undvika att LDAP-databasen bryts.

För att se vilka klasser och attribut som ett schema definierar klickar du på länken Visa . Om du är bekant med schemaformatet och vill redigera en fil klickar du på Redigera istället.
