LDAP zerbitzaria
Modulu honek OpenLDAP direktorio-zerbitzaria konfiguratu eta bere datu baseko objektuak kudeatu ahal izango dituzu. Webminen sistema berean LDAP zerbitzaria instalatuta duzula uste baduzu, orri nagusiak zerbitzariaren konfigurazioa editatzeko eta datu-basea kudeatzeko ikonoak erakutsiko ditu. Modulua urruneko LDAP zerbitzari batekin hitz egiteko konfiguratuta badago, datu basean sartzeko ikonoak baino ez ditu agertuko.

Tokiko edo urruneko zerbitzaria erabiltzen duen ala ez kontrolatu dezakezu Moduluen konfigurazio estekan klik eginez. Tokiko zerbitzari bat kudeatzean bakarrik egongo dira orriak konfigurazio fitxategiak editatzeko eta OpenLDAP zerbitzariaren prozesua gelditzeko, hasteko eta berrabiarazteko.

Zure LDAP datu-basean Unix erabiltzaileak eta taldeak kudeatzeko modua bilatzen baduzu, saiatu LDAP erabiltzaileak eta taldeak modulua Webmin System kategorian azpian. Tokiko edo urruneko LDAP zerbitzari batetik erabiltzaileak eta taldeak biltzeko sistema konfiguratzeko, erabili LDAP Bezeroaren modulua ere, Sistema kategorian.
