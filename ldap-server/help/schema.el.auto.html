Διαχείριση σχήματος
Το σχήμα LDAP καθορίζει ποιες κατηγορίες αντικειμένων και χαρακτηριστικά μπορούν να αποθηκευτούν στη βάση δεδομένων LDAP. Αυτή η σελίδα σάς επιτρέπει να επιλέξετε ποιοι τύποι σχήματος υποστηρίζονται από τον διακομιστή σας, χρησιμοποιώντας το πλαίσιο ελέγχου δίπλα σε κάθε όνομα αρχείου σχήματος. Μόλις πραγματοποιήσετε αλλαγές, κάντε κλικ στην επιλογή Αποθήκευση για να ενημερώσετε το αρχείο διαμόρφωσης OpenLDAP και, στη συνέχεια, Εφαρμογή διαμόρφωσης στην κύρια σελίδα για να τις ενεργοποιήσετε.

Προσέξτε, ωστόσο, να αποεπιλέξετε τα υπάρχοντα αρχεία σχήματος, καθώς αυτό μπορεί να καταστρέψει τον διακομιστή LDAP εάν υπάρχουν ήδη αντικείμενα στη βάση δεδομένων χρησιμοποιώντας τα χαρακτηριστικά που ορίζονται σε αυτά τα αρχεία. Το βασικό σχήμα που περιέχει τις πιο βασικές τάξεις LDAP δεν μπορεί να αποεπιλεχθεί.

Η σειρά των σχημάτων είναι σημαντική, καθώς τα μεταγενέστερα σχήματα μπορούν να αναφέρονται μόνο σε χαρακτηριστικά που ορίστηκαν προηγουμένως. Μπορείτε να αναδιατάξετε τη λίστα χρησιμοποιώντας τα πάνω και κάτω βέλη σε αυτήν τη σελίδα, αλλά και πάλι αυτό πρέπει να γίνει με προσοχή για τα υπάρχοντα σχήματα, προκειμένου να αποφευχθεί η καταστροφή της βάσης δεδομένων LDAP.

Για να δείτε ποιες κατηγορίες και χαρακτηριστικά ορίζει ένα σχήμα, κάντε κλικ στον σύνδεσμο Προβολή . Εάν είστε εξοικειωμένοι με τη μορφή σχήματος και θέλετε να επεξεργαστείτε ένα αρχείο, κάντε κλικ στην επιλογή Επεξεργασία .
