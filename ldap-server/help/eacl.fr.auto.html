Règle de contrôle d'accès
Cette page vous permet de modifier ou de créer une seule règle de contrôle d'accès. La première section Objets accordés détermine à quels DN de votre base de données la règle s'appliquera. Vous pouvez sélectionner Tous les objets pour la base de données entière ou Objet avec DN pour les objets correspondant à tout ce que vous entrez dans le champ adjacent.

Le menu du type de correspondance détermine si la règle s'applique uniquement à cet objet, à ceux qui se trouvent en dessous, ou si le DN est traité comme une expression régulière. Vous pouvez également contrôler les objets qui sont accordés en entrant un filtre LDAP dans le champ Limiter avec le filtre d'objet , comme (objectClass = posixAccount) .

La deuxième partie de la page est un tableau permettant de sélectionner les utilisateurs LDAP ayant accès aux objets. Vous pouvez soit sélectionner plusieurs classes d'utilisateurs générales dans le menu Accorder l'accès à , soit choisir Autre et saisir un DN spécifique.

Le menu Niveau d'accès détermine ce que ces utilisateurs peuvent faire avec les objets. Les niveaux inférieurs les plus puissants impliquent tous ceux au-dessus d'eux, donc un utilisateur avec un accès en écriture peut également lire et rechercher .
