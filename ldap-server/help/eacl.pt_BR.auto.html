Regra de controle de acesso
Esta página permite editar ou criar uma única regra de controle de acesso. A primeira seção Objetos concedidos determina a quais DNs em seu banco de dados a regra se aplicará. Você pode selecionar Todos os objetos para todo o banco de dados ou Objeto com DN para objetos que correspondam ao que você inserir no campo adjacente.

O menu do tipo de correspondência determina se a regra se aplica apenas a esse objeto, àqueles abaixo dele, ou se o DN é tratado como uma expressão regular. Você também pode controlar ainda mais quais objetos são concedidos inserindo um filtro LDAP no campo Limitar com filtro de objetos , como (objectClass = posixAccount) .

A segunda parte da página é uma tabela para selecionar quais usuários LDAP têm acesso aos objetos. Você pode selecionar várias classes de usuário gerais no menu Conceder acesso a , ou escolher Outro e inserir um DN específico.

O menu Nível de acesso determina o que esses usuários podem fazer com os objetos. Os níveis mais baixos e poderosos implicam todos os que estão acima deles; portanto, um usuário com acesso de gravação também pode ler e pesquisar .
