LDAP 서버
이 모듈을 사용하면 OpenLDAP 디렉토리 서버를 구성하고 데이터베이스에서 객체를 관리 할 수 있습니다. Webmin과 동일한 시스템에 LDAP 서버가 설치되어 있다고 가정하면 기본 페이지에 서버 구성 편집 및 데이터베이스 관리에 대한 아이콘이 표시됩니다. 모듈이 원격 LDAP 서버와 통신하도록 구성된 경우 대신 데이터베이스에 액세스하기위한 아이콘 만 표시됩니다.

모듈 구성 링크를 클릭하여 로컬 또는 원격 서버를 사용할지 여부를 제어 할 수 있습니다. 로컬 서버를 관리 할 때만 구성 파일을 편집하고 OpenLDAP 서버 프로세스를 중지, 시작 및 다시 시작하는 페이지를 사용할 수 있습니다.

LDAP 데이터베이스에서 Unix 사용자 및 그룹을 관리하는 방법을 찾고 있다면 Webmin 시스템 카테고리에서 LDAP 사용자 및 그룹 모듈을 사용해보십시오. 로컬 또는 원격 LDAP 서버에서 사용자 및 그룹을 가져 오도록 시스템을 구성하려면 시스템 카테고리에서도 LDAP 클라이언트 모듈을 사용하십시오.
