Administrer skjema
LDAP-skjemaet bestemmer hvilke objektklasser og attributter som kan lagres i LDAP-databasen. På denne siden kan du velge hvilke skjematyper som støttes av serveren din, ved å bruke avkrysningsruten ved siden av hvert skjemafilnavn. Når du har gjort endringer, klikker du på Lagre for å oppdatere OpenLDAP-konfigurasjonsfilen, og deretter bruke konfigurasjon på hovedsiden for å aktivere dem.

Vær forsiktig med å fjerne eksisterende skjemafiler, da dette kan ødelegge LDAP-serveren hvis det allerede finnes objekter i databasen ved å bruke attributtene som er definert i disse filene. Kjernen skjema som inneholder de mest grunnleggende LDAP-klasser kan ikke være de valgt.

Bestilling av skjemaer er viktig, siden senere skjemaer bare kan referere til attributter som er definert tidligere. Du kan bestille rekkefølgen på nytt ved å bruke pil opp og pil ned på denne siden, men igjen bør dette gjøres med omhu for eksisterende skjemaer for å unngå å bryte LDAP-databasen.

For å se hvilke klasser og attributter et skjema definerer, klikker du på koblingen Vis . Hvis du er kjent med skjemaformatet og ønsker å redigere en fil, klikker du på Rediger i stedet.
