เซิร์ฟเวอร์ LDAP
โมดูลนี้ช่วยให้คุณสามารถกำหนดค่าเซิร์ฟเวอร์ไดเรกทอรี OpenLDAP และจัดการวัตถุในฐานข้อมูล สมมติว่าคุณมีเซิร์ฟเวอร์ LDAP ติดตั้งอยู่ในระบบเดียวกับ Webmin หน้าหลักจะแสดงไอคอนสำหรับการแก้ไขการกำหนดค่าเซิร์ฟเวอร์และจัดการฐานข้อมูล หากโมดูลได้รับการกำหนดค่าให้พูดคุยกับเซิร์ฟเวอร์ LDAP ระยะไกลโมดูลจะแสดงเฉพาะไอคอนสำหรับการเข้าถึงฐานข้อมูล

คุณสามารถควบคุมได้ว่าจะใช้เซิร์ฟเวอร์ภายในหรือระยะไกลโดยคลิกที่ลิงค์ โมดูลการ กำหนดค่า เฉพาะเมื่อจัดการเซิร์ฟเวอร์ภายในหน้าจะพร้อมใช้งานเพื่อแก้ไขไฟล์การกำหนดค่าและหยุดให้เริ่มและรีสตาร์ทกระบวนการเซิร์ฟเวอร์ OpenLDAP

หากคุณกำลังมองหาวิธีจัดการผู้ใช้และกลุ่ม Unix ในฐานข้อมูล LDAP ของคุณลอง ใช้ โมดูล ผู้ใช้และกลุ่ม LDAP ภายใต้หมวดหมู่ ระบบ Webmin แทน ในการกำหนดค่าระบบเพื่อดึงข้อมูลผู้ใช้และกลุ่มจากเซิร์ฟเวอร์ LDAP ในตัวหรือระยะไกลให้ใช้โมดูล ไคลเอ็นต์ LDAP เช่นกันในหมวดหมู่ ระบบ
