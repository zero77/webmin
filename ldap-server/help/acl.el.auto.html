Έλεγχος πρόσβασης LDAP
Από προεπιλογή, ένας διακομιστής LDAP επιτρέπει σε κάθε πελάτη που μπορεί να συνδεθεί για να διαβάσει όλα τα αντικείμενα και τα χαρακτηριστικά στη βάση δεδομένων. Ωστόσο, μόνο ο διαχειριστής χρήστης μπορεί να πραγματοποιήσει ενημερώσεις. Αυτό μπορεί να μην είναι ιδανικό για δίκτυα με χρήστες με διαφορετικά επίπεδα εμπιστοσύνης, επομένως το OpenLDAP σάς επιτρέπει να παραχωρείτε διαφορετικά επίπεδα πρόσβασης σε διαφορετικούς χρήστες σε διαφορετικά μέρη της βάσης δεδομένων.

Σε αυτήν τη σελίδα αναφέρονται όλοι οι κανόνες ελέγχου πρόσβασης που έχουν καθοριστεί αυτήν τη στιγμή, εάν υπάρχουν. Για να δημιουργήσετε ένα νέο, κάντε κλικ στο σύνδεσμο Προσθήκη νέου κανόνα ελέγχου πρόσβασης . Για να καταργήσετε πολλούς κανόνες ταυτόχρονα, επιλέξτε τα πλαίσια δίπλα τους και πατήστε το κουμπί Διαγραφή επιλεγμένων κανόνων . Για να αλλάξετε τη σειρά των κανόνων, χρησιμοποιήστε τα πάνω και κάτω βέλη στη δεξιά πλευρά του πίνακα.

Όπως συμβαίνει με τις περισσότερες άλλες αλλαγές διαμόρφωσης διακομιστή LDAP, οι κανόνες ελέγχου πρόσβασης δεν θα τεθούν σε ισχύ έως ότου πατηθεί το κουμπί Εφαρμογή διαμόρφωσης στην κύρια σελίδα της μονάδας.
