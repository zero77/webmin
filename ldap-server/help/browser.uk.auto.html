Огляд бази даних
Ця сторінка дозволяє орієнтуватися на спадкоємність об’єктів у вашій базі даних LDAP, створювати та видаляти об'єкти та редагувати їх атрибути. DN для об'єкта, яким керують, завжди відображається в полі Перегляд - для швидкого переходу до іншого об’єкта, введіть новий DN та натисніть Показати . Для переміщення вгору по спадковості натисніть кнопку Огляд батьків .

Решта сторінки розділена на дві вкладки:

Предмети дитини
У цій таблиці перераховані всі об'єкти під поточною. Ви можете відредагувати суб'єкт, натиснувши його DN, або видалити об'єкти, перевіривши їх та натиснувши кнопку Видалити вибраних дітей . Щоб змінити DN суб'єкта, натисніть на посилання Перейменувати та введіть новий DN у поле, яке з’явиться.

Клацання по кнопці Додати новий суб'єкт відкриє форму для введення реквізитів нового порожнього об'єкта під поточним.

Атрибути об’єкта
У цій таблиці перераховані всі атрибути поточного об'єкта. Щоб відредагувати значення одиниці, натисніть посилання Правка праворуч. Щоб видалити атрибути, поставте прапорець біля них та натисніть кнопку Видалити вибрані атрибути . В обох випадках сервер LDAP буде застосовувати обмеження на дозволені та необхідні атрибути для класу об'єкта.

Новий атрибут можна створити, натиснувши посилання Додати атрибут до об’єкта , де відображатиметься поле для введення імені та значення.
