จัดการสคีมา
LDAP schema กำหนดคลาสอ็อบเจ็กต์และแอ็ตทริบิวต์ที่สามารถเก็บในฐานข้อมูล LDAP ของคุณ หน้านี้ช่วยให้คุณสามารถเลือกประเภทสคีมาที่เซิร์ฟเวอร์ของคุณรองรับโดยใช้ช่องทำเครื่องหมายถัดจากชื่อไฟล์สคีมาแต่ละรายการ เมื่อคุณทำการเปลี่ยนแปลงแล้วให้คลิก บันทึก เพื่ออัปเดตไฟล์กำหนดค่า OpenLDAP จากนั้น ใช้การกำหนดค่า ในหน้าหลักเพื่อเปิดใช้งาน

ระวังการเลือกไฟล์ schema ที่มีอยู่เนื่องจากอาจทำให้เซิร์ฟเวอร์ LDAP ของคุณเสียหายหากมีวัตถุอยู่ในฐานข้อมูลโดยใช้แอตทริบิวต์ที่กำหนดไว้ในไฟล์เหล่านั้น คีมา หลัก ที่มีคลาส LDAP พื้นฐานที่สุดไม่สามารถถูกยกเลิกการเลือกได้

การจัดลำดับสกีมามีความสำคัญเนื่องจากสกีมาในภายหลังสามารถอ้างถึงคุณลักษณะที่กำหนดไว้ก่อนหน้านี้เท่านั้น คุณสามารถจัดลำดับรายการใหม่ได้โดยใช้ลูกศรขึ้นและลงในหน้านี้ แต่ควรทำอีกครั้งด้วยความระมัดระวังสำหรับสกีมาที่มีอยู่เพื่อหลีกเลี่ยงการทำลายฐานข้อมูล LDAP

หากต้องการดูคลาสและแอตทริบิวต์ที่สคีมากำหนดให้คลิกที่ลิงค์ ดู หากคุณคุ้นเคยกับรูปแบบสคีมาและต้องการแก้ไขไฟล์ให้คลิก แก้ไข แทน
