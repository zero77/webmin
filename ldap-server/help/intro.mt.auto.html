Server LDAP
Dan il-modulu jippermettilek li tikkonfigura s-server tad-direttorju OpenLDAP, u timmaniġġja l-oġġetti fid-database tagħha. Jekk nassumu li għandek is-server LDAP installat fl-istess sistema bħal Webmin, il-paġna ewlenija se turi ikoni għall-editjar tal-konfigurazzjoni tas-server u l-immaniġġjar tad-database. Jekk il-modulu huwa kkonfigurat biex jitkellem ma 'server LDAP remot, minflok dan juri biss ikoni għall-aċċess tad-database.

Tista 'tikkontrolla jekk jużax server lokali jew remot billi tikklikkja fuq il-link Module Config . Biss meta tkun qed timmaniġġja server lokali, il-paġni jkunu disponibbli biex jeditjaw il-fajls tal-konfigurazzjoni u jieqfu, jibdew u jerġgħu jibdew il-proċess tas-server OpenLDAP.

Jekk qed tfittex mod kif timmaniġġja utenti u gruppi Unix fid-database LDAP tiegħek, ipprova l-modulu tal- Utenti u Gruppi LDAP taħt il-kategorija tas-Sistema Webmin minflok. Biex tikkonfigura sistema biex iġibu utenti u gruppi minn server LDAP lokali jew remot, uża l-modulu tal- Klijent LDAP , ukoll taħt il-kategorija tas - Sistema .
