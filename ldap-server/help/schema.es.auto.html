Gestionar esquema
El esquema LDAP determina qué clases de objetos y atributos se pueden almacenar en su base de datos LDAP. Esta página le permite seleccionar qué tipos de esquema son compatibles con su servidor, utilizando la casilla de verificación junto a cada nombre de archivo de esquema. Una vez que haya realizado los cambios, haga clic en Guardar para actualizar el archivo de configuración de OpenLDAP, luego en Aplicar configuración en la página principal para activarlos.

Sin embargo, tenga cuidado de deseleccionar los archivos de esquema existentes, ya que esto puede romper su servidor LDAP si ya existen objetos en la base de datos utilizando los atributos definidos en esos archivos. El esquema central que contiene las clases LDAP más básicas no se puede deseleccionar.

El orden de los esquemas es importante, ya que los esquemas posteriores solo pueden referirse a los atributos definidos previamente. Puede reordenar la lista utilizando las flechas hacia arriba y hacia abajo en esta página, pero nuevamente esto debe hacerse con cuidado para los esquemas existentes para evitar romper la base de datos LDAP.

Para ver qué clases y atributos define un esquema, haga clic en el enlace Ver . Si está familiarizado con el formato de esquema y desea editar un archivo, haga clic en Editar en su lugar.
