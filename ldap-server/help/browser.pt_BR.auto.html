Procurar banco de dados
Esta página permite navegar pela hierarquia de objetos no banco de dados LDAP, criar e remover objetos e editar seus atributos. O DN do objeto que está sendo gerenciado é sempre mostrado no campo Navegação - para navegar rapidamente para outro objeto, insira um novo DN e clique em Mostrar . Para subir na hierarquia, clique em Browse Parent .

O restante da página é dividido em duas guias:

Objetos filhos
Esta tabela lista todos os objetos abaixo do atual. Você pode editar um subobjeto clicando em seu DN ou excluí-los, verificando-os e pressionando o botão Remover filhos selecionados. Para alterar o DN de um subobjeto, clique no link Renomear e insira um novo DN no campo que aparece.

Clicar em Adicionar um novo subobjeto exibirá um formulário para inserir os detalhes de um novo objeto vazio abaixo do atual.

Atributos do objeto
Esta tabela lista todos os atributos do objeto atual. Para editar os valores de um, clique no link Editar à direita. Para excluir atributos, marque as caixas ao lado deles e clique no botão Remover atributos selecionados. Nos dois casos, o servidor LDAP aplicará restrições nos atributos permitidos e necessários para a classe do objeto.

Um novo atributo pode ser criado clicando no link Adicionar atributo ao objeto , que exibirá o campo para inserir um nome e um valor.
