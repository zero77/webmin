Controle de Acesso LDAP
Por padrão, um servidor LDAP permite que qualquer cliente que possa se conectar leia todos os objetos e atributos no banco de dados. No entanto, apenas o usuário administrativo pode executar atualizações. Porém, isso pode não ser ideal para redes que possuem usuários com diferentes níveis de confiança; portanto, o OpenLDAP permite conceder níveis de acesso variados a diferentes usuários em diferentes partes do banco de dados.

Esta página lista todas as regras de controle de acesso atualmente definidas, se houver. Para criar um novo, clique no link Adicionar uma nova regra de controle de acesso. Para remover várias regras de uma só vez, marque as caixas próximas a elas e pressione o botão Excluir regras selecionadas. Para alterar a ordem das regras, use as setas para cima e para baixo no lado direito da tabela.

Como na maioria das outras alterações na configuração do servidor LDAP, as regras de controle de acesso não terão efeito até que o botão Aplicar configuração seja clicado na página principal do módulo.
