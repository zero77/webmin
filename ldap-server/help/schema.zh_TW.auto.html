管理架構
LDAP模式確定哪些對像類和屬性可以存儲在LDAP數據庫中。該頁面允許您使用每個模式文件名旁邊的複選框來選擇服務器支持的模式類型。進行更改後，單擊“ 保存”以更新OpenLDAP配置文件，然後在主頁上應用“應用配置”以將其激活。

但是請小心取消選擇現有模式文件，因為如果使用這些文件中定義的屬性在數據庫中已經存在對象，這可能會破壞LDAP服務器。無法取消選擇包含最基本的LDAP類的核心架構。

模式的順序很重要，因為以後的模式只能引用先前定義的屬性。您可以使用此頁面上的向上和向下箭頭對列表進行重新排序，但是再次進行此操作時應注意現有模式，以避免破壞LDAP數據庫。

要查看架構定義的類和屬性，請單擊其“ 查看”鏈接。如果您熟悉架構格式，並且想要編輯文件，請單擊“ 編輯” 。
