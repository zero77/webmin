Gerenciar esquema
O esquema LDAP determina quais classes e atributos de objetos podem ser armazenados no seu banco de dados LDAP. Esta página permite selecionar quais tipos de esquema são suportados pelo servidor, usando a caixa de seleção ao lado de cada nome de arquivo de esquema. Depois de fazer as alterações, clique em Salvar para atualizar o arquivo de configuração do OpenLDAP e, em seguida, em Aplicar configuração na página principal para ativá-los.

Porém, tenha cuidado ao desmarcar os arquivos de esquema existentes, pois isso pode interromper o servidor LDAP se os objetos já existirem no banco de dados usando os atributos definidos nesses arquivos. O esquema principal que contém as classes LDAP mais básicas não pode ser desmarcado.

A ordem dos esquemas é importante, pois os esquemas posteriores podem se referir apenas aos atributos definidos anteriormente. Você pode reordenar a lista usando as setas para cima e para baixo nesta página, mas novamente isso deve ser feito com cuidado com os esquemas existentes para evitar a quebra do banco de dados LDAP.

Para ver quais classes e atributos um esquema define, clique no link Visualizar . Se você estiver familiarizado com o formato do esquema e quiser editar um arquivo, clique em Editar .
