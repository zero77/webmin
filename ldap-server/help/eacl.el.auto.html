Κανόνας ελέγχου πρόσβασης
Αυτή η σελίδα σάς επιτρέπει να επεξεργαστείτε ή να δημιουργήσετε έναν κανόνα ελέγχου πρόσβασης. Η πρώτη ενότητα Αντικείμενα που παρέχονται καθορίζει σε ποια DN στη βάση δεδομένων σας θα εφαρμόζεται ο κανόνας. Μπορείτε είτε να επιλέξετε Όλα τα αντικείμενα για ολόκληρη τη βάση δεδομένων, είτε Αντικείμενο με DN για αντικείμενα που ταιριάζουν με ό, τι εισάγετε στο παρακείμενο πεδίο.

Το μενού τύπου αντιστοίχισης καθορίζει εάν ο κανόνας ισχύει μόνο για αυτό το αντικείμενο, εκείνους που βρίσκονται κάτω από αυτό ή εάν το DN αντιμετωπίζεται ως κανονική έκφραση. Μπορείτε επίσης να ελέγξετε περαιτέρω ποια αντικείμενα χορηγούνται εισάγοντας ένα φίλτρο LDAP στο πεδίο Όριο με φίλτρο αντικειμένων , όπως (objectClass = posixAccount) .

Το δεύτερο μέρος της σελίδας είναι ένας πίνακας για την επιλογή των χρηστών LDAP που έχουν πρόσβαση στα αντικείμενα. Μπορείτε είτε να επιλέξετε πολλές γενικές κατηγορίες χρηστών από το μενού Εκχώρηση πρόσβασης στο μενού, είτε να επιλέξετε Άλλο και να εισαγάγετε ένα συγκεκριμένο DN.

Το μενού Επίπεδο πρόσβασης καθορίζει τι μπορούν να κάνουν αυτοί οι χρήστες με τα αντικείμενα. Τα χαμηλότερα πιο ισχυρά επίπεδα υπονοούν όλα αυτά που βρίσκονται πάνω τους, έτσι ένας χρήστης με πρόσβαση εγγραφής μπορεί επίσης να διαβάσει και να αναζητήσει .
