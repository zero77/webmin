Создать объект
Эта страница предназначена для создания нового объекта под некоторым родителем. Вы должны ввести как минимум суффикс DN для объекта, например, dc = people , и один или несколько классов объектов, таких как top .

Если объект должен иметь начальные атрибуты (обязательные для большинства классов), их необходимо ввести в таблицу «Другие атрибуты» . Если какие-либо обязательные атрибуты отсутствуют (или указаны запрещенные атрибуты), при нажатии кнопки «Создать» будет отображаться ошибка.
