Crear zona
Este formulario le permite crear e instalar opcionalmente una nueva zona en este sistema. Debe ingresar al menos el nombre de la zona, que debe ser único y estar compuesto solo por letras y números. Se puede seleccionar un directorio de archivos de zona, pero el valor predeterminado suele ser el mejor.

Debido a que cada zona generalmente necesita una interfaz de red privada para su servidor, puede ingresar su dirección IP en este formulario y seleccionar la interfaz real a la que está vinculada. Todas las zonas tienen automáticamente una interfaz virtual de bucle invertido como lo0:1 , que no tiene que configurarse manualmente.

Si elige instalar el software del sistema de la zona después de que se haya creado, la nueva zona estará inmediatamente lista para usar. Sin embargo, si desea agregar o cambiar los directorios del paquete, puede optar por no instalar ahora, ya que esto evitará que se realicen estos cambios.

Los directorios de paquetes también se pueden especificar en este formulario, que se agregarán antes de instalar la zona. Los directorios /lib , /platform , /sbin y /usr siempre se incluyen automáticamente en cualquier zona nueva.
