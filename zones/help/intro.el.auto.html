Ζώνες Solaris
Η εγκατάσταση ζωνών στο Solaris παρέχει ένα απομονωμένο περιβάλλον για την εκτέλεση εφαρμογών. Οι διαδικασίες που εκτελούνται σε μια ζώνη δεν επιτρέπεται να παρακολουθούν ή να παρεμβαίνουν σε άλλες δραστηριότητες στο σύστημα. Η πρόσβαση σε άλλες διεργασίες, διασυνδέσεις δικτύου, συστήματα αρχείων, συσκευές και εγκαταστάσεις επικοινωνίας μεταξύ διεργασιών περιορίζεται για την αποτροπή αλληλεπίδρασης μεταξύ διεργασιών σε διαφορετικές ζώνες.

Οι ζώνες παρέχουν έναν εξαιρετικό τρόπο δημιουργίας εικονικών συστημάτων στα οποία οι χρήστες μπορούν να έχουν πλήρη πρόσβαση root, χωρίς να διακυβεύεται η ακεραιότητα του πραγματικού συστήματος ή άλλων ζωνών. Μπορούν επίσης να χρησιμοποιηθούν για τον διαχωρισμό ενός μοναδικού ισχυρού συστήματος σε πολλούς εικονικούς διακομιστές, για παράδειγμα για την εκτέλεση δοκιμών και παραγωγής εφαρμογών.

Κάθε ζώνη έχει τον δικό της ριζικό κατάλογο στον οποίο οι διαδικασίες που εκτελούνται στη ζώνη είναι περιορισμένες. Αυτό είναι παρόμοιο με την τυπική εγκατάσταση Chroot Unix, αλλά πολύ πιο απλό να ρυθμιστεί, καθώς όλες οι απαραίτητες βιβλιοθήκες συστήματος και οι κατάλογοι προγραμμάτων συνδέονται αυτόματα στη ζώνη όταν δημιουργείται.

Οι εικονικές διεπαφές δικτύου μπορούν να εκχωρηθούν σε ζώνες για να τους δώσουν τις δικές τους διευθύνσεις IP, έτσι ώστε διαφορετικοί διακομιστές να μπορούν να εκτελούνται σε διαφορετικές ζώνες όπως θα μπορούσαν να βρίσκονται σε πολλά φυσικά συστήματα.

Αυτή η ενότητα σάς επιτρέπει να δημιουργείτε ζώνες, να κάνετε εκκίνηση και επανεκκίνηση ανεξάρτητα από το κύριο λειτουργικό σύστημα και να διαχειρίζεστε τις διεπαφές δικτύου, τους καταλόγους πακέτων, τα συστήματα αρχείων και άλλα χαρακτηριστικά.
