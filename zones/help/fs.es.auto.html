Sistema de archivos adicional
Esta página le permite montar una unidad o partición del sistema global en un directorio dentro de una zona. Esto puede ser útil si una zona necesita almacenar grandes cantidades de datos y desea aislar esto de los discos utilizados por otras zonas y el sistema global.

Para la mayoría de los tipos de sistemas de archivos, puede seleccionar un directorio de montaje, la partición o dispositivo fuente y las opciones para el montaje. Para algunos tipos que no son compatibles con Webmin, el dispositivo de origen y las opciones deben ingresarse manualmente.

Esta página no se puede usar para crear montajes NFS; en cambio, se pueden crear simplemente usando los comandos de montaje normales dentro de la propia zona.
