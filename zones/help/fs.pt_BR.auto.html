Sistema de arquivos adicional
Esta página permite montar uma unidade ou partição do sistema global em um diretório dentro de uma região. Isso pode ser útil se uma região precisar armazenar grandes quantidades de dados e você quiser isolá-lo dos discos usados por outras regiões e pelo sistema global.

Para a maioria dos tipos de sistemas de arquivos, você pode selecionar um diretório de montagem, a partição ou o dispositivo de origem e as opções para a montagem. Para alguns tipos que não são suportados pelo Webmin, o dispositivo e as opções de origem devem ser inseridos manualmente.

Esta página não pode ser usada para criar montagens NFS - em vez disso, elas podem ser criadas usando simplesmente comandos normais de montagem na própria região.
