Διεπαφή δικτύου
Η διασύνδεση δικτύου μιας ζώνης είναι μια εικονική διεπαφή IP στο παγκόσμιο σύστημα που εκχωρείται ως η κύρια διεπαφή για αυτήν τη ζώνη. Καθένα έχει μια διεύθυνση IP που πρέπει να είναι μοναδική, μια σχετική πραγματική διασύνδεση δικτύου και ένα προαιρετικό netmask. Καθώς δημιουργείται κάποια διασύνδεση σε αυτήν τη σελίδα, θα ενεργοποιηθεί στη ζώνη (υποθέτοντας ότι η ζώνη λειτουργεί).
