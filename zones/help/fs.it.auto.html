File system aggiuntivo
Questa pagina consente di montare un'unità o una partizione dal sistema globale in una directory all'interno di una zona. Ciò può essere utile se una zona deve archiviare grandi quantità di dati e si desidera isolarlo dai dischi utilizzati da altre zone e dal sistema globale.

Per la maggior parte del tipo di file system, è possibile selezionare una directory di montaggio, la partizione o il dispositivo di origine e le opzioni per il montaggio. Per alcuni tipi che non sono supportati da Webmin, il dispositivo di origine e le opzioni devono essere immessi manualmente.

Questa pagina non può essere utilizzata per creare montaggi NFS, ma possono essere creati semplicemente usando i normali comandi di montaggio all'interno della zona stessa.
