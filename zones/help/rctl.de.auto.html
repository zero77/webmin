Ressourcensteuerung
Eine Ressourcensteuerung ist eine Reihe von Beschränkungen für die Verwendung einer Ressource innerhalb der Zone und zugehörige Aktionen, wenn diese überschritten werden. Die zum Zeitpunkt des Schreibens verfügbaren Ressourcen sind:
zone.max-lwps
Begrenzt die Anzahl der Prozesse, die gleichzeitig in der Zone ausgeführt werden können.
zone.cpu-teilen
Legt ein Limit für die Anzahl der FSS-CPU-Freigaben (Fair Share Scheduler) für eine Zone fest.
Für jede Ressource können Sie eine Reihe von Grenzwerten eingeben. Jedes Limit hat einen Berechtigungstyp (der immer privilegiert ist ), ein numerisches Limit und eine Aktion, die ausgeführt werden muss, wenn das Limit überschritten wird. Die typische Aktion ist Verweigern , um zu verhindern, dass eine Anforderung das Limit überschreitet.
