Zusätzliches Dateisystem
Auf dieser Seite können Sie ein Laufwerk oder eine Partition vom globalen System in ein Verzeichnis innerhalb einer Zone einbinden. Dies kann nützlich sein, wenn in einer Zone große Datenmengen gespeichert werden müssen und Sie diese von den Festplatten isolieren möchten, die von anderen Zonen und dem globalen System verwendet werden.

Für die meisten Dateisystemtypen können Sie ein Mount-Verzeichnis, die Quellpartition oder das Quellgerät sowie Optionen für den Mount auswählen. Bei einigen Typen, die von Webmin nicht unterstützt werden, müssen das Quellgerät und die Optionen manuell eingegeben werden.

Diese Seite kann nicht zum Erstellen von NFS-Mounts verwendet werden. Stattdessen können sie einfach mit normalen Mount-Befehlen in der Zone selbst erstellt werden.
