Esportatu Batch fitxategia
Orrialde hau zure sistemako erabiltzaile batzuk edo guztiak esportatzeko erabil daiteke moduluekin erabiltzaileak Sortu, aldatu eta ezabatu fitxategien funtzionaltasunetik. Hau erabilgarria izan daiteke erabiltzaileak beste sistema batera migratu nahi baduzu edo sistema honetako erabiltzaile datu-basearen segurtasun kopia sortu.

Labean dauden eremuak hauek dira:

Batch fitxategiaren helmuga
Eremu honek zehazten du emaitza lortzen den fitxategia pantailan bistaratuko den edo zerbitzarian fitxategi batean gordeko den.

Batch fitxategi formatua
Menu honek, zein formatu erabiliko den zehazten du sortutako batch fitxategian, zure sistemaren formatua hautatuta. Beste sistema bat erabiltzen duen sistema batean erabiltzeko lote bat sortzen ari bazara, orduan OS horretarako egokia den formatua aukeratu beharko litzateke.

Erabiltzaileak esportatu
Atal honetan Unix erabiltzaileak batch fitxategian sartuko diren zehazten da. Sistema honetako erabiltzaile batzuk soilik editatzera mugatuta badago, murrizketa hori hemen aukeratutako guztiekin aplikatuko da.

Ezaugarri hau erabiliz sortutako lote-fitxategiak erabiltzailearen oinarrizko datuak soilik izango ditu, EZ etxeko etxeko direktorioen edukiak, kuotak, Samba pasahitzak edo bestelako ezarpenak. Hori horrela izanik, ez da ordezko segurtasun kopien ordezkoa!
