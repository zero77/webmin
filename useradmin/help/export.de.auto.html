Batchdatei exportieren
Diese Seite kann verwendet werden, um einige oder alle Benutzer auf Ihrem System als Batchdatei zu exportieren, die mit den Modulen Benutzer aus Batchdatei erstellen, ändern und löschen kann . Dies kann nützlich sein, wenn Sie Benutzer auf ein anderes System migrieren oder eine Sicherung der Benutzerdatenbank auf diesem System erstellen möchten.

Die Felder in dieser Forn sind:

Batch-Dateiziel
Dieses Feld bestimmt einfach, ob die resultierende Datei auf dem Bildschirm angezeigt oder in einer Datei auf dem Server gespeichert wird.

Batch-Dateiformat
In diesem Menü wird festgelegt, welches Format für die resultierende Batchdatei verwendet wird, wobei das Format Ihres Systems ausgewählt wird. Wenn Sie einen Stapel zur Verwendung auf einem System erstellen, auf dem ein anderes Betriebssystem ausgeführt wird, sollte das für dieses Betriebssystem geeignete Format ausgewählt werden.

Zu exportierende Benutzer
In diesem Abschnitt wird festgelegt, welche Unix-Benutzer in die Batchdatei aufgenommen werden. Wenn Sie nur einige Benutzer auf diesem System bearbeiten dürfen, gilt diese Einschränkung zusammen mit allen hier ausgewählten.

Die mit dieser Funktion erstellte Batchdatei enthält nur grundlegende Benutzerdetails, NICHT den Inhalt ihrer Basisverzeichnisse, Kontingente, Samba-Kennwörter oder andere Einstellungen. Als solches ist es kein Ersatz für reguläre Backups!
