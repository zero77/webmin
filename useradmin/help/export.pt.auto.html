Exportar arquivo em lote
Esta página pode ser usada para exportar alguns ou todos os usuários do seu sistema como um arquivo em lotes utilizável pelos módulos Criar, modificar e excluir usuários do recurso de arquivo em lote . Isso pode ser útil se você desejar migrar usuários para outro sistema ou criar um backup do banco de dados do usuário neste sistema.

Os campos neste forn são:

Destino do arquivo em lote
Este campo simplesmente determina se o arquivo resultante será exibido na tela ou salvo em um arquivo no servidor.

Formato de arquivo em lote
Este menu determina qual formato será usado para o arquivo em lotes resultante, com o formato do seu sistema selecionado. Se você estiver criando um lote para uso em um sistema executando um sistema operacional diferente, o formato adequado para esse sistema operacional deverá ser selecionado.

Usuários a exportar
Esta seção determina quais usuários do Unix serão incluídos no arquivo em lotes. Se você estiver restrito a editar apenas alguns dos usuários deste sistema, essa restrição será aplicada juntamente com as que você escolher aqui.

O arquivo em lotes criado usando esse recurso incluirá apenas detalhes básicos do usuário, NÃO o conteúdo de seus diretórios pessoais, cotas, senhas do Samba ou outras configurações. Como tal, não substitui os backups regulares!
