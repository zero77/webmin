ID ผู้ใช้
ID ผู้ใช้หรือ UID คือสิ่งที่ระบบใช้เพื่อบังคับใช้สิทธิ์ในไฟล์และกระบวนการ หากผู้ใช้สองคนมี UID เดียวกันพวกเขาจะสามารถเข้าถึงไฟล์อื่น ๆ ได้ดังนั้นผู้ใช้ทุกคนควรมี UID ที่ไม่ซ้ำกัน

เมื่อสร้างผู้ใช้ใหม่คุณจะมีตัวเลือกต่อไปนี้:

อัตโนมัติ
UID จะถูกกำหนดโดยอัตโนมัติโดย Webmin โดยขึ้นอยู่กับการใช้งานในปัจจุบัน
จากการคำนวณ
UID จะถูกคำนวณตามชื่อผู้ใช้โดยใช้อัลกอริทึมการตรวจสอบ
ป้อน
ตัวเลือกสุดท้ายช่วยให้คุณป้อน UID ที่คุณต้องการ

ผู้ใช้ รูท จะมี UID เป็น 0 เสมอหากคุณสร้างผู้ใช้อื่นที่มี UID 0 ผู้ใช้นั้นจะมีอำนาจเหมือนกับ รู ท

หากคุณกำลังแก้ไขผู้ใช้ที่มีอยู่และเปลี่ยน UID, เปลี่ยน ID ผู้ใช้ในไฟล์? ตัวเลือกจะกำหนดว่าไฟล์ใด (ถ้ามี) ที่มีการเปลี่ยนแปลงความเป็นเจ้าของ
