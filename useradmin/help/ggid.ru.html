ID Группы
Как и ID пользователя, ID группы или GID - это идентификатор, который система реально использует для определения прав на файлы и процессы. Если две группы имеют одинаковый GID они будут эквивалентны для системы при проверке прав. При создании новой группы это поле будет автоматически заполнено неиспользуемым GID.
