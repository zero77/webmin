Флаги аккаунта
Для изменения поведения пароля можно установить три разных флага:
  • Только root может сменить пароль
    Если этот флаг установлен, то только пароль пользователя может изменить информацию о пароле.

  • Пользователь должен выбрать новый пароль
    Если этот флаг установлен, то пользователь будет вынужден изменить свой пароль в следующий раз, когда для пользователя будет дана команда входа в систему или команда su.

  • Не проверять ограничения пароля
    Если этот флаг установлен, то при выборе новых паролей необязательно следовать указаниям по паролям.
