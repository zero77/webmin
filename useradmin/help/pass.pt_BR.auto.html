Senha
Quando um usuário efetua login via telnet ou no console, ele normalmente deve fornecer uma senha. Existem quatro opções diferentes para a senha:
  • Pergunte no primeiro login
    Quando o usuário efetua login pela primeira vez via telnet ou no console no modo de texto, o sistema solicita sua senha.

  • Nenhuma senha necessária
    O usuário pode efetuar login no sistema sem fornecer uma senha. Esta opção deve ser usada com cuidado.

  • Nenhum login permitido
    Se essa opção for escolhida, o usuário não poderá fazer o login no sistema, nem mesmo ler e-mails usando o servidor POP3.

  • Senha criptografada
    As senhas Unix são armazenadas em um formato criptografado unidirecional. Essa opção permite inserir a senha de um usuário como uma sequência criptografada (talvez extraída do arquivo de senhas em outro sistema).

  • Senha de texto não criptografado
    Para definir a senha para um usuário, selecione esta opção e digite-a no campo de texto. O Webmin pegará a senha digitada, criptografará e armazenará no arquivo de senhas.
