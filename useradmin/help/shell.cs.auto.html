Shell
Když se uživatel přihlásí přes telnet nebo na konzoli nebo otevře xterm okno v X, spustí se shell program uživatelů pro interpretaci zadaných příkazů. Pokud uživatel nemá platný program shellu nebo pokud nemá žádný program (jako /bin/false ), nebude se moci přihlásit. To může být užitečné při vytváření uživatelů, kteří mohou pouze číst a odesílat e-maily.

V tomto poli jsou uvedeny všechny skořápky používané uživateli ve vašem systému. Můžete si vybrat jednu nebo zvolit Jiné .. a do textového pole níže zadejte úplnou cestu shellu.

Všechny unixové systémy mají Bourne shell ( /bin/sh ) a C Shell ( /bin/csh ). Další běžné a uživatelsky přívětivější skořápky jsou tcsh , zsh a bash .
