Первичная Группа
Это группа, которая по умолчанию будет группой - владельцем любого файла или процесса созданного этим пользователем. Если пользователь обращается к файлу и идентификатор группы данного файла является идентификатором первичной группы данного пользователя, то вступают в силу права группы на данный файл (если только пользователь не является владельцем данного файла).

Если при редактировании существующего пользователя вы изменили первичную группу (если вам это разрешено), опция Изменить ID группы на файлы? определяет какие файлы (если они есть) поменяют свою группу.
