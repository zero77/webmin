배치 파일 내보내기
이 페이지는 시스템에서 일부 또는 모든 사용자를 모듈과 함께 사용할 수있는 배치 파일로 내보내는 데 사용할 수 있습니다. 배치 파일 기능 에서 사용자 작성, 수정 및 삭제 기능. 사용자를 다른 시스템으로 마이그레이션하거나이 시스템에서 사용자 데이터베이스의 백업을 생성하려는 경우에 유용 할 수 있습니다.

이 여관의 필드는 다음과 같습니다.

배치 파일 대상
이 필드는 단순히 결과 파일이 화면에 표시되거나 서버의 파일에 저장되는지 여부를 결정합니다.

배치 파일 형식
이 메뉴는 결과 형식 파일에 사용할 형식을 결정하며 시스템 형식을 선택합니다. 다른 OS를 실행하는 시스템에서 사용할 배치를 작성하는 경우 해당 OS에 적합한 형식을 선택해야합니다.

내보낼 사용자
이 섹션에서는 배치 파일에 포함될 Unix 사용자를 결정합니다. 이 시스템에서 일부 사용자 만 편집 할 수있는 경우 여기에서 선택한 것과 함께 해당 제한이 적용됩니다.

이 기능을 사용하여 작성된 배치 파일에는 홈 디렉토리의 내용, 할당량, Samba 비밀번호 또는 기타 설정이 아닌 기본 사용자 세부 사항 만 포함됩니다. 따라서 정기적 인 백업을 대체하지 않습니다!
