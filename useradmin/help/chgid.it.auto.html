Cambia l'ID del gruppo sui file?
Quando si modifica un utente esistente, se il gruppo primario dell'utente viene modificato, i file di proprietà di quel gruppo rimarranno di proprietà del vecchio GID. Questa opzione può essere utilizzata per aggiornare la proprietà dei file al nuovo GID cercando la home directory dell'utente (Home Dir) o l'intero filesystem (Tutti i file). L'opzione Tutti i file potrebbe richiedere molto tempo, soprattutto se si dispone di molti file system di grandi dimensioni o montati su NFS.
