Hauptgruppe
Dies ist die Gruppe, die standardmäßig alle von diesem Benutzer erstellten Dateien oder Prozesse besitzt. Wenn dieser Benutzer versucht, auf eine Datei zuzugreifen, und die Gruppe dieser Datei die primäre Gruppe des Benutzers ist, gelten die Gruppenberechtigungen für diese Datei (es sei denn, der Benutzer ist der Eigentümer der Datei).

Wenn Sie einen vorhandenen Benutzer bearbeiten und die primäre Gruppe geändert wird, wird die Gruppen-ID für Dateien ändern? Mit dieser Option wird festgelegt, für welche Dateien (falls vorhanden) die Gruppe geändert wurde.
