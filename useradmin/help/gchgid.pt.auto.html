Alterar GID nos arquivos?
Ao editar um grupo, se o ID do grupo for alterado, os arquivos pertencentes a esse grupo permanecerão pertencentes ao GID antigo. Essa opção pode ser usada para atualizar a propriedade dos arquivos para o novo GID pesquisando os diretórios pessoais de todos os usuários que são membros deste grupo (Diretórios Domiciliares) ou todo o sistema de arquivos (Todos os Arquivos). A opção Todos os arquivos pode demorar muito, especialmente se você tiver muitos sistemas de arquivos grandes ou montados em NFS.
