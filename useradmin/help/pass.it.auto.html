Parola d'ordine
Quando un utente accede tramite telnet o alla console, deve normalmente fornire una password. Esistono quattro diverse opzioni per la password:
  • Chiedi al primo accesso
    Quando l'utente accede per la prima volta tramite telnet o alla console in modalità testo, il sistema richiederà all'utente la password.

  • Nessuna password richiesta
    L'utente può accedere al sistema senza fornire una password. Questa opzione dovrebbe essere usata con cura.

  • Nessun accesso consentito
    Se si sceglie questa opzione, l'utente non sarà in grado di accedere al sistema, nemmeno per leggere la posta utilizzando il server POP3.

  • Password criptata
    Le password Unix sono memorizzate in una forma crittografata unidirezionale. Questa opzione ti consente di inserire la password per un utente come stringa crittografata (forse presa dal file della password su un altro sistema).

  • Cancella password di testo
    Per impostare la password per un utente, selezionare questa opzione e inserirla nel campo di testo. Webmin prenderà la password inserita, la crittograferà e la memorizzerà nel file della password.
