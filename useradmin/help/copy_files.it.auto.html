Copia i file nella home directory?
Quando viene creato un nuovo utente e questa opzione è impostata, i file standard come .cshrc e .profile vengono copiati da /etc/skel nella home directory dell'utente.

Selezionata questa opzione non avrà alcun effetto se l'opzione Crea home directory non è impostata anche in quanto non ci sarà home directory in cui copiare.
