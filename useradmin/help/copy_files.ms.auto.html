Salin fail ke direktori utama?
Apabila pengguna baru dibuat dan pilihan ini ditetapkan, fail standard seperti .cshrc dan .profile disalin dari /etc/skel ke direktori utama pengguna.

Memilih pilihan ini tidak akan berpengaruh jika opsi Buat direktori rumah tidak disetel juga, karena tidak akan ada direktori home untuk disalin.
