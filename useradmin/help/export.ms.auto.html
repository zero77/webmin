Eksport Fail Kumpulan
Halaman ini dapat digunakan untuk mengekspor beberapa atau semua pengguna di sistem Anda sebagai file batch yang dapat digunakan dengan modul Membuat, mengubah dan menghapus pengguna dari fitur file batch . Ini boleh berguna jika anda ingin memindahkan pengguna ke sistem lain, atau membuat sandaran pangkalan data pengguna pada sistem ini.

Bidang di forn ini adalah:

Destinasi fail kumpulan
Medan ini hanya menentukan apakah fail yang dihasilkan akan dipaparkan di layar, atau disimpan ke fail di pelayan.

Format fail kumpulan
Menu ini menentukan format yang akan digunakan untuk file batch yang dihasilkan, dengan format sistem anda dipilih. Sekiranya anda membuat kumpulan untuk digunakan pada sistem yang menjalankan OS lain, maka format yang sesuai untuk OS tersebut harus dipilih.

Pengguna untuk mengeksport
Bahagian ini menentukan pengguna Unix mana yang akan dimasukkan dalam fail kumpulan. Sekiranya anda dibatasi hanya untuk mengedit beberapa pengguna pada sistem ini, maka sekatan itu akan berlaku bersama dengan yang anda pilih di sini.

Fail kumpulan yang dibuat menggunakan ciri ini hanya akan merangkumi butiran pengguna asas, BUKAN kandungan direktori rumah mereka, kuota, kata laluan Samba atau tetapan lain. Oleh itu, ia bukan pengganti sandaran biasa!
