Benutzeridentifikation
Die Benutzer-ID oder UID wird vom System tatsächlich verwendet, um Berechtigungen für Dateien und Prozesse zu erzwingen. Wenn zwei Benutzer dieselbe UID haben, können sie auf die jeweils anderen Dateien zugreifen. Daher sollte jeder Benutzer eine eindeutige UID haben.

Wenn Sie einen neuen Benutzer erstellen, haben Sie folgende Möglichkeiten:

Automatisch
Die UID wird von Webmin basierend auf den derzeit verwendeten automatisch zugewiesen.
Berechnet
Die UID wird basierend auf dem Benutzernamen unter Verwendung eines Prüfsummenalgorithmus berechnet.
Trat ein
Mit der letzten Option können Sie eine beliebige UID eingeben.

Der Root- Benutzer hat immer eine UID von 0. Wenn Sie einen anderen Benutzer mit UID 0 erstellen, hat dieser Benutzer die gleichen Befugnisse wie root .

Wenn Sie einen vorhandenen Benutzer bearbeiten und die UID ändern, wird die Benutzer-ID für Dateien ändern? Die Option bestimmt, welche Dateien (falls vorhanden) ihren Besitz geändert haben.
