Directory principale
La home directory di un utente in genere contiene tutti i file di proprietà dell'utente. Quando un utente accede, verrà inserito nella sua directory home e verranno eseguiti script come .cshrc o .profile .

Quando si crea un nuovo utente, Webmin creerà la home directory che si immette in questo campo. Se si modifica la home directory durante la modifica di un utente esistente, Webmin tenterà di rinominare la directory se la home directory sposta se modificata? l'opzione è impostata.
