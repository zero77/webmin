Usuários e Grupos

Introdução aos usuários

Um usuário Unix geralmente é alguém que pode fazer login no sistema, remotamente via telnet ou no console. Todo arquivo pertence a algum usuário e todo processo é executado com os direitos de algum usuário. O acesso a arquivos e processos é determinado pelo usuário no qual você está conectado.

Existem realmente três tipos de usuários em um sistema Unix típico:

  • Usuários Administrativos
    Contas como bin , lp e uccp possuem arquivos como os comandos padrão em /bin , arquivos de spool de impressão e dados UUCP. Na verdade, o login com uma dessas contas normalmente não é permitido - elas são usadas apenas por vários processos do sistema. Esses usuários serão criados quando o sistema operacional for instalado pela primeira vez.

  • Pessoas reais
    Essas são contas pertencentes a usuários reais, criadas pelo administrador do sistema. Você pode permitir que esses usuários façam login remotamente ou talvez apenas envie e receba emails via SMTP e POP3.

  • O usuário root
    O usuário root tem o poder de ler e gravar qualquer arquivo ou diretório e controlar qualquer processo. Essa conta geralmente é usada para fins de administração do sistema e é a conta na qual o Webmin é executado.

No topo da página principal deste módulo, há uma tabela de usuários existentes em seu sistema. Você pode clicar em um usuário para editá-lo ou clicar no link abaixo da tabela para criar um novo usuário. Os usuários cujos nomes de login são mostrados em itálico são aqueles cujas senhas estão temporariamente desabilitadas.

Introdução aos Grupos

Um grupo é simplesmente uma lista de usuários do Unix. Todo usuário pertence a pelo menos um grupo (seu grupo principal) e, opcionalmente, vários outros. Todos os arquivos pertencem a algum grupo e todo processo é executado com as permissões de um grupo.

Abaixo da lista de usuários, há uma tabela de grupos existentes. Você pode clicar no nome de um grupo para editá-lo ou clicar no link abaixo da tabela para criar um novo grupo.

Logins gravados

Sempre que um usuário efetua login por telnet, FTP ou no console, o sistema registra o logon e o logoff subsequente. Na parte inferior da página principal, há um pequeno formulário que permite exibir logins por todos ou por um usuário selecionado.
