Indicateurs de compte
Trois indicateurs différents peuvent être définis pour modifier le comportement du mot de passe:
  • Seul root peut changer le mot de passe
    Si cet indicateur est défini, seul l'utilisateur root peut modifier les informations de mot de passe.

  • L'utilisateur doit choisir un nouveau mot de passe
    Si cet indicateur est défini, l'utilisateur est obligé de changer son mot de passe la prochaine fois qu'une commande de connexion ou une commande su est donnée pour l'utilisateur.

  • Ne pas vérifier les restrictions de mot de passe
    Si cet indicateur est défini, les nouveaux mots de passe choisis n'ont pas besoin de suivre les instructions relatives aux mots de passe.
