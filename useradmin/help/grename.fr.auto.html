Renommer le groupe si le nom d'utilisateur change?
Lors de la modification d'un utilisateur dont le groupe principal est le même que son nom d'utilisateur, cette option peut être utilisée pour renommer le groupe si le nom de l'utilisateur est modifié. Ceci est recommandé par souci de cohérence, mais peut casser d'autres fichiers de configuration qui contiennent l'ancien nom de groupe.
