Επεξεργασία χρήστη
Αυτή η φόρμα σας επιτρέπει να επεξεργαστείτε τις λεπτομέρειες ενός υπάρχοντος χρήστη του Unix. Όλες οι λεπτομέρειες του χρήστη μπορεί να αλλάξουν - ωστόσο, να είστε προσεκτικοί χρήστες του συστήματος επεξεργασίας όπως το root και κανένας, καθώς αυτό μπορεί να σπάσει το σύστημά σας.
