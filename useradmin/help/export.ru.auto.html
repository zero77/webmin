Экспортировать пакетный файл
Эта страница может использоваться для экспорта некоторых или всех пользователей в вашей системе в виде пакетного файла, используемого с модулями Создание, изменение и удаление пользователей из функции пакетного файла . Это может быть полезно, если вы хотите перенести пользователей в другую систему или создать резервную копию базы данных пользователей в этой системе.

Поля на этом участке:

Назначение пакетного файла
Это поле просто определяет, будет ли полученный файл отображаться на экране или сохраняться в файл на сервере.

Пакетный формат файла
Это меню определяет, какой формат будет использоваться для полученного пакетного файла, с выбранным форматом вашей системы. Если вы создаете пакет для использования в системе под управлением другой ОС, то следует выбрать формат, подходящий для этой ОС.

Пользователи для экспорта
Этот раздел определяет, какие пользователи Unix будут включены в пакетный файл. Если вы ограничены только редактированием некоторых пользователей в этой системе, то это ограничение будет применяться вместе с любым, который вы здесь выберете.

Пакетный файл, созданный с использованием этой функции, будет содержать только основные сведения о пользователе, а НЕ содержимое их домашних каталогов, квоты, пароли Samba или другие параметры. Таким образом, это не замена для обычных резервных копий!
