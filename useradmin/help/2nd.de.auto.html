Sekundärgruppen
Eine Liste mit null oder mehr anderen Gruppen, zu denen dieser Benutzer gehört. Wenn dieser Benutzer versucht, auf eine Datei zuzugreifen, und die Gruppe dieser Datei eine der sekundären Gruppen ist, gelten die Gruppenberechtigungen für diese Datei.

Wenn Sie zur Eingabe von Gruppen ein Textfeld verwenden, muss sich jeder Gruppenname in einer separaten Zeile befinden.
