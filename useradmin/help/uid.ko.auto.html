아이디
사용자 ID 또는 UID는 시스템이 파일 및 프로세스에 대한 권한을 시행하기 위해 실제로 사용하는 것입니다. 두 명의 사용자가 동일한 UID를 가지고 있으면 서로의 파일에 액세스 할 수 있으므로 모든 사용자는 고유 한 UID를 가져야합니다.

새 사용자를 만들 때는 다음과 같은 옵션이 있습니다.

자동적 인
UID는 현재 사용중인 UImin을 기반으로 Webmin에 의해 자동으로 할당됩니다.
계획된
UID는 체크섬 알고리즘을 사용하여 사용자 이름을 기반으로 계산됩니다.
입력
마지막 옵션을 사용하면 원하는 UID를 입력 할 수 있습니다.

루트 사용자는 항상 UID가 0입니다. UID 0으로 다른 사용자를 작성하면 해당 사용자는 root 와 동일한 권한을 갖습니다.

기존 사용자를 편집하고 UID를 변경하는 경우 파일에서 사용자 ID를 변경 하시겠습니까? 옵션은 소유권이 변경된 파일 (있는 경우)을 결정합니다.
