Gruppo primario
Questo è il gruppo che per impostazione predefinita sarà proprietario di qualsiasi file o processo creato da questo utente. Se questo utente tenta di accedere a un file e il gruppo di quel file è il gruppo principale dell'utente, verranno applicate le autorizzazioni di gruppo per quel file (a meno che l'utente non sia il proprietario del file).

Se stai modificando un utente esistente e il gruppo primario viene modificato, l' ID gruppo di modifica nei file? L'opzione determina quali file (se presenti) hanno cambiato il loro gruppo.
