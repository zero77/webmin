Cambia l'ID utente sui file?
Quando si modifica un utente esistente, se si modifica l' ID utente dell'utente, i file di proprietà dell'utente rimarranno di proprietà del vecchio UID. Questa opzione può essere utilizzata per aggiornare la proprietà dei file al nuovo UID cercando la home directory degli utenti (Home Dir) o l'intero filesystem (Tutti i file). L'opzione Tutti i file potrebbe richiedere molto tempo, soprattutto se si dispone di molti file system di grandi dimensioni o montati su NFS.
