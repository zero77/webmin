Copiar arquivos para o diretório inicial?
Quando um novo usuário é criado e esta opção é definida, arquivos padrão como .cshrc e .profile são copiados de /etc/skel para o diretório inicial do usuário.

Selecionada essa opção não terá efeito se a opção Criar diretório pessoal também não estiver definida, pois não haverá diretório pessoal para copiar.
