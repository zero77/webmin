Exportera batchfil
Denna sida kan användas för att exportera några eller alla användare på ditt system som en batchfil som kan användas med modulerna Skapa, ändra och ta bort användare från batchfilfunktionen . Detta kan vara användbart om du vill migrera användare till ett annat system eller skapa en säkerhetskopia av användardatabasen på det här systemet.

Fälten på denna forn är:

Batchfildestination
Detta fält bestämmer helt enkelt om den resulterande filen ska visas på skärmen eller sparas i en fil på servern.

Batchfilformat
Den här menyn bestämmer vilket format som ska användas för den resulterande batchfilen, med ditt systems format valt. Om du skapar en batch för användning i ett system som kör ett annat operativsystem, bör det format som är lämpligt för det operativsystemet väljas.

Användare att exportera
Det här avsnittet avgör vilka Unix-användare som ska inkluderas i batchfilen. Om du är begränsad till att bara redigera några av användarna på detta system kommer den begränsningen att gälla tillsammans med alla du väljer här.

Den batchfil som skapats med den här funktionen kommer endast att innehålla grundläggande användarinformation, INTE innehållet i deras hemkataloger, kvoter, Samba-lösenord eller andra inställningar. Som sådan ersätter det inte regelbundna säkerhetskopior!
