Exportar archivo por lotes
Esta página se puede utilizar para exportar algunos o todos los usuarios de su sistema como un archivo por lotes que se puede usar con los módulos Crear, modificar y eliminar usuarios de la función de archivo por lotes . Esto puede ser útil si desea migrar usuarios a otro sistema o crear una copia de seguridad de la base de datos de usuarios en este sistema.

Los campos en este formulario son:

Destino de archivo por lotes
Este campo simplemente determina si el archivo resultante se mostrará en la pantalla o se guardará en un archivo en el servidor.

Formato de archivo por lotes
Este menú determina qué formato se utilizará para el archivo por lotes resultante, con el formato de su sistema seleccionado. Si está creando un lote para usar en un sistema que ejecuta un sistema operativo diferente, entonces debe seleccionarse el formato adecuado para ese sistema operativo.

Usuarios para exportar
Esta sección determina qué usuarios de Unix se incluirán en el archivo por lotes. Si está restringido a editar solo algunos de los usuarios en este sistema, entonces esa restricción se aplicará junto con cualquiera que elija aquí.

El archivo por lotes creado con esta función solo incluirá detalles básicos del usuario, NO el contenido de sus directorios de inicio, cuotas, contraseñas de Samba u otras configuraciones. ¡Como tal, no es un reemplazo para las copias de seguridad regulares!
