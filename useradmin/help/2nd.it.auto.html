Gruppi secondari
Un elenco di zero o più altri gruppi a cui appartiene questo utente. Se questo utente tenta di accedere a un file e il gruppo di quel file è uno dei gruppi secondari, verranno applicate le autorizzazioni di gruppo per quel file.

Se si utilizza una casella di testo per inserire gruppi, ciascun nome di gruppo deve trovarsi su una riga separata.
