Criar / modificar usuário em outros módulos?
Se essa opção estiver definida como Sim, os usuários recém-criados ou modificados serão atualizados em outros módulos do Webmin. Isso permite (por exemplo) que a lista de usuários do Samba seja sincronizada com a lista de usuários do Unix e que as senhas dos usuários do Webmin sejam mantidas sincronizadas com os usuários do Unix.
