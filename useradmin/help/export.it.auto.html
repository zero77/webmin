Esporta file batch
Questa pagina può essere utilizzata per esportare alcuni o tutti gli utenti sul proprio sistema come file batch utilizzabile con i moduli Crea, modifica ed elimina utenti dalla funzione file batch . Ciò può essere utile se si desidera migrare gli utenti su un altro sistema o creare un backup del database degli utenti su questo sistema.

I campi su questo forn sono:

Destinazione del file batch
Questo campo determina semplicemente se il file risultante verrà visualizzato sullo schermo o salvato in un file sul server.

Formato file batch
Questo menu determina quale formato verrà utilizzato per il file batch risultante, con il formato del sistema selezionato. Se si sta creando un batch da utilizzare su un sistema che esegue un sistema operativo diverso, è necessario selezionare il formato adatto a quel sistema operativo.

Utenti da esportare
Questa sezione determina quali utenti Unix saranno inclusi nel file batch. Se si è limitati a modificare solo alcuni degli utenti su questo sistema, tale restrizione si applicherà a qualsiasi altra scelta qui.

Il file batch creato utilizzando questa funzione includerà solo i dettagli di base dell'utente, NON i contenuti delle loro directory home, quote, password Samba o altre impostazioni. Pertanto, non sostituisce i backup regolari!
