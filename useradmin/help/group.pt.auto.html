Grupo primário
Este é o grupo que, por padrão, será proprietário de todos os arquivos ou processos criados por este usuário. Se esse usuário tentar acessar um arquivo e o grupo desse arquivo for o grupo principal do usuário, as permissões de grupo para esse arquivo serão aplicadas (a menos que o usuário seja o proprietário do arquivo).

Se você estiver editando um usuário existente e o grupo principal for alterado, Alterar o ID do grupo nos arquivos? A opção determina quais arquivos (se houver) tiveram seu grupo alterado.
