ID do usuário
O ID do usuário ou UID é o que o sistema realmente usa para impor permissões em arquivos e processos. Se dois usuários tiverem o mesmo UID, eles poderão acessar os arquivos uns dos outros - portanto, cada usuário deverá ter um UID exclusivo.

Ao criar um novo usuário, você terá as seguintes opções:

Automático
O UID será atribuído automaticamente pelo Webmin, com base nos atualmente em uso.
Calculado
O UID será calculado com base no nome de usuário, usando um algoritmo de soma de verificação.
Introduzido
A opção final permite que você insira o UID desejado.

O usuário root sempre possui um UID igual a 0. Se você criar outro usuário com o UID 0, esse usuário terá os mesmos poderes que o root .

Se você estiver editando um usuário existente e alterar o UID, altere o ID do usuário nos arquivos? A opção determina quais arquivos (se houver) tiveram sua propriedade alterada.
