Kontoflaggen
Es gibt drei verschiedene Flags, die gesetzt werden können, um das Kennwortverhalten zu ändern:
  • Nur root kann das Passwort ändern
    Wenn dieses Flag gesetzt ist, kann nur der Root-Benutzer die Kennwortinformationen ändern.

  • Der Benutzer muss ein neues Passwort wählen
    Wenn dieses Flag gesetzt ist, muss der Benutzer sein Kennwort ändern, wenn das nächste Mal ein Anmeldebefehl oder ein su-Befehl für den Benutzer gegeben wird.

  • Überprüfen Sie nicht die Passwortbeschränkungen
    Wenn dieses Flag gesetzt ist, müssen neu ausgewählte Kennwörter nicht den Kennwortrichtlinien entsprechen.
