Home-Verzeichnis
Das Home-Verzeichnis eines Benutzers enthält normalerweise alle Dateien, die diesem Benutzer gehören. Wenn sich ein Benutzer anmeldet, wird er in sein Ausgangsverzeichnis gestellt und Skripte wie .cshrc oder .profile werden ausgeführt.

Beim Erstellen eines neuen Benutzers erstellt Webmin das Ausgangsverzeichnis, das Sie in dieses Feld eingeben. Wenn Sie das Ausgangsverzeichnis ändern, wenn Sie einen vorhandenen Benutzer ändern, versucht Webmin, das Verzeichnis umzubenennen, wenn das Ausgangsverzeichnis verschieben geändert wird. Option ist gesetzt.
