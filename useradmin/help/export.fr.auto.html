Exporter un fichier batch
Cette page peut être utilisée pour exporter tout ou partie des utilisateurs de votre système sous forme de fichier batch utilisable avec les modules Créer, modifier et supprimer des utilisateurs de la fonction fichier batch . Cela peut être utile si vous souhaitez migrer des utilisateurs vers un autre système ou créer une sauvegarde de la base de données d'utilisateurs sur ce système.

Les champs de ce forn sont:

Destination du fichier batch
Ce champ détermine simplement si le fichier résultant sera affiché à l'écran ou enregistré dans un fichier sur le serveur.

Format de fichier batch
Ce menu détermine le format qui sera utilisé pour le fichier de commandes résultant, avec le format de votre système sélectionné. Si vous créez un lot à utiliser sur un système exécutant un système d'exploitation différent, le format approprié pour ce système d'exploitation doit être sélectionné.

Utilisateurs à exporter
Cette section détermine quels utilisateurs Unix seront inclus dans le fichier de commandes. Si vous êtes limité à ne modifier que certains des utilisateurs de ce système, cette restriction s'appliquera avec tous ceux que vous choisissez ici.

Le fichier de commandes créé à l'aide de cette fonction ne comprendra que les détails de base de l'utilisateur, PAS le contenu de leurs répertoires personnels, quotas, mots de passe Samba ou autres paramètres. En tant que tel, il ne remplace pas les sauvegardes régulières!
