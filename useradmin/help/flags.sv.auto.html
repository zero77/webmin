Kontoflaggor
Det finns tre olika flaggor som kan ställas in för att ändra lösenordsbeteende:
  • Endast root kan ändra lösenord
    Om denna flagga är inställd, kan bara rotanvändaren ändra lösenordsinformation.

  • Användaren måste välja ett nytt lösenord
    Om denna flagga är inställd tvingas användaren att ändra sitt lösenord nästa gång ett inloggningskommando eller ett su-kommando ges för användaren.

  • Kontrollera inte lösenordsbegränsningar
    Om denna flagga är inställd behöver nya lösenord som väljs inte följa lösenordets riktlinjer.
