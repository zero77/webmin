Password
Meta utent jilloggja permezz ta ’telnet jew fil-console, normalment għandu jipprovdi password. Hemm erba 'għażliet differenti għall-password:
  • Staqsi għall-ewwel login
    Meta l-utent jidħol għall-ewwel darba permezz tat-telnet jew fil-console fil-modalità test, is-sistema titlob lill-utent għall-password tiegħu.

  • Ebda password meħtieġa
    L-utent jista 'jidħol fis-sistema mingħajr ma jipprovdi password. Din l-għażla għandha tintuża b'attenzjoni.

  • L-ebda login huwa permess
    Jekk tintgħażel din l-għażla, l-utent ma jkun jista 'jidħol fis-sistema xejn, anke jekk jaqra posta bl-użu ta' server POP3.

  • Password kriptata
    Il-passwords Unix jinħażnu f'forma kodifikata one-way. Din l-għażla tippermettilek tidħol il-password għal utent bħala sekwenza kriptata (forsi meħuda mill-fajl tal-password fuq sistema oħra).

  • Tħassar il-password tat-test
    Biex tissettja l-password għal utent, agħżel din l-għażla u daħħalha fil-qasam tat-test. Webmin jieħu l-password li tidħol, jikkriptaha u jaħżenha fil-fajl tal-password.
