Alterar o ID do grupo nos arquivos?
Ao editar um usuário existente, se o Grupo Principal do usuário for alterado, os arquivos pertencentes a esse grupo permanecerão pertencentes ao GID antigo. Essa opção pode ser usada para atualizar a propriedade dos arquivos para o novo GID pesquisando o diretório inicial do usuário (Dir Home) ou o sistema de arquivos inteiro (Todos os arquivos). A opção Todos os arquivos pode demorar muito, especialmente se você tiver muitos sistemas de arquivos grandes ou montados em NFS.
