Créer / modifier un utilisateur dans d'autres modules?
Si cette option est définie sur Oui, les utilisateurs nouvellement créés ou modifiés seront mis à jour dans d'autres modules Webmin. Cela permet (par exemple) de synchroniser la liste d'utilisateurs Samba avec la liste d'utilisateurs Unix et de synchroniser les mots de passe des utilisateurs Webmin avec les utilisateurs Unix.
