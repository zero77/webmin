Passwort
Wenn sich ein Benutzer über Telnet oder an der Konsole anmeldet, muss er normalerweise ein Kennwort angeben. Es gibt vier verschiedene Optionen für das Passwort:
  • Fragen Sie beim ersten Login
    Wenn sich der Benutzer zum ersten Mal über Telnet oder an der Konsole im Textmodus anmeldet, fordert das System den Benutzer zur Eingabe seines Kennworts auf.

  • Kein Passwort erforderlich
    Der Benutzer kann sich ohne Angabe eines Kennworts beim System anmelden. Diese Option sollte mit Vorsicht verwendet werden.

  • Kein Login erlaubt
    Wenn diese Option ausgewählt ist, kann sich der Benutzer überhaupt nicht beim System anmelden, selbst wenn er E-Mails über den POP3-Server liest.

  • Verschlüsseltes Passwort
    Unix-Passwörter werden in einer einseitig verschlüsselten Form gespeichert. Mit dieser Option können Sie das Kennwort für einen Benutzer als verschlüsselte Zeichenfolge eingeben (möglicherweise aus der Kennwortdatei auf einem anderen System entnommen).

  • Klartext-Passwort
    Um das Kennwort für einen Benutzer festzulegen, wählen Sie diese Option aus und geben Sie sie in das Textfeld ein. Webmin nimmt das eingegebene Passwort, verschlüsselt es und speichert es in der Passwortdatei.
