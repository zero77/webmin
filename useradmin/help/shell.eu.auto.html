Shell
Erabiltzaileak telnet bidez edo kontsolan saioa hasten duenean edo Xterm leiho bat irekitzen duenean X erabiltzailearen shell programa exekutatuko da sartutako komandoak interpretatzeko. Erabiltzaileak ez badu baliozko shell programarik edo ezer egiten ez duenik (adibidez /bin/false ), erabiltzaile hori ezin izango da saioa hasi. Hau erabilgarria izan daiteke mezu elektronikoak irakurri eta bidal ditzaketen erabiltzaileak sortzerakoan.

Eremu honetan zerrendatzen dira erabiltzaileek zure sisteman erabiltzen dituzten maskor guztiak. Bat edo beste aukeratu dezakezu .. eta sartu maskorraren bide osoa beheko testu-eremuan.

Unix sistema guztiak Bourne shell ( /bin/sh ) eta C Shell ( /bin/csh ) rekin daude. Erabilgarriak diren beste oskol arruntak eta tcsh , zsh eta bash dira .
