ID utente
L'ID utente o UID è ciò che il sistema utilizza realmente per imporre autorizzazioni su file e processi. Se due utenti hanno lo stesso UID, saranno in grado di accedere agli altri file, quindi ogni utente dovrebbe avere un UID univoco.

Quando crei un nuovo utente, avrai le seguenti opzioni:

Automatico
L'UID verrà assegnato automaticamente da Webmin, in base a quelli attualmente in uso.
Calcolato
L'UID verrà calcolato in base al nome utente, utilizzando un algoritmo di checksum.
inserito
L'ultima opzione ti consente di inserire qualunque UID desideri.

L'utente root ha sempre un UID di 0. Se si crea un altro utente con UID 0, quell'utente avrà gli stessi poteri di root .

Se si sta modificando un utente esistente e si modifica l'UID, l' ID utente modifica nei file? L'opzione determina quali file (se presenti) hanno la proprietà modificata.
