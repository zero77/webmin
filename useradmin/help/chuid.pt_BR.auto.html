Alterar o ID do usuário nos arquivos?
Ao editar um usuário existente, se o ID do usuário for alterado, os arquivos pertencentes a esse usuário permanecerão pertencentes ao UID antigo. Essa opção pode ser usada para atualizar a propriedade dos arquivos para o novo UID pesquisando o diretório inicial do usuário (Diretório Pessoal) ou todo o sistema de arquivos (Todos os Arquivos). A opção Todos os arquivos pode demorar muito, especialmente se você tiver muitos sistemas de arquivos grandes ou montados em NFS.
