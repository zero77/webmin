Переименовать группу, если имя пользователя меняется?
При редактировании пользователя, основная группа которого совпадает с его именем пользователя, этот параметр можно использовать для переименования группы, если имя пользователя изменилось. Это рекомендуется для согласованности, но может сломать другие файлы конфигурации, которые содержат старое имя группы.
