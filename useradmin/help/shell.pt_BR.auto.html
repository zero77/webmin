Concha
Quando um usuário efetua login via telnet ou no console ou abre uma janela xterm em X, o programa shell do usuário será executado para interpretar os comandos digitados. Se um usuário não tiver um programa shell válido ou um que não faça nada (como /bin/false ), esse usuário não poderá efetuar login. Isso pode ser útil ao criar usuários que podem apenas ler e enviar email.

Listadas neste campo estão todos os shells usados pelos usuários no seu sistema. Você pode selecionar um ou escolher Outro. E digite o caminho completo do shell no campo de texto abaixo.

Todos os sistemas Unix vêm com o shell Bourne ( /bin/sh ) e o C Shell ( /bin/csh ). Outros shells comuns e mais fáceis de usar são tcsh , zsh e bash .
