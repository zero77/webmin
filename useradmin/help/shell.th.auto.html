เปลือก
เมื่อผู้ใช้ล็อกอินผ่าน telnet หรือที่คอนโซลหรือเปิดหน้าต่าง xterm ใน X โปรแกรมเชลล์ผู้ใช้จะทำงานเพื่อตีความคำสั่งที่ป้อน หากผู้ใช้ไม่มีโปรแกรมเชลล์ที่ถูกต้องหรือมีโปรแกรมที่ไม่มีอะไรเลย (เช่น /bin/false ) ผู้ใช้นั้นจะไม่สามารถเข้าสู่ระบบได้ สิ่งนี้มีประโยชน์เมื่อสร้างผู้ใช้ที่สามารถอ่านและส่งอีเมลได้เท่านั้น

รายการในฟิลด์นี้เป็นเชลล์ทั้งหมดที่ผู้ใช้ในระบบของคุณใช้ คุณสามารถเลือกหนึ่งอันหรือเลือก อื่น ๆ .. และป้อนเส้นทางแบบเต็มของเชลล์ในฟิลด์ข้อความด้านล่าง

ระบบ Unix ทั้งหมดมาพร้อมกับ Bourne shell ( /bin/sh ) และ C Shell ( /bin/csh ) เชลล์ทั่วไปที่เป็นมิตรกับผู้ใช้อื่น ๆ คือ tcsh , zsh และ bash
