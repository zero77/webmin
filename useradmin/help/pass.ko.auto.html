암호
사용자가 텔넷 또는 콘솔을 통해 로그인 할 때 일반적으로 암호를 제공해야합니다. 비밀번호에는 네 가지 옵션이 있습니다.
  • 처음 로그인 할 때 묻기
    사용자가 텔넷을 통해 또는 콘솔에서 텍스트 모드로 처음 로그인하면 시스템은 사용자에게 자신의 암호를 묻습니다.

  • 비밀번호가 필요하지 않습니다
    사용자는 비밀번호를 제공하지 않고 시스템에 로그인 할 수 있습니다. 이 옵션은주의해서 사용해야합니다.

  • 로그인이 허용되지 않습니다
    이 옵션을 선택하면 사용자는 시스템에 전혀 로그인 할 수 없으며 POP3 서버를 사용하여 메일을 읽을 수 없습니다.

  • 암호화 된 비밀번호
    유닉스 암호는 단방향 암호화 형식으로 저장됩니다. 이 옵션을 사용하면 사용자의 비밀번호를 암호화 된 문자열 (다른 시스템의 비밀번호 파일에서 가져옴)로 입력 할 수 있습니다.

  • 일반 텍스트 비밀번호
    사용자의 비밀번호를 설정하려면이 옵션을 선택하고 텍스트 필드에 입력하십시오. Webmin은 입력 한 비밀번호를 사용하여 암호화하여 비밀번호 파일에 저장합니다.
