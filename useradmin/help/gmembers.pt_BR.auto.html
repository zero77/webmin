Membros
A parte mais importante de um grupo é a lista de usuários que são seus membros. Se um usuário tentar acessar um arquivo e ele for um membro do grupo que possui o arquivo, as permissões de grupo para esse arquivo serão aplicadas.

Esta lista não inclui usuários que possuem esse grupo como grupo principal , apenas usuários que possuem esse grupo como um de seus grupos secundários .

Se você estiver usando uma caixa de texto para inserir membros do grupo, cada nome de usuário deverá estar em uma linha separada.
