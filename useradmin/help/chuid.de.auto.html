Benutzer-ID für Dateien ändern?
Wenn beim Bearbeiten eines vorhandenen Benutzers die Benutzer- ID des Benutzers geändert wird, bleiben die Dateien dieses Benutzers Eigentum der alten UID. Mit dieser Option können Sie den Besitz von Dateien auf die neue UID aktualisieren, indem Sie entweder das Home-Verzeichnis des Benutzers (Home Dir) oder das gesamte Dateisystem (All Files) durchsuchen. Die Option Alle Dateien kann lange dauern, insbesondere wenn Sie viele große oder NFS-gemountete Dateisysteme haben.
