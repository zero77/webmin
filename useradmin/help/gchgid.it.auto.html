Cambia GID sui file?
Quando si modifica un gruppo, se si modifica l' ID del gruppo, i file di proprietà di quel gruppo rimarranno di proprietà del vecchio GID. Questa opzione può essere utilizzata per aggiornare la proprietà dei file al nuovo GID cercando le home directory di tutti gli utenti membri di questo gruppo (Home Directories) o l'intero file system (Tutti i file). L'opzione Tutti i file potrebbe richiedere molto tempo, soprattutto se si dispone di molti file system di grandi dimensioni o montati su NFS.
