껍질
사용자가 텔넷 또는 콘솔을 통해 로그인하거나 X에서 xterm 창을 열면 사용자 쉘 프로그램이 실행되어 입력 된 명령을 해석합니다. 사용자에게 유효한 쉘 프로그램이 없거나 /bin/false 와 같이 아무것도하지 않는 프로그램이 있으면 해당 사용자는 로그인 할 수 없습니다. 전자 메일 만 읽고 보낼 수있는 사용자를 만들 때 유용합니다.

이 필드에는 시스템의 사용자가 사용하는 모든 쉘이 나열됩니다. 하나를 선택하거나 기타를 선택하고 아래 텍스트 필드에 쉘의 전체 경로를 입력하십시오.

모든 Unix 시스템은 Bourne 쉘 ( /bin/sh ) 및 C 쉘 ( /bin/csh )과 함께 제공됩니다. 보다 일반적이고 사용자 친화적 인 다른 쉘은 tcsh , zsh 및 bash 입니다.
