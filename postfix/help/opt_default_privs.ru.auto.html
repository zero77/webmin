Права по умолчанию локального агента доставки
default_privs
Этот параметр указывает права по умолчанию, используемые локальным агентом доставки для доставки во внешний файл или команду.

Эти права используются в отсутствие пользовательского контекста получателя. НЕ УКАЗЫВАЙТЕ ЧАСТНОГО ПОЛЬЗОВАТЕЛЯ ИЛИ ВЛАДЕЛЬЦА POSTFIX.
