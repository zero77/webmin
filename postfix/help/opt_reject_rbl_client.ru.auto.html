DNS-домены для поиска в черном списке
smtpd_client_restrictions: reject_rbl_client
Этот параметр указывает необязательный список доменов DNS, которые публикуют сетевые адреса хостов, занесенных в черный список.
По умолчанию поиск в черном списке RBL отключен.

Укажите список доменов RBL, разделенных запятыми.

Черный список в реальном времени работает следующим образом: отменить сетевой адрес клиента и отказаться от обслуживания, если он указан ниже любого из следующих доменов.
