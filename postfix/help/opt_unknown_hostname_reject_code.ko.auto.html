알 수없는 호스트 이름 거부에 대한 SMTP 서버 응답
unknown_hostname_reject_code
이 매개 변수는 클라이언트가 다음과 같은 reject_unknown_hostname 제한 규칙을 위반할 때 SMTP 서버 응답 코드를 지정합니다.
액세스 거부 : 알 수없는 클라이언트 HELO 호스트 이름 (DNS A 또는 MX 레코드 없음)

unknown_hostname_reject_code 사용되는 접미사 기본값은 코드 501이며 'standard RFC 822'의이 코드는 '매개 변수 또는 인수의 구문 오류'를 의미해야합니다. 따라서 Postfix에서 501 상태 코드는 RFC822 설명을 정확하게 반영하지 않습니다.

혼동을 피하기 위해 예를 들어 577로 변경할 수 있습니다 (또 다른 이유는 코드 501이 다른 종류의 오류에 대한 접미사 기본값 임). 그렇지 않으면 RFC 822를 완전히 이해하지 않는 한이를 변경하지 마십시오.
