Diritti predefiniti dell'agente di consegna locale
default_privs
Questo parametro consente di specificare i diritti predefiniti utilizzati dall'agente di recapito locale per la consegna a file o comando esterno.

Questi diritti vengono utilizzati in assenza di un contesto utente destinatario. NON SPECIFICARE UN UTENTE PRIVILEGIATO O IL PROPRIETARIO POSTFIX.
