Pour quel domaine recevoir du courrier
mydestination
Vous spécifiez les domaines que cette machine fournira localement, au lieu de les transférer vers une autre machine. La valeur par défaut est de recevoir du courrier pour la machine elle-même.

Machine locale : $myhostname, localhost.$mydomain
Domaine entier : $myhostname, localhost.$mydomain, $mydomain

Vous pouvez également spécifier le chemin complet d'un fichier, qui contiendra les noms des domaines pour lesquels cette machine distribuera le courrier localement.

Exemple: /etc/postfix/mydestination


Attention: pour éviter les boucles de distribution de courrier, vous devez répertorier tous les noms d'hôte de la machine, y compris $myhostname et localhost.$mydomain . NE LISTEZ PAS LES DOMAINES VIRTUELS ICI. LISTEZ-LES DANS LE FICHIER VIRTUEL AU LIEU.
