Bannière de voeux SMTP
smtpd_banner
Ce paramètre spécifie le texte qui suit le code d'état 220 dans la bannière d'accueil SMTP. Certaines personnes aiment voir la version mail annoncée. Par défaut, Postfix n'affiche aucune version.

Vous DEVEZ spécifier le $myhostname au début du texte. Lorsque le client SMTP voit son propre nom d'hôte au début d'une bannière d'accueil SMTP, il signale une boucle de messagerie. C'est mieux que d'avoir une fusion de machine.
