Адказ SMTP-сервера на адхіленне дамена
unknown_address_reject_code
Гэты параметр указвае код адказу SMTP-сервера, калі кліент парушае правілы абмежавання reject_unknown_sender_domain або reject_unknown_recipient_domain , якія:
Доступ забаронены: невядомы дамен для адпраўшчыка электроннай пошты адпраўніка або атрымальніка (без запісу DNS A або MX)

Звярніце ўвагу, што значэнне па змаўчанні постфікса, якое выкарыстоўваецца для unknown_address_reject_code з'яўляецца кодам 450 і выкарыстоўваецца і для іншых памылак. Гэты код у "стандартным RFC 822" азначае "Запытаныя дзеянні пошты не прадпрымаюцца: паштовая скрыня занятая альбо доступ забаронены" і папрасіць адпраўніка паспрабаваць яшчэ раз пазней (спадзеючыся, што DNS адкажа правільна пазней).

Не мяняйце гэта, калі вы не маеце поўнага разумення RFC 822.
