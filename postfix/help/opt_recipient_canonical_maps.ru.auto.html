Таблицы для адресов ПОЛУЧАТЕЛЯ
recipient_canonical_maps
Этот параметр указывает необязательные таблицы поиска сопоставления адресов для адресов получателей конвертов и заголовков.

По умолчанию сопоставление адресов только для получателей не выполняется.

$recipient_canonical_maps используется перед поиском в $canonical_maps .
