Mapeo de certificados
El archivo de asignación de certificados SNI asocia nombres de dominio con archivos de certificados, para usar cuando Postfix recibe una conexión SMTP en modo SSL. El formato de la tabla es:
  • Nombre de host : un nombre de host o comodín como *.example.com para que coincida con el nombre de host en la conexión SSL SNI .
  • Archivos de certificado : ruta a un archivo de formato PEM que contiene la clave privada y el certificado a utilizar.
