Pemetaan sijil
Fail pemetaan sijil SNI mengaitkan nama domain dengan fail sijil, untuk digunakan ketika Postfix menerima sambungan SMTP dalam mod SSL. Format jadual adalah:
  • Hostname - Nama host atau wildcard seperti *.example.com untuk memadankan nama host dalam sambungan SSL SNI .
  • Fail sijil - Laluan ke fail format PEM mengandungi kunci peribadi dan sijil untuk digunakan.
