Максимальное время доставки на внешние команды
command_time_limit
Этот параметр ограничивает время доставки для внешних команд. Этот лимит используется локальным агентом доставки и является лимитом времени по умолчанию для доставки по почте.

Примечание: если вы установите для этого ограничения времени большое значение, вы также должны обновить глобальный параметр ipc_timeout . Посмотрите Postfix Разные Опции для деталей.
