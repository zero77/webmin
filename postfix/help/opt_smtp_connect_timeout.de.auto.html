Zeitüberschreitung beim Abschluss von TCP-Verbindungen
smtp_connect_timeout
Dieser Parameter gibt das SMTP-Client-Zeitlimit in Sekunden für den Abschluss einer TCP-Verbindung an.

Wenn innerhalb der Frist keine Verbindung hergestellt werden kann, versucht der SMTP-Client die nächste Adresse in der Mail-Exchange-Liste. Geben Sie 0 an, um das Zeitlimit zu deaktivieren.
