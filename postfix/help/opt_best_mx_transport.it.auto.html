Azione quando elencato come miglior host MX
best_mx_transport
Questo parametro controlla cosa succede quando il sistema locale è elencato come il miglior host MX per una destinazione. Per impostazione predefinita, Postfix segnala un errore "Torna indietro a me stesso" e rimbalza il messaggio.

Specificare local per passare la posta all'agente di consegna locale. È possibile specificare qualsiasi trasporto definito nel file master.cf Postfix.
