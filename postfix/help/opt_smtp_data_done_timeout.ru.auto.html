Тайм-аут ожидания ответа на окончание "."
smtp_data_done_timeout
Этот параметр указывает время ожидания SMTP-клиента в секундах для отправки SMTP "." И для получения ответа сервера.

Если в течение указанного срока ответ не получен, регистрируется предупреждение о том, что почта может быть доставлена несколько раз.

В случае возникновения проблем клиент НЕ пробует следующий адрес в списке почтовых обменников.
