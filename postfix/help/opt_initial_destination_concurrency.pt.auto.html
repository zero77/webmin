Nível de simultaneidade inicial para entrega no mesmo destino
initial_destination_concurrency
Este parâmetro especifica o nível de simultaneidade inicial por destino para entrega paralela no mesmo destino. Esse limite se aplica à entrega via SMTP, via agente de entrega local e via mala direta.

Com a simultaneidade de 1, uma mensagem incorreta é suficiente para bloquear todos os emails em um site. Uma simultaneidade de 2 parece uma escolha razoável.
