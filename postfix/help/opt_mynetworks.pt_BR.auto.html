Redes locais
mynetworks
Este parâmetro especifica a lista de redes locais para esta máquina. A lista é usada pelo software anti-UCE para distinguir clientes locais de estranhos. Consulte permit_mynetworks e smtpd_recipient_restrictions na configuração SMTP.

O padrão é uma lista de todas as redes conectadas à máquina: uma rede completa da classe A (X.0.0.0 / 8), uma rede completa da classe B (XX0.0 / 16) e assim por diante. Se você deseja um controle mais rígido, especifique uma lista de padrões de rede / máscara, em que a máscara especifica o número de bits na parte da rede de um endereço de host. Você também pode especificar o nome do caminho absoluto de um arquivo de padrão em vez de listar os padrões aqui.
