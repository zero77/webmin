Спіс пошуку для наперад
forward_path
Гэты параметр вызначае спіс пошуку. Першы знойдзены файл выкарыстоўваецца.

Наступныя пашырэння:

  • $user (імя атрымальніка),
  • $shell (абалонка атрымальніка),
  • $home (хатні каталог атрымальніка),
  • $recipient (поўны адрас атрымальніка),
  • $extension (пашырэнне адраса атрымальніка),
  • $domain (дамен атрымальніка),
  • $local (увесь атрымальнік мясцовай часткі),
  • $recipient_delimiter .
Укажыце ${name?value} або ${name:value} каб пашырыць значэнне толькі тады, калі $name сапраўды існуе (не існуе).
