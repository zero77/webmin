Quel domaine utiliser dans le courrier sortant
myorigin
Vous spécifiez le domaine qui apparaît dans le courrier publié sur cette machine. La configuration par défaut de Postfix consiste à utiliser le nom de la machine locale, le nom d' hôte .

À moins que vous utilisez un site vraiment petit, vous voulez probablement changer cela en nomdomaine, qui par défaut dans le domaine parent du nom de la machine.
