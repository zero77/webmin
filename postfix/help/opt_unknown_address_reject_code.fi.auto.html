SMTP-palvelimen vastaus verkkotunnuksen hylkäämiseen
unknown_address_reject_code
Tämä parametri määrittää SMTP-palvelimen vastauskoodin, kun asiakas rikkoo reject_unknown_sender_domain tai reject_unknown_recipient_domain rajoitussääntöjä, jotka ovat:
Käyttö evätty: Lähettäjän tai vastaanottajan sähköpostiosoitteen tuntematon verkkotunnus (ilman DNS A- tai MX-tietuetta)

Huomaa, että postfix oletusarvo käytetty unknown_address_reject_code on koodin 450 käytetään myös muita virheitä. Tämä koodi 'standardissa RFC 822' tarkoittaa 'pyydettyä postitoimintaa ei suoritettu: postilaatikko on varattu tai käyttö evätty' ja pyydä lähettäjää yrittämään myöhemmin uudelleen (toivoen, että DNS vastaa myöhemmin oikein).

Älä muuta tätä, ellei sinulla ole täysin ymmärrystä RFC 822: sta.
