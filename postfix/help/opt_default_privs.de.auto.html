Standardrechte des lokalen Zustellungsagenten
default_privs
Dieser Parameter gibt die Standardrechte an, die vom lokalen Übermittlungsagenten für die Übermittlung an eine externe Datei oder einen externen Befehl verwendet werden.

Diese Rechte werden verwendet, wenn kein Empfängerbenutzerkontext vorhanden ist. Geben Sie weder einen privilegierten Benutzer noch den Postfix-Besitzer an.
