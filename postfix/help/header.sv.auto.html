Header Checks
Den valfria rubrikkontrolltabellen gör det möjligt att omdirigera e-post baserat på dess rubriker. Varje post i tabellen har ett POSIX-reguljärt uttryck på vänster sida och en åtgärd till höger.

Det vanliga uttrycket kan vara något som:
/name=[^>]*\.(bat|cmd|exe|com|pif|reg|scr|vb|vbe|vbs)/
för att matcha bilagor vars filnamn slutar med tillägg som anger potentiellt farliga Windows-körbara filer.

Åtgärden kan vara som:
REJECT
för att helt enkelt avvisa e-post som matchar uttrycket, eller:
REDIRECT spam@yourdomain.com
För att vidarebefordra e-post till en annan adress.
