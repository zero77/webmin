Vérifier la table de mappage d'accès des expéditeurs
smtpd_client_restrictions: check_sender_access

Recherchez dans la table d'accès nommée le nom de l'expéditeur, les domaines, l'adresse ou les réseaux obtenus en supprimant les octets les moins significatifs.

Le paramètre access_map_reject_code spécifie le code de réponse pour les résultats REJECT (par défaut: 550).
