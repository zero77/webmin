Имя очереди dirs разбито по подкаталогам
hash_queue_names
Этот параметр указывает имена каталогов очереди, которые разделены на несколько уровней подкаталогов. Хеширование НЕ ДОЛЖНО использоваться с всемирно доступным каталогом maildrop. Хеширование ДОЛЖНО использоваться для каталога отсроченного файла журнала, иначе производительность почтовой системы пострадает.

К сожалению, хэширование входящей или отложенной очереди может фактически замедлить работу почтовой системы ( mailq с пустой очередью может занять несколько секунд), поэтому это следует делать только в случае крайней необходимости.
