Reti locali
mynetworks
Questo parametro consente di specificare l'elenco di reti locali per questa macchina. L'elenco è utilizzato dal software anti-UCE per distinguere i client locali dagli estranei. Vedi permit_mynetworks e smtpd_recipient_restrictions nella configurazione SMTP.

L'impostazione predefinita è un elenco di tutte le reti collegate alla macchina: una rete completa di classe A (X.0.0.0 / 8), una rete completa di classe B (XX0.0 / 16) e così via. Se si desidera un controllo più rigoroso, specificare un elenco di modelli di rete / maschera, in cui la maschera specifica il numero di bit nella parte di rete di un indirizzo host. Puoi anche specificare il percorso assoluto di un file di pattern invece di elencare qui i pattern.
