Elenco di modelli di dominio / rete per i quali è abilitato il registro dettagliato
debug_peer_list
Questo parametro consente di specificare un elenco facoltativo di modelli di dominio o di rete, modelli /file/name o type:name tabelle dei type:name . Quando il nome o l'indirizzo di un client o server SMTP corrisponde a un modello, aumentare il livello di registrazione dettagliato della quantità specificata nel parametro debug_peer_level .

Valori possibili sensibili:

  • 127.0.0.1
  • some.domain
