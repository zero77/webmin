Shell που θα χρησιμοποιηθεί για παράδοση σε εξωτερική εντολή
local_command_shell
Αυτή η παράμετρος ελέγχει τι κέλυφος θα χρησιμοποιηθεί για παράδοση σε εξωτερική εντολή. Από προεπιλογή, οι εξωτερικές εντολές εκτελούνται απευθείας. Οι εντολές δίδονται στο /bin/sh μόνο όταν περιέχουν μετα-χαρακτήρες κελύφους ή ενσωματωμένες εντολές κελύφους.

Καθορίστε ένα διαφορετικό κέλυφος όταν χρειάζεστε περισσότερο έλεγχο για τα προγράμματα που μπορούν να εκτελεστούν από π.χ. αρχεία .forward .

Το "περιορισμένο κέλυφος sendmail" smrsh είναι αυτό που θα χρησιμοποιούν οι περισσότεροι άνθρωποι (το smrsh είναι μέρος των πρόσφατων διανομών sendmail)

Σημείωση: όταν καθορίζεται ένα κέλυφος, καλείται ακόμη και όταν η εντολή δεν περιέχει ενσωματωμένες εντολές κελύφους ή μετα-χαρακτήρες.

Παράδειγμα: /some/where/smrsh -c
