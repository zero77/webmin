Restringir a retransmissão de email
relay_domains
Este parâmetro restringe para quais domínios de nome de host do cliente (e subdomínios) este sistema de email retransmitirá o correio e restringe para quais domínios de destino (e subdomínios) este sistema retransmitirá o email.

Especifique uma lista de hosts ou domínios, /file/name padrões de /file/name ou type:name tabelas de consulta de type:name , separadas por vírgulas e / ou espaço em branco. Um nome de arquivo é substituído por seu conteúdo; uma tabela type:name é correspondida quando um domínio (pai) aparece como chave de pesquisa.

NOTA: O Postfix não encaminhará automaticamente emails para domínios que listam este sistema como host MX principal ou de backup. Consulte a restrição permit_mx_backup , na descrição do parâmetro smtpd_recipient_restrictions .
