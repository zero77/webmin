DNS-Domains für Blacklist-Lookups
smtpd_client_restrictions: reject_rbl_client
Dieser Parameter gibt eine optionale Liste von DNS-Domänen an, die die Netzwerkadressen von Hosts auf der schwarzen Liste veröffentlichen.
Standardmäßig sind RBL-Blacklist-Lookups deaktiviert.

Geben Sie eine Liste von RBL-Domänen an, die durch Kommas getrennt sind.

Die Blackhole-Liste in Echtzeit funktioniert wie folgt: Kehren Sie die Client-Netzwerkadresse um und lehnen Sie den Dienst ab, wenn er unter einer der folgenden Domänen aufgeführt ist.
