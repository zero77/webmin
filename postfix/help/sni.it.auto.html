Mappatura del certificato
Il file di mappatura del certificato SNI associa i nomi di dominio ai file di certificato, da utilizzare quando Postfix riceve una connessione SMTP in modalità SSL. Il formato della tabella è:
  • Nome host : nome host o carattere jolly come *.example.com per abbinare il nome host nella connessione SSL SNI .
  • File di certificato : percorso di un file in formato PEM contenente la chiave privata e il certificato da utilizzare.
