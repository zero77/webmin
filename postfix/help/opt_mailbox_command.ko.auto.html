사서함 배달 대신 사용할 외부 명령
mailbox_command
이 매개 변수는 사서함 배달 대신 사용할 선택적 외부 명령을 지정합니다. 적절한 HOME , SHELL 및 LOGNAME 환경 설정을 사용하여 수신자로 명령이 실행됩니다. 예외 : root 전달은 $default_user 로 수행됩니다.

관심있는 다른 환경 변수 : USER (받는 사람 사용자 이름), EXTENSION (주소 확장), DOMAIN (주소의 도메인 부분) 및 LOCAL (주소 localpart).

다른 Postfix 구성 매개 변수와 달리이 매개 변수는 매개 변수를 대체하지 않습니다. 쉘 구문을보다 쉽게 지정할 수 있습니다 (아래 예 참조).

쉘 메타 문자는 Postfix가 비싼 쉘 프로세스를 강제로 실행하게하므로 피하십시오. Procmail 만으로도 비싸다.

이 기능을 사용하여 메일 시스템 전체를 전달하는 경우 실제 사용자에게 루트를 전달할 메일을 전달할 별칭을 설정해야합니다.

예 :

  • /some/where/procmail
  • /some/where/procmail -a "$EXTENSION"
