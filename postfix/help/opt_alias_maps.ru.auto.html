Базы данных псевдонимов, используемые локальным агентом доставки
alias_maps
Этот параметр указывает список баз данных псевдонимов, используемых локальным агентом доставки. Список по умолчанию зависит от системы. В системах с NIS по умолчанию выполняется поиск в локальной базе данных псевдонимов, а затем в базе данных псевдонимов NIS. Смотрите псевдонимы для деталей синтаксиса.

Типичные конфигурации:

  • dbm:/etc/aliases
  • hash:/etc/postfix/aliases
  • hash:/etc/aliases, nis:mail.aliases
