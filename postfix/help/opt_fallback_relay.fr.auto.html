Hôtes / domaines vers lesquels transférer le courrier sur une destination non valide
fallback_relay
Ce paramètre spécifie zéro ou plusieurs hôtes ou domaines vers lesquels transférer le courrier si une destination de message n'est pas trouvée ou si une destination est inaccessible.

Par défaut, le courrier est renvoyé lorsqu'une destination n'est pas trouvée et la livraison est différée si une destination est inaccessible.
