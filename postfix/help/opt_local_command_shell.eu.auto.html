Shell kanpoko aginduetara entregatzeko erabili
local_command_shell
Parametro honek kanpoko aginduei entregatzeko zer shell erabiliko den kontrolatzen du. Berez, kanpoko komandoak zuzenean exekutatzen dira; komandoak /bin/sh i ematen zaizkie shell meta karaktereak edo shell integratutako komandoak dituztenean bakarrik.

Zehaztu ezberdinak oskol denean zer programa egon adibidez egikaritu ahal kontrol gehiago behar duzu .forward fitxategiak.

"sendmail-en murriztutako shell" smrsh da jende gehienak erabiliko duena ( smrsh azken banaketa banaketaren zati bat da)

Oharra: shell bat zehazten denean, berau agintzen da, nahiz eta komandoak ez dituen osatutako komandoak edo meta karaktereak.

Adibidea: /some/where/smrsh -c
