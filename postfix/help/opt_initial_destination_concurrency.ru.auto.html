Начальный уровень параллелизма для доставки в тот же пункт назначения
initial_destination_concurrency
Этот параметр указывает начальный уровень параллелизма для каждого назначения для параллельной доставки в тот же пункт назначения. Это ограничение применяется к доставке через SMTP, через локального агента доставки и через почтовую программу.

При параллельности 1 достаточно одного плохого сообщения, чтобы заблокировать всю почту на сайте. Параллелизм 2 кажется разумным выбором.
