Maximale Zeit für die Übermittlung an externe Befehle
command_time_limit
Dieser Parameter begrenzt die Zeit für die Übermittlung an externe Befehle. Dieses Limit wird vom lokalen Zustellungsagenten verwendet und ist das Standardzeitlimit für die Zustellung durch den Pipe Mailer.

Hinweis: Wenn Sie dieses Zeitlimit auf einen großen Wert ipc_timeout müssen Sie auch den globalen Parameter ipc_timeout aktualisieren. Weitere Informationen finden Sie unter Verschiedene Optionen für Postfix .
