Header-Checks
Die optionale Header-Prüftabelle ermöglicht die Umleitung von E-Mails basierend auf ihren Headern. Jeder Eintrag in der Tabelle hat auf der linken Seite einen regulären POSIX-Ausdruck und auf der rechten Seite eine Aktion.

Der reguläre Ausdruck kann so etwas sein wie:
/name=[^>]*\.(bat|cmd|exe|com|pif|reg|scr|vb|vbe|vbs)/
um Anhänge abzugleichen, deren Dateinamen mit Erweiterungen enden, die potenziell gefährliche ausführbare Windows-Dateien anzeigen.

Die Aktion kann sein wie:
REJECT
um einfach E-Mails abzulehnen, die dem Ausdruck entsprechen, oder:
REDIRECT spam@yourdomain.com
Mail an eine andere Adresse weiterleiten.
