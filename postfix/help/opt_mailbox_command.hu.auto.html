Külső parancs a postafiók kézbesítése helyett
mailbox_command
Ez a paraméter a választható külső parancsot adja meg, amelyet a postafiók kézbesítése helyett használni kell. A parancs címzettként fut, megfelelő HOME , SHELL és LOGNAME környezeti beállításokkal. Kivétel: a root kézbesítése $default_user .

Egyéb érdeklődő környezeti változók: USER (címzett felhasználóneve), EXTENSION ( EXTENSION ), DOMAIN (a domain domain része) és LOCAL (a cím localpart).

Más Postfix konfigurációs paraméterekkel ellentétben ezt a paramétert nem érintik paraméterek. Ez megkönnyíti a shell szintaxisának meghatározását (lásd az alábbi példát).

Kerülje a héj meta karaktereket, mert ezek kényszerítik a Postfix-ot egy drága shell folyamat futtatására. Procmail önmagában elég drága.

HOGYAN HASZNÁLJA EZT A POSTA RENDSZERBEN SZÁLLÍTÁSÁHOZ, BEÁLLÍTANA ALIÁT, AMELYEK AZ ELSŐ TÁMOGATÁSA A VALÓ FELHASZNÁLÓ SZÁMÁRA.

Példák:

  • /some/where/procmail
  • /some/where/procmail -a "$EXTENSION"
