Максимальное количество получателей на одну доставку сообщения
default_destination_recipient_limit
Параметр указывает ограничение по умолчанию для числа получателей на одну доставку сообщения. Это ограничение по умолчанию для доставки через SMTP, через локального агента доставки и через почтовую программу.
