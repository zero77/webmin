Stellen Sie eine Lieferung vor: wann ...
prepend_delivered_header
Dieser Parameter steuert, wann Postfix einen Message-Header für " Delivered-To: voranstellen soll.

Standardmäßig stellt Postfix beim Weiterleiten von E-Mails und bei der Zustellung an eine Datei (Postfach) und einen Befehl einen Übermittlungs Delivered-To: Header voran. Das Deaktivieren des Headers " Delivered-To: beim Weiterleiten von E-Mails wird nicht empfohlen.
