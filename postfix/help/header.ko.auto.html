헤더 확인
선택적 헤더 검사 테이블을 사용하면 헤더를 기반으로 이메일을 리디렉션 할 수 있습니다. 테이블의 각 항목에는 왼쪽에 POSIX 정규식이 있고 오른쪽에는 작업이 있습니다.

정규식은 다음과 같습니다.
/name=[^>]*\.(bat|cmd|exe|com|pif|reg|scr|vb|vbe|vbs)/
잠재적으로 위험한 Windows 실행 파일을 나타내는 확장명으로 파일 이름이 끝나는 첨부 파일을 일치시킵니다.

동작은 다음과 같습니다.
REJECT
단순히 표현식과 일치하는 메일을 거부하거나 :
REDIRECT spam@yourdomain.com
다른 주소로 메일을 전달합니다.
