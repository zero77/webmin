Псевдоним базы данных, созданные Postfix
alias_database
Этот параметр указывает базы данных псевдонимов, которые создаются с помощью «newaliases» или «sendmail -bi». Это отдельный параметр конфигурации, поскольку alias_maps может указывать таблицы, которые не обязательно находятся под контролем Postfix.

Типичные конфигурации:

  • dbm:/etc/aliases
  • dbm:/etc/mail/aliases
  • hash:/etc/aliases
  • hash:/etc/aliases, hash:/opt/majordomo/aliases
