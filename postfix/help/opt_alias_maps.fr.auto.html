Bases de données d'alias utilisées par l'agent de livraison local
alias_maps
Ce paramètre spécifie la liste des bases de données d'alias utilisées par l'agent de remise local. La liste par défaut dépend du système. Sur les systèmes avec NIS, la valeur par défaut consiste à rechercher la base de données d'alias locale, puis la base de données d'alias NIS. Voir les alias pour les détails de la syntaxe.

Les configurations typiques sont:

  • dbm:/etc/aliases
  • hash:/etc/postfix/aliases
  • hash:/etc/aliases, nis:mail.aliases
