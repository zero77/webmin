금지 된 릴레이에 대한 SMTP 서버 응답
relay_domains_reject_code
이 매개 변수는 메일이 다음과 같은 relay_domains_reject 규칙을 위반할 때 SMTP 서버 응답 코드를 지정합니다.
액세스 거부 : 릴레이 권한이 없거나 릴레이가 아닌 로컬 호스트

relay_domains_reject_code 사용되는 접미사 기본값은 코드 554이며 '표준 RFC 822'의이 코드는 '요청 된 메일 작업이 거부되었습니다 : 액세스가 거부되었습니다'를 의미해야합니다. 따라서 Postfix에서 relay_domains_reject 규칙 위반은보고 된 코드에 대한 RFC822 설명을 정확하게 반영하지 않습니다.

혼동을 피하기 위해 예를 들어 572로 변경할 수 있습니다 (또 다른 이유는 코드 554가 다른 종류의 오류에 대한 접미사 기본값이기도 함). 그렇지 않으면 RFC 822를 완전히 이해하지 않는 한이를 변경하지 마십시오.
