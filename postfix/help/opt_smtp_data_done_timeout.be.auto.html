smtp_data_done_timeout
Гэты параметр паказвае час чакання SMTP кліента ў секундах для адпраўкі SMTP "." І для атрымання адказу сервера.

Калі адказ не атрыманы ў тэрмін, улічваецца папярэджанне аб тым, што пошта можа быць дастаўлена некалькі разоў.

У выпадку ўзнікнення праблем кліент НЕ спрабуе наступны адрас у спісе абменнікаў пошты.
