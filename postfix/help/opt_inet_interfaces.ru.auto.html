Сетевые интерфейсы для получения почты
inet_interfaces
Этот параметр указывает адреса сетевого интерфейса, по которым эта почтовая система получает почту.

По умолчанию программа запрашивает все активные интерфейсы на машине. Параметр также управляет доставкой почты user@[ip.address] .

Типичные конфигурации:

  • all
  • $myhostname
  • $myhostname, localhost
