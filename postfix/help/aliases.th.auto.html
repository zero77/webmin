Mail Aliases
นามแฝงจัดเตรียมกลไกทั่วทั้งระบบเพื่อเปลี่ยนเส้นทางเมลสำหรับผู้รับในท้องถิ่น

ฟิลด์ Alias to... มีหนึ่งรายการขึ้นไปคั่นด้วยเครื่องหมายจุลภาค:

  • address : จดหมายถูกส่งต่อไปยังที่ address ซึ่งเข้ากันได้กับมาตรฐาน RFC 822
  • /file/name : เมลถูกต่อท้ายไปยัง /file/name ดู local (8) สำหรับรายละเอียดการจัดส่งไปยังไฟล์ การจัดส่งไม่ จำกัด เฉพาะไฟล์ทั่วไป ตัวอย่างเช่นหากต้องการกำจัดจดหมายที่ไม่ต้องการให้เบี่ยงเบนไปที่ /dev/null
  • |command : Mail ถูกส่งไปยัง command คำสั่งที่มีอักขระพิเศษเช่นช่องว่างควรอยู่ระหว่างเครื่องหมายคำพูดคู่ ดูท้องถิ่น (8) สำหรับรายละเอียดการจัดส่งไปยังคำสั่ง เมื่อคำสั่งล้มเหลวเอาต์พุตคำสั่งจำนวน จำกัด จะถูกส่งทางไปรษณีย์กลับไปยังผู้ส่ง
  • :include:/file/name : เมลจะถูกส่งไปยังปลายทางที่ระบุไว้ในไฟล์ที่มีชื่อ บรรทัดใน :include: ไฟล์มีไวยากรณ์เหมือนกับด้านขวาของรายการนามแฝง
