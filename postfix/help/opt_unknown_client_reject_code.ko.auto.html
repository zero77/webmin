알 수없는 클라이언트 거부에 대한 SMTP 서버 응답
unknown_client_reject_code
이 매개 변수는 주소 대 이름 맵핑이없는 클라이언트가 다음과 같은 reject_unknown_clients 제한 규칙을 위반할 때 SMTP 서버 응답 코드를 지정합니다.
액세스 거부 : 알 수없는 SMTP 클라이언트 호스트 이름 (DNS A 또는 MX 레코드 없음)

unknown_client_reject_code 사용되는 접미사 기본값은 다른 오류에도 사용되는 코드 450입니다. '표준 RFC 822'의이 코드는 '요청 된 메일 작업이 수행되지 않음 : 사서함 사용 중이거나 액세스가 거부되었습니다'를 의미하며 보낸 사람에게 나중에 다시 시도하도록 요청합니다 (DNS를 호핑하면 나중에 올바르게 응답합니다).

RFC 822에 대한 완전한 이해가 없으면 이것을 변경하지 마십시오.
