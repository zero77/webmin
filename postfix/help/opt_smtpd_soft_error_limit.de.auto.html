Fehleranzahl für das vorübergehende Ignorieren eines Clients
smtpd_soft_error_limit
Dieser Parameter gibt eine Untergrenze für die Fehleranzahl an. Wenn ein SMTP-Client diese Anzahl von Fehlern innerhalb einer Sitzung gemacht hat, wartet der Server auf error_count Sekunden, bevor er auf eine Clientanforderung antwortet.
