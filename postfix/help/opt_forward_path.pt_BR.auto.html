Lista de pesquisa para encaminhar
forward_path
Este parâmetro especifica uma lista de pesquisa. O primeiro arquivo encontrado é usado.

As seguintes expansões são feitas:

  • $user (nome de $user destinatário),
  • $shell (shell do destinatário),
  • $home (diretório inicial do destinatário),
  • $recipient (endereço completo do destinatário),
  • $extension (extensão de endereço do destinatário),
  • $domain (domínio do destinatário),
  • $local (parte $local destinatário inteiro),
  • $recipient_delimiter .
Especifique ${name?value} ou ${name:value} para expandir o valor somente quando $name existir (não).
