Verificar tabela de mapeamento de acesso ao remetente
smtpd_client_restrictions: check_sender_access

Pesquise na tabela de acesso nomeado o nome do remetente, domínios, endereço ou redes obtidos removendo octetos menos significativos.

O parâmetro access_map_reject_code especifica o código de resposta para os resultados de REJECT (padrão: 550).
