Shell akan digunakan untuk penghantaran ke perintah luaran
local_command_shell
Parameter ini mengawal shell apa yang akan digunakan untuk penghantaran ke perintah luaran. Secara lalai, arahan luaran dilaksanakan secara langsung; perintah diberikan kepada /bin/sh hanya apabila mengandungi karakter shell meta atau shell shell built-in.

Tentukan shell yang berbeza apabila anda memerlukan lebih banyak kawalan terhadap program apa yang dapat dijalankan dari fail .forward .

"shell terhad terhad sendmail" smrsh adalah yang akan digunakan oleh kebanyakan orang ( smrsh adalah sebahagian daripada pengedaran sendmail baru-baru ini)

Catatan: ketika shell ditentukan, ia dipanggil walaupun perintah tidak berisi perintah built-in shell atau karakter meta.

Contoh: /some/where/smrsh -c
