Liste der Domänen- / Netzwerkmuster, für die das ausführliche Protokoll aktiviert ist
debug_peer_list
Dieser Parameter gibt eine optionale Liste von Domänen- oder Netzwerkmustern, /file/name Namensmustern oder type:name . Wenn der Name oder die Adresse eines SMTP-Client- oder Server-Hosts mit einem Muster übereinstimmt, erhöhen Sie die ausführliche Protokollierungsstufe um den im Parameter debug_peer_level angegebenen Betrag.

Sinnvolle mögliche Werte:

  • 127.0.0.1
  • einige.Domäne
