Nombre de niveaux de sous-répertoires en dessous du répertoire de file d'attente
hash_queue_depth
Ce paramètre spécifie le nombre de niveaux de sous-répertoire sous les répertoires de files d'attente répertoriés dans le paramètre hash_queue_names .

Plusieurs niveaux de sous-répertoires peuvent accélérer l'accès aux répertoires en réduisant le nombre de fichiers par répertoire.
