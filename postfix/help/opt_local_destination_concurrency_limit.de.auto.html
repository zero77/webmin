Maximale Anzahl paralleler Lieferungen an denselben lokalen Empfänger
local_destination_concurrency_limit
Dieser Parameter begrenzt die Anzahl der parallelen Lieferungen an denselben lokalen Empfänger.

Das Standardlimit wird aus dem Parameter default_destination_concurrency_limit . In der Postfix-Dokumentation wird eine Untergrenze von 2 empfohlen, nur für den Fall, dass jemand einen teuren Shell-Befehl in einer .forward Datei oder in einem Alias (z. B. einem Mailinglisten-Manager) hat. Sie wollen nicht viele davon laufen lassen.
