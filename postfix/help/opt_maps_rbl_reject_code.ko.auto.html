RBL 도메인 위반에 대한 SMTP 서버 응답
maps_rbl_reject_code
이 매개 변수는 메일이 다음과 같은 maps_rbl_domains 규칙을 위반할 때 SMTP 서버 응답 코드를 지정합니다.
액세스 거부 : RBL에 나열된 SMTP 클라이언트

maps_rbl_reject_code 사용되는 접미사 기본값은 코드 554이며 '표준 RFC 822'의이 코드는 '요청 된 메일 작업 거부 : 액세스 거부'를 의미해야합니다. 따라서 Postfix에서 maps_rbl_domains 규칙 위반은보고 된 코드에 대한 RFC822 설명을 정확하게 반영하지 않습니다.

혼동을 피하기 위해 예를 들어 571로 변경할 수 있습니다 (또 다른 이유는 코드 554가 다른 종류의 오류에 대한 접미사 기본값이기도 함). 그렇지 않으면 RFC 822를 완전히 이해하지 않는 한이를 변경하지 마십시오.
