Odmítnout neznámé místní uživatele
local_recipient_maps
Tento parametr určuje volitelné vyhledávací tabulky se všemi uživateli, kteří jsou místní s ohledem na $mydestination ("pro které domény mají přijímat poštu") a $inet_interfaces ("síťová rozhraní pro příjem pošty"). Pokud je tento parametr definován, server SMTP odmítne poštu pro neznámé místní uživatele.

Vložte tabulky s holými uživatelskými unix:passwd.byname jako jsou unix:passwd.byname a unix:passwd.byname mapy. Příklad: $alias_maps unix:passwd.byname .

Pozor: Pokud server SMTP Postfix běží chrootován, možná budete muset kopírovat databázi passwd do vězení. To je závislé na systému.
