Host / domain untuk menghantar surat ke destinasi yang tidak sah
fallback_relay
Parameter ini menentukan hos atau domain sifar atau lebih untuk menyerahkan surat jika tujuan mesej tidak dijumpai, atau jika tujuan tidak dapat dicapai.

Secara lalai, surat akan dilambung apabila destinasi tidak dijumpai, dan penghantaran ditangguhkan sekiranya destinasi tidak dapat dicapai.
