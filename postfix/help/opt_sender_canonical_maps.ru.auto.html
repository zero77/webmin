Таблицы для адресов SENDER
sender_canonical_maps
Этот параметр указывает необязательные таблицы поиска сопоставления адресов для адресов SENDER конвертов и заголовков.

По умолчанию сопоставление адресов только для отправителей не выполняется.

$sender_canonical_maps используется перед $sender_canonical_maps в $canonical_maps .
