Rejeitar usuários locais desconhecidos
local_recipient_maps
Este parâmetro especifica tabelas de pesquisa opcionais com todos os usuários locais em relação a $mydestination ("para quais domínios receber email") e $inet_interfaces ("interfaces de rede para recebimento de email"). Se esse parâmetro for definido, o servidor SMTP rejeitará os emails de usuários locais desconhecidos.

Coloque tabelas com nomes de usuário simples, como mapas unix:passwd.byname e alias. Exemplo: $alias_maps unix:passwd.byname .

Cuidado: se o servidor SMTP do Postfix for executado com chroot, talvez você precise copiar o banco de dados da senha na prisão. Isso depende do sistema.
