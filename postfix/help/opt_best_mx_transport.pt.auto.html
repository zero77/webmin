Ação quando listado como o melhor host MX
best_mx_transport
Este parâmetro controla o que acontece quando o sistema local é listado como o melhor host MX de um destino. Por padrão, o Postfix relata um erro "e-mail volta para mim" e devolve a mensagem.

Especifique local para passar o email para o agente de entrega local. Você pode especificar qualquer transporte definido no arquivo master.cf Postfix.
