Nom des répertoires de file d'attente répartis entre les sous-répertoires
hash_queue_names
Ce paramètre spécifie les noms des répertoires de files d'attente qui sont répartis sur plusieurs niveaux de sous-répertoire. Le hachage NE DOIT PAS être utilisé avec un répertoire maildrop accessible en écriture. Le hachage DOIT être utilisé pour le répertoire de différer le fichier journal, sinon les performances du système de messagerie en souffriront.

Malheureusement, le hachage de la file d'attente entrante ou différée peut en fait ralentir le système de messagerie ( mailq avec une file d'attente vide peut prendre plusieurs secondes), il ne doit donc être effectué qu'en cas d'urgence.
