Адрес назначения для неизвестных получателей
luser_relay
Этот параметр указывает необязательный адрес назначения для неизвестных получателей. По умолчанию почта для неизвестных локальных получателей отклоняется.

Следующие расширения сделаны:

  • $user (имя $user получателя),
  • $shell (оболочка получателя),
  • $home (домашний каталог получателя),
  • $recipient (полный адрес получателя),
  • $extension (расширение адреса получателя),
  • $domain (домен получателя),
  • $local (весь получатель localpart),
  • $recipient_delimiter .
Укажите ${name?value} или ${name:value} чтобы раскрыть значение только тогда, когда $name существует (не существует).

Примеры:

  • $user@other.host
  • $local@other.host
  • admin+$local
