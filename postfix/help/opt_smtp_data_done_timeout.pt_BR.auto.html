Tempo limite ao aguardar resposta para o final de "."
smtp_data_done_timeout
Este parâmetro especifica o tempo limite do cliente SMTP em segundos para enviar o SMTP "." E para receber a resposta do servidor.

Quando nenhuma resposta é recebida dentro do prazo, é registrado um aviso de que o email pode ser entregue várias vezes.

Em caso de problemas, o cliente NÃO tenta o próximo endereço na lista de caixas de correio.
