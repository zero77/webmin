Délai d'expiration pour terminer les connexions TCP
smtp_connect_timeout
Ce paramètre spécifie le délai d'expiration du client SMTP en secondes pour terminer une connexion TCP.

Lorsqu'aucune connexion ne peut être établie dans le délai imparti, le client SMTP essaie la prochaine adresse dans la liste des échangeurs de courrier. Spécifiez 0 pour désactiver le délai.
