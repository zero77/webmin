Απόκριση διακομιστή SMTP στην απόρριψη πελάτη
reject_code
Αυτή η παράμετρος καθορίζει τον κωδικό απόκρισης διακομιστή SMTP όταν ένας πελάτης παραβιάζει τον κανόνα smtpd_client_restrictions που είναι:
Δεν επιτρέπεται η πρόσβαση: Το όνομα κεντρικού υπολογιστή πελάτη SMTP απορρίφθηκε

Σημειώστε ότι η προεπιλεγμένη τιμή μετά την reject_code χρησιμοποιείται για τον κωδικό reject_code είναι ο κωδικός 554 και αυτός ο κωδικός στο «τυπικό RFC 822» πρέπει να σημαίνει «Απορρίφθηκε η ενέργεια αλληλογραφίας που ζητήθηκε: η πρόσβαση δεν επιτρέπεται». Έτσι, με το Postfix, μια smtpd_client_restrictions κανόνα smtpd_client_restrictions δεν αντικατοπτρίζει ακριβώς την εξήγηση του RFC822 για τον αναφερόμενο κώδικα.

Για να αποφύγετε τη σύγχυση, μπορείτε να το αλλάξετε σε 575 για παράδειγμα (ένας άλλος λόγος είναι ότι ο κωδικός 554 είναι επίσης η προεπιλεγμένη τιμή μετά την επιδιόρθωση για άλλα είδη σφαλμάτων), διαφορετικά, μην το αλλάξετε εκτός εάν έχετε πλήρη κατανόηση του RFC 822.
