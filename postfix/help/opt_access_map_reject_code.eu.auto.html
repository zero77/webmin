SMTP zerbitzariaren erantzuna sarbide maparen urratzeari buruz
access_map_reject_code
Parametro honek SMTP zerbitzariaren erantzun kodea zehazten du mezu elektroniko batek access map murrizketa hausten duenean:
Sarbidea ukatu da: access_map urratzea (SMTP bezeroan edo HELO ostalariaren izenean, igorlearen edo hartzailearen helbide elektronikoan)

Kontuan izan access_map_reject_code rako erabiltzen den access_map_reject_code balio lehenetsia 554 kodea dela, eta "RFC 822 estandarrean" kode honek "Eskatutako posta-ekintza baztertu behar du: sarbidea ukatu". Beraz, Postfix-ekin, access map arauaren arau-hausteak ez du zehatz-mehatz adierazten RFC822 kodeari buruzko azalpena.

Nahasmena saihesteko, 570 honetara alda dezakezu (beste arrazoi bat da 554 kodea beste akats mota batzuetarako lehenetsitako balioa da), bestela, ez aldatu hau RFC 822-ren erabateko ulermena izan ezean.
