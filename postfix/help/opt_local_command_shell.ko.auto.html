외부 명령으로 전달하는 데 사용되는 쉘
local_command_shell
이 매개 변수는 외부 명령으로 전달하는 데 사용할 쉘을 제어합니다. 기본적으로 외부 명령은 직접 실행됩니다. 쉘 메타 문자 나 쉘 내장 명령이 포함 된 경우에만 /bin/sh 명령이 제공됩니다.

예를 들어 .forward 파일에서 어떤 프로그램을 실행할 수 있는지 더 세밀하게 제어해야 할 경우 다른 쉘을 지정하십시오.

"sendmail의 제한된 쉘" smrsh 는 대부분의 사람들이 사용할 것입니다 ( smrsh 는 최근 sendmail 배포의 일부입니다)

참고 : 쉘이 지정되면 명령에 쉘 내장 명령이나 메타 문자가 포함되지 않은 경우에도 호출됩니다.

예 : /some/where/smrsh -c
