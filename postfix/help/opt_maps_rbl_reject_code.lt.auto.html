SMTP serverio atsakymas dėl RBL domenų pažeidimo
maps_rbl_reject_code
Šis parametras nurodo SMTP serverio atsakymo kodą, kai laiškas pažeidžia maps_rbl_domains taisyklę:
Prieiga atmesta: SMTP klientas įtrauktas į RBL sąrašą

Atminkite, kad „ maps_rbl_reject_code naudojama „postfix“ numatytoji vertė yra 554 kodas, o šis kodas „standartiniame RFC 822“ turėtų reikšti „Prašomas pašto veiksmas atmestas: prieiga atmesta“. Taigi naudojant „Postfix“ maps_rbl_domains taisyklės pažeidimas tiksliai neatspindi pranešto kodo RFC822 paaiškinimo.

Norėdami išvengti painiavos, galite tai pakeisti, pavyzdžiui, į 571 (kita priežastis, kad 554 kodas taip pat yra numatytoji postfix reikšmė kitokio pobūdžio klaidoms), priešingu atveju nekeiskite to, nebent jūs visiškai suprantate RFC 822.
