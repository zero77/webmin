Disabilita il blocco dei file del kernel nelle cassette postali
sun_mailtool_compatibility
Questo parametro disabilita i blocchi dei file del kernel nelle cassette postali.

Questo è necessario sulle workstation SUN perché il programma mailtool mantiene un esclusivo blocco del kernel mentre la sua finestra è aperta. Il software SUN utilizza solo file user.lock . A meno che non si rimuova tutto il software di posta elettronica SUN, i blocchi del kernel danno solo un falso senso di sicurezza.
