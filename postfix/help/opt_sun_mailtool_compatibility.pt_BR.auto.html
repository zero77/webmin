Desativar bloqueio de arquivo do kernel em caixas de correio
sun_mailtool_compatibility
Este parâmetro desabilita os bloqueios de arquivo do kernel nas caixas de correio.

Isso é necessário nas estações de trabalho SUN, porque o programa mailtool mantém um bloqueio exclusivo do kernel enquanto a janela está aberta. O software SUN usa apenas arquivos user.lock . A menos que você remova todo o software de correio SUN, os bloqueios do kernel apenas fornecem uma falsa sensação de segurança.
