Alias à
Chaque alias peut rediriger le courrier vers plusieurs destinataires, et chaque destinataire peut être l'un des suivants:
Adresse électronique
Soit une adresse e-mail complète comme foo@bar.com , soit le nom d'un autre utilisateur de votre système comme jcameron .
Adresses dans le fichier
Pour ce type d'alias, vous devez spécifier le chemin d'accès complet à un fichier contenant une liste d'adresses e-mail, une par ligne. Chaque adresse peut être soit entièrement qualifiée, soit le nom d'un autre utilisateur de votre système.
Écrire dans un fichier
Dans ce cas, l'e-mail sera écrit dans le fichier spécifié. Pour que tous les e-mails envoyés à une adresse soient supprimés, donnez /dev/null comme nom de fichier.
Feed to Program
Chaque fois qu'un e-mail arrive pour cet alias, le programme spécifié sera exécuté et l'intégralité de l'e-mail (en-têtes et corps) lui sera transmise sur l'entrée standard.
