Percorso del file della cassetta postale dell'utente
home_mailbox
Questo parametro consente di specificare il percorso facoltativo di un file della cassetta postale relativo alla home directory di un utente. Il file della cassetta postale predefinito è /var/spool/mail/user o /var/mail/user .

Specificare "Maildir /" per la consegna in stile qmail (il / è obbligatorio).

Esempi:

  • Mailbox
  • Maildir/
