Einschränkungen für Client-Hostnamen / -Adressen
smtpd_client_restrictions
Dieser Parameter gibt optionale Einschränkungen für SMTP-Client-Hostnamen und -Adressen an.

Standardmäßig sind Verbindungen von jedem Host aus zulässig. Folgende Einschränkungen sind verfügbar: Einschränkungen werden in der angegebenen Reihenfolge angewendet. Die erste Einschränkung, die übereinstimmt, gewinnt.

Geben Sie eine Liste von Einschränkungen an, die durch Kommas und / oder Leerzeichen getrennt sind. Setzen Sie lange Zeilen fort, indem Sie die nächste Zeile mit Leerzeichen beginnen.

Sinnvolle Beispiele:

  • permit_mynetworks, reject_unknown_client
  • reject_maps_rbl, reject_unknown_client
