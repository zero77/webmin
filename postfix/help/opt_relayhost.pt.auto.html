Enviar email de saída pelo host
relayhost
Normalmente, você usa Entregar diretamente para que o Postfix encaminhe qualquer email de saída não destinado aos usuários do seu sistema para o host especificado.

Se você deseja encaminhar todo o correio para um gateway de correio da intranet (por exemplo, se a sua intranet possui um firewall), exceto o correio para a máquina local, você configurará o valor para o gateway de correio. Na maioria das vezes, a organização configurou registros MX internos; portanto, o uso direto do seu nome de domínio será vinculado à máquina do gateway de correio.
