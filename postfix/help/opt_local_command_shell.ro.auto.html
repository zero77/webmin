Shell de utilizat pentru livrare la comandă externă
local_command_shell
Acest parametru controlează ce coajă va fi utilizată pentru livrare la comanda externă. În mod implicit, comenzile externe sunt executate direct; comenzile sunt date la /bin/sh numai atunci când conțin caractere meta shell sau comenzi integrate shell.

Specificați un shell diferit atunci când aveți nevoie de mai mult control asupra programelor care pot fi rulate din ex. Fișiere .forward .

„shell-ul restricționat al sendmail” smrsh este ceea ce vor folosi majoritatea oamenilor ( smrsh face parte din distribuțiile recente de sendmail)

Notă: atunci când este specificată o coajă, aceasta este invocată chiar și atunci când comanda nu conține comenzi încorporate sau caractere meta.

Exemplu: /some/where/smrsh -c
