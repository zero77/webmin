SMTP serverio atsakymas dėl kliento atmetimo
reject_code
Šis parametras nurodo SMTP serverio atsakymo kodą, kai klientas pažeidžia smtpd_client_restrictions taisyklę, kuri yra:
Prieiga atmesta: SMTP kliento pagrindinis vardas atmestas

Atminkite, kad „postfix“ numatytoji reikšmė, naudojama reject_code yra kodas 554, o šis kodas „standartiniame RFC 822“ turėtų reikšti „Prašomas pašto veiksmas atmestas: prieiga atmesta“. Taigi naudojant „Postfix“ smtpd_client_restrictions taisyklės pažeidimas tiksliai neatspindi pranešto kodo RFC822 paaiškinimo.

Norėdami išvengti painiavos, galite tai pakeisti, pavyzdžiui, į 575 (kita priežastis, kad 554 kodas taip pat yra numatytoji „postfix“ vertė kitokio pobūdžio klaidoms), priešingu atveju nekeiskite to, nebent jūs visiškai suprantate RFC 822.
