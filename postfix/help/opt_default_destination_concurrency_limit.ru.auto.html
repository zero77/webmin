Максимальное количество параллельных доставок в один и тот же пункт назначения
default_destination_concurrency_limit
Параметр указывает ограничение по умолчанию для числа параллельных доставок в один и тот же пункт назначения. Это ограничение по умолчанию для доставки через SMTP, через локального агента доставки и через почтовую программу.
