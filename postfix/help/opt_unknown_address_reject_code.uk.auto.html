Відповідь SMTP-сервера на відхилення домену
unknown_address_reject_code
Цей параметр вказує код відповіді сервера SMTP, коли клієнт порушує правила обмеження reject_unknown_sender_domain або reject_unknown_recipient_domain які є:
Заборонено доступ: невідомий домен для електронної адреси відправника або одержувача (без запису DNS A або MX)

Зауважте, що значення за замовчуванням постфіксу, яке використовується для unknown_address_reject_code є кодом 450, який також використовується для інших помилок. Цей код у "стандартному RFC 822" означає "Запрошена пошта не здійснена: поштова скринька зайнята або доступ заборонено", і попросіть відправника спробувати пізніше (сподіваючись, що DNS відповість правильно пізніше).

Не змінюйте це, якщо ви повністю не розумієте RFC 822.
