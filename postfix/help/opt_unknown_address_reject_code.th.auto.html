การตอบสนองของเซิร์ฟเวอร์ SMTP เกี่ยวกับการปฏิเสธโดเมน
unknown_address_reject_code
พารามิเตอร์นี้ระบุรหัสการตอบสนองเซิร์ฟเวอร์ SMTP เมื่อลูกค้าละเมิด reject_unknown_sender_domain หรือ reject_unknown_recipient_domain กฎข้อ จำกัด ที่มี:
ปฏิเสธการเข้าถึง: โดเมนที่ไม่รู้จักสำหรับที่อยู่อีเมลของผู้ส่งหรือผู้รับ (ไม่มีระเบียน DNS A หรือ MX)

โปรดทราบว่าค่าเริ่มต้น postfix ที่ใช้สำหรับ unknown_address_reject_code คือรหัส 450 ที่ใช้สำหรับข้อผิดพลาดอื่น ๆ รหัสนี้ใน 'มาตรฐาน RFC 822' หมายถึง 'ไม่ได้ทำการร้องขอจดหมาย: กล่องจดหมายไม่ว่างหรือปฏิเสธการเข้าถึง' และขอให้ผู้ส่งลองอีกครั้งในภายหลัง (หวังว่า DNS จะตอบได้อย่างถูกต้องในภายหลัง)

อย่าเปลี่ยนแปลงสิ่งนี้หากคุณไม่เข้าใจ RFC 822 โดยสมบูรณ์
