Perintah luaran untuk digunakan dan bukannya penghantaran peti mel
mailbox_command
Parameter ini menentukan perintah luaran pilihan untuk digunakan dan bukannya penghantaran peti mel. Perintah dijalankan sebagai penerima dengan tetapan persekitaran HOME , SHELL dan LOGNAME . Pengecualian: penghantaran untuk root dilakukan sebagai $default_user .

Pemboleh ubah persekitaran lain yang menarik: USER (nama pengguna penerima), EXTENSION (peluasan alamat), DOMAIN (bahagian domain alamat), dan LOCAL (bahagian tempatan alamat).

Tidak seperti parameter konfigurasi Postfix yang lain, parameter ini tidak dikenakan penggantian parameter. Ini untuk mempermudah menentukan sintaks shell (lihat contoh di bawah).

Elakkan watak shell meta kerana mereka akan memaksa Postfix menjalankan proses shell yang mahal. Procmail sahaja cukup mahal.

JIKA ANDA MENGGUNAKAN INI UNTUK MENGHANTAR SISTEM MAIL SELURUH, ANDA HARUS MENGHASILKAN ALIAS YANG MENGHADAPI MAJLIS UNTUK ROOT KEPADA PENGGUNA YANG SEBENAR.

Contoh:

  • /some/where/procmail
  • /some/where/procmail -a "$EXTENSION"
