ย้ายการแมป
ไฟล์ที่ถูกย้ายที่เลือกกำหนดจัดเตรียมข้อมูลที่ใช้ในข้อความตีกลับ "user ย้ายไปยัง new_location"

การค้นหาตารางไม่คำนึงถึงขนาดตัวพิมพ์

ฟิลด์ "แผนที่ไปยัง ... " ระบุข้อมูลติดต่อเช่นที่อยู่อีเมลหรืออาจเป็นที่อยู่หรือหมายเลขโทรศัพท์ เป็นหนึ่งในสิ่งต่อไปนี้:

  • user@domain : จับคู่ user@domain แบบฟอร์มนี้มีความสำคัญมากกว่ารูปแบบอื่น ๆ ทั้งหมด
  • user : จับคู่ user@site เมื่อไซต์คือ $myorigin เมื่อไซต์อยู่ในรายการ $mydestination หรือเมื่อไซต์นั้นปรากฏใน $inet_interfaces
  • @domain : จับคู่ทุกที่อยู่ในโดเมน แบบฟอร์มนี้มีลำดับความสำคัญต่ำสุด
ส่วนขยายที่อยู่: เมื่อการค้นหาล้มเหลวและ address localpart มีตัวคั่นผู้รับที่เป็นตัวเลือก (เช่น user+foo@domain ) การค้นหาจะทำซ้ำสำหรับที่อยู่ที่ไม่ได้ user@domain (เช่น user@domain )
