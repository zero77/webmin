Ответ SMTP-сервера на отклонение домена
unknown_address_reject_code
Этот параметр указывает код ответа сервера SMTP, когда клиент нарушает правила ограничения reject_unknown_sender_domain или reject_unknown_recipient_domain , которые:
Доступ запрещен: неизвестный домен для адреса электронной почты отправителя или получателя (без записи DNS A или MX)

Обратите внимание, что значением по умолчанию постфикса, используемым для unknown_address_reject_code является код 450, также используемый для других ошибок. Этот код в «стандартном RFC 822» означает «Запрошенное почтовое действие не выполнено: почтовый ящик занят или доступ запрещен» и попросите отправителя повторить попытку позже (надеясь, что DNS ответит правильно позже).

Не изменяйте это, если у вас нет полного понимания RFC 822.
