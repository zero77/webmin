Оболочка, используемая для доставки на внешнюю команду
local_command_shell
Этот параметр определяет, какая оболочка будет использоваться для доставки внешней команде. По умолчанию внешние команды выполняются напрямую; Команды передаются /bin/sh только если они содержат метасимволы оболочки или встроенные команды оболочки.

Укажите другую оболочку, когда вам нужно больше контроля над тем, какие программы можно запускать, например, из файлов .forward .

smrsh - это то, что будет использовать большинство людей (« smrsh является частью последних дистрибутивов sendmail).

Примечание: когда указана оболочка, она вызывается, даже если команда не содержит встроенных в оболочку команд или метасимволов.

Пример: /some/where/smrsh -c
