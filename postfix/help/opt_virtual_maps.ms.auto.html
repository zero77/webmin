Jadual carian pemetaan domain
virtual_maps
Parameter ini menentukan jadual carian pilihan untuk mengalihkan alamat tertentu atau bahkan melengkapkan domain ke alamat lain. Ini biasanya digunakan untuk melaksanakan sokongan domain maya.

Secara lalai, pengalihan alamat tidak dilakukan.

Contoh:

  • dbm:/etc/postfix/virtual
  • hash:/etc/postfix/virtual
  • hash:/etc/postfix/virtual, nis:virtual
  • hash:/etc/postfix/virtual, netinfo:/virtual
