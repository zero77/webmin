Alias To
Jeder Alias kann E-Mails an mehrere Empfänger umleiten, und jeder Empfänger kann einer der folgenden sein:
E-Mail-Addresse
Entweder eine vollständige E-Mail-Adresse wie foo@bar.com oder der Name eines anderen Benutzers auf Ihrem System wie jcameron .
Adressen in Datei
Für diese Art von Alias müssen Sie den vollständigen Pfad zu einer Datei angeben, die eine Liste von E-Mail-Adressen enthält, eine pro Zeile. Jede Adresse kann entweder vollständig qualifiziert sein oder der Name eines anderen Benutzers auf Ihrem System.
In Datei schreiben
In diesem Fall wird eine E-Mail in die angegebene Datei geschrieben. Geben Sie als Dateinamen /dev/null an, damit alle E-Mails an eine Adresse verworfen werden.
Feed zum Programm
Jedes Mal, wenn eine E-Mail-Nachricht für diesen Alias eingeht, wird das angegebene Programm ausgeführt und die gesamte E-Mail-Nachricht (Header und Text) an die Standardeingabe übergeben.
