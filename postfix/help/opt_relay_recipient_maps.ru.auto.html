Карта разрешенных адресов для ретрансляции
Если для этой опции задана карта, такая как hash:/etc/postfix/relay_recipients , только адреса с ключом в этой карте будут приниматься, когда электронная почта пересылается через эту систему. Это может быть полезно, чтобы спаммеры не отправляли электронную почту с поддельными адресами, которые будут возвращены при отправке электронной почты на целевой сервер.
