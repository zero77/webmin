Bancos de dados alternativos usados pelo agente de entrega local
alias_maps
Este parâmetro especifica a lista de bancos de dados de alias usados pelo agente de entrega local. A lista padrão depende do sistema. Em sistemas com NIS, o padrão é procurar o banco de dados de alias local e, em seguida, o banco de dados de alias do NIS. Veja aliases para detalhes de sintaxe.

As configurações típicas são:

  • dbm:/etc/aliases
  • hash:/etc/postfix/aliases
  • hash:/etc/aliases, nis:mail.aliases
