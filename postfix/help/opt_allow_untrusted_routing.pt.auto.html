Permitir roteamento não confiável
allow_untrusted_routing
Este parâmetro controla se o Postfix encaminhará as mensagens com o roteamento especificado pelo remetente (usuário [@%!] Site remoto [@%!]) De clientes não confiáveis para destinos abençoados pelo parâmetro relay_domains .

Por padrão, clientes não confiáveis não têm permissão para especificar roteamento. Isso fecha uma brecha desagradável na retransmissão aberta, na qual um host MX de backup pode ser enganado no encaminhamento de lixo eletrônico para um host MX principal, que depois o envia para o mundo.
