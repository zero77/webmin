Réseaux locaux
mynetworks
Ce paramètre spécifie la liste des réseaux locaux sur cette machine. La liste est utilisée par le logiciel anti-UCE pour distinguer les clients locaux des étrangers. Voir permit_mynetworks et smtpd_recipient_restrictions dans la configuration SMTP.

La valeur par défaut est une liste de tous les réseaux connectés à la machine: un réseau complet de classe A (X.0.0.0 / 8), un réseau complet de classe B (XX0.0 / 16), etc. Si vous souhaitez un contrôle plus strict, spécifiez une liste de modèles de réseau / masque, où le masque spécifie le nombre de bits dans la partie réseau d'une adresse d'hôte. Vous pouvez également spécifier le chemin d'accès absolu d'un fichier de signatures au lieu de répertorier les signatures ici.
