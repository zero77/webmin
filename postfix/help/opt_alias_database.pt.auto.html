Bancos de dados de alias criados pelo Postfix
alias_database
Este parâmetro especifica os bancos de dados de alias criados com "newaliases" ou "sendmail -bi". Este é um parâmetro de configuração separado, porque alias_maps pode especificar tabelas que não são necessariamente todas sob controle do Postfix.

As configurações típicas são:

  • dbm:/etc/aliases
  • dbm:/etc/mail/aliases
  • hash:/etc/aliases
  • hash:/etc/aliases, hash:/opt/majordomo/aliases
