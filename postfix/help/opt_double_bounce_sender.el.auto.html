Διεύθυνση αποστολέα για αναπήδηση αλληλογραφίας
double_bounce_sender
Αυτή η παράμετρος καθορίζει τη διεύθυνση αποστολέα για αλληλογραφία που πρέπει να απορριφθεί όταν δεν μπορεί να παραδοθεί. Αυτό πρέπει να είναι ένα μοναδικό όνομα. Όλα τα μηνύματα σε αυτό το όνομα απορρίπτονται σιωπηλά, προκειμένου να τερματιστούν οι βρόχοι αναπήδησης αλληλογραφίας.
