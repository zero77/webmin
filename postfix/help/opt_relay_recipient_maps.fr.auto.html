Carte des adresses autorisées pour le relais
Lorsque cette option est définie sur une carte comme hash:/etc/postfix/relay_recipients , seules les adresses avec une clé dans cette carte seront acceptées lorsque le courrier électronique est relayé via ce système. Cela peut être utile pour empêcher les spammeurs d'envoyer des e-mails avec des adresses falsifiées qui seront renvoyées lorsque les e-mails seront relayés vers le serveur de destination.
