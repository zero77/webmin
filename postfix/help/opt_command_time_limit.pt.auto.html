Tempo máximo para entrega a comandos externos
command_time_limit
Este parâmetro limita a quantidade de tempo para entrega a comandos externos. Esse limite é usado pelo agente de entrega local e é o prazo padrão para entrega pela mala direta.

Nota: se você definir esse limite de tempo para um valor grande, também deverá atualizar o parâmetro global ipc_timeout . Consulte Opções diversas do Postfix para obter detalhes.
