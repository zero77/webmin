Таблицы "Пользователь переехал на новое место"
relocated_maps
Этот параметр указывает необязательные таблицы с контактной информацией для пользователей, хостов или доменов, которые больше не существуют. См. Перемещенное сопоставление для деталей.

По умолчанию эта функция отключена.

Разумное возможное значение: hash:/etc/postfix/relocated
