Maximale Anzahl von Empfängern pro lokaler Nachrichtenübermittlung
local_destination_recipient_limit
Dieser Parameter begrenzt die Anzahl der Empfänger pro lokaler Nachrichtenübermittlung. Das Standardlimit wird aus dem Parameter default_destination_recipient_limit .

Der Warteschlangenmanager begrenzt jedoch die Anzahl der Empfänger pro lokaler Übermittlungsanforderung auf genau 1, sodass dieser Parameter keine Auswirkung hat.
