Action lorsque répertorié comme meilleur hôte MX
best_mx_transport
Ce paramètre contrôle ce qui se passe lorsque le système local est répertorié comme le meilleur hôte MX pour une destination. Par défaut, Postfix signale une erreur de "boucle de retour vers moi" et renvoie le message.

Spécifiez local pour transmettre le courrier à l'agent de livraison local. Vous pouvez spécifier n'importe quel transport défini dans le fichier master.cf Postfix.
