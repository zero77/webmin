Para que domínio receber emails
mydestination
Você especifica quais domínios essa máquina fornecerá localmente, em vez de encaminhar para outra máquina. O padrão é receber e-mails para a própria máquina.

Máquina local : $myhostname, localhost.$mydomain
Domínio inteiro : $myhostname, localhost.$mydomain, $mydomain

Você também pode especificar o caminho completo de um arquivo, que conterá os nomes de domínios para os quais esta máquina entregará correio localmente.

Exemplo: /etc/postfix/mydestination


Cuidado: para evitar loops de entrega de correio, você deve listar todos os nomes de host da máquina, incluindo $myhostname e localhost.$mydomain . NÃO LISTAR DOMÍNIOS VIRTUAIS AQUI. LISTA-OS NO ARQUIVO VIRTUAL DENTRO.
