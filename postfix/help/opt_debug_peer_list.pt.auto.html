Lista de padrões de domínio / rede para os quais o log detalhado está ativado
debug_peer_list
Este parâmetro especifica uma lista opcional de padrões de domínio ou rede, /file/name padrões de /file/name ou tabelas type:name . Quando um nome ou endereço de host do cliente ou servidor SMTP corresponder a um padrão, aumente o nível de log detalhado pela quantidade especificada no parâmetro debug_peer_level .

Valores possíveis sensíveis:

  • 127.0.0.1
  • some.domain
