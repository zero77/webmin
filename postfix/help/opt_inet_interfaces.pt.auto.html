Interfaces de rede para recebimento de correio
inet_interfaces
Este parâmetro especifica os endereços da interface de rede em que este sistema de email recebe emails.

Por padrão, o software reivindica todas as interfaces ativas na máquina. O parâmetro também controla a entrega de email para o user@[ip.address] .

As configurações típicas são:

  • all
  • $myhostname
  • $myhostname, localhost
