不明なクライアントのSMTPサーバーの応答が拒否されました
unknown_client_reject_code
このパラメーターは、アドレスから名前へのマッピングを持たないクライアントが次のreject_unknown_clients制限ルールに違反する場合のSMTPサーバー応答コードを指定します。
アクセス拒否：不明なSMTPクライアントのホスト名（DNS AまたはMXレコードなし）

unknown_client_reject_code使用されるpostfixのデフォルト値は、他のエラーにも使用されるコード450であることに注意してください。 「標準RFC 822」のこのコードは、「要求されたメールアクションは実行されません：メールボックスがビジーまたはアクセスが拒否されました」を意味し、送信者に後で再試行するよう依頼します（DNSが後で正しく応答することを期待しています）。

RFC 822を完全に理解していない限り、これを変更しないでください。
