Εξωτερική εντολή για χρήση αντί παράδοσης γραμματοκιβωτίου
mailbox_command
Αυτή η παράμετρος καθορίζει την προαιρετική εξωτερική εντολή για χρήση αντί για παράδοση γραμματοκιβωτίου. Η εντολή εκτελείται ως παραλήπτης με τις κατάλληλες ρυθμίσεις περιβάλλοντος HOME , SHELL και LOGNAME . Εξαίρεση: η παράδοση για root γίνεται ως $default_user .

Άλλες περιβαλλοντικές μεταβλητές ενδιαφέροντος: USER (όνομα χρήστη παραλήπτη), EXTENSION (επέκταση διεύθυνσης), DOMAIN (τμήμα τομέα διεύθυνσης) και LOCAL (το LOCAL τμήμα διεύθυνσης).

Σε αντίθεση με άλλες παραμέτρους διαμόρφωσης Postfix, αυτή η παράμετρος δεν υπόκειται σε αντικαταστάσεις παραμέτρων. Αυτό γίνεται για να διευκολύνεται ο καθορισμός σύνταξης κελύφους (δείτε το παρακάτω παράδειγμα).

Αποφύγετε τους χαρακτήρες μετα-φλοιού γιατί θα αναγκάσουν το Postfix να εκτελέσει μια ακριβή διαδικασία κελύφους. Procmail μόνο είναι αρκετά ακριβό.

ΕΑΝ ΧΡΗΣΙΜΟΠΟΙΕΙΤΕ ΑΥΤΟ ΓΙΑ ΤΗΝ ΠΑΡΑΔΟΣΗ ΤΟΥ MAIL ΣΥΣΤΗΜΑΤΟΣ, ΠΡΕΠΕΙ ΝΑ ΕΓΚΑΤΑΣΤΑΣΕΤΕ ΜΙΑ ΑΛΛΙΑ ΠΟΥ ΠΡΟΒΛΕΠΕΙ ΤΗ ΜΑΛΛΙΑ ΓΙΑ ΡΟΖ ΣΕ ΠΡΑΓΜΑΤΙΚΟ ΧΡΗΣΤΗ.

Παραδείγματα:

  • /some/where/procmail
  • /some/where/procmail -a "$EXTENSION"
