Procesos del servidor
Esta página se puede usar para administrar los diversos procesos del servidor utilizados por Postfix para recibir y procesar correos electrónicos. A menos que sea un experto en la configuración de Postfix, generalmente no se deben editar, sin embargo, desea habilitar o deshabilitar ciertos procesos del servidor para activar y desactivar las funciones de Postfix como SMTP sobre SSL.
