Стварыць файл часопіса
Гэтая старонка выкарыстоўваецца для ўстаноўкі кручэння для новага файла часопіса. Найважнейшае поле - гэта шлях да файла для часопіса , які дазваляе ўводзіць месцазнаходжанне часопіса, які трэба паварочваць, альбо падпісны знак абалонкі, які адпавядае некалькіх часопісах.
