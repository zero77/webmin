Pacchetti software cluster
Questo modulo Webmin consente di gestire l'installazione di utenti e gruppi su più server in un cluster da un'unica interfaccia. È simile al modulo Utenti e gruppi, ma con la possibilità di eseguire la maggior parte delle funzioni (come la creazione, l'aggiornamento e l'eliminazione) degli utenti su molti server Webmin in un'unica operazione.

La parte superiore della pagina principale sotto l'intestazione Server gestiti elenca altri server Webmin i cui utenti e gruppi sono gestiti da questo modulo. Per aggiungere un server a questo elenco, è necessario innanzitutto aggiungerlo al modulo Server Webmin, con un nome utente e una password specificati per accedere a Webmin su quel server. È quindi possibile selezionare il server dall'elenco accanto al pulsante Aggiungi server.

Quando viene aggiunto un server, il modulo verificherà che stia utilizzando lo stesso formato di file utente di questo server, quindi scarica un elenco di tutti gli utenti e gruppi per la memorizzazione nella cache locale. Un server che verrà sempre visualizzato nell'elenco di quelli disponibili per l'aggiunta è "questo server", in modo da poter gestire gli utenti localmente e su più server remoti da un'unica interfaccia.

Quando è presente almeno un'icona in Server gestiti, è possibile utilizzare i moduli Trova utenti e Trova gruppi per cercare utenti e gruppi su tutti i server e i pulsanti Aggiungi utente e Aggiungi gruppo per aggiungere a tutti i server. Il pulsante Sincronizza può essere utilizzato per creare automaticamente tutti gli utenti esistenti su alcuni server nel cluster ma non su altri.
