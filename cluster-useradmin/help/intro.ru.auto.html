Пакеты программ кластера
Этот модуль Webmin позволяет управлять настройками пользователей и групп на нескольких серверах в кластере из одного интерфейса. Он похож на модуль «Пользователи и группы», но с возможностью выполнять большинство функций (таких как создание, обновление и удаление) пользователей на многих серверах Webmin за одну операцию.

В верхней части главной страницы под заголовком «Управляемые серверы» перечислены другие серверы Webmin, пользователи и группы которых управляются этим модулем. Чтобы добавить сервер в этот список, вы должны сначала добавить его в модуль Webmin Servers, указав имя пользователя и пароль для входа в Webmin на этом сервере. Затем вы можете выбрать сервер из списка рядом с кнопкой «Добавить сервер».

Когда сервер добавлен, модуль проверяет, использует ли он тот же формат файла пользователя, что и этот сервер, а затем загружает список всех пользователей и групп для локального кэширования. Один сервер, который всегда будет отображаться в списке доступных для добавления, - это «этот сервер», так что вы можете управлять пользователями локально и на нескольких удаленных серверах из одного интерфейса.

Если в разделе «Управляемые серверы» есть хотя бы один значок, вы можете использовать формы «Найти пользователей» и «Найти группы» для поиска пользователей и групп на всех серверах, а также кнопки «Добавить пользователя» и «Добавить группу» для добавления на все серверы. Кнопка Синхронизировать может использоваться для автоматического создания любых пользователей, которые существуют на некоторых серверах в кластере, но не на других.
