Packages logiciels de cluster
Ce module Webmin vous permet de gérer la configuration des utilisateurs et des groupes sur plusieurs serveurs dans un cluster à partir d'une seule interface. Il est similaire au module Utilisateurs et groupes, mais avec la possibilité d'effectuer la plupart des fonctions (comme la création, la mise à jour et la suppression) d'utilisateurs sur de nombreux serveurs Webmin en une seule opération.

La partie supérieure de la page principale sous l'en-tête Serveurs gérés répertorie les autres serveurs Webmin dont les utilisateurs et les groupes sont gérés par ce module. Pour ajouter un serveur à cette liste, vous devez d'abord l'ajouter au module Serveurs Webmin, avec un nom d'utilisateur et un mot de passe spécifiés pour vous connecter à Webmin sur ce serveur. Vous pouvez ensuite sélectionner le serveur dans la liste à côté du bouton Ajouter un serveur.

Lorsqu'un serveur est ajouté, le module vérifie qu'il utilise le même format de fichier utilisateur que ce serveur, puis télécharge une liste de tous les utilisateurs et groupes pour la mise en cache localement. Un serveur qui apparaîtra toujours dans la liste de ceux disponibles pour l'ajout est «ce serveur», afin que vous puissiez gérer les utilisateurs localement et sur plusieurs serveurs distants à partir de la même interface.

Une fois qu'il y a au moins une icône sous Serveurs gérés, vous pouvez utiliser les formulaires Rechercher des utilisateurs et Rechercher des groupes pour rechercher des utilisateurs et des groupes sur tous les serveurs, et les boutons Ajouter un utilisateur et Ajouter un groupe pour ajouter à tous les serveurs. Le bouton Synchroniser peut être utilisé pour créer automatiquement tous les utilisateurs qui existent sur certains serveurs du cluster mais pas sur d'autres.
