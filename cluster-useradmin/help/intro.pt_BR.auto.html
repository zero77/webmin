Pacotes de software de cluster
Este módulo Webmin permite gerenciar a configuração de usuários e grupos em vários servidores em um cluster a partir de uma interface. É semelhante ao módulo Usuários e Grupos, mas com a capacidade de executar a maioria das funções (como criação, atualização e exclusão) de usuários em muitos servidores Webmin em uma operação.

A parte superior da página principal, sob o título Servidores Gerenciados, lista outros servidores Webmin cujos usuários e grupos estão sendo gerenciados por este módulo. Para adicionar um servidor a esta lista, você deve primeiro adicioná-lo ao módulo Servidores Webmin, com um nome de usuário e senha especificados para efetuar login no Webmin nesse servidor. Em seguida, você pode selecionar o servidor na lista ao lado do botão Adicionar servidor.

Quando um servidor é adicionado, o módulo verifica se está usando o mesmo formato de arquivo do usuário que este servidor e baixa uma lista de todos os usuários e grupos para armazenamento em cache localmente. Um servidor que sempre aparecerá na lista daqueles disponíveis para adição é 'this server', para que você possa gerenciar usuários localmente e em vários servidores remotos a partir de uma única interface.

Quando houver pelo menos um ícone em Servidores gerenciados, você poderá usar os formulários Localizar usuários e Localizar grupos para procurar usuários e grupos em todos os servidores e os botões Adicionar usuário e Adicionar grupo para adicionar a todos os servidores. O botão Sincronizar pode ser usado para criar automaticamente qualquer usuário que exista em alguns servidores no cluster, mas não em outros.
