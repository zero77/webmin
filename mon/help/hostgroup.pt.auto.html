Ajuda do MON para grupos de hosts e seleção de serviços

Se você já possui o arquivo "mon.cf", esta página carrega os valores padrão de Grupos de Host, Membros e Serviços da configuração existente.

Cada grupo de host inserido é definido como Padrão como Observar. Pode ser alterado para NÃO se você não deseja assistir ao grupo de host.

A menos que os serviços para grupos de hosts (para assistir) sejam selecionados, não é possível prosseguir para o próximo estágio de configuração.
