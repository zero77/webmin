Справка MON для выбора групп и услуг

Если у вас уже есть файл «mon.cf», эта страница загружает значения по умолчанию для групп узлов, участников и служб из существующей конфигурации.

Каждой введенной группе хостов присваивается значение по умолчанию для просмотра. Его можно изменить на НЕТ, если вы не хотите смотреть группу хостов.

Если не выбраны сервисы для групп хостов (для просмотра), переход к следующему этапу настройки невозможен.
