Fichiers de copie de cluster
Ce module vous permet de configurer des transferts planifiés de fichiers d'un serveur maître vers d'autres serveurs d'un cluster Webmin. Cela peut être utile pour distribuer des fichiers comme /etc/hosts , httpd.conf et d'autres pour lesquels aucun protocole de mise en réseau comme NIS ou LDAP n'est disponible.

La page principale du module répertorie toutes les copies planifiées définies et comporte un lien pour en créer une nouvelle. Pour chaque copie, vous pouvez définir les fichiers source, le répertoire de destination, les serveurs cibles et les heures d'exécution. Les cibles doivent d'abord avoir été créées dans le module Webmin Servers Index avec un identifiant et un mot de passe.
