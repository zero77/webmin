Cluster Copiați fișiere
Acest modul vă permite să configurați transferuri programate de fișiere de la un server master la alte servere dintr-un cluster Webmin. Acest lucru poate fi util pentru distribuirea fișierelor precum /etc/hosts , httpd.conf și altele pentru care nu este disponibil niciun protocol de rețea precum NIS sau LDAP.

Pagina principală a modulului listează toate copiile programate definite și are un link pentru crearea unei noi. Pentru fiecare copie puteți defini fișierele sursă, directorul destinație, serverele țintă și orele la care puteți rula. Țintele trebuie să fi fost create mai întâi în modulul Index Web Servers cu o autentificare și o parolă.
