Cluster-Kopierdateien
Mit diesem Modul können Sie geplante Übertragungen von Dateien von einem Master-Server auf andere Server in einem Webmin-Cluster einrichten. Dies kann nützlich sein, um Dateien wie /etc/hosts , httpd.conf und andere zu verteilen, für die kein Netzwerkprotokoll wie NIS oder LDAP verfügbar ist.

Die Hauptseite des Moduls listet alle definierten geplanten Kopien auf und enthält einen Link zum Erstellen einer neuen. Für jede Kopie können Sie die Quelldateien, das Zielverzeichnis, die Zielserver und die Ausführungszeiten definieren. Die Ziele müssen zuerst im Webmin Servers Index- Modul mit einem Login und einem Passwort erstellt worden sein.
