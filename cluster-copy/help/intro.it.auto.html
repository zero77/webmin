Cluster Copia file
Questo modulo consente di impostare trasferimenti pianificati di file da un server master ad altri server in un cluster Webmin. Questo può essere utile per distribuire file come /etc/hosts , httpd.conf e altri per i quali non è disponibile alcun protocollo di rete come NIS o LDAP.

La pagina principale del modulo elenca tutte le copie pianificate definite e ha un collegamento per crearne una nuova. Per ogni copia è possibile definire i file di origine, la directory di destinazione, i server di destinazione e i tempi in cui eseguire. Le destinazioni devono essere state create per la prima volta nel modulo Indice server Webmin con un login e una password.
