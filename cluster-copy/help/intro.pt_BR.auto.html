Arquivos de cópia de cluster
Este módulo permite configurar transferências agendadas de arquivos de um servidor mestre para outros servidores em um cluster Webmin. Isso pode ser útil para distribuir arquivos como /etc/hosts , httpd.conf e outros para os quais nenhum protocolo de rede como NIS ou LDAP está disponível.

A página principal do módulo lista todas as cópias agendadas definidas e possui um link para criar uma nova. Para cada cópia, você pode definir os arquivos de origem, diretório de destino, servidores de destino e horários para execução. Os destinos devem ter sido criados primeiro no módulo Índice de servidores Webmin com um login e senha.
