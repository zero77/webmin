Cluster Copy Files
Este módulo le permite configurar transferencias programadas de archivos desde un servidor maestro a otros servidores en un clúster de Webmin. Esto puede ser útil para distribuir archivos como /etc/hosts , httpd.conf y otros para los que no hay ningún protocolo de red como NIS o LDAP disponible.

La página principal del módulo enumera todas las copias programadas definidas y tiene un enlace para crear una nueva. Para cada copia, puede definir los archivos de origen, el directorio de destino, los servidores de destino y las horas en las que se ejecutará. Los objetivos deben haberse creado primero en el módulo Índice de servidores Webmin con un nombre de usuario y contraseña.
