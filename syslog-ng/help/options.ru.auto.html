Параметры ведения журнала
Эта страница позволяет редактировать параметры, которые применяются ко всему серверу Syslog-NG, такие как обработка имен хостов и поведение поиска DNS. Его также можно использовать для изменения значений по умолчанию, которые применяются ко всем адресатам журналов, таким как владельцы и разрешения для файлов журналов. Однако они могут быть переопределены для каждого пункта назначения.
