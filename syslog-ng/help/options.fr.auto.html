Options de journalisation
Cette page vous permet de modifier les options qui s'appliquent à l'ensemble du serveur Syslog-NG, telles que la remise des noms d'hôte et le comportement des recherches DNS. Il peut également être utilisé pour modifier les valeurs par défaut qui s'appliquent à toutes les destinations de journal, telles que les propriétaires et les autorisations sur les fichiers journaux. Cependant, ceux-ci peuvent être remplacés par destination.
