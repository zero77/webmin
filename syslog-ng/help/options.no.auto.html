Alternativer for logging
Denne siden lar deg redigere alternativer som gjelder for hele Syslog-NG-serveren, for eksempel overlevering av vertsnavn og oppførsel til DNS-oppslag. Det kan også brukes til å endre standardverdier som gjelder alle loggdestinasjoner, for eksempel eiere og tillatelser til loggfiler. Disse kan imidlertid overstyres per bestemmelsessted.
