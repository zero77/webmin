Στόχοι καταγραφής
Οι στόχοι καταγραφής είναι ο πιο σημαντικός τύπος αντικειμένου στη διαμόρφωση Syslog-NG, καθώς συγκεντρώνουν πηγές, φίλτρα και προορισμούς για να προσδιορίσουν ακριβώς τι καταγράφεται πού. Κάθε στόχος έχει μία ή περισσότερες πηγές, μηδέν ή περισσότερα φίλτρα (για να επιλέξετε ποια μηνύματα θα εξάγονται) και έναν ή περισσότερους προορισμούς (για να ελέγξετε πού γράφονται τα μηνύματα). Όλα αυτά μπορούν να επιλεγούν εύκολα όταν δημιουργείτε ή επεξεργάζεστε έναν στόχο καταγραφής.

Αυτή η σελίδα παραθέτει όλους τους γνωστούς στόχους και σας επιτρέπει να προσθέσετε νέους, να επεξεργαστείτε υπάρχοντες ή να διαγράψετε πολλούς ταυτόχρονα. Προσέξτε προσεκτικά τους στόχους που είχαν ρυθμιστεί από προεπιλογή, καθώς αυτό μπορεί να αλλάξει το περιεχόμενο και τα ονόματα των κρίσιμων αρχείων καταγραφής συστήματος. Γενικά, είναι ασφαλές να προσθέσετε τους δικούς σας στόχους.
