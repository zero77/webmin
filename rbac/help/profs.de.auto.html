Profile
Ein Ausführungsprofil ist ein Mechanismus, mit dem die Befehle und Berechtigungen gebündelt werden, die zum Ausführen einer bestimmten Funktion erforderlich sind. Jedes der auf dieser Seite aufgelisteten Profile verfügt über eine Reihe von Berechtigungen und eine Liste von Unterprofilen, deren Berechtigungen es ebenfalls erbt.

Möglicherweise möchten Sie Ihrem System zusätzliche Berechtigungen hinzufügen, wenn keine der hier aufgeführten Berechtigungen die genaue Zugriffskontrolle bietet, die Sie benötigen. Bestehende Standardberechtigungen sollten jedoch sorgfältig geändert werden, da sich eine Änderung auf die Berechtigungen von Systembenutzern wie root und lp auswirken kann.
