Autorisations
Une autorisation est un droit attribué aux utilisateurs et aux profils qui est vérifié par certains programmes privilégiés pour déterminer si les utilisateurs peuvent exécuter des fonctionnalités restreintes. Chacun a un nom de code court et une description lisible par l'homme.

En règle générale, vous n'aurez pas besoin d'ajouter ou de modifier les autorisations de cette page, sauf si vous développez votre propre logiciel qui utilise RBAC pour déterminer ce que les utilisateurs sont autorisés à faire.
