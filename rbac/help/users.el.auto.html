Χαρακτηριστικά χρήστη
Αυτή η σελίδα μπορεί να χρησιμοποιηθεί για την εκχώρηση προφίλ και εξουσιοδοτήσεων σε χρήστες του Unix, προκειμένου να τους παραχωρηθούν πρόσθετα δικαιώματα. Ένα προφίλ καθορίζει ένα σύνολο εξουσιοδοτήσεων που εκχωρούνται στον χρήστη και δίνει εντολές που ο χρήστης μπορεί να εκτελέσει με υψηλότερα δικαιώματα (ονομάζεται προφίλ εκτέλεσης).

Επιπλέον, αυτή η ενότητα μπορεί να χρησιμοποιηθεί για τη δημιουργία ρόλων, οι οποίοι μπορούν επίσης να έχουν προφίλ και εξουσιοδοτήσεις. Μόλις δημιουργηθεί ένας ρόλος, ένας χρήστης μπορεί να κληρονομήσει όλα τα δικαιώματα του ρόλου, γεγονός που καθιστά εύκολη τη χορήγηση των ίδιων δικαιωμάτων πρόσβασης σε πολλούς χρήστες.
