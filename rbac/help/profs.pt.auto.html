Perfis
Um perfil de execução é um mecanismo usado para agrupar os comandos e autorizações necessários para executar uma função específica. Cada um dos perfis listados nesta página possui um conjunto de autorizações e uma lista de sub-perfis cujas autorizações ele também herda.

Você pode adicionar autorizações adicionais ao seu sistema se nenhuma das listadas aqui fornecer o nível exato de controle de acesso necessário. No entanto, as autorizações padrão existentes devem ser modificadas com cuidado, pois alterá-las pode afetar os privilégios dos usuários dos sistemas, como root e lp .
