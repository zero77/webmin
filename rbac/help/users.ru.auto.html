Атрибуты пользователя
Эта страница может использоваться для назначения профилей и авторизаций пользователям Unix, чтобы предоставить им дополнительные привилегии. Профиль определяет набор полномочий, предоставленных пользователю, и команды, которые пользователь может выполнять с более высокими привилегиями (так называемые профили выполнения).

Кроме того, этот раздел можно использовать для создания ролей, которые также могут иметь профили и полномочия. После создания роли пользователь может унаследовать все привилегии роли, что упрощает предоставление одинаковых прав доступа нескольким пользователям.
