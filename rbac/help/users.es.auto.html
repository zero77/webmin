Atributos del usuario
Esta página se puede utilizar para asignar perfiles y autorizaciones a usuarios de Unix, a fin de otorgarles privilegios adicionales. Un perfil define un conjunto de autorizaciones otorgadas al usuario y comandos que el usuario puede ejecutar con mayores privilegios (llamados perfiles de ejecución).

Además, esta sección se puede usar para crear roles, que también pueden tener perfiles y autorizaciones. Una vez que se crea un rol, un usuario puede heredar todos los privilegios del rol, lo que facilita otorgar los mismos derechos de acceso a múltiples usuarios.
