Кіраванне рэсурсамі
Гэтая старонка можа выкарыстоўвацца для адлюстравання абмежаванняў рэсурсаў, якія ў цяперашні час актыўныя ў сістэме для нейкага працэсу, праекта, зоны ці задачы. Гэта можа быць карысна для забеспячэння таго, што налады, зробленыя вамі ў раздзеле " Праекты " гэтага модуля, альбо модуль " Зоны Solaris" на самай справе выкарыстоўваюцца.
