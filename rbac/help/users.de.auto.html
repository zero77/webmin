Benutzerattribute
Auf dieser Seite können Unix-Benutzern Profile und Berechtigungen zugewiesen werden, um ihnen zusätzliche Berechtigungen zu erteilen. Ein Profil definiert eine Reihe von Berechtigungen, die dem Benutzer erteilt wurden, sowie Befehle, die der Benutzer mit höheren Berechtigungen ausführen kann (sogenannte Ausführungsprofile).

Darüber hinaus können in diesem Abschnitt Rollen erstellt werden, die auch Profile und Berechtigungen enthalten können. Sobald eine Rolle erstellt wurde, kann ein Benutzer alle Berechtigungen der Rolle erben, wodurch es einfach ist, mehreren Benutzern dieselben Zugriffsrechte zu gewähren.
