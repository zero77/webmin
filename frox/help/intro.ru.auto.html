Frox FTP Proxy
Frox - это сервер, который может перехватывать и пересылать FTP-соединения, сделанные клиентами, которые маршрутизируются через ваш сервер. Это позволяет кэшировать обычно загружаемый контент FTP, обеспечивать контроль доступа, по которому клиентам разрешено использовать FTP, и ограничивать объем используемой полосы пропускания. Это как эквивалент Squid, но для протокола FTP вместо HTTP.

Это не особенно полезно, если ваша система не является шлюзом между внутренней сетью и Интернетом - если вы просто ищете способ кэширования содержимого FTP в одной системе, лучше использовать Squid.

Для правильной работы Frox необходимо создать правило брандмауэра, чтобы перенаправить весь маршрутизируемый трафик через порт 21 в порт, который прослушивает Frox. Если вы используете Linux и используете модуль брандмауэра Linux для управления брандмауэром, этот модуль может легко настроить это правило для вас.
