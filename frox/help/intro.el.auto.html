Διαμεσολαβητής Frox FTP
Το Frox είναι ένας διακομιστής που μπορεί να παρακολουθεί και να προωθεί συνδέσεις FTP που πραγματοποιούνται από πελάτες που δρομολογούνται μέσω του διακομιστή σας. Αυτό σας επιτρέπει να αποθηκεύσετε προσωρινά την προσωρινή μνήμη περιεχομένου FTP, να επιβάλλετε τον έλεγχο πρόσβασης για το ποιοι πελάτες επιτρέπεται να χρησιμοποιούν FTP και να περιορίσετε το μέγεθος του εύρους ζώνης που μπορεί να καταναλωθεί. Αυτό είναι το ισοδύναμο του Squid, αλλά για το πρωτόκολλο FTP αντί για HTTP.

Δεν είναι ιδιαίτερα χρήσιμο εκτός εάν το σύστημά σας είναι πύλη μεταξύ ενός εσωτερικού δικτύου και του Διαδικτύου - εάν ψάχνετε απλώς έναν τρόπο αποθήκευσης περιεχομένου FTP σε ένα μόνο σύστημα, είναι καλύτερα να χρησιμοποιήσετε το Squid.

Για να λειτουργεί σωστά το Frox, πρέπει να δημιουργηθεί ένας κανόνας τείχους προστασίας για να ανακατευθύνει όλη τη δρομολογημένη κίνηση στη θύρα 21 στη θύρα που ακούει το Frox. Εάν χρησιμοποιείτε Linux και χρησιμοποιείτε τη λειτουργική μονάδα Linux Firewall για να διαχειριστείτε το τείχος προστασίας σας, αυτή η μονάδα μπορεί να ρυθμίσει αυτόν τον κανόνα για εσάς εύκολα.
