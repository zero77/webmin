Frox FTP Proxy
Frox è un server in grado di intercettare e inoltrare le connessioni FTP effettuate dai client che vengono instradati attraverso il server. Ciò consente di memorizzare nella cache il contenuto FTP scaricato comunemente, imporre il controllo dell'accesso su quali client sono autorizzati a utilizzare FTP e limitare la quantità di larghezza di banda che può essere consumata. Questo come l'equivalente di Squid, ma per il protocollo FTP anziché HTTP.

Non è particolarmente utile a meno che il tuo sistema non sia gateway tra una rete interna e Internet - se stai solo cercando un modo per memorizzare nella cache il contenuto FTP su un singolo sistema, è meglio usare Squid.

Affinché Frox funzioni correttamente, è necessario creare una regola firewall per reindirizzare tutto il traffico indirizzato sulla porta 21 alla porta su cui Frox è in ascolto. Se usi Linux e usi il modulo Linux Firewall per gestire il tuo firewall, questo modulo può impostare facilmente questa regola per te.
