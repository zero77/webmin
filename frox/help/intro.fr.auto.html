Proxy FTP Frox
Frox est un serveur qui peut intercepter et transférer des connexions FTP établies par des clients qui sont routés via votre serveur. Cela vous permet de mettre en cache le contenu FTP fréquemment téléchargé, d'appliquer le contrôle d'accès sur les clients autorisés à utiliser FTP et de limiter la quantité de bande passante pouvant être consommée. Celui-ci est l'équivalent de Squid, mais pour le protocole FTP au lieu de HTTP.

Il n'est pas particulièrement utile à moins que votre système ne soit une passerelle entre un réseau interne et Internet - si vous cherchez simplement un moyen de mettre en cache le contenu FTP sur un seul système, il est préférable d'utiliser Squid à la place.

Pour que Frox fonctionne correctement, une règle de pare-feu doit être créée pour rediriger tout le trafic routé sur le port 21 vers le port sur lequel Frox écoute. Si vous utilisez Linux et utilisez le module Pare-feu Linux pour gérer votre pare-feu, ce module peut configurer cette règle pour vous facilement.
