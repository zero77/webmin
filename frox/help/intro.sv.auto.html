Frox FTP Proxy
Frox är en server som kan fånga och vidarebefordra FTP-anslutningar gjorda av klienter som dirigeras via din server. Detta gör att du kan cache vanligt nedladdat FTP-innehåll, verkställa åtkomstkontroll som klienter tillåter att använda FTP och begränsa mängden bandbredd som kan konsumeras. Detta av det som ekvivalent med bläckfisk, men för FTP-protokollet istället för HTTP.

Det är inte särskilt användbart om inte ditt system är gateway mellan ett internt nätverk och internet - om du bara letar efter ett sätt att cache-FTP-innehåll på ett enda system är det bättre att använda Squid istället.

För att Frox ska fungera ordentligt måste en brandväggsregel skapas för att omdirigera all dirigerad trafik på port 21 till porten som Frox lyssnar på. Om du kör Linux och använder Linux-brandväggsmodulen för att hantera din brandvägg kan den här modulen enkelt konfigurera denna regel för dig.
