Frox FTP Proxy
Frox เป็นเซิร์ฟเวอร์ที่สามารถสกัดกั้นและส่งต่อการเชื่อมต่อ FTP ที่ทำโดยไคลเอนต์ที่ถูกกำหนดเส้นทางผ่านเซิร์ฟเวอร์ของคุณ สิ่งนี้ช่วยให้คุณสามารถแคชเนื้อหา FTP ที่ดาวน์โหลดโดยทั่วไปบังคับใช้การควบคุมการเข้าถึงที่อนุญาตให้ลูกค้าใช้ FTP และ จำกัด ปริมาณแบนด์วิดท์ที่สามารถใช้ได้ สิ่งนี้มันเทียบเท่า Squid แต่สำหรับโปรโตคอล FTP แทนที่จะเป็น HTTP

ไม่มีประโยชน์อย่างยิ่งหากระบบของคุณเป็นเกตเวย์ระหว่างเครือข่ายภายในและอินเทอร์เน็ต - หากคุณเพียงแค่มองหาวิธีการแคชเนื้อหา FTP ในระบบเดียวคุณควรใช้ Squid แทน

เพื่อให้ Frox ทำงานอย่างถูกต้องจะต้องสร้างกฎไฟร์วอลล์เพื่อเปลี่ยนเส้นทางปริมาณข้อมูลที่กำหนดเส้นทางทั้งหมดในพอร์ต 21 ไปยังพอร์ต Frox รอฟัง หากคุณใช้ Linux และใช้โมดูล Linux Firewall เพื่อจัดการไฟร์วอลล์ของคุณโมดูลนี้สามารถตั้งค่ากฎนี้ให้คุณได้อย่างง่ายดาย
