Frox FTP Proxy
Frox er en server som kan avskjære og videresende FTP-tilkoblinger laget av klienter som blir dirigert gjennom serveren din. Dette lar deg buffer ofte nedlastet FTP-innhold, håndheve tilgangskontroll som klienter tillater å bruke FTP og begrense mengden båndbredde som kan forbrukes. Dette av det som tilsvarer Squid, men for FTP-protokollen i stedet for HTTP.

Det er ikke spesielt nyttig med mindre systemet er inngangsport mellom et internt nettverk og internett - hvis du bare leter etter en måte å buffe FTP-innhold på et enkelt system, er det bedre å bruke Squid i stedet.

For at Frox skal fungere ordentlig, må det opprettes en brannmurregel for å omdirigere all dirigert trafikk på port 21 til porten Frox lytter på. Hvis du kjører Linux og bruker Linux Firewall-modulen til å administrere brannmuren, kan denne modulen enkelt konfigurere denne regelen for deg.
