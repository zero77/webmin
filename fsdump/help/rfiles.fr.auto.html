Fichiers à restaurer
Si cette option est définie, seuls les fichiers répertoriés seront restaurés à partir de la sauvegarde. Les noms de fichiers doivent être relatifs au système de fichiers à partir duquel la sauvegarde a été effectuée, donc si vous sauvegardez / home et souhaitez simplement restaurer / home / foo, vous devez entrer foo dans ce champ.
