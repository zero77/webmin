Rimontare con l'opzione noatime durante il backup?
Quando questa opzione è abilitata, il filesystem di cui verrà eseguito il backup verrà rimontato con l'opzione noatime prima dell'avvio del backup. Ciò impedisce alle letture dei file eseguite dal processo di backup di aggiornare l'ora dell'ultimo accesso di ciascun file e quindi riduce l'attività del disco.

Al termine del backup, verranno ripristinate le opzioni di montaggio del filesystem originale.
