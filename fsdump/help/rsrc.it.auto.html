Ripristina da file o dispositivo
Questa opzione determina da dove vengono letti i dati del dump del backup. Può essere un file locale o un dispositivo a nastro oppure un file o un dispositivo su un host remoto. Perché i ripristini da un host remoto funzionino, deve supportare il protocollo rmt , che di solito non è attivato di default per motivi di sicurezza.
