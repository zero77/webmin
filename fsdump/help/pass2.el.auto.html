Κωδικός πρόσβασης για σύνδεση SSH / FTP
Κατά τη δημιουργία αντιγράφων ασφαλείας ή την επαναφορά μέσω SSH ή FTP, αυτή η επιλογή μπορεί να χρησιμοποιηθεί για τον καθορισμό κωδικού πρόσβασης για την απομακρυσμένη σύνδεση. Αυτό απαιτείται όταν χρησιμοποιείτε FTP ή εάν δεν χρησιμοποιείτε τον έλεγχο ταυτότητας SSH και βασικού κλειδιού.
