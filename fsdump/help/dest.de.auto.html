Backup auf
Dies kann entweder der Pfad zu einer Datei sein, in die die Sicherung geschrieben wird, der Pfad zu einem Bandlaufwerksgerät (wie /dev/st0 ) oder eine Datei oder ein Gerät auf einem Remote-Host. Sicherungen auf einem Remote-System erfolgen über das SSH-Protokoll oder optional über das ältere unsichere RSH-Protokoll.
