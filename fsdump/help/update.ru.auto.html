Обновить файл /etc/dumpdates?
Эта опция может быть включена только при резервном копировании всей файловой системы. Если этот параметр включен, имя устройства, уровень, дата и время резервного копирования в исходной файловой системе будут записываться в удобочитаемой форме в /etc/dumpdates . На каждой файловой системе может быть только одна запись. Этот параметр полезен при использовании параметра командной строки dump -W для отображения самой последней даты дампа и уровня каждой файловой системы.
