Comando de copia de seguridad remota
Por defecto, el comando dump usará el comando rsh y el protocolo para hacer una copia de seguridad en otros servidores. Sin embargo, este campo se puede usar para seleccionar un comando diferente, como ssh , que es más seguro y se usa más comúnmente.

En los sistemas Linux y BSD, también puede seleccionar la opción FTP , que utilizará el programa de cliente FTP incorporado de Webmin para transferir la copia de seguridad hacia o desde el servidor remoto.
