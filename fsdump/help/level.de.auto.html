Dump Level
Die Speicherauszugsebene steuert, welche Dateien in die Sicherung geschrieben werden. Wenn Stufe 0 ausgewählt ist, wird jede Datei unter dem zu sichernden Verzeichnis gespeichert. Andernfalls werden nur Dateien geschrieben, die seit dem letzten Speicherauszug einer niedrigeren Ebene neu oder geändert wurden.

Wenn Sie ein Verzeichnis sichern, das kein Dateisystem-Mountpunkt ist, kann nur die Ebene 0 ausgewählt werden.
