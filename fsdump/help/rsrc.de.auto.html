Von Datei oder Gerät wiederherstellen
Diese Option bestimmt, woher die Backup-Dump-Daten gelesen werden. Dies kann entweder eine lokale Datei oder ein Bandgerät oder eine Datei oder ein Gerät auf einem Remote-Host sein. Damit Wiederherstellungen von einem Remote-Host funktionieren, muss das rmt- Protokoll unterstützt werden, das aus Sicherheitsgründen normalerweise nicht standardmäßig aktiviert ist.
