Backup do sistema de arquivos
Este módulo permite fazer backup e restaurar sistemas de arquivos inteiros usando a família de comandos dump and restore . Os backups do sistema de arquivos normalmente são feitos em fita, mas também podem ser salvos em um arquivo em outro sistema de arquivos se você não tiver uma unidade de fita.

Em alguns sistemas operacionais, o comando tar será usado ao fazer backup de um sistema de arquivos que não possui um dump nativo. Por exemplo, NFS, FAT e Reiserfs não possuem comandos de backup específicos do sistema de arquivos. O formato tar não pode armazenar certas informações avançadas do arquivo, como atributos ACLs, EXT e XFS, portanto, é preferível usar o formato dump sempre que possível.

A página principal do módulo contém uma lista de backups configurados (se houver), um botão para adicionar um novo backup e um botão para restaurar um backup de algum tipo de sistema de arquivos.
