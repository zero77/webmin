Il backup è suddiviso su più file?
Se è stato eseguito un backup su più file utilizzando la suddivisione su più file? opzione nella pagina di backup, è necessario selezionare questa opzione durante il ripristino. Il nome file da ripristinare avrà quindi 001 , 002 e così via ad esso aggiunto per ottenere i nomi effettivi dei file di backup.
