Удалить файл / стереть ленту перед резервным копированием?
Если выполняется резервное копирование в файл, если для этого параметра установлено значение «Да», команда резервного копирования удалит файл перед продолжением. При резервном копировании на ленту этот параметр приведет к удалению ленты перед продолжением.
