Basta aggiungere nuovi file per l'archiviazione?
Quando è impostato su No (come per impostazione predefinita), la destinazione del backup verrà sostituita con un archivio di tutti i file attualmente presenti nelle directory di origine. Ma se si seleziona Sì , verranno aggiunti solo i file che sono stati modificati dalla creazione dell'archivio. Ciò è utile quando si eseguono backup incrementali. Tuttavia, non può essere utilizzato con archivi compressi o remoti.
