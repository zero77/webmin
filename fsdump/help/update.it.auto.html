Aggiornare il file /etc/dumpdates?
Questa opzione può essere abilitata solo durante il backup di un intero file system. Se abilitato, il nome del dispositivo del file system di origine, il livello, la data e l'ora del backup verranno registrati in forma leggibile dall'uomo in /etc/dumpdates . Può esserci una sola voce per file system per ogni livello. Questa opzione è utile quando si utilizza l'opzione della riga di comando dump -W per visualizzare la data e il livello di dump più recenti di ciascun file system.
