Ignorer les erreurs de lecture sur les fichiers?
Lorsque cette option est définie sur Oui , toutes les erreurs de lecture des fichiers qui devraient être incluses dans la sauvegarde seront ignorées. Il est utile pour ignorer le fichier d' erreur généralement inutile modifié à mesure que nous le lisons , mais peut masquer les problèmes d'autorisation ou de corruption de fichier.
