Comando de restauración remota
Por defecto, el comando de restore usará el comando y el protocolo rsh para restaurar desde otros servidores. Sin embargo, este campo se puede usar para seleccionar un comando diferente, como ssh , que es más seguro y se usa más comúnmente. SSH solo funcionará si la configuración SSH del usuario root permite inicios de sesión sin contraseña desde este sistema al servidor remoto.
