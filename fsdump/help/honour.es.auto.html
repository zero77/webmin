¿Excluir siempre los archivos marcados?
Si se establece esta opción, las copias de seguridad siempre omitirán los archivos que se hayan marcado con el comando chattr o el Administrador de archivos, incluso cuando se realice una copia de seguridad de nivel 0. Normalmente, solo se omiten para copias de seguridad de nivel 1 o superior.
