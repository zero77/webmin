Comando di ripristino remoto
Per impostazione predefinita, il comando restore utilizzerà il comando rsh e il protocollo per ripristinare da altri server. Tuttavia, questo campo può essere utilizzato per selezionare un comando diverso come ssh , che è più sicuro e più comunemente usato. SSH funzionerà solo se la configurazione SSH dell'utente root consente accessi senza password da questo sistema al server remoto.
