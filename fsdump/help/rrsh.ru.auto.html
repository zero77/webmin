Команда удаленного восстановления
По умолчанию команда восстановления будет использовать команду rsh и протокол для восстановления с других серверов. Однако это поле можно использовать для выбора другой команды, например ssh , которая более безопасна и используется чаще. SSH будет работать только в том случае, если конфигурация SSH корневого пользователя разрешает вход без пароля из этой системы на удаленный сервер.
