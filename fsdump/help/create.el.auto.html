Δημιουργία αντιγράφων ασφαλείας
Σε αυτήν τη σελίδα μπορείτε να εισαγάγετε τις λεπτομέρειες ενός νέου διαμορφωμένου αντιγράφου ασφαλείας. Μόλις ολοκληρώσετε την εισαγωγή των λεπτομερειών αυτού του αντιγράφου ασφαλείας, κάντε κλικ στο κουμπί Δημιουργία στο κάτω μέρος της σελίδας για να το αποθηκεύσετε.
