Backup del filesystem
Questo modulo consente di eseguire il backup e il ripristino di interi file system utilizzando la famiglia di comandi dump e restore . I backup del file system vengono normalmente eseguiti su nastro, ma possono anche essere salvati in un file su un altro file system se non si dispone di un'unità nastro.

Su alcuni sistemi operativi, verrà utilizzato il comando tar quando si esegue il backup di un filesystem che non ha un dump nativo. Ad esempio, NFS, FAT e Reiserfs non hanno comandi di backup specifici del file system. Il formato tar non è in grado di memorizzare alcune informazioni avanzate sui file, come gli attributi ACL, EXT e XFS, quindi è preferibile utilizzare il formato dump ove possibile.

La pagina principale del modulo contiene un elenco di backup configurati (se presenti), un pulsante per aggiungere un nuovo backup e un pulsante per ripristinare un backup di un certo tipo di filesystem.
