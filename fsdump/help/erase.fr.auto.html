Supprimer le fichier / effacer la bande avant la sauvegarde?
Si vous sauvegardez dans un fichier, la définition de cette option sur Oui entraînera la commande de sauvegarde pour supprimer le fichier avant de continuer. Si vous sauvegardez sur une bande, cette option entraînera l'effacement de la bande avant de continuer.
