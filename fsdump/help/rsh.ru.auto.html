Команда удаленного резервного копирования
По умолчанию команда dump будет использовать команду rsh и протокол для резервного копирования на другие серверы. Однако это поле можно использовать для выбора другой команды, например ssh , которая более безопасна и используется чаще.

В системах Linux и BSD вы также можете выбрать параметр FTP , который будет использовать встроенную программу клиента Webmin для передачи резервной копии на удаленный сервер или с него.
