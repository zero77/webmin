Livello di scarico
Il livello di dump controlla quali file verranno scritti nel backup. Se viene scelto il livello 0, tutti i file nella directory di cui eseguire il backup verranno salvati. In caso contrario, verranno scritti solo i file nuovi o modificati dall'ultimo dump di un livello inferiore.

Quando si esegue il backup di una directory che non è un punto di montaggio del filesystem, l'unico livello che può essere selezionato in 0.
