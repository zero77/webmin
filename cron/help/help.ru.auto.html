Запланированные команды и задания

Эта страница справки является неполной

Вступление

Этот модуль можно использовать для настройки команд, которые будут запускаться автоматически в запланированное время. Он использует стандартный демон Unix cron , который может запускать команды от имени разных пользователей в выбранные минуты, часы, дни месяца, дни недели и месяцы года. Каждое задание принадлежит пользователю и при выполнении выполняется с разрешениями этого пользователя.

На главной странице модуля перечислены все известные запланированные задания, сгруппированные пользователем для запуска. Некоторые системы будут иметь задания, которые являются частью операционной системы и принадлежат таким пользователям, как root или sys . Также могут быть задания, созданные пользователями в вашей системе с помощью команды crontab , принадлежащей пользователям, которые их создали.

Создание нового запланированного задания

Чтобы создать новое задание, нажмите « Создать новое запланированное задание» под списком существующих заданий. Появится форма для ввода следующих параметров:
  • Пользователь для запуска от имени
    Имя пользователя Unix, под которым будет выполняться задание.

  • Активный?
    Если задание не активно, оно все равно будет отображаться в списке известных заданий, но не будет выполняться по расписанию.

  • команда
    Фактическая команда Unix для запуска во время расписаний. Это может быть любая команда, которую вы вводите в командной строке.

  • Вход в команду
    Если это поле не пустое, то все, что вы вводите, будет предоставлено в качестве ввода для команды при ее запуске. Например, если команда была mail foo@bar.com и input hello world , то введенный текст будет отправлен по электронной почте foo@bar.com .

  • Раз бежать
    Время выполнения команды выбирается путем указания минут, часов, дней месяца, месяцев года и дней недели для запуска. Для каждого из них вы можете выбрать « Все» или выбрать время запуска команды.

    Например, если вы выбрали 5 для часа и вторник для дня недели, то команда будет выполняться каждую минуту с 5:00 до 5:59 по вторникам.
    Кроме того, если вы выбрали 30 для минуты и 14 для часа, то команда будет выполняться каждый день в 14:30.

Заполнив форму, нажмите кнопку « Создать» внизу страницы. Новая работа вступит в силу немедленно.

Редактирование существующего запланированного задания

Чтобы отредактировать существующее задание, нажмите на команду из списка заданий на главной странице. Форма для редактирования существующей работы такая же, как форма для создания новой работы, описанная выше. После того, как вы внесли необходимые изменения, нажмите кнопку « Сохранить» в нижней части страницы.

Чтобы удалить работу, нажмите кнопку « Удалить» в нижней части формы редактирования работы. Это немедленно удалит работу без запроса подтверждения.
