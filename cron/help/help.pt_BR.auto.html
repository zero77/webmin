Introdução

Este módulo pode ser usado para configurar comandos para serem executados automaticamente em horários programados. Ele usa o daemon cron Unix padrão, que pode executar comandos como vários usuários em minutos, horas, dias do mês, dias da semana e meses do ano selecionados. Todo trabalho é de propriedade de um usuário e, quando executado, é executado com as permissões desse usuário.

A página principal do módulo lista todos os trabalhos agendados conhecidos, agrupados pelo usuário para serem executados como. Alguns sistemas terão trabalhos que fazem parte do sistema operacional, pertencentes a usuários como root ou sys . Também pode haver trabalhos criados por usuários em seu sistema usando o comando crontab , de propriedade dos usuários que os criaram.

Criando um novo trabalho agendado

Para criar um novo trabalho, clique em Criar novo trabalho agendado abaixo da lista de trabalhos existentes. Isso exibirá um formulário para inserir os seguintes parâmetros:
  • Usuário para executar como
    O nome de login dos usuários do Unix em que a tarefa será executada.

  • Ativo?
    Se um trabalho não estiver ativo, ele ainda será exibido na lista de trabalhos conhecidos, mas não será executado dentro do cronograma.

  • Comando
    O comando Unix real a ser executado no horário das agendas. Pode ser qualquer comando que você digitar na linha de comando.

  • Entrada para comando
    Se esse campo não estiver vazio, o que você digitar será fornecido como entrada para o comando quando ele for executado. Por exemplo, se o comando fosse mail foo@bar.com e o hello world de entrada, o texto de entrada seria enviado por email para foo@bar.com .

  • Tempos a executar
    Os horários em que um comando é executado são escolhidos especificando os minutos, horas, dias do mês, meses do ano e dias da semana a serem executados. Para cada um desses itens, você pode escolher Tudo ou selecionar os horários em que o comando deve ser executado.

    Por exemplo, se você escolher 5 para a hora e terça-feira para o dia da semana, o comando será executado a cada minuto, das 5:00 às 5:59 horas, às terças-feiras.
    Como alternativa, se você escolher 30 para o minuto e 14 para a hora, o comando será executado todos os dias às 14:30.

Depois de preencher o formulário, clique no botão Criar na parte inferior da página. O novo trabalho entrará em vigor imediatamente.

Editando um trabalho agendado existente

Para editar um trabalho existente, clique no comando na lista de trabalhos na página principal. O formulário para editar um trabalho existente é o mesmo que para criar um novo trabalho, descrito acima. Depois de fazer as alterações necessárias, clique no botão Salvar na parte inferior da página.

Para excluir um trabalho, clique no botão Excluir na parte inferior do formulário de edição do trabalho. Isso excluirá imediatamente o trabalho sem solicitar confirmação.
