Προγραμματισμένες εντολές και εργασίες

Αυτή η σελίδα βοήθειας είναι ελλιπής

Εισαγωγή

Αυτή η ενότητα μπορεί να χρησιμοποιηθεί για τη ρύθμιση εντολών που θα εκτελούνται αυτόματα σε προγραμματισμένες ώρες. Χρησιμοποιεί τον τυπικό δαίμονα Unix cron , ο οποίος μπορεί να εκτελεί εντολές ως διάφορους χρήστες σε επιλεγμένα λεπτά, ώρες, ημέρες του μήνα, ημέρες της εβδομάδας και μήνες του έτους. Κάθε εργασία ανήκει σε έναν χρήστη και όταν εκτελείται εκτελείται με τα δικαιώματα αυτού του χρήστη.

Η κύρια σελίδα της ενότητας παραθέτει όλες τις γνωστές προγραμματισμένες εργασίες, ομαδοποιημένες από τον χρήστη για εκτέλεση ως. Ορισμένα συστήματα θα έχουν εργασίες που αποτελούν μέρος του λειτουργικού συστήματος, που ανήκουν σε χρήστες όπως root ή sys . Μπορεί επίσης να υπάρχουν εργασίες που δημιουργούνται από χρήστες στο σύστημά σας χρησιμοποιώντας την εντολή crontab , που ανήκει στους χρήστες που τις δημιούργησαν.

Δημιουργία νέας προγραμματισμένης εργασίας

Για να δημιουργήσετε μια νέα εργασία, κάντε κλικ στο Δημιουργία νέας προγραμματισμένης εργασίας κάτω από τη λίστα των υπαρχουσών εργασιών. Αυτό θα εμφανίσει μια φόρμα για την εισαγωγή των ακόλουθων παραμέτρων:
  • Χρήστης για εκτέλεση ως
    Το όνομα σύνδεσης των χρηστών Unix στην οποία θα εκτελεστεί η εργασία.

  • Ενεργός?
    Εάν μια εργασία δεν είναι ενεργή, θα εξακολουθεί να εμφανίζεται στη λίστα των γνωστών εργασιών, αλλά δεν θα εκτελείται σύμφωνα με το χρονοδιάγραμμα.

  • Εντολή
    Η πραγματική εντολή Unix που θα εκτελεστεί κατά την ώρα προγραμματισμού. Αυτή μπορεί να είναι οποιαδήποτε εντολή που θα πληκτρολογήσατε στη γραμμή εντολών.

  • Εισαγωγή στην εντολή
    Εάν αυτό το πεδίο δεν είναι κενό, τότε ό, τι εισαγάγετε θα παρέχεται ως είσοδος στην εντολή όταν εκτελείται. Για παράδειγμα, εάν η εντολή ήταν mail foo@bar.com και η είσοδος hello world , τότε το κείμενο εισαγωγής θα αποσταλεί μέσω email στο foo@bar.com .

  • Ώρες να τρέξετε
    Οι ώρες εκτέλεσης μιας εντολής επιλέγονται καθορίζοντας τα λεπτά, ώρες, ημέρες του μήνα, μήνες του έτους και ημέρες εκτέλεσης της εβδομάδας. Για καθένα από αυτά μπορείτε είτε να επιλέξετε Όλα , είτε να επιλέξετε τις ώρες που πρέπει να εκτελεστεί η εντολή.

    Για παράδειγμα, εάν επιλέξατε 5 για την ώρα και την Τρίτη για την ημέρα της εβδομάδας, τότε η εντολή θα εκτελείται κάθε λεπτό από τις 5:00 π.μ. έως τις 5:59 π.μ. την Τρίτη.
    Εναλλακτικά, εάν επιλέξατε 30 για το λεπτό και 14 για την ώρα, τότε η εντολή θα εκτελείται καθημερινά στις 2:30 μ.μ.

Μόλις συμπληρώσετε τη φόρμα, κάντε κλικ στο κουμπί Δημιουργία στο κάτω μέρος της σελίδας. Η νέα δουλειά θα τεθεί σε ισχύ αμέσως.

Επεξεργασία υπάρχουσας προγραμματισμένης εργασίας

Για να επεξεργαστείτε μια υπάρχουσα εργασία, κάντε κλικ στην εντολή από τη λίστα εργασιών στην κύρια σελίδα. Η φόρμα για την επεξεργασία μιας υπάρχουσας εργασίας είναι η ίδια με τη φόρμα για τη δημιουργία μιας νέας εργασίας, που περιγράφεται παραπάνω. Αν έχετε κάνει ό, τι χρειάζεται, κάντε κλικ στο κουμπί Αποθήκευση στο κάτω μέρος της σελίδας.

Για να διαγράψετε μια εργασία, κάντε κλικ στο κουμπί Διαγραφή στο κάτω μέρος της φόρμας επεξεργασίας εργασίας. Αυτό θα διαγράψει αμέσως την εργασία χωρίς να ζητήσει επιβεβαίωση.
