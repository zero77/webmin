Να μην επιτρέπεται η εκτέλεση δυαδικών αρχείων;
Επιτρέψτε την εκτέλεση τυχόν δυαδικών αρχείων στο εγκατεστημένο σύστημα αρχείων.
Αυτή η επιλογή μπορεί να είναι χρήσιμη για έναν διακομιστή που διαθέτει συστήματα αρχείων που περιέχουν δυαδικά αρχεία για αρχιτεκτονικές εκτός από τη δική της.

επιλογές προσάρτησης: exec / noexec
