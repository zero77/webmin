Ausführung von Binärdateien nicht zulassen?
Ermöglichen Sie die Ausführung von Binärdateien im bereitgestellten Dateisystem.
Diese Option kann für einen Server nützlich sein, der über Dateisysteme verfügt, die Binärdateien für andere als eigene Architekturen enthalten.

Mount-Optionen: exec / noexec
