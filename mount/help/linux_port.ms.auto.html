Pelabuhan NFS
Nilai numerik port untuk disambungkan ke pelayan NFS dihidupkan.
Sekiranya nombor port adalah 0 (lalai), tanyakan portmapper host jauh untuk nombor port yang akan digunakan. Sekiranya daemon NFS host jauh tidak didaftarkan dengan portmappernya, sebaliknya nombor port NFS 2049 akan digunakan.
