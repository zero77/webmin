NFS Port
Den numeriske verdien for porten som skal kobles til NFS-serveren.
Hvis portnummeret er 0 (standard), må du spørre den eksterne verts portmapper for portnummeret som skal brukes. Hvis den eksterne verts NFS-demonen ikke er registrert med portmapper, brukes standard NFS-portnummer 2049 i stedet.

monteringsalternativ: port
standard: 0
