Порт NFS
Числовое значение порта для подключения к серверу NFS.
Если номер порта равен 0 (по умолчанию), запросите номер порта для удаленного хоста. Если демон NFS удаленного хоста не зарегистрирован в своем portmapper, вместо него используется стандартный номер порта NFS 2049.

опция монтирования: порт
по умолчанию: 0
