Serveur PPP Dialin
Ce module vous permet de configurer votre machine Linux avec un modem connecté en tant que serveur de numérotation qui permet à d'autres machines de s'y connecter, de se connecter et d'ouvrir des sessions PPP. Il peut également être utilisé pour autoriser les connexions et les connexions PPP à partir d'une autre machine connectée directement à la vôtre via un câble null-modem.

Afin de configurer l'accès à distance, vous devez suivre ces étapes:

  1. Dans la page Configuration du port série, ajoutez une entrée pour le port sur lequel votre modem est connecté. L 'démarrera Mgetty sur ce port pour répondre automatiquement aux appels téléphoniques reçus par votre modem et afficher une invite de connexion. Si tout ce que vous voulez faire est d'autoriser les connexions texte uniquement via un modem ou à partir d'un terminal stupide, c'est la seule étape que vous devez faire.

  2. Dans la page Options PPP, définissez l'adresse IP qui sera attribuée pour la connexion des clients et toutes les autres options dont vous avez besoin. Le plus utile est «Détecter automatiquement les connexions PPP sur les ports série? Cependant, si cette option est activée, vous devez également activer "Exiger l'authentification?" pour empêcher les clients non fiables de se connecter.

  3. Si l'authentification est activée, utilisez la page Comptes PPP pour créer des comptes pour toutes les personnes par lesquelles vous souhaitez autoriser la connexion. Pour chaque compte, le serveur doit être défini sur «Tout» et les adresses valides sur «Autoriser tout».
