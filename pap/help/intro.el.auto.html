Διακομιστής κλήσης PPP
Αυτή η ενότητα σάς επιτρέπει να ρυθμίσετε το μηχάνημά σας Linux με ένα συνημμένο μόντεμ ως διακομιστή κλήσης που επιτρέπει σε άλλα μηχανήματα να καλέσουν σε αυτό, να συνδεθούν και να ανοίξουν συνεδρίες PPP. Μπορεί επίσης να χρησιμοποιηθεί για να επιτρέψει συνδέσεις και συνδέσεις PPP από άλλο μηχάνημα που συνδέεται απευθείας με το δικό σας μέσω καλωδίου null-modem.

Για να ρυθμίσετε την πρόσβαση μέσω τηλεφώνου, πρέπει να ακολουθήσετε τα παρακάτω βήματα:

  1. Στη σελίδα Διαμόρφωση σειριακής θύρας, προσθέστε μια καταχώριση για τη θύρα στην οποία είναι συνδεδεμένο το μόντεμ σας. Θα ξεκινήσει το Mgetty σε αυτήν τη θύρα για να απαντά αυτόματα στις τηλεφωνικές κλήσεις που λαμβάνονται από το μόντεμ σας και θα εμφανίζει μια προτροπή σύνδεσης. Εάν το μόνο που θέλετε να κάνετε είναι να επιτρέπετε συνδέσεις μόνο με κείμενο μέσω μόντεμ ή από έναν ανόητο τερματικό, αυτό είναι το μόνο βήμα που πρέπει να κάνετε.

  2. Στη σελίδα Επιλογές PPP, ορίστε τη διεύθυνση IP που θα εκχωρηθεί για τη σύνδεση πελατών και οποιωνδήποτε άλλων επιλογών χρειάζεστε. Το πιο χρήσιμο είναι "Αυτόματη ανίχνευση συνδέσεων PPP σε σειριακές θύρες;", κάτι που θα προκαλέσει σε οποιονδήποτε πελάτη που ξεκινά αμέσως μια συνεδρία PPP μετά τη σύνδεση να παραλείψει τη συνήθη σύνδεση unix με κείμενο. Ωστόσο, εάν αυτή η επιλογή είναι ενεργοποιημένη, θα πρέπει επίσης να ενεργοποιήσετε την επιλογή "Απαιτείται έλεγχος ταυτότητας;" για να αποτρέψετε τη σύνδεση μη αξιόπιστων πελατών.

  3. Εάν είναι ενεργοποιημένος ο έλεγχος ταυτότητας, χρησιμοποιήστε τη σελίδα Λογαριασμοί PPP για να δημιουργήσετε λογαριασμούς για όλα τα άτομα από τα οποία θέλετε να επιτρέψετε τη σύνδεση. Για κάθε λογαριασμό, ο διακομιστής θα πρέπει να οριστεί σε "Οποιοδήποτε" και οι έγκυρες διευθύνσεις σε "Να επιτρέπονται".
