PPP Dialin Server
Этот модуль позволяет вам настроить ваш Linux-компьютер с подключенным модемом в качестве сервера удаленного доступа, который позволяет другим компьютерам подключаться к нему, входить в систему и открывать сеансы PPP. Его также можно использовать для разрешения входа в систему и PPP-соединений с другого компьютера, подключенного напрямую к вашему через нуль-модемный кабель.

Для настройки доступа по телефонному соединению необходимо выполнить следующие действия:

  1. На странице конфигурации последовательного порта добавьте запись для порта, к которому подключен модем. Mgetty запустит этот порт, чтобы автоматически отвечать на телефонные звонки, полученные модемом, и отображать запрос на вход в систему. Если все, что вы хотите сделать, это разрешить только текстовые входы через модем или через тупой терминал, это единственный шаг, который вам нужно сделать.

  2. На странице параметров PPP укажите IP-адрес, который будет назначен для подключения клиентов, и любые другие необходимые параметры. Наиболее полезным является «Автоматически обнаруживать PPP-соединения на последовательных портах?», Что приводит к тому, что любой клиент, который сразу же начинает сеанс PPP после подключения, пропускает обычный текстовый вход в Unix. Однако, если эта опция включена, вам также следует включить «Требовать аутентификацию?» чтобы предотвратить подключение ненадежных клиентов.

  3. Если аутентификация включена, используйте страницу учетных записей PPP, чтобы создать учетные записи для всех людей, которым вы хотите разрешить вход в систему. Для каждой учетной записи сервер должен иметь значение «Любой», а действительные адреса - «Разрешить любой».
