PPP 다이얼 인 서버
이 모듈을 사용하면 다른 시스템이 전화 접속 서버로 전화 접속하여 PPP 세션을 열 수 있도록 전화 접속 서버로 연결된 모뎀으로 Linux 시스템을 설정할 수 있습니다. 널 모뎀 케이블을 통해 사용자 컴퓨터에 직접 연결된 다른 컴퓨터의 로그인 및 PPP 연결을 허용하는 데에도 사용할 수 있습니다.

전화 접속 액세스를 설정하려면 다음 단계를 수행해야합니다.

  1. Serial Port Configuration (직렬 포트 구성) 페이지에서 모뎀이 연결된 포트에 대한 항목을 추가하십시오. 해당 포트에서 Mgetty가 시작되어 모뎀이 수신 한 전화에 자동으로 응답하고 로그인 프롬프트가 표시됩니다. 모뎀이나 벙어리 터미널을 통한 텍스트 전용 로그인 만 허용하면이 단계 만 수행하면됩니다.

  2. PPP 옵션 페이지에서 클라이언트 연결에 필요한 IP 주소와 필요한 다른 옵션을 설정하십시오. 가장 유용한 방법은 '직렬 포트에서 PPP 연결 자동 감지?'입니다. 이는 연결 후 즉시 PPP 세션을 시작하는 클라이언트가 일반적인 텍스트 기반 유닉스 로그인을 건너 뛰도록합니다. 그러나이 옵션이 켜져 있으면 '인증 필요?'도 활성화해야합니다. 신뢰할 수없는 클라이언트가 연결되지 않도록합니다.

  3. 인증이 활성화 된 경우 PPP 계정 페이지를 사용하여 로그인을 허용하려는 모든 사람의 계정을 만듭니다. 각 계정에 대해 서버는 '임의'로 설정하고 유효한 주소는 '허용'으로 설정해야합니다.
