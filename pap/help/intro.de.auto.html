PPP-Einwahlserver
Mit diesem Modul können Sie Ihren Linux-Computer mit einem angeschlossenen Modem als Einwahlserver einrichten, über den andere Computer sich einwählen, sich anmelden und PPP-Sitzungen öffnen können. Es kann auch verwendet werden, um Anmeldungen und PPP-Verbindungen von einem anderen Computer zuzulassen, der über ein Nullmodemkabel direkt mit Ihrem verbunden ist.

Um den Einwahlzugriff einzurichten, müssen Sie die folgenden Schritte ausführen:

  1. Fügen Sie auf der Seite Konfiguration des seriellen Anschlusses einen Eintrag für den Anschluss hinzu, an den Ihr Modem angeschlossen ist. Das startet Mgetty an diesem Port, um von Ihrem Modem empfangene Anrufe automatisch zu beantworten und eine Anmeldeaufforderung anzuzeigen. Wenn Sie nur Anmeldungen nur über Text über ein Modem oder von einem dummen Terminal aus zulassen möchten, ist dies der einzige Schritt, den Sie ausführen müssen.

  2. Legen Sie auf der Seite PPP-Optionen die IP-Adresse fest, die für die Verbindung von Clients zugewiesen wird, sowie alle anderen Optionen, die Sie benötigen. Am nützlichsten ist "PPP-Verbindungen an seriellen Schnittstellen automatisch erkennen?", Was dazu führt, dass jeder Client, der sofort nach der Verbindung eine PPP-Sitzung startet, die übliche textbasierte Unix-Anmeldung überspringt. Wenn diese Option aktiviert ist, sollten Sie auch "Authentifizierung erforderlich?" Aktivieren. um zu verhindern, dass nicht vertrauenswürdige Clients eine Verbindung herstellen.

  3. Wenn die Authentifizierung aktiviert ist, erstellen Sie auf der Seite PPP-Konten Konten für alle Personen, für die Sie die Anmeldung zulassen möchten. Für jedes Konto sollte der Server auf "Beliebig" und die gültigen Adressen auf "Beliebig zulassen" eingestellt sein.
