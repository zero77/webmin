Servidor PPP Dialin
Este módulo le permite configurar su máquina Linux con un módem conectado como un servidor de marcación que permite a otras máquinas marcar, iniciar sesión y abrir sesiones PPP. También se puede utilizar para permitir inicios de sesión y conexiones PPP desde otra máquina conectada directamente a la suya a través de un cable de módem nulo.

Para configurar el acceso telefónico, debe seguir estos pasos:

  1. En la página Configuración del puerto serie, agregue una entrada para el puerto en el que está conectado su módem. Iniciará Mgetty en ese puerto para responder automáticamente las llamadas telefónicas recibidas por su módem y mostrará un mensaje de inicio de sesión. Si todo lo que quiere hacer es permitir inicios de sesión de solo texto a través de módem o desde una terminal tonta, este es el único paso que debe hacer.

  2. En la página de Opciones de PPP, configure la dirección IP que se asignará para conectar clientes y cualquier otra opción que necesite. Lo más útil es "¿Detectar automáticamente conexiones PPP en puertos seriales?", Lo que hará que cualquier cliente que inicie inmediatamente una sesión PPP después de conectarse se salte el inicio de sesión habitual de Unix basado en texto. Sin embargo, si esta opción está activada, también debe habilitar '¿Requerir autenticación?' para evitar que clientes no confiables se conecten.

  3. Si la autenticación está habilitada, use la página Cuentas PPP para crear cuentas para todas las personas a las que desea permitir el inicio de sesión. Para cada cuenta, el servidor debe establecerse en 'Cualquiera' y las direcciones válidas en 'Permitir cualquiera'.
