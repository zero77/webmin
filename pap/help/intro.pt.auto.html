Servidor de discagem PPP
Este módulo permite que você configure sua máquina Linux com um modem conectado como um servidor de discagem que permite que outras máquinas discem, façam login e abram sessões PPP. Também pode ser usado para permitir logins e conexões PPP de outra máquina conectada diretamente à sua via cabo de modem nulo.

Para configurar o acesso à discagem, siga estas etapas:

  1. Na página Configuração da porta serial, adicione uma entrada para a porta na qual o modem está conectado. O Mgetty iniciará nessa porta para atender automaticamente as chamadas telefônicas recebidas pelo modem e exibir um prompt de login. Se tudo o que você deseja é permitir logins somente de texto via modem ou em um terminal idiota, esta é a única etapa que você precisa executar.

  2. Na página Opções de PPP, defina o endereço IP que será atribuído para conectar os clientes e quaisquer outras opções necessárias. O mais útil é 'Detectar automaticamente conexões PPP em portas seriais?', O que fará com que qualquer cliente que inicie imediatamente uma sessão PPP após a conexão salte o login unix baseado em texto usual. No entanto, se essa opção estiver ativada, você também deverá ativar 'Exigir autenticação?' para impedir que clientes não confiáveis se conectem.

  3. Se a autenticação estiver ativada, use a página Contas PPP para criar contas para todas as pessoas pelas quais você deseja permitir o login. Para cada conta, o servidor deve ser definido como 'Qualquer' e os endereços válidos para 'Permitir qualquer'.
