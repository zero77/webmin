Asynchronní zápisy
"Ano" - Zápisy na server se provádějí asynchronně. Když váš klient NFS odešle požadavek na zápis na server NFS, vrátí server pozitivní odpověď klientovi před zapisováním jakýchkoli dat na disk. Váš klient pak může odeslat další požadavek na zápis bez čekání na dokončení předchozího zápisu na serveru. Výkon zápisu je lepší u asynchronních zápisů než u synchronních zápisů. Pokud však server havaruje po odeslání pozitivní odpovědi klientovi, ale před zápisem dat na disk, dojde ke ztrátě dat a klient není informován.

„Ne“ - Zápisy na server se provádějí synchronně. Když váš klient NFS odešle požadavek na zápis na server NFS, server zapíše data na svůj disk a poté vrátí pozitivní odpověď vašemu klientovi. Před odesláním další žádosti o zápis musí klient počkat, než server zapíše data na disk. Výkon zápisu není u synchronních zápisů tak dobrý jako u asynchronních zápisů.

Výchozí hodnota je „Ne“.
