Introdução aos compartilhamentos NFS

HPUX NFS

Este módulo permite configurar diretórios (chamados de exportações no HPUX) compartilhados via NFS do seu sistema. A primeira página mostra uma lista de diretórios e os clientes para os quais são exportados, permitindo clicar no cliente para editar suas opções ou começar a exportar um novo diretório.

Abaixo da lista de compartilhamentos na primeira página, há um botão que ativará a configuração atual.

O HPUX não permite reexportar um diretório NFS montado de outro servidor - apenas os diretórios nos sistemas de arquivos podem ser exportados. Se um diretório compartilhado contiver o ponto de montagem de outro sistema de arquivos local, os clientes NFS desse compartilhamento não verão os arquivos abaixo desse ponto de montagem. O servidor deve reexportar cada sistema de arquivos local separadamente, para ser montado pelos clientes no ponto correto na hierarquia de diretórios.
