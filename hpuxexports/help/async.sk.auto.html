Asynchrónne zápisy
„Áno“ - Zápisy na server sa vykonávajú asynchrónne. Keď váš klient NFS odošle žiadosť o zápis na server NFS, server vráti klientovi pozitívnu odpoveď predtým, ako zapíše akékoľvek údaje na disk. Váš klient potom môže odoslať ďalšiu požiadavku na zápis bez čakania na dokončenie predchádzajúceho zápisu na serveri. Výkon zápisu je lepší pri asynchrónnom zápise ako pri synchrónnom zápise. Ak však server havaruje po odoslaní pozitívnej odpovede klientovi, ale pred zápisom údajov na disk, údaje sa stratia a klient nebude upovedomený.

„Nie“ - Zápisy na server sa vykonávajú synchrónne. Keď váš klient NFS odošle žiadosť o zápis na server NFS, server zapíše údaje na svoj disk a potom vráti pozitívnu odpoveď vášmu klientovi. Pred odoslaním ďalšej žiadosti o zápis musí klient počkať, kým server nezapíše údaje na disk. Zápisový výkon nie je pri synchronných zápisoch taký dobrý ako pri asynchrónnych zápisoch.

Predvolená hodnota je „Nie“.
