Редагувати пакет
У верхньому розділі цієї сторінки показано подробиці пакета, встановленого на одному або декількох керованих хостах. У випадку з пакетом, який знаходиться на декількох серверах, повний опис та інші деталі будуть взяті з перших серверів, перелічених у нижній частині сторінки.

Нижче деталі пакета знаходяться кнопки для переліку файлів, що складають пакет, та для видалення пакета або з одного сервера, або з усіх серверів. Внизу - таблиця піктограм, по одному для кожного з серверів, на яких, як відомо, встановлений пакет.
