Pacotes de software de cluster
Este módulo Webmin permite gerenciar o software instalado em vários servidores em um cluster a partir de uma interface. É semelhante ao módulo Pacotes de Software, mas com a capacidade de executar a maioria das funções (como instalação, consulta e exclusão de pacotes) em muitos servidores Webmin em uma operação.

A parte superior da página principal, sob o título Servidores gerenciados, lista outros servidores Webmin cujo software está sendo gerenciado por este módulo. Para adicionar um servidor a esta lista, você deve primeiro adicioná-lo ao módulo Servidores Webmin, com um nome de usuário e senha especificados para efetuar login no Webmin nesse servidor. Em seguida, você pode selecionar o servidor na lista ao lado do botão Adicionar servidor.

Quando um servidor é adicionado, o módulo verifica se está usando o mesmo sistema de gerenciamento de pacotes que este servidor (como RPM ou Debian) e, em seguida, baixa uma lista de todos os pacotes instalados para cache local.

Um servidor que sempre aparecerá na lista daqueles disponíveis para adição é 'this server', para que você possa gerenciar pacotes localmente e em vários servidores remotos a partir de uma única interface.

Quando houver pelo menos um ícone em Servidores Gerenciados, você poderá usar o formulário Pacotes Instalados para procurar pacotes em todos os servidores ou clicar no botão Atualizar listas de pacotes para buscar novamente a lista de pacotes de todos os servidores, se os pacotes tiverem sido adicionados ou excluídos. através de algum outro módulo ou método.

Na parte inferior da página, há um formulário para instalar novos pacotes em todos os servidores gerenciados. Você pode optar por instalar a partir de um arquivo neste servidor, de um arquivo carregado ou de um arquivo baixado via HTTP ou FTP de outro site. Ao instalar a partir de um arquivo baixado, você pode escolher se cada servidor gerenciado deve baixar novamente o pacote do site original ou receber uma cópia desse servidor. A primeira opção será mais rápida se o site em que o pacote estiver estiver dentro da sua rede local.
