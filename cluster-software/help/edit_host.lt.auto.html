Tvarko serveris
Šiame puslapyje pateikiama išsami informacija apie vieną iš serverių, kurio programinės įrangos paketus tvarko šis modulis, ir į serverį įdiegtų paketų medį. Medį galite išplėsti spustelėdami aplanko piktogramas arba per išsamią informaciją apie paketą spustelėdami jo nuorodą medyje.

Norėdami pašalinti šį serverį iš valdomų sąrašų, spustelėkite mygtuką, esantį žemiau serverio informacijos.
