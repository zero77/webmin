Serveur géré
Cette page affiche les détails de l'un des serveurs dont les packages logiciels sont gérés par ce module, ainsi qu'une arborescence de packages installés sur le serveur. Vous pouvez développer l'arborescence en cliquant sur les icônes des dossiers, ou via les détails d'un package en cliquant sur son lien dans l'arborescence.

Pour supprimer ce serveur de la liste gérée, cliquez sur le bouton sous les détails du serveur.
