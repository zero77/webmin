Server gestionat
Această pagină prezintă detaliile unuia dintre serverele ale căror pachete software sunt gestionate de acest modul și un arbore de pachete instalate pe server. Puteți extinde arborele făcând clic pe pictogramele folderului sau prin detaliile unui pachet făcând clic pe linkul său din copac.

Pentru a elimina acest server din lista gestionată, faceți clic pe butonul de sub detaliile serverului.
