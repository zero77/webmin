Kluster software paketeak
Webmin modulu honek kluster batean zerbitzari anitzetan instalatutako softwarea kudeatzeko aukera ematen du interfaze batetik. Software Paketeen moduluaren antzekoa da, baina Webmin zerbitzari askotan funtzio gehienak (adibidez, paketeak instalatzea, galdetzea eta ezabatzea) egiteko gaitasuna du.

Kudeatutako zerbitzariak goiburuko azpiko orrialde nagusiaren goialdean modulu honek kudeatzen dituen beste Webmin zerbitzari batzuk agertzen dira. Zerbitzari bat zerrenda honetan gehitzeko, lehenik Webmin Servers moduluan gehitu behar duzu, erabiltzaile izena eta pasahitza zehaztuta zerbitzari horretan Webmin sartzeko. Zerbitzaria Gehitu Zerbitzaria botoiaren ondoko zerrendatik hauta dezakezu.

Zerbitzaria gehitzen denean, moduluak egiaztatu egingo du zerbitzari honen paketeak kudeatzeko sistema bera erabiltzen duela (adibidez, RPM edo Debian), eta, ondoren, lokalean gordetzeko instalatutako pakete guztien zerrenda deskargatuko du.

Gehitzeko erabilgarri daudenen zerrendan beti agertuko den zerbitzaria da "zerbitzari hau", eta, beraz, interfaze batetik paketeak lokalki eta urruneko zerbitzarietan kudeatu ahal izango dituzu.

Kudeatutako zerbitzariak azpian gutxienez ikonoa dagoenean, Instalatutako Paketeak inprimakia erabil dezakezu zerbitzarietan paketeak bilatzeko edo sakatu Eguneratu paketeen zerrendako botoia berriro zerbitzari guztietatik paketeak berriro biltzeko. beste modulu edo metodo batzuen bidez.

Orriaren behealdean kudeatutako zerbitzari guztietan pakete berriak instalatzeko formularioa dago. Zerbitzari honetako fitxategietatik, kargatutako fitxategitik edo HTTP edo FTP bidez deskargatutako fitxategi batetik beste gune batzuetatik instalatu dezakezu. Deskargatutako fitxategi batetik instalatzerakoan, kudeatutako zerbitzari bakoitzak paketea jatorrizko gunetik berriro deskargatu behar duen ala ez aukeratu dezakezu edo zerbitzari honetatik kopia bat jaso. Lehenengo aukera azkarragoa izango da paketea dagoen tokia sare lokalaren barruan badago.
