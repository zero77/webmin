Server Immexxi
Din il-paġna turi d-dettalji ta 'wieħed mis-servers li l-pakketti tas-softwer tagħhom huma ġestiti minn dan il-modulu, u siġra ta' pakketti installati fuq is-server. Tista 'tespandi s-siġra billi tikklikkja fuq l-ikoni tal-fowlder, jew permezz tad-dettalji ta' pakkett billi tikklikkja fuq il-link tagħha fis-siġra.

Biex tneħħi dan is-server mil-lista ġestita, ikklikkja fuq il-buttuna taħt id-dettalji tas-server.
