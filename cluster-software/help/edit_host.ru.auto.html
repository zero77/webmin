Управляемый сервер
На этой странице отображаются сведения об одном из серверов, программные пакеты которых управляются этим модулем, а также дерево пакетов, установленных на сервере. Вы можете развернуть дерево, нажав на значки папок, или с помощью сведений о пакете, щелкнув по его ссылке в дереве.

Чтобы удалить этот сервер из списка управляемых, нажмите кнопку под информацией о сервере.
