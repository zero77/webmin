Modifier le package
La section supérieure de cette page affiche les détails d'un package installé sur un ou plusieurs hôtes gérés. Dans le cas d'un package qui est sur plusieurs serveurs, la description complète et les autres détails seront tirés des premiers serveurs répertoriés au bas de la page.

Sous les détails du package se trouvent des boutons pour répertorier les fichiers qui composent le package et pour désinstaller le package d'un seul serveur ou de tous les serveurs. En bas se trouve un tableau d'icônes, une pour chacun des serveurs sur lesquels le package est connu pour être installé.
