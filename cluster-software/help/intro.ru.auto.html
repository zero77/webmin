Пакеты программ кластера
Этот модуль Webmin позволяет управлять программным обеспечением, установленным на нескольких серверах в кластере, из одного интерфейса. Он похож на модуль Software Packages, но с возможностью выполнять большинство функций (например, установку, запрос и удаление пакетов) на многих серверах Webmin за одну операцию.

В верхней части главной страницы под заголовком «Управляемые серверы» перечислены другие серверы Webmin, программное обеспечение которых управляется этим модулем. Чтобы добавить сервер в этот список, вы должны сначала добавить его в модуль Webmin Servers, указав имя пользователя и пароль для входа в Webmin на этом сервере. Затем вы можете выбрать сервер из списка рядом с кнопкой «Добавить сервер».

Когда сервер добавлен, модуль проверяет, использует ли он ту же систему управления пакетами, что и этот сервер (например, RPM или Debian), а затем загружает список всех установленных пакетов для локального кэширования.

Один сервер, который всегда будет отображаться в списке доступных для добавления, - это «этот сервер», так что вы можете управлять пакетами локально и на нескольких удаленных серверах из одного интерфейса.

Если в разделе «Управляемые серверы» есть хотя бы один значок, вы можете использовать форму «Установленные пакеты» для поиска пакетов на всех серверах или нажать кнопку «Обновить списки пакетов», чтобы повторно получить список пакетов со всех серверов, если пакеты были добавлены или удалены через какой-то другой модуль или метод.

Внизу страницы находится форма для установки новых пакетов на все управляемые серверы. Вы можете выбрать установку из файла на этом сервере, из загруженного файла или из файла, загруженного через HTTP или FTP с какого-либо другого сайта. При установке из загруженного файла вы можете выбрать, должен ли каждый управляемый сервер повторно загружать пакет с исходного сайта или получать копию с этого сервера. Первый вариант будет быстрее, если сайт, на котором находится пакет, находится в вашей локальной сети.
