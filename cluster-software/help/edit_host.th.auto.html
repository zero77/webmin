เซิร์ฟเวอร์ที่มีการจัดการ
หน้านี้แสดงรายละเอียดของหนึ่งในเซิร์ฟเวอร์ที่โมดูลซอฟต์แวร์ได้รับการจัดการโดยโมดูลนี้และแผนผังของแพ็คเกจที่ติดตั้งบนเซิร์ฟเวอร์ คุณสามารถขยายแผนผังโดยคลิกที่ไอคอนโฟลเดอร์หรือผ่านรายละเอียดของแพ็คเกจโดยคลิกที่ลิงก์ในแผนผัง

หากต้องการลบเซิร์ฟเวอร์นี้ออกจากรายการที่มีการจัดการให้คลิกที่ปุ่มด้านล่างรายละเอียดเซิร์ฟเวอร์
