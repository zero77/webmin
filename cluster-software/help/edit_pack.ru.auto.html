Редактировать пакет
В верхнем разделе этой страницы показаны сведения о пакете, установленном на одном или нескольких управляемых хостах. В случае пакета, который находится на нескольких серверах, полное описание и другие сведения будут взяты с первых серверов, перечисленных внизу страницы.

Ниже сведений о пакете находятся кнопки для вывода списка файлов, составляющих пакет, и для удаления пакета с одного или всех серверов. Внизу находится таблица значков, по одному для каждого из серверов, на которых, как известно, установлен пакет.
