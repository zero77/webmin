Modifica pacchetto
La sezione superiore di questa pagina mostra i dettagli di un pacchetto installato su uno o più host gestiti. Nel caso di un pacchetto che si trova su più server, la descrizione completa e altri dettagli verranno presi dai primi server elencati nella parte inferiore della pagina.

Di seguito i dettagli del pacchetto sono i pulsanti per elencare i file che compongono il pacchetto e per disinstallare il pacchetto da un singolo server o da tutti i server. Nella parte inferiore è presente una tabella di icone, una per ciascuno dei server su cui è noto che il pacchetto è installato.
