Διαχειριζόμενος διακομιστής
Αυτή η σελίδα εμφανίζει τις λεπτομέρειες ενός από τους διακομιστές του οποίου τα πακέτα λογισμικού διαχειρίζονται αυτή η λειτουργική μονάδα και ένα δέντρο πακέτων εγκατεστημένο στον διακομιστή. Μπορείτε να επεκτείνετε το δέντρο κάνοντας κλικ στα εικονίδια του φακέλου ή μέσω των λεπτομερειών ενός πακέτου κάνοντας κλικ στον σύνδεσμό του στο δέντρο.

Για να καταργήσετε αυτόν τον διακομιστή από τη διαχειριζόμενη λίστα, κάντε κλικ στο κουμπί κάτω από τις λεπτομέρειες του διακομιστή.
