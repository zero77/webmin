Redigera paketet
Det övre avsnittet på denna sida visar detaljerna för ett paket som är installerat på en eller flera hanterade värdar. När det gäller ett paket som finns på flera servrar kommer den fullständiga beskrivningen och andra detaljer att tas från de första servrarna som listas längst ner på sidan.

Nedanför paketinformationen finns knappar för att lista de filer som utgör paketet och för att avinstallera paketet från antingen en enda server eller alla servrar. Längst ner finns en tabell med ikoner, en för var och en av servrarna som paketet är känt för att vara installerat på.
