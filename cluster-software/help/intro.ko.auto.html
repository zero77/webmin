클러스터 소프트웨어 패키지
이 Webmin 모듈을 사용하면 하나의 인터페이스에서 클러스터의 여러 서버에 설치된 소프트웨어를 관리 할 수 있습니다. 소프트웨어 패키지 모듈과 유사하지만 한 번의 작업으로 많은 Webmin 서버에서 대부분의 기능 (패키지 설치, 쿼리 및 삭제 등)을 수행 할 수 있습니다.

관리 서버 제목 아래의 기본 페이지 상단에는이 모듈에서 소프트웨어를 관리하는 다른 Webmin 서버가 나열됩니다. 이 목록에 서버를 추가하려면 먼저 해당 서버에서 Webmin에 로그인하도록 사용자 이름과 비밀번호를 지정하여 Webmin 서버 모듈에 서버를 추가해야합니다. 그런 다음 서버 추가 버튼 옆의 목록에서 서버를 선택할 수 있습니다.

서버가 추가되면 모듈은이 서버와 동일한 패키지 관리 시스템 (예 : RPM 또는 Debian)을 사용하고 있는지 확인한 다음 로컬로 캐싱하기 위해 설치된 모든 패키지 목록을 다운로드합니다.

추가 할 수있는 서버 목록에 항상 표시되는 하나의 서버는 '이 서버'이므로 하나의 인터페이스에서 로컬 및 여러 원격 서버의 패키지를 관리 할 수 있습니다.

관리되는 서버 아래에 아이콘이 하나 이상 있으면 설치된 패키지 양식을 사용하여 모든 서버에서 패키지를 검색하거나 패키지 목록 새로 고침 단추를 클릭하여 패키지가 추가 또는 삭제 된 경우 모든 서버에서 패키지 목록을 다시 가져올 수 있습니다 다른 모듈이나 방법을 통해.

페이지 맨 아래에는 모든 관리 서버에 새 패키지를 설치하기위한 양식이 있습니다. 이 서버의 파일, 업로드 된 파일 또는 다른 사이트에서 HTTP 또는 FTP를 통해 다운로드 한 파일에서 설치하도록 선택할 수 있습니다. 다운로드 한 파일에서 설치하는 경우 각 관리 서버가 원본 사이트에서 패키지를 다시 다운로드해야하는지 또는이 서버로부터 사본을 수신해야하는지 선택할 수 있습니다. 패키지가있는 사이트가 로컬 네트워크 내에 있으면 이전 옵션이 더 빠릅니다.
