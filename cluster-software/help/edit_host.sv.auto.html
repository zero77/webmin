Hanterad server
Den här sidan visar detaljerna om en av servrarna vars programvarupaket hanteras av den här modulen och ett paketträd installerat på servern. Du kan utöka trädet genom att klicka på mappikonerna, eller via detaljerna om ett paket genom att klicka på länken i trädet.

För att ta bort den här servern från den hanterade listan, klicka på knappen nedanför serverinformationen.
