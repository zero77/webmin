Spravovaný server
Tato stránka zobrazuje podrobnosti o jednom ze serverů, jejichž softwarové balíčky jsou spravovány tímto modulem, a strom balíčků nainstalovaných na serveru. Strom můžete rozbalit kliknutím na ikony složek nebo prostřednictvím podrobností o balíčku kliknutím na jeho odkaz ve stromu.

Chcete-li tento server odebrat ze spravovaného seznamu, klikněte na tlačítko pod údaji o serveru.
