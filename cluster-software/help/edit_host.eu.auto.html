Kudeatutako zerbitzaria
Orrialde honetan modulu honek kudeatzen dituen software paketeen xehetasunak eta zerbitzarian instalatutako paketeen zuhaitza erakusten dira. Zuhaitza zabaldu dezakezu karpetaren ikonoetan klik eginez edo pakete baten xehetasunetan zuhaitzean dagoen estekan klik eginez.

Zerbitzaria kudeatutako zerrendatik kentzeko, egin klik zerbitzariaren xehetasunen azpian dagoen botoian.
