Програмні пакети кластерів
Цей модуль Webmin дозволяє керувати програмним забезпеченням, встановленим на декількох серверах в кластері, з одного інтерфейсу. Він схожий з модулем програмних пакетів, але з можливістю виконувати більшість функцій (наприклад, встановлення, запит та видалення пакетів) на багатьох серверах Webmin за одну операцію.

У верхній частині головної сторінки під заголовком Керовані сервери перелічені інші сервери Webmin, програмним забезпеченням яких керує цей модуль. Щоб додати сервер до цього списку, спершу слід додати його до модуля серверів Webmin, вказавши ім'я користувача та пароль для входу в Webmin на цьому сервері. Потім можна вибрати сервер зі списку поруч із кнопкою Додати сервер.

Коли сервер додано, модуль перевірить, чи використовує він ту саму систему управління пакунками, як і цей сервер (наприклад, RPM або Debian), а потім завантажить список усіх встановлених пакетів для кешування локально.

Один сервер, який завжди відображатиметься у списку доступних для додавання, - це "цей сервер", щоб ви могли керувати пакетами локально та на декількох віддалених серверах з одного інтерфейсу.

Після того, як у керованих серверах з’явиться хоча б один значок, ви можете скористатися формою встановлених пакетів для пошуку пакетів на всіх серверах або натиснути кнопку Обновити списки пакунків, щоб повторно отримати список пакетів з усіх серверів, якщо пакети додані чи видалені через інший модуль чи метод.

Внизу сторінки розміщена форма для встановлення нових пакетів на всіх керованих серверах. Ви можете встановити з файлу на цьому сервері, із завантаженого файлу або з файлу, завантаженого через HTTP або FTP з іншого веб-сайту. Встановлюючи з завантаженого файлу, ви можете вибрати, чи повинен кожен керований сервер повторно завантажувати пакунок з оригінального сайту або отримувати копію з цього сервера. Колишній варіант буде швидше, якщо сайт, на якому знаходиться пакет, знаходиться у вашій локальній мережі.
