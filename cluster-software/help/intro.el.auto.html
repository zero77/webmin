Πακέτα λογισμικού συμπλέγματος
Αυτή η λειτουργική μονάδα Webmin σας επιτρέπει να διαχειριστείτε το λογισμικό που είναι εγκατεστημένο σε πολλούς διακομιστές σε ένα σύμπλεγμα από τη μία διεπαφή. Είναι παρόμοιο με τη λειτουργική μονάδα πακέτων λογισμικού, αλλά με τη δυνατότητα εκτέλεσης των περισσότερων λειτουργιών (όπως εγκατάσταση, ερώτηση και διαγραφή πακέτων) σε πολλούς διακομιστές Webmin σε μία λειτουργία.

Το πάνω μέρος της κύριας σελίδας κάτω από την επικεφαλίδα Διαχειριζόμενοι διακομιστές παραθέτει άλλους διακομιστές Webmin των οποίων το λογισμικό διαχειρίζεται αυτή η ενότητα. Για να προσθέσετε έναν διακομιστή σε αυτήν τη λίστα, πρέπει πρώτα να τον προσθέσετε στη μονάδα διακομιστών Webmin, με καθορισμένο όνομα χρήστη και κωδικό πρόσβασης για σύνδεση στο Webmin σε αυτόν τον διακομιστή. Στη συνέχεια, μπορείτε να επιλέξετε το διακομιστή από τη λίστα δίπλα στο κουμπί Προσθήκη διακομιστή.

Όταν προστίθεται ένας διακομιστής, η λειτουργική μονάδα θα ελέγξει για να βεβαιωθεί ότι χρησιμοποιεί το ίδιο σύστημα διαχείρισης πακέτων με αυτόν τον διακομιστή (όπως RPM ή Debian) και, στη συνέχεια, κατεβάστε μια λίστα με όλα τα εγκατεστημένα πακέτα για προσωρινή αποθήκευση.

Ένας διακομιστής που θα εμφανίζεται πάντα στη λίστα αυτών που είναι διαθέσιμοι για προσθήκη είναι "αυτός ο διακομιστής", ώστε να μπορείτε να διαχειρίζεστε πακέτα τοπικά και σε πολλούς απομακρυσμένους διακομιστές από τη μία διεπαφή.

Μόλις υπάρχει τουλάχιστον ένα εικονίδιο στην ενότητα Διαχειριζόμενοι διακομιστές, μπορείτε να χρησιμοποιήσετε τη φόρμα εγκατεστημένων πακέτων για να αναζητήσετε πακέτα σε όλους τους διακομιστές ή να κάνετε κλικ στο κουμπί Ανανέωση λιστών πακέτων για να ανακτήσετε τη λίστα πακέτων από όλους τους διακομιστές εάν έχουν προστεθεί ή διαγραφούν πακέτα μέσω κάποιας άλλης ενότητας ή μεθόδου.

Στο κάτω μέρος της σελίδας υπάρχει μια φόρμα για την εγκατάσταση νέων πακέτων σε όλους τους διαχειριζόμενους διακομιστές. Μπορείτε να επιλέξετε να εγκαταστήσετε από ένα αρχείο σε αυτόν τον διακομιστή, από ένα μεταφορτωμένο αρχείο ή από ένα αρχείο που κατεβάσατε μέσω HTTP ή FTP από κάποια άλλη τοποθεσία. Κατά την εγκατάσταση από ένα ληφθέν αρχείο, μπορείτε να επιλέξετε εάν κάθε διαχειριζόμενος διακομιστής θα πρέπει να πραγματοποιήσει εκ νέου λήψη του πακέτου από τον αρχικό ιστότοπο ή να λάβει αντίγραφο από αυτόν τον διακομιστή. Η προηγούμενη επιλογή θα είναι ταχύτερη εάν ο ιστότοπος στον οποίο βρίσκεται το πακέτο βρίσκεται εντός του τοπικού σας δικτύου.
