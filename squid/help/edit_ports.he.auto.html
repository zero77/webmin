נמלים ורשתות
יציאת פרוקסי : ברירת מחדל היא 3128. ערך זה מגדיר את היציאה שרת הדיונון שלך יקשיב לבקשות http. יהיה צורך להגדיר דפדפני לקוח כדי להתחבר לפרוקסי ביציאה זו, או שתצטרך להתקין גם שקיפות.

יציאת ICP : מספר היציאה שאליה דיונון שולח ומקבל בקשות ICP אל המטמון של השכנים וממנה. ברירת המחדל היא 3130. לביטול השימוש ב- "0". עשוי להיות מחוספס עם -u בשורת הפקודה.

כתובת TCP נכנסת : ברירת מחדל היא ללא אחת. מגדיר את שקע HTTP שמקבל חיבורים מלקוחות ומטמונים אחרים.

כתובת TCP יוצאת : ברירת המחדל היא ללא. מגדיר את שקע HTTP עבור חיבורים שנעשו לשרתים מרוחקים ולמטמונים אחרים.

כתובת UDP יוצאת : ברירת המחדל היא ללא. מגדיר את שקע ה- ICP שקבל מנות ממטמונים אחרים.

כתובת UDP נכנסת : ברירת מחדל היא ללא אחת. מגדיר את השקע עבור מנות ICP שנשלחות למטמונים אחרים.

מאגר קבלת TCP : גודל המאגר TCP. ברירות המחדל למערכת ההפעלה מקבלות מאגר.
