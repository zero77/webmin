Ports und Netzwerk
Proxy-Port : Der Standardwert ist 3128. Dieser Wert legt fest, welchen Port Ihr Squid-Server auf http-Anforderungen überwacht. Client-Browser müssen so konfiguriert sein, dass sie eine Verbindung zu einem Proxy an diesem Port herstellen, oder es muss auch Transparenz eingerichtet werden.

ICP-Port : Die Portnummer, an die Squid ICP-Anforderungen an und von benachbarten Caches sendet und empfängt. Die Standardeinstellung ist 3130. Zum Deaktivieren verwenden Sie "0". Kann in der Befehlszeile mit -u überschrieben werden.

Eingehende TCP-Adresse : Standardmäßig keine. Definiert den HTTP-Socket, der Verbindungen von Clients und anderen Caches akzeptiert.

Ausgehende TCP-Adresse : Standardmäßig keine. Definiert den HTTP-Socket für Verbindungen zu Remoteservern und anderen Caches.

Ausgehende UDP-Adresse : Standardmäßig keine. Definiert den ICP-Socket, der Pakete von anderen Caches empfängt.

Eingehende UDP-Adresse : Standardmäßig keine. Definiert den Socket für ICP-Pakete, die an andere Caches gesendet werden.

TCP-Empfangspuffer : Größe des TCP-Puffers. Standardmäßig wird der OS-Empfangspuffer verwendet.
