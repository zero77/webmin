Aukera administratiboak
Exekutatu Unix erabiltzaile gisa : lehenetsia 'inork' edo 'txipiroiak' sistemaren menpe dago. Cache erro gisa exekutatzen bada, bere ID efektiboa / erreala balio honetara aldatuko da.

Exekutatu Unix talde gisa : lehenetsia ' talde taldea ' edo 'txipiroia' sistemaren menpe dago. Cache erro gisa exekutatzen bada, bere talde ID eraginkorra / erreala balio honetara aldatuko da.

Proxy autentifikazioen eremua : proxy-autentifikazioari bezeroari jakinarazi behar zaion erreinuaren izena zehazten du (erabiltzaileak eta pasahitza galdetzean erabiltzaileak ikusiko duen testuaren zati bat).

Cache kudeatzailearen helbide elektronikoa : sistema "web-maisua" edo "root" menpekoa da. Cachea hiltzen bada tokiko cache kudeatzailearen tokiko helbide elektronikoa.

Ostalari izena ikusgai : ostalari izen berezi bat aurkeztu nahi baduzu, eta abarretan, zehaztu hau. Bestela, gethostname () itzuleraren balioa erabiliko da. Kluster batean cache anitz badituzu eta IP-bidaltzeari buruzko akatsak lortzen badituzu, ezarpen honekin izenak banakoak ezarri behar dituzu.

Ostalari izen bakarra : ikusgai ostalari izen bera duten makina bat baino gehiago nahi badituzu, makina bakoitzari ostalari izen desberdina eman behar diozu, birbidaltzeko begiztak antzeman ahal izateko.

Cache iragartzeko ostalaria : Ostalariaren izena non iragarkiak bidaliko diren. Aurki itzazu The Squid ohiko ohiko galderei buruzko hierarkiei buruz. Lehenetsita dago 'tracker.ircache.net' mezuan iragarkia gaitzen denean iragarkia gaitzen denean.

Cache iragartzeko portua : Ohar osteko tokia bidaliko da. Iragarkia gaituta dagoenean 3131 lehenetsi da.

Deialdia egiteko epea : Iragarkien maiztasuna. Gaitu oharrak cache-ko helmugan hau 'egun 1' ezarrita. Lehenespena '0' da iragarkia desaktibatzen duena.
