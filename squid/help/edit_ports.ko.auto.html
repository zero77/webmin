포트 및 네트워킹
프록시 포트 : 기본값은 3128입니다.이 값은 Squid 서버가 http 요청을 수신 할 포트를 설정합니다. 이 포트에서 프록시에 연결하도록 클라이언트 브라우저를 구성하거나 투명도를 설정해야합니다.

ICP 포트 : Squid가 인접 캐시와 ICP 요청을주고받는 포트 번호입니다. 기본값은 3130입니다. 비활성화하려면 "0"을 사용하십시오. 명령 행에서 -u로 대체 될 수 있습니다.

들어오는 TCP 주소 : 기본값은 none입니다. 클라이언트 및 기타 캐시의 연결을 허용하는 HTTP 소켓을 정의합니다.

발신 TCP 주소 : 기본값은 none입니다. 원격 서버 및 기타 캐시에 대한 연결을위한 HTTP 소켓을 정의합니다.

발신 UDP 주소 : 기본값은 none입니다. 다른 캐시에서 패킷을 수신하는 ICP 소켓을 정의합니다.

수신 UDP 주소 : 기본값은 none입니다. 다른 캐시로 전송되는 ICP 패킷의 소켓을 정의합니다.

TCP 수신 버퍼 : TCP 버퍼의 크기입니다. OS 수신 버퍼가 기본값입니다.
