Порты и сети
Порт прокси : по умолчанию 3128. Это значение определяет, какой порт ваш сервер Squid будет прослушивать для запросов http. Клиентские браузеры должны быть настроены для подключения к прокси-серверу на этом порту, или прозрачность также должна быть настроена.

Порт ICP : номер порта, на который Squid отправляет и получает запросы ICP от соседних кешей. По умолчанию 3130. Для отключения используйте «0». Может быть переопределено с -u в командной строке.

Входящий TCP-адрес : по умолчанию нет. Определяет сокет HTTP, который принимает соединения от клиентов и других кэшей.

Исходящий TCP-адрес : по умолчанию нет. Определяет сокет HTTP для подключений к удаленным серверам и другим кэшам.

Исходящий UDP-адрес : по умолчанию нет. Определяет сокет ICP, получающий пакеты от других кэшей.

Входящий адрес UDP : по умолчанию нет. Определяет сокет для пакетов ICP, отправляемых в другие кэши.

TCP приемный буфер : размер буфера TCP. По умолчанию используется буфер приема ОС.
