Задержка бассейнов

Задержка пулов обеспечивает средство ограничения использования полосы пропускания клиентами. С пулами задержки клиенты могут быть ограничены на индивидуальном уровне или в группах. ACL используются для применения ограничений полосы пропускания к пользователям.

На этой странице отображаются все существующие пулы и их ограничения. Также есть раздел для глобальных опций, который на данный момент имеет только одну опцию.

Начальный уровень ковша
Из-за того, как работают пулы задержек, новому клиенту потенциально может быть разрешено использовать больше, чем его справедливая доля пропускной способности, потому что его новый сегмент будет пустым. В перегруженной сети это будет означать, что другие соединения от клиентов с почти полными сегментами будут лишены пропускной способности в течение некоторого периода времени. Этот параметр по умолчанию равен 50% и, вероятно, может быть оставлен в покое, если ваши максимальные значения не очень велики (максимальное значение при настройке пределов фактически применяется к размеру корзины для пула).
