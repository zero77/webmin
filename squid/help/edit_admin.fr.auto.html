Options administratives
Exécuter en tant qu'utilisateur Unix : la valeur par défaut est «personne» ou «calmar» dépendant du système. Si le cache est exécuté en tant que root, il changera son ID effectif / réel en cette valeur.

Exécuter en tant que groupe Unix : la valeur par défaut est 'nogroup' ou 'squid' selon le système. Si le cache est exécuté en tant que root, il changera son ID de groupe effectif / réel en cette valeur.

Domaine d'authentification proxy : spécifie le nom de domaine qui doit être signalé au client pour l'authentification proxy (partie du texte que l'utilisateur verra lorsqu'il sera invité à saisir son nom d'utilisateur et son mot de passe).

Adresse e-mail du gestionnaire de cache : la valeur par défaut est «webmaster» ou «root» selon le système. Adresse e-mail du gestionnaire de cache local qui recevra le courrier en cas de disparition du cache.

Nom d'hôte visible : si vous souhaitez présenter un nom d'hôte spécial dans les messages d'erreur, etc., définissez-le. Sinon, la valeur de retour de gethostname () sera utilisée. Si vous avez plusieurs caches dans un cluster et obtenez des erreurs sur le transfert IP, vous devez les définir pour avoir des noms individuels avec ce paramètre.

Nom d'hôte unique : si vous souhaitez avoir plusieurs machines avec le même nom d'hôte visible, vous devez attribuer à chaque machine un nom d'hôte unique différent afin que les boucles de transfert puissent être détectées.

Cache annoncent l'hôte : nom d'hôte où les messages d'annonce seront envoyés. Pour en savoir plus sur les hiérarchies de cache, consultez la FAQ Squid . Par défaut, «tracker.ircache.net» lorsque l'annonce est activée en définissant une période d'annonce.

Port d'annonce du cache : port hôte sur lequel les messages d'annonce seront envoyés. La valeur par défaut est 3131 lorsque l'annonce est activée.

Période d'annonce : Fréquence des annonces. Activez les annonces dans une hiérarchie de cache en définissant ce paramètre sur «1 jour». La valeur par défaut est «0», ce qui désactive l'annonce.
