โปรแกรมผู้ช่วย
ความกว้างคอลัมน์ FTP : ค่าเริ่มต้นคือ 32 ตั้งค่าความกว้างของรายการ ftp ควรตั้งค่าให้พอดีกับความกว้างของเบราว์เซอร์มาตรฐาน การตั้งค่านี้เล็กเกินไปอาจตัดชื่อไฟล์ยาวเมื่อเรียกดูไซต์ ftp

Anon FTP login : ค่าเริ่มต้นคือ 'Squid @' หากคุณต้องการให้รหัสผ่านการเข้าสู่ระบบแบบไม่ระบุชื่อมีข้อมูลมากขึ้น (และเปิดใช้งานการใช้เซิร์ฟเวอร์ picky ftp) ให้ตั้งค่านี้เป็นสิ่งที่ resonable สำหรับโดเมนของคุณเช่น wwwuser@somewhere.net

สาเหตุที่ทำให้เกิดปัญหานี้โดยค่าเริ่มต้นคือคำขอสามารถทำในนามของผู้ใช้ในโดเมนใด ๆ ขึ้นอยู่กับวิธีการใช้แคช เซิร์ฟเวอร์ ftp บางแห่งยังตรวจสอบว่าที่อยู่อีเมลนั้นถูกต้อง (เช่น perl.com)

โปรแกรม Squid DNS : ค่าเริ่มต้นคือ dnsserver ระบุตำแหน่งของไฟล์เรียกทำงานสำหรับกระบวนการ dnslookup ค่าเริ่มต้นคือเนมเซิร์ฟเวอร์แคชที่ไม่มีการบล็อกที่ติดตั้งกับ Squid ตามค่าเริ่มต้น

จำนวนโปรแกรม DNS : ค่าเริ่มต้นคือ 5 จำนวนกระบวนการที่ให้บริการการค้นหาชื่อ DNS สำหรับแคชที่โหลดจำนวนมากบนเซิร์ฟเวอร์ขนาดใหญ่คุณควรเพิ่มค่านี้เป็นอย่างน้อย 10 ค่าสูงสุดคือ 32

ผนวกโดเมนเข้ากับคำขอ : ค่าเริ่มต้นปิดอยู่ โดยปกติแล้ว 'dnsserver' จะปิดใช้งานตัวเลือกตัวแก้ไข RES_DEFNAMES (ดู res_init (3)) สิ่งนี้จะป้องกันแคชในลำดับชั้นจากการตีความชื่อโฮสต์เดียวขององค์ประกอบภายในเครื่อง ในการอนุญาตให้ dnsserver จัดการกับชื่อองค์ประกอบเดียวให้เปิดใช้งานตัวเลือกนี้

ที่อยู่เซิร์ฟเวอร์ DNS : ค่าเริ่มต้นคือการใช้ไฟล์ resolv.conf ใช้สิ่งนี้หากคุณต้องการระบุรายการเซิร์ฟเวอร์ชื่อ DNS (ที่อยู่ IP) เพื่อใช้แทนที่ระบุในไฟล์ /etc/resolv.conf ของคุณ

โปรแกรมการล้างแคช : ค่าเริ่มต้นคือ 'unlinkd' ระบุตำแหน่งของไฟล์เรียกทำงานสำหรับกระบวนการลบไฟล์ สิ่งนี้อาจไม่จำเป็นถ้าคุณใช้ async-io เนื่องจากเธรดถูกจัดการโดยเธรดในบางเวอร์ชัน

โปรแกรม Squid ping : ค่าเริ่มต้นเป็น 'pinger' ระบุตำแหน่งของไฟล์สั่งการสำหรับกระบวนการส่งสัญญาณ สิ่งนี้มีประโยชน์เฉพาะเมื่อคุณกำหนดค่า Squid (ระหว่างการทำตาม) ด้วยตัวเลือก '--enable-icmp'

โปรแกรมเปลี่ยนเส้นทางที่กำหนดเอง : เริ่มต้นที่ไม่มี ระบุตำแหน่งของไฟล์สั่งการสำหรับตัวเปลี่ยนเส้นทาง URL เนื่องจากพวกเขาสามารถทำงานได้เกือบทุกฟังก์ชั่นจึงไม่มีการรวมไว้ ดู Release-Notes สำหรับข้อมูลเกี่ยวกับวิธีการเขียน

จำนวนโปรแกรมเปลี่ยนเส้นทาง : ค่าเริ่มต้นถึง 5 หากกำหนดค่าตัวเปลี่ยนเส้นทาง จำนวนของตัวเปลี่ยนเส้นทางที่จะประมวลผล หากคุณเริ่ม Squid น้อยเกินไปจะต้องรอให้พวกเขาประมวลผลงานค้างของ URL ทำให้ช้าลง หากคุณเริ่มต้นมากเกินไปพวกเขาจะใช้ RAM และทรัพยากรระบบอื่น ๆ
