Porty a vytváranie sietí
Proxy port : Predvolené je 3128. Táto hodnota nastavuje, na ktorých portoch bude váš server Squid počúvať pre požiadavky http. Prehliadače klientov budú musieť byť nakonfigurované tak, aby sa na tomto porte pripájali k serveru proxy, alebo bude potrebné nastaviť aj priehľadnosť.

ICP port : Číslo portu, na ktorý Squid posiela a prijíma požiadavky ICP do a zo susedných vyrovnávacích pamätí. Predvolená hodnota je 3130. Na deaktiváciu použite „0“. Môže byť potlačený príkazom -u na príkazovom riadku.

Prichádzajúca adresa TCP : Predvolené žiadne. Definuje soket HTTP, ktorý prijíma pripojenia od klientov a iných vyrovnávacích pamätí.

Odchádzajúca TCP adresa : Predvolené žiadne. Definuje soket HTTP pre pripojenia k vzdialeným serverom a iným cache.

Odchádzajúca adresa UDP : Predvolené žiadne. Definuje pakety prijímajúce soket ICP z iných vyrovnávacích pamätí.

Prichádzajúca adresa UDP : Predvolené žiadne. Definuje soket pre pakety ICP zasielané do iných vyrovnávacích pamätí.

Príjem vyrovnávacej pamäte TCP : Veľkosť vyrovnávacej pamäte TCP. Predvolené hodnoty OS dostávajú vyrovnávaciu pamäť.
