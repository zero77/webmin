Парты і сеткі
Порт проксі : па змаўчанні 3128. Гэта значэнне ўсталёўвае, які порт будзе праслухоўваць ваш сервер Squid для запытаў http. Для падлучэння да проксі на гэтым порце трэба наладзіць браўзэры кліентаў, альбо празрыстасць таксама павінна быць наладжана.

Порт ICP : нумар порта, куды Squid адпраўляе і прымае запыты ICP у кэш суседа і з яго. Па змаўчанні - 3130. Каб адключыць выкарыстанне "0". У камандным радку можа быць пераменены з -u.

Уваходны адрас TCP : па змаўчанні няма. Вызначае сокет HTTP, які прымае злучэнні ад кліентаў і іншыя кэшы.

Выходны адрас TCP : Па змаўчанні няма. Вызначае гняздо HTTP для злучэння, выкананага з аддаленымі серверамі і іншымі кэшамі.

Выходны адрас UDP : Па змаўчанні няма. Вызначае прыём пакетаў ICP-сокетаў з іншых кэшаў.

Адрас, які ўваходзіць UDP : па змаўчанні няма. Вызначае разетку для пакетаў ICP, якія адпраўляюцца ў іншыя кэшы.

TCP буфер прыёму : Памер буфера TCP. Па змаўчанні АС атрымлівае буфер.
