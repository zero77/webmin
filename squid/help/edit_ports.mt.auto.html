Portijiet u Netwerking
Port Proxy : Defaults għal 3128. Dan il-valur jistabbilixxi liema port is-server tal-Klamari tiegħek se jisma 'fuq it-talbiet http. Il-brawżers tal-klijenti se jkollhom jiġu kkonfigurati biex jaqbdu ma 'prokura f'dan il-port, jew it-trasparenza se tkun meħtieġa wkoll.

Port ICP : In-numru tal-port fejn il-Klamar jibgħat u jirċievi talbiet mill-ICP minn u mill-caches tal-ġar. In-nuqqas huwa 3130. Biex tiddiżattiva l-użu "0". Jista 'jiġi mwarrab b'u fuq il-linja tal-kmand.

Indirizz TCP li jmiss : Default għal xejn. Jiddefinixxi s-socket HTTP li jaċċetta konnessjonijiet mill-klijenti u cache oħra.

Indirizz TCP ħiereġ: Default għal xejn. Jiddefinixxi s-socket HTTP għal konnessjonijiet magħmula għal servers remoti u cache oħra.

Indirizz UDP ħiereġ: Nuqqasijiet għal xejn. Jiddefinixxi s-socket ICP li jirċievi pakketti minn cache oħra.

Indirizz UDP li jmiss : Default għal xejn. Jiddefinixxi s-sokit għall-pakketti ICP li jintbagħtu lil cache oħra.

Buffer li tirċievi TCP : Daqs tal-buffer TCP. In-nuqqasijiet fl-OS jirċievu buffer.
