Hilfsprogramme
FTP-Spaltenbreite : Die Standardeinstellung ist 32. Legt die Breite der FTP-Listen fest. Dies sollte so eingestellt werden, dass es in die Breite eines Standardbrowsers passt. Wenn Sie diese Einstellung zu klein einstellen, werden beim Durchsuchen von FTP-Sites möglicherweise lange Dateinamen abgeschnitten.

Anon FTP-Login : Standard ist 'Squid @'. Wenn Sie möchten, dass das anonyme Anmeldekennwort informativer ist (und die Verwendung wählerischer FTP-Server ermöglicht), stellen Sie dies auf einen für Ihre Domain vernünftigen Wert ein, z. B. wwwuser@somewhere.net.

Der Grund, warum dies standardmäßig domänenlos ist, besteht darin, dass die Anforderung im Namen eines Benutzers in einer beliebigen Domäne erfolgen kann, je nachdem, wie der Cache verwendet wird. Einige FTP-Server überprüfen auch, ob die E-Mail-Adresse gültig ist (z. B. perl.com).

Squid DNS-Programm : Standard ist dnsserver. Geben Sie den Speicherort der ausführbaren Datei für den DNS-Lookup-Prozess an. Der Standardwert ist ein nicht blockierender Caching-Nameserver, der standardmäßig mit Squid installiert wird.

Anzahl der DNS-Programme : Die Standardeinstellung ist 5. Die Anzahl der Prozesse, die für die Suche nach DNS-Namen erstellt wurden. Bei stark ausgelasteten Caches auf großen Servern sollten Sie diesen Wert wahrscheinlich auf mindestens 10 erhöhen. Das Maximum beträgt 32.

Domäne an Anforderungen anhängen: Standard ist deaktiviert. Normalerweise deaktiviert der 'dnsserver' die Resolver-Option RES_DEFNAMES (siehe res_init (3)). Dies verhindert, dass Caches in einer Hierarchie Einzelkomponenten-Hostnamen lokal interpretieren. Aktivieren Sie diese Option, damit dnsserver Einzelkomponentennamen verarbeiten kann.

DNS-Serveradressen : Standardmäßig wird die Datei resolv.conf verwendet. Verwenden Sie diese Option, wenn Sie eine Liste der zu verwendenden DNS-Nameserver (IP-Adressen) anstelle der in Ihrer Datei /etc/resolv.conf angegebenen angeben möchten.

Cache -Löschprogramm: Standard ist 'unlinkd'. Geben Sie den Speicherort der ausführbaren Datei für den Löschvorgang an. Dies ist möglicherweise nicht erforderlich, wenn Sie async-io verwenden, da es in einigen Versionen von einem Thread verarbeitet wird.

Squid-Ping-Programm : Der Standardwert ist "Pinger". Geben Sie den Speicherort der ausführbaren Datei für den Pinger-Prozess an. Dies ist nur nützlich, wenn Sie Squid (während der Kompatibilität) mit der Option '--enable-icmp' konfiguriert haben.

Benutzerdefiniertes Umleitungsprogramm : Standardmäßig keine. Geben Sie den Speicherort der ausführbaren Datei für den URL-Redirector an. Da sie fast jede Funktion ausführen können, ist keine enthalten. Informationen zum Schreiben finden Sie in den Versionshinweisen.

Anzahl der Umleitungsprogramme : Der Standardwert ist 5, wenn ein Redirector konfiguriert ist. Die Anzahl der Redirector-Prozesse, die erzeugt werden sollen. Wenn Sie zu wenig Squid starten, müssen Sie darauf warten, dass sie einen Rückstand an URLs verarbeiten, wodurch dieser verlangsamt wird. Wenn Sie zu viele starten, verwenden sie RAM und andere Systemressourcen.
