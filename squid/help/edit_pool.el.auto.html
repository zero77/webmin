Επεξεργασία ομάδας καθυστέρησης

Αριθμός πισίνας
Αυτός είναι ένας αριθμός που ξεκινά από το 1, ο οποίος επιλέγεται από εσάς από το Webmin. Απλώς επισημαίνει την πισίνα έτσι ώστε περισσότερες από μία ομάδες μπορούν να διαμορφωθούν και να αντιμετωπιστούν διαφορετικά από το Squid.

Μάθημα πισίνας
Αυτό καθορίζει τον τύπο κατηγορίας της πισίνας. Κάθε ομάδα καθυστέρησης μπορεί να επιλέξει μια τάξη από τις κατηγορίες 1, 2 ή 3. Οι τάξεις έχουν τις ακόλουθες έννοιες: "1-Συγκεντρωτικό", "2-Συγκεντρωτικό και Ατομικό", "3-Συγκεντρωτικό, Ατομικό και Δίκτυο". Η πιο απλή, κατηγορία 1, καθορίζει ένα εύρος ζώνης που δεν μπορούν να ξεπεράσουν όλοι οι πελάτες στους οποίους ανήκει αυτή η ομάδα. Η Κλάση 2 καθορίζει τόσο ένα συνολικό εύρος ζώνης όσο και ένα εύρος ζώνης που μπορεί να χρησιμοποιήσει κάθε άτομο, δημιουργώντας μια κληρονομιά ορίων όπου όλα τα άτομα που συνδυάζονται πρέπει να ταιριάζουν στο σύνολο ενώ κάθε άτομο περιορίζεται από το ατομικό όριο. Τέλος, η Κλάση 3 καθορίζει ένα συνολικό και μεμονωμένο εύρος ζώνης εκτός από ένα εύρος ζώνης δικτύου που ισχύει για ένα ολόκληρο υποδίκτυο. Η κλάση 3 είναι συνήθως χρήσιμη μόνο σε πολύ μεγάλα δίκτυα που διαθέτουν περισσότερα από ένα δίκτυα υποδικτύου.

Τα όρια μπορούν να καθοριστούν σε οποιαδήποτε μονάδα προτιμάται, αν και θα αποθηκευτούν στο αρχείο διαμόρφωσης Squid σε byte. Το πρώτο πεδίο είναι η μέση απόδοση που θα κατανέμεται στην ομάδα, ενώ το δεύτερο είναι το μέγεθος του κάδου που χρησιμοποιείται, το οποίο αντιστοιχεί περίπου στο μέγιστο εύρος ζώνης.

Αφού δημιουργήσετε μια ομάδα για πρώτη φορά, θα εμφανιστεί μια νέα ενότητα στο κάτω μέρος της σελίδας, επιτρέποντας την επιλογή σε ποια ACL θα εφαρμοστεί η ομάδα. Τα ACL δημιουργούνται στη σελίδα ελέγχου πρόσβασης και μπορεί να χρησιμοποιηθεί οποιοδήποτε ACL για να επιλέξετε σε ποιες συνδέσεις θα εφαρμοστεί μια ομάδα καθυστέρησης. Επιτρέποντας ένα ACL θα εφαρμόσει το όριο καθυστέρησης στις αντίστοιχες συνδέσεις, ενώ η άρνηση ενός ACL θα προκαλέσει τις αντίστοιχες συνδέσεις να παρακάμψουν την ομάδα καθυστέρησης. Επομένως, εκτός εάν όλες οι συνδέσεις θα έχουν την ίδια ομάδα καθυστέρησης που εφαρμόζεται σε αυτές, θα θελήσετε να προσθέσετε δύο ACL, η πρώτη θα είναι η ACL στην οποία θέλετε να εφαρμόσετε την ομάδα καθυστέρησης και η δεύτερη θα είναι όλες οι ACL έτσι ότι μόνο οι επιλεγμένοι πελάτες σας θα περιοριστούν.

Μην ξεχάσετε να εφαρμόσετε αλλαγές μετά τη δημιουργία μιας νέας ομάδας και την προσθήκη ACL για αντιστοίχιση.
