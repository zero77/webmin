Askotariko aukerak
Hasierako DNS proben helbideak : Squid-ek DNS zerbitzariek kontsultatuko dute DNS bilaketak behar bezala funtzionatzen duten probatzeko.

SIGUSR1 erregistro fitxategiaren biraketak : 10 edo 0 lehenetsiak, sistemaren menpe. 'Txipiroiak -k biratu' idazten duzunean egin beharreko logfileen biraketa kopurua zehazten du. Lehenetsia 10 da, hau da 0 luzapenarekin biratuko da. Logfile_rotate 0 -ra ezarrita, biraketa desgaituko da, baina fitxategiak itxita daude eta berriro irekiko dira. Horri esker, bihurketa-seinalea bidali baino lehen fitxategi-izenak berrizendatu ahal izango dituzu.

Kontuan izan, 'squid -k biratu' komandoak normalean USR1 seinale bat bidaltzen du exekutatzen den txipiroiaren prozesura. Zenbait egoeratan (adibidez Linux-en Async I / O-rekin), USR1 beste helburu batzuetarako erabiltzen da, beraz, -k biratzeak beste seinale bat erabiltzen du. Hobe da 'txipiroi-k biratu' erabiltzeko ohitura izatea 'hiltzea -USR1 erabili beharrean '.

Akats mezuen testua : errore mezuetan sartzeko HTML testua. Bihur ezazu hau "mailto" URL zure administratzailearen helbidera edo, agian, besterik gabe, zure erakundeen web orrialderako esteka.

Hau zure errore mezuetan sartzeko, akats txantiloiaren fitxategiak berridatzi behar dituzu ("akatsak" direktorioan aurkitzen dira). 'Err_html_text' lerroa agertzea nahi duzun tokian, sartu% L etiketa akats txantiloiaren fitxategian.

Bezero bakoitzeko estatistikak : bezero bakoitzeko estatistikak biltzea desgaitu nahi baduzu, hemen desaktibatu client_db.

X-Forwarded-For goiburua : Lehenespenezkoa 'aktibatuta dago'. Ezarrita badago, Squid-ek zure sistemaren IP helbidea edo izena HTTP eskaeretan sartuko du aurrera. Lehenespenez, HTTP-n aurrera begira eskatzen du. Lehenespenez, itxura hau da:

X-Birbidalitako-For: 192.1.2.3

Hau desgaitzen baduzu, honela agertuko da:

X-Forwarded-For: ezezaguna

Log ICP kontsultak : lehenetsia 'aktibatuta dago'. Ezarrita badago, ICP kontsultak access.log-era erregistratu dira. Baliteke hau desaktibatzea zure ICP karga OSO altua bada gauzak azkartzeko edo erregistroaren analisia errazteko.

Gutxieneko zuzeneko saltokia : Lehenetsia 4. ICMP pinging-ak erabiltzen badituzu, egin zuzeneko bilaketak hontzetik urruntzen ez diren guneetarako.

Gorde memoria etorkizuneko erabilerarako : lehenetsia "bai" da. Ezarrita badago, Squid-ek erabilitako memoria (baina erabili gabeko) memoria gordetzen du etorkizunean erabiltzeko. Memoria zure sisteman premium bada eta zure malloc liburutegiak Squid errutinak gainditzen baditu, ezgaitu hau.

Gorde beharreko memoria kopurua : lehenetsia "Mugagabea" da. Zero ez den balioa ezarrita badago, Squid-ek memoria multzoetan gordetako (baina erabili gabeko) memoria zehaztuta mantenduko du gehienez. Muga hori gainditzen duten doako () eskaera guztiak zure malloc liburutegiak kudeatuko ditu. Txipiroiak ez du memoriarik aurrez esleitzen, besterik gabe, libre geratuko liratekeen objektuak seguru gordetzen ditu () d. Horrela, segurua da memory_pools_limit balio nahiko altu batean ezartzea, nahiz eta zure konfigurazioak memoria gutxiago erabiliko duen.

Ezarrita (lehenetsita) edo zero ez bada, Squid-ek ahal duen memoria guztia gordeko du. Hau da, ez da mugarik gordeko segurtasunerako erabilitako memoria guztian.

Pasatu beharreko goiburuak : lehenetsitakoak "Goiburu guztiak" dira. Aukera honek 'http_anonymizer' aukera zaharra ordezkatzen du askoz konfiguragarriago den zerbaitengatik. Orain zehaztu dezakezu goiburuak zein baimendu behar diren edo irteerako eskaeretatik kenduko diren.

Aukera hau erabiltzeko bi metodo daude. Baimendu ditzakezu goiburu zehatzak (horrela beste guztiak ukatuz) edo goiburu zehatzak ukatu (beraz, gainerako guztiak baimendu).

Erabiltzaile agente faltsua : lehenetsia 'bat ez da'. Erabiltzaile-agente goiburua goiburuekin hautatzen baduzu aukera pasatzeko , zenbait zerbitzariri zure eskaerari uko egin diezaioke. Erabili hau gezurrezko bat lortzeko.

WCCP Router helbidea : Erabili aukera hau Squiderako WCCP `` etxeko '' bideratzailea definitzeko.

WCCP sarrerako helbidea : erabili aukera hau WCCP mezuak interfaze bakarrean jaso nahi badituzu. Ez ezazu aukera hau erabili zenbat interfaze dituzun edo interfaze bakarra duzula badakizu.

WCCP irteerako helbidea : erabili aukera hau WCCP mezuak interfaze bakarrean bidali nahi badituzu. Ez ezazu aukera hau erabili zenbat interfaze dituzun edo interfaze bakarra duzula badakizu.
