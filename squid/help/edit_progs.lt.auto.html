Pagalbininkų programos
FTP stulpelio plotis : Numatytasis yra 32. Nustatomas FTP sąrašų plotis. Tai turėtų būti nustatyta taip, kad atitiktų standartinės naršyklės plotį. Jei nustatysite tai per mažą, naršydami ftp svetainėse galite iškirpti ilgus failų pavadinimus.

Anon FTP prisijungimas : numatytoji reikšmė yra „Squid @“. Jei norite, kad anoniminis prisijungimo slaptažodis būtų informatyvesnis (ir įgalintų naudoti smulkmeniškus FTP serverius), nustatykite tai, kad jūsų domenas būtų tinkamas, pvz., Wwwuser@somewhere.net.

Pagal nutylėjimą tai nėra domeno priežastis, kad užklausą galima pateikti bet kurio domeno vartotojo vardu, atsižvelgiant į tai, kaip naudojama talpykla. Kai kurie ftp serveriai taip pat patvirtina, kad el. Pašto adresas galioja (pvz., Perl.com).

Kalmarų DNS programa : Numatytasis yra „dnsserver“. Nurodykite vykdomosios programos dnslookup proceso vietą. Numatytoji yra neužblokuojanti talpyklos vardų serveris, kuris pagal numatytuosius nustatymus yra įdiegtas su „Squid“.

DNS programų skaičius : Numatytasis yra 5. Procesų, kurių buvo imtasi teikiant DNS vardų paiešką, skaičius. Tikriausiai turėtumėte padidinti šią vertę iki mažiausiai 10. Jei įrenginys yra labai įkeltas į didelius serverius, turėtumėte padidinti jį bent iki 10. Didžiausia yra 32.

Pridėti domeną prie užklausų : Numatytasis yra išjungtas. Paprastai „dnsserver“ išjungia „RES_DEFNAMES“ sprendimo parinktį (žr. „Res_init“ (3)). Tai neleidžia hierarchijos talpykloms aiškinti lokaliai vienkomponenčių pagrindinių kompiuterių pavadinimų. Norėdami leisti „dnsserver“ tvarkyti vieno komponento pavadinimus, įgalinkite šią parinktį.

DNS serverio adresai : Numatytasis failas yra „resolv.conf“. Naudokite tai, jei norite nurodyti naudotinų DNS vardų serverių (IP adresų) sąrašą, o ne tuos, kurie pateikiami jūsų /etc/resolv.conf faile.

Talpyklos išvalymo programa : numatytoji reikšmė yra „atsieta“. Nurodykite vykdomojo failo, kurio failai ištrinami, vietą. Tai gali būti nereikalinga, jei naudojate „async-io“, nes kai kuriose versijose jį tvarko sriegis.

Kalmarų ping programa : pagal nutylėjimą „pinger“. Nurodykite vykdymo vietą „pinger“ procesui. Tai naudinga tik tuo atveju, jei „Squid“ (suderinimo metu) sukonfigūravote pasirinkdami „--enable-icmp“.

Individualizuota peradresavimo programa : numatytosios nėra. Nurodykite URL peradresavimo programos vykdomosios programos vietą. Kadangi jie gali atlikti beveik bet kokią funkciją, jų nėra. Norėdami sužinoti, kaip juos rašyti, skaitykite „Release-Notes“.

Peradresavimo programų skaičius : Numatyta iki 5, jei sukonfigūruotas peradrektorius. Peradresavimo procesų, kurių metu nerimaujama, skaičius. Jei paleisite per mažai „Kalmarų“, turės palaukti, kol jie apdoros URL atsilikimą, sulėtindami jį. Jei paleisite per daug, jie naudos RAM ir kitus sistemos išteklius.
