Κωδικοί πρόσβασης Cache Manager
Από προεπιλογή, η διεπαφή ιστού cache manager δεν απαιτεί έλεγχο ταυτότητας για πρόσβαση σε καμία από τις λειτουργίες της. Αυτή η σελίδα σάς επιτρέπει να ορίσετε έναν ή περισσότερους κωδικούς πρόσβασης που ισχύουν για ορισμένες ή όλες τις ενέργειες στη διαχείριση προσωρινής μνήμης.

Για να καθορίσετε έναν μόνο κωδικό πρόσβασης για όλες τις ενέργειες, απλώς επιλέξτε Κωδικοί πρόσβασης και ενέργειες που καθορίζονται παρακάτω , στη συνέχεια στη στήλη Κωδικός πρόσβασης επιλέξτε Ορισμός και εισαγάγετε τον κωδικό πρόσβασής σας στο παρακείμενο πλαίσιο. Στη συνέχεια, επιλέξτε το πλαίσιο Χρήση αυτού του κωδικού πρόσβασης για όλες τις ενέργειες και κάντε κλικ στην επιλογή Αποθήκευση .
