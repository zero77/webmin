Editar grupo de retardo

Número de piscina
Este es un número que comienza en 1, que Webmin elige para usted. Simplemente etiqueta el grupo para que Squid pueda configurar y tratar de forma diferente más de un grupo.

Clase de piscina
Esto define el tipo de clase del grupo. Cada grupo de retraso puede tener una clase seleccionada de las clases 1, 2 o 3. Las clases tienen los siguientes significados: "1-Agregado", "2-Agregado e Individual", "3-Agregado, Individual y Red". El más simple, clase 1, especifica un ancho de banda que todos los clientes a los que pertenece este grupo no pueden superar. La clase 2 especifica un ancho de banda agregado y un ancho de banda que cada individuo puede usar, creando una jerarquía de límites donde todos los individuos combinados deben caber en el agregado mientras cada individuo está limitado por el límite individual. Finalmente, la Clase 3 especifica un ancho de banda agregado e individual además de un ancho de banda de red que se aplica a una subred completa. La clase 3 generalmente solo es útil en redes muy grandes que tienen más de una subred de red.

Los límites se pueden especificar en cualquier unidad preferida, aunque se guardará en el archivo de configuración de Squid en bytes. El primer campo es el rendimiento promedio que se asignará a la agrupación, mientras que el segundo es el tamaño del depósito utilizado, que corresponde aproximadamente a un ancho de banda máximo de ruptura.

Después de crear un grupo por primera vez, aparecerá una nueva sección en la parte inferior de la página, que permite seleccionar a qué ACL se aplicará el grupo. Las ACL se crean en la página Control de acceso , y cualquier ACL se puede usar para seleccionar a qué conexiones se aplicará un grupo de retraso. Al permitir una ACL, se aplicará el grupo de retraso a las conexiones coincidentes, mientras que si se niega una ACL, las conexiones coincidentes evitarán el grupo de retraso. Por lo tanto, a menos que todas las conexiones tendrán el mismo retardo de la piscina aplicada a ellos, usted desee agregar dos ACL, el primero será el ACL desea aplicar la piscina demora a, y el segundo será el todo de manera ACL que solo sus clientes seleccionados estarán restringidos.

No olvide aplicar los cambios después de crear un nuevo grupo y agregar ACL para que coincidan.
