Έλεγχος πρόσβασης κεφαλίδας
Αυτή η σελίδα μπορεί να χρησιμοποιηθεί για τον περιορισμό των κεφαλίδων HTTP που θα μεταφέρει το Squid από πελάτες σε διακομιστές. Μπορούν να δημιουργηθούν διαφορετικοί περιορισμοί για διαφορετικούς πελάτες ή διακομιστές επιλέγοντας ACL για τους οποίους ισχύει κάθε κανόνας ελέγχου πρόσβασης.
