Uso de memória
Limite de uso de memória : o padrão é 8 MB.

NOTA: Este parâmetro não especifica o tamanho máximo do processo. Ele coloca um limite em um aspecto do uso da memória do Squid. O Squid também usa memória para outras coisas. Seu processo provavelmente se tornará duas ou três vezes maior que o valor que você coloca aqui.

Especifica a quantidade ideal de memória a ser usada para:

  • Objetos em trânsito
  • Objetos Quentes
  • Objetos em cache negativo

Os dados para esses objetos são armazenados em blocos de 4 kB. Este parâmetro especifica o limite superior ideal para o tamanho total de blocos de 4 kB alocados. Objetos em trânsito têm a maior prioridade.

Objetos em trânsito têm prioridade sobre os outros. Quando for necessário espaço adicional para os dados recebidos, objetos em cache e quentes negativos serão liberados. Em outras palavras, os objetos em cache e quentes em negativo preencherão qualquer espaço não utilizado não necessário para objetos em trânsito.

Se as circunstâncias exigirem, esse limite será excedido. Especificamente, se a taxa de solicitação de entrada exigir mais do que essa quantidade de memória para armazenar objetos em trânsito, o Squid excederá esse limite para atender às novas solicitações. Quando a carga diminuir, os blocos serão liberados até que a marca d'água alta seja atingida. Depois disso, os blocos serão usados para armazenar objetos quentes.

Tamanho do cache do FQDN : O padrão é 1024. Número máximo de entradas de cache do Nome de Domínio Totalmente Qualificado.

Marca d'água de memória alta : O padrão é 95%. Quando essa marca alta é atingida, objetos quentes e objetos em trânsito serão liberados para liberar memória.

Marca d'água de memória baixa : o padrão é 90%. Depois que a marca d'água alta é atingida, o Squid libera memória até que o uso caia abaixo dessa marca.

Marca d'água do disco : o padrão é 95%. Quando essa marca é atingida, o Squid começa a liberar agressivamente o disco.

Marca d'água de disco baixa : o padrão é 90%. Após atingir a marca d'água, o Squid continuará lavando o disco até que o uso caia abaixo dessa marca.

Tamanho máximo do objeto em cache : O padrão é 4096 kB. Essa configuração limita o tamanho dos objetos que serão armazenados pelo cache. Qualquer coisa acima desse tamanho nunca será armazenada em cache e deve ser carregada do servidor de origem sempre.

Tamanho do cache do endereço IP : o padrão é 1024. Número de entradas a serem armazenadas no cache IP.

Marca d'água máxima do cache IP : o padrão é 95%. O ponto em que o Squid começa a liberar as entradas do cache IP.

Marca d'água de cache baixo do IP : o padrão é 90%. Após o início da limpeza do cache IP, ele continuará até que o número de entradas no cache IP caia abaixo dessa porcentagem.
