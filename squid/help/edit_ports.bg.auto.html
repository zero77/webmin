Портове и мрежи
Прокси порт : По подразбиране 3128. Тази стойност задава какъв порт вашият Squid сървър ще слуша за http заявки. Клиентските браузъри ще трябва да бъдат конфигурирани да се свързват с прокси на този порт, или прозрачността също ще трябва да бъде настроена.

ICP порт : номерът на порта, където Squid изпраща и получава ICP заявки до и от съседните кешове. По подразбиране е 3130. За деактивиране използвайте "0". Може да се отмени с -u в командния ред.

Входящ TCP адрес : По подразбиране няма. Дефинира HTTP сокета, който приема връзки от клиенти и други кешове.

Изходящ TCP адрес : По подразбиране няма. Дефинира HTTP сокета за връзки, направени към отдалечени сървъри и други кешове.

Изходящ UDP адрес : По подразбиране няма. Определя ICP сокет, получаващи пакети от други кешове.

Входящ UDP адрес : По подразбиране няма. Определя гнездото за ICP пакетите, които се изпращат към други кешове.

TCP буфер за получаване : Размер на TCP буфер. По подразбиране към OS получават буфер.
