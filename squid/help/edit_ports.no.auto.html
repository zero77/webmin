Porter og nettverk
Proxy-port : Som standard er 3128. Denne verdien angir hvilken port Squid-serveren skal lytte på for http-forespørsler. Klientlesere må konfigureres for å koble til en proxy på denne porten, eller transparens må også konfigureres.

ICP-port : Portnummeret hvor blekksprut sender og mottar ICP-forespørsler til og fra nabokuffere. Standard er 3130. For å deaktivere bruk "0". Kan overstyres med -u på kommandolinjen.

Inngående TCP-adresse : Standard er ingen. Definerer HTTP-kontakten som godtar tilkoblinger fra klienter og andre hurtigbuffer.

Utgående TCP-adresse : Standard er ingen. Definerer HTTP-kontakten for tilkoblinger til eksterne servere og andre hurtigbuffer.

Utgående UDP-adresse : Standard er ingen. Definerer ICP-socket som mottar pakker fra andre cacher.

Inngående UDP-adresse : Standard er ingen. Definerer kontakten for ICP-pakker som skal sendes ut til andre hurtigbuffer.

TCP-mottaksbuffer : Størrelse på TCP-bufferen. Standard for OS mottar buffer.
