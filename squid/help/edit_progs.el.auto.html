Βοηθητικά προγράμματα
Πλάτος στήλης FTP : Η προεπιλογή είναι 32. Ορίζει το πλάτος των λιστών ftp. Αυτό πρέπει να ρυθμιστεί ώστε να ταιριάζει στο πλάτος ενός τυπικού προγράμματος περιήγησης. Η ρύθμιση αυτή πολύ μικρή μπορεί να αποκόψει μεγάλα ονόματα αρχείων κατά την περιήγηση σε ιστότοπους ftp.

Σύνδεση Anon FTP : Η προεπιλογή είναι «Squid @». Αν θέλετε ο ανώνυμος κωδικός πρόσβασης σύνδεσης να είναι πιο ενημερωτικός (και να επιτρέψετε τη χρήση επιλεκτικών διακομιστών ftp), ορίστε το σε κάτι που να ανταποκρίνεται στον τομέα σας, όπως wwwuser@somwhere.net.

Ο λόγος για τον οποίο αυτό είναι χωρίς τομέα από προεπιλογή είναι ότι το αίτημα μπορεί να υποβληθεί εκ μέρους ενός χρήστη σε οποιονδήποτε τομέα, ανάλογα με τον τρόπο χρήσης της προσωρινής μνήμης. Ορισμένοι διακομιστές ftp επικυρώνουν επίσης ότι η διεύθυνση email είναι έγκυρη (για παράδειγμα perl.com).

Πρόγραμμα DNS Squid : Η προεπιλογή είναι dnsserver. Καθορίστε τη θέση του εκτελέσιμου για τη διαδικασία dnslookup. Η προεπιλογή είναι ένας διακομιστής ονομάτων προσωρινής αποθήκευσης χωρίς αποκλεισμό που εγκαθίσταται με το Squid από προεπιλογή.

Αριθμός προγραμμάτων DNS : Η προεπιλογή είναι 5. Ο αριθμός των διαδικασιών που δημιουργήθηκαν για την αναζήτηση ονόματος DNS υπηρεσίας. Για βαριά φορτωμένες κρυφές μνήμες σε μεγάλους διακομιστές, πιθανότατα θα πρέπει να αυξήσετε αυτήν την τιμή σε τουλάχιστον 10. Το μέγιστο είναι 32.

Προσθήκη τομέα σε αιτήματα : Η προεπιλογή είναι απενεργοποιημένη. Κανονικά το "dnsserver" απενεργοποιεί την επιλογή RES_DEFNAMES resolver (βλ. Res_init (3)). Αυτό εμποδίζει την προσωρινή ερμηνεία των cache σε μια ιεραρχία τοπικά ονόματα κεντρικών στοιχείων. Για να επιτρέψετε στο dnsserver να χειρίζεται ονόματα ενός συστατικού, ενεργοποιήστε αυτήν την επιλογή.

Διευθύνσεις διακομιστή DNS : Προεπιλογή είναι η χρήση του αρχείου resolv.conf. Χρησιμοποιήστε αυτό εάν θέλετε να καθορίσετε μια λίστα διακομιστών ονομάτων DNS (διευθύνσεις IP) για χρήση αντί εκείνων που δίνονται στο αρχείο /etc/resolv.conf.

Πρόγραμμα εκκαθάρισης προσωρινής μνήμης : Η προεπιλογή είναι «αποσύνδεση». Καθορίστε τη θέση του εκτελέσιμου για τη διαδικασία διαγραφής αρχείων. Αυτό μπορεί να μην είναι απαραίτητο εάν χρησιμοποιείτε το async-io, καθώς διαχειρίζεται ένα νήμα σε ορισμένες εκδόσεις.

Πρόγραμμα Squid ping : Προεπιλογές στο «pinger». Καθορίστε τη θέση του εκτελέσιμου για τη διαδικασία pinger. Αυτό είναι χρήσιμο μόνο εάν διαμορφώσατε το Squid (κατά τη διάρκεια της συμμόρφωσης) με την επιλογή «--enable-icmp».

Προσαρμοσμένο πρόγραμμα ανακατεύθυνσης : Προεπιλογή σε κανένα. Καθορίστε τη θέση του εκτελέσιμου για την ανακατεύθυνση διευθύνσεων URL. Δεδομένου ότι μπορούν να εκτελέσουν σχεδόν οποιαδήποτε λειτουργία, δεν περιλαμβάνεται. Ανατρέξτε στις Σημειώσεις έκδοσης για πληροφορίες σχετικά με τον τρόπο σύνταξης.

Αριθμός προγραμμάτων ανακατεύθυνσης : Προεπιλογή σε 5 εάν έχει ρυθμιστεί μια ανακατεύθυνση. Ο αριθμός των διαδικασιών ανακατεύθυνσης προς αναπαραγωγή. Εάν ξεκινήσετε πάρα λίγα, το Squid θα πρέπει να περιμένει να επεξεργαστεί μια καθυστέρηση διευθύνσεων URL, επιβραδύνοντάς το. Εάν ξεκινήσετε πάρα πολλά θα χρησιμοποιούν RAM και άλλους πόρους συστήματος.
