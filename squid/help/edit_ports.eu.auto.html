Portuak eta Sareak
Proxy portua : 3128 balio lehenetsiak. Balio honek zure Squid zerbitzariak http eskaerei zer entzungo dion zehazten du. Bezeroen arakatzaileak portu honetako proxy batera konektatzeko konfiguratu beharko dira edo gardentasuna konfiguratu beharko da.

ICP portua : Squid-ek ICP eskaerak bidaltzen eta jasotzen dituen portu zenbakia auzoko kaxetatik etortzen da. Lehenespenez 3130. "0" erabiltzea desgaitzeko. Ezin da -u-rekin komando lerroan baliogabetu.

Sarrerako TCP helbidea : inor ez. Bezeroen eta beste cacheen konexioak onartzen dituen HTTP entxufea definitzen du.

Irteerako TCP helbidea : inor ez da lehenetsi. HTTP entxufeak urruneko zerbitzarietarako eta beste cache batzuetarako konexioetarako definitzen du

Irteerako UDP helbidea : Lehenetsitakoak ez ditu. Beste hainbat katxeetako paketeak jasotzen dituzten ICP entxufeak definitzen ditu.

UDP helbide sarrerako helbidea : bat ere ez. ICP paketeak beste cacheetara bidaltzen diren entxufeak definitzen ditu.

TCP jaso bufferra : TCP bufferraren tamaina. OSaren lehenetsitakoek bufferra jasotzen dute.
