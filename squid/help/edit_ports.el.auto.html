Θύρες και δικτύωση
Θύρα διακομιστή μεσολάβησης : Προεπιλογές σε 3128. Αυτή η τιμή καθορίζει σε ποια θύρα θα ακούει ο διακομιστής Squid για αιτήματα http. Τα προγράμματα περιήγησης πελάτη θα πρέπει να ρυθμιστούν ώστε να συνδέονται σε διακομιστή μεσολάβησης σε αυτήν τη θύρα, ή θα πρέπει επίσης να ρυθμιστεί η διαφάνεια.

Θύρα ICP : Ο αριθμός θύρας όπου το Squid στέλνει και λαμβάνει αιτήματα ICP από και προς γειτονικές κρυφές μνήμες. Η προεπιλογή είναι 3130. Για απενεργοποίηση χρησιμοποιήστε το "0". Μπορεί να παρακαμφθεί με -u στη γραμμή εντολών.

Διεύθυνση εισερχόμενης TCP : Προεπιλογή σε καμία. Ορίζει την υποδοχή HTTP που δέχεται συνδέσεις από πελάτες και άλλες κρυφές μνήμες.

Εξερχόμενη διεύθυνση TCP : Προεπιλογή σε καμία. Ορίζει την υποδοχή HTTP για συνδέσεις που πραγματοποιούνται σε απομακρυσμένους διακομιστές και άλλες κρυφές μνήμες.

Εξερχόμενη διεύθυνση UDP : Προεπιλογή σε καμία. Ορίζει τα πακέτα λήψης της υποδοχής ICP από άλλες κρυφές μνήμες.

Εισερχόμενη διεύθυνση UDP : Προεπιλογή σε καμία. Καθορίζει την υποδοχή για τα πακέτα ICP που αποστέλλονται σε άλλες κρυφές μνήμες.

TCP buffer λήψης : Μέγεθος του buffer TCP. Οι προεπιλογές στο λειτουργικό σύστημα λαμβάνουν buffer.
