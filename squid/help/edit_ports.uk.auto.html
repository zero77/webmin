Порти та мережі
Порт проксі : за замовчуванням 3128. Це значення встановлює, який порт буде прослуховувати ваш сервер Squid для запитів http. Клієнтські браузери повинні бути налаштовані для підключення до проксі на цьому порту, або прозорість також повинна бути налаштована.

Порт ICP : номер порту, куди Squid надсилає та приймає запити ICP до кешів сусідів та з них. За замовчуванням - 3130. Щоб вимкнути використання "0". Може бути замінено на -u в командному рядку.

Вхідна TCP-адреса : за замовчуванням немає. Визначає HTTP-сокет, який приймає з'єднання з клієнтами та іншими кешами.

Вихідна TCP-адреса : за замовчуванням немає. Визначає сокет HTTP для з'єднань, віддалених на віддалені сервери та інші кеші.

Вихідна адреса UDP : за замовчуванням немає. Визначає ICP-розетку, що приймає пакети з інших кешів.

Вхідна адреса UDP : за замовчуванням немає. Визначає сокет для пакетів ICP, що надсилаються в інші кеші.

Буфер прийому TCP : розмір буфера TCP. За замовчуванням ОС приймає буфер.
