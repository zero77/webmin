Porty a sítě
Proxy port : Výchozí hodnota 3128. Tato hodnota nastavuje, na jakém portu bude váš server Squid poslouchat požadavky http. Klientské prohlížeče budou muset být nakonfigurovány tak, aby se připojily k serveru proxy na tomto portu, nebo bude nutné nastavit také průhlednost.

ICP port : Číslo portu, na který Squid odesílá a přijímá ICP požadavky do az mezipaměti sousedů. Výchozí hodnota je 3130. Pro deaktivaci použijte "0". Může být přepsán -u na příkazovém řádku.

Příchozí TCP adresa : Výchozí na žádnou. Definuje soket HTTP, který přijímá připojení od klientů a jiných mezipamětí.

Odchozí TCP adresa : Výchozí na žádnou. Definuje soket HTTP pro připojení ke vzdáleným serverům a jiným mezipaměti.

Odchozí adresa UDP : Výchozí žádná. Definuje pakety pro příjem soketu ICP z jiných mezipamětí.

Příchozí adresa UDP : výchozí žádné. Definuje soket pro ICP pakety odeslané do jiných mezipaměti.

Vyrovnávací paměť TCP : Velikost vyrovnávací paměti TCP. Výchozí nastavení pro OS přijímá vyrovnávací paměť.
