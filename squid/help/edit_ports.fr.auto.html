Ports et mise en réseau
Port proxy : la valeur par défaut est 3128. Cette valeur définit le port sur lequel votre serveur Squid écoutera les requêtes http. Les navigateurs clients devront être configurés pour se connecter à un proxy sur ce port, ou la transparence devra également être configurée.

Port ICP : numéro de port où Squid envoie et reçoit des requêtes ICP vers et depuis les caches voisins. La valeur par défaut est 3130. Pour désactiver, utilisez "0". Peut être remplacé par -u sur la ligne de commande.

Adresse TCP entrante : par défaut, aucune. Définit le socket HTTP qui accepte les connexions des clients et autres caches.

Adresse TCP sortante : par défaut, aucune. Définit le socket HTTP pour les connexions établies aux serveurs distants et autres caches.

Adresse UDP sortante : par défaut, aucune. Définit le socket ICP recevant les paquets d'autres caches.

Adresse UDP entrante : par défaut, aucune. Définit le socket pour les paquets ICP envoyés à d'autres caches.

Tampon de réception TCP : taille du tampon TCP. Par défaut, le tampon de réception du système d'exploitation.
