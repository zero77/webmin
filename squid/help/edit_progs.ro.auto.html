Programe de ajutor
Lățimea coloanei FTP : implicit este 32. Setează lățimea listărilor ftp. Acest lucru ar trebui să fie setat pentru a se potrivi cu lățimea unui browser standard. Dacă setați acest lucru prea mic, puteți tăia numele de fișiere lungi atunci când navigați pe site-uri ftp.

Logare FTP Anon : implicit este „Squid @”. Dacă doriți ca parola de autentificare anonimă să fie mai informativă (și să permită utilizarea serverelor ftp picky), setați acest lucru pe ceva rezonabil pentru domeniul dvs., cum ar fi wwwuser@somewhere.net.

Motivul pentru care acest lucru este fără domeniu în mod implicit este faptul că solicitarea poate fi făcută în numele unui utilizator din orice domeniu, în funcție de modul în care este folosit cache-ul. Unele server ftp validează, de asemenea, că adresa de e-mail este validă (de exemplu, perl.com).

Programul DNS Squid : Implicit este dnsserver. Specificați locația executabilului pentru procesul dnslookup. Valoarea implicită este un server de nume de memorie în cache care nu blochează instalat în mod implicit cu Squid.

Numărul de programe DNS : Valoarea implicită este 5. Numărul de procese generate pentru căutarea numelor DNS de serviciu. Pentru cache-urile puternic încărcate pe serverele mari, probabil că ar trebui să creșteți această valoare la cel puțin 10. Maximul este de 32.

Adăugați domeniu la solicitări : implicit este dezactivat. În mod normal „dnsserver” dezactivează opțiunea de rezolvare RES_DEFNAMES (vezi res_init (3)). Acest lucru împiedică cache-urile dintr-o ierarhie să interpreteze local numele de host cu o singură componentă. Pentru a permite dnsserver să gestioneze nume cu o singură componentă, activați această opțiune.

Adresele serverului DNS : implicit este să folosiți fișierul rezolv.conf. Folosiți acest lucru dacă doriți să specificați o listă de servere de nume DNS (adrese IP) pe care să le utilizați în loc de cele date în fișierul dvs. /etc/resolv.conf.

Programul de ștergere a memoriei în cache : implicit este „deblocat”. Specificați locația executabilului pentru procesul de ștergere a fișierelor. Este posibil să nu fie necesar dacă utilizați async-io, deoarece este tratat de un thread în unele versiuni.

Program de ping calmar : Default la „pinger”. Specificați locația executabilului pentru procesul de pinger. Acest lucru este util numai dacă ați configurat Squid (în timpul conformării) cu opțiunea '--enable-icmp'.

Program de redirecționare personalizat : Implicite la oricare. Specificați locația executabilului pentru redirectorul URL. Deoarece pot îndeplini aproape orice funcție, nu este inclusă niciuna. Consultați Notele de versiune pentru informații despre cum puteți scrie unul.

Număr de programe de redirecționare : Valori implicite la 5 dacă este configurat un redirector. Numărul proceselor de redirector pentru a genera. Dacă începeți prea puțini, Squid va trebui să aștepte ca aceștia să proceseze o listă de adrese URL, încetinind-o. Dacă începeți prea multe, vor folosi memoria RAM și alte resurse de sistem.
