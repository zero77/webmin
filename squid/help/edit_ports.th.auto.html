พอร์ตและระบบเครือข่าย
พอร์ตพร็อกซี : ค่าเริ่มต้นที่ 3128 ค่านี้ตั้งค่าพอร์ตที่เซิร์ฟเวอร์ Squid ของคุณจะรับฟังสำหรับคำขอ HTTP เบราว์เซอร์ไคลเอ็นต์จะต้องได้รับการกำหนดค่าให้เชื่อมต่อกับพร็อกซีบนพอร์ตนี้หรือจะต้องตั้งค่าความโปร่งใสด้วย

พอร์ต ICP : หมายเลขพอร์ตที่ Squid ส่งและรับคำขอ ICP ไปและกลับจากแคชเพื่อนบ้าน ค่าเริ่มต้นคือ 3130 หากต้องการปิดใช้งาน "0" อาจถูกเขียนทับด้วย -u บนบรรทัดคำสั่ง

ที่อยู่ TCP ขาเข้า : ค่าเริ่มต้นเป็น none กำหนดซ็อกเก็ต HTTP ที่ยอมรับการเชื่อมต่อจากไคลเอนต์และแคชอื่น ๆ

ที่อยู่ TCP ขาออก : ไม่มีค่าเริ่มต้น กำหนดซ็อกเก็ต HTTP สำหรับการเชื่อมต่อกับเซิร์ฟเวอร์ระยะไกลและแคชอื่น ๆ

ที่อยู่ UDP ขาออก : ค่าเริ่มต้นเป็น none กำหนดซ็อกเก็ต ICP ที่รับแพ็กเก็ตจากแคชอื่น

ที่อยู่ UDP ขาเข้า : ค่าเริ่มต้นเป็น none กำหนดซ็อกเก็ตสำหรับแพ็คเก็ต ICP ที่ถูกส่งไปยังแคชอื่น

TCP ได้รับบัฟเฟอร์ : ขนาดของบัฟเฟอร์ TCP ค่าเริ่มต้นของระบบปฏิบัติการรับบัฟเฟอร์
