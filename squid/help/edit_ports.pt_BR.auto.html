Portas e redes
Porta do proxy : o padrão é 3128. Esse valor define em qual porta o servidor Squid escutará solicitações http. Os navegadores do cliente precisarão ser configurados para conectar-se a um proxy nessa porta, ou a transparência também precisará ser configurada.

Porta ICP : O número da porta onde o Squid envia e recebe solicitações de ICP para e de caches vizinhos. O padrão é 3130. Para desativar, use "0". Pode ser substituído por -u na linha de comando.

Endereço TCP de entrada : O padrão é nenhum. Define o soquete HTTP que aceita conexões de clientes e outros caches.

Endereço TCP de saída : o padrão é nenhum. Define o soquete HTTP para conexões feitas com servidores remotos e outros caches.

Endereço UDP de saída : o padrão é nenhum. Define o soquete ICP que recebe pacotes de outros caches.

Endereço UDP de entrada : O padrão é nenhum. Define o soquete para pacotes ICP sendo enviados para outros caches.

Buffer de recebimento TCP : tamanho do buffer TCP. O padrão é o buffer de recebimento do SO.
