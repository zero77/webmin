Administrative Optionen
Als Unix-Benutzer ausführen: Die Standardeinstellung ist "Niemand" oder "Tintenfisch", abhängig vom System. Wenn der Cache als Root ausgeführt wird, ändert er seine effektive / reale ID in diesen Wert.

Als Unix-Gruppe ausführen: Die Standardeinstellung ist "Nogroup" oder "Squid", abhängig vom System. Wenn der Cache als Root ausgeführt wird, ändert er seine effektive / reale Gruppen-ID in diesen Wert.

Proxy-Authentifizierungsbereich : Gibt den Bereichsnamen an, der dem Client zur Proxy-Authentifizierung gemeldet werden soll (Teil des Textes, den der Benutzer sieht, wenn er zur Eingabe seines Benutzernamens und Kennworts aufgefordert wird).

E-Mail-Adresse des Cache-Managers : Die Standardeinstellung ist "Webmaster" oder "Root", abhängig vom System. E-Mail-Adresse des lokalen Cache-Managers, der E-Mails erhält, wenn der Cache stirbt.

Sichtbarer Hostname : Wenn Sie in Fehlermeldungen usw. einen speziellen Hostnamen angeben möchten, definieren Sie diesen. Andernfalls wird der Rückgabewert von gethostname () verwendet. Wenn Sie mehrere Caches in einem Cluster haben und Fehler bei der IP-Weiterleitung erhalten, müssen Sie festlegen, dass diese mit dieser Einstellung individuelle Namen haben.

Eindeutiger Hostname : Wenn Sie mehrere Computer mit demselben sichtbaren Hostnamen haben möchten, müssen Sie jedem Computer einen anderen eindeutigen Hostnamen zuweisen, damit Weiterleitungsschleifen erkannt werden können.

Cache Announce Host : Hostname, an den Ansagemeldungen gesendet werden. Weitere Informationen zu Cache-Hierarchien finden Sie in den häufig gestellten Fragen zu The Squid . Der Standardwert ist 'tracker.ircache.net', wenn die Ansage durch Festlegen eines Ansagezeitraums aktiviert wird.

Cache-Ankündigungsport : Host-Port, an den Ankündigungsnachrichten gesendet werden. Der Standardwert ist 3131, wenn die Ansage aktiviert ist.

Ankündigungszeitraum : Häufigkeit der Ankündigungen. Aktivieren Sie Ankündigungen für eine Cache-Hierarchie, indem Sie diese auf "1 Tag" setzen. Der Standardwert ist '0', wodurch die Ansage deaktiviert wird.
