Programi za pomoć
FTP širina stupca : Zadana je 32. Postavlja širinu ftp popisa. Ovo bi trebalo biti postavljeno na širinu standardnog preglednika. Ako je ovo premalo postavljeno, može se oduzeti dugačka datoteka kada pregledavate ftp web stranice.

Anon FTP prijava : Zadana vrijednost je "Liganj @". Ako želite da lozinka za anonimnu prijavu bude informativnija (i omogućite upotrebu izbirljivih ftp servera), postavite to na nešto dostupno za vašu domenu, poput wwwuser@somewhere.net.

Razlog zašto je ovo zadano bez domene je taj što se zahtjev može uputiti u ime korisnika na bilo kojoj domeni, ovisno o načinu na koji se koristi predmemorija. Neki ftp poslužitelj također potvrđuje da je adresa e-pošte valjana (na primjer perl.com).

DNS program lignje : Zadano je dnsserver. Navedite lokaciju izvršnog programa za dnslookup postupak. Zadani je poslužitelj imena ne blokirajući predmemoriranje koji je sa Squidom postavljen prema zadanim postavkama.

Broj DNS programa : Zadano je 5. Broj postupaka pokrećenih pretraživanja DNS usluga. Za jako učitane predmemorije na velikim poslužiteljima vjerovatno biste trebali povećati ovu vrijednost na najmanje 10. Maksimalno je 32.

Dodavanje domene zahtjevima : zadano je isključeno. 'Dnsserver' obično onemogućuje RES_DEFNAMES rezolucijsku opciju (vidi res_init (3)). Ovo sprečava predmemorije u hijerarhiji lokalno tumačenje jednokomponentnih imena hosta. Da biste dnsserveru mogli upravljati jednokomponentnim imenima, omogućite ovu opciju.

Adresa DNS poslužitelja : Zadana je upotreba datoteke rezolucije.conf. Upotrijebite ovo ako želite odrediti popis poslužitelja imena DNS (IP adrese) koji ćete koristiti umjesto onih koji se nalaze u vašoj /etc/resolv.conf datoteci.

Program za brisanje predmemorije : zadani je "prekinuti vezu". Navedite mjesto izvršnog postupka za brisanje datoteke. Ovo vam možda neće biti potrebno ako koristite async-io jer u nekim verzijama njime upravlja nit.

Program ping za lignje : podrazumijeva se 'pinger'. Navedite lokaciju izvršnog elementa za proces pinger. Ovo je korisno samo ako ste konfigurirali lignje (za vrijeme dopisivanja) opcijom '--enable-icmp'.

Prilagođeni program za preusmjeravanje : prema zadanim postavkama ne postoji. Navedite lokaciju izvršnog URL preusmjerivača. Budući da mogu obavljati gotovo bilo koju funkciju, nije uključena niti jedna. Pogledajte napomene o izdanju za informacije o tome kako ih napisati.

Broj programa za preusmjeravanje : Zadano je 5 ako je konfiguriran redirektor. Broj procesa preusmjeravanja na mrijest. Ako započnete premalo lignje, morat ćete ih pričekati da obrade zaostatke URL-ova, usporavajući ga. Ako ih pokrenete previše, oni će koristiti RAM i ostale resurse sustava.
