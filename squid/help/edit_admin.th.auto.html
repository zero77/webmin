เรียกใช้ในฐานะผู้ใช้ Unix : ค่าเริ่มต้นคือ 'nobody' หรือ 'squid' ขึ้นอยู่กับระบบ หากแคชถูกเรียกใช้ในฐานะรูทแคชจะเปลี่ยน ID ที่มีประสิทธิภาพ / จริงเป็นค่านี้

เรียกใช้เป็นกลุ่ม Unix : ค่าเริ่มต้นคือ 'nogroup' หรือ 'squid' ขึ้นอยู่กับระบบ หากแคชถูกเรียกใช้ในฐานะรูทแคชจะเปลี่ยน ID กลุ่มที่มีประสิทธิภาพ / แท้จริงเป็นค่านี้

ขอบเขตการรับรองความถูกต้องของพร็อกซี : ระบุชื่อขอบเขตที่จะรายงานไปยังไคลเอนต์สำหรับการตรวจสอบความถูกต้องของพร็อกซี (ส่วนหนึ่งของข้อความที่ผู้ใช้จะเห็นเมื่อได้รับพร้อมท์ชื่อผู้ใช้และรหัสผ่าน)

ที่อยู่อีเมลของผู้จัดการแคช : ค่าเริ่มต้นคือ 'เว็บมาสเตอร์' หรือ 'รูท' ขึ้นอยู่กับระบบ ที่อยู่อีเมลของผู้จัดการแคชในท้องถิ่นที่จะรับจดหมายหากแคชนั้นตาย

ชื่อโฮสต์ที่มองเห็นได้ : หากคุณต้องการนำเสนอชื่อโฮสต์พิเศษในข้อความแสดงข้อผิดพลาด ฯลฯ ให้กำหนดชื่อนี้ มิฉะนั้นจะใช้ค่าส่งคืนของ gethostname () หากคุณมีหลายแคชในคลัสเตอร์และรับข้อผิดพลาดเกี่ยวกับการส่งต่อ IP คุณต้องตั้งค่าให้มีชื่อบุคคลที่มีการตั้งค่านี้

ชื่อโฮสต์ที่ไม่ซ้ำกัน : หากคุณต้องการมีเครื่องหลายเครื่องที่มี ชื่อโฮสต์ที่มองเห็นได้ เหมือนกันคุณจะต้องให้ ชื่อโฮสต์ที่ไม่ซ้ำ กันแต่ละเครื่องแตกต่าง กัน เพื่อให้สามารถตรวจพบลูปการส่งต่อ

แคชประกาศโฮสต์ : ชื่อโฮสต์ที่จะส่งข้อความประกาศ ค้นหาเพิ่มเติมเกี่ยวกับลำดับชั้นแคชจาก The Squid FAQ ค่าเริ่มต้นเป็น 'tracker.ircache.net' เมื่อเปิดใช้งานการประกาศโดยการตั้งค่าช่วงเวลาการประกาศ

แคชประกาศพอร์ต : โฮสต์พอร์ตที่จะส่งข้อความประกาศ ค่าเริ่มต้นเป็น 3131 เมื่อเปิดใช้งานการประกาศ

ระยะเวลาประกาศ : ความถี่ของการประกาศ เปิดใช้งานการประกาศให้กับลำดับความสำคัญแคชโดยการตั้งค่านี้เป็น '1 วัน' ค่าเริ่มต้นคือ '0' ซึ่งจะปิดการประกาศ
