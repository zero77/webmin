Использование памяти
Ограничение использования памяти : по умолчанию 8 МБ.

ПРИМЕЧАНИЕ. Этот параметр не указывает максимальный размер процесса. Это накладывает ограничение на один аспект использования памяти Squid. Squid использует память и для других вещей. Ваш процесс, вероятно, станет в два или три раза больше, чем значение, которое вы здесь указали.

Он определяет идеальный объем памяти, который будет использоваться для:

  • Транзитные объекты
  • Горячие объекты
  • Объекты с отрицательным кэшированием

Данные для этих объектов хранятся в блоках по 4 кБ. Этот параметр указывает идеальный верхний предел общего размера выделенных блоков 4 КБ. Транзитные объекты имеют наивысший приоритет.

Транзитные объекты имеют приоритет над остальными. Когда для входящих данных требуется дополнительное пространство, объекты с отрицательным кэшированием и горячие объекты будут освобождены. Другими словами, объекты с отрицательным кэшированием и горячие объекты будут заполнять любое неиспользуемое пространство, не необходимое для объектов в пути.

Если обстоятельства требуют, этот предел будет превышен. В частности, если вашей частоте входящих запросов требуется больше, чем этот объем памяти для хранения транзитных объектов, Squid превысит этот лимит для удовлетворения новых запросов. Когда нагрузка уменьшится, блоки будут освобождены до тех пор, пока не будет достигнута максимальная температура. После этого блоки будут использоваться для хранения горячих объектов.

Размер кэша полного доменного имени : по умолчанию 1024. Максимальное количество записей кэша полных доменных имен.

Верхний предел памяти : по умолчанию 95%. Когда эта высокая оценка достигнута, горячие и транзитные объекты будут освобождены для освобождения памяти.

Память низкого уровня : по умолчанию 90%. После достижения максимальной отметки Squid освобождает память до тех пор, пока уровень использования не опустится ниже этой отметки.

Верхняя отметка диска : по умолчанию 95%. Когда эта высокая оценка будет достигнута, Squid начнет агрессивно очищать диск.

Марка диска ниже уровня воды : по умолчанию 90%. После того, как будет достигнут максимальный уровень воды, Squid продолжит очищать диск до тех пор, пока уровень использования не опустится ниже этого уровня.

Максимальный размер кэшируемого объекта : по умолчанию 4096 кБ. Этот параметр ограничивает размер объектов, которые будут храниться в кэше. Все, что превышает этот размер, никогда не будет кэшироваться и должно загружаться с исходного сервера каждый раз.

Размер кэша IP-адреса : по умолчанию 1024. Количество записей, которые будут сохранены в кэше IP.

Верхняя отметка IP-кэша : по умолчанию 95%. Точка, в которой Squid начинает сбрасывать записи кеша IP.

Низкая отметка кеша IP : по умолчанию 90%. После того как начнется очистка IP-кэша, он будет продолжаться до тех пор, пока количество записей в IP-кэше не станет ниже этого процента.
