Ανανέωση κανόνων
Αυτή η σελίδα μπορεί να χρησιμοποιηθεί για τη δημιουργία κανόνων που αντικαθιστούν τον προεπιλεγμένο χρόνο ανανέωσης του Squid για διευθύνσεις URL που αντιστοιχούν σε ορισμένες κανονικές εκφράσεις. Μπορεί να είναι χρήσιμο για την αλλαγή του χρόνου προσωρινής αποθήκευσης για συγκεκριμένους ιστότοπους ή πρωτόκολλα, αντί για την προσαρμογή των καθολικών ρυθμίσεων.
