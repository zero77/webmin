Оновити правила
Ця сторінка може використовуватися для створення правил, які переосмислюють час оновлення кальмарів за замовчуванням для URL-адрес, що відповідають певним регулярним виразам. Це може бути корисно для зміни часу кешу для певних веб-сайтів або протоколів, а не для налаштування глобальних налаштувань.
