Обновить правила
Эта страница может использоваться для создания правил, которые переопределяют стандартное время обновления Squid для URL, соответствующих определенным регулярным выражениям. Это может быть полезно для изменения времени кэширования для определенных веб-сайтов или протоколов, а не для настройки глобальных настроек.
