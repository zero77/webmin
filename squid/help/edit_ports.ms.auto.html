Pelabuhan dan Rangkaian
Port proksi : Lalai ke 3128. Nilai ini menetapkan port yang akan didengarkan oleh pelayan Squid anda untuk permintaan http. Penyemak imbas pelanggan perlu dikonfigurasi untuk menyambung ke proksi di port ini, atau ketelusan juga perlu disiapkan.

Port ICP : Nombor port tempat Squid menghantar dan menerima permintaan ICP ke dan dari cache jiran. Lalai adalah 3130. Untuk melumpuhkan penggunaan "0". Mungkin diganti dengan -u pada baris perintah.

Alamat TCP Masuk : Lalai tidak ada. Mentakrifkan soket HTTP yang menerima sambungan dari klien dan cache lain.

Alamat TCP Keluar : Lalai tidak ada. Mentakrifkan soket HTTP untuk sambungan yang dibuat ke pelayan jauh dan cache lain.

Alamat UDP Keluar : Lalai tidak ada. Menentukan paket penerimaan soket ICP dari cache lain.

Alamat UDP Masuk : Lalai tidak ada. Mentakrifkan soket untuk paket ICP yang dihantar ke cache yang lain.

TCP menerima buffer : Ukuran buffer TCP. Lalai ke OS menerima penyangga.
