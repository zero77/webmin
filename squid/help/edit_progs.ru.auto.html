Вспомогательные программы
Ширина столбца FTP : по умолчанию 32. Устанавливает ширину списков FTP. Это должно быть установлено, чтобы соответствовать ширине стандартного браузера. Установка этого значения может обрезать длинные имена файлов при просмотре FTP-сайтов.

Anon FTP login : по умолчанию установлено значение «Squid @». Если вы хотите, чтобы анонимный пароль для входа был более информативным (и позволял использовать разборчивые ftp-серверы), установите для этого домена что-то подходящее, например, wwwuser@somewhere.net.

Причина, по которой это по умолчанию является бездоменной, заключается в том, что запрос может быть сделан от имени пользователя в любом домене, в зависимости от того, как используется кэш. Некоторые ftp-сервер также проверяют правильность адреса электронной почты (например, perl.com).

Программа Squid DNS : по умолчанию используется dnsserver. Укажите местоположение исполняемого файла для процесса dnslookup. По умолчанию используется неблокирующий кеширующий сервер имен, который по умолчанию устанавливается вместе со Squid.

Количество программ DNS : по умолчанию - 5. Количество процессов, созданных для поиска служб DNS. Для сильно загруженных кешей на больших серверах вам, вероятно, следует увеличить это значение как минимум до 10. Максимальное значение - 32.

Добавить домен к запросам : по умолчанию отключено. Обычно dnsserver отключает опцию решателя RES_DEFNAMES (см. Res_init (3)). Это не позволяет кэшам в иерархии интерпретировать однокомпонентные имена хостов локально. Чтобы разрешить dnsserver обрабатывать однокомпонентные имена, включите эту опцию.

Адреса DNS-серверов : по умолчанию используется файл resolv.conf. Используйте это, если вы хотите указать список DNS-серверов имен (IP-адресов) для использования вместо тех, которые указаны в вашем файле /etc/resolv.conf.

Программа очистки кэша : по умолчанию «unlinkd». Укажите расположение исполняемого файла для процесса удаления файла. Это может не понадобиться, если вы используете async-io, так как он обрабатывается потоком в некоторых версиях.

Программа Squid ping : по умолчанию используется «pinger». Укажите местоположение исполняемого файла для процесса пингера. Это полезно только в том случае, если вы настроили Squid (во время компиляции) с параметром --enable-icmp.

Пользовательская программа перенаправления : по умолчанию нет. Укажите расположение исполняемого файла для перенаправителя URL. Так как они могут выполнять практически любую функцию, она не включена. См. Примечания к выпуску для получения информации о том, как написать один.

Количество программ перенаправления : по умолчанию 5, если перенаправитель настроен. Количество процессов редиректора для появления. Если вы запустите слишком мало, Squid придется подождать, пока они обработают задел URL, замедляя его. Если вы запустите слишком много, они будут использовать оперативную память и другие системные ресурсы.
