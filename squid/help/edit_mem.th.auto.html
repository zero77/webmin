การใช้ความจำ
ขีด จำกัด การใช้หน่วยความจำ : เริ่มต้นที่ 8MB

หมายเหตุ: พารามิเตอร์นี้ไม่ได้ระบุขนาดกระบวนการสูงสุด มันวางข้อ จำกัด ในด้านหนึ่งของการใช้หน่วยความจำของ Squid Squid ใช้หน่วยความจำสำหรับสิ่งอื่นเช่นกัน กระบวนการของคุณอาจจะใหญ่เป็นสองเท่าหรือสามเท่าของมูลค่าที่คุณใส่ไว้ที่นี่

มันระบุจำนวนหน่วยความจำในอุดมคติที่จะใช้สำหรับ:

  • วัตถุระหว่างทาง
  • วัตถุที่น่าสนใจ
  • วัตถุที่แคชลบ

ข้อมูลสำหรับวัตถุเหล่านี้จะถูกเก็บไว้ในบล็อก 4 kB พารามิเตอร์นี้ระบุขีด จำกัด สูงสุดในอุดมคติของขนาดทั้งหมดของบล็อก 4 kB ที่จัดสรร วัตถุระหว่างทางมีความสำคัญสูงสุด

วัตถุระหว่างทางมีลำดับความสำคัญเหนือวัตถุอื่น เมื่อต้องการพื้นที่เพิ่มเติมสำหรับข้อมูลที่เข้ามาจะมีการลบแคชและวัตถุร้อน กล่าวอีกนัยหนึ่งวัตถุที่เป็นลบและแคชร้อนจะเติมเต็มพื้นที่ใด ๆ ที่ไม่ได้ใช้ซึ่งไม่จำเป็นสำหรับวัตถุระหว่างทาง

หากสถานการณ์จำเป็นต้องใช้ขีด จำกัด นี้จะเกิน หากอัตราการร้องขอขาเข้าของคุณต้องการหน่วยความจำมากกว่านี้เพื่อเก็บออบเจ็กต์ระหว่างทาง Squid จะเกินขีด จำกัด นี้เพื่อตอบสนองคำขอใหม่ เมื่อโหลดลดลงบล็อกจะถูกปล่อยให้เป็นอิสระจนกว่าจะถึงเครื่องหมายน้ำสูง หลังจากนั้นบล็อกจะถูกใช้เพื่อเก็บวัตถุร้อน

ขนาดแคช FQDN : เริ่มต้นที่ 1024 จำนวนสูงสุดของรายการแคชชื่อโดเมนที่ผ่านการรับรองโดยสมบูรณ์

เครื่องหมายหน่วยความจำสูง : เริ่มต้นที่ 95% เมื่อถึงเครื่องหมายสูงวัตถุร้อนและในวัตถุการขนส่งจะถูกปล่อยออกสู่หน่วยความจำฟรี

เครื่องหมายหน่วยความจำต่ำ : เริ่มต้นที่ 90% หลังจากถึงเครื่องหมายน้ำสูง Squid จะเพิ่มหน่วยความจำจนกว่าการใช้งานจะลดลงต่ำกว่าเครื่องหมายนี้

เครื่องหมายดิสก์สูงน้ำ : ค่าเริ่มต้นถึง 95% เมื่อถึงเครื่องหมายสูงนี้ Squid จะเริ่มล้างข้อมูลในดิสก์อย่างจริงจัง

แผ่นดิสก์ที่มีน้ำน้อย : เริ่มต้นที่ 90% หลังจากถึงเครื่องหมายน้ำสูงแล้ว Squid จะยังคงล้างข้อมูลดิสก์ต่อไปจนกว่าการใช้งานจะลดลงต่ำกว่าเครื่องหมายนี้

ขนาดวัตถุที่แคชสูงสุด : ค่าเริ่มต้นคือ 4096 kB การตั้งค่านี้ จำกัด ขนาดของวัตถุที่จะถูกเก็บไว้ในแคช สิ่งใดที่มีขนาดเกินขนาดนี้จะไม่ถูกแคชและจะต้องโหลดจากเซิร์ฟเวอร์ต้นทางทุกครั้ง

ขนาดแคชของที่อยู่ IP : ค่าเริ่มต้นคือ 1024 จำนวนรายการที่จะเก็บไว้ในแคช IP

เครื่องหมาย IP ระดับสูงของแคช : ค่าเริ่มต้นเป็น 95% จุดที่ Squid เริ่มล้างรายการแคช IP

เครื่องหมาย IP ต่ำ - ต่ำของแคช : ค่าเริ่มต้นเป็น 90% หลังจากการล้างแคช IP เริ่มขึ้นจะดำเนินต่อไปจนกว่าจำนวนรายการในแคช IP จะลดลงต่ำกว่าเปอร์เซ็นต์นี้
