Editar pool de atraso

Número do pool
Este é um número começando com 1, escolhido por você pelo Webmin. Ele simplesmente rotula o pool para que mais de um pool possa ser configurado e tratado de forma diferente pelo Squid.

Classe de piscina
Isso define o tipo de classe do pool. Cada pool de atrasos pode ter uma classe selecionada das classes 1, 2 ou 3. As classes têm os seguintes significados: "1-Agregado", "2-Agregado e Individual", "3-Agregado, Individual e Rede". A mais simples, classe 1, especifica uma largura de banda que todos os clientes aos quais esse pool pertence não podem superar. A Classe 2 especifica uma largura de banda agregada e uma largura de banda que cada indivíduo pode usar, criando uma hierarquia de limites em que todos os indivíduos combinados devem se encaixar no agregado, enquanto cada indivíduo é limitado pelo limite individual. Por fim, a Classe 3 especifica uma largura de banda agregada e individual, além de uma largura de banda de rede que se aplica a uma sub-rede inteira. A classe 3 geralmente é útil apenas em redes muito grandes que possuem mais de uma sub-rede.

Os limites podem ser especificados em qualquer unidade preferida, embora sejam salvos no arquivo de configuração do Squid em bytes. O primeiro campo é a taxa de transferência média que será alocada para o pool, enquanto o segundo é o tamanho do balde usado, que corresponde aproximadamente a uma largura de banda máxima que pode ser estourada.

Depois de criar um pool pela primeira vez, uma nova seção aparecerá na parte inferior da página, permitindo a seleção de quais ACLs o pool se aplicará. As ACLs são criadas na página Controle de Acesso e qualquer ACL pode ser usada para selecionar a quais conexões um pool de atrasos se aplicará. Permitir que uma ACL aplique o conjunto de atrasos às conexões correspondentes, enquanto Negar uma ACL fará com que as conexões correspondentes ignorem o conjunto de atrasos. Assim, a menos que todas as conexões terão o mesmo atraso piscina aplicada a eles, você vai querer adicionar duas ACLs, o primeiro será o ACL você deseja aplicar a piscina atraso para, ea segunda será a todos ACL assim que apenas seus clientes selecionados serão restritos.

Não se esqueça de aplicar as alterações depois de criar um novo pool e adicionar ACLs para corresponder.
