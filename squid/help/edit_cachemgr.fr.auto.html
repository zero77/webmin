Mots de passe du gestionnaire de cache
Par défaut, l'interface Web du gestionnaire de cache de Squid ne nécessite pas d'authentification pour accéder à ses fonctions. Cette page vous permet de définir un ou plusieurs mots de passe qui s'appliquent à certaines ou à toutes les actions dans le gestionnaire de cache.

Pour spécifier un mot de passe unique pour toutes les actions, sélectionnez simplement Mots de passe et actions spécifiés ci - dessous , puis dans la colonne Mot de passe , sélectionnez Définir sur et entrez votre mot de passe dans la case adjacente. Cochez ensuite la case Utiliser ce mot de passe pour toutes les actions , puis cliquez sur Enregistrer .
