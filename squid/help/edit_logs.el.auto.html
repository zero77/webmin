Ξύλευση
Αρχείο καταγραφής πρόσβασης : Η προεπιλογή εξαρτάται από το σύστημα, συνήθως κάπου στο / var . Καταγράφει τη δραστηριότητα αιτήματος πελάτη. Περιέχει μια καταχώριση για κάθε ληφθέν αίτημα HTTP και ICP.

Αρχείο καταγραφής εντοπισμού σφαλμάτων : Η προεπιλογή εξαρτάται από το σύστημα. Αρχείο καταγραφής προσωρινής μνήμης. Εδώ πηγαίνουν γενικές πληροφορίες σχετικά με τη συμπεριφορά της προσωρινής μνήμης. Μπορείτε να αυξήσετε τον όγκο των δεδομένων που καταγράφονται σε αυτό το αρχείο με τις επιλογές εντοπισμού σφαλμάτων .

Αρχείο καταγραφής αποθήκευσης : Η προεπιλογή εξαρτάται από το σύστημα. Καταγράφει τις δραστηριότητες του διαχειριστή αποθήκευσης. Δείχνει ποια αντικείμενα εξάγονται από την προσωρινή μνήμη και ποια αντικείμενα αποθηκεύονται και για πόσο καιρό. Για απενεργοποίηση, εισαγάγετε "κανένα". Δεν υπάρχουν πραγματικά βοηθητικά προγράμματα για την ανάλυση αυτών των δεδομένων, ώστε να μπορείτε να τα απενεργοποιήσετε με ασφάλεια.

Αρχείο μεταδεδομένων προσωρινής μνήμης : Η προεπιλογή είναι swap.log σε κάθε κατάλογο κρυφής μνήμης. Αυτό το αρχείο καταγραφής διατηρεί τα μεταδεδομένα των αντικειμένων που είναι αποθηκευμένα στο δίσκο. Χρησιμοποιείται για την αναδημιουργία της προσωρινής μνήμης κατά την εκκίνηση. Κανονικά αυτό το αρχείο βρίσκεται σε κάθε κατάλογο κρυφής μνήμης, αλλά μπορείτε να καθορίσετε ένα εναλλακτικό όνομα διαδρομής εδώ. Σημειώστε ότι πρέπει να δώσετε ένα πλήρες όνομα αρχείου, όχι μόνο έναν κατάλογο. Δεδομένου ότι αυτό είναι το ευρετήριο για ολόκληρη τη λίστα αντικειμένων, ΔΕΝ ΜΠΟΡΕΙ να το περιστρέφετε περιοδικά!

Εάν το% s μπορεί να χρησιμοποιηθεί στο όνομα του αρχείου, τότε θα αντικατασταθεί με μια αναπαράσταση του ονόματος καταλόγου της προσωρινής μνήμης όπου κάθε / αντικαθίσταται με το '.'. Αυτό απαιτείται για να επιτρέπεται η προσθήκη / αφαίρεση γραμμών καταλόγου προσωρινής μνήμης όταν χρησιμοποιείται ένα εναλλακτικό αρχείο καταγραφής ανταλλαγής προσωρινής μνήμης.

Εάν υπάρχουν περισσότεροι από ένας κατάλογοι προσωρινής μνήμης και το% s δεν χρησιμοποιείται στο όνομα, τότε αυτά τα αρχεία καταγραφής ανταλλαγής θα έχουν ονόματα όπως:

  • cache_swap_log.00
  • cache_swap_log.01
  • cache_swap_log.02

Η αριθμημένη επέκταση (η οποία προστίθεται αυτόματα) αντιστοιχεί στη σειρά των γραμμών καταλόγου cache στη διαμόρφωση. Εάν αλλάξετε τη σειρά των γραμμών καταλόγου προσωρινής μνήμης σε αυτό το αρχείο, τότε αυτά τα αρχεία καταγραφής ΔΕΝ θα αντιστοιχούν στη σωστή καταχώριση καταλόγου προσωρινής μνήμης (εκτός εάν τα μετονομάσετε χειροκίνητα). Σας συνιστούμε να ΜΗ χρησιμοποιείτε αυτήν την επιλογή. Είναι καλύτερα να διατηρείτε αυτά τα αρχεία καταγραφής σε κάθε κατάλογο κρυφής μνήμης.

Χρησιμοποιήστε τη μορφή αρχείου καταγραφής HTTPD : Προεπιλογές στο αριθ. Η προσωρινή μνήμη μπορεί να μιμηθεί τη μορφή αρχείου καταγραφής που χρησιμοποιούν πολλά προγράμματα «httpd». Για να απενεργοποιήσετε / ενεργοποιήσετε αυτήν την εξομοίωση, ορίστε αυτήν την τιμή σε απενεργοποίηση ή ενεργοποίηση. Η προεπιλογή είναι να χρησιμοποιήσετε τη μορφή εγγενούς καταγραφής δεδομένου ότι περιλαμβάνει χρήσιμες πληροφορίες που χρησιμοποιούν οι αναλυτές καταγραφής συγκεκριμένων καλαμαριών.

Καταγραφή κεφαλίδων MIME : Προεπιλογές στο αριθ. Η προσωρινή μνήμη μπορεί να καταγράψει τόσο το αίτημα όσο και τις κεφαλίδες MIME απόκρισης για κάθε συναλλαγή HTTP. Οι κεφαλίδες κωδικοποιούνται με ασφάλεια και θα εμφανίζονται ως δύο πεδία παρενθέσεων στο τέλος του αρχείου καταγραφής πρόσβασης (είτε για τις εγγενείς είτε για τις μορφές αρχείου καταγραφής που είναι εξομοιωμένες με httpd). Για να ενεργοποιήσετε αυτήν την καταγραφή, ενεργοποιήστε την.

Αρχείο καταγραφής παράγοντα χρήστη : Η προεπιλογή δεν είναι καμία. Εάν διαμορφωθεί με την επιλογή διαμόρφωσης "--enable-useragent-log", το Squid θα γράψει το πεδίο User-Agent από αιτήσεις HTTP στο όνομα αρχείου που καθορίζεται εδώ.

Αρχείο PID : Η προεπιλογή εξαρτάται από το σύστημα. Ένα όνομα αρχείου για να γράψετε το αναγνωριστικό διαδικασίας. Για απενεργοποίηση, εισαγάγετε "κανένα".

Εκτελέστε αναζήτηση RFC931 ident για ACL : Η προεπιλογή δεν είναι καμία. Εκτελέστε αναζήτηση ταυτότητας για να βρείτε το όνομα χρήστη των πελατών που έχουν πρόσβαση σε οτιδήποτε υπάρχει σε αυτά τα ACL.

Χρονικό όριο ταυτότητας RFC931 : Η προεπιλογή είναι 30 δευτερόλεπτα. Μέγιστος χρόνος αναμονής για αιτήματα IDENT. Εάν αυτό είναι πολύ υψηλό και έχετε ενεργοποιήσει τυχόν αναζητήσεις ταυτότητας, τότε ενδέχεται να είστε ευάλωτοι σε άρνηση υπηρεσίας, έχοντας ταυτόχρονα πολλά αιτήματα ταυτότητας.

Καταγράψτε τα πλήρη ονόματα κεντρικών υπολογιστών : Η προεπιλογή είναι όχι. Καταγράφει το πλήρες όνομα κεντρικού υπολογιστή των πελατών.

Netmask καταγραφής : Η προεπιλογή είναι 255.255.255.255. Ένα netmask για διευθύνσεις πελατών σε αρχεία log και έξοδο cachemgr. Αλλάξτε αυτό για να προστατεύσετε το απόρρητο των πελατών cache. Ένα netmask 255.255.255.0 θα καταγράψει όλες τις IP σε αυτό το εύρος με το τελευταίο ψηφίο να έχει οριστεί στο "0".

Επιλογές εντοπισμού σφαλμάτων : Η προεπιλογή είναι "ALL, 1". Οι επιλογές καταγραφής ορίζονται ως ενότητα, επίπεδο όπου κάθε αρχείο προέλευσης έχει μια μοναδική ενότητα. Τα χαμηλότερα επίπεδα έχουν ως αποτέλεσμα λιγότερα Πλήρης εντοπισμός σφαλμάτων (επίπεδο 9) μπορεί να οδηγήσει σε ένα πολύ μεγάλο αρχείο καταγραφής, γι 'αυτό να είστε προσεκτικοί. Η μαγική λέξη "ALL" ορίζει επίπεδα εντοπισμού σφαλμάτων για όλες τις ενότητες. Συνιστούμε να τρέχετε κανονικά με το "ALL, 1".

Πίνακας κεφαλίδων MIME : Όνομα διαδρομής στον πίνακα MIME του Squid. Δεν πρέπει να αλλάξετε, αλλά το προεπιλεγμένο αρχείο περιέχει παραδείγματα και πληροφορίες μορφοποίησης εάν το κάνετε.
