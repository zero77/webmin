Διάφορες επιλογές
Διευθύνσεις δοκιμής εκκίνησης DNS : Διακομιστές DNS που ο Squid θα υποβάλει ερώτημα για να ελέγξει εάν οι αναζητήσεις DNS λειτουργούν σωστά.

Περιστροφές αρχείου καταγραφής SIGUSR1 : Από προεπιλογή έως 10 ή 0, ανάλογα με το σύστημα. Καθορίζει τον αριθμό των περιστροφών αρχείων καταγραφής που πρέπει να κάνετε όταν πληκτρολογείτε "squid -k rotate". Η προεπιλογή είναι 10, η οποία θα περιστραφεί με επεκτάσεις 0 έως 9. Η ρύθμιση logfile_rotate σε 0 θα απενεργοποιήσει την περιστροφή, αλλά τα αρχεία καταγραφής εξακολουθούν να είναι κλειστά και να ανοίγουν ξανά. Αυτό θα σας επιτρέψει να μετονομάσετε τα αρχεία καταγραφής μόνοι σας πριν από την αποστολή του σήματος περιστροφής.

Σημειώστε ότι η εντολή "squid -k rotate" στέλνει κανονικά ένα σήμα USR1 στην τρέχουσα διαδικασία καλαμαριού. Σε ορισμένες καταστάσεις (π.χ. σε Linux με Async I / O), το USR1 χρησιμοποιείται για άλλους σκοπούς, έτσι το -k rotate χρησιμοποιεί άλλο σήμα. Είναι καλύτερο να συνηθίσετε να χρησιμοποιείτε το «squid -k rotate» αντί του «kill -USR1» ".

Κείμενο μηνυμάτων σφάλματος : Κείμενο HTML για συμπερίληψη στα μηνύματα σφάλματος. Ορίστε αυτήν τη διεύθυνση URL "mailto" στη διεύθυνση διαχειριστή σας ή ίσως απλά έναν σύνδεσμο προς τη σελίδα του οργανισμού σας.

Για να το συμπεριλάβετε στα μηνύματα σφάλματος, πρέπει να ξαναγράψετε τα αρχεία προτύπων σφαλμάτων (βρίσκονται στον κατάλογο "σφάλματα"). Όπου θέλετε να εμφανίζεται η γραμμή "err_html_text", εισαγάγετε μια ετικέτα% L στο αρχείο προτύπου σφάλματος.

Στατιστικά στοιχεία ανά πελάτη : Εάν θέλετε να απενεργοποιήσετε τη συλλογή στατιστικών στοιχείων ανά πελάτη, απενεργοποιήστε το πρόγραμμα πελάτη_db εδώ.

X-Forwarded-For header : Η προεπιλογή είναι 'on'. Εάν οριστεί, το Squid θα συμπεριλάβει τη διεύθυνση IP ή το όνομα του συστήματός σας στα αιτήματα HTTP που το προωθεί. Από προεπιλογή, φαίνεται ότι το HTTP το ζητάει προς τα εμπρός. Από προεπιλογή μοιάζει με αυτό:

X-Forwarded-For: 192.1.2.3

Εάν το απενεργοποιήσετε, θα εμφανίζεται ως:

X-Forwarded-For: άγνωστο

Καταγραφή ερωτημάτων ICP : Η προεπιλογή είναι "on". Εάν οριστεί, τα ερωτήματα ICP καταγράφονται στο access.log. Μπορείτε να απενεργοποιήσετε αυτό εάν το φορτίο ICP σας είναι πολύ υψηλό για να επιταχύνετε τα πράγματα ή για να απλοποιήσετε την ανάλυση καταγραφής.

Ελάχιστο άμεσο λυκίσκο : Η προεπιλογή είναι 4. Εάν χρησιμοποιείτε το υλικό ping ping ICMP, κάντε άμεσες ανακτήσεις για ιστότοπους που δεν είναι περισσότερο από αυτό.

Διατηρήστε τη μνήμη για μελλοντική χρήση : Η προεπιλογή είναι «ναι». Εάν ρυθμιστεί, το Squid θα διατηρεί διαθέσιμες δεξαμενές εκχωρημένης (αλλά μη χρησιμοποιημένης) μνήμης για μελλοντική χρήση. Εάν η μνήμη είναι εξαιρετική στο σύστημά σας και πιστεύετε ότι η βιβλιοθήκη malloc σας ξεπερνά τις ρουτίνες του Squid, απενεργοποιήστε το.

Ποσότητα μνήμης που πρέπει να διατηρηθεί : Η προεπιλογή είναι "Απεριόριστη". Εάν οριστεί σε μη μηδενική τιμή, το Squid θα διατηρήσει το μέγιστο καθορισμένο όριο της εκχωρημένης (αλλά μη χρησιμοποιημένης) μνήμης στις ομάδες μνήμης. Όλα τα δωρεάν αιτήματα () που υπερβαίνουν αυτό το όριο θα αντιμετωπιστούν από τη βιβλιοθήκη malloc σας. Το Squid δεν εκχωρεί εκ των προτέρων καμία μνήμη, αλλά διατηρεί αντικείμενα που διαφορετικά θα ήταν ελεύθερα () d. Επομένως, είναι ασφαλές να ρυθμίσετε το memory_pools_limit σε μια αρκετά υψηλή τιμή, ακόμη και αν η διαμόρφωσή σας θα χρησιμοποιεί λιγότερη μνήμη.

Εάν δεν οριστεί (προεπιλογή) ή οριστεί στο μηδέν, το Squid θα διατηρήσει όλη τη μνήμη που μπορεί. Δηλαδή, δεν θα υπάρχει όριο στη συνολική ποσότητα μνήμης που χρησιμοποιείται για την ασφαλή φύλαξη.

Κεφαλίδες για να περάσετε : Η προεπιλογή είναι "Όλες οι κεφαλίδες". Αυτή η επιλογή αντικαθιστά την παλιά "http_anonymizer" επιλογή με κάτι που είναι πολύ πιο διαμορφώσιμο. Μπορείτε τώρα να καθορίσετε ακριβώς ποιες κεφαλίδες θα επιτρέπονται ή ποιες θα πρέπει να καταργηθούν από εξερχόμενα αιτήματα.

Υπάρχουν δύο μέθοδοι για τη χρήση αυτής της επιλογής. Μπορείτε είτε να επιτρέψετε συγκεκριμένες κεφαλίδες (με αυτόν τον τρόπο να αρνηθείτε όλες τις άλλες), είτε να αρνηθείτε συγκεκριμένες κεφαλίδες (επιτρέποντας έτσι σε όλες τις άλλες).

Fake User-Agent : Η προεπιλογή είναι «κανένας». Εάν φιλτράρετε την κεφαλίδα User-Agent με την επιλογή Headers να περάσει , μπορεί να προκαλέσει ορισμένους διακομιστές Web να απορρίψουν το αίτημά σας. Χρησιμοποιήστε αυτό για να ψεύσετε ένα.

Διεύθυνση δρομολογητή WCCP : Χρησιμοποιήστε αυτήν την επιλογή για να ορίσετε τον δρομολογητή "οικιακού" WCCP για το Squid.

Εισερχόμενη διεύθυνση WCCP : Χρησιμοποιήστε αυτήν την επιλογή εάν απαιτείται η λήψη μηνυμάτων WCCP σε μία μόνο διεπαφή. ΜΗΝ χρησιμοποιείτε αυτήν την επιλογή εάν δεν είστε βέβαιοι πόσες διεπαφές έχετε ή αν γνωρίζετε ότι έχετε μόνο μία διεπαφή.

Εξερχόμενη διεύθυνση WCCP : Χρησιμοποιήστε αυτήν την επιλογή εάν απαιτείται η αποστολή μηνυμάτων WCCP σε μία μόνο διεπαφή. ΜΗΝ χρησιμοποιείτε αυτήν την επιλογή εάν δεν είστε βέβαιοι πόσες διεπαφές έχετε ή αν γνωρίζετε ότι έχετε μόνο μία διεπαφή.
