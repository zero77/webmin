PPTP VPN Client
Mit diesem Modul können Sie mithilfe des PPTP-Protokolls VPN-Verbindungen zu anderen Servern herstellen. Das Modul verwendet das Standard-Linux-PPTP-Client-Programm und den PPP-Daemon. Auf Remoteservern muss ein PPTP-Daemon ausgeführt werden, der mit dem PPTP-VPN-Servermodul von Webmin konfiguriert werden kann.

Es können mehrere Tunnel definiert werden, von denen jeder einen Remote-Server zum Herstellen einer Verbindung, einen Anmeldenamen und ein Kennwort haben muss. Einem Tunnel können auch mehrere statische Routen zugeordnet sein, die beim Verbinden aufgerufen werden. Standardmäßig wird bei Aktivierung nur eine Route zum Server am anderen Ende des Tunnels erstellt.

Sobald ein Tunnel erstellt wurde, kann er über die Schaltfläche Verbinden mit unten auf der Hauptseite aktiviert werden. Es können mehrere Tunnel gleichzeitig aktiv sein, und diejenigen, die aktiv sind, können mit der Schaltfläche Trennen von heruntergefahren werden.

Auf der Hauptseite befindet sich auch die Schaltfläche " Globale PPP-Optionen bearbeiten" zum Bearbeiten von Einstellungen, die für alle Tunnel gelten. Die wichtigsten sind diejenigen, die sich auf MPPE beziehen, ein Verschlüsselungsprotokoll, das von Microsoft VPN-Servern zum Sichern von PPTP-Verbindungen verwendet wird. Damit MPPE funktioniert, ist jedoch Unterstützung sowohl im PPP-Daemon als auch im Kernel erforderlich. PPPd-Versionen 2.4.2 und höher unterstützen MPPE nativ, und auch für ältere Versionen gibt es einen Patch.
