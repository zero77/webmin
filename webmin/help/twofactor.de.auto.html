Zwei-Faktor-Authentifizierung
Die Zwei-Faktor-Authentifizierung kann verwendet werden, um einem Webmin-Konto eine zusätzliche Sicherheitsebene sowie den regulären Benutzernamen und das Kennwort hinzuzufügen. Wenn diese Option aktiviert ist, muss ein Benutzer ein zusätzliches Authentifizierungstoken bereitstellen, das normalerweise von einer Smartphone-App, einem Hardware-OTP-Generator oder einer SMS stammt.

Auf dieser Seite können Sie einen Authentifizierungsanbieter für Webmin-Benutzer auf Ihrem System auswählen. Nach der Aktivierung kann sich jeder Benutzer für zwei Faktoren im Webmin-Benutzermodul registrieren. Die verfügbaren Zwei-Faktor-Anbieter sind:

Authy
Dies ist ein kommerzieller Dienst, der eine Smartphone-App und eine Website für die Verwaltung von Zwei-Faktor-Benutzern bereitstellt. Jeder Benutzer muss ein Konto auf der Website von Authy erstellen und es mit seinem Smartphone verknüpfen.

Google Authenticator
Dies ist eine Smartphone-App, die das Standard-TOPT-Protokoll implementiert. Jeder Benutzer muss einen QR-Code mit der App scannen, um seine Token mit dem Webmin-Server zu verknüpfen.
