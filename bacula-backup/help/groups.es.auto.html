Grupos Bacula
Esta página enumera los grupos de hosts, cada uno de los cuales es un conjunto de hosts que pueden ser respaldados simultáneamente por un solo trabajo de copia de seguridad del grupo Bacula . Los hosts en un grupo se toman de una base de datos del grupo de nodos del Administrador OCM o del módulo Índice de servidores Webmin, según la configuración de este módulo.

Para que las copias de seguridad del grupo bacula funcionen, cada una debe tener instalado al menos el demonio del archivo Bacula y todos los hosts del grupo deben tener la misma contraseña para este director.
