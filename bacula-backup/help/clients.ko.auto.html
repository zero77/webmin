백업 클라이언트
Bacula 클라이언트는 파일을 백업 할 수있는 시스템입니다. 파일을 백업하려는 모든 클라이언트가이 페이지에 나열되어 있고 각 클라이언트에서 Bacula 파일 데몬이 실행 중이어야합니다. 일반적인 단일 시스템 설정에서이 서버에 나열해야합니다.

원격 클라이언트를 추가 할 때 호스트 이름 외에도 해당 Bacula 파일 데몬의 이름과 데몬의 암호도 알아야합니다. 이들은 클라이언트 시스템의 /etc/bacula/bacula-fd.conf 파일에 설정되어 있습니다.

표시된 경우 클라이언트 페이지의 TLS 옵션은이 디렉터가 클라이언트와 통신 할 때 암호화를 사용할지 여부를 결정합니다. TLS를 사용하려면 SSL 인증서 및 키를 생성하고 해당 경로를이 양식의 해당 필드에 입력해야합니다.
