Sicherungsjobs der Bacula-Gruppe
Diese Seite listet Sicherungsjobs auf, die so eingerichtet sind, dass sie auf allen Hosts in einer Bacula-Gruppe gleichzeitig ausgeführt werden. Sie haben dieselben Einstellungen wie Standard-Sicherungsjobs, außer dass Sie keine Standardjobs definieren können, um Einstellungen zwischen Jobs zu teilen. Anstatt einen zu sichernden Client auszuwählen, müssen Sie stattdessen eine zuvor definierte Bacula-Gruppe auswählen.
