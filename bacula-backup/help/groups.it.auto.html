Gruppi Bacula
Questa pagina elenca i gruppi di host, ognuno dei quali è un insieme di host che possono essere contemporaneamente supportati da un singolo processo di backup del gruppo Bacula . Gli host di un gruppo sono presi da un database del gruppo di nodi di OCM Manager o dal modulo Indice server Webmin, a seconda della configurazione di questo modulo.

Perché i backup del gruppo bacula funzionino, ognuno deve avere almeno il demone del file Bacula installato e tutti gli host del gruppo devono avere la stessa password per questo direttore.
