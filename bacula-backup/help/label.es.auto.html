Volumen de etiqueta
El etiquetado marca un volumen (como una cinta cargada o un directorio de respaldo) como parte de un grupo de volúmenes seleccionado y, por lo tanto, utilizable para respaldos. Debe etiquetar al menos un volumen antes de que las copias de seguridad realizadas a través de Bacula funcionen; si no lo hace, la copia de seguridad se detendrá hasta que se etiquete un volumen adecuado.

Para las copias de seguridad de archivos, el nombre del volumen determina el nombre del archivo en el directorio de destino en el que realmente se escribirá. Para las cintas, el nombre será la etiqueta de la cinta.
