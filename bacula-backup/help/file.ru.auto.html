Конфигурация File Daemon
Эта форма может быть использована для настройки параметров демона файла Bacula, работающего в системе. Значения по умолчанию, как правило, будут правильными, хотя вы можете настроить имя демона и количество одновременных заданий. Если ваша система Bacula поддерживает безопасность TLS, параметры TLS на этой странице можно использовать для защиты связи между файловым демоном, директором и демонами хранилища.
