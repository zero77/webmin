Volume d'étiquette
L'étiquetage marque un volume (tel qu'une bande ou un répertoire de sauvegarde chargé) comme faisant partie d'un pool de volumes sélectionné, et donc utilisable pour les sauvegardes. Vous devez étiqueter au moins un volume avant que les sauvegardes effectuées via Bacula fonctionnent - si vous ne le faites pas, la sauvegarde s'arrêtera jusqu'à ce qu'un volume approprié soit étiqueté.

Pour les sauvegardes dans des fichiers, le nom du volume détermine le nom du fichier dans le répertoire de destination qui sera réellement écrit. Pour les bandes, le nom sera l'étiquette de la bande.
