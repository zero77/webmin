Seturi de fișiere
Un set de fișiere este o listă de fișiere și directoare pe care Bacula le poate face backup ca parte a unei lucrări. Fiecare set trebuie să aibă un nume unic, o listă de fișiere pe care să le includă și o listă opțională de fișiere pe care să le excludă. Această din urmă caracteristică poate fi utilă pentru a omite fișierele sau directoarele care nu sunt critice de sub unul dintre directoarele care trebuie excluse.
