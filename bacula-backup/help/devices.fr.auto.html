Périphériques de stockage
Contrairement à toutes les autres pages de ce module, celle-ci configure le démon de stockage Bacula plutôt que le directeur. Il vous permet de contrôler les unités de bande et les répertoires utilisés pour les sauvegardes. Chaque entrée de la liste définit un périphérique de stockage, dont chacun doit avoir un nom unique, un fichier ou un répertoire de périphérique (comme /dev/st0 ou /backup ) et un nom de type de support.

Les périphériques définis ici peuvent être référencés sur la page Storage Daemons , qui peut à son tour être utilisée dans des travaux de sauvegarde. Si vous souhaitez créer un nouveau répertoire dans lequel sauvegarder, il doit être ajouté ici et à la liste des démons. De plus, tout nouveau répertoire doit être étiqueté en premier (en utilisant la page Label Volume ) avant que Bacula ne puisse y écrire.

Sachez que les modifications apportées à cette page ne seront activées que lorsque le bouton Redémarrer Bacula sera cliqué sur la page principale du module.
