Storage Daemon Configuration
Dette skjemaet kan brukes til å justere innstillingene for Bacula-lagringsdemonen som kjører på systemet. Standardverdiene vil vanligvis være riktige, selv om du kan justere daemonnavnet og antall samtidige jobber. Hvis Bacula-systemet ditt støtter TLS-sikkerhet, kan TLS-alternativene på denne siden brukes til å sikre kommunikasjon mellom lagrings- og fildemoner.
