Diretores de Daemon de Arquivos
Esta página lista todos os diretores que têm permissão para se conectar a este daemon de arquivo. Se você estiver adicionando este daemon de arquivo a um diretor remoto, a senha na entrada do Cliente de Backup deverá corresponder à senha listada aqui.

Se o TLS estiver ativado para um diretor, a comunicação entre esse diretor e este daemon de arquivo será criptografada e verificada usando o certificado e a chave selecionados.
