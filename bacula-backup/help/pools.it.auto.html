Pool di volumi
Nel contesto di Bacula, un pool è un insieme di volumi dedicati a qualche scopo. Un volume è in genere un nastro o una directory di destinazione specifici, ognuno dei quali è etichettato con un nome univoco. Quando si crea un processo di backup, è possibile selezionare quale pool verrà utilizzato per i file di destinazione e quindi quali volumi.

Un volume può essere aggiunto a un pool utilizzando la pagina Volume etichetta , che contrassegna un nastro inserito o una directory di destinazione con un nome univoco.
