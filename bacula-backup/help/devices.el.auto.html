Συσκευές αποθήκευσης
Σε αντίθεση με όλες τις άλλες σελίδες σε αυτήν την ενότητα, αυτή διαμορφώνει τον δαίμονα αποθήκευσης Bacula και όχι τον σκηνοθέτη. Σας επιτρέπει να ελέγχετε ποιες συσκευές και καταλόγους ταινιών χρησιμοποιούνται για αντίγραφα ασφαλείας. Κάθε καταχώριση στη λίστα ορίζει μια συσκευή αποθήκευσης, καθεμία από τις οποίες πρέπει να έχει ένα μοναδικό όνομα, ένα αρχείο ή κατάλογο συσκευής (όπως /dev/st0 ή /backup ) και ένα όνομα τύπου μέσου.

Οι συσκευές που ορίζονται εδώ μπορούν να αναφερθούν στη σελίδα Storage Daemons , η οποία με τη σειρά της μπορεί να χρησιμοποιηθεί σε εργασίες δημιουργίας αντιγράφων ασφαλείας. Εάν θέλετε να δημιουργήσετε έναν νέο κατάλογο για δημιουργία αντιγράφων ασφαλείας, πρέπει να προστεθεί τόσο εδώ όσο και στη λίστα δαιμόνων. Επιπλέον, οποιοσδήποτε νέος κατάλογος πρέπει να επισημανθεί πρώτα (χρησιμοποιώντας τη σελίδα Όγκος ετικέτας ) προτού ο Bacula γράψει σε αυτόν.

Λάβετε υπόψη ότι οι αλλαγές που έγιναν σε αυτήν τη σελίδα θα ενεργοποιηθούν μόνο όταν κάνετε κλικ στο κουμπί Επανεκκίνηση Bacula στην κύρια σελίδα της μονάδας.
