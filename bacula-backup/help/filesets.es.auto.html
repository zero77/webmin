Conjuntos de archivos
Un conjunto de archivos es una lista de archivos y directorios que Bacula puede respaldar como parte de un trabajo. Cada conjunto debe tener un nombre único, una lista de archivos para incluir y una lista opcional de archivos para excluir. Esta última característica puede ser útil para omitir archivos o directorios no críticos en uno de los directorios a excluir.
