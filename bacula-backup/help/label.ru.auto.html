Объем метки
Маркировка помечает том (например, загруженную ленту или каталог резервного копирования) как часть выбранного пула томов и, таким образом, может использоваться для резервного копирования. Вы должны пометить как минимум один том, прежде чем будут работать любые резервные копии, выполненные с помощью Bacula - в противном случае резервное копирование будет остановлено, пока не будет помечен подходящий том.

Для резервных копий файлов имя тома определяет имя файла в целевом каталоге, в который будет записана запись. Для лент название будет меткой ленты.
