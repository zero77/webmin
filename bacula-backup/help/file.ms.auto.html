Fail Konfigurasi Daemon
Bentuk ini dapat digunakan untuk menyesuaikan pengaturan untuk daemon file Bacula yang berjalan di sistem. Lalai biasanya akan betul, walaupun anda dapat menyesuaikan nama daemon dan jumlah pekerjaan serentak. Sekiranya sistem Bacula anda menyokong keselamatan TLS, pilihan TLS di halaman ini dapat digunakan untuk melindungi komunikasi antara daemon file, pengarah dan daemon penyimpanan.
