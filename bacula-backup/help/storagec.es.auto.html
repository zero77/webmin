Configuración de almacenamiento de daemon
Este formulario se puede utilizar para ajustar la configuración del demonio de almacenamiento Bacula que se ejecuta en el sistema. Los valores predeterminados generalmente serán correctos, aunque puede ajustar el nombre del daemon y la cantidad de trabajos simultáneos. Si su sistema Bacula es compatible con la seguridad TLS, las opciones TLS en esta página se pueden usar para asegurar la comunicación entre el almacenamiento y los demonios de archivos.
