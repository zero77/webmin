File Daemon Directors
Esta página enumera todos los directores que pueden conectarse a este demonio de archivo. Si está agregando este demonio de archivo a un director remoto, la contraseña en su entrada de Cliente de respaldo debe coincidir con la contraseña que se enumera aquí.

Si TLS está habilitado para un director, la comunicación entre ese director y este demonio de archivo se cifrará y verificará utilizando el certificado y la clave seleccionados.
