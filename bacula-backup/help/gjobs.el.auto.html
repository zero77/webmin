Bacula Group Backup Jobs
Αυτή η σελίδα παραθέτει εφεδρικές εργασίες που έχουν ρυθμιστεί για εκτέλεση σε όλους τους κεντρικούς υπολογιστές σε μια ομάδα Bacula ταυτόχρονα. Έχουν τις ίδιες ρυθμίσεις με τις τυπικές εργασίες δημιουργίας αντιγράφων ασφαλείας, εκτός από το ότι δεν μπορείτε να ορίσετε προεπιλεγμένες εργασίες για κοινή χρήση ρυθμίσεων μεταξύ εργασιών. Επίσης, αντί να επιλέξετε έναν πελάτη για δημιουργία αντιγράφων ασφαλείας, θα πρέπει αντί να επιλέξετε μια ομάδα Bacula που έχει οριστεί προηγουμένως.
