Trabalhos de backup do Bacula Group
Esta página lista as tarefas de backup configuradas para execução em todos os hosts em um grupo Bacula de uma só vez. Eles têm as mesmas configurações dos trabalhos de backup padrão, exceto que você não pode definir trabalhos padrão para compartilhar configurações entre os trabalhos. Além disso, em vez de selecionar um cliente para backup, você deve selecionar um grupo Bacula definido anteriormente.
