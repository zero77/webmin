Наборы файлов
Набор файлов - это список файлов и каталогов, которые Bacula может создать в качестве части задания. Каждый набор должен иметь уникальное имя, список включаемых файлов и необязательный список исключаемых файлов. Эта последняя функция может быть полезна для пропуска некритических файлов или каталогов в одном из каталогов для исключения.
