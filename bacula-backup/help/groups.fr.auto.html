Groupes Bacula
Cette page répertorie les groupes d'hôtes, chacun étant un ensemble d'hôtes pouvant être simultanément soutenu par une seule tâche de sauvegarde de groupe Bacula . Les hôtes d'un groupe proviennent d'une base de données de groupe de nœuds OCM Manager ou du module Webmin Servers Index, selon la configuration de ce module.

Pour que les sauvegardes du groupe bacula fonctionnent, chacun doit avoir au moins le démon de fichier Bacula installé et tous les hôtes du groupe doivent avoir le même mot de passe pour ce directeur.
