Εκτελέστε εργασία δημιουργίας αντιγράφων ασφαλείας
Αυτή η φόρμα μπορεί να χρησιμοποιηθεί για να ξεκινήσει η άμεση εκτέλεση μιας εργασίας δημιουργίας αντιγράφων ασφαλείας Bacula. Για να εκτελέσετε μια εργασία, το μόνο που χρειάζεται είναι να το επιλέξετε από τη λίστα και να επιλέξετε εάν το Webmin θα πρέπει να περιμένει την εμφάνιση της τελικής του κατάστασης.
