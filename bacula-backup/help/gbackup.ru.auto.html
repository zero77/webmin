Запустить групповое задание Bacula
Эту форму можно использовать для немедленного запуска задания резервного копирования Bacula, которое выполняется на всех хостах в группе Bacula. Чтобы запустить его, просто выберите его из списка и нажмите кнопку «Резервировать сейчас» . Поскольку задание выполняется на нескольких хостах, его окончательный статус отображаться не будет.
