Emplois Bacula Group Backup
Cette page répertorie les travaux de sauvegarde configurés pour s'exécuter simultanément sur tous les hôtes d'un groupe Bacula. Ils ont les mêmes paramètres que les travaux de sauvegarde standard, sauf que vous ne pouvez pas définir de travaux par défaut pour partager les paramètres entre les travaux. De plus, au lieu de sélectionner un client à sauvegarder, vous devez plutôt sélectionner un groupe Bacula précédemment défini.
