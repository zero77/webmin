Διαμόρφωση αρχείου Daemon
Αυτή η φόρμα μπορεί να χρησιμοποιηθεί για να προσαρμόσετε τις ρυθμίσεις για το δαίμονα του αρχείου Bacula που εκτελείται στο σύστημα. Οι προεπιλογές θα είναι συνήθως σωστές, αν και μπορείτε να προσαρμόσετε το όνομα δαίμονα και τον αριθμό των ταυτόχρονων εργασιών. Εάν το σύστημά σας Bacula υποστηρίζει την ασφάλεια TLS, οι επιλογές TLS σε αυτήν τη σελίδα μπορούν να χρησιμοποιηθούν για την ασφαλή επικοινωνία μεταξύ του δαίμονα, του διευθυντή και των δαιμόνων αποθήκευσης.
