File Daemon Configuration
Ez az űrlap felhasználható a rendszeren futó Bacula fájldémon beállításainak módosítására. Az alapértelmezett beállítások általában helyesek, bár beállíthatják a démon nevét és az egyidejű feladatok számát. Ha a Bacula rendszere támogatja a TLS biztonságot, az ezen az oldalon található TLS beállítások felhasználhatók a fájldémon, a rendező és a tárolódémonok közötti kommunikáció biztosítására.
