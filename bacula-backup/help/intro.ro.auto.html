Backup Bacula
Bacula este un set de programe de calculator care vă permite (sau administratorului de sistem) să gestionați copie de rezervă, recuperare și verificare a datelor computerului într-o rețea de computere de diferite tipuri. De asemenea, Bacula poate rula integral pe un singur computer și poate face backup la diferite tipuri de suporturi, inclusiv bandă și disc.

În termeni tehnici, este un program de rezervă bazat pe client / server în rețea. Bacula este relativ ușor de utilizat și eficient, oferind în același timp multe funcții avansate de gestionare a stocării care ușurează găsirea și recuperarea fișierelor pierdute sau deteriorate. Datorită designului modular, Bacula este scalabil de la mici sisteme computerizate unice la sisteme constând din sute de calculatoare situate pe o rețea mare.

Sistemul Bacula este împărțit în trei demoni separati, care teoretic pot rula pe diferite gazde. Acestea sunt:

Director
Directorul stochează locuri de muncă, clienți și majoritatea altor setări de configurare și este responsabil de inițierea lucrărilor de rezervă. Acest modul Webmin trebuie să fie executat pe sistemul care are instalat directorul.
Fișier Daemon
Acest demon este responsabil de citirea fișierelor reale pentru a fi salvate. O configurație Bacula poate avea mai mult de un demon de fișier, fiecare rulat pe un sistem pentru a fi salvat.
Daemon de stocare
Demonul de stocare este responsabil pentru scrierea datelor pe suportul de rezervă final, cum ar fi o unitate sau un fișier. De obicei, veți avea nevoie doar de un astfel de demon (de obicei rulat pe aceeași gazdă ca directorul), dar o configurație Bacula mai complexă poate avea mai multe sisteme cu diferite unități de bandă, pentru a răspândi încărcarea de rezervă.
Acest modul Webmin poate gestiona un sistem care are una sau mai multe demonii Bacula instalate.
