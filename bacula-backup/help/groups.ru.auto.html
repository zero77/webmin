Группы Bacula
На этой странице перечислены группы хостов, каждая из которых представляет собой набор хостов, которые могут одновременно поддерживаться одним заданием резервного копирования группы Bacula . Хосты в группе берутся из базы данных группы узлов OCM Manager или из модуля индекса серверов Webmin, в зависимости от конфигурации этого модуля.

Чтобы резервные копии группы bacula работали, на каждом должен быть установлен хотя бы демон файлов Bacula, и все хосты в группе должны иметь одинаковый пароль для этого директора.
