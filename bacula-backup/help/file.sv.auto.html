File Daemon Configuration
Detta formulär kan användas för att justera inställningarna för Bacula-fildemonen som körs på systemet. Standardvärdena är vanligtvis korrekta, även om du kan justera namnet på namnet och antalet samtidiga jobb. Om ditt Bacula-system stöder TLS-säkerhet kan TLS-alternativen på den här sidan användas för att säkra kommunikation mellan fildemonen, regissören och lagringsdemonerna.
