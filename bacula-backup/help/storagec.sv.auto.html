Storage Daemon Configuration
Detta formulär kan användas för att justera inställningar för Bacula-lagringsdemon som körs på systemet. Standardvärdena är vanligtvis korrekta, även om du kan justera namnet på namnet och antalet samtidiga jobb. Om ditt Bacula-system stöder TLS-säkerhet kan TLS-alternativen på den här sidan användas för att säkra kommunikation mellan lagrings- och fildemoner.
