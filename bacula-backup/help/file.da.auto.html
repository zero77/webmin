File Daemon-konfiguration
Denne formular kan bruges til at justere indstillinger for Bacula-fildemonen, der kører på systemet. Standardindstillingerne vil typisk være korrekte, selvom du kan justere daemonnavnet og antallet af samtidige job. Hvis dit Bacula-system understøtter TLS-sikkerhed, kan TLS-indstillingerne på denne side bruges til at sikre kommunikation mellem fildemonen, instruktøren og lagringsdemonerne.
