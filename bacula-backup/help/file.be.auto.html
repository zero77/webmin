Канфігурацыя файла Дэмана
Гэтая форма можа быць выкарыстана для налады налад для дэмана файла Bacula, які працуе ў сістэме. Звычайна значэнні па змаўчанні з'яўляюцца правільнымі, хоць вы можаце наладзіць імя дэмана і колькасць адначасовых заданняў. Калі ваша сістэма Bacula падтрымлівае бяспеку TLS, параметры TLS на гэтай старонцы можна выкарыстоўваць для забеспячэння сувязі паміж дэманамі файла, дырэктарам і дэманамі захоўвання.
