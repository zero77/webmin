Configuration du fichier démon
Ce formulaire peut être utilisé pour ajuster les paramètres du démon de fichier Bacula exécuté sur le système. Les valeurs par défaut sont généralement correctes, bien que vous puissiez ajuster le nom du démon et le nombre de travaux simultanés. Si votre système Bacula prend en charge la sécurité TLS, les options TLS de cette page peuvent être utilisées pour sécuriser la communication entre le démon de fichier, le directeur et les démons de stockage.
