Configuración del demonio de archivo
Este formulario se puede usar para ajustar la configuración del demonio de archivo Bacula que se ejecuta en el sistema. Los valores predeterminados generalmente serán correctos, aunque puede ajustar el nombre del daemon y la cantidad de trabajos simultáneos. Si su sistema Bacula es compatible con la seguridad TLS, las opciones TLS en esta página se pueden usar para asegurar la comunicación entre el demonio de archivo, el director y los demonios de almacenamiento.
