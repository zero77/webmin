File Daemon Director
หน้านี้แสดงรายชื่อกรรมการทั้งหมดที่ได้รับอนุญาตให้เชื่อมต่อกับไฟล์ daemon นี้ หากคุณกำลังเพิ่มไฟล์ daemon นี้ไปยังผู้กำกับระยะไกลรหัสผ่านในรายการ ไคลเอ็นต์สำรองข้อมูล จะต้องตรงกับรหัสผ่านที่แสดงไว้ที่นี่

หากเปิดใช้งาน TLS สำหรับผู้กำกับการสื่อสารระหว่างผู้กำกับและไฟล์ daemon นี้จะถูกเข้ารหัสและตรวจสอบโดยใช้ใบรับรองและคีย์ที่เลือก
