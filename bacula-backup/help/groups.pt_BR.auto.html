Grupos Bacula
Esta página lista grupos de hosts, cada um dos quais é um conjunto de hosts que podem ser apoiados simultaneamente por uma única tarefa de backup em grupo do Bacula . Os hosts em um grupo são obtidos de um banco de dados de grupo de nós do OCM Manager ou do módulo Webmin Servers Index, dependendo da configuração deste módulo.

Para que os backups do grupo bacula funcionem, cada um deve ter pelo menos o daemon de arquivos Bacula instalado e todos os hosts no grupo devem ter a mesma senha para este diretor.
