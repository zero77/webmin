ไฟล์ Daemon Configuration
แบบฟอร์มนี้สามารถใช้เพื่อปรับการตั้งค่าสำหรับไฟล์ Bacula daemon ที่รันบนระบบ โดยทั่วไปแล้วค่าเริ่มต้นจะถูกต้องแม้ว่าคุณจะสามารถปรับชื่อ daemon และจำนวนงานที่เกิดขึ้นพร้อมกันได้ หากระบบ Bacula ของคุณรองรับการรักษาความปลอดภัย TLS ตัวเลือก TLS ในหน้านี้สามารถใช้เพื่อรักษาความปลอดภัยการสื่อสารระหว่าง daemon ไฟล์, Director และ Storage daemons
