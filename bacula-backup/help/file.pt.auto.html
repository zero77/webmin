Configuração de Daemon de Arquivo
Este formulário pode ser usado para ajustar as configurações do daemon de arquivos Bacula em execução no sistema. Os padrões normalmente estão corretos, embora você possa ajustar o nome do daemon e o número de trabalhos simultâneos. Se o seu sistema Bacula suportar a segurança TLS, as opções TLS nesta página poderão ser usadas para proteger a comunicação entre o daemon de arquivo, o diretor e os daemons de armazenamento.
