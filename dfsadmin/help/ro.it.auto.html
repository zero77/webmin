Accesso in sola lettura
Ciò determina quali host hanno accesso in sola lettura a questa condivisione. Esistono tre opzioni per questa opzione:
  • Nessuna
    Nessun host ha accesso in sola lettura.
  • Tutti gli host
    Qualsiasi host può montare questa condivisione in sola lettura. Se sei connesso a Internet, ciò significa che chiunque nel mondo può leggere i file nella directory esportata.
  • Host elencati
    Gli host o gli indirizzi IP elencati di seguito possono montare questa condivisione in sola lettura. Gli host corrispondenti elencati di seguito possono montare questa condivisione in sola lettura. Gli host possono essere specificati nei seguenti modi:

    Nome host o indirizzo IP (come ftp.foo.com o 1.2.3.4 )
    Corrisponde all'host con il nome o l'indirizzo indicato
    Netgroup (come ingegneria )
    Corrisponde a qualsiasi host membro del netgroup
    Dominio DNS (come .foo.com )
    Corrisponde a qualsiasi host nel dominio
    Rete (come @ 10.254.1 )
    Corrisponde a qualsiasi host nella rete
    Rete / maschera di rete (come @ 10.254.1 / 24 )
    Corrisponde a qualsiasi host nella rete

    Inoltre, è possibile aggiungere il prefisso a una qualsiasi delle notazioni precedenti con un - , il che significa che l'host, il netgroup, il dominio o la rete non possono montare questa condivisione. Questo può essere utile per consentire l'accesso da parte di un set di host, ma negare un membro.
