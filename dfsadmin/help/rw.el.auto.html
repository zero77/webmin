Πρόσβαση ανάγνωσης-εγγραφής
Αυτό καθορίζει ποιοι οικοδεσπότες έχουν πρόσβαση ανάγνωσης-εγγραφής σε αυτό το κοινόχρηστο στοιχείο. Υπάρχουν τρεις επιλογές για αυτήν την επιλογή:
  • Κανένας
    Κανένας οικοδεσπότης δεν έχει πρόσβαση ανάγνωσης-εγγραφής.
  • Όλοι οι οικοδεσπότες
    Οποιοσδήποτε κεντρικός υπολογιστής μπορεί να τοποθετήσει αυτήν την κοινή χρήση ανάγνωσης-εγγραφής Εάν είστε συνδεδεμένοι στο Διαδίκτυο, αυτό σημαίνει ότι οποιοσδήποτε στον κόσμο μπορεί να διαβάσει, να γράψει και να διαγράψει αρχεία στον εξαγόμενο κατάλογο. Χρησιμοποιήστε αυτήν την επιλογή με προσοχή.
  • Λίστα κεντρικών υπολογιστών
    Οι κεντρικοί υπολογιστές ή οι διευθύνσεις IP που αναφέρονται παρακάτω μπορούν να προσαρμόσουν αυτήν την κοινή χρήση ανάγνωσης-εγγραφής. Οι αντίστοιχοι οικοδεσπότες που αναφέρονται παρακάτω μπορούν να τοποθετήσουν αυτήν την κοινή χρήση ανάγνωσης-εγγραφής. Οι κεντρικοί υπολογιστές μπορούν να καθοριστούν με τους ακόλουθους τρόπους:

    Όνομα κεντρικού υπολογιστή ή διεύθυνση IP (όπως ftp.foo.com ή 1.2.3.4 )
    Ταιριάζει με τον οικοδεσπότη με το δεδομένο όνομα ή διεύθυνση
    Netgroup (όπως μηχανική )
    Αντιστοιχεί σε κάθε κεντρικό υπολογιστή που είναι μέλος της ομάδας δικτύου
    Τομέας DNS (όπως .foo.com )
    Αντιστοιχεί σε οποιοδήποτε κεντρικό υπολογιστή στον τομέα
    Δίκτυο (όπως @ 10.254.1 )
    Αντιστοιχεί σε οποιοδήποτε κεντρικό υπολογιστή στο δίκτυο
    Δίκτυο / netmask (όπως @ 10.254.1 / 24 )
    Αντιστοιχεί σε οποιοδήποτε κεντρικό υπολογιστή στο δίκτυο

    Επιπλέον, οποιαδήποτε από τις παραπάνω συμβολές μπορεί να προθετηθεί από ένα - , πράγμα που σημαίνει ότι ο κεντρικός υπολογιστής, το netgroup, ο τομέας ή το δίκτυο δεν επιτρέπεται να προσαρτήσει αυτό το κοινόχρηστο στοιχείο. Αυτό μπορεί να είναι χρήσιμο για να επιτρέπεται η πρόσβαση από ένα σύνολο κεντρικών υπολογιστών, αλλά αρνείται ένα μέλος.
