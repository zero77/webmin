Каждый общий ресурс имеет экспортируемый каталог, описание для клиентов NFS, три списка хостов с различными уровнями доступа к этому общему ресурсу и ряд других параметров.

Общий ресурс может быть экспортирован только для чтения или для чтения и записи в список клиентов. По умолчанию пользователь root в клиентских системах не является доверенным сервером, если только клиент не находится в списке корневого доступа.
