การเข้าถึงแบบอ่านอย่างเดียว
สิ่งนี้จะกำหนดว่าโฮสต์ใดที่มีการเข้าถึงแบบแบ่งใช้นี้แบบอ่านอย่างเดียว มีตัวเลือกสามตัวเลือกสำหรับตัวเลือกนี้:
  • ไม่มี
    ไม่มีโฮสต์ที่มีการเข้าถึงแบบอ่านอย่างเดียว
  • โฮสต์ทั้งหมด
    โฮสต์ใด ๆ สามารถเมานต์การแชร์นี้แบบอ่านอย่างเดียว หากคุณเชื่อมต่อกับอินเทอร์เน็ตหมายความว่าทุกคนในโลกสามารถอ่านไฟล์ในไดเรกทอรีที่ส่งออกได้
  • รายชื่อโฮสต์
    โฮสต์หรือที่อยู่ IP ที่แสดงด้านล่างนี้สามารถใช้การแชร์นี้เป็นแบบอ่านอย่างเดียว โฮสต์ที่จับคู่ในรายการด้านล่างสามารถเมานต์แชร์นี้แบบอ่านอย่างเดียว โฮสต์สามารถระบุได้ด้วยวิธีต่อไปนี้:

    ชื่อโฮสต์หรือที่อยู่ IP (เช่น ftp.foo.com หรือ 1.2.3.4 )
    จับคู่โฮสต์กับชื่อหรือที่อยู่ที่ระบุ
    Netgroup (เช่น วิศวกรรม )
    ตรงกับโฮสต์ใด ๆ ที่เป็นสมาชิกของ netgroup
    โดเมน DNS (เช่น. foo.com )
    ตรงกับโฮสต์ใด ๆ ในโดเมน
    เครือข่าย (เช่น @ 10.254.1 )
    จับคู่โฮสต์ใด ๆ ในเครือข่าย
    เครือข่าย / netmask (เช่น @ 10.254.1 / 24 )
    จับคู่โฮสต์ใด ๆ ในเครือข่าย

    นอกจากนี้สัญลักษณ์ใด ๆ ข้างต้นสามารถนำหน้าด้วย - ซึ่งหมายความว่าโฮสต์ netgroup โดเมนหรือเครือข่าย ไม่ได้ รับอนุญาตให้เชื่อมต่อการแชร์นี้ สิ่งนี้มีประโยชน์สำหรับการอนุญาตให้เข้าถึงโดยชุดโฮสต์ แต่ปฏิเสธสมาชิกหนึ่งราย
