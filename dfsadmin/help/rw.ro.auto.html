Acces de citire-scriere
Aceasta determină ce gazde au acces de citire-scriere la acest share. Există trei opțiuni pentru această opțiune:
  • Nici unul
    Nici o gazdă nu are acces la citire-scriere.
  • Toate gazdele
    Orice gazdă poate monta această partajare citire-scriere. Dacă sunteți conectat la Internet, acest lucru înseamnă că oricine din lume poate citi, scrie și șterge fișierele din directorul exportat. Utilizați această opțiune cu precauție.
  • Gazde enumerate
    Gazdele sau adresele IP enumerate mai jos pot monta această partajare citire-scriere. Potrivirea gazdelor enumerate mai jos poate monta această partajare citire-scriere. Gazdele pot fi specificate în următoarele moduri:

    Nume gazdă sau adresă IP (cum ar fi ftp.foo.com sau 1.2.3.4 )
    Se potrivește cu gazda cu numele sau adresa dată
    Netgroup (cum ar fi inginerie )
    Se potrivește cu orice gazdă care este membru al grupului net
    Domeniul DNS (cum ar fi .foo.com )
    Se potrivește cu orice gazdă din domeniu
    Rețea (cum ar fi @ 10.254.1 )
    Se potrivește cu orice gazdă din rețea
    Network / netmask (cum ar fi @ 10.254.1 / 24 )
    Se potrivește cu orice gazdă din rețea

    În plus, oricare dintre notările de mai sus poate fi prefixată de a - , ceea ce înseamnă că gazda, grupul net, domeniul sau rețeaua nu are voie să monteze această partajare. Acest lucru poate fi util pentru a permite accesul unui set de gazde, dar refuzând un membru.
