Introdução aos compartilhamentos NFS

Solaris NFS

Este módulo permite configurar diretórios (chamados de compartilhamentos no Solaris) exportados via NFS do seu sistema Solaris. A primeira página mostra uma lista de diretórios e os clientes para os quais são exportados, permitindo que você clique no nome do diretório para editar suas opções ou começar a compartilhar um novo diretório.

Abaixo da lista de compartilhamentos na primeira página, há um botão que ativará a configuração atual. Isso é feito interrompendo e iniciando os processos do servidor NFS em execução.

O Solaris não permite reexportar um diretório NFS montado de outro servidor - apenas os diretórios nos sistemas de arquivos locais podem ser exportados. Se um diretório compartilhado contiver o ponto de montagem de outro sistema de arquivos local, os clientes NFS desse compartilhamento não verão os arquivos abaixo desse ponto de montagem. O servidor deve reexportar cada sistema de arquivos local separadamente, para ser montado pelos clientes no ponto correto na hierarquia de diretórios.
