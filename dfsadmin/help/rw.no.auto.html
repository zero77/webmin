Lese-skrivetilgang
Dette avgjør hvilke verter som har lese-skrivetilgang til denne delingen. Det er tre valg for dette alternativet:
  • Ingen
    Ingen verter har lese-skrive tilgang.
  • Alle verter
    Enhver vert kan montere denne delingen lese-skrive. Hvis du er koblet til Internett, betyr dette at alle i verden kan lese, skrive og slette filer i den eksporterte katalogen. Bruk dette alternativet med forsiktighet.
  • Oppførte verter
    Vertene eller IP-adressene som er oppført nedenfor, kan montere denne deles lese-skrive. Matchende verter som er oppført nedenfor, kan montere denne delingen lese-skrive. Verter kan spesifiseres på følgende måter:

    Vertsnavn eller IP-adresse (som ftp.foo.com eller 1.2.3.4 )
    Matcher verten med det oppgitte navnet eller adressen
    Netgroup (som engineering )
    Matcher alle vert som er medlem av nettgruppen
    DNS-domene (som .foo.com )
    Matcher hvilken som helst vert i domenet
    Nettverk (som @ 10.254.1 )
    Matcher hvilken som helst vert i nettverket
    Nettverk / nettmaske (som @ 10.254.1 / 24 )
    Matcher hvilken som helst vert i nettverket

    I tillegg kan hvilken som helst av notasjonene ovenfor være forhåndsinnstilt av en - , noe som betyr at verten, netgruppen, domenet eller nettverket ikke har lov til å montere denne delingen. Dette kan være nyttig for å tillate tilgang av et sett verter, men å nekte et medlem.
