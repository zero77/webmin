Acesso de leitura e gravação
Isso determina quais hosts têm acesso de leitura e gravação a esse compartilhamento. Existem três opções para esta opção:
  • Nenhum
    Nenhum host tem acesso de leitura e gravação.
  • Todos os hosts
    Qualquer host pode montar esse compartilhamento de leitura e gravação. Se você estiver conectado à Internet, isso significa que qualquer pessoa no mundo pode ler, gravar e excluir arquivos no diretório exportado. Use esta opção com cuidado.
  • Anfitriões listados
    Os hosts ou endereços IP listados abaixo podem montar esse compartilhamento de leitura e gravação. Os hosts correspondentes listados abaixo podem montar esse compartilhamento de leitura e gravação. Os hosts podem ser especificados das seguintes maneiras:

    Nome do host ou endereço IP (como ftp.foo.com ou 1.2.3.4 )
    Corresponde ao host com o nome ou endereço
    Grupo de rede (como engenharia )
    Corresponde a qualquer host que seja membro do grupo de rede
    Domínio DNS (como .foo.com )
    Corresponde a qualquer host no domínio
    Rede (como @ 10.254.1 )
    Corresponde a qualquer host na rede
    Rede / máscara de rede (como @ 10.254.1 / 24 )
    Corresponde a qualquer host na rede

    Além disso, qualquer uma das notações acima pode ser prefixada com a - , o que significa que o host, grupo de rede, domínio ou rede não tem permissão para montar esse compartilhamento. Isso pode ser útil para permitir o acesso por um conjunto de hosts, mas negando um membro.
