Доступ только для чтения
Это определяет, какие хосты имеют доступ только для чтения к этой общей папке. Есть три варианта для этого варианта:
  • Никто
    Хосты не имеют доступа только для чтения.
  • Все хозяева
    Любой хост может монтировать этот общий ресурс только для чтения. Если вы подключены к Интернету, это означает, что любой человек в мире может читать файлы в экспортированном каталоге.
  • Перечисленные хосты
    Хосты или IP-адреса, перечисленные ниже, могут монтировать этот общий ресурс только для чтения. Соответствующие хосты, перечисленные ниже, могут монтировать этот общий ресурс только для чтения. Хосты могут быть указаны следующими способами:

    Имя хоста или IP-адрес (например, ftp.foo.com или 1.2.3.4 )
    Соответствует хосту с указанным именем или адресом
    Netgroup (например, инжиниринг )
    Соответствует любому хосту, который является членом сетевой группы
    DNS-домен (например, .foo.com )
    Соответствует любому хосту в домене
    Сеть (как @ 10.254.1 )
    Соответствует любому хосту в сети
    Сеть / сетевая маска (например, @ 10.254.1 / 24 )
    Соответствует любому хосту в сети

    Кроме того, любой из приведенных выше обозначений может начинаться с префикса - , что означает, что хосту, сетевой группе, домену или сети не разрешено монтировать этот общий ресурс. Это может быть полезно для разрешения доступа множеству хостов, но запрета одного участника.
