Πρόσβαση μόνο για ανάγνωση
Αυτό καθορίζει ποιοι κεντρικοί υπολογιστές έχουν πρόσβαση μόνο για ανάγνωση σε αυτό το κοινόχρηστο στοιχείο. Υπάρχουν τρεις επιλογές για αυτήν την επιλογή:
  • Κανένας
    Κανένας κεντρικός υπολογιστής δεν έχει πρόσβαση μόνο για ανάγνωση.
  • Όλοι οι οικοδεσπότες
    Οποιοσδήποτε κεντρικός υπολογιστής μπορεί να τοποθετήσει αυτήν την κοινή χρήση μόνο για ανάγνωση. Εάν είστε συνδεδεμένοι στο Διαδίκτυο, αυτό σημαίνει ότι οποιοσδήποτε στον κόσμο μπορεί να διαβάσει αρχεία στον εξαγόμενο κατάλογο.
  • Λίστα κεντρικών υπολογιστών
    Οι κεντρικοί υπολογιστές ή οι διευθύνσεις IP που αναφέρονται παρακάτω μπορούν να προσαρτήσουν αυτήν την κοινή χρήση μόνο για ανάγνωση. Οι αντίστοιχοι κεντρικοί υπολογιστές που αναφέρονται παρακάτω μπορούν να προσαρτήσουν αυτήν την κοινή χρήση μόνο για ανάγνωση. Οι κεντρικοί υπολογιστές μπορούν να καθοριστούν με τους ακόλουθους τρόπους:

    Όνομα κεντρικού υπολογιστή ή διεύθυνση IP (όπως ftp.foo.com ή 1.2.3.4 )
    Ταιριάζει με τον οικοδεσπότη με το δεδομένο όνομα ή διεύθυνση
    Netgroup (όπως μηχανική )
    Αντιστοιχεί σε κάθε κεντρικό υπολογιστή που είναι μέλος της ομάδας δικτύου
    Τομέας DNS (όπως .foo.com )
    Αντιστοιχεί σε οποιοδήποτε κεντρικό υπολογιστή στον τομέα
    Δίκτυο (όπως @ 10.254.1 )
    Αντιστοιχεί σε οποιοδήποτε κεντρικό υπολογιστή στο δίκτυο
    Δίκτυο / netmask (όπως @ 10.254.1 / 24 )
    Αντιστοιχεί σε οποιοδήποτε κεντρικό υπολογιστή στο δίκτυο

    Επιπλέον, οποιαδήποτε από τις παραπάνω συμβολές μπορεί να προθετηθεί από ένα - , πράγμα που σημαίνει ότι ο κεντρικός υπολογιστής, το netgroup, ο τομέας ή το δίκτυο δεν επιτρέπεται να προσαρτήσει αυτό το κοινόχρηστο στοιχείο. Αυτό μπορεί να είναι χρήσιμο για να επιτρέπεται η πρόσβαση από ένα σύνολο κεντρικών υπολογιστών, αλλά αρνείται ένα μέλος.
