Cada compartilhamento possui um diretório que está sendo exportado, uma descrição para clientes NFS, três listas de hosts com diferentes níveis de acesso a esse compartilhamento e várias outras opções.

Um compartilhamento pode ser exportado somente leitura ou leitura e gravação para uma lista de clientes. Por padrão, o usuário root nos sistemas clientes não é confiável para o servidor, a menos que o cliente também esteja na lista de acesso root.
