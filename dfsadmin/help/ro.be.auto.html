Доступ толькі для чытання
Гэта вызначае, якія хасты маюць доступ толькі да чытання. Для гэтага варыянту ёсць тры варыянты:
  • Не
    Няма хостаў, якія маюць доступ толькі для чытання.
  • Усе гаспадары
    Любы хост можа ўсталяваць гэтую долю толькі для чытання. Калі вы падлучаныя да Інтэрнэту, гэта азначае, што любы чалавек у свеце можа чытаць файлы ў экспартаваным каталогу.
  • Пералічаныя гаспадары
    Пералічаныя ніжэй хосты ці IP-адрасы могуць усталяваць гэтую долю толькі для чытання. Адпаведныя хасты, пералічаныя ніжэй, могуць усталяваць гэтую долю толькі для чытання. Хасты могуць быць паказаны наступнымі спосабамі:

    Імя хаста ці IP-адрас (напрыклад, ftp.foo.com або 1.2.3.4 )
    Супадае з хостам з зададзеным імем або адрасам
    Netgroup (напрыклад, інжынерыя )
    Супадае з любым хостам, які ўваходзіць у сетку
    Дамен DNS (напрыклад, .foo.com )
    Супадае з любым хостам у дамене
    Сетка (напрыклад, @ 10.254.1 )
    Супадае з любым хостам у сетцы
    Сетка / сеткавая маска (напрыклад, @ 10.254.1 / 24 )
    Супадае з любым хостам у сетцы

    Акрамя таго, любы з вышэйпералічаных абазначэнняў можа быць прэфіксам а - азначае, што хост, сеткавая група, дамен ці сетка не могуць усталяваць гэтую агульную долю. Гэта можа быць карысна для дазволу набору хастоў, але забароны аднаго члена.
