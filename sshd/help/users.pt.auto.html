Autenticação
Esta página é para configurar opções relacionadas à autenticação para o seu servidor SSH. Algumas das opções menos óbvias são:
Permitir autenticação por senha
Se isso estiver definido como Não, os usuários precisarão ter seu diretório .ssh configurado com uma chave pública antes de poderem fazer login. Se definido como Sim, o usuário poderá inserir uma senha para autenticação se a validação não puder ser feita usando uma chave pública.

Permitir autenticação RSA?
Se estiver definido como Não, os usuários sempre precisarão digitar sua senha, mesmo que sua chave pública tenha sido configurada.

Verificar permissões nos arquivos principais?
Se definido como Sim, o servidor SSH verificará o diretório .ssh de cada usuário e os diretórios pai para garantir que eles não sejam graváveis em grupo e o arquivo .ssh / identity para garantir que não seja legível por mais ninguém. Se definido como Não, nenhuma verificação de permissões de arquivo será feita.
