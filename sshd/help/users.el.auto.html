Αυθεντικοποίηση
Αυτή η σελίδα προορίζεται για τη διαμόρφωση επιλογών που σχετίζονται με τον έλεγχο ταυτότητας για τον διακομιστή SSH σας. Μερικές από τις λιγότερο προφανείς επιλογές είναι:
Να επιτρέπεται ο έλεγχος ταυτότητας με κωδικό πρόσβασης
Εάν έχει οριστεί σε Όχι, οι χρήστες θα πρέπει να έχουν τη ρύθμιση καταλόγου .ssh με ένα δημόσιο κλειδί για να μπορέσουν να συνδεθούν. Εάν οριστεί σε Ναι, θα επιτρέπεται στον χρήστη να εισάγει έναν κωδικό πρόσβασης για έλεγχο ταυτότητας, εάν η επικύρωση δεν μπορεί να γίνει χρησιμοποιώντας ένα δημόσιο κλειδί.

Να επιτρέπεται ο έλεγχος ταυτότητας RSA;
Εάν έχει οριστεί σε Όχι, οι χρήστες θα πρέπει πάντα να εισάγουν τον κωδικό πρόσβασής τους, ακόμη και αν έχει ρυθμιστεί το δημόσιο κλειδί τους.

Έλεγχος δικαιωμάτων σε βασικά αρχεία;
Εάν οριστεί σε Ναι, ο διακομιστής SSH θα ελέγξει τον κατάλογο .ssh κάθε χρήστη και τυχόν γονικούς καταλόγους για να βεβαιωθεί ότι δεν είναι εγγράψιμοι στην ομάδα και το αρχείο .ssh / ταυτότητας για να βεβαιωθείτε ότι δεν είναι αναγνώσιμο από κανέναν άλλο. Εάν οριστεί σε Όχι, δεν θα γίνει έλεγχος των αδειών αρχείων.
