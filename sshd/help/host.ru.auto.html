На этой странице отображаются параметры для одного хоста, хостов, имена которых соответствуют некоторому шаблону, или для всех хостов. Некоторые из менее очевидных вариантов:
Войти как пользователь
Обычно, если имя пользователя не указано в командной строке ssh , имя текущего пользователя используется для входа на удаленный SSH-сервер. Однако этот параметр можно использовать для указания другого имени пользователя по умолчанию для определенного хоста или хостов.

Побег персонаж
При интерактивном входе в систему по SSH можно использовать escape-символ, чтобы разорвать соединение и закрыть или приостановить его.

Сжатие SSH трафика?
Если эта опция включена, клиент SSH сжимает все данные, отправленные на этот хост, с помощью алгоритма gzip . Это может быть полезно, если вы копируете большие файлы с помощью scp по медленной ссылке.

Локальные порты для пересылки на сервер
В этой таблице вы можете ввести номера локальных портов, которые будут перенаправлены на некоторый хост и порт сервером SSH. Это может быть полезно, если ваш единственный доступ к какой-либо сети осуществляется через SSH-вход на один компьютер в этой сети и вы хотите получить доступ к другим службам, таким как веб-серверы или POP-серверы.

Порты сервера для пересылки на локальный
В этой таблице вы можете ввести номера портов на сервере, которые будут перенаправлены на некоторый хост и порт в сети клиентского компьютера.
