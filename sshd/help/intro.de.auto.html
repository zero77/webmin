SSH-Server
SSH ist ein Protokoll, mit dem sich Benutzer über ein Netzwerk bei Ihrem Computer anmelden können, ähnlich wie bei Telnet. Alle SSH-Verbindungen werden jedoch an beiden Enden verschlüsselt und authentifiziert, um zu verhindern, dass die Angreifer Kennwörter erfassen oder eine SSH-Anmeldung auf einen anderen Server umleiten.

Da SSH so eingerichtet werden kann, dass anstelle des von Telnet verwendeten normalen Benutzernamens und Kennworts RSA-Schlüssel für die Authentifizierung verwendet werden, können sich Benutzer bei einem Remoteserver anmelden, ohne jedes Mal ein Kennwort eingeben zu müssen. Dies bietet ein ähnliches Verhalten wie RSH, jedoch ohne die Sicherheitsprobleme, die durch die Authentifizierung von Benutzern allein anhand ihrer IP-Adresse entstehen.

Mit diesem Modul können Sie verschiedene Optionen für den auf Ihrem System installierten SSH-Server sowie die Option konfigurieren, die verwendet wird, wenn ein SSH-Client eine Verbindung von Ihrem Computer zu einem anderen Server herstellt. Nachdem Sie Änderungen an den Serveroptionen vorgenommen haben, müssen Sie unten auf der Seite auf die Schaltfläche Änderungen übernehmen klicken, damit die Änderungen wirksam werden.
