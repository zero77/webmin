Servidor SSH
O SSH é um protocolo que permite aos usuários acessar sua máquina através de uma rede, como o telnet. No entanto, todas as conexões SSH são criptografadas e autenticadas nas duas extremidades para impedir que os invasores capturem senhas ou redirecionem um logon SSH para outro servidor.

Como o SSH pode ser configurado para usar chaves RSA para autenticação, em vez do nome de usuário e senha normais usados pelo telnet, os usuários podem efetuar login em um servidor remoto sem precisar digitar uma senha a cada vez. Isso fornece um comportamento semelhante ao RSH, mas sem os problemas de segurança resultantes da autenticação de usuários com base apenas no endereço IP.

Este módulo permite configurar várias opções para o servidor SSH instalado no seu sistema, bem como a opção usada quando um cliente SSH faz uma conexão da sua máquina com outro servidor. Depois de fazer qualquer alteração nas opções do servidor, você deve clicar no botão Aplicar Alterações na parte inferior da página para que as alterações tenham efeito.
