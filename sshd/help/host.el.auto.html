Αυτή η σελίδα εμφανίζει επιλογές για έναν κεντρικό υπολογιστή, κεντρικούς υπολογιστές των οποίων τα ονόματα ταιριάζουν με κάποιο μοτίβο ή για όλους τους κεντρικούς υπολογιστές. Μερικές από τις λιγότερο προφανείς επιλογές είναι:
Συνδεθείτε ως χρήστης
Κανονικά, εάν δεν δίνεται όνομα χρήστη στη γραμμή εντολών ssh , το όνομα του τρέχοντος χρήστη χρησιμοποιείται για σύνδεση στον απομακρυσμένο διακομιστή SSH. Ωστόσο, αυτή η επιλογή μπορεί να χρησιμοποιηθεί για τον καθορισμό διαφορετικού προεπιλεγμένου ονόματος χρήστη για έναν συγκεκριμένο κεντρικό υπολογιστή ή κεντρικούς υπολογιστές.

Αποδράστε χαρακτήρα
Όταν κάνετε μια διαδραστική σύνδεση SSH, ο χαρακτήρας διαφυγής μπορεί να χρησιμοποιηθεί για διακοπή της σύνδεσης και κλείσιμο ή αναστολή του.

Συμπίεση κυκλοφορίας SSH;
Εάν αυτή η επιλογή είναι ενεργοποιημένη, ο πελάτης SSH θα συμπιέσει όλα τα δεδομένα που αποστέλλονται σε αυτόν τον κεντρικό υπολογιστή με τον αλγόριθμο gzip . Αυτό μπορεί να είναι χρήσιμο αν αντιγράφετε μεγάλα αρχεία με scp μέσω αργού συνδέσμου.

Τοπικές θύρες για προώθηση στον διακομιστή
Σε αυτόν τον πίνακα μπορείτε να εισαγάγετε τοπικούς αριθμούς θύρας που θα προωθηθούν σε κάποιον κεντρικό υπολογιστή και θύρα από τον διακομιστή SSH. Αυτό μπορεί να είναι χρήσιμο εάν η μόνη πρόσβασή σας σε κάποιο δίκτυο είναι μέσω σύνδεσης SSH σε ένα μηχάνημα σε αυτό το δίκτυο και θέλετε να έχετε πρόσβαση σε άλλες υπηρεσίες όπως διακομιστές ιστού ή POP.

Θύρες διακομιστή για προώθηση σε τοπικό
Σε αυτόν τον πίνακα μπορείτε να εισαγάγετε αριθμούς θύρας στο διακομιστή που θα προωθηθούν σε κάποιο κεντρικό υπολογιστή και θύρα στο δίκτυο του υπολογιστή-πελάτη.
