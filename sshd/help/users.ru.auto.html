Аутентификация
Эта страница предназначена для настройки параметров аутентификации для вашего SSH-сервера. Некоторые из менее очевидных вариантов:
Разрешить аутентификацию по паролю
Если для этого параметра установлено значение Нет, пользователям потребуется настроить каталог .ssh с открытым ключом, прежде чем они смогут войти в систему. Если установлено значение Да, пользователю будет разрешено вводить пароль для аутентификации, если проверка не может быть выполнена с использованием открытого ключа.

Разрешить аутентификацию RSA?
Если для этого параметра установлено значение «Нет», пользователям всегда нужно будет вводить свой пароль, даже если был установлен их открытый ключ.

Проверить разрешения для ключевых файлов?
Если установлено значение «Да», сервер SSH будет проверять каталог каждого пользователя .ssh и любые родительские каталоги, чтобы убедиться, что они не доступны для записи в группе, и файл .ssh / identity, чтобы убедиться, что он не доступен для чтения кем-либо еще. Если установлено значение Нет, проверка прав доступа к файлам не производится.
