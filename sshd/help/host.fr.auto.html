Cette page affiche les options pour un seul hôte, les hôtes dont les noms correspondent à un modèle ou tous les hôtes. Certaines des options les moins évidentes sont:
Connectez-vous en tant qu'utilisateur
Normalement, si aucun nom d'utilisateur n'est donné sur la ligne de commande ssh , le nom de l'utilisateur actuel est utilisé pour se connecter au serveur SSH distant. Cependant, cette option peut être utilisée pour spécifier un nom d'utilisateur par défaut différent pour un hôte ou des hôtes particuliers.

Caractère d'échappement
Lorsque vous effectuez une connexion SSH interactive, le caractère d'échappement peut être utilisé pour rompre la connexion et la fermer ou la suspendre.

Compresser le trafic SSH?
Si cette option est activée, le client SSH compressera toutes les données envoyées à cet hôte avec l'algorithme gzip . Cela peut être utile si vous copiez des fichiers volumineux avec scp sur une liaison lente.

Ports locaux à transmettre au serveur
Dans ce tableau, vous pouvez entrer des numéros de port locaux qui seront transmis à un hôte et un port par le serveur SSH. Cela peut être utile si votre seul accès à un réseau est via la connexion SSH à une machine sur ce réseau et que vous souhaitez accéder à d'autres services tels que les serveurs Web ou POP.

Ports de serveur à transmettre au local
Dans ce tableau, vous pouvez entrer des numéros de port sur le serveur qui seront transférés vers un hôte et un port du réseau de la machine client.
