Opções diversas
Esta página contém opções que não se enquadram em nenhuma das outras categorias. Algumas das opções menos óbvias, mas úteis, são:
Permitir encaminhamento de conexão X11?
Se definido como Sim, os usuários que fizerem um login SSH a partir de uma máquina Unix poderão executar aplicativos X no servidor e ter a conexão X ativada de volta através da conexão SSH para a exibição local.

Recurso de registro do sistema
O recurso syslog usado para registrar mensagens de erro e informações do servidor SSH. Isso pode ser usado em conjunto com o módulo System Logs para controlar em qual arquivo as mensagens SSH são gravadas.

Intervalo de regeneração da chave do servidor
Com que frequência o servidor SSH gerou novamente a chave usada para criptografar conexões. Se você é paranóico em relação à segurança, defina-o como um número menor.
