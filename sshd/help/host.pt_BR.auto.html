Esta página exibe opções para um único host, hosts cujos nomes correspondem a algum padrão ou todos os hosts. Algumas das opções menos óbvias são:
Entrar como usuário
Normalmente, se nenhum nome de usuário for fornecido na linha de comando ssh , o nome do usuário atual será usado para efetuar login no servidor SSH remoto. No entanto, esta opção pode ser usada para especificar um nome de usuário padrão diferente para um ou mais hosts específicos.

Personagem de escape
Ao fazer um logon SSH interativo, o caractere de escape pode ser usado para interromper a conexão e fechá-la ou suspendê-la.

Compactar tráfego SSH?
Se essa opção estiver ativada, o cliente SSH compactará todos os dados enviados para esse host com o algoritmo gzip . Isso pode ser útil se você estiver copiando arquivos grandes com scp em um link lento.

Portas locais a serem encaminhadas para o servidor
Nesta tabela, você pode inserir números de porta local que serão encaminhados para algum host e porta pelo servidor SSH. Isso pode ser útil se seu único acesso a alguma rede for via login SSH em uma máquina nessa rede e você desejar acessar outros serviços, como servidores Web ou POP.

Portas do servidor para encaminhar para local
Nesta tabela, você pode inserir números de porta no servidor que serão encaminhados para alguns hosts e portas na rede da máquina cliente.
