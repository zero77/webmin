기타 옵션
이 페이지에는 다른 범주에 맞지 않는 옵션이 있습니다. 덜 명백한 옵션 중 일부는 유용하지만 다음과 같습니다.
X11 연결 전달을 허용 하시겠습니까?
예로 설정하면 Unix 시스템에서 SSH 로그인을하는 사용자는 서버에서 X 응용 프로그램을 실행할 수 있으며 SSH 연결을 통해 X 연결을 로컬 디스플레이로 다시 터널링 할 수 있습니다.

시스템 로그 기능
SSH 서버에서 오류 및 정보 메시지를 기록하는 데 사용되는 syslog 기능입니다. 이것은 시스템 로그 모듈과 함께 사용되어 어떤 파일 SSH 메시지를 쓸 것인지 제어 할 수 있습니다.

서버 키 재생성 간격
SSH 서버가 연결 암호화에 사용 된 키를 재생성하는 빈도입니다. 보안에 대한 편집증이라면, 이것을 더 낮은 숫자로 설정하십시오.
