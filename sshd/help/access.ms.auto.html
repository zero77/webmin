Kawalan Akses
Bergantung pada versi SSH yang anda gunakan, satu atau dua jenis kawalan akses dapat disediakan di halaman ini.

Untuk kawalan akses berasaskan host, anda boleh membenarkan atau menolak host dengan memasukkan senarai nama host (seperti www.foo.com ), alamat IP atau corak nama host (seperti *.foo.com ). Kawalan akses seperti ini tidak disokong oleh semua versi pelayan SSH.

Untuk kawalan akses berdasarkan pengguna atau kumpulan, nama pengguna boleh berupa akaun masuk tempatan (seperti jcameron ), corak akaun (seperti jcam * ) atau gabungan nama masuk dan nama host pelanggan (seperti jcameron@www.foo.com ). Kumpulan yang dibenarkan atau ditolak juga boleh menjadi nama kumpulan biasa atau corak nama kumpulan, tetapi tidak boleh memasukkan nama host.
