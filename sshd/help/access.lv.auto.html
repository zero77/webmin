Piekļuves kontrole
Atkarībā no izmantotās SSH versijas šajā lapā var iestatīt viena vai divu veidu piekļuves kontroli.

Uz resursdatoriem balstītas piekļuves kontrolei varat atļaut vai noraidīt resursdatorus, ievadot saimniekdatoru vārdu (piemēram, www.foo.com ), IP adrešu vai resursdatora nosaukuma paraugu (piemēram, *.foo.com ) sarakstu. Tomēr šāda veida piekļuves kontroli neatbalsta visas SSH servera versijas.

Uz lietotāju vai grupu balstītai piekļuves kontrolei lietotāju vārdi var būt gan vietējie pieteikšanās konti (piemēram, jcameron ), kontu paraugi (piemēram, jcam * ), gan pieteikšanās un klienta resursdatora kombinācija (piemēram, jcameron@www.foo.com ). Atļautās vai noraidītās grupas var būt arī vienkāršie grupu nosaukumi vai grupas nosaukuma paraugi, bet tajos nevar iekļaut resursdatora nosaukumus.
