Pilihan Hos Pelanggan
Halaman ini memaparkan corak hos dan hos yang mana pilihan pelanggan SSH telah ditentukan. Ini digunakan ketika program klien SSH dijalankan di sistem anda untuk masuk ke pelayan SSH lain, dengan set pilihan yang berlaku ditentukan oleh nama host yang ditentukan dalam perintah ssh .

Biasanya secara lalai satu-satunya pilihan yang ditentukan adalah pilihan yang berlaku untuk semua host pelanggan. Walau bagaimanapun, anda boleh membuat set pilihan tambahan untuk mengawal bagaimana pengguna masuk ke pelayan atau pelayan tertentu yang namanya sesuai dengan beberapa corak.
