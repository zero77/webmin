Διακομιστής SSH
Το SSH είναι ένα πρωτόκολλο που επιτρέπει στους χρήστες να συνδέονται στο μηχάνημά σας μέσω δικτύου, όπως το telnet. Ωστόσο, όλες οι συνδέσεις SSH είναι κρυπτογραφημένες και επικυρωμένες και στα δύο άκρα για να αποτρέψουν οι εισβολείς να καταγράψουν κωδικούς πρόσβασης ή να ανακατευθύνουν μια σύνδεση SSH σε άλλον διακομιστή.

Επειδή το SSH μπορεί να ρυθμιστεί ώστε να χρησιμοποιεί κλειδιά RSA για έλεγχο ταυτότητας αντί για το κανονικό όνομα χρήστη και κωδικό πρόσβασης που χρησιμοποιείται από το telnet, οι χρήστες μπορούν να συνδεθούν σε έναν απομακρυσμένο διακομιστή χωρίς να χρειάζεται να εισάγουν κωδικό πρόσβασης κάθε φορά. Αυτό παρέχει παρόμοια συμπεριφορά με το RSH, αλλά χωρίς τα προβλήματα ασφαλείας που προέρχονται από τον έλεγχο ταυτότητας των χρηστών μόνο με βάση τη διεύθυνση IP τους.

Αυτή η ενότητα σάς επιτρέπει να διαμορφώσετε διάφορες επιλογές για τον διακομιστή SSH που είναι εγκατεστημένος στο σύστημά σας, καθώς και την επιλογή που χρησιμοποιείται όταν ένας πελάτης SSH πραγματοποιεί σύνδεση από το μηχάνημά σας σε άλλο διακομιστή. Αφού πραγματοποιήσετε αλλαγές στην επιλογή διακομιστή, πρέπει να κάνετε κλικ στο κουμπί Εφαρμογή αλλαγών στο κάτω μέρος της σελίδας για να εφαρμοστούν οι αλλαγές.
