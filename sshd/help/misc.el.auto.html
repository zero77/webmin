Διάφορες επιλογές
Αυτή η σελίδα περιέχει επιλογές που δεν ταιριάζουν σε καμία από τις άλλες κατηγορίες. Μερικές από τις λιγότερο προφανείς επιλογές αλλά χρήσιμες είναι:
Να επιτρέπεται η προώθηση σύνδεσης X11;
Εάν οριστεί σε Ναι, οι χρήστες που πραγματοποιούν είσοδο SSH από μηχανή Unix θα μπορούν να εκτελούν εφαρμογές X στον διακομιστή και να έχουν τη σύνδεση X συντονισμένη μέσω της σύνδεσης SSH στην τοπική τους οθόνη.

Δυνατότητα καταγραφής συστήματος
Η δυνατότητα syslog που χρησιμοποιείται για την καταγραφή μηνυμάτων σφαλμάτων και πληροφοριών από τον διακομιστή SSH. Αυτό μπορεί να χρησιμοποιηθεί σε συνδυασμό με τη λειτουργική μονάδα καταγραφής συστήματος για τον έλεγχο σε ποιο αρχείο γράφονται τα μηνύματα SSH.

Διάστημα αναγέννησης κλειδιού διακομιστή
Πόσο συχνά ο διακομιστής SSH δημιούργησε ξανά το κλειδί που χρησιμοποιείται για την κρυπτογράφηση συνδέσεων. Εάν είστε παρανοϊκοί για την ασφάλεια, ορίστε το σε χαμηλότερο αριθμό.
