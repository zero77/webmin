Trabalho em rede
Esta página é para configurar opções de rede para o servidor SSH. Algumas das opções menos óbvias são:
Tempo limite inativo
Se isso estiver definido e não houver nada enviado pelo cliente ou servidor durante o período definido, o cliente será desconectado.

Desconectar se o cliente falhou?
Se definido como Sim, o servidor SSH verificará periodicamente se o cliente ainda está ativo. Caso contrário (devido a um erro de rede ou falha na máquina do cliente), ele será desconectado.

Permitir encaminhamento de TCP?
Se essa opção estiver configurada, os usuários poderão encapsular conexões TCP de máquinas clientes para máquinas na rede do servidor e também permitir conexões de volta do servidor para máquinas na rede cliente.
