Server SSH
SSH huwa protokoll li jippermetti lill-utenti jidħlu fil-magna tiegħek fuq netwerk, bħal telnet. Madankollu, il-konnessjonijiet SSH kollha huma kriptati u awtentikati fiż-żewġt itruf biex jipprevjenu l-attakkanti milli jaqbdu passwords jew jirindirizzaw login SSH għal server ieħor.

Minħabba li SSH jista 'jiġi installat biex juża ċ-ċwievet RSA għall-awtentikazzjoni minflok il-username u l-password normali użati minn telnet, l-utenti jistgħu jidħlu fis-server remot mingħajr ma jkollhom bżonn jidħlu password kull darba. Dan jipprovdi mġieba simili għal RSH, iżda mingħajr il-problemi ta 'sigurtà li jiġu mill-utenti ta' l-awtentikazzjoni bbażati fuq l-indirizz IP tagħhom biss.

Dan il-modulu jippermettilek li tikkonfigura diversi għażliet għas-server SSH installat fis-sistema tiegħek, kif ukoll għażla użata meta klijent SSH jagħmel konnessjoni mill-magna tiegħek għal server ieħor. Wara li tagħmel kwalunkwe tibdil fl-għażla tas-server, trid tikklikkja l-buttuna Applika Tibdil fil-qiegħ tal-paġna biex il-bidliet jidħlu fis-seħħ.
