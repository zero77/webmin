Halaman ini memaparkan pilihan untuk satu host, host yang namanya sesuai dengan beberapa corak, atau semua host. Beberapa pilihan yang kurang jelas adalah:
Log masuk sebagai pengguna
Biasanya jika tidak ada nama pengguna yang diberikan pada baris perintah ssh , nama pengguna semasa digunakan untuk masuk ke pelayan SSH jauh. Walau bagaimanapun, pilihan ini dapat digunakan untuk menentukan nama pengguna lalai yang berbeza untuk host atau host tertentu.

Watak melarikan diri
Semasa membuat log masuk SSH interaktif, watak melarikan diri dapat digunakan untuk memutuskan hubungan dan menutup atau menangguhkannya.

Memampatkan trafik SSH?
Sekiranya pilihan ini diaktifkan, pelanggan SSH akan memampatkan semua data yang dihantar ke host ini dengan algoritma gzip . Ini boleh berguna jika anda menyalin fail besar dengan scp melalui pautan perlahan.

Port tempatan untuk diteruskan ke pelayan
Dalam jadual ini anda boleh memasukkan nombor port tempatan yang akan diteruskan ke beberapa host dan port oleh pelayan SSH. Ini berguna sekiranya satu-satunya akses anda ke beberapa rangkaian adalah melalui log masuk SSH ke satu mesin di rangkaian tersebut, dan anda ingin mengakses perkhidmatan lain seperti pelayan web atau POP.

Pelabuhan pelayan untuk diteruskan ke tempatan
Dalam jadual ini anda boleh memasukkan nombor port pada pelayan yang akan diteruskan ke beberapa host dan port di rangkaian mesin pelanggan.
