Аутентифікація
Ця сторінка призначена для налаштування параметрів, пов'язаних з автентифікацією, для вашого SSH-сервера. Деякі з менш очевидних варіантів:
Дозволити автентифікацію паролем
Якщо для цього встановлено значення Ні, користувачам потрібно буде встановити їх .ssh- каталог із відкритим ключем, перш ніж мати можливість увійти. Якщо встановлено значення Так, користувачеві буде дозволено ввести пароль для автентифікації, якщо перевірку неможливо виконати за допомогою відкритого ключа.

Дозволити аутентифікацію RSA?
Якщо для цього встановлено значення Ні, користувачам завжди потрібно буде ввести свій пароль, навіть якщо їх відкритий ключ налаштовано.

Перевірити дозволи на ключові файли?
Якщо встановлено значення Так, SSH-сервер перевіряє .ssh- каталог кожного користувача та будь-які батьківські каталоги, щоб переконатися, що вони не підлягають запису групи, а також .ssh / ідентифікаційний файл, щоб переконатися, що його ніхто не читає. Якщо встановлено значення Ні, перевірка дозволів на файли не проводиться.
