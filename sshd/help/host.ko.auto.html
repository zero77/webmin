이 페이지에는 단일 호스트, 이름이 일부 패턴과 일치하는 호스트 또는 모든 호스트에 대한 옵션이 표시됩니다. 덜 명백한 옵션 중 일부는 다음과 같습니다.
사용자로 로그인
일반적으로 ssh 명령 행에 사용자 이름이 없으면 현재 사용자 이름이 원격 SSH 서버에 로그인하는 데 사용됩니다. 그러나이 옵션을 사용하여 특정 호스트에 다른 기본 사용자 이름을 지정할 수 있습니다.

탈출 캐릭터
대화식 SSH 로그인을 수행 할 때 이스케이프 문자를 사용하여 연결을 끊고 종료하거나 중단 할 수 있습니다.

SSH 트래픽을 압축 하시겠습니까?
이 옵션을 사용하면 SSH 클라이언트는 gzip 알고리즘을 사용하여이 호스트로 전송 된 모든 데이터를 압축합니다. 느린 링크를 통해 scp 를 사용하여 큰 파일을 복사하는 경우 유용 할 수 있습니다.

서버로 전달할 로컬 포트
이 표에는 SSH 서버가 일부 호스트 및 포트로 전달할 로컬 포트 번호를 입력 할 수 있습니다. 이는 일부 네트워크에 대한 유일한 액세스가 해당 네트워크의 한 시스템에 대한 SSH 로그인을 통해 이루어지고 웹 또는 POP 서버와 같은 다른 서비스에 액세스하려는 경우 유용합니다.

로컬로 전달할 서버 포트
이 테이블에서 서버의 포트 번호를 입력하여 클라이언트 시스템 네트워크의 일부 호스트 및 포트로 전달할 수 있습니다.
