Δικτύωση
Αυτή η σελίδα προορίζεται για τη διαμόρφωση επιλογών δικτύωσης για τον διακομιστή SSH. Μερικές από τις λιγότερο προφανείς επιλογές είναι:
Ανενεργό χρονικό όριο
Εάν έχει ρυθμιστεί και δεν έχει αποσταλεί τίποτα από τον πελάτη ή τον διακομιστή κατά τη διάρκεια της περιόδου που ορίστηκε, ο πελάτης θα αποσυνδεθεί.

Αποσύνδεση εάν ο πελάτης έχει καταρρεύσει;
Εάν οριστεί σε Ναι, ο διακομιστής SSH θα ελέγχει περιοδικά για να δει εάν ο πελάτης είναι ακόμα ζωντανός. Εάν όχι (λόγω σφάλματος δικτύου ή διακοπής λειτουργίας υπολογιστή-πελάτη) θα αποσυνδεθεί.

Να επιτρέπεται η προώθηση TCP;
Εάν αυτή η επιλογή έχει οριστεί, οι χρήστες μπορούν να διοχετεύσουν συνδέσεις TCP από υπολογιστές-πελάτες σε μηχανήματα στο δίκτυο του διακομιστή και επίσης να επιτρέψουν συνδέσεις από τον διακομιστή σε υπολογιστές στο δίκτυο πελατών.
