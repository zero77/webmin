Options d'hôte client
Cette page affiche les hôtes et les modèles d'hôtes pour lesquels les options du client SSH ont été définies. Ils sont utilisés lorsque le programme client SSH est exécuté sur votre système pour vous connecter à un autre serveur SSH, l'ensemble des options applicables étant déterminé par le nom d'hôte spécifié dans la commande ssh .

Normalement, par défaut, les seules options définies seront celles qui s'appliquent à tous les hôtes clients. Cependant, vous pouvez créer des ensembles d'options supplémentaires afin de contrôler la façon dont les utilisateurs se connectent à d'autres serveurs spécifiques ou aux serveurs dont les noms correspondent à un modèle.
