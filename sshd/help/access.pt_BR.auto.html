Controle de acesso
Dependendo da versão do SSH que você está usando, um ou dois tipos de controle de acesso podem ser configurados nesta página.

Para controle de acesso baseado em host, você pode permitir ou negar hosts digitando uma lista de nomes de host (como www.foo.com ), endereços IP ou padrões de nome de host (como *.foo.com ). Esse tipo de controle de acesso não é suportado por todas as versões do servidor SSH.

Para controle de acesso baseado em usuário ou grupo, os nomes de usuário podem ser contas de login local (como jcameron ), padrões de conta (como jcam * ) ou uma combinação de login e nome de host do cliente (como jcameron@www.foo.com ). Grupos permitidos ou negados também podem ser nomes simples de grupo ou padrões de nome de grupo, mas não podem incluir nomes de host.
