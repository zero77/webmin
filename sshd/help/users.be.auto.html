Аўтэнтыфікацыя
Гэта старонка для налады параметраў аўтэнтыфікацыі для вашага SSH-сервера. Некаторыя з менш відавочных варыянтаў:
Дазволіць праверку сапраўднасці з дапамогай пароля
Калі для гэтага ўстаноўлена Не, карыстальнікам трэба будзе наладзіць іх .ssh- каталог з адкрытым ключом, перш чым мець магчымасць увайсці ў сістэму. Калі ўстаноўлена "Так", карыстачу будзе дазволена ўвесці пароль для праверкі сапраўднасці, калі праверку немагчыма зрабіць з дапамогай адкрытага ключа.

Дазволіць праверку сапраўднасці RSA?
Калі для гэтага ўстаноўлена Не, карыстальнікам заўсёды трэба будзе ўвесці свой пароль, нават калі быў адкрыты іх адкрыты ключ.

Праверце дазволы на ключавыя файлы?
Калі ўстаноўлена "Так", сервер SSH правярае .ssh каталог кожнага карыстальніка і любыя бацькоўскія каталогі, каб пераканацца, што яны не падлягаюць запісу ў групу, і .ssh / файл ідэнтыфікацыі, каб пераканацца, што ён не чытаецца кім-небудзь іншым. Калі ўстаноўлена "Не", праверка дазволаў файлаў не будзе ажыццяўляцца.
