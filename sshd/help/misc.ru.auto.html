Разные варианты
Эта страница содержит параметры, которые не вписываются ни в одну из других категорий. Некоторые из менее очевидных, но полезных вариантов:
Разрешить переадресацию соединения X11?
Если установлено значение «Да», пользователи, которые выполняют SSH-вход с компьютера Unix, смогут запускать X-приложения на сервере и получать туннельное X-соединение через SSH-соединение с локальным дисплеем.

Системный журнал
Средство системного журнала, которое используется для регистрации сообщений об ошибках и информационных сообщений от сервера SSH. Это можно использовать вместе с модулем «Системные журналы» для управления тем, в какой файл записываются сообщения SSH.

Интервал регенерации ключа сервера
Как часто сервер SSH повторно генерирует ключ, используемый для шифрования соединений. Если вы не уверены в безопасности, установите для этого параметра меньшее значение.
