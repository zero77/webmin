ตัวเลือกเบ็ดเตล็ด
หน้านี้มีตัวเลือกที่ไม่สอดคล้องกับหมวดหมู่อื่น ๆ ตัวเลือกบางอย่างที่ชัดเจนน้อยลง แต่มีประโยชน์คือ:
อนุญาตการส่งต่อการเชื่อมต่อ X11 หรือไม่
หากตั้งค่าเป็นใช่ผู้ใช้ที่ลงชื่อเข้าใช้ SSH จากเครื่อง Unix จะสามารถเรียกใช้แอปพลิเคชัน X บนเซิร์ฟเวอร์และให้การเชื่อมต่อ X ถูกเชื่อมต่อกลับผ่านการเชื่อมต่อ SSH ไปยังจอแสดงผลในเครื่อง

ระบบบันทึกการทำงาน
สิ่งอำนวยความสะดวก syslog ที่ใช้เพื่อบันทึกข้อผิดพลาดและข้อความข้อมูลจากเซิร์ฟเวอร์ SSH สิ่งนี้สามารถใช้ร่วมกับโมดูล System Logs เพื่อควบคุมข้อความ SSH ของไฟล์ที่เขียน

ช่วงเวลาการสร้างคีย์เซิร์ฟเวอร์
เซิร์ฟเวอร์ SSH สร้างคีย์ที่ใช้สำหรับการเข้ารหัสการเชื่อมต่อบ่อยครั้งเพียงใด หากคุณหวาดระแวงเกี่ยวกับความปลอดภัยให้ตั้งเป็นจำนวนที่ต่ำกว่า
