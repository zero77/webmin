Options diverses
Cette page contient des options qui ne rentrent dans aucune des autres catégories. Certaines des options les moins évidentes mais utiles sont:
Autoriser le transfert de connexion X11?
S'il est défini sur Oui, les utilisateurs qui établissent une connexion SSH à partir d'une machine Unix pourront exécuter des applications X sur le serveur et voir la connexion X tunnelée via la connexion SSH vers leur affichage local.

Fonction de journal système
La fonction syslog utilisée pour consigner les messages d'erreur et d'information du serveur SSH. Ceci peut être utilisé conjointement avec le module System Logs pour contrôler dans quel fichier les messages SSH sont écrits.

Intervalle de régénération des clés du serveur
Fréquence à laquelle le serveur SSH a recréé la clé utilisée pour chiffrer les connexions. Si vous êtes paranoïaque à propos de la sécurité, définissez-le sur un nombre inférieur.
