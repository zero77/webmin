Server SSH
SSH este un protocol care permite utilizatorilor să se conecteze la aparatul dvs. printr-o rețea, la fel ca telnet. Cu toate acestea, toate conexiunile SSH sunt criptate și autentificate la ambele capete pentru a împiedica atacatorii să capteze parole sau să redirecționeze o autentificare SSH către un alt server.

Deoarece SSH poate fi configurat pentru a utiliza cheile RSA pentru autentificare în loc de numele de utilizator și parola obișnuite utilizate de telnet, utilizatorii se pot autentifica pe un server de la distanță fără a fi nevoie să introduceți o parolă de fiecare dată. Acest lucru oferă un comportament similar cu RSH, dar fără probleme de securitate care provin din autentificarea utilizatorilor doar pe adresa lor IP.

Acest modul vă permite să configurați diferite opțiuni pentru serverul SSH instalat pe sistemul dvs., precum și opțiunea folosită atunci când un client SSH face o conexiune de la mașina dvs. la un alt server. După efectuarea oricărei modificări de opțiuni de server, trebuie să faceți clic pe butonul Aplicați modificările din partea de jos a paginii pentru ca modificările să aibă efect.
