SSH 서버
SSH는 사용자가 텔넷과 같이 네트워크를 통해 시스템에 로그인 할 수있는 프로토콜입니다. 그러나 공격자가 암호를 캡처하거나 SSH 로그인을 다른 서버로 리디렉션하지 못하도록 모든 SSH 연결이 양쪽 끝에서 암호화 및 인증됩니다.

텔넷에서 사용하는 일반 사용자 이름 및 비밀번호 대신 인증을 위해 RSA 키를 사용하도록 SSH를 설정할 수 있으므로 사용자는 매번 비밀번호를 입력하지 않고도 원격 서버에 로그인 할 수 있습니다. 이는 RSH와 유사한 동작을 제공하지만 IP 주소만으로 사용자를 인증 할 때 발생하는 보안 문제는 없습니다.

이 모듈을 사용하면 시스템에 설치된 SSH 서버에 대한 다양한 옵션과 SSH 클라이언트가 시스템에서 다른 서버로 연결할 때 사용되는 옵션을 구성 할 수 있습니다. 서버 옵션을 변경 한 후에는 페이지 하단의 변경 사항 적용 버튼을 클릭하여 변경 사항을 적용해야합니다.
