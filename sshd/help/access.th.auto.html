การควบคุมการเข้าถึง
ขึ้นอยู่กับเวอร์ชั่นของ SSH ที่คุณใช้สามารถตั้งค่าการควบคุมการเข้าถึงหนึ่งหรือสองชนิดในหน้านี้

สำหรับการควบคุมการเข้าถึงโดยใช้โฮสต์คุณสามารถอนุญาตหรือปฏิเสธโฮสต์โดยป้อนรายการชื่อโฮสต์ (เช่น www.foo.com ) ที่อยู่ IP หรือรูปแบบชื่อโฮสต์ (เช่น *.foo.com ) การควบคุมการเข้าถึงชนิดนี้ไม่ได้รับการรองรับโดยเซิร์ฟเวอร์ SSH ทุกรุ่น

สำหรับการควบคุมการเข้าถึงตามผู้ใช้หรือกลุ่มชื่อผู้ใช้อาจเป็นบัญชีล็อกอินท้องถิ่น (เช่น jcameron ) รูปแบบบัญชี (เช่น jcam * ) หรือการรวมกันของชื่อเข้าสู่ระบบและชื่อโฮสต์ของลูกค้า (เช่น jcameron@www.foo.com ) กลุ่มที่ได้รับอนุญาตหรือถูกปฏิเสธอาจเป็นชื่อกลุ่มธรรมดาหรือรูปแบบชื่อกลุ่ม แต่ไม่สามารถมีชื่อโฮสต์ได้
