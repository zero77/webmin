Параметры хоста клиента
На этой странице отображаются узлы и шаблоны узлов, для которых определены параметры клиента SSH. Они используются, когда клиентская программа SSH запускается в вашей системе для входа на другой сервер SSH, причем применяемый набор параметров определяется именем хоста, указанным в команде ssh .

Обычно по умолчанию определяются только те параметры, которые применяются ко всем клиентским хостам. Однако вы можете создать дополнительные наборы параметров, чтобы контролировать, как пользователи входят в систему на других конкретных серверах или серверах, имена которых соответствуют некоторому шаблону.
