Serveur SSH
SSH est un protocole qui permet aux utilisateurs de se connecter à votre machine via un réseau, un peu comme telnet. Cependant, toutes les connexions SSH sont cryptées et authentifiées aux deux extrémités pour empêcher les attaquants de capturer des mots de passe ou de rediriger une connexion SSH vers un autre serveur.

Étant donné que SSH peut être configuré pour utiliser des clés RSA pour l'authentification au lieu du nom d'utilisateur et du mot de passe normaux utilisés par telnet, les utilisateurs peuvent se connecter à un serveur distant sans avoir à entrer un mot de passe à chaque fois. Cela fournit un comportement similaire à RSH, mais sans les problèmes de sécurité qui proviennent de l'authentification des utilisateurs basée uniquement sur leur adresse IP.

Ce module vous permet de configurer diverses options pour le serveur SSH installé sur votre système, ainsi que l'option utilisée lorsqu'un client SSH établit une connexion entre votre machine et un autre serveur. Après avoir apporté des modifications aux options du serveur, vous devez cliquer sur le bouton Appliquer les modifications au bas de la page pour que les modifications prennent effet.
