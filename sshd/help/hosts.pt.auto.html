Opções de host do cliente
Esta página exibe hosts e padrões de host para os quais as opções do cliente SSH foram definidas. Eles são usados quando o programa cliente SSH é executado em seu sistema para efetuar login em outro servidor SSH, com o conjunto de opções aplicáveis sendo determinado pelo nome do host especificado no comando ssh .

Normalmente, por padrão, as únicas opções definidas serão aquelas que se aplicam a todos os hosts clientes. No entanto, você pode criar conjuntos adicionais de opções para controlar como os usuários efetuam login em outros servidores ou servidores específicos cujos nomes correspondem a algum padrão.
