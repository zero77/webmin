หน้านี้จะแสดงตัวเลือกสำหรับโฮสต์เดียวโฮสต์ที่มีชื่อตรงกับรูปแบบบางส่วนหรือโฮสต์ทั้งหมด ตัวเลือกที่ชัดเจนน้อยกว่าคือ:
เข้าสู่ระบบในฐานะผู้ใช้
โดยปกติหากไม่มีการกำหนดชื่อผู้ใช้ในบรรทัดคำสั่ง ssh ชื่อของผู้ใช้ปัจจุบันจะถูกใช้เพื่อเข้าสู่เซิร์ฟเวอร์ SSH ระยะไกล อย่างไรก็ตามตัวเลือกนี้สามารถใช้เพื่อระบุชื่อผู้ใช้เริ่มต้นที่แตกต่างกันสำหรับโฮสต์หรือโฮสต์ที่เฉพาะเจาะจง

หลบหนีตัวละคร
เมื่อทำการล็อกอิน SSH แบบโต้ตอบอักขระเลี่ยงสามารถใช้เพื่อแยกการเชื่อมต่อและปิดหรือระงับ

บีบอัดทราฟฟิก SSH หรือไม่
หากเปิดใช้งานตัวเลือกนี้ไคลเอ็นต์ SSH จะบีบอัดข้อมูลทั้งหมดที่ส่งไปยังโฮสต์นี้ด้วยอัลกอริทึม gzip สิ่งนี้มีประโยชน์หากคุณกำลังคัดลอกไฟล์ขนาดใหญ่ที่มี scp ผ่านลิงค์ช้า

พอร์ตในเครื่องเพื่อส่งต่อไปยังเซิร์ฟเวอร์
ในตารางนี้คุณสามารถป้อนหมายเลขพอร์ตท้องถิ่นซึ่งจะถูกส่งต่อไปยังโฮสต์และพอร์ตโดยเซิร์ฟเวอร์ SSH สิ่งนี้มีประโยชน์หากการเข้าถึงเครือข่ายบางอย่างของคุณคือการเข้าสู่ระบบ SSH ไปยังเครื่องหนึ่งในเครือข่ายนั้นและคุณต้องการเข้าถึงบริการอื่น ๆ เช่นเว็บหรือเซิร์ฟเวอร์ POP

พอร์ตเซิร์ฟเวอร์เพื่อส่งต่อไปยังโลคัล
ในตารางนี้คุณสามารถป้อนหมายเลขพอร์ตบนเซิร์ฟเวอร์ซึ่งจะถูกส่งต่อไปยังโฮสต์และพอร์ตบางส่วนในเครือข่ายของเครื่องไคลเอนต์
