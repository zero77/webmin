Auf dieser Seite werden Optionen für einen einzelnen Host, Hosts, deren Namen einem bestimmten Muster entsprechen, oder für alle Hosts angezeigt. Einige der weniger offensichtlichen Optionen sind:
Melden Sie sich als Benutzer an
Wenn in der ssh- Befehlszeile kein Benutzername angegeben ist, wird normalerweise der Name des aktuellen Benutzers verwendet, um sich beim Remote-SSH-Server anzumelden. Diese Option kann jedoch verwendet werden, um einen anderen Standardbenutzernamen für einen bestimmten Host oder bestimmte Hosts anzugeben.

Escape-Zeichen
Bei einer interaktiven SSH-Anmeldung kann das Escape-Zeichen verwendet werden, um die Verbindung zu trennen und zu schließen oder auszusetzen.

SSH-Verkehr komprimieren?
Wenn diese Option aktiviert ist, komprimiert der SSH-Client alle an diesen Host gesendeten Daten mit dem gzip- Algorithmus. Dies kann nützlich sein, wenn Sie große Dateien mit scp über einen langsamen Link kopieren.

Lokale Ports, die an den Server weitergeleitet werden sollen
In dieser Tabelle können Sie lokale Portnummern eingeben, die vom SSH-Server an einen Host und einen Port weitergeleitet werden. Dies kann nützlich sein, wenn Sie nur über die SSH-Anmeldung an einem Computer in diesem Netzwerk auf ein Netzwerk zugreifen und auf andere Dienste wie Web- oder POP-Server zugreifen möchten.

Server-Ports zur Weiterleitung an lokale
In dieser Tabelle können Sie Portnummern auf dem Server eingeben, die an einen Host und einen Port im Netzwerk des Clientcomputers weitergeleitet werden.
