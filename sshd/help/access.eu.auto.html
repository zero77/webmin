Sarbide-kontrola
Erabiltzen duzun SSH bertsioaren arabera, sarbide kontrol mota bat edo bi konfigura daiteke orrialde honetan.

Ostalaritzako sarbide kontrolerako, ostalariak baimendu edo ukatu ditzakezu ostalarien izenen ( www.foo.com adibidez), IP helbideen edo ostalariaren izenen patroien (sartuta *.foo.com ) bezalako zerrendak sartuz. Sarbide kontrol mota hau SSH zerbitzariaren bertsio guztiek ez dute onartzen.

Erabiltzaile edo taldean oinarritutako sarbide kontrolerako, erabiltzaile izenak hasierako kontu kontuak ( jcameron bezalakoak), kontu ereduak ( jcam * bezalakoak) edo saioa eta bezero ostalariaren izena konbinatzea ( jcameron@www.foo.com adibidez) izan daitezke. Onartutako edo ukatutako taldeak talde arrunten izenak edo talde izenen ereduak ere izan daitezke, baina ezin dituzte ostalariaren izenak sartu.
