Сервисы, использующие LDAP
В системе Unix различные типы системной информации (например, пользователи, группы и имена хостов) могут быть найдены из разных источников. В простейшем случае все они берутся из локальных файлов в каталоге /etc , таких как /etc/passwd для пользователей, /etc/group для групп и /etc/hosts для имен хостов.

Тем не менее, можно выбрать дополнительные источники для этих данных. Если ваша система является клиентом LDAP, вам необходимо использовать эту страницу, чтобы включить запрос к сконфигурированному серверу LDAP для получения информации о пользователях и группах. Вы должны быть осторожны, чтобы не отключить источник данных Files для пользователей или групп, поскольку это может скрыть важных системных пользователей, таких как root . Вместо этого все, что вам нужно сделать, это выбрать LDAP в качестве второго источника.
