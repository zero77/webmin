Cliente LDAP
Este módulo le permite configurar su sistema como cliente para un servidor LDAP, que proporciona usuarios y grupos de Unix a múltiples sistemas en una red. LDAP es una forma conveniente de compartir cuentas entre sistemas, especialmente aquellos que comparten directorios de inicio usando NFS.

LDAP también se puede usar para compartir servicios de Internet, direcciones de host, grupos de red y otra información. Sin embargo, como estos son típicamente estáticos o provistos por DNS, estas capacidades rara vez se usan.
