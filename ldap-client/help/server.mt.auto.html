Konfigurazzjoni tas-Server LDAP
Din il-paġna tippermettilek tagħżel is-server LDAP li magħha se tikkuntattja din is-sistema tal-klijent sabiex tinkiseb informazzjoni dwar l-utent u l-grupp. L-iktar qasam importanti huwa l- isem tal-host tas-server LDAP , li fih trid tidħol il-hostname jew l-indirizz IP tas-server LDAP fuq in-netwerk tiegħek. Jekk għandek aktar minn server replikat wieħed, dawn kollha jistgħu jiddaħħlu hawn.

L- għeluq tal-Logħob għall-utenti mhux tal-għeruq u l- Password għall-utenti mhux tal-għeruq għandhom jimtlew b’isem tal-utent u password aċċettati mis-server LDAP rispettivament. Il-login normalment ikun isem distint LDAP, bħal cn = Manager, dc = dominju tiegħi, dc = com .

Minħabba li dan il-login u password huma viżibbli għall-utenti kollha Unix fis-sistema tiegħek, tipikament huwa ssettjat għal dak ta 'utent LDAP bi privileġġi limitati (bħal aċċess li jinqara biss, u l-ebda kapaċità li tara l-passwords). Minħabba li l-passwords għandhom bżonn jiġu ċċekkjati f’xi sitwazzjonijiet, l- Login għall-utent għerq u l- Password għall- għelieqi tal- utent tal-għeruq għandhom jiġu ssettjati għal isem tal-utent u password LDAP li jkun qara l-aċċess għas-server LDAP kollu. Din it-tieni password tinħażen f'fajl li jinqara biss mill-utent ta 'l- għeruq , u b'hekk huwa sikur minn utenti regolari ta' Unix mhux fdati.
