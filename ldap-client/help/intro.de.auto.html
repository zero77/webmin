LDAP-Client
Mit diesem Modul können Sie Ihr System als Client für einen LDAP-Server konfigurieren, der Unix-Benutzer und -Gruppen für mehrere Systeme in einem Netzwerk bereitstellt. LDAP ist eine bequeme Möglichkeit, Konten zwischen Systemen freizugeben, insbesondere solchen, die Home-Verzeichnisse mithilfe von NFS gemeinsam nutzen.

LDAP kann auch verwendet werden, um Internetdienste, Hostadressen, Netzgruppen und andere Informationen gemeinsam zu nutzen. Da diese jedoch normalerweise statisch sind oder von DNS bereitgestellt werden, werden diese Funktionen nur selten verwendet.
