Services utilisant LDAP
Sur un système Unix, différents types d'informations système (tels que les utilisateurs, les groupes et les noms d'hôte) peuvent être recherchés à partir de différentes sources. Dans le cas le plus simple, ils sont tous extraits des fichiers locaux du répertoire /etc , tels que /etc/passwd pour les utilisateurs, /etc/group pour les groupes et /etc/hosts pour les noms d'hôte.

Cependant, il est possible de sélectionner des sources supplémentaires pour ces données. Si votre système est un client LDAP, vous devrez utiliser cette page pour activer l'interrogation du serveur LDAP configuré pour les informations sur les utilisateurs et les groupes. Vous devez faire attention à ne pas désactiver la source de données Fichiers pour les utilisateurs ou les groupes, car cela masquerait les utilisateurs système importants tels que root . Au lieu de cela, tout ce que vous devez faire est de sélectionner LDAP comme deuxième source.
