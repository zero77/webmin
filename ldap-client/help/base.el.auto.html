Βάσεις αναζήτησης LDAP
Οι βάσεις δεδομένων ενός διακομιστή LDAP είναι οργανωμένες κληρονομικά, συνήθως με διαφορετικούς κλάδους στο δέντρο για χρήστες, ομάδες και άλλες υπηρεσίες. Αυτή η σελίδα σάς επιτρέπει να επιλέξετε πού στη βάση δεδομένων LDAP θα αναζητήσετε διαφορετικούς τύπους πληροφοριών και πόσο βαθιά θα αναζητήσετε το καθένα.

Από προεπιλογή, όλες οι αναζητήσεις θα πραγματοποιούνται κάτω από την καθολική βάση αναζήτησης . Ωστόσο, επειδή οι περισσότεροι διακομιστές θέτουν χρήστες και ομάδες σε διαφορετικά δευτερεύοντα δέντρα, πιθανότατα θα πρέπει να ορίσετε τους χρήστες Base για Unix και Base για Unix ομάδες στα διακεκριμένα ονόματα για αυτά τα δευτερεύοντα δέντρα. Οι κωδικοί πρόσβασης Base for Unix πρέπει να οριστούν στο ίδιο DN με το Base για χρήστες Unix , καθώς οι πληροφορίες κωδικού πρόσβασης αποθηκεύονται σχεδόν πάντα με άλλα χαρακτηριστικά χρήστη.
