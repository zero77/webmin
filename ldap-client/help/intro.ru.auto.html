Клиент LDAP
Этот модуль позволяет вам настроить вашу систему в качестве клиента для сервера LDAP, который предоставляет пользователям и группам Unix несколько систем в сети. LDAP - это удобный способ совместного использования учетных записей между системами, особенно теми, которые используют домашние каталоги с использованием NFS.

LDAP также может использоваться для обмена интернет-сервисами, адресами хостов, сетевыми группами и другой информацией. Однако, поскольку они обычно являются статическими или предоставляются DNS, эти возможности используются редко.
