Bases de búsqueda LDAP
La base de datos de un servidor LDAP se organiza de forma jerárquica, generalmente con diferentes ramas en el árbol para usuarios, grupos y otros servicios. Esta página le permite seleccionar en qué parte de la base de datos LDAP buscar diferentes tipos de información y qué tan profundo buscar cada uno.

Por defecto, todas las búsquedas se realizarán en la base de búsqueda Global . Sin embargo, debido a que la mayoría de los servidores colocan a los usuarios y grupos bajo diferentes subárboles, probablemente necesitará configurar la Base para usuarios de Unix y la Base para grupos de Unix con los nombres distinguidos para esos subárboles. La base para las contraseñas de Unix debe establecerse en el mismo DN que la base para los usuarios de Unix , ya que la información de la contraseña casi siempre se almacena con otros atributos de usuario.
