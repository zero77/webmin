Konfigurácia servera LDAP
Táto stránka vám umožňuje vybrať server LDAP, s ktorým bude tento klientsky systém kontaktovať, aby sa získali informácie o užívateľoch a skupinách. Najdôležitejším poľom sú názvy hostiteľov servera LDAP , do ktorých musíte zadať názov hostiteľa alebo IP adresu servera LDAP vo vašej sieti. Ak máte viac ako jeden replikovaný server, môžete ich sem zadať všetky.

Polia Login for non-root users a Password for non-root users musia byť vyplnené užívateľským menom a heslom akceptovaným serverom LDAP. Prihlasovanie bude zvyčajne rozlišovacím menom LDAP, napríklad cn = Manager, dc = my-domain, dc = com .

Pretože toto prihlasovacie meno a heslo sú viditeľné pre všetkých používateľov systému Unix vo vašom systéme, zvyčajne je nastavené na používateľské meno LDAP s obmedzenými oprávneniami (napríklad prístup len na čítanie a bez možnosti prezerania hesiel). Pretože v niektorých situáciách je potrebné skontrolovať heslá, polia Login for root user a Password for root user by mali byť nastavené na užívateľské meno a heslo LDAP, ktorý má prístup na čítanie na celý server LDAP. Toto druhé heslo je uložené v súbore, ktorý je čitateľný iba pre koreňového užívateľa, a je teda chránený pred bežnými nedôveryhodnými používateľmi Unixu.
