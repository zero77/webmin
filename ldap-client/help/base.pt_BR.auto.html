Bases de pesquisa LDAP
As bases de dados de um servidor LDAP são organizadas hierarquicamente, geralmente com diferentes ramificações na árvore para usuários, grupos e outros serviços. Esta página permite selecionar onde no banco de dados LDAP procurar diferentes tipos de informações e qual a profundidade de busca de cada uma.

Por padrão, todas as pesquisas serão feitas na base de pesquisa Global . No entanto, como a maioria dos servidores coloca usuários e grupos em diferentes subárvores, você provavelmente precisará definir os usuários do Base for Unix e os grupos do Base for Unix com os nomes distintos dessas subárvores. As senhas do Base for Unix devem ser definidas com o mesmo DN dos usuários do Base for Unix , pois as informações da senha são quase sempre armazenadas com outros atributos do usuário.
