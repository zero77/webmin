Authentifizierungsoptionen
Die Optionen auf dieser Seite steuern, wie Kennwörter in der LDAP-Datenbank überprüft werden. Sie können sie verwenden, um gültige Benutzer auf Mitglieder einer bestimmten Gruppe zu beschränken oder das Verschlüsselungsformat für gespeicherte Kennwörter zu steuern. Sie können jedoch für die meisten Systeme normalerweise unverändert bleiben.
