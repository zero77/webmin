Основы поиска LDAP
База данных сервера LDAP организована иерархически, обычно с разными ветвями в дереве для пользователей, групп и других служб. Эта страница позволяет вам выбрать, где в базе данных LDAP искать различные типы информации, и как глубоко искать каждый из них.

По умолчанию все поиски будут выполняться в глобальной базе поиска . Однако, поскольку большинство серверов помещают пользователей и группы в разные поддеревья, вам, вероятно, потребуется установить для базовых пользователей Unix и базовых групп Unix различающиеся имена для этих поддеревьев. Для паролей Base for Unix должно быть установлено то же DN, что и для Base для пользователей Unix , поскольку информация о паролях почти всегда хранится с другими атрибутами пользователя.
