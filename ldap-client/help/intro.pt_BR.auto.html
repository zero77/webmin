Cliente LDAP
Este módulo permite que você configure seu sistema como um cliente para um servidor LDAP, que fornece usuários e grupos Unix a vários sistemas em uma rede. O LDAP é uma maneira conveniente de compartilhar contas entre sistemas, especialmente aqueles que compartilham diretórios pessoais usando o NFS.

O LDAP também pode ser usado para compartilhar serviços de Internet, endereços de host, grupos de rede e outras informações. No entanto, como normalmente são estáticos ou fornecidos pelo DNS, esses recursos raramente são usados.
