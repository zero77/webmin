Servicios que usan LDAP
En un sistema Unix, se pueden buscar varios tipos de información del sistema (como usuarios, grupos y nombres de host) desde diferentes fuentes. En el caso más simple, todos estos se toman de archivos locales en el directorio /etc , como /etc/passwd para usuarios, /etc/group para grupos y /etc/hosts para nombres de host.

Sin embargo, es posible seleccionar fuentes adicionales para estos datos. Si su sistema es un cliente LDAP, deberá utilizar esta página para habilitar la consulta del servidor LDAP configurado para obtener información de usuarios y grupos. Debe tener cuidado de no desactivar la fuente de datos de Archivos para usuarios o grupos, ya que esto ocultaría usuarios importantes del sistema, como root . En cambio, todo lo que debe hacer es seleccionar LDAP como la segunda fuente.
