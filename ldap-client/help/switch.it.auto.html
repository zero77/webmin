Servizi tramite LDAP
Su un sistema Unix, è possibile cercare vari tipi di informazioni di sistema (come utenti, gruppi e nomi host) da diverse fonti. Nel caso più semplice questi sono tutti presi da file locali nella directory /etc , come /etc/passwd per utenti, /etc/group per gruppi e /etc/hosts per nomi host.

Tuttavia, è possibile selezionare fonti aggiuntive per questi dati. Se il tuo sistema è un client LDAP, dovrai utilizzare questa pagina per abilitare l'interrogazione del server LDAP configurato per informazioni su utenti e gruppi. Prestare attenzione a non disattivare l'origine dati di File per utenti o gruppi, in quanto ciò nasconderebbe importanti utenti del sistema come root . Invece, tutto ciò che dovresti fare è selezionare LDAP come seconda fonte.
