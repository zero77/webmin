Serviços usando LDAP
Em um sistema Unix, vários tipos de informações do sistema (como usuários, grupos e nomes de host) podem ser consultados de diferentes fontes. No caso mais simples, todos eles são obtidos de arquivos locais no diretório /etc , como /etc/passwd para usuários, /etc/group para grupos e /etc/hosts para nomes de host.

No entanto, é possível selecionar fontes adicionais para esses dados. Se o seu sistema for um cliente LDAP, você precisará usar esta página para ativar a consulta ao servidor LDAP configurado para obter informações sobre usuários e grupos. Você deve tomar cuidado para não desativar a fonte de dados Arquivos para usuários ou grupos, pois isso ocultaria usuários importantes do sistema, como raiz . Em vez disso, tudo que você precisa fazer é selecionar LDAP como a segunda fonte.
