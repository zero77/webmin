Πελάτης LDAP
Αυτή η ενότητα σάς επιτρέπει να διαμορφώσετε το σύστημά σας ως πελάτη για διακομιστή LDAP, ο οποίος παρέχει στους χρήστες Unix και ομάδες σε πολλά συστήματα σε ένα δίκτυο. Το LDAP είναι ένας βολικός τρόπος κοινής χρήσης λογαριασμών μεταξύ συστημάτων, ειδικά εκείνων που μοιράζονται οικιακούς καταλόγους χρησιμοποιώντας NFS.

Το LDAP μπορεί επίσης να χρησιμοποιηθεί για κοινή χρήση υπηρεσιών Διαδικτύου, διευθύνσεων κεντρικού υπολογιστή, ομάδων δικτύου και άλλων πληροφοριών. Ωστόσο, καθώς αυτά είναι συνήθως στατικά ή παρέχονται από το DNS, αυτές οι δυνατότητες σπάνια χρησιμοποιούνται.
