Υπηρεσίες που χρησιμοποιούν LDAP
Σε ένα σύστημα Unix, μπορείτε να αναζητήσετε διάφορους τύπους πληροφοριών συστήματος (όπως χρήστες, ομάδες και ονόματα κεντρικών υπολογιστών) από διαφορετικές πηγές. Στην απλούστερη περίπτωση, όλα αυτά λαμβάνονται από τοπικά αρχεία στον κατάλογο /etc , όπως /etc/passwd για χρήστες, /etc/group για ομάδες και /etc/host για ονόματα κεντρικών υπολογιστών.

Ωστόσο, είναι δυνατή η επιλογή πρόσθετων πηγών για αυτά τα δεδομένα. Εάν το σύστημά σας είναι πελάτης LDAP, θα πρέπει να χρησιμοποιήσετε αυτήν τη σελίδα για να ενεργοποιήσετε το ερώτημα του διαμορφωμένου διακομιστή LDAP για πληροφορίες χρήστη και ομάδας. Πρέπει να είστε προσεκτικοί για να μην απενεργοποιήσετε την πηγή δεδομένων αρχείων για χρήστες ή ομάδες, καθώς αυτό θα αποκρύψει σημαντικούς χρήστες του συστήματος, όπως το root . Αντ 'αυτού, το μόνο που πρέπει να κάνετε είναι να επιλέξετε το LDAP ως τη δεύτερη πηγή.
