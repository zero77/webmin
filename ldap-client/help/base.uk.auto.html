Бази пошуку LDAP
Бази даних серверів LDAP організовані герархічно, як правило, з різними гілками дерева, для користувачів, груп та інших служб. Ця сторінка дозволяє вибрати, де в базі даних LDAP можна шукати різні типи інформації та як глибоко їх шукати.

За замовчуванням усі пошукові запити здійснюватимуться в рамках глобальної бази пошуку . Однак, оскільки більшість серверів розміщують користувачів та групи під різними під-деревами, вам, ймовірно, потрібно буде встановити Бази для користувачів Unix та груп Base для Unix розрізненими іменами для цих під дерев. Паролі Base for Unix повинні бути встановлені на той самий DN, що і Base для користувачів Unix , оскільки інформація про пароль майже завжди зберігається з іншими атрибутами користувача.
