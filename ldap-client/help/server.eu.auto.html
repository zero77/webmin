LDAP zerbitzariaren konfigurazioa
Orrialde honek bezero sistema honekin harremanetan jarriko den LDAP zerbitzaria hautatzeko aukera ematen du erabiltzaileen eta taldeen informazioa eskuratzeko. Eremu garrantzitsuena LDAP zerbitzariaren ostalariaren izenak dira , eta bertan sartu behar duzu zure sarean LDAP zerbitzariaren ostalariaren izena edo IP helbidea. Erreplikatutako zerbitzari bat baino gehiago baduzu, hemen sartu daitezke guztiak.

Erro ez diren erabiltzaileentzako saioa eta erroa ez duten erabiltzaileentzako pasahitzak LDAP zerbitzariak onartutako erabiltzaile izena eta pasahitza bete behar dira hurrenez hurren. Saioa normalean LDAP bereizgarria izango da, cn = Manager bezala, dc = my-domain, dc = com .

Saioa hastea eta pasahitza zure sisteman Unix-eko erabiltzaile guztientzat ikusgai dagoenez, normalean pribilegio mugatuak dituen LDAP erabiltzailearentzat ezarrita dago (esaterako, irakurtzeko sarbidea eta pasahitzak ikusteko gaitasunik ez). Pasahitzak zenbait egoeratan egiaztatu behar direlako, root erabiltzailearentzako saioa hasi eta Pasahitza root erabiltzaileentzako eremuetarako LDAP zerbitzari osora eta pasahitza LDAP zerbitzari osora irakurri duen pasahitza ezarri behar da. Bigarren pasahitz hau root erabiltzaileak bakarrik irakur dezakeen fitxategi batean gordetzen da eta, beraz, fidagarriak ez diren Unix erabiltzaileen artean segurua da.
