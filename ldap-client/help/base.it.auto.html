Basi di ricerca LDAP
Le banche dati di un server LDAP sono organizzate in modo gerarchico, in genere con diversi rami nella struttura per utenti, gruppi e altri servizi. Questa pagina consente di selezionare la posizione nel database LDAP in cui cercare diversi tipi di informazioni e la profondità della ricerca di ciascuna.

Per impostazione predefinita, tutte le ricerche verranno eseguite nella base di ricerca globale . Tuttavia, poiché la maggior parte dei server inserisce utenti e gruppi in alberi secondari diversi, sarà probabilmente necessario impostare Base per utenti Unix e Base per gruppi Unix sui nomi distinti per tali alberi secondari. Le password di Base per Unix devono essere impostate sullo stesso DN di base per gli utenti Unix , poiché le informazioni sulla password vengono quasi sempre archiviate con altri attributi utente.
