Options d'authentification
Les options de cette page contrôlent la façon dont les mots de passe sont vérifiés dans la base de données LDAP. Vous pouvez les utiliser pour restreindre les utilisateurs valides aux membres d'un certain groupe ou contrôler le format de chiffrement des mots de passe stockés. Cependant, ils peuvent généralement rester inchangés pour la plupart des systèmes.
