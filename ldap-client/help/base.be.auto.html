Базы пошуку LDAP
Базы дадзеных сервера LDAP арганізаваны герархічна, звычайна з рознымі галінамі дрэва для карыстальнікаў, груп і іншых службаў. Гэтая старонка дазваляе выбраць, дзе ў базе дадзеных LDAP можна шукаць розныя тыпы інфармацыі і як глыбока іх шукаць.

Па змаўчанні ўсе пошукавыя запыты будуць ажыццяўляцца ў рамках глабальнай базы пошуку . Аднак, паколькі большасць сервераў змяшчае карыстальнікаў і групы пад розныя дрэвы, вам, напэўна, трэба будзе ўсталяваць Base для карыстальнікаў Unix і Base для Unix, каб яны адрознівалі імёны гэтых дрэў. Паролі Base для Unix павінны быць усталяваны такім жа DN, як і Базы для карыстальнікаў Unix , бо інфармацыя пра пароль амаль заўсёды захоўваецца разам з іншымі атрыбутамі карыстальніка.
