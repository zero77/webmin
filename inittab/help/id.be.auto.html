Назва працэсу
Id (назва працэсу) уяўляе сабой унікальную паслядоўнасць 1-4 знакаў, якая ідэнтыфікуе запіс у файле inittab (для версій sysvinit, складзеных з бібліятэк <5.2.18 альбо a.out бібліятэкі, ліміт складае 2 знакі).

Заўвага: Для gettys або іншых працэсаў уводу ў поле павінна быць суфікс tty адпаведнага tty, напрыклад1 для tty1. У адваротным выпадку ўлік уваходу можа працаваць няправільна.
