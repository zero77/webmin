Processo
Este campo contém a linha de comandos do programa que deve ser executado no nível de execução especificado (por exemplo, /etc/init.d/rc, /sbin/getty ...).
Se o campo do processo começar com um caractere `+ ', o init não fará contabilização de utmp e wtmp para esse processo. Isso é necessário para as gettys que insistem em fazer suas próprias tarefas domésticas utmp /wtmp. Este também é um bug histórico.
