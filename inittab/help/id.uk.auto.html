Назва процесу
Id (назва процесу) - це унікальна послідовність 1-4 символів, яка ідентифікує запис у файлі inittab (для версій sysvinit, складених з бібліотек <5.2.18 або a.out бібліотеки, обмеження становить 2 символи).

Примітка. Для gettys або інших процесів входу в поле повинно бути суфікс tty відповідного tty, наприклад1 для tty1. Інакше облік входу може працювати неправильно.
