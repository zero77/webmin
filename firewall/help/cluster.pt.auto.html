Servidores de firewall de cluster
Esta página pode ser usada para enviar sua configuração de firewall para vários sistemas simultaneamente, para que você possa gerenciar um cluster de sistemas de firewall a partir de uma única interface. Para que um servidor possa ser adicionado a esta página, ele deve ter o Webmin 1.170 ou superior instalado e ser adicionado no módulo Índice de Servidores Webmin. Ele também deve ter os programas de tabelas ipi (6) necessários instalados e configurados corretamente.
