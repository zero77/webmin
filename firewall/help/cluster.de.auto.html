Cluster-Firewall-Server
Auf dieser Seite können Sie Ihre Firewall-Konfiguration gleichzeitig auf mehrere Systeme übertragen, sodass Sie einen Cluster von Firewall-Systemen über eine einzige Schnittstelle verwalten können. Bevor ein Server zu dieser Seite hinzugefügt werden kann, muss Webmin 1.170 oder höher installiert und im Webmin Servers Index-Modul hinzugefügt sein. Außerdem müssen die erforderlichen ipi (6) -Tabellenprogramme installiert und ordnungsgemäß konfiguriert sein.
