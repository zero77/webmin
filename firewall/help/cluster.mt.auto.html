Cluster Firewall Servers
Din il-paġna tista 'tintuża biex timbotta l-konfigurazzjoni tal-firewall tiegħek għal diversi sistemi fl-istess ħin, sabiex tkun tista' timmaniġġja gruppi ta 'sistemi tal-firewall minn interface waħda. Qabel ma server ikun jista 'jiżdied ma' din il-paġna, irid ikun installat Webmin 1.170 jew aktar u jkun miżjud fil-modulu ta 'Indiċi ta' Servers Webmin. Għandu jkollu wkoll il-programmi neċessarji tat- tabelli tal-ipi (6) installati u kkonfigurati sewwa.
