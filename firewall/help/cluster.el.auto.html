Διακομιστές τείχους προστασίας συμπλέγματος
Αυτή η σελίδα μπορεί να χρησιμοποιηθεί για να ωθήσει τη διαμόρφωση του τείχους προστασίας σε πολλά συστήματα ταυτόχρονα, έτσι ώστε να μπορείτε να διαχειριστείτε ένα σύμπλεγμα συστημάτων τείχους προστασίας από μία μόνο διεπαφή. Για να μπορέσει να προστεθεί ένας διακομιστής σε αυτήν τη σελίδα, πρέπει να έχει εγκατεστημένο το Webmin 1.170 ή παραπάνω και να προστεθεί στη μονάδα Webmin Servers Index. Πρέπει επίσης να έχει τα απαραίτητα προγράμματα ipi (6) για πίνακες εγκατεστημένα και σωστά διαμορφωμένα.
