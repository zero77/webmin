Server firewall cluster
Questa pagina può essere utilizzata per inviare la configurazione del firewall a più sistemi contemporaneamente, in modo da poter gestire un cluster di sistemi firewall da un'unica interfaccia. Prima di poter aggiungere un server a questa pagina, è necessario che Webmin 1.170 o versione successiva sia installato e che sia aggiunto nel modulo Indice server Webmin. Deve inoltre disporre dei programmi di tabelle ipi (6) necessari installati e configurati correttamente.
