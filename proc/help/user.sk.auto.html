Process Manager - User View
Táto stránka zobrazuje všetky spustené procesy, zoskupené podľa používateľa, ktorý vlastní proces. Procesy sú zoradené podľa využitia CPU pod menom každého užívateľa, ktorý má spustený jeden alebo viac procesov. Pre každý proces sú zobrazené PID, CPU a príkaz. Kliknutím na PID sa zobrazia ďalšie informácie.
