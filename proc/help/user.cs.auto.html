Process Manager - User View
Tato stránka zobrazuje všechny spuštěné procesy, seskupené podle uživatele, který vlastní proces. Procesy jsou seřazeny podle využití CPU pod jménem každého uživatele, který má spuštěn jeden nebo více procesů. Pro každý proces jsou zobrazeny PID, využití CPU a příkaz. Kliknutím na PID se zobrazí další informace.
