แก้ไขกระบวนการ
หน้านี้แสดงรายละเอียดของกระบวนการทำงาน ที่ด้านบนข้อมูลต่อไปนี้จะปรากฏขึ้น:
คำสั่ง
โปรแกรมและอาร์กิวเมนต์บรรทัดคำสั่งสำหรับกระบวนการนี้
ID กระบวนการ
ID กระบวนการที่ไม่ซ้ำกัน
กระบวนการปกครอง
กระบวนการที่สร้างสิ่งนี้ คลิกที่มันเพื่อแสดงรายละเอียดของผู้ปกครอง
ผู้ใช้งาน
ผู้ใช้ที่เป็นเจ้าของกระบวนการนี้และด้วยสิทธิ์ที่มันจะทำงาน
ใช้ CPU
เปอร์เซ็นต์ของเวลา CPU ที่กระบวนการนี้ใช้อยู่
ขนาด
จำนวนหน่วยความจำที่ครอบครองโดยกระบวนการนี้ บางส่วนนี้อาจใช้ร่วมกันโดยกระบวนการอื่น ๆ
เวลาทำงาน
จำนวนเวลา CPU ทั้งหมดที่ใช้โดยกระบวนการนี้ตั้งแต่เริ่มทำงานเป็นนาที ถ้ากระบวนการนั้นใช้งาน CPU สูงมากสิ่งนี้จะไม่เหมือนกับจำนวนนาทีที่ผ่านไปนับตั้งแต่เริ่มต้น
ระดับที่ดี
ลำดับความสำคัญที่กระบวนการนี้ทำงาน ข้อมูลเพิ่มเติม เกี่ยวกับระดับที่ดีสามารถใช้ได้
ข้อมูลเพิ่มเติมอาจแสดงขึ้นอยู่กับรุ่นของ Unix ที่คุณใช้งาน

ด้านล่างข้อมูลกระบวนการเป็นปุ่มสำหรับส่งสัญญาณไปยังกระบวนการ เลือกสัญญาณที่คุณต้องการส่งจากรายการจากนั้นคลิกปุ่ม ส่งสัญญาณ สัญญาณที่มีประโยชน์ที่สุดคือ KILL และ TERM เพื่อฆ่ากระบวนการ

หากกระบวนการนี้มีลูก ๆ พวกเขาจะปรากฏที่ด้านล่างของหน้า คลิกที่ ID จากรายการเพื่อแสดงข้อมูลรายละเอียด
