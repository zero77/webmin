Gestionnaire de processus - Vue de zone
Cette page affiche tous les processus en cours d'exécution, regroupés par zone Solaris dans laquelle ils se trouvent. Les processus sont classés par utilisation de l'UC sous le nom de chaque zone ayant un ou plusieurs processus en cours d'exécution. Pour chaque processus, le PID, l'utilisation du CPU et la commande sont affichés. Le PID peut être cliqué pour afficher plus d'informations.
