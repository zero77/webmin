ผู้จัดการกระบวนการ - มุมมองผู้ใช้
หน้านี้แสดงกระบวนการที่กำลังทำงานอยู่ทั้งหมดจัดกลุ่มตามผู้ใช้ที่เป็นเจ้าของกระบวนการ กระบวนการถูกจัดอันดับโดยการใช้งาน CPU ด้านล่างชื่อของผู้ใช้แต่ละคนที่มีกระบวนการทำงานอย่างน้อยหนึ่งกระบวนการ สำหรับแต่ละกระบวนการ PID การใช้งาน CPU และคำสั่งจะปรากฏขึ้น สามารถคลิก PID เพื่อแสดงข้อมูลเพิ่มเติม
