Χρήση συστήματος αρχείων
Αυτό το κριτήριο αναζήτησης εντοπίζει διαδικασίες που χρησιμοποιούν οποιοδήποτε αρχείο ή κατάλογο στο επιλεγμένο σύστημα αρχείων. Εάν μια διαδικασία έχει ένα αρχείο ανοιχτό για ανάγνωση ή γραφή ή βρίσκεται σε έναν κατάλογο στο επιλεγμένο σύστημα αρχείων, θα συμπεριληφθεί στα αποτελέσματα αναζήτησης. Αυτό μπορεί να είναι χρήσιμο για την εύρεση διαδικασιών που αποτρέπουν την αποπροσαρμογή ενός συστήματος αρχείων.
