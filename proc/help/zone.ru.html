Менеджер процессов
Эта страница отображает все процессы запущенные в вашей системе, сгруппированные по Solaris zone. Список процессов в каждой зоне отсортирован по степени использования CPU. Для каждого процесса отображается его PID, загрузка CPU и команда. Для просмотра дополнительной информации о процессе, можно щелкнуть на его PID.
