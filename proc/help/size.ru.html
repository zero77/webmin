Активные процессы - Память
Эта страница отображает список всех запущенных процессов, отсортированный по количеству использованной памяти. Для каждого процесса можно увидеть его PID, владельца, количество занимаемой памяти в килобайтах и команду. Для просмотра дополнительной информации о процессе, щелкните на его PID.

Этот вид может помочь при поиске команд, использующих чрезмерно много памяти. Тем не менее, использование памяти может быть ошибочно высоким для таких процессов, как например X-сервер, в используемую память которого, может входить отображаемый кадровый буфер. Несколько экземпляров одного процесса (такие как httpd или bash), могут совместно использовать одни и те же объекты в памяти, таким образом, общее количество занятой ими памяти может казаться больше, чем есть на самом деле.
