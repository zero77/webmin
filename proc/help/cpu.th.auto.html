ผู้จัดการกระบวนการ - ดู CPU
หน้านี้แสดงกระบวนการทำงานทั้งหมดในระบบของคุณเรียงลำดับตามการใช้งาน CPU สำหรับแต่ละกระบวนการ PID เจ้าของ CPU และคำสั่งจะปรากฏขึ้น สามารถคลิก PID เพื่อแสดงข้อมูลเพิ่มเติม

มุมมองนี้มีประโยชน์สำหรับการค้นหากระบวนการที่ใช้เวลา CPU มากเกินไป การใช้ประโยชน์ของแต่ละกระบวนการแสดงเป็นเปอร์เซ็นต์ของ CPU ทั้งหมดที่มี ซึ่งหมายความว่าในระบบปกติผลรวมของกระบวนการทั้งหมดจะน้อยกว่า 100% หากกระบวนการอย่างน้อยหนึ่งกระบวนการทำงานหนักบางงานจะใช้พลังงาน CPU ทั้งหมดของระบบของคุณ
