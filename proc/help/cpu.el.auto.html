Διαχειριστής επεξεργασίας - Προβολή CPU
Αυτή η σελίδα εμφανίζει όλη τη διαδικασία που εκτελείται στο σύστημά σας, ταξινομημένη με βάση τη χρήση της CPU. Για κάθε διαδικασία εμφανίζεται το PID, ο ιδιοκτήτης, η CPU και η εντολή. Μπορείτε να κάνετε κλικ στο PID για να εμφανίσετε περισσότερες πληροφορίες.

Αυτή η προβολή είναι χρήσιμη για την εύρεση διεργασιών που χρησιμοποιούν υπερβολικό χρόνο CPU. Η χρήση κάθε διαδικασίας εμφανίζεται ως ποσοστό της συνολικής διαθέσιμης CPU. Αυτό σημαίνει ότι σε ένα κανονικό σύστημα, το άθροισμα όλων των διαδικασιών θα είναι μικρότερο από 100%. Μόνο εάν μία ή περισσότερες διεργασίες εκτελούν κάποια εντατική εργασία θα χρησιμοποιηθεί όλη η ισχύς του CPU του συστήματός σας.
