Τρέχουσες Διαδικασίες - Προβολή Μνήμης
Αυτή η σελίδα εμφανίζει όλες τις τρέχουσες διαδικασίες, ταξινομημένες ανά μνήμη που χρησιμοποιείται. Για κάθε διαδικασία εμφανίζεται το PID, ο ιδιοκτήτης, το μέγεθος σε kB και η εντολή. Κάντε κλικ στο PID για να εμφανίσετε περισσότερες πληροφορίες σχετικά με μια διαδικασία που εκτελείται.

Αυτή η προβολή μπορεί να είναι χρήσιμη για την εύρεση των εντολών που χρησιμοποιούν υπερβολική ποσότητα μνήμης στο σύστημά σας. Ωστόσο, η χρήση μνήμης μπορεί να είναι παραπλανητικά υψηλή για διαδικασίες όπως ο διακομιστής X, οι οποίες μπορεί να περιλαμβάνουν αντιστοιχισμένα buffer καρέ στη χρήση της μνήμης τους. Πολλές παρουσίες της ίδιας διαδικασίας (όπως httpd ή bash ) μοιράζονται μεγάλο μέρος της μνήμης τους, κάνοντας τη συνολική τους χρήση να φαίνεται υψηλότερη από ό, τι είναι πραγματικά.
