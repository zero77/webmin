Process Manager - Visão do Usuário
Esta página exibe todos os processos em execução, agrupados pelo usuário que possui o processo. Os processos são classificados pela utilização da CPU abaixo do nome de cada usuário que possui um ou mais processos em execução. Para cada processo, o PID, o uso e o comando da CPU são exibidos. É possível clicar no PID para exibir mais informações.
