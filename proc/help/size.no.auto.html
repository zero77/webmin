Kjører prosesser - Minnevisning
Denne siden viser alle kjørende prosesser, rangert etter brukt minne. For hver prosess vises PID, eier, størrelse i kB og kommando. Klikk på PID for å vise mer informasjon om en prosess.

Denne visningen kan være nyttig for å finne hvilke kommandoer som bruker store mengder minne på systemet ditt. Imidlertid kan minnebruk være misvisende høy for prosesser som X-serveren som kan inkludere kartlagte rammebuffere i minnebruken. Flere forekomster av den samme prosessen (for eksempel httpd eller bash ) deler mye av minnet, slik at deres totale utnyttelse virker høyere enn den egentlig er.
