Διαχειριστής διεργασιών - Προβολή χρήστη
Αυτή η σελίδα εμφανίζει όλες τις τρέχουσες διαδικασίες, ομαδοποίηση ανά χρήστη που κατέχει τη διαδικασία. Οι διεργασίες ταξινομούνται κατά τη χρήση της CPU κάτω από το όνομα κάθε χρήστη που εκτελεί μία ή περισσότερες διεργασίες. Για κάθε διαδικασία εμφανίζεται το PID, η χρήση της CPU και η εντολή. Μπορείτε να κάνετε κλικ στο PID για να εμφανίσετε περισσότερες πληροφορίες.
