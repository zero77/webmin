Запуск працэсаў - прагляд памяці
На гэтай старонцы паказаны ўсе запушчаныя працэсы, ранжыраваныя па выкарыстанай памяці. Для кожнага працэсу адлюстроўваюцца PID, уладальнік, памер у кБ і каманда. Націсніце на PID, каб паказаць дадатковую інфармацыю аб запушчаным працэсе.

Гэта прадстаўленне можа быць карысна для пошуку, якія каманды выкарыстоўваюць празмерную колькасць памяці ў вашай сістэме. Аднак выкарыстанне памяці можа быць памылкова высокім для такіх працэсаў, як X-сервер, які можа ўключаць адлюстраваныя буферы кадраў пры выкарыстанні памяці. Некалькі асобнікаў аднаго працэсу (напрыклад, httpd або bash ) падзяляюць вялікую частку сваёй памяці, дзякуючы чаму іх агульная загрузка выглядае вышэй, чым ёсць на самай справе.
