Λειτουργία εκτέλεσης
Εκτελέστε στο παρασκήνιο
Η εντολή θα ξεκινήσει στο παρασκήνιο και το πρόγραμμα περιήγησής σας θα κατευθυνθεί στη λίστα των διαδικασιών που εκτελούνται.

Περιμένετε μέχρι να ολοκληρωθεί
Η εντολή θα εκτελεστεί και θα εμφανιστεί η έξοδος της. Αυτή η επιλογή δεν πρέπει να χρησιμοποιείται για εντολές που εκτελούνται για πάντα, όπως διαδικασίες διακομιστή.
