Prozessmanager - CPU-Ansicht
Diese Seite zeigt alle laufenden Prozesse auf Ihrem System, geordnet nach CPU-Auslastung. Für jeden Prozess werden PID, Eigentümer, CPU und Befehl angezeigt. Die PID kann angeklickt werden, um weitere Informationen anzuzeigen.

Diese Ansicht ist nützlich, um Prozesse zu finden, die übermäßig viel CPU-Zeit verbrauchen. Die Auslastung jedes Prozesses wird als Prozentsatz der insgesamt verfügbaren CPU angezeigt. Dies bedeutet, dass auf einem normalen System die Summe aller Prozesse weniger als 100% beträgt. Nur wenn ein oder mehrere Prozesse eine intensive Aufgabe ausführen, wird die gesamte CPU-Leistung Ihres Systems genutzt.
