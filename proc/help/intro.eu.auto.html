Unix Prozesuak
Prozesua zure sisteman exekutatzen ari den programa bat da. Zure web arakatzailea, leiho kudeatzailea, terminal leihoa eta X zerbitzaria zuzenean elkarreragiten dituzun prozesuak dira. Beste prozesu asko atzeko planoan exekutatzen dira, hala nola web zerbitzariak eta sistemako beste zeregin batzuk. Prozesu berri bat sortzen da ls edo pwd bezalako komando bat idazten duzun bakoitzean, hala ere, prozesu horiek iraupen laburrekoak izan ohi dira.

Prozesu bakoitzak ID bakarra du, prozesuaren IDa edo PID izenekoa. Aldi berean exekutatzen den prozesu bakoitzak ID desberdinak dituen bitartean, denboran zehar PIDak berriro erabil daitezke.

Hasierako prozesua alde batera utzita ( init izenarekin normalean) bakoitzak sortutako prozesua guraso bat du. Adibidez, vi zure shell gonbidapenetik exekutatzen baduzu, vi gurasoen prozesua zure maskorra izango da. Prozesu batek edozein seme-alaba izan ditzake, baina guraso bakarra.

Prozesu bakoitza erabiltzaile eta talde batzuen baimenekin exekutatzen da, fitxategiak eta direktorioetara sartzean aplikatzen direnak. Erabiltzaileek eta prozesuek jabea duten beste prozesu batzuk baino ezin dituzte hiltzen, edozer hil dezakeen root-a izan ezik.
