กระบวนการ Unix
กระบวนการเป็นเพียงโปรแกรมที่ทำงานอยู่บนระบบของคุณ เว็บเบราว์เซอร์ตัวจัดการหน้าต่างหน้าต่างเทอร์มินัลและเซิร์ฟเวอร์ X เป็นกระบวนการทั้งหมดที่คุณโต้ตอบโดยตรง กระบวนการอื่นจำนวนมากทำงานในพื้นหลังเช่นเว็บเซิร์ฟเวอร์และงานระบบอื่น ๆ กระบวนการใหม่จะถูกสร้างขึ้นทุกครั้งที่คุณพิมพ์คำสั่งเช่น ls หรือ pwd แม้ว่ากระบวนการดังกล่าวมักจะมีอายุสั้น

ทุกกระบวนการมี ID เฉพาะที่เรียกว่า ID กระบวนการหรือ PID ในขณะที่ทุกกระบวนการทำงานในเวลาหนึ่งมี ID ที่แตกต่างกันอาจใช้ PID ในช่วงเวลาหนึ่ง

นอกเหนือจากกระบวนการเริ่มต้น (โดยทั่วไปเรียกว่า init ) แต่ละกระบวนการมีกระบวนการหลักที่สร้างขึ้น ตัวอย่างเช่นถ้าคุณรัน vi จาก shell prompt กระบวนการหลักของ vi จะเป็น shell ของคุณ กระบวนการสามารถมีลูกได้จำนวนเท่าใดก็ได้ แต่มีเพียงหนึ่งรายการหลักเท่านั้น

แต่ละกระบวนการทำงานโดยได้รับอนุญาตจากผู้ใช้และกลุ่มซึ่งใช้เมื่อเข้าถึงไฟล์และไดเรกทอรี ผู้ใช้และกระบวนการอาจฆ่ากระบวนการอื่น ๆ ที่พวกเขาเป็นเจ้าของด้วยข้อยกเว้นของ root ที่สามารถฆ่าสิ่งใด ๆ
