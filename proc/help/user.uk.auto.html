Менеджер процесів - Перегляд користувача
На цій сторінці відображаються всі запущені процеси, згруповані за користувачем, якому належить процес. Процеси класифікуються по використанню процесора нижче імені кожного користувача, у якого працює один або кілька процесів. Для кожного процесу відображаються PID, використання процесора та команда. PID можна натиснути, щоб відобразити більше інформації.
