Gerenciador de processos - exibição da CPU
Esta página mostra todo o processo em execução no seu sistema, ordenado pela utilização da CPU. Para cada processo, o PID, proprietário, CPU e comando são exibidos. É possível clicar no PID para exibir mais informações.

Essa visualização é útil para localizar processos que estão usando quantidades excessivas de tempo de CPU. A utilização de cada processo é mostrada como uma porcentagem do total de CPU disponível. Isso significa que, em um sistema normal, a soma de todos os processos será menor que 100%. Somente se um ou mais processos estiverem executando alguma tarefa intensiva será usada toda a energia da CPU do seu sistema.
