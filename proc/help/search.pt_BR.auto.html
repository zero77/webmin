Process Manager - Formulário de Pesquisa
Este formulário permite procurar processos que correspondam a alguns critérios. Quando o botão Pesquisar é clicado, uma lista de processos correspondentes será exibida abaixo do formulário. Para cada um, o PID, proprietário, uso da CPU e comando são exibidos. Clique no PID para exibir mais informações sobre um processo.

Abaixo da lista de resultados da pesquisa, há um botão para enviar um sinal para os processos correspondentes. Selecione o sinal que você deseja enviar na lista e clique no botão Enviar sinal . Os sinais mais úteis são KILL e TERM , para eliminar os processos em execução.
