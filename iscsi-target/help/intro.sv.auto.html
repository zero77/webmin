iSCSI-mål
Den här modulen låter dig exportera diskenheter till andra system via nätverket med iSCSI-protokollet. Varje exporterad disk kallas ett mål , och befintliga mål listas på modulens startsida. För att skapa ett nytt mål, klicka på Lägg till ett nytt iSCSI- målknapp, ange ett namn och välj den diskenhet eller fil som ska exporteras.

Inloggningar och lösenord som krävs från klienter kan antingen ställas in per mål eller för alla mål på sidan Autentiseringsinställningar . ISCSI-protokollet tillåter också mål att autentisera sig till klienter (även känd som initiatorer).

Globala inställningar relaterade till paketstorlekar och iSCSI-protokollet kan ändras på sidan Anslutningsinställningar . Timeouts för kommunikation med klienter kan ändras på sidan Timeout-inställningar .

Som standard kan alla klienter med giltig inloggning ansluta till dina exporterade mål. Du kan dock begränsa IP: erna klienter kan ansluta från med sidan Tillåtna klientadresser . Alternativt kan du begränsa IP: erna på servern som klienter kan ansluta till med hjälp av sidan Tillåtna serveradresser .

Längst ner på modulens startsida finns knappar för att stoppa, starta och starta om iSCSI-servern. Konfigurationsändringar kommer inte att tillämpas förrän servern startas om.
